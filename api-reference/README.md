# HarmonyOS API Reference

HarmonyOS 开发者 API 参考文档镜像。

数据来源: https://developer.huawei.com/consumer/cn/doc/harmonyos-references/

## 同步

```bash
python3 sync.py           # 全量同步
python3 sync.py --sample 10  # 测试模式
```
