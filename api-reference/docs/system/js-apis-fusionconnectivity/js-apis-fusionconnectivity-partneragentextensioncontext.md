# @ohos.FusionConnectivity.PartnerAgentExtensionContext (设备状态通知能力上下文)

PartnerAgentExtensionContext模块是三方外设的发现和连接管理功能的上下文。

![](./img/note_3.0-zh-cn.png)

- 本模块接口从API version 23开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。
- 仅可在Stage模型下使用。
