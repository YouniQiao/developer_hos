# ParamListNode*

```
typedef struct ParamListNode* ParamList
```

#### 概述

事件参数列表节点。用于组织和管理事件参数列表信息，通过ParamListNode可以构建参数链表，支持多参数事件的参数传递。

起始版本： 8
