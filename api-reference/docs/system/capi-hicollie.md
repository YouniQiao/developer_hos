# HiCollie

#### 概述

提供检测业务线程卡死、卡顿，以及上报卡死事件的能力。本模块函数可用于：

（1）注册应用业务线程卡死的周期性检测任务；

（2）注册应用业务线程卡顿检测的回调函数；

（3）上报应用业务线程卡死事件。

起始版本： 12
