# OH_ComposeOptions

```
typedef struct OH_ComposeOptions OH_ComposeOptions
```

#### 概述

OH_ComposeOptions是native层封装的合成HDR选项参数结构体，用于设置合成选项参数。用于指定合成HDR所用的参数，例如目标像素格式。

起始版本： 23
