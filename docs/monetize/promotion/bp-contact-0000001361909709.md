# 联系方式

当您使用应用推广功能遇到问题时，可以按照以下顺序寻求帮助。

1. 通过[智能客服](https://developer.huawei.com/consumer/cn/customerService/#/bot-dev-top/faq-top/faq-talk-top)查找问题解决方案。
2. 添加客服企业QQ：4000886183，寻求解决方案。

   ![](./img/945d3155b642.png) 

   客服企业QQ仅用于咨询应用推广相关问题。
3. 可以通过[在线工单系统](https://developer.huawei.com/consumer/cn/support/feedback/#/add/13?level2=111)与我们进行联系，提交后的工单可在开发者联盟页面，点击右上角头像图标，选择“我的客服”，查看工单处理进展。
4. 如果以上方法仍未解决您的问题，发送邮件至邮箱[developer@huawei.com](mailto:developer@huawei.com)，联系对口商务或客服。
