# 概述

延迟Deeplink的推广任务投放成功后，您可以通过<strong>[客户端(方式一)](/docs/monetize/promotion/bp-functions-deeplink-inquire-client-0000001348653289)</strong>和<strong>[API接口(方式二)](/docs/monetize/promotion/bp-functions-deeplink-inquire-api-0000001296132894)</strong>两种方式查询。

两种方式对比如下：

| <strong>查询方式</strong> | <strong>是否需要发布新版本</strong> | <strong>开发工作量</strong> |
| --- | --- | --- |
| <strong>[客户端(方式一)](/docs/monetize/promotion/bp-functions-deeplink-inquire-client-0000001348653289)</strong> | 是 | 稍大 |
| <strong>[API接口(方式二)](/docs/monetize/promotion/bp-functions-deeplink-inquire-api-0000001296132894)</strong> | 否 | 稍小 |

请根据实际需要选择对应的方式查询。
