# 查询商品库分享情况

【简介】查询经理账户某一个商品库的分享情况

<strong>请求地址</strong>

https://ads.cloud.huawei.com/openapi/v2/tools/dpa/product/share\_store\_detail/query

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| manager\_id | string | 是 | 经理账户 |
| store\_id | string | 是 | 商品库ID |

<strong>请求示例</strong>

POST openapi/v2/tools/dpa/product/share\_store\_detail/query HTTP/1.1

Accept:application/json

Content-Type:application/json

Authorization:Bearer CgB6e3x9ERGComr9dENxZX22iBk+mLuf1yGtQVPUjPJUMrstfKlqpdXk+kfHU9J8ZJ/soYIZHZzT446GeSYumluQuhsK7jvz4kz1Bkms4CLI/rE=

```
{
"manager_id": "659318074012379008",
"store_id": "45001295"
}
```

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1 | 商品库列表 |

data（Struct1）定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| manager\_id | string | 经理账户 |
| store\_id | string | 商品库ID |
| advertiser\_ids | string[] | 该商品库共享的广告主id集合，不包括该商品库的所有者 |
