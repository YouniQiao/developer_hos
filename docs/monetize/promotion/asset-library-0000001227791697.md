# 素材库

您可以在“<strong>工具</strong>“-&gt;“<strong>素材库</strong>”自行上传图片或视频素材到素材库并编辑素材名称，方便您在创建广告任务时直接使用；通过审核的创意素材会自动保存到素材库。

![](./img/zh-cn_image_0000001182553580_b934ce67781a.png)

- 图片素材：您可以上传/删除图片素材。
  - 类型：JPG, JPEG, PNG, GIF
  - 文件大小：不超过500KB
  - 尺寸：960\*150, 960\*300, 900\*750, 1404\*180, 2184\*270, 1080\*170, 1080\*432, 640\*360, 160\*160, 1080\*607, 225\*150, 216\*216, 450\*450, 1080\*1920, 1080\*1620, 1920\*1080, 1080\*150, 1080\*270, 1080\*96, 720\*1440, 720\*1280, 1248\*400, 2022\*486, 256\*256, 400\*400, 120\*120, 984\*422, 640\*960, 1080\*430, 1280\*1665, 392\*392, 2496\*468, 1440\*2560, 2934\*3306, 320\*50, 210\*150, 312\*432, 608\*608, 1080\*310, 1080\*504, 1312\*768, 640\*256, 640\*184, 480\*800, 800\*450, 250\*250, 728\*90, 300\*250, 336\*280, 1045\*396, 1045\*224, 334\*136, 1200\*420, 285\*120, 2200\*3120, 880\*552, 1248\*534, 576\*720, 192\*192, 300\*50, 320\*480, 480\*320, 1024\*768, 768\*1024。
- 视频素材：您可以上传/删除视频。
  - 类型：封装为H.264编码的MP4视频格式
  - 文件大小：不超过100MB
  - 尺寸：640\*360, 720\*1280, 720\*1080, 1280\*720, 1920\*1080, 720\*1440, 880\*552, 320\*480, 480\*320, 640\*480, 480\*640, 1080\*1920。
