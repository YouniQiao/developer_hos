# 推广产品

鲸鸿动能广告平台支持推广多类型产品，如Android应用、网页、商品等，具体如下：

| 营销目标 | 计划类型 | 推广产品 |
| --- | --- | --- |
| 销售转化 | 展示广告 | 网页、Android应用、快应用/快游戏 |
| 搜索广告 | Android应用 |
| 商品广告 | 商品 |
| 品牌认知度 | 展示广告 | 网页、Android应用 |
| 智能应用推广 | 应用广告 | Android应用（仅支持AG应用） |
| 无明确目的导向 | 展示广告 | 网页、Android应用、快应用/快游戏 |
| 搜索广告 | Android应用 |
| 商品广告 | 商品 |
| 应用广告 | Android应用（仅支持AG应用） |
