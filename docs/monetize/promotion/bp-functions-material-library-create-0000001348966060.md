# 新建素材库内容

1. 登录[华为应用市场应用推广平台](https://ads.huawei.com/cn/)， 在顶部菜单栏点击【工具】页签，确认推广范围为“应用市场应用推广”。选择“资产管理”——“素材库”。

   ![](./img/zh-cn_image_0000002583371071_9b9014a24227.png "点击放大")
2. 在当前页面点击“上传素材”。

   ![](./img/zh-cn_image_0000002583291079_1d4c4113074d.png "点击放大")
3. 在弹框中按照图片规格要求上传素材。完成后，点击“确定”。

   ![](./img/note_3.0-zh-cn_69fb636aa0fa.png) 

   一次最多可上传20张图片。

   ![](./img/zh-cn_image_0000002552731084_a662407c1bb2.png "点击放大")
