# DeliveryHour

| 参数 | 必选(M)/可选(O) | 类型 | 描述 |
| --- | --- | --- | --- |
| day | M | String | 填写具体星期。  取值范围：   - Monday - Tuesday - Wednesday - Thursday - Friday - Saturday - Sunday |
| startHMs | M | String | 投放开始时段。  格式：hhmm |
| endHMs | M | String | 投放结束时段。  格式：hhmm |
