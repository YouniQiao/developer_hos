# 2025年7月

<strong>【规则放宽】医疗健康行业-素材样式、广审表可修改范围</strong>

1.支持素材添加无利益关系的装饰性元素，如：弧线设计、色块突出、气泡框、色块突出。

2.支持素材更改背景颜色、文字字体、图片规格，如：横版图改竖版。

3.支持在素材中增加或更改地址、联系电话、网址。

4.支持删减已过审的样件表素材，但需明确推广内容。

【规则调整】通用-虚假诱导类

1.不得模拟手机系统故障或恶意软件感染的视觉呈现方式，或通过图案或文案的冗余堆砌来对用户造成感官干扰。

2.不得使用按钮设计作为素材主视觉，包含但不限于：上下条按钮、单双圈按钮、矩形按钮等。

3.素材使用引导用户操作的行为文案作为主视觉，人工对创意评分为1分。
