# 常见问题

1. <strong>充值时银行转账主体是否需要和账户注册的主体保持一致吗？</strong>

   您的银行转账主体需和鲸鸿动能广告账户主体保持一致。若不一致，将导致充值失败。请在转账前仔细确认主体信息。
2. <strong>我的实际充值币种和账户的默认币种不一致怎么办？</strong>

   若您的实际充值币种和账户币种不一致，我们会根据到款时的汇率转换成账户币种为您充值，因汇率带来的损益需由您承担，因此请尽量使用账户的默认币种充值。
3. <strong>广告费充值金额有要求吗？</strong>

   您好，对于充值金额不做限制，但出于对广告效果的保证，建议您预留较为充足的广告预算。
4. <strong>广告费充值后，发票怎么获取？</strong>

   按照您鲸鸿动能广告账户的注册地不同，我们将按照充值金额或月消耗金额开具发票，具体请参考[发票信息](/docs/monetize/promotion/invoice-0000001051704326)。
5. <strong>如何申请开具形式发票？</strong>

   如果您尚未充值，需要在充值前申请开具发票支撑充值转账，您可以申请形式发票，形式发票不可替代正式发票，详情请参考[形式发票申请](/docs/monetize/promotion/invoice-0000001051704326#形式发票申请)。
6. <strong>银行账户转账充值，完成银行转账后充值金额迟迟未充值到广告账户，如何解决？</strong>

   当完成银行转账后，还需要在鲸鸿动能广告平台提交充值申请。在充值界面右上角单击![](./img/zh-cn_image_0000001758706573_9023e6ab9948.png)-&gt;“<strong>充值</strong>”提交充值申请；填写到账金额及充值备注,上传付款凭证；确认充值信息并提交，等待鲸鸿动能广告平台审核。

   一般情况下，银行转账1-3个工作日到账，到账后鲸鸿动能广告平台会在3个工作日内完成充值审批，提交充值申请后关注申请状态。
7. <strong>充值订单审核被拒绝，提示审核不通过</strong> <strong>，如何解决？</strong>

   请按照审核拒绝理由修改充值订单并提交审核；若有其他充值问题请通过[在线提单](https://developer.huawei.com/consumer/cn/support/feedback/#/)处理，并附上广告账户信息、充值主体名称和带有付款主体的付款凭证。
