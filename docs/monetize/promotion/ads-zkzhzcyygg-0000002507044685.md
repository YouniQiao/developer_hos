# 直客账户注册

![](./img/note_3.0-zh-cn_022efdda037b.png) 

以下内容为同时开通推广范围为“展示广告网络”和“应用市场应用推广”双权限的直客账户，如只需开通单一权限直客账户根据页面提示自行选择即可。

## 直客账户开通步骤

1. <strong>注册华为账号</strong>

   1）登录鲸鸿动能官网[https://ads.huawei.com](https://ads.huawei.com/)（建议您使用Chrome浏览器），单击页面右上角“<strong>立即开始</strong>”。

   ![](./img/zh-cn_image_0000002474933096_fd5642b9eefc.png "点击放大")

   2）进入华为账号注册界面，可选择手机号注册或者邮箱地址注册方式，注册完成后跳转至登录界面，登录该账号。

   若您的手机号此前已经注册过华为账号，输入验证码之后将会弹出“<strong>此号码已被注册，请登录</strong>”弹窗，此时请单击弹窗中的<strong>“登录”</strong>，若忘记密码，可通过<strong>‘找回密码’</strong>功能重置后登录。

   ![](./img/zh-cn_image_0000002516362832_dee258809c35.png "点击放大")

   3）账号登录成功后，跳转至“<strong>选择投放区域</strong>”界面，请选择“<strong>中国大陆</strong>”区域 。

   ![](./img/zh-cn_image_0000002547937253_308b54c62e53.png "点击放大")

   4）区域选择完成后，跳转至“<strong>请选择您的企业类</strong> <strong>型</strong>”界面，请选择“<strong>直客广告主</strong>”账户类型。

   ![](./img/zh-cn_image_0000002547937287_62ba2cb41f9d.png "点击放大")

   5）账户类型选择完之后，需要您选择推广类型和签署协议，勾选完协议之后请单击“确定并进入认证中心”进行推广类型勾选。

   ![](./img/zh-cn_image_0000002554975124_5cc2006c94e9.png "点击放大")

   推广类型可以选择“应用市场应用推广”和“展示广告网络”。

   ![](./img/zh-cn_image_0000002548018471_d921e6a5fd7a.png "点击放大")
2. <strong>实名认证</strong>

   完成华为账号注册之后，需要进行实名认证，请根据实际情况，认证方式选择“<strong>个人开发者</strong>”或“<strong>企业开发者</strong>”，认证方式提交后不可修改。

   ![](./img/zh-cn_image_0000002548060509_a8f6a22b16be.png "点击放大")

   ![](./img/note_3.0-zh-cn_2c3e8b192cfe.png) 

   <strong>个人开发者：</strong>

   账号归属个人，适用群体为个人、个体工商户

   支持人脸识别认证、个人银行卡认证、身份证人工审核认证等

   需年满18周岁以上

   同一个人信息只能在华为开发者联盟实名认证1个账号

   <strong>企业开发者：</strong>

   账号归属企业，适用群体为企业、党政及国家机关、事业单位、民办非企业单位、社会团体、社会组织

   支持企业对公账户打款认证、企业资料+法人扫脸认证等

   支持开增值税专用发票

   <strong>如果您是“个人开发者认证”</strong>

   您可以选择两种认证方式：人脸识别认证（即时完成）或个人银行卡认证（即时完成）。

   ![](./img/zh-cn_image_0000002548100519_53b6cec1696f.png "点击放大")

   - 方式一：人脸识别认证，您需要完善个人实名认证信息，首先需要通过微信或华为浏览器扫描二维码进行人脸识别，人脸识别完成之后，根据实际情况填写联系人邮箱，联系人手机，所在地区等页面信息，勾选相关协议单击“下一步”等待审核通过即可。

   ![](./img/zh-cn_image_0000002516380706_d75fcea63889.png "点击放大")

   ![](./img/zh-cn_image_0000002516540624_07bec2deddc0.png "点击放大")

   - 方式二：个人银行卡认证，如您选择的是个人银行卡实名认证，您需要填写真实姓名，身份证号码，银行卡号以及联系人手机，然后单击“下一步”补充资料。

   ![](./img/zh-cn_image_0000002548060519_edc8ef4b9b53.png "点击放大")

   根据页面提示补充完资料单击“下一步”等待审核通过即可。

   ![](./img/zh-cn_image_0000002548100527_d75b5a97ac91.png "点击放大")

   ![](./img/zh-cn_image_0000002516380716_dee22267e071.png "点击放大")
