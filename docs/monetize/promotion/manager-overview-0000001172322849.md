# 概述

同一公司使用多个账户进行广告投放时，希望能够对多个账户统一管理。通过给广告账户绑定经理账户，您可以在登录经理账户后方便的在各个广告账户之间切换登录并进行数据查看等操作。注册经理账户时需要提供公司的邓白氏编码或营业执照完成公司信息的验证，注册完成后可以关联同一公司注册的直客账户或子客账户进行管理。经理账户一共可以绑定1000个广告账户，直客与子客均可以使用经理账户。

<strong>经理账户</strong> <strong>使用限制：</strong>

- 同一经理账户管理的广告账户只能全部是直客账户或者全部是子客账户，无法同时管理两种广告账户类型。在管理直客账户时，要求管理的所有直客账户站点、币种相同。在管理子客账户类型时，要求管理的所有子客账户必须为同一个L2服务商下开户的子客账户且子客的站点、币种必须一致。
- 同一经理账户只能管理归属同一运营站点的广告账户，鲸鸿动能广告根据账号注册国家划分4个运营站点：亚非拉、欧洲、俄罗斯、中国。鲸鸿动能广告业务的站点，由华为数据中心统一规划。
- 一个广告账户只能被关联到一个经理账户，且只有审核状态正常的广告账户才能被关联到经理账户，审核中、已停用的广告账户不支持关联。
- 当经理账户的注册地为中国大陆时，币种只能选择人民币，无法绑定币种为非人民币的直客或者子客。

<strong>经理账户具体功能如下</strong>：

| 功能模块 | 功能介绍 |
| --- | --- |
| 账户访问 | 访问经理账户管理的各个广告账户 |

<strong>使用经理账户管理子客账户时，其与L2服务商的权限差异如下：</strong>

| 功能 | 经理账户 | 服务商账户 |
| --- | --- | --- |
| 关联不同站点账户 | - | √ |
