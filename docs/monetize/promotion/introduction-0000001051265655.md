# 鲸鸿动能广告简介

鲸鸿动能是华为终端智慧营销平台，依托于华为1+8+N全场景生态及自有媒体营销能力，汇聚全球三方优质流量，以丰富的商业资源和强大的广告技术，为全球广告主和三方网盟提供广告营销和流量变现服务，助力伙伴实现商业增长的同时提升品牌价值。覆盖200+广告主行业，截至2025年9月30日，全球超8.5万个应用集成Ads Kit。

- 鲸鸿动能广告政策请参考：[广告审核规范](/docs/monetize/promotion/overview-guangaoshenheguifan-0000001188925990)。
- 投放指导更新点请查看：[新功能和公告](/docs/monetize/promotion/adsrizhi-0000001143181107)。

![](./img/note_3.0-zh-cn_59bee5bae968.png) 

鲸鸿动能广告服务仅对企业开放注册，暂不支持个人注册广告账户，注册地在非中国大陆区域的个人开发者可以通过[服务商](/docs/monetize/promotion/partnerinformation-0000001058393913)投放广告。
