# 查询维纳斯线索列表

【简介】广告主获取维纳斯平台的线索列表，线索来源于维纳斯落地页。

<strong>请求地址</strong>

https://ads.cloud.huawei.com/ads/v1/tools/landingpage/lead/query

<strong>请求方法</strong>

<strong>GET</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| page | integer | 是 | 搜索页码，大于等于1，默认值：1 |
| page\_size | integer | 是 | 一页展示数量，取值范围 10~50，默认值: 10 |
| filtering | Struct | 是 | 过滤条件 |

filtering (Struct)定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>必选</strong> | <strong>描述</strong> |
| campaign\_id | string | 否 | 计划ID，  最大长度不得超50；  只能填写数字；  不能使用“^”和换行； |
| adgroup\_id | long | 否 | 任务ID |
| creative\_id | long | 否 | 创意ID |
| lead\_status | string | 否 | 线索状态，详见[【线索状态】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section1079462524412) |
| lead\_type | string | 是 | 线索类型，详见[【线索类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section148761437443) |
| created\_begin\_time | string | 否 | 线索创建的开始时间，  格式如下:  "yyyy-MM-dd HH:mm:ss"； |
| created\_end\_time | string | 否 | 线索创建的结束时间，  格式如下:  "yyyy-MM-dd HH:mm:ss"； |

<strong>请求示例</strong>

GET /ads/v1/landingPage/lead/query HTTP/1.1

Accept:application/json

Content-Type:application/json

Authorization:Bearer CgB6e3x9ERGComr9dENxZX22iBk+mLuf1yGtQVPUjPJUMrstfKlqpdXk+kfHU9J8ZJ/soYIZHZzT446GeSYumluQuhsK7jvz4kz1Bkms4CLI/rE=

```
{
"page": 1,
"page_size": 10,
"filtering": {
"lead_status": "LEAD_STATUS_INVALID",
"lead_type": "LEAD_TYPE_FORM_SUBMIT"
}
}
```

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1 | 线索列表 |

Struct1定义

|  |  |  |
| --- | --- | --- |
| <strong>类型</strong> | <strong>参数名称</strong> | <strong>描述</strong> |
| total | integer | 总条数 |
| data | Struct2[] | 线索列表 |

data (Struct2)定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| campaign\_id | string | 计划ID |
| adgroup\_id | long | 任务ID |
| creative\_id | long | 创意ID |
| landing\_page\_id | string | 落地页ID |
| landing\_page\_name | string | 落地页名称 |
| landing\_page\_url | string | 落地页地址 |
| marketing\_com\_id | string | 营销组件ID |
| lead\_id | string | 线索ID |
| lead\_type | string | 线索类型，详见[【线索类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section148761437443) |
| lead\_status | string | 线索状态，详见[【线索状态】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section1079462524412) |
| created\_time | string | 线索提交时间 |
