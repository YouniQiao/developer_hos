# 查询任务数据报表

1. 登录[华为应用市场应用推广平台](https://ads.huawei.com/cn/)，选择“报表”。

   ![](./img/b0ce9024efd1.png)
2. 您可以在报表中选择某一个智能分包查看投放情况。另外在报表查询结果中也会显示渠道包名和渠道号信息。

   ![](./img/e4a69ae39c6e.png "点击放大")
