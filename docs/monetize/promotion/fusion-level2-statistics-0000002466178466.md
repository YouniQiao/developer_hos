# 推广统计

原客户投放伙伴子账户的数据管理界面，整合升级后入口为：服务商管理平台——推广统计——消耗统计，可查看维度不变。

您可以通过消耗统计页面查看客户投放伙伴子账户的总消耗、总曝光、总下载，以及投放操作账户的消耗明细数据。
