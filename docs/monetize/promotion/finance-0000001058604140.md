# 服务商财务管理

## 概述

**广告账户成功注册后，您需要对广告账户进行充值，才能成功投放广告。当前仅服务商账户可以充值，充值完毕后转账给下一级。**

**广告账户中有多个余额类型，各余额类型的扣费顺序为：纯赠送金\_华为自有媒体>纯赠送金\_通用>充值赠送金\_华为自有媒体>充值赠送金\_通用>返利金(阿斯比格/华为服务(香港))\_华为自有媒体>返利金（阿斯比格/华为服务(香港)）\_通用>现金>授信。**

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ab/v3/dxetb828Qv-uaEd0oo50MA/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260530T050413Z&HW-CC-Expire=86400&HW-CC-Sign=8C03024DA8C51EF1E2E5718D40124590BBD03768ABA31048F463F2E9D216FA19) 

- 充值赠送金需搭配与华为商定达成的比例，与其他余额类型（现金+返利金+授信）一起进行消耗。
- 纯赠送金支持单独消耗，单独转账，不受按比例消耗限制，纯赠送金不会影响子客账户余额平衡。

## 充值方式

- **服务商系统中，服务商可以通过[线下充值](#section14769552165611)和[线上充值](#section395361975716)的方式进行充值，仅服务商账户可以充值，子客服务商账户与子客账户均不可充值，充值完成后，服务商需要[转账](#section17794102114232)给下一级，只有子客账户有余额才能正常投放广告。**
  - **[线下充值](#section14769552165611)：**您可以通过银行转账到[指定银行账户](#ZH-CN_TOPIC_0000001058604140__li136043713466)的方式进行充值，银行转账之后需要您在服务商账户中提交充值申请，审批通过后充值进现金账户余额。
  - **[线上充值](#section395361975716)**：您可以通过线上充值的方式在广告投放之前先充值，金额将会充值到现金账户余额。
- **充值的银行账号信息如下：**
  - 鲸鸿动能广告平台提供阿斯比格和华为服务（香港）两个银行收款账户，您需要根据主要投放区域选择阿斯比格或华为服务（香港）充值。
  - 您的充值币种需要与您服务商账户注册时选择的币种保持一致。账户的计费也按照该币种计算。例如：您账户注册时选择的币种是美元，则您需要用美元充值，您的账户计费也将以美元计算。

### 服务商充值-线下充值

**操作流程：**

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/f4/v3/KdpQZsgEShKhtN1mH2rRVg/zh-cn_image_0000001209990538.png?HW-CC-KV=V1&HW-CC-Date=20260530T050413Z&HW-CC-Expire=86400&HW-CC-Sign=F353FAE6366E659A1F3314C60A5254688ADF5A315D174157000C9F407040BA62)

**操作步骤：**

1. 银行转账。

   您需要使用您的企业银行账户转账至鲸鸿动能广告平台提供的[银行账户](#ZH-CN_TOPIC_0000001058604140__li136043713466)，转账币种需与您服务商账户注册时选择的币种保持一致。为保障广告计划顺利创建并投放，选择线下银行账户转账模式，请提前与银行确认转账时效（打款到账时间为3-5个工作日）。

   ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/1a/v3/yaUFitKXRKaWvuHv0qrnAQ/notice_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260530T050413Z&HW-CC-Expire=86400&HW-CC-Sign=D709804B4A18FC0BFCD60FA805D3782A0047C7C3C624D114B12BC3AB22A5A797) 

   - 线下充值不支持个人银行账户充值。
   - 线下充值仅支持企业银行账户充值，且您用来充值的银行账户必须与广告账户主体一致，否则将无法充值到广告账户中。
2. 服务商平台提交充值申请并提交审核。

   **单击“财务管理”->"充值开票****"**，填写相关信息，补充转账凭证，并提交审核，审核时间为1-3个工作日。

   ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/39/v3/_OSrKWEuRki9BRgWLSGvHw/zh-cn_image_0000001481287896.png?HW-CC-KV=V1&HW-CC-Date=20260530T050413Z&HW-CC-Expire=86400&HW-CC-Sign=D319F46410D53F922741F75BD94423D213869BA509B1D889EBC57052F65A1ACF "点击放大")

   ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b1/v3/NrwvEM_kQW6S91Mx2kSnWw/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260530T050413Z&HW-CC-Expire=86400&HW-CC-Sign=F84A87718D160BD159D1615790732FBA8144627B0A3CB7C8857B48ED23917A81) 

   - 如果您一次性线下充值少于1000美元或1000欧元，银行会收取一定的手续费，手续费的金额需要您与银行确认。实际到款金额与汇款金额不一致。提交充值申请时，需要填写实际到账的金额。
   - 如果您一次性线下充值超过1000美元或1000欧元，华为承担中间行转账手续费。
   - 涉及退款时，华为仅退还实际到账金额，详情可参考[鲸鸿动能广告协议](https://developer.huawei.com/consumer/cn/doc/promotion/ad-agreements-0000001169499170#ZH-CN_TOPIC_0000001169499170__li2061184515243)中“预付费”内容。
3. 充值到账。

   充值订单审核完成后，您可以在首页看到余额，也可以在“**财务管理**“->"**充值记录**"中查看充值记录。

## 授信账户

- 授信账户：指的是您（服务商）通过向鲸鸿动能广告平台申请开通授信账户（授信额度存在有效期），并分配一定的授信额度，服务商可以将授信额度通过转账的方式转到子客服务商的账户中，再由子客服务商通过转账的方式转到子客的账户中，或者服务商可以将授信额度通过转账的方式直接转到子客的账户中，子客进行消耗后，服务商的账户中才会产生授信账单，服务商需要按照“线下充值”的方式进行还款。
  - **授信线下充值：**您可以通过银行转账到[指定银行账户](#ZH-CN_TOPIC_0000001058604140__li1720515314473)的方式进行还款，银行转账之后需要您在鲸鸿动能广告平台提交还款申请，审批通过后，还款成功。
- 额度到期：如果您的授信额度到期，子客已产生消耗，您仍然需要还款；如果子客并未进行消耗，则子客账户中的通过授信获得的余额将会清零，您无需还款。
- 还款完成：还款完成后，系统将恢复服务商授信账户额度，您可以在授信账户中的“详情”查看：授信截止日期、状态、总额度、可用额度。

  ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/db/v3/qOzpf6HDSOGCxbUMlu4l7w/zh-cn_image_0000001346776417.png?HW-CC-KV=V1&HW-CC-Date=20260530T050413Z&HW-CC-Expire=86400&HW-CC-Sign=833BCA0AAD2E91742D3BA544DC1E9A4B77EBD3D40735650AA2D92E1B067ABDD9)
- 授信通知：以下几种情况，您的账户将会收到鲸鸿动能广告平台的授信通知。
  - 额度恢复完成。
  - 授信账户额度低于20%。比例默认为20%，您也可以修改比例。
  - 授信账户快到期。
  - 授信账户到期。
