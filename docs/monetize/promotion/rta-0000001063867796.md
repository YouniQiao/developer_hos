# RTA

## 广告主自行配置RTAID

## 操作指引

“投放端” -&gt;“工具” -&gt;“投放辅助”-&gt;“RTA配置管理”

![](./img/note_3.0-zh-cn_4f0d424f3a47.png) 

需要广告主账户开启RTA权限时，才有“RTA配置管理”选项。

![](./img/zh-cn_image_0000001681676008_b643c6d0017a.png)

点击RTA配置管理，广告主可以配置/编辑RTAID，RTAID格式要求与当前RTA要求一致。

![](./img/zh-cn_image_0000001729874949_3eea9969593e.png)

1. 单击“编辑”，输入RTAID。
2. 编辑完成后，请单击“确定”，如需修改，可再次单击“编辑”。
