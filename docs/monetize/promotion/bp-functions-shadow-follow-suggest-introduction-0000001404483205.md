# 功能介绍

在影子投放推广任务过程中创建子任务时，您需要选择定向跟随的目标应用。往往因为缺少充足认知或不熟悉追随条件，导致无法快速找到符合条件的追随应用。

影子投放跟随应用建议功能，借助大数据和算法计算应用与应用之间的关联下载概率、搜索热度、应用关联性等特征，提供了一个建议应用清单辅助投放，提高投放效率。
