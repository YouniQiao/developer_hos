# 推广统计

原客投主账户的数据管理界面，整合后入口：服务商管理平台——推广统计——消耗统计，可查看维度不变。

您可以通过消耗统计页面查看子账户投放总消耗、总曝光、总下载，以及分账户的消耗明细数据。
