# 代理应用管理

代理应用管理页面用于查看和管理代理关系信息。

![](./img/zh-cn_image_0000001553152725_d671d3e944ac.png "点击放大")

- 点击代理客户端ID链接，查看代理关系详情。
- 代理关系生效前，可点击“确认”来确认代理关系。若代理方式为线下授权时，则无需点击“确认”来确认代理关系。
- 代理关系生效后，可点击“终止”来终止代理关系。
