# 活动数据查询

- <strong>查询奖品流水</strong>

奖品流水数据可以在【AppGallery Connect &gt;我的应用&gt;应用类型&gt;运营&gt;活动管理&gt;奖品流水】中查看

![](./img/9e86dea4eb70.png "点击放大")

- <strong>查询活动效果</strong>

活动效果可在【AppGallery Connect &gt;常用服务&gt;分发分析】中查看

运营分析支持活动ID或名称搜索查看结果

![](./img/7a7b1eeb746f.png "点击放大")
