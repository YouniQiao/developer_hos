# 活动数据查询

- <strong>查询奖品流水</strong>

奖品流水数据可以在【AppGallery Connect &gt;我的应用&gt;应用类型&gt;运营&gt;活动管理&gt;奖品流水】中查看

![](./img/0000000000011111111.20211216102804.25151380047527023783548643911717_50521215032314_2800_A32DEE29F2D335B0EC2218426DBAECB6DCD80F5ABF181E70EBD313845575039D_9e86dea4eb70.png "点击放大")

- <strong>查询活动效果</strong>

活动效果可在【AppGallery Connect &gt;常用服务&gt;分发分析】中查看

运营分析支持活动ID或名称搜索查看结果
