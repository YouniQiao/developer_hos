# 删除人群

【简介】合作伙伴更新人群包,

人群包内容变化时可通过该接口更新。

特殊说明：增量更新可能存在失败情况，建议根据实际情况固定周期全量同步一次。

<strong>请求地址</strong>

https://svc-drcn.ads.huawei.com/opendmp/v1/audience\_package/delete

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>长度</strong> | <strong>必填</strong> | <strong>说明</strong> |
| dmpID | String | 255 | Y | 三方DMP标识。 |
| audienceIDs | List&lt;String&gt; | - | Y | 人群包id列表。  最长100个。 |

<strong>请求示例</strong>

POST http://$\{base\_address\}/opendmp/v1/audience/delete

HTTP/1.1

Headers:

Content-Type: application/json

Authorization：Bearer DAEAAIX7IS\*\*\*\*\*Pd81PDmWiM1KS2

Body:

```
{
    "dmpID": " 829411234539267456",
    "audienceIDs": ["联调测试test001"]  //人群特殊标志符
}
```
