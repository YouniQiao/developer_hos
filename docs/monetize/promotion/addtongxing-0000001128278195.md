# 特性通行名单

鲸鸿动能广告平台部分特性处于测试阶段，只对有意愿试用且通过审核的用户开放。如果您希望试用这些功能，可以通过[在线提单](https://developer.huawei.com/consumer/cn/support/feedback/#/)进行申请，申请内容需要包含：账户ID、特性名称和预算。申请结果以申请单中答复为准。当前可以申请试用的功能清单如下：

| 特性名称 | 特性描述 | 特性说明 |
| --- | --- | --- |
| - 推广非华为应用市场上架的应用 | 此功能允许您创建任务推广在非华为应用市场上架的应用。 | [非华为应用市场应用](/docs/monetize/promotion/displayad-0000001052424318#ZH-CN_TOPIC_0000001052424318__zh-cn_topic_0000001176471121_li115004572714) |
| - 转化追踪 | 此功能允许您中国大陆的广告账户使用转化追踪功能。 | [转化跟踪](/docs/monetize/promotion/tracking-gaishu-0000001139892539) |
| - 转化价值回传 | 此功能允许您转化价值回传的功能。 | [转化价值](/docs/monetize/promotion/tracking-own-analytics-0000001138842000#ZH-CN_TOPIC_0000001138842000__li768310531217) |
| - 多平台智能推广 | 如果您的应用在多个应用市场上架，此功能允许您使用“多平台智能推广”。 | [多平台智能推广](/docs/monetize/promotion/displayad-0000001052424318#ZH-CN_TOPIC_0000001052424318__li815516371017) |
| - 通用投放应用管理 | 此功能允许您在应用管理中添加所有平台的应用。 | [通用投放应用管理](/docs/monetize/promotion/appmanagement-0000001182393586#section1638488102219) |
| - 线上充值 | 此功能允许您充值时使用线上充值方式。 | [线上充值](/docs/monetize/promotion/online-top-up-0000001148173308) |
| - 秘钥共享 | 此功能允许您在多个广告账户中使用同一个应用关联的秘钥。 | [秘钥共享](/docs/monetize/promotion/analytics-association-0000001227791699) |
| - CPI任务效果助手 | 此功能允许您在投放CPI任务时，使用效果助手功能。 | [CPI任务效果助手](/docs/monetize/promotion/displayad-0000001052424318#ZH-CN_TOPIC_0000001052424318__li1526844152718) |
| - 试玩落地页 | 此功能允许您投放试玩落地页。 | [试玩落地页](/docs/monetize/promotion/displayad-0000001052424318#ZH-CN_TOPIC_0000001052424318__li116410333319) |
| - 应用市场广告支持转化跟踪 | 此功能允许您对应用市场广告进行广告数据跟踪。 | [应用市场广告支持转化跟踪](/docs/monetize/promotion/tracking-gaishu-0000001139892539) |
| - 商品中心、投放展示商品广告 | 此功能允许您使用商品中心的功能，用于管理商品广告的商品库，同时允许您投放展示商品广告。 | [商品中心](/docs/monetize/promotion/item-center-0000001251845484)  [投放展示商品广告](/docs/monetize/promotion/sp-ad-0000001601823656#section11600154892311) |
| - 应用内行为数据 | 此功能允许您将应用内行为数据创建的受众人群，用于广告任务中。 | [应用内行为数据](/docs/monetize/promotion/customized-audience-0000001182393560#section110171114447) |
| - 应用留存 | 此功能允许您使用应用留存中的所有转化指标。 | [应用留存](/docs/monetize/promotion/tracking-shu-0000001139892541#ZH-CN_TOPIC_0000001139892541__li14948201152720) |
| - 自动版位 | 此功能允许您只需上传相应图片、视频等元素，系统自动为您选择效果较好的位置进行展示广告，无需手动选择版位。 | [自动版位](/docs/monetize/promotion/overview-cjjjgg-0000001182873508#ZH-CN_TOPIC_0000001182873508__li425902485920) |
