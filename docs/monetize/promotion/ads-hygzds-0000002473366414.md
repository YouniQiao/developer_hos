# 电商

## 一、不得通过虚假描述、夸大承诺、误导性表述欺骗或诱导消费者

1.不得以不存在的商品或者服务欺骗、误导消费者。

2.不得对商品的性能、功能、产地、用途、质量、成分、价格、生产者、有效期限、允诺等，或者对服务的内容、提供者、形式、质量、价格、允诺等做虚假宣传。 如：“央视上榜品牌”——【央视关于从未颁发“央视上榜品牌”等称号的声明】。

3.不得使用虚构、伪造或者无法验证的科研成果、统计资料、调查结果、文摘、引用语等信息作证明材料。

|  |  |  |
| ×问题解析：虚假宣传“央视上榜品牌” | ×问题解析：效果虚假“肌肤嫩回20岁” | ×问题解析：不得使用“具体时间/天数”+效果描述，如：提供真实性资质证明，可支持 |

## 二、禁止假冒伪劣、不正当竞争与违规促销

<strong>1.</strong> <strong>商品售卖合规要求：</strong>

a.暂不支持山寨产品投放（含知名产品仿制品、高价值产品超低价销售）。

- 支持低价类促销活动，如“限时限量”“低价秒杀”等，但不得涉及山寨产品投放。
- 禁止山寨产品投放，或利用其它知名品牌热度混淆产品厂家关系，诱导用户购买。如：“茅台酒30元一瓶”（实际产品名为茅台镇酱香酒）。

b.不得销售假冒伪劣商品，售价需符合市场认知，禁止过分低于市场价的售卖行为。

- 不支持投放无品牌、杂牌、低端品牌等价格远低于正常售价的低价商品。
- 不支持涉及模仿知名品牌标志性产品包装、logo、品牌名称、产地等，如：茅台镇酒。

| --- | --- | --- |

| ×问题解析：山寨知名矿泉水 | ×问题解析：商品低于市场价 | ×问题解析：无品牌产品 |

<strong>2</strong> <strong>.公平竞争要求</strong>

广告文案不得使用或变相使用对比、贬低、抨击其他同类经营者的描述。

<strong>3</strong> <strong>.有奖销售要求</strong>

a.抽奖式有奖销售最高奖金额不得超过50000元。

b.以物品使用权为奖品的，按物品自身价格评估金额（非使用权价格，如汽车 1 个月使用权按汽车原价算）。

<strong>4</strong> <strong>.促销条款规范</strong>

活动内容不得出现 “本活动由 XXX 享有最终解释权” 的格式条款。

## 三、关于推广内容一致性的要求

1.广告文案、素材、落地页三者推广内容需保持一致，不得出现广告文案与素材内容不一致、外层素材推广内容在落地页无体现的情况。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> | <strong>通过示例</strong> |
| --- | --- | --- |
|  |  |  |
| ×问题解析：文案描述与商品不一致 | ×问题解析：素材价格与落地页价格不一致 | √：外层商品、满减活动在落地页体现且保持一致 |

2.广告使用单品素材投放广告，如外层素材商品在落地页无直接体现，且落地页为产品聚合页，在无法验证外层产品价格与实际售卖价格是否一致的情况下，外层素材请勿出现特定价格描述，请使用：xx元起、低至XX元或不出现具体价格。

## 四、不得使用挑衅、命令语气、祈求式、必须类文案

1.挑衅式，如：“居然买不起”“不买不要点”“不买不是男人”“敢不敢”“不看后悔”“不穿鞋不要点”等。

2.命令祈使式，如：“您该换表了”“是男人必须买”“非买不可”“一定要”等。

3.祈求式如：“帮帮我”“求求您”“拜托了”。

4.必须类，如：“必备”“必买”“必入”“必囤”等。

## 五、不得使用夸大、虚假、无法证实的对价格及功能的不合理描述

1.涉及折扣类，如：“0利润” “击破底价”“击穿底价”。

2.涉及价格类，如：“市场最低价” 。

3.涉及具体销量排名、比较和数量类，如：“年度销量第一腕表（畅销腕表可通过）”“成交量遥遥领先”。

4.明星同款（出现具体人物，需提供具体明星授权书，可通过）；抖音同款（提供具体抖音授权书，可通过），如：“达康书记同款”。

5.涉及明显功能夸大，如：“比光脚还透气”“比冰箱还清凉”。

6.其他夸大场景：“一家人穿都够了”、“借钱也要买”、“偷偷买了三件套老婆震惊”、“穿上您能上天”、 “老板涨工资原来是抢到这三件碉堡”。

7.不得有任何虚构原价、虚假折扣等价格欺诈以及虚假营销的行为；禁止在广告中出现 “原价”“内部价”“员工价” 等描述，可使用为 “吊牌价”“指导价。

## 六、广告信息表述误导/不清晰

1.活动投放提示语审核规范

广告涉及活动推广，需满足以下规范：未体现参与活动的具体流程、步骤、条件等规则，需在外层素材或落地页首屏注明活动参与提示语，如：该活动存在参与条件或资格限制，请在参与活动前进入 app 内查看活动规则详情（大图字体最小尺寸 16px，小图最小尺寸 12px ）。

2.附赠有奖销售活动审核规范

广告推广内容若为附赠式销售，文案或素材中需写明具体赠品内容，如：素材不能仅出现“买一送一”的文案，误导用户，需补充赠品名称或展示赠品图，如非送同款需加引号区分，如：买一送“X”，附赠奖品价值需与描述一致。

如：买一瓶面霜赠送一片面膜，不支持描述为买一送一，需要给赠品数量添加引号，如买一送“一”，或在外层素材图片添加赠品图例。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> | <strong>通过示例</strong> |
| --- | --- | --- |
|  |  |  |
| ×问题解析：未注明参与规则的“限时限量”活动 | ×问题解析：未注明参与规则的“0元下单”活动 | √支持同款买一送一展示赠品图的 |

3.限时描述审核规范

广告内容中出现的限时描述，请在外层素材或落地页详细注明活动时间如：周年庆、今天、今日、特惠日等无明确指向的日期，均需标明具体时间。

a.不得使用未明确指向时间的倒计时描述，如：“活动倒计时 XX 天”。

b.涉及降价描述类，如：今日降价（“今日”无明确日期指向，无法证实）、 “只卖一天”“仅限当日”“明日涨价” 等（其中“只卖一天”“仅限当日”若有明确的活动时间，可以使用）。

| --- | --- | --- |
|  |  |  |
| ×问题解析：未明确时间“限时”描述 | ×问题解析：未明确时间“倒计时”设计 | ×问题解析：未明确时间“仅剩一天”描述 |

4.不得使用容易引起歧义的词语

1.偏色情类，如：“搞大了”等。

2.价格歧义类，如：

a.不得使用易引起歧义的价格描述（如：“？元，可能让用户以为是几块钱，但实际售价并非几块钱，可能让人误解只需1块钱或2块钱”）

b.不得使用“下单不付款”（易误导免费购买）、“归您了”（未购买时误导商品归属）、“寻物启事”（易误导真实需求）等模糊表述。

## 直营电商审核规范补充

<strong>一、落地页要求</strong>

1.在广告投放环节，仅支持使用第三方电商平台（如京东、淘宝、天猫等）的商品购买链接，其他形式的链接暂不支持使用。

2.不得出现虚假的客户下单跑马灯、购买人数、商品热销量、余量等的数据展示（系统统计除外）。

3.不得出现转账付款信息，如支付宝、银行账户、微信付款二维码等。
