# 游戏

## 一、政治敏感

1.广告图片不得使用任何军用logo、标志、特殊符号等可代表各国或各国军队的标识。

2.广告图片不允许使用不完整中国地图、国旗元素等素材。

3.现代战争类游戏不允许使用战争实景图的素材。

4.不得出现弘扬封建制度，如“回到明朝做皇帝”。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> | <strong>驳回示例</strong> |
| --- | --- | --- |
|  |  |  |
| 问题解析：不完整地图与国旗元素 | 问题解析：战争实景图 | 问题解析：弘扬封建制度 |

5.不得出现政治敏感词语，如“治理国家”“造反”等。

6.文案内容不得带有政治不良诱导：如：“如果您是皇帝您会如何治理国家”等政治敏感文案或模拟现今社会中容易引起社会反响的内容。

7.广告图片素材禁止出现妨碍社会安定的元素，如：人民揭竿起义、重写历史篇章。

8.广告图片素材不得出现明示或暗示社会敏感内容的元素，如 ：暗示拆迁的“官民冲突”。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> | <strong>驳回示例</strong> |
| --- | --- | --- |
|  |  |  |
| 问题解析：不良政治导向 | 问题解析：妨碍社会安定“重写历史篇章” | 问题解析：官民冲突+下跪场景 |

## 二、暴力恐怖血腥

1. 广告素材禁止出现腐烂、尸体、僵尸类等形象。
2. 广告素材禁止出现多头、多手、多足的巨兽类形象。
3. 广告素材禁止出现巨兽打斗场景、相互吞噬、撕咬等血腥场景，如：肢体撕裂、浴血、咬食等。
4. 请勿出现伤害身体的过程、溅血场景、爆头场景等，影响用户体验。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> | <strong>驳回示例</strong> |
| --- | --- | --- |
|  |  |  |
| 问题解析：恐怖僵尸形象 | 问题解析：多头多足的巨兽形象 | 问题解析：大面积红色的血腥画面 |

## 三、赌博相关

1.游戏行业广告不允许出现赚钱、以金币或积分等方式变相兑换现金的描述，例如：提现、打金、现金到账、装备回收换钱、游戏积分金币兑换或赢取现金等。

2.禁止涉嫌赌博场景/内容，例如：“一夜暴富、炸金花、老虎机、百家乐、骰宝、21点、牌九、梭哈、炸金花、赢三张、牛牛、德州”等。

3.禁止使用涉嫌赌博场景/元素，例如：筹码、线下真实的扑克牌等。

## 四、低俗/性暗示

1.文案不支持使用歧义或夸大等涉及不良诱导的内容如：终于找到一款大人玩的塔防了、这游戏18岁以下慎入、花前月下来双修太有意思了（注：若描述场景不暧昧，如果单纯描述游戏玩法，可使用“双修”描述)。

2.不支持以玩家性别占比为噱头作宣传；如：这游戏女性玩家占90% （注：换装类游戏可通过）。

3.不得使用色情、挑逗、突出敏感部位、过于暴露的人物的形象。

4.不得使用导向不良的暧昧场景+亲密动作的素材。

5.广告不得使用洞房图及带有“生子”导向的图和文案描述，可以使用“结婚”图片及描述。

## 五、不良行为示范

1.文案不得使用涉及性别、人格、职业侮辱等挑衅命令的描述，如：不玩不是真男人，通关我叫您爸爸。

2.文案不得使用诋毁、谩骂、脏话粗话、骚扰攻击他人等不文明用语、低俗流行语，如：撕逼、变态、蛋疼、草泥马等。

3.广告图片素材不得出现有悖社会主义良好风尚的场景，如：“惩罚女囚”、“侍寝”、 “朝拜”、 “下跪”。

## 六、不良价值观

1.文案请勿出现沉迷游戏类描述，如：通宵打关、一玩就停不下来、这游戏可以玩一整天、玩到忘记时间。

2.游戏广告，支持使用游戏中玩家常使用的游戏密语，不支持带有贬低人群、沉迷游戏的词汇描述，如：可使用：氪金、GG、开黑、落地成盒、吃鸡、猥琐发育等；不支持使用：非洲人、非酋、肝帝、肝爆、屌丝等。

3.不得描述用户为了玩游戏不顾一切等不符合常理的内容，如：玩这游戏，才是真正的生活。

4.不得误导，诱发未成年人模仿违反社会道德和犯罪行为的内容；如：杀人换装备，一键当打手，做山贼，体验一把烧杀抢掠的快感。

5.不得出现违反社会道德风尚，如“三妻四妾”、“后宫选妃”等。

6.禁止种族、宗教、文化、性别歧视，不得带有对特殊群体的歧视、惩罚内容，如：对囚犯有特殊对待、惩罚女奴等描述。

## 七、承诺描述

文案不得出现承诺化、不符合游戏盈利模式的内容，如：无付费窗口、充值入口被我焊死了、装备爆率99%等（注：可允许使用的爆率描述，如：99倍掉落、爆率提升9.9）。

## 八、禁投词汇/形象/场景

1.广告内容不能出现“摸金”“倒斗”“盗墓”等违法词汇。

2.广告文案不得出现以下关键词/敏感词：如：BUG、私服、SF、外挂、QQ尾巴、木马病毒、魔兽世界插件、魔兽世界大脚等键词等；包含游戏类私服、外挂软件、私服服务器租赁的网站提交的游戏类关键词等。

3.广告素材不得出现囚犯形象。

4.广告素材不得涉及烟草等相关内容（含动漫、游戏抽烟形象）。

## 九、私域流量引流

广告文案不得涉嫌暗示或明示引导玩家私下交易的描述如：加微信、加QQ、进群有福利等。

## 十、诱导点击

素材不得涉及诱导点击的元素误导用户、诱骗用户点击广告；包括但不限于关闭、好的、知道了等与用户预期不一致的按钮（注：可使用游戏截图中的与用户预期一致的按钮元素，如游戏截图中本身的领取按钮、抽奖按钮可以使用）。
