# 成功案例

## 客户需求

某应用当前投放任务对目标人群的定位精度不够高，导致实现应用的激活投放成本较高，且下载激活数也不够理想。

## 解决方案

某应用使用oCPD任务投放后，相比CPD任务投放，应用激活率提升了200%+，激活成本降低，成功实现了降本提效。同时运营不需要频繁调整任务，节省了运营成本。
