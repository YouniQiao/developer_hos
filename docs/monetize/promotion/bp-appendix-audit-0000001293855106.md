# 应用推广审核原则

应用推广审核原则具体如下：

- 应用需符合华为应用市场的应用上架标准，可随时接受应用审核人员复审。
- 应用不能在未通知并获得用户同意之前采集、传送或使用用户隐私数据。
- 开发、利用程序暗中收集用户密码或用户私人数据将会被驳回。
- 程序包在安装后，必须自动创建启动的图标或明确提示用户运行应用的启动文件路径。
- 安装后运行的界面分辨率必须能与主流华为终端屏幕的分辨率（1920 x 1080、1280×720、960×540、854×480 等）相适配，必须体验良好，并请确保当用户对终端进行横竖屏切换操作时不会出现显示或输入等功能异常。
- 全屏模式运行的应用应充分使用屏幕的全部空间，整个界面位置应居中显示。
- 应用界面及其中的字体及图片展示清晰，不允许出现无法辨识和操作的情况；应用的操作按钮、操作界面等规格尺寸正常，大小匹配根据比例显示。
- 要求应用界面简单、精致、富有创造性以及经过精心设计。不允许用户界面太过复杂或者过于简单。
- 应用需适配海思芯片，有一定的口碑和用户基础，符合华为终端用户、华为应用市场用户的喜好风格。
