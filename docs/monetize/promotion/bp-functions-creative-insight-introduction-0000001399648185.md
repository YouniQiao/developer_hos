# 业务介绍

为了帮助开发者有效提升推广的整体创意质量，华为应用市场应用推广平台推出创意洞察功能，在生产创意前给出有效的辅助信息，帮助开发者理解哪些创意更受用户喜好，哪些元素、文案、创意布局值得参考。

![](./img/db27b31aecb2.png) 

此功能仅针对直客、代客开放。
