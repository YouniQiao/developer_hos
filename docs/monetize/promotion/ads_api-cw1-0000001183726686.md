# 查询经理账户下属广告主现金消耗和返利金消耗

【简介】通过此接口可以查询经理账户下属广告主的现金消耗和返利金消耗数据。

![](./img/note_3.0-zh-cn_c08d1402d026.png) 

使用时注意：需要经理账户授权使用。

<strong>请求地址</strong>

https://ads.cloud.huawei.com/openapi/v2/manager/report/query

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| manager\_id | string | 否 | 经理账户ID,与advertiser\_id\_list必须有一个是必传。 |
| position\_business\_types | string[] | 否 | 版位商务类型  说明：直客才有合约消耗数据  详见[【版位商务类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#版位商务类型) |
| advertiser\_id\_list | long[] | 否 | 广告主ID,与manager\_id必须有一个是必传，AG用户仅允许传1个。 |
| sign\_party | string[] | 否 | 签约主体  详见[【签约主体】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#签约主体) |
| account\_media\_type | int | 否 | 虚拟账户类型  不传默认查所有虚拟账户类型数据  详见[【虚拟账户类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#虚拟账户类型) |
| start\_time | string | 是 | 开始时间  格式如下:  "yyyy-MM-dd"； |
| end\_time | string | 是 | 结束时间  格式如下:  "yyyy-MM-dd"； |
| user\_service\_type | string | 否 | 账户推广业务类型，不填默认查询ADS侧数据，详见[【账户推广业务类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#账户推广业务类型) |

<strong>请求示例</strong>

POST openapi/v2/manager/report/query

HTTP/1.1

Accept:application/json

Content-Type:application/json

Authorization:Bearer CgB6e3x9ERGComr9dENxZX22iBk+mLuf1yGtQVPUjPJUMrstfKlqpdXk+kfHU9J8ZJ/soYIZHZzT446GeSYumluQuhsK7jvz4kz1Bkms4CLI/rE=

```
{
"start_time": "2021-10-08",
"end_time": "2021-10-08",
"advertiser_id_list":[741936499657844608],
"manager_id": "715928367165122432"
}
```

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1 | 返回数据 |

data(Struct1)定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| consume\_info\_list | Struct2[] | 列表 |

consume\_info\_list(Struct2)定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| advertiser\_id | string | 广告主ID |
| user\_expense\_sum | bigdecimal | 总消耗 |
| financial\_task\_record | Struct3 | 商务类型消耗详情 |

financial\_task\_record (Struct3)定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| business\_type | string | 商务类型  详见[【商务类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#商务类型) |
| task\_expense\_sum | bigdecimal | 总消耗 |
| financial\_account\_records | Struct4[] | 账本消耗详情 |

financial\_ account\_record (Struct4)定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| account\_type | string | 账户类型，值为：  cash：现金  gift：赠送  rebate：返利  flare\_star\_ticket：耀星  credit:信用卡  credit\_limit：信用额度 |
| expense\_sum | bigdecimal | 总消耗 |
