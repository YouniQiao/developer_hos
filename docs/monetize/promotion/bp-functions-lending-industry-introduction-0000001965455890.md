# 功能介绍

## 功能简介

贷款行业解决方案，在推荐、搜索页面设置专属于贷款类应用流量场景，还可根据模型预估对用户进行分维度金融价值评分，支持开发者按不同维度的价值等级对人群进行分层和出价，助开发者实现高效、精准获客。

## 功能特点

### 卡片式专区

特有流量场景“贷款精选卡片”外显样式吸引眼球， “可借额度”与“年化利率”等直观信息为用户对比提供便利，助力提升下载转化率。

### 支持“排除负向人群”投放

排除负向人群可为开发者过滤掉有多头借贷、团队欺诈、反催收等行为特征的用户群，支持开发者上传负向样本人群，在投放时进行排除，系统会同时根据您选择的负向样本人群进行算法模型分析，自动找到与样本人群具有相似特征的负向人群进行一并排除，帮助降低获客成本与获客风险。

### 支持为不同子任务设置不同的推广创意

- 支持区分不同子任务设置不同的转化目标、目标人群定向以及对应的投放创意，以不同创意素材匹配不同的定向人群，提升推广创意的转化率。
- 贷款行业解决方案的投放创意区别于普通推广投放，支持自定义的外显信息（预估额度、利率、营销标签等）。

### 支持对不同贷款价值用户分层出价

通过开发者回传的贷款用户行为数据（至少包含完件、授信、支用）以及APP行为数据建设贷款行业模型，对大盘用户进行分维度金融价值打分。支持开发者在回传了“授信”及“支用”行为数据后，在创建解决方案任务时按不同价值衡量维度（信用度、用款意愿、资质等级）及区分价值等级（高价值、中价值、默认）对投放人群进行分层和出价，达到优化转化效果和成本的目的。

### 独立模型

目前贷款行业解决方案的流量场景和算法模型为方案内计划独享，需要针对新的流量场景和算法模型回传数据以训练专有模型。投放账户需要先跑几天CPD任务用以机器学习（即为学习期任务），需先创建至少一个CPD任务，积累至少一个转化回传，才可建oCPD任务。
