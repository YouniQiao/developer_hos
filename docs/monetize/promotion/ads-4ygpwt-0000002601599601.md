# 2026年4月高频问题Q&A

<strong>Q1：国内的房地产经纪公司可以推广海外房产吗？</strong>

<strong>A：</strong>当前国内的房地产经纪公司推广海外房产投资及租赁属于禁投内容。

<strong>Q2：什么样的产品属于儿童邪典？支持投放吗？</strong>

<strong>A：</strong>儿童邪典是一种以儿童熟悉的卡通、动画、儿歌等形式为伪装，暗中植入暴力、色情、恐怖、自残、虐待等违背公序良俗的有害内容，专门针对低龄儿童传播的有害文化产品及相关传播现象。其本质是利用儿童的认知弱点和好奇心，传递扭曲价值观的不良信息。属于禁投内容。

<strong>Q3：服务商账户充值成功后，开票需要多久？</strong>

<strong>A：</strong>订单充值成功后，可以在开发者联盟后台点击“申请开票”或等系统于15个工作日后自动触发开票申请。如果有多条订单，当月订单会进行合并开票。开票周期为当月充值次月开出发票。

<strong>Q4：通过试投放联调时为什么资产页面有数据，试投放任务页面却没有？</strong>

<strong>A：</strong>如果试投放任务关联的设备恰好是账户内通行名单管理中的终端设备，就没有数据。

<strong>Q5：如果想要不同计划带不同的自定义参数，该怎么操作？</strong>

<strong>A：</strong>当前在计划层级还没有办法，但是可以通过计划id参数（campagin\_id）来区分；如果有任务层级的监测地址，可以不同任务配置不同的自定义参数。

<strong>Q6：开户产品为店铺，是否可以推广店铺内的多款商品？</strong>

<strong>A：</strong>同一品牌下，不同商品可以在同一账户投放，但单独推单品，需要提供对应产品资质补充在投放资质（广告资质）处。如果开户时推广内容是单品，则需要提供产品资质到行业资质处以供审核。

<strong>Q7：鸿蒙手机如何重置OAID？</strong>

<strong>A：</strong>操作方法:

①、首先需要有应用获取过OAID；

②、打开系统设置-隐私和安全-跨应用关联-里面有查看OAID；

③、跨应用关联页面，把所有应用的跟踪权限关了，再任意打开一个，就可以触发OAID的重置。

<strong>Q8：维纳斯落地页可以投放应用促活吗？</strong>

<strong>A：</strong>维纳斯落地页当前有应用下载落地页和网页落地页，暂不支持投放促活，促活一般使用应用直达链接直接引入应用活动页，需要在创建广告时选择应用促活。

<strong>Q9：如果以AI功能为核心卖点的推广产品，需提供AI相关资质，一般需要提供哪些资质？</strong>

<strong>A：</strong>互联网信息服务算法备案截图；互联网信息服务安全评估报告；如涉及具有舆论属性或社会动员能力的生成式人工智能服务，需完成生成式人工智能（大语言模型）上线备案，需提供备案截图。具体请以实际推广产品以及审核要求为准。

<strong>Q10：开户时如果选择对公银行打款认证，会有几次机会？打款账号是什么？</strong>

<strong>A：</strong>一般是有2次打款机会，每次打款后有2次输入验证机会，当前打款账户主体名称为银联商务支付股份有限公司，具体以实际打款情况为准，打款账号随机。

<strong>Q11：对公银行打款认证搜索不到开户支行怎么办？</strong>

<strong>A：</strong>首先可以通过搜索12位支行号快速定位支行，支行号非开户行号，可联系银行查询确认。如果支行号可以搜索到对应开户支行，则表示系统支持您的支行，还请进行确认并查询重试。如支行号搜不到，可反馈平台具体的银行信息和开户行号或更换其他支行信息进行尝试。

<strong>Q12：专科医院可以投放吗？</strong>

<strong>A：</strong>鲸鸿动能平台当前还不支持专科医院推广投放，关于禁投内容可查看官网帮助中心查看明细：[平台禁投内容](/docs/monetize/promotion/ads_shenhe03-0000001057185555)。

<strong>Q13：广告素材使用了推广应用内的操作截图做封面，为什么无法过审？</strong>

<strong>A：</strong>建议首先对素材封面进行检查，如游戏产品可查看素材封面是否涉及违背用户意愿的虚假按钮“关闭按钮、暂停按钮、返回按钮等”，如有涉及则无法审核通过，如无相关问题，可进行申诉。

<strong>Q14：金融贷款应用如果放贷方不是应用开发者也不是应用运营主体，可以准入吗？</strong>

<strong>A：</strong>如果应用内放贷方不是应用开发者也不是应用运营主体，需要按贷超及助贷服务行业和资质进行提交，以开户最终审核结果为准。

<strong>Q15：</strong> <strong>对于平台开户审核的资质要求，是否有资质示例可查看分辨？</strong>

<strong>A：</strong>关于开户资质证照，平台官网帮助中心有明确的示例说明可查看：[资质法律依据与示例](/docs/monetize/promotion/ads-zhizizhenzhaoshilishuoming-0000002021275500)。

<strong>Q16：</strong> <strong>如何删除被推送的人群包？</strong>

<strong>A：</strong>如果被推送的账户有任务在使用该人群包，是无法删除的，需先删除对应任务后才支持删除。

<strong>Q17：</strong> <strong>如何查看极速开户（复制账户）的主账户？</strong>

<strong>A：</strong>如果您是极速开户的复制户，要查看来源账户（主账户）可通过工具-广告账户管理-账户信息页面中的账户来源即可获知。
