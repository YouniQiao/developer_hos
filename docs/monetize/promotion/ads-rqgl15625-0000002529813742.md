# 查询人群包的基础信息

【简介】合作伙伴查询已创建的人群包的基础信息。

<strong>请求地址</strong>

https://svc-drcn.ads.huawei.com/opendmp/v1/audience/query

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>长度</strong> | <strong>必填</strong> | <strong>说明</strong> |
| dmpID | String | 255 | Y | 三方DMP标识。 |
| audienceIDs | List&lt;String&gt; | - | N | 人群包id列表。  最长100个。 |

<strong>请求示例</strong>

POST http://$\{base\_address\}\}/opendmp/v1/audience/query

HTTP/1.1

Headers:

Content-Type: application/json

Authorization：Bearer DAEAAIX7IS\*\*\*\*\*Pd81PDmWiM1KS2

Body:

```
{
    "dmpID": " 829411234539267456",
    "audienceIDs": ["联调测试test001"]  //人群特殊标志符
}
```

<strong>响应字段</strong>

<strong>AudienceInfo</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>Parameter name</strong> | <strong>Type</strong> | <strong>Length</strong> | <strong>Mandatory</strong> | <strong>Description</strong> |
| id | String | 128 | Yes | 指定的人群标识，按照人群标识做精确匹配。  Id不支持编辑 |
| name | List&lt;LanguageInfo&gt; | - | No | 人群名称。  支持多语.  第一个值为默认值。最多200个。 |
| description | List&lt;LanguageInfo&gt; | - | No | 人群描述。  支持多语，  第一个值为默认值。最多200个 |
| scale | String | 128 | No | 预估覆盖人数 |
| status | Integer | 1 | No | 人群状态：  0 计算中  1 就绪  -1 错误 |

<strong>LanguageInfo</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>Parameter name</strong> | <strong>Type</strong> | <strong>Length</strong> | <strong>Mandatory</strong> | <strong>Description</strong> |
| language | String | 3 | Yes | 语种。  传ISO639-1 Code |
| country | String | 2 | No | 语言国家，ISO 3166-1 alpha-2两位字母编码。  传中国-CN |
| value | String | 256 | No | 传人群包名称 |
