# 查询归因报表

创建任务后，您可以在归因报表中查看您的归因转化报表数据。

## 操作步骤

1. 登录[华为应用市场应用推广平台](https://ads.huawei.com/cn/)。
2. 点击“工具”页签，在“资产管理”中选择“数据资产”，进入“数据资产”页面。

   ![](./img/0000000000011111111.20260212172439.37543686988460387409652086378736:50001231000000:2800:0E45A9BF6BD91039069F445EB0E9F3C09A71BB8F0DC6A9012CE5D372352371E6_630abcdd1bf7.png "点击放大")
3. 在数据源“操作”中点击“详情”。

   ![](./img/0000000000011111111.20260212172439.35625807723967959148973509461888:50001231000000:2800:7DE6B0286C86502CA51408B0F8E36F8800B8DCCCC8DD2C4ACD7491D5313CDD68_6d7ccdc1f9ea.png "点击放大")
4. 在“归因报表”中可以显示所有已选择归因目标的任务，包括任务的曝光、下载、出价、深度转化成本、注册、激活等数据。报表支持筛选时间周期、任务类型、计费类型、转化目标。![](./img/0000000000011111111.20260212172439.43585036364877440355375453856739:50001231000000:2800:CE367BCA09D532953087E357C7562DC7765523165AF9C9871A100BBC0891C5A6_9c4ec298bf80.png "点击放大")
