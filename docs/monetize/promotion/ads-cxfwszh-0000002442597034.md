# 查询服务商账户下属账户列表

【简介】：通过此接口可以查询服务商账户下属账户列表<strong>（不含邀请中和已销户状态）</strong>

<strong>请求地址：</strong>

https://ads.cloud.huawei.com/openapi/v1/agency/advertiser/query

<strong>请求方法：</strong>

GET

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| agencyId | string | 否 | 服务商账户 说明：  1、若一级服务商登录：  i.传自身或不传，则查询一级服务商下属子客服务商列表、一级服务商直接关联的子客列表。  ii.传子客服务商，则查询子客服务商直接关联的子客列表。  2、若子客服务商登录：  i.传自身或不传，则查询子客服务商直接关联的子客列表。 |
| page | integer | 是 | 搜索页码，取值范围1~10000 |
| pageSize | integer | 是 | 一页展示数量，取值范围 10 ~ 50 |
| filtering | Struct1 | 否 | 过滤条件 |

filtering (Struct1）定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| userType | string | 否 | 需要查询的用户类型过滤条件。取值见[服务商查询用户过滤类型](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#服务商查询用户过滤类型)枚举值 |

<strong>请求示例</strong>

GET openapi/v1/agency/advertiser/query HTTP/1.1

Accept:application/json

Content-Type:application/json

Authorization:Bearer CgB6e3x9ERGComr9dENxZX22iBk+mLuf1yGtQVPUjPJUMrstfKlqpdXk+kfHU9J8ZJ/soYIZHZzT446GeSYumluQuhsK7jvz4kz1Bkms4CLI/rE=

```
{
    "page": 1,
    "pageSize": 50,
    "agencyId": "111",
    "filtering": {
        "userType": "ADVERTISER"
    }
}
```

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1 | 用户列表 |

data(Struct1)定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| total | integer | 总条数 |
| agencyList | Struct2[] | 子客服务商列表 |
| advertiserInfoList | Struct3[] | 子客列表 |

agencyList（Struct2）定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| agencyId | string | 子客服务商账户ID |
| nickName | string | 账号昵称 |
| corpName | string | 企业名称 |

advertiserInfoList（Struct3）定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| advertiserId | string | 子客账户ID |
| nickName | string | 账号昵称 |
| corpName | string | 企业名称 |
