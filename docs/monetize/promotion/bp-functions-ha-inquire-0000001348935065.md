# 查询HA归因报表

当您选择HA归因方式时，HA会自动帮您回传数据用于提升推广效果。

1. 登录[华为应用市场应用推广平台](https://developer.huawei.com/consumer/cn/service/apcs/app/home.html)，点击右上角“管理中心”，进入“管理中心”页面。
2. 点击“工具”页签，在“资产管理”中选择“数据资产”，进入“数据资产”页面。

   ![](./img/0000000000011111111.20251117124639.05548251967767081248578442208836_50001231000000_2800_CC52B1B1AAE7D7CBDFD36D83605F931563C3A07E39272737D8A80356FA1EA24F_e25b67c44e17.png)
3. 在“数据资产”页面，在需要查看的数据源后，点击“详情”，进入详情页面。

   ![](./img/0000000000011111111.20251117124639.23380822102076244182129771698741_50001231000000_2800_6A729C8288D022633F419CB11906A265D210C578EDDEE1DF77746BADD772B0FA_b52b8873d768.png "点击放大")
4. 点击“归因报表”页签，查看回传数据信息。

   ![](./img/0000000000011111111.20251117124639.10337882594167741501963238610617_50001231000000_2800_37885B680D8163D2DEE266674EBBD2B39C3793B2C736C3787FED6D3D4339EA7E_a6525f307072.png "点击放大")
