# 食品&酒水

<strong>食品行业审核规则</strong>

## 一、虚假宣传

1.夸大绝对化描述

a.不得含有“最新科学”、“最新技术”、“最先进加工工艺”等绝对化的描述。

b.不得使用“纯天然”“无添加”等描述。

| --- | --- | --- |
|  |  |  |
| ×问题解析：纯天然 | ×问题解析：无添加 | √支持使用真实性的配料成分、口感等描述 |

<strong>2</strong> <strong>.功效虚假</strong>

a.食品广告不得出现与药品相混淆的用语，不得直接或间接地宣传治疗作用，也不得借助宣传某些成分的作用明示或暗示该食品的治疗作用，如：

- 不得涉及特殊化妆品功效宣传
- 不得宣传具有疾病/症状的治疗与改善功效
- 不得涉及中医功效/治疗话术，引人误导其具有医疗作用
- 不得明示/暗示其具有减肥效果
- 不得声称具有保健功能
- 不得涉及壮阳、丰胸、增高等违规功效
- 不得与其他食品、相关药品、保健食品、化妆品等进行对比

| --- | --- | --- |
|  |  |  |
| ×问题解析：不得宣传治疗作用 | ×问题解析：不得明示减肥作用 | ×问题解析：不得声称保健功能 |

b.广告中涉及“转基因”、“非转基因”相关描述，需符合相关要求：

- 不得含有虚假的内容，不得欺骗和误导消费者。如：宣称“花生油” 、“橄榄油” 、“葵花籽油”等非转基因，利用消费者专业知识的缺乏，容易造成该类作物可能同时存在“转基因”和“非转基因”产品的错误认知。
- 对于宣传“非转基因”的产品广告，不得使用“更健康”、“更安全”等误导性词汇

## 婴幼儿食品审核规则

<strong>一、功效虚假</strong>

1.不得明示或者暗示可以替代母乳，不得使用母乳哺乳中的妇女和婴儿的形象。

2.不得明示或者暗示具有益智、增加抵抗力或者免疫力、保护肠道等功能性表述。

<strong>二、表述误导/不清晰</strong>

不得使用进口奶源、源自国外牧场、生态牧场、进口原料”等模糊信息，涉及原料来源的（如生乳、原料乳粉等原料来源），应当如实标明具体来源地或者来源，如：新西兰进口奶源、源自澳大利亚牧场等。

## 酒水广告审核规则

<strong>一、不良行为示范</strong>

1.广告中禁止出现饮酒形象，同时不得诱导、怂恿饮酒或者宣传无节制饮酒，不得出现饮酒动作不得诱导、怂恿饮酒或者宣传无节制饮酒；不得出现饮酒动作。

2.不得出现潜在危险的活动，如：驾驶车、船、飞机等。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> | <strong>驳回示例</strong> |
| --- | --- | --- |
|  |  |  |
| ×问题解析：未成年饮酒形象 | ×问题解析：不得出现饮酒形象 | ×问题解析：不得出现危险驾驶场景 |

<strong>二、夸大绝对化描述</strong>

1.不得把个人、商业、社会、体育、性生活或者其他方面的成功归因于饮酒。

例如：“壮阳补肾、提高性生活、事业有成、成功人士的选择、重振雄风。

2.不得使用医疗用语或者易与药品混淆的用语。

<strong>三、借用公信力宣传</strong>

禁止使用“专（特）供XX国家机关”、“XXX省（市）指定专用产品”、 “军队特需”等国家机关名义或类似内容。
