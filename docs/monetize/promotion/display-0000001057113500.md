# 展示广告网络

## 概述

展示广告网络是指在华为自有应用和通过华为流量变现的联盟应用的广告位宣传您的应用或网页*，*在创建展示广告计划时，“投放网络”选择”展示广告网络”，展示广告网络支持多种不同的广告样式，每个样式通常会被打包到不同的广告版位，在创建投放任务时，您可以通过选择版位投放不同的广告样式。

- 展示广告网络支持投放的广告为：
  - **展示应用广告**：您可以选择在哪些展示广告网络版位上推广您的应用，您可以控制各个广告素材的组合方式和定向条件等。
  - **展示网页广告**：展示网页广告是指在[展示广告位](/docs/monetize/promotion/display-0000001057113500)上对您的网页进行推广，用户看见图片/视频后，单击进入您想要推广的网页，增加网页的浏览量和转化量。
  - **展示商品广告**：如果您希望推广自己的零售商品，可以在展示广告网络上投放商品广告来增加流量，触达更多优质潜在用户。

- 展示广告网络包含以下版位：
  - **华为媒体版位**：在华为自有的应用上展示您的广告，例如：华为视频、华为音乐等。
  - **三方媒体版位**：在接入鲸鸿动能广告变现服务的三方应用上展示您的广告。
  - **三方ssp版位**：三方ssp版位属于“[其他首选资源](/docs/monetize/promotion/overview-cjjjgg-0000001182873508#ZH-CN_TOPIC_0000001182873508__zh-cn_topic_0000001205953939_li184556354214)“内，指的是在与华为有合作的三方变现服务的三方应用上展示您的广告。
