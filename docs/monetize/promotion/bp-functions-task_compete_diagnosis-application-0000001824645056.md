# 申请开通

任务竞争力诊断功能为允许清单开通制，需要申请的开发者请联系相关运营或填写申请模板后发送邮件至[developer@huawei.com](mailto:developer@huawei.com)，审核人员评估后会在1-2个工作日内进行答复

|  |  |
| --- | --- |
| 申请开通竞争力诊断，您可以发送邮件至developer@huawei.com或联系对应的行业运营进行申请，请按照如下表格填写邮件。 | |
| 邮件标题 | 请按照“应用名称-申请竞争力诊断功能”格式填写。 |
| 邮件内容 | 应用名称：应用在华为应用市场详情页展示的名称。 |
| 应用ID：应用的唯一标识。 |
| 申请内容：申请开通任务竞争力诊断 |
