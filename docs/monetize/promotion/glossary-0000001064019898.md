# 术语表

|  |  |
| --- | --- |
| <strong>词汇</strong> | <strong>解释</strong> |
| 计划 | 在推广计划级别，您可以根据营销目标设置将您的哪类产品以何种广告形式投放到哪个广告网络，并为您的产品设置日预算。这些设置在计划创建完成之后不支持修改。 |
| 任务 | 在任务级别，您可以设置待推广的产品、投放类型、投放版位、投放时间、出价等，还可以设置地域、运营商、语言、设备、应用行为等定向条件。不同的投放网络对应的定向条件有所不同。 |
| 创意 | 创意是广告主展现给用户的推广内容，包括文字、图片或者视频素材等内容。创意决定了是否能吸引潜在受众，从而采取行动促成转化。 |
| eCPM | effective cost per mille，指的是每一千次展示可以获得的广告收入。 |
| pCTR | Predict Click Through Rate，预估点击率。 |
| CPM | Cost Per Mille，按千次曝光计费。选择该出价方式，您的广告在展示时就会产生费用，按照每一千次展示的费用来计算。用户的单击行为不会再进行扣费。 |
| CPC | Cost Per Click，按单次点击计费。选择该出价方式，您的广告在展示时不会扣费，只有当用户点击了广告，才会扣除相应的点击费用。 |
| CPD | Cost Per Download，按单次下载计费。 |
| CPI | Cost Per Install，按单次安装计费。 |
| CPA | Cost Per Action，按广告的实际投放效果付费。效果包括付费、激活、表单提交。 |
| oCPC | Optimized Cost Per Click，按预估转化完成率与目标转化完成成本自动优化出价。 |
| CPT | Cost per Time，即按时长付费，仅支持合约模式销售。 |
| 关键词 | 您可以设置希望在用户搜索哪些关键词时对您的应用进行推广。 |
| 否定关键词 | 在用户搜索的内容匹配上您设置的任意否定关键词时，您的广告都不会被展示。 |
| 精确匹配 | 当用户搜索词与您设置的关键词完全一致时，您的广告才有展现机会。精确匹配可以匹配关键词的单复数，进行时、过去式等变体，并且顺序必须保持一致。 |
| 短语匹配 | 当用户的搜索词完全包含您的关键词，系统有可能会展现您的广告。短语匹配可以匹配关键词的单复数、进行时、过去式等，可以有间隔但是顺序必须保持一致。 |
| 广泛匹配 | 此种方式能够较大范围的将用户的搜索词与您设置的关键词进行匹配，广泛匹配可以匹配关键词的同义词、近义词、单复数、进行时、过去式等，乱序，或者包含，您的广告都会展现。 |
| OAID/GAID | OAID/GAID指的是一种非永久性设备标识符，可在保护用户个人数据隐私安全的前提下，向用户提供个性化广告。HMS手机一般用OAID，GMS手机一般用GAID。 |
