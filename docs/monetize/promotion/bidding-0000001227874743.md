# 竞价广告

竞价广告是一种由广告主在鲸鸿动能广告平台自主投放，自主管理,通过调整价格来进行排名，按照广告效果付费的网络广告形式。在各媒体上以不固定位置出现，可以进行多维度定向，精准捕捉潜在人群，灵活操作定向维护、出价、创意及多种优化工具，使您可以随时控制广告的投放，获得较大回报。

鲸鸿动能广告支持的[竞价广告](https://developer.huawei.com/consumer/cn/doc/promotion/ads-gaishu-0000001095807128)有展示广告和应用市场广告。计费方式为CPM、CPC、CPI、CPD、oCPC。
