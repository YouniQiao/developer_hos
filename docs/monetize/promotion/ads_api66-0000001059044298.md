# 广告主数据

广告主通过本接口获取广告主基本数据，包括花费、点击量、点击均价等。

![](./img/note_3.0-zh-cn_eaf44f9343f0.png) 

本接口不支持查询合约数据。

<strong>请求地址</strong>

https://ads.cloud.huawei.com/openapi/v2/reports/advertiser/query

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 是 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| time\_granularity | string | 是 | 可选(非必填)："STAT\_TIME\_GRANULARITY\_DAILY"天粒度。  详见[【时间粒度】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section147576327545) |
| start\_date | string | 是 | 起始时间 |
| end\_date | string | 是 | 结束时间 |
| page | integer | 否 | 搜索页码 默认值1，取值范围1~10000 |
| page\_size | integer | 否 | 一页展示数量，默认值20  取值范围 1~10000 |
| topn | integer | 否 | 汇总数据指标 |
| flow\_resource | integer | 否 | 资源类型 |
| campaign\_type | integer | 否 | 计划类型 |
| index\_screen\_list | UserNewDspPortalIndexScreen[] | 否 | 指标筛选 |
| dimension\_type | UserNewDspPortaldimensionType | 否 | 维度细分 |
| time\_line | string | 否 | 可选(非必填)：时间口径，默认请求时间口径  详见[【时间口径】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section101781612185919) |
| group\_by | string[] | 否 | 分组  枚举值 包括 DATE、HOUR、ADGROUP\_ID、COUNTRY、SEARCH\_KEY\_WORD、DEAL\_ID、CAMPAIGN\_ID、ADVERTISER\_ID、CREATIVE\_ID、POSITION\_ID  按天月，必传对应维度+时间，按summary，必传对应维度 |

UserNewDspPortaldimensionType:

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| dimension | string | 否 | 维度细分指标 |
| data | string[] | 否 | 筛选指标 |

UserNewDspPortalIndexScreen:

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| index\_screen | string | 否 | 筛选指标，字符串形式：effective\_cost：花费  effective\_impression\_count：展示量  effective\_click\_count：点击量  effective\_per\_click：点击均价  effective\_download\_count：下载量  effective\_click\_ratio：点击率  effective\_click\_download\_ratio：点击下载率 |
| type | integer | 否 | 筛选指标类型   1. 大于等于，只需要赋值low\_value、 2. 小于等于，只需要赋值up\_value 3. 介于，需要赋值low\_value up\_value |
| low\_value | float | 否 | 下限值 |
| up\_value | float | 否 | 上限值 |
