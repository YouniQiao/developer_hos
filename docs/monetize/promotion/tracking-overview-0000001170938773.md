# 概述

## 概述

鲸鸿动能广告支持使用业界常用的监测平台进行转化效果跟踪，当前支持的监测平台如下：

| 三方监测平台 | 监测目标 | OAID/GAID归因支持的SDK版本 | Referrer归因支持的SDK版本 | 是否支持回传 |
| --- | --- | --- | --- | --- |
| [AppsFlyer](/docs/monetize/promotion/appsflyer-0000001122291488) | 效果 | 支持5.4.0以上 | 支持6.14.0及以上版本 | 是 |
| [Adjust](/docs/monetize/promotion/adjust-0000001121931660) | 效果 | 支持4.2.2以上版本 | 支持4.28.6及以上版本 | 是 |
| [Sizmek](/docs/monetize/promotion/sizmek-0000001168691457) | 品牌 | 支持 | - | 否 |
| [Kochava](/docs/monetize/promotion/kochava-0000001168891345) | 效果 | 支持3.8.0及以上版本 | - | 是 |
| [Tenjin](/docs/monetize/promotion/tenjin-0000001197115949) | 效果 | 支持1.12.8以上版本 | - | 是 |
| [Airbridge](/docs/monetize/promotion/airbridge-0000001197036041) | 效果 | 支持 | - | 是 |
| [Branch](/docs/monetize/promotion/branch-0000001272007784) | 效果 | 支持4.4.0以上版本 | - | 否 |
| [Singular](/docs/monetize/promotion/singular-0000001360983758) | 效果 | 支持12.0.7及以上版本 | - | 否 |
| [MyTracker](/docs/monetize/promotion/mytracker-0000001439462594) | 效果 | 3.0.10及以上版本 | 3.0.10及以上版本 | 是 |
| [AppMetrica](/docs/monetize/promotion/appmetrica-0000001439303334) | 效果 | 5.2.0 及以上版本 | 5.2.0 及以上版本 | 是 |

## 曝光、点击数据监测流程

如果您想使用任何一个三方监测平台监测曝光、点击数据，您只需要按照如下步骤进行操作。如果您想监测更多的数据，例如：安装、激活、付费等数据，您可以参考步骤1中“三方监测平台”列表。

1. 在三方监测平台获取曝光和点击监测链接。

   详细操作请参照下表：

   | 三方监测平台 | 是否需要设置监测链接 | 监测链接获取指导 |

   | [Adjust](/docs/monetize/promotion/adjust-0000001121931660) | 是 | - |

   - 如果您后期修改了关联分析工具中的曝光/点击监测链接，您需要重新对任意一个指标进行[手动测试](/docs/monetize/promotion/tracking-app-overview-0000001209244840#ZH-CN_TOPIC_0000001209244840__section105501517172)，测试成功后新的曝光/点击监测链接才生效，其他的指标启用状态，与修改链接前保持一致。
   - 如果您使用的监测链接未包含Referrer参数（例如：af\_ref=\_\_REFERRER\_\_），请重新在三方监测平台拷贝监测链接并填入鲸鸿动能广告平台。
2. 在鲸鸿动能广告平台新建资产。
   1. 需要为您希望跟踪的每一个应用使用指定的监测工具新建资产，详细请参考[新建资产](/docs/monetize/promotion/tracking-app-overview-0000001209244840#ZH-CN_TOPIC_0000001209244840__li8351194812211)。
3. 在鲸鸿动能广告平台创建广告任务。

   您在上传广告创意时，系统将会自动关联到创意中的曝光/点击监测链接（自动关联的链接不要修改，避免影响跟踪数据）。

   ![](./img/note_3.0-zh-cn_680e3d7148db.png) 

   如果您后期修改了关联分析工具中的曝光/点击监测链接，投放网络为展示广告的任务，修改后的监测链接将自动同步到任务中；投放网络为应用市场的任务，您需要编辑任务，进入到创意，重新单击“提交”，此时监测链接生效。
