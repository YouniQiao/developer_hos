# 功能介绍

创意预审库让开发者能够在投放推广任务前，单独提交创意素材进行预审核。素材通过审核后，开发者创建推广任务时可以从创意预审库中选择素材，无需再审核，即可直接投放任务。多个推广任务支持使用同一份创意预审库中通过审核的素材，避免素材重复审核，帮助开发者减少等待时间、提升投放效率。
