# SearchkeyWordDetail

![](./img/4422621bdd28.png) 

该对象仅为兼容历史数据，后续该模型会删除。

| 参数 | 必选(M)/可选(O) | 类型 | 描述 |
| --- | --- | --- | --- |
| key | O | String | 关键词。 |
| keyMatchType | O | String | 匹配方式：   - ACCURATE：精确匹配 - FUZZY：泛匹配 |
| searchListPrice | O | double | 排名出价。 |
| fastSearchPrice | O | double | 直达出价。 |
| keyStatus | O | String | 关键词状态。 |
