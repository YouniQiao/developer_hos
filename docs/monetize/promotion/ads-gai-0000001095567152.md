# 概述

如果您希望更多用户下载您的Android应用，可以在鲸鸿动能广告平台使用下述方式进行推广。为了达到较佳推广效果，我们推荐您同时创建所有推广类型，并在投放过程中根据效果进行持续优化。

<strong>由于每个广告计划仅支持一种推广类型，您需要为每个推广类型创建一条独立的推广计划。</strong>

|  |  | 支持推广的应用所属平台 | | | | 广告展示位置 | |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 推广类型 | 说明 | 华为应用市场 | GooglePlay | 其他安卓应用商店 | 第三方监测渠道 | 华为自有媒体 | 三方媒体 |
| 应用市场展示广告 | 在应用市场展示广告位上对您的应用进行推广。 | ✔ | - | - | - | ✔ | - |
| 应用市场搜索广告 | 用户在华为应用市场上搜索关键词时对您的应用进行推广 | ✔ | - | - | - | ✔ | - |
| 智能应用广告 | 在接入鲸鸿动能广告变现服务的三方应用上及华为应用上展示您的应用广告。在智能推广模式下，您只需要输入关键元素，系统将对各个元素进行智能组合和优化，适配到各个不同的广告类型。 | ✔ | - | - | - | ✔ | ✔ |
