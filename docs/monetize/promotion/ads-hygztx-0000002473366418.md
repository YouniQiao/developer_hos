# 通讯

## 一、信息模糊缺失

1. 品牌来源要求

a.广告主为运营商，来源可单独使用运营商名称，如：XX移动。

b.广告主与运营商为合作关系，来源不可单独使用运营商名称，可以为【广告主核心词汇+产品名称】或单独体现【广告主核心词汇】。

| <strong>通过示例</strong> | <strong>通过示例</strong> |
| --- | --- |
|  |  |
| 解析：广告主为运营商主体，可使用自身运营商名称 | 解析：支持使用“广告主名称+推广产品”作为品牌来源 |

2.信息需规范展示

a.广告推广号卡业务，需要明确资费信息时间周期；涉及资费信息需明确时间周期，如19元/月、360G/年等，不得混淆月享套餐内容和年享套餐内容。

b.推广号卡业务，落地页需体现具体资费详情表单说明，需保证资费详情表单清晰完整，且体现以下内容：

- 月费：明确月费资费标准（若涉及资费活动需明确活动规则）。
- 流量：明确具体流量属性及资费标准，如：

  ①专属流量：明确是具体哪些app\时段\地域的定向流量，每月多少G专属/定向流量。

  ②通用流量：每月多少G通用流量。

  ③赠送流量：明确具体额度以及使用限制。
- 语音：明确资费标准。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> | <strong>通过示例</strong> |
| --- | --- | --- |
|  |  |  |
| 问题解析：仅标识135G/ 29元，未明确时间周期 | 问题解析：只提及次月资费，未明确首月月租，无法通过 | 通过解析：详细列出月租&套餐内容&套外资费信息 |

c.广告落地页运营商信息需规范展示：

- 推广通讯号卡业务，落地页必须体现运营商信息，且仅能体现一家运营商，不得描述为任意运营商都可以用。
- 落地页需补充添加广告主公司联系电话或自有公众号（不得涉及引导添加个人微信号）。
- 广告推广号卡业务，需在落地页显著位置清晰展示电子标识牌。

| <strong>通过示例</strong> | <strong>通过示例</strong> |
| --- | --- |
|  |  |
| 解析：落地页注明运营商+广告主电话 | 解析：号卡业务在落地页合规添加电子标识牌 |

## 二、不实承诺

1.内容真实性

a.广告推广号卡业务，业务内容需真实，通用流量资费需真实准确，不得以不合理低价诱导用户，例如不得低于1元/G；以下场景可通过：仅单纯涉及流量包。赠送流量除外，但需要明确具体额度及使用限制。

b.广告推广号卡业务，不得宣传通用流量无限量。

c.通用流量不得涉及“流量不限量”、“无限流量”等表述。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> |
| --- | --- |
|  |  |
| 问题解析：29元100GB，非正常资费标准 | 问题解析：不符合实际情况 |

2.噱头描述

a.不得使用不符合实际的时间描述，如在时间词包前后增加限定词，误导用户只能在今天或一定时间内才能享受相关权益。如：仅限/截至\\{时间\\}、仅限今日/今日特惠/今日截止/最后一天/明日涨价、 \\{日期\\}起 、 \\{日期\\}仅剩X个。

b.不得使用不符合实际的空间描述，如在地点词包前后增加限定词，误导用户只可在限定区域获得相关权益。如：仅限\\{地点\\}、 \\{地点\\}附近x米。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> |
| --- | --- |
|  |  |
| 问题解析：用虚假倒计时制造紧迫感，误导用户 | 问题解析：非定向类产品，不得虚构使用地点为卖点 |

## 三、诱导点击

1.模仿性诱导

a.不得涉及虚假按钮类素材，按钮涉及“关闭、跳过、返回、忽略、解锁” 等。

b.不得涉及虚假通话/视频界面、直播界面。

c.不得涉及操作后无法实现其功能的设计，如：滑动、指纹、解锁、光标等样式设计。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> |
| --- | --- |
|  |  |
| 问题解析：虚假视讯界面 | 问题解析：图片样式误导用户输入手机号 |

## 四、隐私场景

1.手机号隐私

a.手机号卡业务外层素材 (标题、图片、视频) 不得出现完整的手机号码，需至少隐藏4位数。

b.推广号卡业务，落地页可体现完整手机号码。

2.个人隐私协议

a.落地页为第三方网站，需体现必要文件，例如《入网许可协议》《个人信息收集证明》、隐私声明等，经用户同意后，方可收集用户信息，如个人姓名、身份证信息、电话、地址等。

b.《入网许可协议》和《个人信息收集证明》 、隐私声明等必要文件需可勾选、可点击，其中内容需详细可查。注：该类文件不得为自动勾选

| <strong>驳回示例</strong> | <strong>通过示例</strong> |
| --- | --- |
|  |  |
| 问题解析：缺少协议书 | 解析：协议齐全、可点击可勾选 |
