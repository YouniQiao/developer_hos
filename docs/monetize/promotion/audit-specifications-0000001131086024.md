# 审核规范

<strong>【活动应用】：</strong>华为应用市场在架应用。

<strong>【活动时间】：</strong>1天&lt;=活动时间&lt;=7天，活动开始时间统一为10:00点。

老用户回流（指定用户发奖）活动时间：开始时间即为奖品发放时间

<strong>【活动奖品数量】：</strong>奖品数量分为“初次创建”（即奖品库存）和“初次添加至活动”两种；

（1）“初次创建”奖品数量&gt;=当前华为渠道日均非更新\*活动时间\*1.5倍效果，其中当奖品类型为“华为优惠券”时，需配置为“不限量”（结算以实际使用为准）；

（2）“初次添加至活动”奖品数量，视情况而定，系统支持在线追加奖品库存中剩余奖品数量。（温馨提示：若活动过程中礼包数量不足将引起用户投诉。）

<strong>【活动目的】：</strong>

（1）“提升安装量（安装有奖）”：无限制；

（2）“提升新增注册（首登有奖）”：应用内登录方式须含华为帐号；

（3）“提升活跃（连续登录有奖）”：应用内登录方式须含华为帐号；

（4）“提升收入（充值有奖）”：应用内支付方式须含华为支付；

（5）“老用户回流（指定用户发奖）”：无限制。

<strong>【活动素材】：</strong>建议参考系统示例。

<strong>【活动创建时间】：</strong>建议至少提前5个工作日, 申请时间需提前两个工作日。

<strong>【活动与奖品描述】：</strong>建议参考页面提示，同时避免出现错别字、歧义句等。
