# 离线资产管理

## 功能简介

离线资产管理为有一方数据回传分析的客户，提供统一上传接口，并且一次上传即可实现多种业务场景。离线资产涵盖营销度量（转化跟踪）、联合建模、人群拓展，落地页事件上报等分析场景，可从根本上提升数据上传效率，更好地利用后向链路数据优化广告投放。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ef/v3/sYqBKFVZQcqOFVCKVN2Zfw/zh-cn_image_0000001639974248.png?HW-CC-KV=V1&HW-CC-Date=20260530T050720Z&HW-CC-Expire=86400&HW-CC-Sign=9C1B80871562DD76BE1782788A2FD5B59096BD2938FAF0B02EBBAF4E2EC1DE48)

1. 营销度量（转化跟踪）：通过鲸鸿动能广告平台，利用科学的转化归因方法，将离线上传的事件归因至具体广告任务。
2. 联合建模：使用数据训练专属您的私有模型，帮您优化广告效果，提升广告跑量效果。
3. 人群拓展：使用数据作为种子人群进行人群拓展，生成可用于投放的人群包，需要在鲸鸿功能DMP中进行拓展操作。
4. 落地页事件上报：将不同状态的维纳斯落地页的线索ID离线上传。
