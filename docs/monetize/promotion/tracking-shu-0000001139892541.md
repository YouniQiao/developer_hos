# 转化数据

## 概述

完成应用跟踪或线索跟踪后，回传的转化数据可以在鲸鸿动能广告平台查看，如果您在广告平台没有看到相应的转化数据，您需要检查应用跟踪或者线索跟踪回传配置是否正确。

## 查看数据

进入“推广”或“报表”菜单下的相关页面，选择“自定义列”，选择您想要的数据进行查看。

- <strong>推广</strong>：您可以从“计划”、“任务”、“创意”、“元素”、“关键词”页面，分别查看“自定义列”的数据。
- <strong>报表</strong>：您可以从“账户”、“计划”、“任务”、“创意”、“国家/地区报表”页面，分别查看“自定义列”的数据。

- “华为分析预置事件参考”中的$AppFirstOpen为自动采集事件，不需要埋点，其他均需要进行埋点并在“华为分析 ”-&gt; “管理” -&gt; ”事件管理”中注册，详情可参考[预置事件使用指导](https://developer.huawei.com/consumer/cn/doc/development/HMSCore-Guides/android-usage-guide-0000001135571462)。
- “应用打开”指标中：
  - 如果您在新建资产时，勾选了<strong>智能跟踪</strong>且回传了“应用打开”指标，则系统自动创建“留存（计算）”指标。
  - 如果您<strong>未勾选智能跟踪</strong>，则不自动创建“留存（计算）”指标。
  - 如果您已经创建“留存（回传）”指标，则无法自动创建“留存（计算）”指标。
- 如果您需要使用“应用留存”指标的功能，需要申请[特性通行名单](https://developer.huawei.com/consumer/cn/doc/promotion/addtongxing-0000001128278195#ZH-CN_TOPIC_0000001128278195__li1597554939)。
  - 如果您没有创建“留存”转化指标（包括不限于HMS、回传），从华为分析回传应用打开事件，系统会自动创建“留存（计算）”。
  - 如果您有创建“留存”转化指标，从华为分析回传应用打开事件，系统不会自动创建“留存（计算）”。
- “次日留存（计算）”指标中，同一个用户，0-24小时不算次留，第24-48小时打开算次留，第72-96小时算3日留，第168-192小时算7日留。
