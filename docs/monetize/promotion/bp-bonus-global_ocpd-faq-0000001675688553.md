# FAQ

## 具体怎么定义保障周期？

例如任务8月1日新建，8月2日审核通过，8月3日产生首次曝光数据，则保障时间为8月3日及其之后的10个自然日（8月3日至8月12日）。

## 一天内将任务先暂停后启动同时修改出价 1 次，能够满足保障政策吗？

不可以。暂停和启动各算一次，任务共有3次操作，已超过1天2次的修改上限。

![](./img/849fb6c2d90b.png) 

修改预算不占用次数限制。
