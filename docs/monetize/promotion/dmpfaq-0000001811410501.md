# FAQ

## FAQ

<strong>Q1：定向包中的每一个用户都能收到我投放的广告吗？</strong>

<strong>A：</strong>不一定，定向包起的是一个过滤作用，限定投放的人群范围，这部分人群依然会参与竞价，竞价成功的才会投放。

<strong>Q2：上传自有人群会泄露我的用户隐私吗？</strong>

<strong>A：</strong>不会，鲸鸿动能DMP平台坚决保护用户个人信息及广告主数据安全，对于广告主上传的自有人群，平台只根据用户需求匹配与计算，不查看人群具体内容。

<strong>Q3：上传自有人群失败是什么原因？</strong>

<strong>A：</strong>请检查上传的格式是否符合鲸鸿动能DMP平台的格式要求，如非格式问题但依然上传失败，请联系运营人员协助排查。

<strong>Q4:</strong> <strong>人群包是否支持自动更新？</strong>

<strong>A：</strong>细分受众人群（标签人群）、重定向人群、拓展人群、组合人群支持定期自动更新，号码文件人群不支持更新。
