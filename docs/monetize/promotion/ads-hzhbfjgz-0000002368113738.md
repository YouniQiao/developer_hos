# 合作伙伴分级权益规则（试行）

为进一步提升客户服务体验、助力合作伙伴高效开展业务，特试行推出<strong>【鲸鸿动能合作伙伴分级权益】</strong>

<strong>评定周期与升级规则：</strong>

1.评定周期：<strong>每个季度评定一次</strong>，每个季度第一周根据上个季度的日均消耗进行定级。

2.若您的产品上一季度在鲸鸿动能的投放消耗环比增长1倍，本季度可获得一次评级基础上增加一个等级的越级服务。（自然年度限1次）

3.<strong>按照推广内容进行定级</strong>，消耗只计算通过鲸鸿动能平台下单的消耗。该规则自发布之日起生效，试行周期到2025.12.31。

4.权益申请：您可联系您所在行业对应的鲸鸿动能商务联系人发起权益申请。

5.定级查询：您可通过您账户中的站内信查看您的产品定级情况或联系鲸鸿动能商务联系人确认。
