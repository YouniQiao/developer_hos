# 查询版位

【简介】通过此接口可以查询DSP版位信息。

<strong>请求地址</strong>

https://ads.cloud.huawei.com/ads/v1/tools/position/query

<strong>请求方法</strong>

<strong>GET</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| filtering | Struct1 | 否 | 过滤条件 |

filtering (Struct1)定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| product\_type | string | 否 | 推广产品  详见[【推广产品】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section1526319223472) |
| price\_type | string | 否 | 付费方式  详见[【付费方式】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section1898795494716) |
| creative\_size\_id\_list | string[] | 否 | 查询指定版位ID列表 |
| objective\_type | string | 否 | 推广目的  详见[【推广目的】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section122581352419) |
| category | string | 否 | 版位所属分类  详见[【版位所属分类】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section1294412322443)  注：全域智投版位不支持此字段 |
| flow\_resources | string[] | 否 | 详见[【投放网络】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section11983101283716) |
| campaign\_type | string | 否 | 计划类型  取值见[【计划类型】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section65101236574)枚举值 |
| vip | string | 否 | vip 版位 0不是 1是  详见[【VIP版位】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section1550115264219)  注：全域智投版位不支持此字段 |
| is\_support\_keyword | string | 否 | 是否支持关键词设置  详见[【是否支持关键字类型】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section2610111444318) |
| smart\_delivery | string | 否 | 是否查询全域智投版位  详见[【全域智投版位】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section1468814571620), 默认查询非全域智投版位 |

creative\_size\_placement\_list(Struct6)定义

|  |  |  |
| --- | --- | --- |

| is\_support\_multiple\_creatives | string | 是否支持多创意  详见[【是否支持多创意】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section11124133654815) |

creative\_size\_support\_scale(Struct8)定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| creative\_size\_scale | string | 当前规格下所有图片和视频素材的尺寸，多值用X标识，如3图为X3，不同尺寸的图片或视频以逗号分隔 |
| spec\_ids | string[] | 此规格尺寸下所有的规格ID |

creative\_size\_operation\_info(Struct4)定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| support\_product\_type | string | 支持的推广产品，多值以逗号分隔  详见[【推广产品】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section1526319223472) |
| support\_objective\_type | string | 支持的推广目的，多值以逗号分隔。推广产品为应用时有效  详见[【推广目的】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section122581352419) |
| is\_support\_time\_period | string | 是否支持选择投放时段  详见[【是否选择支持投放时段】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section112801154813) |
| is\_support\_multiple\_creatives | string | 是否支持多创意  详见[【是否支持多创意】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section11124133654815) |
| min\_duration\_days | integer | 投放此版位的任务最短投放时间，单位天。默认不限制 |
| max\_duration\_days | integer | 投放此版位的任务最长投放时间，单位天。默认不限制 |
