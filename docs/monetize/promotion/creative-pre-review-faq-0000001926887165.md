# FAQ

- <strong>新建投放推广任务时，为什么点击“选择预审创意”，提示“该展示类型下，暂无预审创意”？</strong>

  请前往创意预审库检查是否存在与推广任务“创意展示”类型一致的素材，并且该素材已通过审核。例如，如果您在创意预审库中只提交了一份“横版大图”素材，该素材通过审核，但您在创建推广任务时选择了“横版视频”，则无法选择预审创意。
- <strong>新建投放推广任务时，选择了相应的预审创意素材并进行修改，是否会触发新的审核？</strong>

  这样的操作会导致预审创意失效。您修改以后提交的素材，会作为普通创意（非预审创意）重新触发任务创意审核流程。
- <strong>创意预审库最多可支持创建多少个预审创意素材？</strong>

  目前最多可支持创建20个预审创意素材。
