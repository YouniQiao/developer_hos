# 智能扩展

智能扩展是人群定向的一种方式。该能力通过算法和大数据能力，找到与开发者的种子人群（通常指有转化行为的价值用户）具有相似特征的人群，扩展种子人群量级，以满足推广定向投放需求。

种子人群分行业lookalike是智能扩展的一项新能力，基于不同行业的转化目标和用户特征，建立行业级lookalike模型，支持选择正样本和负样本，进行人群智能扩展，以帮助开发者更精准地定向人群，提高转化率，降低获客成本。

| <strong>功能对比</strong> | <strong>旧智能扩展</strong> | <strong>新智能扩展</strong> |
| --- | --- | --- |
| <strong>算法模型</strong> | 通用模型 | 行业级lookalike模型 |
| <strong>算法模型数量</strong> | 1个：通用 | 12个：通用、电商、泛娱乐、金融、金融授信、旅游、汽车、视频、地产、短剧、游戏、阅读 |
| <strong>选择人群</strong> | 一般人群 | 正样本人群：开发者的种子人群，一般指具有转化行为的价值用户。  负样本人群：与种子人群特征相反或相关性不大的人群。 |
