# 对接归因

根据需要选择您的归因对接方案，建议您优先使用监测链接进行对接。

## 监测链接归因

参考[监测链接](https://developer.huawei.com/consumer/cn/doc/promotion/bp-functions-link-introduction-0000001299018740)完成监测链接方案的对接。

![](./img/0000000000011111111.20251117124629.88213537060465626348951391175302_50001231000000_2800_EA86FC39FF56213A4C4A9973D3B4B498D49B36C829D13A61159619E9BF1626B2_645b43c342e3.png) 

- 4个监测链接，需要至少填写1个，一般填写在“下载完成”。
- 请确保监测链接中增加\_\_CALLBACK\_\_参数。
- 监测链接配置完成后，请务必使用[链路测试](https://developer.huawei.com/consumer/cn/doc/promotion/bp-functions-link-test-0000001352018577)进行联调测试。

参考[智能分包](https://developer.huawei.com/consumer/cn/doc/promotion/bp-functions-intelligent-subcontract-introduction-0000001285128664)完成智能分包方案的对接。
