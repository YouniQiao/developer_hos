# 审核时间

1. 审核周期为素材提交后的1个工作日。
2. 审核时间为工作日9:00到17:00。 说明：工作日17:00前提交的素材，当天审核完；非审核时间提交的素材自动推迟到下一个工作日审核。

![](./img/880cb22136bf.png) 

广告素材修改后提交将重新记录广告素材审核时间。
