# 2025年11月

<strong>典型驳回行业：教育、游戏、电商、小说阅读。</strong>

<strong>典型驳回问题场景主要涵盖：</strong>

一、广告内容禁止投放伪科学资讯内容。

二、广告中禁止出现 “原价”等描述。

三、广告内容不得使用带有不良诱导、沉迷游戏的描述。

四、广告素材禁止展示丑陋、恶心不适的画面，影响用户体验。

五、广告不得涉及带有明显性暗示的场景。

<strong>一、广告内容禁止投放伪科学资讯内容</strong>

1.禁止使用无科学依据的营养、养生、护肤知识、生活窍门/建议，通过噱头、未经证实的知识吸引人关注。

| --- | --- | --- |
|  |  |  |
| ×问题解析：不支持使用正常食物与疾病划等号 | ×问题解析：不支持将普通食物“神话”包装有医疗保健效果 | √支持使用符合常理的资讯内容 |

2.禁止利用虚构事迹推广产品或服务。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> | <strong>通过示例</strong> |
| --- | --- | --- |
|  |  |  |
| ×问题解析：不支持虚构事迹描述，夸大推广商品实际作用 | ×问题解析：不支持虚构事迹描述，夸大推广商品实际作用 | √支持使用符合常理的资讯内容 |

3.禁止无科学依据的商品功效分析/建议/介绍。

1.禁止出现易引发恐惧或抵触的生物形象：包括真实或高仿真的五毒（蛇、蝎子、蜈蚣、蟾蜍、壁虎）与软体动物（如蜘蛛）。
