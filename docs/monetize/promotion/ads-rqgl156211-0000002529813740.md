# 创建人群

【简介】新增人群用于投放

<strong>请求地址</strong>

https://svc-drcn.ads.huawei.com/opendmp/v1/audience/add

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>长度</strong> | <strong>必填</strong> | <strong>说明</strong> |
| dmpID | String | 255 | Y | 三方DMP标识 |
| audiences | List&lt;AudienceInfo&gt; | - | Y | 人群列表  最长100个 |

<strong>AudienceInfo</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>Parameter name</strong> | <strong>Type</strong> | <strong>Length</strong> | <strong>Mandatory</strong> | <strong>Description</strong> |
| id | String | 128 | Yes | 指定的人群标识，按照人群标识做精确匹配。 |
| name | List&lt;LanguageInfo&gt; | - | Yes | 人群名称。  支持多语.  第一个值为默认值。最多200个。 |
| description | List&lt;LanguageInfo&gt; | - | No | 人群描述。  支持多语，  第一个值为默认值。最多200个。 |
| scale | String | 128 | Yes | 覆盖人数。 |
| url | AudiencePackage |  | Yes | 人群包下载信息，新增/更新人群包时必填 |
| price | List&lt;BusinessInfo&gt; |  | No | 传空值即可 |

<strong>AudiencePackage</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>参数名</strong> | <strong>类型</strong> | <strong>长度</strong> | <strong>必填</strong> | <strong>说明</strong> |
| locationType | int | 1 | N | 固定传1  1-华为OBS |
| downloadUrls | List&lt;String&gt; | - | Y | 人群包地址，  每个人群不超过100个文件路径。  国内传：objectID（即DMPID/文件名称.zip） |
| deviceIDType | int | 1 | N | 人群包中的设备ID类型 |

<strong>BusinessInfo</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>Parameter name</strong> | <strong>Type</strong> | <strong>Length</strong> | <strong>Mandatory</strong> | <strong>Description</strong> |
| type | int | - | No | 传空 |
| price | Float | - | Yes | 固定传0即可 |
| currency | String | 3 | No | 传空 |

<strong>LanguageInfo</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>Parameter name</strong> | <strong>Type</strong> | <strong>Length</strong> | <strong>Mandatory</strong> | <strong>Description</strong> |
| language | String | 3 | Yes | 语种。  传ISO639-1 Code |
| country | String | 2 | No | 语言国家，ISO 3166-1 alpha-2两位字母编码。  传中国-CN |
| value | String | 256 | Yes | 传人群包名称 |

<strong>请求示例</strong>

POST http://$\\{base\_address\\}/opendmp/v1/audience/add

HTTP/1.1

Headers:

Content-Type: application/json

Authorization：Bearer DAEAAIX7IS\*\*\*\*\*Pd81PDmWiM1KS2

Body:

```
{
    "dmpID": "829411234539267456", //Ads运营提供的dmpID
    "audiences": [{
            "id": "联调测试test001", // 人群特殊标志符，必填 不重复
            "scale": "5001", // 预计人群计算覆盖人群
            "name": [{
                    "language": "zh", //支持多语
                    "value": "华为测试001" //人群名称
                },
            "description": [{
                    "language": "zh",  //支持多语
                    "value": "华为测试人群1号"  //人群描述
                },
            "url": {
                "locationType": "1", //固定传1
                "downloadUrls": ["829411234539267456/A.zip"], //上传至OBS文件路径
                "deviceIDType": "3" //人群包中的设备ID类型
            }
        }]
}
```
