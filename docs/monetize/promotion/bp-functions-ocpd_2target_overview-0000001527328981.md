# 功能介绍

oCPD双目标出价功能适用于多指标衡量推广效果的应用，最大程度满足开发者对浅层和深层转化目标的考核要求。

oCPD双目标出价功能支持如下场景：

- <strong>激活-首日ROI：</strong>在oCPD浅层转化目标激活应用出价前提下，增加深度转化目标首日ROI设置。
- <strong>激活-次日留存和激活-7日留存</strong>：在oCPD浅层转化目标激活应用出价前提下，增加深度转化目标次日留存率和7日留存率设置。
