# 服务商财务管理

## 概述

**广告账户成功注册后，您需要对广告账户进行充值，才能成功投放广告。当前仅服务商账户可以充值，充值完毕后转账给下一级。**

**广告账户中有多个余额类型，各余额类型的扣费顺序为：纯赠送金\_华为自有媒体>纯赠送金\_通用>充值赠送金\_华为自有媒体>充值赠送金\_通用>返利金(阿斯比格/华为服务(香港))\_华为自有媒体>返利金（阿斯比格/华为服务(香港)）\_通用>现金>授信。**

![](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250513090207.82545982410540270479187609997352:50001231000000:2800:9C30DD87B412EE2861F3237C848265D6134C9F98BB751645F22C76A5A4C8B057.png) 

- 充值赠送金需搭配与华为商定达成的比例，与其他余额类型（现金+返利金+授信）一起进行消耗。

- 纯赠送金支持单独消耗，单独转账，不受按比例消耗限制，纯赠送金不会影响子客账户余额平衡。

## 充值方式

- **服务商系统中，服务商可以通过**[线下充值](#ZH-CN_TOPIC_0000001328517614__section5790142118232)**和**[线上充值](#ZH-CN_TOPIC_0000001328517614__section4436101052314)**的方式进行充值，仅服务商账户可以充值，子客服务商账户与子客账户均不可充值，充值完成后，服务商需要**[转账](#ZH-CN_TOPIC_0000001328517614__section17794102114232)**给下一级，只有子客账户有余额才能正常投放广告。**
  - [线下充值](#ZH-CN_TOPIC_0000001328517614__section5790142118232)**：**您可以通过银行转账到[指定银行账户](#ZH-CN_TOPIC_0000001328517614__li136043713466)的方式进行充值，银行转账之后需要您在服务商账户中提交充值申请，审批通过后充值进现金账户余额。
  - [线上充值](#ZH-CN_TOPIC_0000001328517614__section4436101052314)：您可以通过线上充值的方式在广告投放之前先充值，金额将会充值到现金账户余额。
- **充值的银行账号信息如下：**
  - 鲸鸿动能广告提供阿斯比格和华为服务（香港）两个银行收款账户，您需要根据主要投放区域选择阿斯比格或华为服务（香港）充值。
  - 您的充值币种需要与您服务商账户注册时选择的币种保持一致。账户的计费也按照该币种计算。例如：您账户注册时选择的币种是美元，则您需要用美元充值，您的账户计费也将以美元计算。

## 服务商充值-线下充值

**操作流程：**

![](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250513090208.80959372893852681198384642291991:50001231000000:2800:5CB64A03EFC2ED3FD7B1E793652822364D9907FA3F9DE1FE8EC50B68C20AC178.png)

**操作步骤：**

1. 银行转账。

   您需要使用您的企业银行账户转账至鲸鸿动能广告提供的[银行账户](#ZH-CN_TOPIC_0000001328517614__li136043713466)，转账币种需与您服务商账户注册时选择的币种保持一致。为保障广告计划顺利创建并投放，选择线下银行账户转账模式，请提前与银行确认转账时效（打款到账时间为3-5个工作日）。

   ![](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250513090208.51654920043917931105265481613530:50001231000000:2800:83BC85D1FCAD07C05642267374958BF5A85015E8E23122F68DBC001932783369.png) 

   - 线下充值不支持个人银行账户充值。
   - 线下充值仅支持企业银行账户充值，且您用来充值的银行账户必须与广告账户主体一致，否则将无法充值到广告账户中。

2. 服务商平台提交充值申请并提交审核。

   **点击“财务管理”>"充值开票"**，填写相关信息，补充转账凭证，并提交审核，审核时间为1-3个工作日。

   ![](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250513090208.93588283648420276947971425950342:50001231000000:2800:40AD43E112E217084D07E9A2281F6D42EA1982127D291AC1BD6AF5F72361B5F2.png "点击放大")

3. 充值到账。

   充值订单审核完成后，您可以在首页看到余额，也可以在“**财务管理**“>"**充值记录**"中查看充值记录。

## 服务商充值-线上充值

![](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250513090208.34408974915259606155356908093311:50001231000000:2800:3A62BB321B93FD75297D244E8892BE5B61D7F157D800B32D21F3E19014956794.png) 

如果您需要使用此功能，需要申请[特性通行名单](https://developer.huawei.com/consumer/cn/doc/distribution/promotion/addtongxing-0000001128278195)。

- 申请条件：
  - 只支持注册地为下述国家/地区的广告账户使用银行卡线上充值功能：

    德国、英国、爱尔兰、以色列、法国、意大利、西班牙、波兰、罗马尼亚、捷克、芬兰、土耳其、乌克兰，新加坡、马来西亚、泰国、印度尼西亚、中国香港、菲律宾、沙特阿拉伯、埃及、南非、印度、墨西哥、智利、秘鲁、哥伦比亚。
  - 已通过实名认证的服务商可以使用线上充值方式进行现金账户充值，充值完成后实时到账。
  - 当前仅支持以下币种的服务商账户使用线上充值功能：EUR、CNY、JPY、GBP。
  - 如果您修改了鲸鸿动能广告账户注册企业名称，您需要重新申请线上充值的使用功能，企业名称变更请参考[企业名称修改](/docs/monetize/promotion/bpos-delivery-task-account-manage-information-0000001328357938)。
  - 充值的银行卡户名必须和鲸鸿动能广告账户注册名称一致，不支持绑定个人银行卡。
  - 您即将绑定的银行卡必须未被任何其它鲸鸿动能广告账户使用过。
  - 您即将绑定的银行卡暂时仅支持借记卡。
- **充值步骤**：

## 授信账户

- 授信账户：指的是您（服务商）通过向鲸鸿动能广告申请开通授信账户（授信额度存在有效期），并分配一定的授信额度，服务商可以将授信额度通过转账的方式转到子客服务商的账户中，再由子客服务商通过转账的方式转到子客的账户中，或者服务商可以将授信额度通过转账的方式直接转到子客的账户中，子客进行消耗后，服务商的账户中才会产生授信账单，服务商需要按照“线下充值”的方式进行还款。
  - **线下充值：**您可以通过银行转账到[指定银行账户](#ZH-CN_TOPIC_0000001328517614__li1720515314473)的方式进行还款，银行转账之后需要您在鲸鸿动能广告界面提交还款申请，审批通过后，还款成功。
- 额度到期：如果您的授信额度到期，子客已产生消耗，您仍然需要还款；如果子客并未进行消耗，则子客账户中的通过授信获得的余额将会清零，您无需还款。
- 还款完成：还款完成后，系统将恢复服务商授信账户额度，您可以在授信账户中的“详情”查看：授信截止日期、状态、总额度、可用额度。

  ![](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250513090209.35288117545223583546465632087309:50001231000000:2800:57AA05B0F69CC5A42D086FA9BAA1AFC47D99E17B8BD96802DF80404973717149.png)
- 授信通知：以下几种情况，您的账户将会收到鲸鸿动能广告的授信通知：
  - 额度恢复完成。
  - 授信账户额度低于20%。比例默认为20%，您也可以修改比例。
  - 授信账户快到期。
  - 授信账户到期。

### 授信账户申请

您需要发送申请邮件向对应区域进行申请，鲸鸿动能广告会根据您提供的信息进行信用等级评估，如果评估通过，根据评估结果给予授信额度，并在签署相关协议后生效。

- 申请邮件模板：
  - 公司名称。
  - 公司财报。
- 区域邮箱：详情请参考鲸鸿动能广告官网中的“[商务合作](https://ads.huawei.com/usermgtportal/home/index.html#/support)”。

### 授信线下充值

1. 银行转账。

   您需要先进行账户线下充值，可以使用银行转账的充值方式，请提前与银行确认转账时效（公对公跨行转账到账时间一般为1-3个工作日）。

   ![](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250513090209.24009539879921387591267221973791:50001231000000:2800:4EDF10C6003C2DEA44ECEE1FB36758B1CCE4CF701C58D34C83EF7CDAC2B7C1A2.png) 

   - 线下充值不支持个人银行账户充值。
   - 线下充值仅支持企业银行账户充值，且您用来充值的银行账户必须与广告账户主体一致，否则将无法正常还款，影响授信使用。

2. 在鲸鸿动能广告平台提交还款申请。

   点击“首页”，点击账户总览中的“充值”，填写充值信息。

   ![](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250513090209.37282122318938823005062436509427:50001231000000:2800:8DF2BDEBD606D414E547EF08879406D1A2207254BCAAC5E3852DD2B040501B39.png "点击放大")

   - 充值类型：选择“线下充值”。
   - 投放区域：按照鲸鸿动能广告为您开票的主体进行充值。
     - 投放区域选择欧洲，则收款方为阿斯比格。
     - 投放区域选择亚洲、非洲、拉丁美洲，则收款方为华为服务(香港)。
   - 充值用途：选择“授信账户还款”。
   - 还款金额：系统将会提示您最大充值金额为\*\*\*，请在限额内输入金额。
   - 打款凭证：上传转账的银行截图或银行流水等转账资料的扫描件作为凭证，付款主体必须与广告账户开户主体保持一致。
   - 发票抬头：默认选择您在开户时填写的发票信息，无法修改。
   - 充值备注：请注明付款账号的开户名称、开户银行及银行账号。

3. 确认还款信息并提交审核。

   审核时间：一般为3个工作日。
