# (可选)授权客户投放伙伴管理账户

![](./img/0000000000011111111.20260226163844.35616619102590169120765780518132_50001231000000_2800_4CF27FCD5AFA970C9DBC54A5ADC9428663D24D096E7E40D72B9798E568C634D6_5f54506b23cb.png) 

在进行授权前，需要先申请获得推广权限，具体请参见[申请推广评测](https://developer.huawei.com/consumer/cn/doc/promotion/bp-start-guest-apply-evaluation-0000001346654709)。

<strong>授权应用参加客户激励活动获得的激励金（如：星火计划、月度激励等）将通过被授权的客户投放伙伴发放。</strong>

应用授权前，需要暂停所有“执行”状态的推广任务。同样，应用取消授权给客户投放伙伴，也需要客户投放伙伴投放操作账户暂停所有“执行”状态的推广任务。

直客如果需要客户投放伙伴推广，则进行如下操作。

## 操作步骤

1. 以直客账户登录[华为应用市场应用推广平台](https://ads.huawei.com/cn/)，点击“工具”—“投放辅助”—“应用授权管理”，进入授权页面。

   ![](./img/0000000000011111111.20260226163844.53131141938061792695623472085908_50001231000000_2800_A45A4640E9A4AFAEBD517CE98D3A6031F18B36463522BE8D06950039A85A90DE_e1618647b406.png "点击放大")
2. 点击“申请授权”，填写相关授权信息后，点击“提交审核”。当审核通过后，即可由授权的客户投放伙伴投放操作账户（子客账户）进行应用推广。

   ![](./img/0000000000011111111.20260226163844.78123569068279517422308122915464_50001231000000_2800_1E6C2FEFAF967C32C3908EFF1C3D71A8971CAD34EC24F9B2746EB32849637EAC_97511727c9bf.png "点击放大")

   ![](./img/0000000000011111111.20260226163844.94630819251239236978356159496365_50001231000000_2800_8D9C4DC23CE96E962110E20D5A3FA7DCF3519FB5415C4598207DA13B32C81404_58bb9d2746b8.png "点击放大")

   ![](./img/0000000000011111111.20260226163844.24740405994118740823032692478911_50001231000000_2800_FBC4EDA76F5ACBD84CF37FDC8D20A24F156EE08AF2DDFDDEC993F1B1E2EAEE9B_1e64e59d19a0.png) 

   “投放操作账户ID”配置项可由客户投放伙伴子账户（子客服务商）登录[服务商管理平台](https://ads.huawei.com/cn/)，点击“首页”—“子客清单”，查看子客ID字段获取。

   ![](./img/0000000000011111111.20260226163844.01545508316294407275462922739261_50001231000000_2800_B93EB11ECF858F18F89F7B9DCF028E26547069D9C383197E02F11FEF09DD20C2_2cf6b8bf7c99.png "点击放大")
3. 如果直客账户已完成授权后，想取消授权，则点击“取消授权”即可。

   ![](./img/0000000000011111111.20260226163845.80986441362255866301961232252055_50001231000000_2800_AEE44B33FA0C395B39EB1740EA7FADE7E48CD37253C332961F3202CCAF9B0927_80e08add667f.png "点击放大")
