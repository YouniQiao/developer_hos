# 更新人群包基础信息

【简介】合作伙伴更新人群信息列表

<strong>请求地址</strong>

https://svc-drcn.ads.huawei.com/opendmp/v1/audience/update

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>长度</strong> | <strong>必填</strong> | <strong>说明</strong> |
| dmpID | String | 255 | Y | 三方DMP标识。 |
| audiences | List&lt;AudienceInfo&gt; | - | Y | 人群列表。  最长100个。 |

<strong>AudienceInfo</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>Parameter name</strong> | <strong>Type</strong> | <strong>Length</strong> | <strong>Mandatory</strong> | <strong>Description</strong> |
| id | String | 128 | Yes | 指定的人群标识，按照人群标识做精确匹配。  Id不支持编辑 |
| name | List&lt;LanguageInfo&gt; | - | No | 人群名称。  支持多语.  第一个值为默认值。最多200个。 |
| description | List&lt;LanguageInfo&gt; | - | No | 人群描述。  支持多语，  第一个值为默认值。最多200个 |
| scale | String | 128 | No | 覆盖人数 |
| price | List&lt;BusinessInfo&gt; | - | No | 传空 |

<strong>BusinessInfo</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>Parameter name</strong> | <strong>Type</strong> | <strong>Length</strong> | <strong>Mandatory</strong> | <strong>Description</strong> |
| type | int | - | No | 传空 |
| price | Float | - | No | 传空 |
| currency | String | 3 | No | 传空 |

<strong>LanguageInfo</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>Parameter name</strong> | <strong>Type</strong> | <strong>Length</strong> | <strong>Mandatory</strong> | <strong>Description</strong> |
| language | String | 3 | Yes | 语种。  传ISO639-1 Code |
| country | String | 2 | No | 语言国家，ISO 3166-1 alpha-2两位字母编码。  传中国-CN |
| value | String | 256 | No | 传人群包名称 |

<strong>请求示例</strong>

POST http://$\\{base\_address\\}/opendmp/v1/audience/update

HTTP/1.1

Headers:

Content-Type: application/json

Authorization：Bearer DAEAAIX7IS\*\*\*\*\*Pd81PDmWiM1KS2

Body:

```
{
    "dmpID": "829411234539267456", //Ads运营提供的dmpID
    "audiences": [{
            "id": "联调测试test001", // 人群特殊标志符，必填 不重复
            "scale": "5001", // 预计人群计算覆盖人群
            "name": [{
                    "language": "zh", //支持多语
                    "value": "华为测试001_update" //人群名称
                },
            "description": [{
                    "language": "zh",  //支持多语
                    "value": "华为测试人群1号更新描述"  //人群描述
                }            ]
        }
    ]
}
```

<strong>响应字段</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 参数名 | 类型 | 长度 | 是否必填 | 说明 |
| code | int | - | Y | 错误码 |
| message | String | 2048 | N | 描述 |
| value | object | - | N | NA |
