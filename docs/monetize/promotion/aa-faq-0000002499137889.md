# FAQ

| <strong>问题分类</strong> | <strong>问题</strong> | <strong>答案</strong> |
| --- | --- | --- |
| 账户体系 | 整合升级后，哪些账号会合并为一个账户进行投放？ | 仅当您的华为账号在“鲸鸿动能”和“应用市场应用推广”两个平台均注册为直客/直客管理者账户类型时，这两个广告账户才会合并，统一为一个账户进行投放。其他类型的账户不会合并，仍将分别投放。  服务商账户，仅可选择一个推广范围。如果您的华为账号在应用推广和鲸鸿动能平台分别开通过不同类型的账户，整合升级后登录时需先选择账户类型，随后才能进入对应平台进行操作。 |
| 账户体系 | 服务商账户整合升级后，子客账户可以选择多个推广范围吗？怎么操作？ | 服务商账户整合升级后，子客账户仅可选择一个推广范围。如果您的华为账号在应用推广和鲸鸿动能平台分别开通过不同类型的账户，整合升级后登录时需先选择账户类型，随后才能进入对应平台进行操作。 |
| 账号登录 | 是否可以使用应用市场的应用推广账号和密码登录鲸鸿动能平台？ | 是的，您可以直接使用原有应用市场的应用推广账号和密码登录鲸鸿动能平台，无需重新注册。 |
| 账号登录 | 指定账号持有者——只有子客账户需要强指定，还是所有账户类型割接流程都需指定一个账户持有者的华为账号？ | 仅应用市场应用推广的客户投放伙伴子账户和投放操作账户需要指定持有人账号，其他账户类型不指定 |
| 账号登录 | 账号持有人是怎么设定的指定规则？ | 授权规则：  1）账户已授权管理员：默认授权您在应用市场应用推广最早授权为“管理员”角色的华为账号为账户持有人。  2）账户未授权管理员，已授权浏览员：系统自动处理迁移存量广告账户，不指定授权账户持有人，您可后续自行授权账户持有人。原来客户投放子伙伴账户浏览员，整合升级后角色为观察员；原投放操作账户浏览员，整合升级后角色为数据分析师。  3）未授权管理员和浏览员：您的客户投放子伙伴账户、投放操作账户在处理账户迁移前未授权华为账号，系统自动处理迁移存量广告账户，不授权账户持有人，您可后续自行授权账户持有人。 |
| 账号登录 | 现有账号持有人的华为账号想要变更，怎么处理？ | 客户投放伙伴子账户：使用子账户账号登录服务商管理平台——账户管理——账号信息，进行绑定、换绑  投放操作账户：使用投放操作账户登录投放端——工具——广告账号管理，进行账号持有人的绑定、换绑 |
| 账号登录 | 有些“账户ID”变了，是什么原因？ | 对于直客及客户投放伙伴主账户，应用推广原先在右上角客户信息中仅展示客户ID（即UPID）；整合升级后，系统将自动生成账户ID，右上角账户信息将同时展示账户ID和客户ID（UPID）。 |
| 账号登录 | 客户投放子账户管理员新创建了子客账户后，部分账户协作者登录后页面不展示新邀请的子客账户，是什么原因？ | 新创建的子客账户需要确保已在子客服务商平台里分配给账户的协作者，对应的二代账户持有人把对应的账户授权给对应的二代操作员（管理员）或观察员（浏览员） |
| 账号登录 | 整合升级后，原应用市场应用推广投放平台还能登录使用吗？ | 整合升级后，原应用市场应用推广投放平台将无法登录使用，登录页面会自动跳转至鲸鸿动能投放端。您可在跳转后的页面点击“登录”进入鲸鸿动能平台，也可以直接访问鲸鸿动能平台并使用原有账号登录。  鲸鸿动能平台登录入口：https://ads.huawei.com/usermgtportal/home/index.html#/ |
| 投放相关 | 平台融合升级后，账户数据、投放效果、OCPX模型以及API的使用会受到影响吗？ | 平台融合升级不会影响账户内的数据、投放效果或OCPX模型的使用，您在应用市场应用推广支持的所有工具和产品能力仍可正常使用。投放端整合升级后，现有任务的投放和模型预估均不受影响，历史API对接等仍可继续使用。需要注意的是，API依旧为两套体系，应用市场应用推广API的接口范围不变，原有API权限与功能均保持不变。 |
| 投放相关 | 整合升级后，OCPX投放目标是否会合并？ | 不会合并，不同投放范围独立管理，投放任务类型、支持的计费方式以及投放目标均未兼容，仍由各自团队分别维护和优化相关功能。 |
| 投放相关 | 整合升级后，投放流量是否会合并？账户如何选择投放范围？ | 整合升级后，投放流量不会合并。直客账户拥有双权限的客户，在账户进行任务创建时，需要先选择投放范围（展示广告网络或应用市场应用推广），选择后方可继续新建任务。 |
| 投放工具 | 监测链接、智能分包、人群定向等投放工具会合并吗？ | 不会合并，相关投放工具仍维持两套体系。对于直客双权限客户，可以在工具栏中分别查看展示广告网络和应用市场应用推广对应的工具能力，工具的使用和管理保持独立。 |
| 投放层级FAQ | 创建任务，无法选择已有计划 | 应用推广中，计划层与任务层一一对应，一个计划只能绑定一个任务。如果该计划下已有任务，则无法新建多个任务 |
| 投放层级FAQ | 计划层级不可以暂停？ | 计划层级暂未适配计划启停的开关，应用市场应用推广的任务启停操作仍在任务层级操作。 |
| 投放层级FAQ | 计划预算是否支持次日预算? | 未适配，仅鲸鸿动能展示广告网络支持 |
| 投放层级FAQ | 应用推广中的计划层级与任务层级是什么关系？存量广告任务迁移后如何处理？ | 计划层级是为适配新增的广告结构层级而引入的。在应用推广中，计划层级与任务层级一一对应，即一个计划只能绑定一个任务。存量广告任务迁移后，系统会在账户内自动为每个广告任务生成一个对应的计划层级，计划层级的名称与原任务层级名称保持一致。 |
| 数据报表 | 针对应用市场应用推广范围，是否支持新增汇总行？ | 未适配，后续支持 |
| 数据报表 | 升级后是否支持查看分时数据？ | 不支持 |
| 数据报表 | 升级后，子客登录报表查看有同等金额的转出和转入记录，对方账户ID是二代账户ID | 整合升级时会把钱从子客转到二代，整合升级后会自动把钱转回子客 |
| 权限申请 | 直客团队账户权限（经理账户权限）是不是申请一个权限就可以了？ | 仅需开通【应用市场应用推广】权限时，请联系应用市场应用推广相关运营人员进行申请。目前，直客团队账户（经理账户）依然采用两套白名单管理。如需同时开通【应用市场应用推广】和【鲸鸿动能广告】权限，请分别联系对应平台的运营人员，分别提交权限开通申请。 |
| 权限申请 | 只需要开通“应用市场应用推广”的权限，应该怎么办？ | 您可以联系“应用市场应用推广”相关运营人员申请开通或者通过客服邮箱developer@huawei.com发送申请 |
| 开户相关 | 服务商客户投放伙伴子账户、子客账户开户接收邮箱可以重复吗? | 开户接收邮箱可以重复，但是点击邮件内链接，进行注册开户的华为账号不可重复。 |
| 财务资金相关 | 充值金额是否共用？ | 整合升级后，充值入口统一为鲸鸿动能广告投放基金（原应用市场推广基金将逐步下线）。充值后，资金可同时用于应用市场应用推广和鲸鸿动能展示广告网络推广。 |
| 财务资金相关 | 后续充值资金只充值到鲸鸿动能基金吗？ | 整合升级后，推广资金充值/扣款都在鲸鸿动能广告投放基金 |
| 财务资金相关 | 原未处理的结算/发票是否全部转移至鲸鸿动能处理？ | 不影响，开票均在开发者联盟平台处理 |
| Marketing API | API调用权限是否打通，接口是否合并？ | API接口仍维持两套体系，客户在应用市场应用推广的API权限及接口调用范围保持不变。 |
