# 定向

## 概述

鲸鸿动能广告的竞价广告目前支持的定向条件包括地域、语言、App行为、性别、年龄、App兴趣、设备、联网方式、自定义人群等。通过合理设置广告定向，可以将您的广告仅展示给定向人群，提升转化率。如果您设置的定向条件过窄，可能会影响您的广告流量。

## 定向设置

**不限：**对所有的定向，默认选择不限，此时您的广告将会投放给所有用户。您可以根据以下定向来设置您广告的投放人群：

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/51/v3/K3tMa9buTsCf74b5-h_W7w/zh-cn_image_0000001445376242.png?HW-CC-KV=V1&HW-CC-Date=20260530T050720Z&HW-CC-Expire=86400&HW-CC-Sign=E67C94646B8B5E52E0F01DF2922814DACDDED177451A6D31275A2FF8A19CF5A4 "点击放大")

- **地域：**您可以按照不限、账户默认注册地、自定义地理位置三种方式选择用户当前所在地域进行投放定向。您选择账户默认注册地及自定义地理位置时，您需要选择人群状态以及包含/排除：
  - 地域选择：
    - 账户默认注册地：如果您的注册国家/地区是德国，则此处默认显示德国，选择后，广告将会投放给德国的用户。
    - 自定义地理位置：您可以按照大洲、国家或地区、州/省、城市进行定向。

    地域定向支持的国家请参考[海外地域字典数据](https://developer.huawei.com/consumer/cn/doc/promotion/attachments-0000001532611905#ZH-CN_TOPIC_0000001532611905__li66991118709)

    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e3/v3/_lpkwFKUTaOmAhhtbgkW4g/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260530T050720Z&HW-CC-Expire=86400&HW-CC-Sign=EBC4A7E15E887371817E38DE7DC554060B1E71772BCFC251EE695D19098DA845) 

    如果您的投放地域为“俄罗斯”，根据俄罗斯政府颁布的新广告法案要求，广告账户必须填写税号或邓白氏编码或营业执照号（三者选其一），如已完成认证，请忽略该提示。

    - 邓白氏编码或营业执照号补充方式：此操作请通过[在线提单](https://developer.huawei.com/consumer/cn/support/feedback/#/)联系鲸鸿动能广告平台或者联系您的广告运营，发起广告账户[实名认证](https://developer.huawei.com/consumer/cn/doc/start/itrna-0000001076878172)，发起后，您登录广告账户，单击“工具”->“广告账户管理”，补充认证信息并等待审核，审核通过后即可进行广告投放。
    - 税号补充方式：仅直客、一级服务商可填写税号，详情请参考[查看&修改发票信息](https://developer.huawei.com/consumer/cn/doc/promotion/invoice-0000001051704326#section929018425482)。
  - 人群状态：
    - 当前在这里的人：当前位置在该区域的人。
    - 是否接受漫游流量：漫游流量是指设备出厂国家/地区与用户当前所在国家/地区不一致的流量，但此场景下用户当前所在国家/地区（IP）仍属于您所选定向国家/地区，广告带来的所有曝光均来源于定向国家/地区。
      - 如果您勾选“接受漫游流量”，广告按照您选定的国家/地区进行投放，但是可能会存在漫游流量，我们建议您开启此功能，可以在您的定向范围内，为您的广告获取更多曝光机会。

        举例：假设您想投放在智利（与华为服务（香港）有限公司进行签约），您勾选了漫游流量，那么您可能会购买到当前在智利的意大利用户流量（与Aspiegel SE进行签约）的流量。所以您在开票时，会收到来自华为服务（香港）有限公司、 Aspiegel SE的发票。
      - 如果您未勾选“接受漫游流量”，则广告严格按照您选定的国家/地区进行投放。
  - 包含/排除：
    - 包含：您的广告将会被展示给这些地域的用户。
    - 排除：您的广告将不会被展示给排除地域的用户。

  在您推广目的为应用下载或应用促活时，请注意选择的投放地域应该是您的应用已经上架的地域。

  当您投放网络为应用市场时，您设置的推广国家/地区，只支持从此应用已经[审核通过](https://developer.huawei.com/consumer/cn/doc/promotion/review-0000001052064324)的国家中进行选择，同一任务中可以选择多个国家/地区进行投放。

  当您投放**商品广告**时，地域自动默认按商品的“目标销售国家”作为国家定向，不支持更改国家定向，单击“自定义”，支持选择到该国家具体的“州/省”或者“城市”。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/4d/v3/gDpikVphRL-tSJD8nWCZiQ/zh-cn_image_0000001445696266.png?HW-CC-KV=V1&HW-CC-Date=20260530T050720Z&HW-CC-Expire=86400&HW-CC-Sign=4AB557BE7A19A4AEB7C5C3C3EE7D5CA2D7937F7159E7C66DA5ED9BC5A416B73F)

- **运营商**：您可以定向使用特定运营商的用户。应用市场展示广告不支持使用运营商定向功能。
- **语言**：通过语言定向，您可以选择希望覆盖使用哪种语言的用户，您的广告将会展示给手机上设置这些语言的用户。如果您未选择语言，您的广告创意语言应该与投放地域主流语言一致。
- **性别**：指定只投放给某个性别的用户。
- **年龄**：您可以覆盖那些可能在年龄段上符合您的特定要求的潜在客户。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/1f/v3/zqsPRlIkSdaT2GwxusFoIw/zh-cn_image_0000001226625357.png?HW-CC-KV=V1&HW-CC-Date=20260530T050720Z&HW-CC-Expire=86400&HW-CC-Sign=423D5AEDE0A094E7B1E17167AAD66D4212596A6109D11C7AAE25D28C5E40A764)

- **App安装：**当您的推广产品为应用，您需要选择用户的应用状态，广告可以对“已安装”和“未安装”的用户进行投放。
  - 如果您想投放给未安装您应用的用户提升下载，请选择“未安装”。
  - 如果您想对已安装您应用的用户进行应用促活，请选择“已安装”。
- **App行为：**根据用户对某类App的使用行为进行定向，支持三个定向条件：
  - 一个月内活跃：当前设备上安装有此类应用，且在过去一个月内至少使用过一次。
  - 已安装：当前设备上安装有此类应用。
  - 未安装：当前设备上未安装此类应用。
- **App兴趣：**根据用户对每类App的兴趣程度进行定向，将用户对每类App的兴趣划分为3个分类，分为不限、中等兴趣用户、高兴趣用户。
- **设备：**您可以根据用户使用的机型品牌进行定向，可以投放给指定品牌的手机用户。
- **联网方式：**您可以根据用户设备的联网方式进行定向，可以投放给wifi、2G、3G、4G、5G的用户。
- **自定义人群**：您可以通过指定投放人群投放给某个人群，也可以指定不向某个人群投放。

  ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/d/v3/LYV7H2F8R0u2rAPQJJlB5w/zh-cn_image_0000001180865666.png?HW-CC-KV=V1&HW-CC-Date=20260530T050720Z&HW-CC-Expire=86400&HW-CC-Sign=9E155006660D7A5B9B3537B54F1FCC272277DB10A7F73555D2530B84151DAEBF)

  - **类别**：指的是人群包的类别，分为公共人群包、专属人群包、私有人群包。更多请参考[人群包](https://developer.huawei.com/consumer/cn/doc/promotion/overview-renqunbao-0000001228033193)。
  - **区域**：指当前人群包所在的区域，分为中国、俄罗斯、欧盟(GDPR)、亚非拉、全球等，每个区域只保存本区域内用户数据，投放时只能选择投放目标区域内的人群包。
  - **覆盖量**：指的是人群包中包含的人群总数。人群包定向和其它定向条件会同时生效，如果人群包包含的用户太少会导致任务覆盖较少。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/0f/v3/sU2Qiso7Rky-hiDec7TJnQ/zh-cn_image_0000001195932460.png?HW-CC-KV=V1&HW-CC-Date=20260530T050720Z&HW-CC-Expire=86400&HW-CC-Sign=24EB2A62CA32272E4BF86B5076A11A61010C65AA2E35722BD360ADE8048B7F1F "点击放大")

- **内容：**您希望自己的广告在什么内容上展示，利用媒体上用户浏览的内容来匹配对应内容的广告任务；开启此定向，可以实现广告更有针对性地覆盖受众群体，目前支持”游戏”和”应用”两类内容，可同时多选。
  - 游戏：您可以选择与自己广告相匹配的游戏类型，支持多选。
  - 应用：您可以选择在哪类应用上匹配投放，支持多选。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a7/v3/_A7toP1gTReG6zjqjyqIrQ/zh-cn_image_0000001377984265.png?HW-CC-KV=V1&HW-CC-Date=20260530T050720Z&HW-CC-Expire=86400&HW-CC-Sign=11EB5D381E29C4C4C4C7C0525DBDFB3EC16E0A02FC2D1465C11F11030AD5B482 "点击放大")

- 智能扩量：

  在您已选的定向条件上进行智能扩量投放，自动学习已有用户和潜在用户的行为、兴趣等特征，为每个广告寻找相关的用户，自动探索更多适合您广告的人群。若您的初始定向条件选择了地域、性别，开启智能扩量时，智能扩量的地域、性别选项为可放开定向，年龄、App行为、自定义人群等选项不可勾选，则系统会在您的广告计划积累到一定转化量后，在地域、性别条件放开限制，进行智能扩量投放，自动探索更多适合广告计划的人群。

  例如：您在定向中选择“法国”、“男”，在“智能扩量”中选择“地域”为“可放开定向“，则系统会在控制成本的前提下逐渐探索法国地区以外的男性用户，但不会有女性人群。

  - 修改：如果在广告投放之前，您修改了定向条件，则智能扩量自动随之修改。如果在广告投放之后，您中途修改了任务的定向（部分定向需要重新[审核](https://developer.huawei.com/consumer/cn/doc/promotion/review-0000001052064324#section10690257178)），或者修改了智能扩量的定向，广告已经建立起来的人群将会受到影响，广告需要重新跑量，请您谨慎修改。
  - 暂停：如果您中途暂停了智能扩量，广告无需重新审核。
  - 复制：如果您复制已启用智能扩量的广告计划，则智能扩量功能也会复制成功。

## 定向包设置

如果多条任务使用相同的定向，单击“**保存为定向包**”且完成操作，系统生成该定向包，更多请参考[定向包管理](https://developer.huawei.com/consumer/cn/doc/promotion/adddingxiang-0000001082374774)。
