# 查询渠道包列表

【简介】通过此接口可以查询渠道包列表数据，需要向运营申请渠道包白名单权限

<strong>请求地址</strong>

https://ads.cloud.huawei.com/ads/v1/tools/channel\_package/query

<strong>请求方法</strong>

<strong>GET</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| filtering | Struct1 | 否 | 过滤条件，若此字段不传，或传空则视为无限制条件 |
| page | integer | 是 | 搜索页码，  取值范围1~10000 |
| page\_size | integer | 是 | 一页展示数量，  取值范围 10~50 |

filtering (Struct1)定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| channel\_package\_name | string | 否 | 渠道包名或者应用包名  不能包含“|”、“^”、“\n”字符 |
| status | string | 否 | 渠道包状态  详见[【渠道包状态】](/docs/monetize/promotion/ads_api73-0000001058884698#渠道包状态) |

<strong>请求示例</strong>

GET ads/v1/tools/channel\_package/paging\_package/query HTTP/1.1

Accept:application/json

Content-Type:application/json

Authorization:Bearer CgB6e3x9ERGComr9dENxZX22iBk+mLuf1yGtQVPUjPJUMrstfKlqpdXk+kfHU9J8ZJ/soYIZHZzT446GeSYumluQuhsK7jvz4kz1Bkms4CLI/rE=

```
{
"filtering": {
"channel_package_name": "324",
"status": "AUDIT_NOT_PASS"
},
"page": 1,
"page_size": 10
}
```

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1[] | 渠道包列表 |

Struct1定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| total | integer | 定向包总条数 |
| data | Struct2[] | 渠道包列表 |

Struct2定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| channel\_package\_id | string | 渠道包ID |
| channel\_package\_name | string | 渠道包名 |
| app\_name | string | 应用名 |
| package\_name | string | 应用包名 |
| version\_code | string | 应用市场版本号 |
| url | string | 上传成功后的渠道包地址 |
| status | string | 渠道包状态  详见[【渠道包状态】](/docs/monetize/promotion/ads_api73-0000001058884698#渠道包状态) |
