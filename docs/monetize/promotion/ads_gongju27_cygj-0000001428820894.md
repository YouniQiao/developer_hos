# 创意工具简介

创意工具是鲸鸿动能为广告主提供的一站式创意制作平台，聚合落地页工具、图片工具、视频工具、创意雷达和互动平台等功能，助力广告主更好地进行广告素材的创作、优化和管理。

<strong>[落地页工具](https://developer.huawei.com/consumer/cn/doc/promotion/ads_gongju14_1-0000001477131173)</strong>

提供30+落地页模板，广告主可直接替换素材输出落地页；提供丰富的营销组件，广告主可自定义组合不同组件快速搭建落地页。

<strong>[图片工具](https://developer.huawei.com/consumer/cn/doc/promotion/ads_gongju15-0000001408836838)</strong>

提供图片尺寸拓展、模板制图和智能制图等功能，广告主可通过尺寸拓展来智能适配版位素材尺寸；通过模板制图替换元素生成图片；使用智能制图生成多样化商品图。

<strong>[视频工具](https://developer.huawei.com/consumer/cn/doc/promotion/ads_gongju16-0000001408676854)</strong>

提供视频尺寸拓展、智能生成视频等功能，广告主可通过尺寸拓展实现视频尺寸的无损拓展、通过智能视频自动衍生视频素材。

<strong>[创意雷达](https://developer.huawei.com/consumer/cn/doc/promotion/ads_gongju17-0000001458756785)</strong>

洞察鲸鸿动能平台不同行业点击率、转化率高的图片创意、视频创意和落地页，通过标签分析、行业受众分析、热力分析等为优质创意制作提供参考。

<strong>[文案助手](https://developer.huawei.com/consumer/cn/doc/promotion/ads_wazs-0000001455760176)</strong>

维纳斯文案助手工具是鲸鸿动能平台为广告主免费提供的文案创作工具，系统根据行业和关键词，通过智能算法自动生成优质文案，帮助广告主提高文案创作效率。
