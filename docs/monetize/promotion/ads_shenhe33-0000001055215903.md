# 鲸鸿动能平台规范

为了保证广告主在鲸鸿动能平台上的展示内容，符合国家的法律法规及相关政策，为提供公开、明确的平台使用规范，保护广大用户合法权益，鲸鸿动能平台有权对广告予以审核，鲸鸿动能依法制定本规范，作为广告审核的依据，广告主应当遵循本规范，对于不符合规范的广告，不得发布。

针对本规范中未明确定义、模糊、未明确提及的内容，鲸鸿动能对该规范有最终解释权。本规范将根据国家法律、法规、以及行业环境的变化随时调整，以最新发布的版本为准。

## 素材要求

### （一）政治敏感

1. 不得使用或者变相使用国家机关、国家机关工作人员的名义或者形象。不得利用党和国家领导人的形象（含各种卡通虚拟形象特型演员），包括现任、离任或者已故的党和国家领导人的形象作商业促销宣传。
2. 不得出现与不良社会事件相关的政治人物。
3. 不得涉及时事政治及国家等敏感元素。
4. 不得使用破坏国家、民族形象的政史、党史、军史等内容。
5. 不得危害国家社会公共秩序，如：聚众扰乱公共场所、破坏历史文化遗产。
6. 不得宣传损害国家利益、危害国家安全、反对国家政策、反民主的内容，如：内涵国家政治体制、吹捧美、日等西方国家。
7. 不得侮辱、贬低英雄烈士。
8. 时政新闻类，不得发布党政新闻，如：地区冲突，国际关系等。

### （二）暴力恐怖血腥

1. 暴恐人物
   1. 不得涉及组织恐怖活动、极端主义活动，或与恐怖组织相关的人员。
2. 暴力恐怖场景
   1. 不得出现血腥、恐怖、暴力，命案、黑社会群架等易引起用户恐慌的画面图片。
   2. 不得展示危害动/植物生命安全的内容，包括但不限于捕杀国家保护动物、虐待虐杀动物、损害珍稀植物等。
   3. 不得出现夜间探险、探秘、灵异类的内容或其他带有惊悚和恐怖言语的行为。
   4. 不得出现动物互相伤害、吞噬、撕咬等血腥场景，如肢体撕裂、浴血、咬食等内容。
3. 暴力恐怖用语
   1. 不得出现恐怖、灵异鬼怪、惊悚的言语描述。
4. 冷热武器
   1. 不得出现售卖枪支、弹药及其他武器类，包括各类枪支、弹药及相关器材，如枪械、枪械仿制品（如仿真枪）、子（炮）弹、消音器、火药、说明书、包装盒等。

### （七）虚假宣传

1. 功效虚假
   1. 不得通过虚假描述、夸大承诺、误导性表述欺骗或诱导消费者
      - 不得对产品或服务的效果做夸大、保证性承诺的表述，如：承诺产品具体作用范围（包括但不限于：面积/长度/距离/承诺产品作用具体时间/有效期时长等）。
      - 不得涉及虚假承诺奖励金额描述，如邀请一人赚18元，实际并无该奖励。
      - 不得使用前后效果对比素材，包括但不限于：项目前后功效对比，痤疮前后对比图。
      - 涉及 “转基因”“非转基因” 相关描述，需符合相关要求：

        ①不得含有虚假的内容，不得欺骗和误导消费者（如：宣称 “花生油”“橄榄油”“葵花籽油” 等非转基因，利用消费者专业知识缺乏造成错误认知）。

        ②对于宣传 “非转基因” 的产品广告，不得使用 “更健康”“更安全” 等误导性词汇。

      |  |  |
      | 问题解析：效果虚假“年轻十岁” | 问题解析：质量虚假“耐用十年” |
   2. 不得混淆普通食品与保健品、药品的属性，也不得通过直接或间接方式对普通食品的功效进行虚假宣传
      - 不得借助宣传某些成分的作用明示或暗示该食品的治疗、保健、特殊营养等作用。
      - 不得明示或者暗示具有益智、增加抵抗力或者免疫力、保护肠道等功能性表述。
      - 不得出现与药品相混淆的用语或宣传含有新资源食品中的成分或者特殊营养成分。
      - 不得与其他食品、保健食品、药品等进行功效对比。
   3. 禁止投放伪科学资讯内容，包括不限于以下场景
      - 禁止使用无科学依据的营养、养生、护肤知识、生活窍门/建议，通过噱头、未经证实的知识吸引人关注

        ① 将普通食物或日常行为“神话”包装有医疗保健效果；如“血压稳如泰山，认准1块钱蔬菜，一吃安神、二吃利尿、三吃调血脂““西蓝花富含胆碱和叶酸，协同健脑，让你拥有年轻体态”。

        ②通过分裂正常的健康行为和饮食，来创造耸人听闻的“知识点”，制造虚假健康对立与夸大风险；如“早上喝水和晚上喝水对身体效果那可不一样”“晨起空腹喝水和饭后喝水，效果差别太大了！”“大蒜尽量少吃，最好不吃？看完才明白”。

        ③将复杂的健康问题简单归因于单一食物或行为，将其塑造为“万能钥匙”或“绝对禁忌”，使用绝对化指令且缺乏依据，来诱导受众群体类要不要吃某类食品、做某个动作；如“想要好身体，少吃鸡蛋/牛奶，多吃三果”。

        ④无科学依据的中老年养生知识类，来制造健康焦虑/渴望，提供不现实的解决方案，去诱导用户使用广告所推广的商品；如“人老了，真正需要的是XX”。
      - 禁止利用虚构事迹（尤其是高龄老人）推广产品或服务

        ① 通过虚构事迹（利用高龄老人来作为信任背书），制造健康焦虑/渴望，提供神化单一方案/食物，进行推广产品或服务；如“84岁悬崖跳水第一人，能做到这样的壮举全靠这一点”“105岁老人的长寿窍门，不靠顿顿补品，竟是这常见好物”等。
      - 禁止无科学依据的商品功效分析/建议/介绍

        ① 通过推荐常规健康认知中不被提倡（甚至被认为有健康风险）或无依据的饮食、行为和神化未知物品，来针对受众群体进行营销推广商品；如“年龄大了，建议多喝隔夜茶，现在知道也不晚”。
   4. 特殊品类限制
      - 乳制品：不得明示或者暗示可以替代母乳，不得使用母乳哺乳中的妇女和婴儿的形象。
      - 酒类：不得把个人、商业、社会、体育、性生活或者其他方面的成功归因于饮酒，不得含有不科学的明示或暗示（如：壮阳补肾、提高性生活、事业有成、成功人士的选择、重振雄风）。
2. 夸大绝对化描述
   1. 广告内容不得在商品性能、质量、使用效果及行业地位等方面使用夸大性、绝对化的描述，需确保文案客观真实：
      - 不得出现 “最高级”“国家级”“顶级”“极品”“第一品牌”“保证”“绝对”“一定”“至尊”“最受欢迎”“极致”“冠军” 等绝对化用语。
      - 不得使用夸大、虚假、无法证实的对价格及功能的不合理描述：①涉及折扣类，如：“0 利润”“击破底价”“击穿底价”；②涉及价格类，如：“市场最低价”。
   2. 不得使用带有命令、强迫意味的祈使句式，避免对消费者形成心理施压或不当引导，如：是男人必须买、一定要买。
   3. 学历教育限制：学历教育不支持“0”基础“描述。
   4. 预包装食品不得使用 “纯天然”“无添加” 等描述。
   5. 盲盒广告不得使用奖品名称+100%中奖、奖品名称+无空包等虚假承诺性描述。
   6. 流量管家类限制：非运营商/虚拟运营商类工具软件不具备网络资源与通信权限，不得宣传无法实现的功能或服务，如：分发/激活/赠送流量、增强信号、网络优化、修改网络配置等。

3. 价格/促销虚假
   1. 推广标准卡、普卡等长期可申信用卡，禁用 “快被抢光了”“限量限时”“最后一波” 等哄抢式文案。
   2. 售卖商品，其赠品价值超商品售价或与其接近。
   3. 不得有任何虚构原价、虚假折扣等价格欺诈以及虚假营销的行为；禁止在广告中出现 “原价”“内部价”“员工价” 等描述，可使用为 “吊牌价”“指导价。
4. 借用公信力宣传
   1. 禁止借科研、教育、机关、专业人士等权威主体的名义/形象背书，避免误导消费者：
      - 不得利用科研单位、学术机构、教育机构、行业协会、专业人士、受益者的名义或者形象作推荐、证明。
      - 不得明示或暗示和机关单位关系密切，如特殊渠道、绿色通道、官方入口。
      - 不得明示或暗示有相关考试机构或其工作人员、考试命题人员参与教育、培训（如：高考命题组专家参与培训、阅卷老师公开课）。
      - 不得使用真实的国家资格证书或学历证书、教学楼等标志性建筑作为素材暗示教学效果，包括证书内页展示、真实证书展示。
      - 不得出现医护形象对保险产品进行推荐宣传。
      - 不得出现“国家免检产品”等涉及质量免检的内容。

      不得以新闻报道名义，实质进行商品或服务广告宣传（混淆“新闻资讯”与“商业广告”属性，误导用户认为是客观报道）。

      不得含有特供、专供、国家机关专项等内容，注：客观地区专供可支持。
   3. 禁止借政策、内部消息等虚假内容进行宣传
      - 不得利用地方/国家政策或政策预测类内容做广告宣传。
      - 不得利用虚假政策或无法证实的虚假内容提醒误导用户，如：内部消息、政策关联。

5. 不实承诺

      - 不得在宣传中对学员招生录取使用含有降低招生标准或招生档次的明示或者暗示性描述，如：补录、走后门等。
      - 不得出现不符合考试制度及学制的宣传：

        ①本科类：“一年制学本科”“初高中也能拿本科”。

        ①不得使用“1年拿本科证书”“1个月拿资格证”等“具体时间+具体效果”描述。

        ②不得使用“毕业月薪过万”“培训前60分/培训后110分”等保证培训效果的表述。

6. 虚假保险陈述
   1. 不得误导、诱导营销：不得以展示不实、不全保费，炒作产品，非法提供保险合同约定外利益等方式诱导消费者投保，如：“首月0元”“首月低价” “免费保障”，“限时特惠首月立减\*\*元” “限售、限时、限量” “剩余X份”，“会员日补贴” “分享有礼”“红包补贴” “实物抽奖”等表述。
7. 虚假理财陈述

   2. 商业办公类房地产广告发布时，应当表现商业办公用途，不得出现“公寓”等带有居住功能的用语，不得使用“卧室”“厨房”等混淆房屋使用性质的画面或描述。
8. 虚假理财陈述
   1. 禁止对投资收益、效果作保证性承诺或暗示，避免误导用户忽视风险
      - 有投资回报预期的商品或服务广告，不得对未来效果、收益或相关情况作保证性承诺，不得明示或暗示保本、无风险、保收益。
      - 不对未来效果、收益或相关情况做保证性承诺，不得明示或暗示保本、无风险、保收益，不宣传 “帮您解套”“跟庄”“涨停”“稳定获利” 等内容。
      - 不得使用 “股票池/股票名单” 搭配暗示收益的修饰语（如：“精选股票池”“优质股票池”“核心股票池”“公布短线名单”），仅支持 “股票池”“股票池资讯/新闻/板块” 等中性表述。

   2. 股票、基金推广，禁止直接推荐或泄露明确信息
      - 不支持出现具体股票名称或股票代码（含用 \* 号模糊后仍可识别的样式，如 “万科 \*\*”）；仅支持“号代替股票代码/ 模糊股票信息”（如：股票代码 00\*\*2）。

### （九）诱导点击

1. 虚假按钮/提示
   1. 不得发布明显虚假欺骗性内容，如：“手机中毒”“版本需更新”“网络存在异常”等不实信息，误导用户。
   2. 不得使用违背用户意愿、功能无法实现的虚假设计，如设置虚假的“关闭”“X”“跳过”按钮，阻碍用户正常操作。
   3. 不得仿冒系统消息推送、仿系统操作功能等诱导用户点击广告，例如使用“登录领取”“应用菜单栏”“您还未开通XX”“请继续解锁”等易混淆为系统原生内容的表述。
   4. 不得通过“引导用户交互的描述+功能无法实现的按钮”组合形式诱导用户操作，包括但不限于设置诱导用户填写信息的虚假输入框。
   5. 不得展示虚假数据，如非系统统计的客户下单跑马灯、虚构的购买人数、商品热销量、库存余量等，需确保数据真实可追溯。

2. 模仿类诱导

   广告内容需具备清晰可识别性，确保消费者能明确辨别其广告属性；同时不得包含虚假、引人误解的内容，也不得通过设计或文案欺骗、误导、诱导用户点击，具体禁止情形如下：

   1. 禁止使用恶意诱导点击的设计元素，例如模拟手机碎裂、屏幕脏污等易误导用户产生设备异常认知的视觉效果。
   2. 禁止使用通知类描述模拟官方消息，如：重大通知”等表述伪装官方推送，混淆广告与官方信息边界。
   3. 广告素材主视觉不得仿冒第三方知名支付软件设计，也不得采用仿官方新闻的样式，避免误导用户认为是正规平台内容。
   4. 禁止使用单一弹窗样式、“按钮+鼠标手”样式作为素材设计，此类设计易诱导用户误触。
   5. 不得通过视觉或文案设计干扰用户感官，包括但不限于模拟手机系统故障、恶意软件感染的呈现方式，以及通过图案或文案冗余堆砌造成的视觉干扰。
   6. 素材主视觉不得使用按钮类设计，包含上下条按钮、单双圈按钮、矩形按钮等，避免以功能按钮假象诱导点击。
   7. 广告文案不得模仿官方传媒消息风格，例如模拟警方新闻等官方发布的内容形式，防止用户混淆信息来源。

### （十一）一致性要求

1. 行业及内容一致
   1. 推广内容需与开户主体、行业保持一致，如：“开户为A行业，素材内容必须是A行业；不得出现B行业。
   2. 文案-素材-落地页需保持一致，不得出现 “文案与素材不一致”“外层素材内容在落地页无体现”的情况。
   3. 视频内容一致性：广告视频的语音需与字幕保持一致；若使用非汉语语音，需添加中文字幕。
2. 推广目的一致性
   1. 推广目标为应用下载，落地页不得为网页推广，且不得二跳。
   2. 推广目标非应用下载类，落地页不得进行应用下载推广。
   3. 当推广产品为”快应用“或”网页“，落地页推广功能需一致，不得通过网页拉端到快应用。
   4. 外层引导文案需与落地页内容及功能相关联，如落地页推广促活，外层引导文案不得描述为下载。
3. 电商行业
   1. 价格一致性：

      ①使用单品素材投放时，外层素材价格需与落地页一致。

      ②使用单品素材投放广告，如外层素材商品在落地页无直接体现，且落地页为产品聚合页，在无法验证外层产品价格与实际售卖价格是否一致的情况下，外层素材请勿出现特定价格描述，请使用：xx元起、低至XX元或不出现具体价格。

   活动利益点匹配：外层素材涉及的活动奖品、奖金等利益点，需在落地页有相符信息，不得出现不切实际的奖励描述。
4. 阅读短剧
   1. 小说广告创意不得误导用户，禁止将小说内容伪装成新闻资讯、娱乐八卦等非小说类内容。
5. 金融行业
   1. 内容与条款一致：互联网保险营销宣传内容（如保障范围、理赔条件、保费金额）需与保险合同条款保持一致。
6. 房地产行业
   1. 楼盘名称标注：若预售证名称与广告推广楼盘名称不一致，需在广告素材中同时注明【注册名称】和【推广名称】。
7. 盲盒
   1. 推广盲盒活动，外层素材中展示的奖品需与奖品池中的商品保持一致，包含品牌、规格、型号等。
8. 旅游资讯
   1. 目的地/景点素材匹配：广告文案、素材中使用目的地/景点/城市（尤其地标性建筑）描述时，需与配图严格匹配（如描述“日本东京”，配图不得为其他城市地标）。

### （十四）资质要求

1. 行业资质要求
   1. 教育行业
      - 教育机构不得在未提供金融资质的前提下，使用 “学费零首付”“零利息”“学费分期” 等金融手段招生，若与银行合作贷款，需提供合作协议与银行《金融许可证》。
      - 涉及学费分期等内容，需提供有权从事相关业务的资质，或与银行的分期业务合作凭证。
   2. 金融行业
      - 担保服务：无担保资质不得在广告中宣传增信服务（如提供担保、信用保险）及兜底承诺等变相增信服务。
      - 贷超平台：涉及具体贷款产品时，

        ①需上传与该贷款产品的合作协议及产品合法放贷资质；

        ②贷款产品需符合《小额贷款审核规则》。
   3. 证券投顾/诊股：

      ①诊股广告涉及股市即时行情信息，需上传《证券交易所批准发布的证明》；

      ②教育广告涉及证券投资类（投顾/炒股）相关内容，需提供《 经营证券期货业务许可证》或《证券、期货投资咨询业务许可证》，二选一上传。
   4. 汽车行业
      - 广告中出现具体机构贷款服务，需补充与金融机构的合作协议，并在广告中说明相关金融业务和活动介绍。
   5. 房地产行业
      - 涉及具体预售楼盘需提供《预售许可证》，涉及现房需提供《销售许可证》。
      - 若许可证资质与广告推广楼盘项目不一致，需补充广告推广中对应楼盘项目的资质。
   6. 短剧行业
      - 非短剧行业投放短剧：必传《计算机软件著作权登记证书》、《增值电信业务经营许可证》、《网络文化经营许可证》；选传《信息网络传播视听节目许可证》或《信息网络传播视听节目备案》。
      - 所有短剧推广：广告内容涉及短剧推广，需提交该剧的《网络剧片发行许可证》或节目登记备案资质。
   7. 医疗健康行业
      - 在线问诊平台：

        ① 在线问诊属医疗广告且涉及具体药品信息，需提交广告审查表；

        ②问诊形式为义诊，需向机构所在地上级卫健委申请报批，并提交报批证明。
      - 医药交易平台：

        ①推广素材不涉及具体药品、医疗器械、保健食品且仅品牌推广，可不提供广告审查表；

        ②推广多商品/多品类素材，且不涉及功效描述，可不提供广审表。
      - 医疗健康类商品（医疗、药品、保健食品等）：

      ① 欲发布医疗广告必须事先取得广审。

      ② 保健食品、药品、特殊医学用途配方食品、医疗器械广告，若不涉及商品信息、功效等需广审的内容（仅集团品宣或人文宣传），可不要求广审表，但不得使用代言人形象。

      ③ 不得对已有广审结果的文案、图片、视频、落地页进行修改，不得将广审文案添加到已通过广审的落地页，已审查通过的广告内容需改动时，应当重新申请广告审查。
2. 广告资质要求（含荣誉/数据真实性证明等）
   1. 电商行业
      - 特殊材质商品：广告明确描述 “真皮、头层牛皮、桑蚕丝、100% 桑蚕丝” 等特殊材质时，需提供含对应材质证明的质检报告。

      - 专利商品：广告使用专利产品时，需提供真实有效的专利证书，并在素材中标明专利号、专利种类；禁止使用未授权、已终止/撤销/无效的专利作广告。
      - 带特殊标志商品：广告涉及“国家地理协会标志”时，需提供真实有效的资质证明。
      - 儿童相关商品：不得将“小金盾”标志与“国家审批、质量认证” 等宣传用语挂钩，避免混淆产品监管审批或质量认证状态。

      - 市政配套描述：广告涉及楼盘周边交通、公园、医院等市政设施时，需提供以该楼盘为参照地点的地图截图（需为知名地图软件），证明设施真实性。

      - 师资介绍：广告内容出现教师师资介绍（尤其学科类），若涉及具体教师，需提交该教师的教师资格证书。
   4. 金融行业
      - 讲师资质：推广内容中出现金融/理财讲师介绍时，需提供对应讲师的金融行业从业资格证，如：《证券分析师资格证》、《证券从业资格证》。
   5. 网赚类
      - 金额宣传：网赚类 APP广告涉及金额价格超 100 元时，需提供 3 份用户获得实物的物流截图证明，验证收益真实性。
   6. 其他
   - 奖项/荣誉宣传：广告展示奖项、评比结果、荣誉称号时，需提交对应官方书面证书或官网截图。
   - 排名/比较宣传：涉及 “全网排名第一” 等具体排名、比较和数量类描述时，需提供相关证明资质。
   - 引证内容规范：广告使用数据、统计资料、调查结果、文摘、引用语等引证内容时，需真实准确、标明出处；若有适用范围和有效期限，需明确表示。
   - 个人荣誉/机构资质：推广内容涉及 “协会头衔、个人专业技术职称、获奖荣誉” 等时，需提交对应资质证明。
3. 知产相关资质
   1. 存储介质推广：广告涉及存储介质时，禁止出现音乐、电影、视频、MV等可能涉嫌版权侵权的宣传内容。
   2. 版权作品使用：广告中使用注册商标、版权作品（如：绘画、影视剧、音乐、影视、游戏、动漫等版权内容），需提交权利人（公司/个人）的合法授权或合作协议。
   3. 知名IP使用：
      - 使用知名人物、动漫形象宣传（含文字描述、图片、明星同款），需提供名人/经纪公司的肖像授权书、代言合同，或动漫权利人的授权/合作协议。
      - 使用第三方知名游戏的标志性人物、logo、元素、场景，需事先取得授权证明。
      - 涉及知名影视剧剧照、综艺节目，需补充授权证明。
   4. 教育行业
      - 大学/学院字样使用：①广告涉及第三方院校名称或使用 “大学”“学院” 做宣传，需补充校方授权；②未获国家认可的高等教育机构，不得使用 “大学”“学院” 名义。
      - 院校标志性建筑使用：经营性培训机构不得直接或间接使用第三方院校的学校大门、教学楼等标志性建筑；若出现带院校名称的大门或知名院校标志性建筑，需校方授权后方可使用。
   5. 金融行业
      - 金融机构名义禁用：不得使用 “中国人民银行”“央行” 字眼及名义宣传。
      - 第三方金融工具授权：①信用卡管家类软件或其他第三方金融工具，提及 “XX 银行信用信息查询” 时，必须有 XX 银行的授权或合作证明；②非银行官方机构使用金融卡类素材，未获品牌授权的，不得出现 “银联”“万事达”“VISA” 等信用卡组织Logo。
      - 未经授权禁止使用：未经金融机构或其他金融产品授权，不得出现其品牌信息。
   6. 医疗行业
      - 合作医疗机构宣传：广告仅可体现合作医疗机构名称，但需同时提供与该医院的合作协议，以及合作医院的《医疗许可证》。
   7. 工具行业
      - 推广地图导航类APP，涉及xx卫星导航系统相关的宣传内容，需提交相关官方授权。
   8. 第三方品牌误导
      - 广告文案不得使用让人误以为是第三方品牌官方行为的文案（如非华为官方，不得使用 “华为推荐/华为投资/华为合作企业” 等描述，仅用户权益类描述如“XX 游戏华为用户首批体验” 可过审）。

### （十五）素材质量要求

1. 不得出现内容显示异常。
2. 视频声音须完整且无杂音，不得出现体验不佳的突兀声音，如：尖叫声、爆破音。
3. 视频中不得出现静帧、卡顿、黑屏、无声音等。
4. 广告内容不得出现非完整页面/页面结构凌乱/留白过多等明显样式问题。

### （十六）风格调性要求

### （十七）落地页交互合规

1. 下载要求
   1. 落地页下载功能需由鲸鸿动能提供，仅支持合规下载格式，不得涉及.doc、.exe等非指定下载格式。
   2. 禁止使用静默下载模式，需确保用户明确感知下载操作。
   3. 落地页仅支持通过按钮触发下载，仅点击下载按钮区域可启动下载，其他区域点击不得触发下载功能。
   4. 使用维纳斯落地页，下载按钮需符合尺寸与数量要求：落地页仅允许添加1个下载按钮，按钮高度不得超过40像素，按钮位置可自主选择。
   5. 下载按钮提示语需满足以下规范：
   - 提示语必须包含“下载安装”字样，可使用“下载安装”“点击下载安装”“点击按钮下载安装”等表述。
   - 提示语字体需清晰可辨，不得出现字号过小、与背景色混淆等导致用户无法识别的情况。
2. 广告返回链路要求
   1. 广告落地页支持使用弹窗，但弹窗需具备可关闭功能，且不得出现反复弹出的情况，避免干扰用户操作。
   2. 开屏版位推广目的为“应用促活”的广告，其应用直达链接需添加“一键返回”能力：点击返回按钮后可跳转回华为媒体页面，且返回按钮需清晰标注“返回”字样，确保用户识别与操作。
   3. 调起广告落地页后，禁止出现“广告套广告”场景，例如：落地页开屏后叠加APP开屏，（如：5秒广告+5秒广告）、弹窗彩蛋后再弹出APP自运营弹窗等多层广告叠加形式。
   4. 用户调起应用后，需保证应用可正常退出并返回原浏览页面，回退路径不得超过2次返回键。
3. 链接格式要求
   1. 落地页不允许出现“死链”，死链即点击后无法抵达的页面，包含：
      - 协议死链（如404、403、503状态）
      - 内容死链（内容不存在、已删除、空白页等）
      - 域名失效（因网页安全被屏蔽等）
      - 无法正常自动加载的外部链接（如加载在中国大陆地区不可访问的境外地址\*.goolge.com,\*.facebook.com 等）
      - url链接需要通过鲸鸿动能安全测试，不得包含恶意操作和擅自修改跳转页面内容（恶意操作或者擅自更改落地页将可能被终止投放或封禁账号）
   2. 应用直达链接仅可使用deeplink地址，不支持使用intent地址。
4. 个人隐私协议相关要求
   1. 落地页涉及个人数据收集，须说明获取信息的原因和用途，并添加隐私协议、个人信息保护相关协议等内容，且不得默认勾选同意，需由用户自主勾选确认。

5. 落地页交互要求
   1. 落地页添加客服好友、引导关注公众号/服务号等引流行为需告知用户获取许可，不得直接调用。
   2. 落地页不得使用的虚假链接，如涉及二跳跳转，需在页面上有明确的跳转引导提示，使用引导按钮跳转时需注意，跳转热区应限定在按钮范围。
   3. 针对https的落地页内可添加不超过3个以上的“浮窗”。
6. 耗费流量行为
   1. 落地页打开需静默无声，不得有自动播放的音频、视频或其他未经消费者同意严重消耗其网络流量的内容。
   2. 网络最佳环境下，落地页加载时间，即用户从点击到完全打开落地页所需的时长不得超过5秒；落地页内所含的所有其他可跳转的页面，加载时间均不超过5秒。
7. 应用或快应用交互要求
   1. 未经用户许可，不得默认勾选创建桌面快捷图标，强制加桌。
   2. 未经用户许可，不得静默/捆绑下载安装其他APP。
   3. 快应用内的广告视作广告内容的一部分，请确保一跳页面所展示的广告内容符合鲸鸿动能平台要求。
   4. 应用打开一跳展示内容需与外层内容/服务保持一致。
   5. 应用使用中不得通过频繁弹出悬浮窗广告，或强制用户点击、浏览广告等方式，中断用户操作或对正常使用造成不当干扰。
