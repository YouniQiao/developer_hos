# 2025年8月

<strong>【规则优化】通用-关于推广内容一致性的要求</strong>

1. 广告使用单品素材投放广告，如外层素材商品在落地页无直接体现，且落地页为产品聚合页，在无法验证外层产品价格与实际售卖价格是否一致的情况下，外层素材请勿出现特定价格描述，请使用：xx元起、低至XX元或不出现具体价格。

|  |  |
| --- | --- |
| ❌具体价格、折扣+聚合页无体现 | ✅仅供参考 |
|  |  |

<strong>【规则优化】通用-网赚类</strong>

1. 广告素材需添加金额描述或警示语。如“具体金额以实际活动规则为准”、“实物兑换以实际活动规则为准“, 或说明金额最高上限，如：最高100元。

|  |
| --- |
| ✅仅供参考 |
|  |

<strong>【规则新增】通用-推广内容需明确，不支持模糊性描述</strong>

1.广告标题文案描述需完整，不得单独使用地域名、品牌名、模仿自然人称谓等不完整描述。

2.品牌来源要求：可使用品牌名称（如： app名称、商标名称）或企业主体简称。

3.推广产品要求：广告素材需体现行业或产品特性，确保明确表达出所推广的内容。

|  |  |  |
| --- | --- | --- |
| ❌单独使用自然人称谓 | ❌未使用品牌名称或企业主体简称 | ❌素材无文案，未明确表达推广内容 |
|  |  |  |

<strong>【规则优化】通用-文案要求</strong>

1.在中国大陆境内发布的广告，文案不得使用繁体字、方言、少数民族文字、汉语拼音等不规范汉字，如使用外国文字，需与中文翻译一起使用。

|  |  |  |
| --- | --- | --- |
| ❌纯藏语内容 | ❌纯英文内容 | ❌繁体字 |
|  |  |  |

<strong>【规则调整】平台禁投-小说题材、高价盲盒</strong>

1.近现代战争、兵王类小说题材，支持全量投放。

2.售卖高价盲盒产品（价值＞200），转为禁投管控。

|  |  |
| --- | --- |
| ✅兵王类小说 | ❌高价盲盒（价值&gt;200） |
|  |  |

<strong>【规则新增】平台禁投“军队文职培训”</strong>

1.禁止推广军队文职培训相关内容。

|  |
| --- |
| ❌不支持投放“军队文职培训” |
