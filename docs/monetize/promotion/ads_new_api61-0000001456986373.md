# 查询操作日志列表

【简介】通过此接口可以查询操作日志信息。默认查询时间范围为30天，如果超过30天请分段查询。定向部分日志变更需要通过【操作日志中定向包信息转换接口】进行解析

<strong>请求地址</strong>

https://ads.cloud.huawei.com/ads/v1/tools/operation\_log/query

<strong>请求方法</strong>

<strong>GET</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| page | int | 是 | 页数 取值范围1-10000 |
| page\_size | int | 是 | 每页数量 取值范围1-50 |
| start\_date | string | 是 | 开始时间(yyyy-MM-dd) |
| end\_date | string | 是 | 结束时间(yyyy-MM-dd)  时间范围最大30天 |
| operation\_type | string | 否 | [【操作类型】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section478931125811) |
| operator\_type | string | 否 | [【操作者类型】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section1692524785813) |
| operation\_target\_type | string | 否 | [【操作对象类型】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section62876581580)  用于选择类型，根据此字段可以选填下方参数，CAMPAIGN，ADGROUP，CREATIVE，TARGETING\_TEMPLATE，ACCOUNT仅可传入一个 |
| campaign\_ids | string[] | 否 | 计划ID列表【operation\_target\_type为CAMPAIGN时，可使用该字段进行筛选】 |
| adgroup\_ids | string[] | 否 | 任务ID列表【operation\_target\_type 为ADGROUP时，可使用该字段进行筛选】 |
| creative\_ids | string[] | 否 | 创意ID列表【operation\_target\_type 为CREATIVE时，可使用该字段进行筛选】 |
| targeting\_ids | string[] | 否 | 定向包ID列表【operation\_target\_type 为TARGETING\_TEMPLATE时，可使用该字段进行筛选】 |

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1 | 操作日志响应结果 |

Struct1定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| total | int | 总数 |
| data | Struct2[] | 操作日志列表 |

Struct2定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| operation\_id | string | 操作Id |
| operation\_time | string | 操作时间(yyyy-MM-dd HH:mm:ss) |
| operation\_type | string | 操作类型 |
| operator\_type | string | 操作者类型 |
| campaign\_id | string | 计划id |
| adgroup\_id | long | 任务id |
| creative\_id | long | 创意id |
| targeting\_id | string | 定向包id |
| operation\_target | string | 操作对象 |
| operation\_target\_type | string | 操作对象类型 |
| details | Struct3[] | 变更详细信息列表，包含定向包的变更详情 |

Struct3定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| change\_item | string | 日志记录修改  类型详见[【操作日志修改项说明】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_api721-0000001641010849#section107891895811) |
| before | string | 修改之前 |
| after | string | 修改之后 |
