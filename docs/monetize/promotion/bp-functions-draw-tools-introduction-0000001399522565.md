# 功能介绍

制图工具作为开发者便捷高效的创意助手，旨在帮助开发者激发创意灵感、简化创意素材产出流程、提升创意素材制作效率。 “制图工具”涵盖“模板制图”、“图片自适应衍生”两种功能。

- 模板制图：开发者可在投放前、投放中使用“模板制图”功能。平台针对各行业特点提供丰富多样的行业模板，开发者可选择合适的模板，参考其风格、排版、色彩等，进行简易编辑和高级编辑，快速生成符合推广诉求的创意素材。该功能在提供创意方向、提升创意产能的基础上，进而提升投放效果。
- 图片自适应衍生：开发者在投放时，可通过此功能将素材智能剪裁、衍生为横版大图、焦点图等素材规格。平台通过自适应算法，将图片进行等比例缩放或高斯模糊，保留图片主要内容和清晰度，批量衍生为适合投放的多种规格尺寸，支持下载至本地和保存至素材库。创建任务时，可通过上传图片或从素材库选择并使用。
