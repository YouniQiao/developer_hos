# 问题解答FAQ

<strong>【问题一】</strong>提前多久提交活动审核？

您至少提前5个工作日（新游首发请提前3个工作日）提交奖品、活动、公告审核（涉及审批、驳回、修改等），避免活动开始时电子流仍为【待审核】状态，导致活动无法按时上线。

<strong>【问题二】</strong>运营活动参与时间可以设置多长时间？

活动时间最长不超过7天，应用类活动开始时间请统一为10:00点。

<strong>【问题三】</strong>首发新游可以申请哪些类型的活动？

除了安装类和首登类活动不支持申请，其他类型均支持。因为申请安装和首登活动后，内测、预约、预下载用户无法在首发活动期获得奖品，引发用户投诉。

<strong>【问题四】</strong>活动状态/审核意见等如何查询？

活动创建成功后，您可以通过：华为开发者联盟&gt;管理中心&gt;应用服务&gt;联运服务&gt;活动管理&gt;进入[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)&gt;我的应用&gt;应用类型&gt;运营&gt;活动管理界面查询活动状态、审核意见等。如有疑问，可联系运营人员。

<strong>【问题五】</strong>礼包/第三方卡券兑换码上传格式？

- 文件格式必须是ANSI编码的txt格式。
- 文件中每行一条数据排列，请勿添加其他注释。
- 激活码/兑换码只能由英文大小写字母和数字组成。
- 文件中不能存在重复码。

<strong>【问题六】</strong>如何设置礼包奖品数量（单位：份）？

为保障用户体验良好，礼包数量需覆盖活动期间所有参与用户，如登录类活动请按日活人数\*活动天数\*3倍预估，建议根据活跃度配置10万~100万（库存礼包到期将自动作废）。若活动过程中礼包数量不足将引起用户投诉。

<strong>【问题七】</strong>如何设置优惠券奖品数量（单位：份）？

为保障用户体验良好，优惠券数量建议配置为“不限量”，避免活动过程中奖品数量不足引起用户投诉。最终结算以实际消耗为准。

<strong>【问题八】</strong>同期最多支持申请几个活动？

为合理分配资源，同一应用/游戏最高可同时开展3个不同类型的开放活动。如安装送券/礼包（1个）、登录送券/礼包（1个）、充值送券/礼包（1个）。

<strong>【问题九】</strong>重复申请同类型活动，是否会被驳回？

为合理分配资源，同一个应用/游戏，当期活动结束后30天之内不得重复申请完全相同的活动。如：登录类活动若申请9月1日-9月3日、9月26日-9月28日2个时间段，则26-28日的活动将被驳回。

<strong>【问题十】</strong>设置华为优惠券作为活动奖品时，成本由谁承担？

优惠券成本由您承担，以有效期内（X天）优惠券实际消耗金额为准，由系统自动在当月收入中扣减。举例说明如下：

- 活动规则

9月15日-9月20日，安装本应用/游戏即可领取10元专属优惠券，每人限获得1次奖励。

- 活动成本

假设参与活动人数1万人，则优惠券发放金额：1万人\*10元=10万元。

假设有效期内优惠券使用率90%，则优惠券使用金额：10万元\*90%=9万元。

- 结算方式

活动发放的优惠券部分照常参与分成，假设贵司当月流水100万，其中现金流水80万，优惠券流水20万（含活动的9万），双方五五分成（若为联运应用类则按三七分成），则贵司当月应收100万元\*50%=50万元，系统扣减活动成本后最终收入：50万元-9万元=41万元。

<strong>【问题十一】</strong>审核中的活动和奖品可以进行信息的修改吗？

需要先撤销审核中的活动和奖品，编辑修改后重新提交审核。

具体路径：[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html) &gt;我的应用&gt;应用类型&gt;运营&gt;活动管理&gt;活动管理&gt;撤销中查看。

<strong>【问题十二】</strong>活动结束后奖品数据怎么查看？

- 奖品流水数据可以在[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html) &gt;我的应用&gt;应用类型&gt;运营&gt;活动管理&gt;奖品流水中查看；
- 已结束的活动数据可以在[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html) &gt;我的应用&gt;应用类型&gt;运营&gt;活动管理&gt;活动数据中查看。

<strong>【问题十三】</strong>优惠券活动奖品要求？

您为已首发超过1个月的游戏申请安装/连登1天/首登类运营活动（限游戏类运营活动）时，活动奖品中的优惠券满减比例超过优惠券面额40%或者无门槛券的，不予通过。

<strong>［问题十四］活动结算模式收入抵减模式仍可使用吗？</strong>

仍可使用。

<strong>［问题十五］充值预购模式团队成员都可修改权限吗？</strong>

仅账户持有者可修改权限。

<strong>［问题十六］充值预购模式如何添加新的团队成员</strong>

点击“添加成员账号”，按要求填写手机、邮箱等信息。
