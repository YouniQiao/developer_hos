# 2026年AG小游戏行业激励政策

## <strong>政策周期</strong>

2026年12月31日。

## <strong>面向对象</strong>

游戏行业客户（仅限标准联运游戏参加）。

## <strong>参与条件</strong>

无需报名，默认符合规则的产品均可参与。

## <strong>激励内容</strong>

### <strong>IAP-小游戏</strong>

1. 通过AppGallery买量带来的小游戏新用户，对该用户在该小游戏内产生的内购变现流水30%作为推广激励金返货给开发者，该用户在该游戏在后续365天内产生的流水均按该激励执行，365天之后激励失效。
2. 该政策仅适用于HarmonyOS 4.x及以下版本，通过该政策获取的返货也仅能使用于HarmonyOS 4.x及以下版本的投放。

## <strong>激励发放形式和使用</strong>

|  |  |  |
| --- | --- | --- |
| <strong>投放</strong> | <strong>发放周期</strong> | <strong>有效期</strong> |
| 激励金充值 | 月度 | 180天 |

## <strong>结算方式</strong>

1. 有内购变现的属于IAP小游戏。
2. 结算周期以月度为单位，激励金自下发之日起有效期180天。
3. 账单和激励金额数据以AppGallery数据为准。
4. 如发现不符合激励条件的情况，AppGallery将对数据进行复核，并仅在符合激励条件的范围内发放激励，金额以复核调整后的为准。次月发放后，如果客户有异议，可以通过发邮件向AppGallery反馈问题，AppGallery进行数据复核。
5. 如涉及数据差异的，数据差异在AppGallery数据的10%以内时，以AppGallery数据为准；如差异超过10%，请于五个工作日内发正式邮件反馈数据差异排查，双方友好协商一致确认最终数据；如未及时反馈，视为确认以AppGallery数据为准。

## <strong>补充说明</strong>

1. 激励金会充值到获得游戏所在的开发者推广账户中，账户中所有游戏都可以使用这笔金额。当账户中含激励金时，账户下所有游戏的推广费用都优先消耗激励金。
2. 仅适用于中国大陆注册并推广的开发者报名参加。
3. 激励金不支持退款申请，请尽快使用，以免过期。
4. 适用政策期间，推广产品及相关推广投放应保持合法合规运营，不得出现违反法律法规或侵犯第三方合法权益的情形，否则AppGallery有权取消推广产品的政策适用资格并采取相关处罚措施。
5. <strong>异常作弊、违规操作、恶意修改预算/出价、利用系统漏洞、滥用系统机制、可疑或非法交易等不当行为不可享受激励。经AppGallery合理判定服务商客户存在前述行为的，AppGallery有权拒绝发放激励并追回已发放的激励，并采取相应处罚措施，由此造成的损失或赔偿责任由服务商客户承担。</strong>
6. <strong>如因产品功能、计算错误、系统错误等任何原因导致激励超额发放、重复发放或错误发放，一经发现，AppGallery有权追回多发部分或按核实后的应发激励调整发放。</strong>
7. <strong>对于应追回的激励金额，AppGallery可按以下方式扣除或追缴：</strong>

   <strong>（1）从下期激励中扣除；</strong>

   <strong>（2）从服务商或客户激励账户中直接扣除；</strong>

   <strong>（3）从服务商或客户的充值账户等其他支付、结算账户中扣除；</strong>

   <strong>（4）从AppGallery尚未向服务商或客户支付的各种激励、费用、分成、代收款等款项中直接扣除；</strong>

   <strong>（5）从服务商或客户在华为平台的其他激励、账户或待结算款项中扣除；</strong>

   <strong>（6）采取其他可行的方式要求服务商或客户在30个工作日内退回、支付或赔偿。</strong>
8. 如对本政策有任何疑问，请联系AppGallery。
