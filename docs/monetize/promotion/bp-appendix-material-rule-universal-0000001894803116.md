# 通用合规要求

## 授权及证明资质规范

1. <strong>品牌，商标、版权作品</strong>

   广告中使用的注册商标、版权作品（如照片、绘画、影视剧等）等内容的，广告主须提交前述内容权利人（公司或个人）的商标合法授权或合作协议。
2. <strong>知名人物、动漫形象、第三方游戏、影视剧、综艺</strong>
   - 广告中使用知名人物、知名动漫的名义或形象进行宣传，包括但不限于文字描述、图片展示、明星同款等，须提供名人本人或其经纪公司与品牌方签订的肖像授权书或代言合同，或者提供动漫权利人（公司或个人）的合法授权或合作协议；
   - 广告中使用第三方知名游戏标志性人物、logo、元素、场景等，应当事先取得其授权证明；
   - 广告中涉及知名影视剧剧照、综艺节目等，应补充其授权证明。
3. <strong>素人模特</strong>
   - 广告中出现素人人物肖像，需提供个人肖像承诺函或华为承诺函。
   - 广告素材使用AI人像需提供免责函。
4. <strong>奖项、荣誉称号</strong>

   广告中展示获得奖项、评比结果、荣誉称号等信息的，广告主须提交相应的官方书面证书或官网截图。
5. <strong>第三方合作</strong>

   广告中若声称商品/服务是特定媒体、会议、活动的专用、指定、推荐、合作商品/服务等的，广告主须提供其与相应媒体、会议主办方、活动主办方签订的合作协议。
6. <strong>使用数据、统计资料、调查结果</strong>

   广告中若出现引用具体销售数据（如月销量X万件、累计销售X件）、统计资料（为X万用户提供过服务）、调查结果（如市场占有率、实验数据等），广告主须提供相应的第三方报告；如果相关数据为广告主内部数据，广告主提供数据证明/系统后台截图或华为承诺函。
7. <strong>进出口证明</strong>
   - 广告中涉及“法国“、“意大利”、“欧美”等产地，需补充提供进出口检验检疫证明或者海关报关单；
   - 广告中若涉及出口，需出示出口企业资质，如：海外对外贸易经营备案登记。
8. <strong>专利表述</strong>

   广告中使用专利产品或专利方法，应当标明专利号和专利种类，未取得专利权的，不得在广告中谎称取得专利权，禁止使用未授予专利权的专利申请和已经终止、撤销、无效的专利作广告。
9. <strong>国家地理协会标志</strong>

   国家地理协会标志广告中如涉及国家地理协会标志，需提供真实有效的资质证明。

   ![](./img/488a3684e1f0.png) 

   广告授权及证明资质应满足以下要求：

   - 资质的登记主体：应与系统中录入的、和华为签订的相关协议及其它相关文件中的出现的广告主的名称一致。
   - 资质的有效期限：应涵盖广告的投放期限及产品、服务的提供期限。
   - 资质的登记事项：应与要证明的事项一致。
   - 授权人名称：应与广告主提交的反映原始生产商、提供商等原始所有者相关资质或权利证明中的主体名称一致。
   - 被授权人名称：应与广告主的名称一致。
   - 授权标的：广告所涉及的商品或服务应与授权的商品或服务一致；广告所涉及的商品或服务的品牌的商标应与授权的商品或服务的品牌的商标、授权人有权的商标权一致。
   - 授权期限：应涵盖广告的投放期限及产品、服务的提供期限。
   - 授权地域：应涵盖广告所提及的产品、服务的销售区域。
   - 授权事项：应包含广告主可以自行销售被授权人的产品、服务等事项

## 华为品牌相关规范

1. 电子产品竞品：广告素材禁止使用非华为品牌的手机、平板、电脑、电视、智能手表、耳机、智能音响、智能眼镜等。出现电子产品的，需使用华为产品。

   竞品手机说明：

   - 广告卡片（外显图片标题视频）不得出现任何明显竞品品牌名称、logo，或以明显竞品品牌旗下产品作为素材内容主体（包括竞品产品突出元素，如“灵动岛”、“home键”、“摄像头”“按键”、“状态栏”等
   - 竞品品牌主要包括：VIVO、OPPO、一加、realme、小米、魅族、中兴、联想、苹果、三星、荣耀等。
2. 未经我司许可不能以任何形式直接或间接地使用“HUAWEI”、“华为”品牌名义进行联合活动宣传；广告不得使用让人误以为是华为官方行为的文案，如：华为推荐/华为用户专享/华为投资/华为合作企业/华为金融生态企业打造XX/恭喜华为用户获得牛股微信号等。但是关于用户权益相关的描述，比如：XX游戏华为用户首批体验，可以过审。
3. 内部流量暂不支持推广华为产品低价抢购活动，日常售卖价格不得低华为商城、华为京东旗舰店、华为天猫旗舰店三者的最低价格；
4. 二手交易平台投放买卖华为系列产品，不支持出现具体买卖价格，建议描述为高价回收/低价购买等。
5. 汽车竞品：三方平台使用汽车做素材设计时，需使用鸿蒙智行合作的车型。

## 其他合规要求

### 合规要求（文案、图片、视频、落地页等）

1. 广告文案支持使用网络用语，但不得使用不符合逻辑、不通顺、不完整的语句、句式、词组；不得使用容易引起歧义/导向不良/粗俗的文案描述。
2. 广告文案中的标点符号需符合国家《标点符号用法》，相同符号的连用不得超过3个，如“！！！”；文案中不得使用特殊符号,如表情符号“：）”“-。-”、emoji表情等，若使用的特殊 符号为大众日常经常使用且易让人理解、不存在歧义，则可支持使用；图片中书名号《》 只能用于书本、游戏、影视作品推广。
3. 文案不得含有言语声明、诽谤、谩骂、挑衅、威胁、命令、贬低他人性质的内容，禁止出现妨害社会良好风尚的内容。
4. 文案需积极向上，不得出现有价值观和负面情绪不正确引导的文案。
5. 广告不得使用时事政治、负面社会问题等全民热点内容作为广告宣传点为自身利益做广告引导。
6. 如无授权，素材中不得包含央视相关内容（如央视、中央电视台、cctv等）的推荐或荐、认证含义（如央视上榜品牌、cctv推荐、央视明星品牌等）及央视重点栏目的名称（如：新闻联播、焦点访谈等），但地理位置性类描述除外，例如位于央视大楼东侧。
7. 不得含有虚假广告内容，广告有下列情形之一的，为虚假广告：
   - 不得对商品的性能、功能、产地、用途、质量、成分、价格、生产者、有效期限、允诺等，或者对服务的内容、提供者、形式、质量、价格、允诺等做虚假宣传；
   - 不得使用夸大、虚假、无法证实的价格描述；不得有任何虚构原价、虚假折扣等价格欺诈以及虚假营销的行为；禁止在广告中出现“原价”、”内部价”“员工价”等描述，可使用为“吊牌价”“指导价”；
   - 禁止以新闻报道名义，实质进行商品或服务广告宣传；
   - 对商品/服务的描述应当有合理、有效的证明资料支持，不可捏造或夸大内容，不可使用不符合事实认知或逻辑的描述；
   - 不得投放与市场价格差异过大，价格超低或明显造假的商品，如：非促销期间，知名品牌商品价格远低于正品的市场价，华为手机5折、LV1000元等；
   - 不得对产品或服务的效果做夸大、保证性承诺的表述，如承诺产品具体作用范围（包括但不限于：面积/长度/距离）/承诺产品作用具体时间/有效期时长。
8. 广告文案不得涉嫌可能损害未成年人和残疾人身心健康等歧义内容。
9. 广告文案使用旅游攻略式的标题，如“国内最好吃的五条美食街，你去过几条？”、“最值得去的9座古镇，你去过几个？”使用这类型的标题时，落地页需拉起对应的内容介绍，否则需修改为与落地页推广内容强关联的文案与素材。
10. 广告文案不得模仿官方传媒消息，如模拟警方新闻等。
11. 广告文案不得使用或变相使用对比/贬低/抨击其他同类经营者的描述。
12. 广告文案不得使用易引起用户不良联想的乡村风俗类描述。
13. 品牌来源请勿超过7个字符，需为品牌名称、广告主名称、广告主简称。
14. 特殊用途化妆品中，不得涉及育发、脱毛、美乳、健美、除臭，其中祛斑、防晒仅允许物理遮盖，不得涉及具体疗效。
15. 家装广告，通过家装设计使得空间效果变大或节省空间的描述，支持使用装修后效果变大、空间扩容等描述，但不允许使用暗示建筑面积变大的描述，如：
    - 允许：有效利用空间扩容30%、装修后效果增大20平方米
    - 不允许：让家变大20%或20平方米，让家再大20%或20平方米
16. 广告文案中允许出现座机号码、400等服务号码，暂不支持个人电话号码
17. 广告推广内容不得出现金币/积分等兑换实物活动，如“积分兑换P40”，可以出现其他形式的“赢取/抽奖得实物活动”（银行/大型综合电商/知名第三方支付平台除外）。
18. 广告推广内容涉及存储介质，禁止出现音乐、电影、视频、MV等相关的可能涉嫌版权侵权宣传内容。
19. 广告文案不得使用让人误以为是第三方品牌官方行为的文案。
20. 广告文案不得搭配使用限定用户属性（性别、年龄、地域等） +绝对化的明显夸大类描述，如：男人都在买、深圳的女人都在穿、谁都买得起等广告素材。
21. 不得涉及明显虚假、谣言、诋毁历代君王/文人的古今野史类创意。
22. 广告文案不得使用各类易引起用户焦虑的内容：
    - 不得涉及金钱焦虑，如通过对社会经济情况或个人收入/存款情况的负面描述，刻意加强大众对金钱问题的忧虑；
    - 不得涉及容貌焦虑，如将容貌与工作发展、生活状况、心理情绪等进行归因或关联，制造容貌焦虑；
    - 不得涉及教育焦虑，如刻意营造优质教育资源稀缺的紧张氛围，过度强调教育资源重要性制造社会焦虑情绪；
    - 不得利用贫富差距、职业或身份歧视、故意“制造焦虑”等形式渲染对立情绪；
    - 其他易引起用户焦虑情绪的内容/场景。
23. 推广商务服务（包含中介服务、户籍服务、法律服务等）不得涉及以下内容：
    - 不得利用地方/国家政策或政策预测类内容做广告宣传；
    - 不得明示或暗示和机关单位关系密切，如特殊渠道、绿色通道、官方入口；
    - 不得涉及不良导向内容，如拉横幅、群体聚集、刑事申诉、钻法律漏洞等描述；
    - 推广积分入户业务，不得涉及无需门槛或门槛较低即可办理入户的虚假描述；
24. 图片内容需保证文字清晰可读，不得影响用户的正常浏览使用，推广意图明确。
25. 不得出现比例失调、界面不完整、文字过小、文字展示不完整的画面，影响整体排版体验；Logo或元素不得遮挡广告关键文字信息。
26. 图片需结合实际推广页面包含适当的文字说明的内容，如产品介绍，活动宣传。
27. 关于推广内容一致性的要求：
    - 广告文案、素材、落地页三者推广内容需保持一致，不得出现广告文案与素材内容不一致、外层素材推广内容在落地页无体现的情况；
    - 外层素材中的活动/利益点/内容需与落地页保持一致，如图片所涉及活动奖品，需要在落地页有相符信息，不得出现不切实际的奖品及奖金；
28. 图片不得出现点击后无法实现其功能的虚假按钮/虚假提示/仿系统功能操作的按键设计等诱导点击的内容或欺骗性内容。

    虚假按钮如：关闭、跳过、返回

    滑动解锁虚假提示，如：“立即领取”、“立即下载”“立即体验”、“立即购买”、“立即使用”等（可以使用“立即查看”）
29. 图片中不能出现真实的人民币、硬币相关元素；同时不得出现比特币等虚拟货币相关元素。
30. 广告中不得展示低俗、恶俗、含有不良暗示的画面/内容
    - 不得涉及带有明显性暗示的场景，如：凌乱的室内/床/沙发等；
    - 不得展示穿着带有引诱性质的女性形象，如：穿着情趣制服、情趣内衣、肤色装、透视装、男士白衬衣、真空凸点、内衣外穿、刻意突出敏感部位的紧身衣，裸露胸罩或内裤或仅用被子、浴巾等遮盖身体的大尺度裸露身体的女性形象，包括男性赤裸上身（运动健身类素材除外）；
    - 不得涉及明显暗示性的动作和表情，如：刻意抚摸身体、舔舐食物、舔咬嘴唇、真人接吻及其他暗示性动作等；
    - 不得涉及易产生明显性暗示的拍摄视角，如：俯拍女性胸部、特写局部身体等；
    - 不得涉及任何带有性暗示的文案描述，如：宅男福利天堂、老司机才懂；
    - 不得使用女性搭配荧光粉色、红色、紫色、蓝色等擦边不良场所的背景灯光。
31. 图片不得使用真实证件作为元素内容。如：“身份证”、“社保卡”、“护照”；也不支持出现涉及个人隐私的营业执照/合同//公积金卡/社保卡/简历/工资条等。
32. 广告外层素材不允许使用闪烁、高光等方式突出按钮。
33. 广告文案与素材中如使用目的地/景点/城市等描述，需与素材中搭配的图片相匹配，特别是地标性建筑景点，如描述为巴黎，配图不允许为其他城市的地标性景点，请严格按照此要求上传广告素材。
34. 不允许使用模拟弹窗样式/红包卡券的设计。
35. 广告图片禁止展示恶心不适的画面，包括不限于：
    - 禁止出现注射针头、病变人体器官、长斑、秃顶、刻意放大的人体部位等；
    - 禁止出现真实或仿真度较高的五毒与软体动物形象，如：蛇、蝎子、蜈蚣、蟾蜍、壁虎、蜘蛛；
    - 不得出现展示清理排泄物、分泌物等场景。
36. 不得出现负面消极场景，包含不限于：
    - 不得使用虚假的卖惨营销行为，如：撑不下，只卖最后一天了；
    - 不得使用表情痛苦狰狞、抱头哭、展示抑郁、躁怒等的人物情绪；
    - 不得展示真实灾难场景画面，如：地震、交通事故；
    - 不得使用坟墓、丧葬等敏感场景。
37. 广告图片禁止出现血腥、恐怖、暴力、反党、反社会，易引起用户恐慌的画面图片。
38. 广告图片不得展示易影响人身安全的危险行为。
39. 广告图片请勿出现与不良社会事件相关的人物。
40. 广告图片不得模仿第三方知名支付软件设计。
41. 广告图片/落地页不允许出现虚假倒计时。
42. 广告图片不得出现转账付款信息，如支付宝、银行账户、微信付款二维码等。
43. 电商行业使用单品素材投放促活任务，外层素材价格需与落地页保持一致，如促活链接调起为产品聚合页，在无法验证外层产品价格与实际售卖价格是否一致的情况下，外层素材请勿出现特定价格描述，请使用xx元起。
44. 广告暂不支持山寨产品投放，包括但不限于：知名产品仿制品、日常认知中的高价值产品以低廉价格销售的产品，如：
    - 19.9元/1套的中高档茶具；
    - 19.9元/1件的珠宝翡翠玉石；
    - 19.9元/1双的真皮鞋；
    - 50元/10斤的贵州茅台镇酒；
    - 高仿的服饰鞋帽箱包、或外贸原单等商品。
45. 广告图片允许在添加“截图到微信扫描”提示语的情况下使用企业二维码、小程序码（金融行业除外），但不得出现个人的二维码、条形码图、个人微信号、个人QQ号、QQ群号或其他社交软件号码等文字内容，禁止添加个人好友的行为。
46. 广告不得出现违背“厉行节约、反对浪费”的社会良好风尚的内容，如：吃播。
47. 广告中禁止使用中国地图元素，包含中国地图/中国局部地图 。
48. 广告请勿出现法律法规未明确可食用的水生、陆生动物形象。

51. 针对视频类素材，需遵守如下规定：

52. 落地页和URL规范
    - 落地页与开屏展示内容直接相关，提供与所宣传的内容相关的实用信息，其中首屏位置须排列用户最感兴趣，相关性最强的内容。

    - 落地页不允许出现自动弹窗。
    - 落地页不允许使用静默下载。
    - 落地页不得接调用、开启QQ、微信、QQ群等相关功能。
    - 落地页不得出现非完整页面/页面结构凌乱/留白超过四分之一等明显样式问题。
    - 以应用下载为推广目的落地页不允许有二跳。
    - 合理设计页面，让访问者能够轻松找信息，避免元素较多，引起用户无谓找寻。
    - 如果要求访问者提供个人信息，须清楚说明获取信息的原因和用途。
    - 落地页及落地页内不允许出现自构建的应用下载链接（此功能由华为应用市场提供）。
    - 落地页打开需静默无声，不得有自动播放的音频、视频或其他未经消费者同意严重消耗其网络流量的内容。
    - 目标url链接不得为“.doc/.exe”或直接下载文件链接；可正常访问不出现病毒及而已代码；不得含有本规范禁止的全部内容。

      协议死链（如404、403、503状态）；

      内容死链（内容不存在、已删除、空白页等）；

      域名失效（因网页安全被屏蔽等）；

      无法正常自动加载的外部链接（如加载在中国大陆地区不可访问的境外地址\*.google.com,\*.facebook.com等）。

    - url指向页面的内容合法性，不得含有禁止开户的内容，不涉及未提交行业资质的内容。

    - 落地页中图片不允许使用不可交互的虚假链接。
    - 针对https的落地页内可添加不超过3个以上的 “浮窗”。

      仅允许使用低频闪烁按钮，不允许使用高频快闪设计；

      不允许使用高饱和度、高亮度的按钮颜色。
