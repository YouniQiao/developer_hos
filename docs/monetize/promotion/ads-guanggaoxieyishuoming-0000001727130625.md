# 鲸鸿动能平台协议

- 如果鲸鸿动能协议有更新，您在进入鲸鸿动能平台后，系统会弹出协议更新页面，同时在鲸鸿动能官网帮助中心您也可以查看协议的更新内容以及完整的协议，您可以选择“立即签署”或“延期签署”。
  - 立即签署：如果您选择立即签署，协议将于xxxx年xx月生效，签署后进入鲸鸿动能平台。
  - 延期签署：如果您选择延期签署，您必须在协议生效前完成签署，否则您将无法使用鲸鸿动能平台。鲸鸿动能平台会在协议生效前天提醒您协议更新时间，如果在协议更新后30天内，您未签署时继续使用鲸鸿动能相关服务 ，则代表您知晓、同意、并遵循鲸鸿动能协议中修改的条款，协议将于XXXX年XX月生效。
- 如果您开通了华为联盟提供的[团队账号](https://developer.huawei.com/consumer/cn/doc/start/team-account-guides-0000001053785552)功能，您需要使用主账号登录鲸鸿动能平台并重新签署协议。
- 如果您开通了鲸鸿动能提供的团队账号功能，您可以使用主账号登录鲸鸿动能平台并签署协议，也可以授权您的管理员进行签署协议，详情可查看[授权签署协议权限](/docs/monetize/promotion/addtuandui-0000001079312694#ZH-CN_TOPIC_0000001079312694__li16934182711012)。
- 如果您想了解华为开发者侧的协议，请参考[服务协议](/docs/monetize/monetization/agreement-0000001062394234)。
