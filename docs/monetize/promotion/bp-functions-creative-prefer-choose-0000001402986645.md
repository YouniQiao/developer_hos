# 选择创意优选

1. 创建或修改推广任务时，若该任务支持推广创意功能，勾选“编辑辅助创意”。

   ![](./img/zh-cn_image_0000002555979847_c3e1151f6c49.png "点击放大")
2. 进入“推广创意”页面，在“创意展示模式”区域勾选“优选模式”或“轮播模式”，完成后点击“提交创意”。

   ![](./img/zh-cn_image_0000002525059904_fa8097e1f28f.png "点击放大")
