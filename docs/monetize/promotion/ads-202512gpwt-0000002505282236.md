# 2025年12月

<strong>典型驳回行业：软件应用-地图导航、电商、金融</strong>

<strong>典型驳回问题场景主要涵盖：</strong>

一、地图导航类APP“官方授权”资质要求

二、广告不得使用容易引起歧义的词语

三、广告涉及禁投内容（高频禁投）

四、广告素材不允许出现切割、重叠、拉伸、文字剪裁等影响用户观看的质量问题

五、金融广告不得涉及承诺描述

<strong>一、</strong> <strong>推广地图导航类APP，涉及xx卫星导航系统相关的宣传内容，如：“卫星导航”、“卫星定位”、“免费卫星导航”等描述，需提交相关官方授权</strong>

<strong>五、金融广告不得涉及承诺描述，如：不得对未来效果、收益或者与其相关的情况做出保证性承诺，不得明示或者暗示保本、无风险或者保收益</strong>

1.不得出现任何明示或暗示金融内幕消息的内容，如：“股市内参”、“揭秘”、“内部消息”

2.不得对产品使用的效果描述不得涉及明示/暗示金钱变化
