# 商品数据

【简介】广告主通过本接口获取动态商品分析报表。

<strong>请求地址</strong>

https://ads.cloud.huawei.com/openapi/v2/reports/dpa/query

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> | |
| --- | --- | --- | --- | --- |
| advertiser\_id | long | 是 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 | |
| time\_granularity | string | 是 | "STAT\_TIME\_GRANULARITY\_DAILY"天粒度。  详见[【时间粒度】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#时间粒度)  不支持小时STAT\_TIME\_GRANULARITY\_HOURLY | |
| filtering | Struct1 | 否 | 筛选条件 | |
| page | integer | 否 | 搜索页码  默认值1  取值范围1~10000。 | |
| page\_size | integer | 否 | 每页展示的数据条数  默认值20  取值范围 1~10000。 | |
| start\_date | string | 是 | 起始时间。 | |
| end\_date | string | 是 | 结束时间。 | |
| order\_field | string | 否 | 排序字段，所有的统计指标均可参与排序。  筛选指标，字符串形式样例：  cost：花费  click\_count：点击量  download\_count：下载量  install\_count：安装量  active\_count\_normalized：激活量  register\_count：注册 | |
| topn | integer | 否 | 汇总数据指标。 | |
| index\_screen\_list | Struct3 | 否 | 指标筛选。  --按照指标值的区间范围筛选 | |
| dimension\_list | Struct2 | 否 | 维度细分。  --按照维度字段值筛选记录 | |
| time\_line | string | 否 | 时间口径。 | |
| group\_by | string[] | 否 | 数据分组维度，枚举值  当dpa\_data\_type= dpa\_product时，支持DATE，PRODUCT\_LIB\_ID，POSITION\_ID，PRODUCT\_ID，FIRST\_CATEGORY，SECOND\_CATEGORY，  THIRD\_CATEGORY，PRODUCT\_BRAND；  当dpa\_data\_type= dpa\_product\_ library时，支持DATE，PRODUCT\_LIB\_ID，COUNTRY  当dpa\_data\_type= dpa\_creative\_template时，支持DATE， CREATIVE\_TEMPLATE\_ID，POSITION\_ID；  按天或月，必传对应维度+时间；按总，必传对应维度 | |
| dpa\_data\_type | string | 是 | 枚举值，包括：商品投放、创意模板  dpa\_product  dpa\_product\_library  dpa\_creative\_template | |

filtering(Struct1)参数：

| 参数名称 | 类型 | 是否必选 | 描述 |
| --- | --- | --- | --- |
| product\_lib\_id | string[] | 否 | 商品库ID  dpa\_data\_type为dpa\_product或dpa\_product\_library |
| creative\_template\_id | string[] | 否 | 创意模板ID  dpa\_data\_type为dpa\_creative\_template |

dimension\_list（Struct2）参数：

| 参数名称 | 类型 | 是否必选 | 描述 |
| --- | --- | --- | --- |
| dimension | string | 否 | 筛选维度  当dpa\_data\_type= dpa\_product时，支持adposition\_id：版位ID;  product\_lib：商品库ID；  product\_id：商品ID；  first\_category：商品一级类目；  second\_category：商品二级类目；  third\_category：商品三级类目；  product\_brand：商品品牌；  当dpa\_data\_type= dpa\_product\_library时，支持  adposition\_id：版位ID;  product\_lib：商品库ID；  country：国家。  当dpa\_data\_type=dpa\_creative\_template时，支持  adposition\_id：版位ID。 |
| data | string[] | 否 | 筛选指标。 |

index\_screen\_list（Struct3）参数：

| 参数名称 | 类型 | 是否必选 | 描述 |
| --- | --- | --- | --- |
| index\_screen | string | 否 | 筛选指标，取值为：  effective\_cost：花费  effective\_impression\_count：展示量  effective\_click\_count：点击量  effective\_per\_click：点击均价  effective\_download\_count：下载量。 |
| type | integer | 否 | 筛选指标类型，取值为：  1．大于等于，只需要赋值low\_value，type=1、  2．小于等于，只需要赋值up\_value，type=2  3．介于，需要赋值low\_value up\_value，type=0。 |
| low\_value | float | 否 | 下限值。 |
