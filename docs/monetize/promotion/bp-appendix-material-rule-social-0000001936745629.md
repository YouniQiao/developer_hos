# 社交

## 社交&网络直播行业通用审核规范

<strong>1. 不得使用违背公序良俗等内容：</strong>

1）广告文案需符合社会良好风尚，不得使用涉及低俗、打擦边球、不良暗示、容易引起歧义的文案，包含但不限于以下内容：“深夜孤单”、“刺激”、“过瘾”、“爽”、“床上直播”、“深夜直播”、 “秘密直播”等，如：性感女神深夜直播，劲歌热舞嗨不停；千万别看她直播，你想看的她都有，太刺激了。

2）不得含有不良导向词汇，如：“大波”、“艳遇”、“18禁”、“老司机才懂”等。

3）不得使用容易产生歧义的“约”字的搭配，如：单身独居美女在家睡衣与你约聊；寂寞妹纸等你来约；不闲聊，直接见，快快约到你所爱。

4）不得使用强调急迫性的“求”字的搭配，包括但不限于“求爱”、“求约”、“求聊”等。

5）不得宣传不良价值观，包括但不限于：伦理、拜金主义、贪污、受贿、婚外恋、一夜情、同性恋以及其他引人消极颓废的行为。

6）不得涉及“磕炮”“连睡”等敏感词，包括但不限于：语音做爱、语音连睡、连麦睡觉、语音哄睡等。

<strong>2. 文案基础要求：</strong>

1）不得使用撩拨型口头语文案，如：“你过来，我想撩你一下！”。

2）不得使用负面、歧视语气、指向特定人群的网络用语，包括但不限于： “水友”、“打工妹”等。如：农村美女在家直播，比城里人还会玩；打工妹玩直播，不嫌弃的加她一起玩。

<strong>3. 图片基础要求：</strong>

广告素材不允许使用虚假时间标识、虚假播放按钮等元素。

<strong>4. 广告素材人物形象不得含有暴露、低俗、动作不雅等：</strong>

1）衣着：不得使用暴露/暧昧/带有引诱性质、突出敏感部位、穿着透视/肤色/紧身、动作不雅的暗示或隐含性信息的形象。

不得穿着情趣制服、情趣内衣、暴露装、肤色装、透视装、真空凸点、内衣外穿、刻意突出敏感部位的紧身衣、三角短裤或呈现三角状短裤、衣衫不整等。

不得裸露胸罩或内裤，或仅用被子、浴巾、衣物等遮盖身体。

不得穿着过于紧身导致敏感部位突出，不得对局部敏感部位聚焦或具有特殊修饰等镜头作为素材创意。

不得大尺度裸露身体，包括男性赤裸上身。

2）表情/动作：不得使用带有性暗示、性挑逗或其他低级趣味的行为，包括但不限于：

不得使用暗示不良诱导动作，如手指向某些敏感部位等引发不良暗示的姿势；

不得揉挤胸部或刻意对着镜头抖胸、扭臀、抚摸敏感部位等；

不得拉扯衣服、脱/穿丝袜或内衣胸罩等刻意表现低级趣味的动作；

聚焦胸部、臀部、腿部、脚部等敏感部位博取眼球；

不得使用性用品或暗示性的道具进行展示；

不得出现湿身诱惑、 嘟嘴、咬唇、咬手指、吐舌头、暧昧眼神迷离、刻意露肩、撩裙子、撩衣服等带有性引诱性质的人物形象；

禁止出现抽烟形象。

3）妆容：不得使用带有低俗、不雅、恐怖妆容或在身体各部位描画低俗文字或图案。

4）声音：视频播放低俗音乐、模仿叫床、暧昧语气等低俗声音。

5）场景&元素：

不得出现美女+浴室、卫生间、卧室等敏感场景；

不得出现涉及两性隐私或AV女优，如苍老师

## 社交行业审核规范

<strong>社交广告应当遵循以下规范：</strong>

1）为避免不良导向，社交广告不得使用“摇一摇”、 “漂流瓶” 、“查看附近的人” 等介绍作为广告。

2）不得出现通过模拟虚拟定位功能泄漏个人信息的描述，包括但不限于“附近”、“周边”等，（可使用同城描述），如：附近200米的美女主动打招呼，等你来约。

3）社交广告内容请勿涉及付费相关描述，易引起歧义。

## 网络直播行业审核规范

<strong>直播广告文案特殊要求：</strong>

1）不得出现通过模拟虚拟定位功能泄漏个人信息的描述，包括但不限于“附近”、“周边”等，可使用同城描述 驳回案例：附近200米的美女主动打招呼，等你来约。

2）网络直播类广告不得对主播使用诱惑性称谓，如：“熟女”、“少妇”、“小妖精”等

3）网络直播类广告请勿出现视频交友、视频邀约的描述，如：漂亮女主播给你发来一个视频邀约/视频电话
