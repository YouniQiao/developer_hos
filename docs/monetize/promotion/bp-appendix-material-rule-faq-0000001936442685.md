# FAQ

## 审核时间

<strong>Q1：素材审核周期多久？</strong>

<strong>A：</strong>合约资源的素材审核周期为3个工作日，建议尽早提交素材审核，竞价素材工作日当天17:00前提交，会当天审核完毕，晚于17:00提交的任务将于第二个工作日完成审核。

<strong>Q2：华为应用市场素材任务审核时间是什么？</strong>

<strong>A：</strong>工作日：09:00-17:00，当天完成17:00前提交的素材审核；

非工作日：非审核时间提交的素材，推迟到下一个工作日审核。

## 素材规范

<strong>Q1：上传素材提示“素材不合法，请重试”，是什么原因？</strong>

<strong>A：</strong>可能是以下三种情况：

1）素材尺寸不符合版位尺寸要求。

2）素材格式不符合规范要求。

3）素材文件大小不符合规范要求。

<strong>Q2：素材资质有多个图片文件，怎么上传？</strong>

<strong>A：</strong>可以把多个图片压缩成ZIP格式再上传。

<strong>Q3：已经创建素材任务并提交，但忘记提供资质证明该如何补充？</strong>

<strong>A：</strong>在素材审核之前，进入已提交的素材任务页面，补充上传相关资质证明即可。

<strong>Q4：素材使用了第三方品牌的名称，是否需要提供第三方授权资质？</strong>

<strong>A：</strong>需要提供第三方品牌授权证明。

<strong>Q5：推广代言人使用公司自己的员工，是否需要提供肖像授权？</strong>

<strong>A：</strong>需提供员工本人肖像授权证明。

<strong>Q6：推广素材使用AI人像，需要提供什么资质？</strong>

<strong>A：</strong>素材使用AI人像需提供盖章的《华为应用市场推广免责函》。
