# 行业准入规则

## 鲸鸿动能广告禁止内容

![](./img/caution_3.0-zh-cn_657b87192b37.png) 

推广广告在符合鲸鸿动能广告平台的广告政策外，需始终遵守投放国家/地区所有适用的法律及法规。

| 行业 | 说明 |
| --- | --- |
| 违法行业 | 毒品买卖、人口及人体体液、器官贩运和买卖、濒危动植物及其制品售卖、洗钱。 |
| 性服务、成人用品及成人内容 | 宣传直接或间接的性服务、成人用品，脱衣舞俱乐部、色情电影院、性玩具、成人杂志、性功能增强产品、相亲网站、姿势性感的模特等内容。 |
| 爆炸物(含烟花) | 炸弹，烟花鞭炮，手榴弹；炸弹制造相关的网站，炸弹零部件的3D打印指南、软件或设备。 |
| 保释担保服务 | 商业保释担保代理，保释担保融资服务、赏金猎人、保释担保人、移民保释服务、羁押保释服务。 |
| 烟草及相关制品 | 包含烟草的任何产品，模拟吸烟的产品。 |
| 处方药 | 处方药的销售或使用。 |
| 消遣式药物 | 可卡因、冰毒、海洛因和其他非法阿片类药物、大麻、可卡因替代品、甲氧麻黄酮、“合法兴奋剂”；针管、水烟筒、大麻咖啡馆；交流吸毒技巧或建议的论坛。 |
| 戒瘾服务 | 包括瘾症临床治疗提供商、戒瘾中心和互助组织等提供康复支持服务的机构、为戒瘾服务机构提供潜在客户信息或转介客户的机构、面向药瘾和酒瘾患者的应急热线，部分属于非医疗机构,如Rehab, AA, 戒酒疗养院等。 |
| 枪支武器 | 枪支及相关的产品，刀具及其他武器。 |
| 经济援助 | 公益，众筹、募捐等相关广告。 |
| 收养 | 禁止私人及非法收养。 |
| 占卜起名 | 禁止投放占星术、超自然现象相关的广告（如：超能力、预知未来、占星、骨相学、看手相、命理学、读心术、催眠术、信仰疗法、巫术等）。 |
| 医疗美容 | 医疗美容相关内容。 |
| 人体艺术 | 人体艺术相关内容。 |
| 不安全的补充剂 | 不安全的补充剂，包括但不限于合成类固醇、壳聚糖、紫草、脱氢表雄酮、麻黄和人类生长激素。 |
| 金融借贷（高利贷） | 禁止宣传高利贷，私人借贷服务。 |
| 多层次传销 | 推广赚钱机会的广告必须详细说明相关的产品或业务模式，且不得推广投资少回报快的业务模式，包括多层次传销。 |
| 一分钱竞拍 | 广告不得推广一分钱拍卖、娱乐式拍卖或其他类似的商业模式。 |
| 政治宣传 | &gt; 禁止宣传违反当地法律法规要求的政治内容。  &gt;禁止政治选举内容。  &gt;禁止宣传政治敏感内容。 |

## 鲸鸿动能广告受限内容

![](./img/caution_3.0-zh-cn_0c2a8df8c8f7.png) 

推广广告在符合鲸鸿动能广告平台的广告政策外，需始终遵守投放国家/地区所有适用的法律及法规。

| 行业 | 描述 |
| --- | --- |
| 博彩 | 包括但不限于线下赌场、在线赌博、在线类赌博的非赌场游戏、社交类赌博,彩票，投注，bingos,纸牌游戏，游乐场游戏等。 |
| 酒精饮料 | &gt;禁止误导：将酒精与社会成就/性成功等关联；有镇静/刺激/强身健体/治病等功效。  &gt;禁止不负责任：出现未成年人和孕妇形象；酒后驾车/运动等场景；鼓励饮酒。  &gt;需明示健康警告及年龄限制：必须包含健康警告“滥用酒精对健康有害”，以及年龄限制。  &gt;禁止向未成年人投放，而且只能定向到明确允许展示酒精饮料广告的国家/地区。  受限制的酒精饮料的示例：啤酒、葡萄酒、清酒、烈性酒、香槟、烈性葡萄酒、无酒精的啤酒、无酒精的葡萄酒和无酒精的蒸馏酒。 |
| 医疗服务 | &gt; 禁止夸大效果：保证疗效；治愈不治之症。  &gt;禁止特殊医疗服务：堕胎广告、产前性别鉴定服务、催眠疗法、推测性和实验性医学疗法。  &gt;禁止过度宣传：广告内容只能包括机构名称、患者类型、设施、费用；禁止提及除姓名、地址、工作时间、从业许可范围以外的信息。  &gt;禁止宣传招募临床受试者的内容。 |
| 医疗器械 | &gt;禁止误导：不能夸大或对设备的功效、使用进行错误引导；只能根据其标签、使用说明、技术资料中所述预期目的进行宣传。  &gt;禁止特殊器械：国家/地区卫生系统报销的医疗器械不能向公众宣传；医疗保健专业人员使用的医疗器械。  &gt; 特殊产品宣传限制：如艾滋病病毒检测产品，节育器械等。 |
| 非处方药（制造商） | &gt;禁止夸大药品效果：保证疗效；声称产品没有副作用/绝对安全/可提升幸福感等。  &gt;禁止误导药物使用：鼓励过度使用/滥用药物，给人制造不需要看医生的错觉。  &gt;推广非处方药的广告必须遵守所有适用的当地法律，以及强制或既定的行业规范、准则、许可和审批，并采用符合本地适用法律的年龄和国家/地区定位标准。 |
| 赡养养育(养老院，福利院，青少年养育及老年赡养) | 禁止临终关怀服务，私人代养、寄养及代孕。 |
| 健康产品/服务  &gt; 减肥、增重 | &gt; 禁止误导：不能暗示节食者可以随意饮食，且仍能减重。  &gt; 禁止专业代言：禁止包含个别医生或健康专业人士的建议内容，以及与减肥相关的健康声明。  &gt;禁止特殊产品：禁止与减肥或控制体重有关的人绒毛膜促性腺激素(hCG)产品，或与合成代谢类固醇一起推广的产品。  &gt; 禁止包含“前后对比图”，或者描绘意料之外或不切实际的效果的图片。  &gt; 推广减肥商品和服务的广告仅能面向成年人的受众投放。 |
| 健康产品/服务  &gt; 保健品 | &gt; 禁止误导：禁止宣称保健品可以预防/缓解疾病，或可代替正常饮食和运动。  &gt; 禁止夸大：禁止夸大产品带来的减肥效果和健康功能。 |
| 婴儿产品  &gt; 配方奶粉 | &gt;禁止美化误导：不能通过宣传产品特性、性能效果、预防和治疗功能误导消费者；禁止美化奶粉、忽视母乳喂养的重要性。  &gt; 需明示母乳重要性：需明示母乳对婴儿而言唯一安全且易获得的食品；应始终提倡母乳喂养。  &gt;禁止 12个月以下婴儿奶粉及配方食品广告。 |
| 玩具 | &gt;禁止危险产品：禁止宣传可能对人产生身体、心理伤害的产品。  &gt;产品信息明示：必须清楚说明须分开购买的商品(如电池、油画、附加配件等)；正确的产品标签和安全信息。 |
| 食品饮料  &gt; HFSS(垃圾食品及甜食) | &gt;禁止鼓励不健康的生活方式：禁止破坏健康饮食习惯、鼓励过度消费/摄入。  &gt;禁止误导：禁止宣称产品可以代替正常饮食，或具有治疗功能/营养价值。  &gt;禁止医学专业人士代言：禁止医疗从业人员为产品背书，出现“医学界推荐”等声明。 |
| 金融&gt;个人理财 | 个人理财类广告（如：信用卡、借贷、财政计划、避险基金、保险、投资、共同基金、期权、退休计划、股票、纳税计划等）。 |
| 小说 | 禁止宣传科幻类、幻想类小说广告。 |
| 宠物 | 禁止宣传涉及虐待动物的内容。 |
| 化妆品 | &gt;禁止虚假误导：保证疗效、声称没有副作用；对产品功效的宣传只能依据科学测试。  &gt; 明示产品风险：使用化妆品时的健康风险必须明示。  &gt; 不得通过涉嫌种族歧视的肤色对比凸显化妆品效果等。 |
| 汽车/机动车 | 不得通过超速场景凸显机动车性能等。 |
| 房地产 | 有些国家/地区需政府及相关机构审批资质。 |
| 通信服务(如VPN、电信等) | 禁止宣传数据交易及涉及用户隐私的相关内容。 |
| 约会交友 | &gt; 内容：禁止以下内容。  – 发布成人交友和邮购新娘类服务。  – 用文字指出有人已经或将要通过所推广的服务搜索某用户或与其接触，如“ 已有 5 名女士搜索过你！”。  &gt;性暗示、性交易、援交为目的的交友软件及广告内容。 |
| 婚恋中介 | &gt;禁止宣传前卫的婚恋理念，如闪婚，裸婚等。  &gt;禁止宣传个人财产，如有车有房，存款多少等。 |
| 教育培训 | 不得做出升学承诺等。 |
| 特殊教育 | 不得涉嫌特殊人群歧视等。 |
| 农药化肥 | 禁止宣传违规产品：禁止未在相关部门注册的农药/化肥的广告。 |
| 文物和艺术品交易 | 禁止有争议的文物或艺术品广告宣传。 |
| 生物技术广告 | 禁止不符合公序良俗的生物技术广告内容。 |
| 金融产品/服务 | &gt;禁止复合型投机性金融产品/差价合约、 复合型投机性金融产品/即期外汇、债务服务、加密（虚拟）货币、首次代币发售、二元期权。  推广官方认可机构的信用卡申请或金融服务的广告必须明确且充分地公开相关费用，包括在广告落地页公开年利率、手续费、利率和相关产品发行实体的真实地址等信息。  &gt;推广信用卡、贷款或保险服务的广告只能定位成年人的用户。  &gt;推广信用卡、贷款或保险服务的广告不得直接要求用户输入个人财务信息，包括信用卡信息。广告不得推广误导性或欺骗性的学生贷款合并、免除或重组服务。  &gt;宣传金融产品和服务时，必须遵守广告定位到的任何区域的国家/地区和地方性法规，例如，按照当地法律要求加入特定的披露信息。 |
| 金融产品/服务  &gt; 个人贷款 | &gt; 要求披露尽量多的信息，包括还款期、最高年利率和代表性贷款中费用示例。  &gt; 不允许宣传短期个人贷款(还款周期不超过60天)。  &gt;禁止抵押贷款或断供房出售广告中可能存在的欺诈虚假信息，保证防止断供房出售、要求用户绕过贷款人直接向公司或第三方付款等。 |
| 律师/律所 | 禁止误导：广告必须客观，不得歪曲事实。 |
| 会计师(个人) | 禁止涉嫌偷税漏税，商业贿赂等暗示内容。 |
| 订阅服务 | 在落地页内容设置空复选框，用户必须勾选才能接受条款和条件。不得设置默认勾选的复选框。  在落地页内容中加入您的隐私权政策链接。  清楚显示所推广服务的价格，包括所有经常性费用的详细信息。即使条款和条件有提及价格，广告的落地页中也必须显示价格。  广告落地页的内容必须清楚说明和显示订阅条款和条件。您可以在登录界面展示条款和条件的摘要版本，前提是此版本能够清楚说明关于订阅服务的主要信息（包括如何取消服务）。  如果广告中宣传的方案或服务要求订阅，则广告必须面向成年人的用户。  广告文案必须清楚说明订阅条款，例如标明“需要订阅”。  广告文本必须准确描述商品或服务，且必须充分描述商业模式，不得隐瞒隐形优惠的具体细节。  提供免费试用优惠的广告必须清楚说明参加试用的所有条件。如果免费试用之外有一次性收费，落地页内容必须清楚显示何时收费以及费用为多少。 |
| 版权内容 | 不允许投放未经授权使用受版权保护的内容的广告。如果您已取得合法授权，可以使用受版权保护的内容，请申请认证后再投放广告。 |
