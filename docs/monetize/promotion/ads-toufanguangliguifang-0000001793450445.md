# 鲸鸿动能投放违规管理规范

## 一、总则

**1.1** **目的**

为保护投放方、用户、鲸鸿动能平台以及华为软件技术有限公司及其关联公司（以下称“华为公司”）的合法权益，规范投放方的合作行为、推广投放行为及履约行为，建立健康有序的生态环境，特制定本规范。

**1.2** **适用范围**

通过鲸鸿动能平台发布信息宣传推广其产品或服务的企业，或通过鲸鸿动能平台进行信息发布的企业（简称“投放方”）的投放行为均适用本规范。包括自行投放的企业或受委托为他人投放的服务商。

所有投放方应严格遵守法律法规、协议约定及平台规则和规范要求，违反相关合作协议及平台规则的，除依据本管理规范外，华为公司有权依据相关合作协议、本规范及相关鲸鸿动能平台规则的约定对投放方采取处罚措施，追究投放方的法律责任。投放方按本规范接受处罚的，并不能免除或抵消其应当承担的违约责任或其他法律责任。

**2.2** **违规行为认定**

2.2.1违规信息来源

投放方的违规信息的来源包括但不限于华为公司内部审核或巡检，消费者投诉或举报，监管部门监督指令，其他第三方反馈等信息。

2.2.2违规行为取证

华为公司在进行调查过程中有权对举报人、投诉人、被举报投放方的相关信息进行取证留存。

取证的形式及内容包括但不限于：即时聊天软件对话截图、email记录、电话录音、系统数据处理记录、文档原件、合同原件或传真、发票（收据）原件或传真、网站内容截图等。

在对投放方进行违规调查期间，华为公司有权自行独立判断是否先行按照本规范中的规定对投放方采取一项或多项处理措施。

2.2.3违规行为认定

投放方应当对其推广产品、推广内容、推广行为、履约行为以及使用平台相关功能和服务的合法合规性负责，华为公司有权对前述事项进行审查，但无论是否属于华为公司查责任范围内，一旦投放方推广产品、推广内容、推广行为或履约行为等为相关法律法规禁止发布或若发布将很可能导致违法、违规风险，或存在妨害平台秩序或侵害消费者或第三方合法权益，或存在严重违背社会公序良俗、违反平台相关规则情形，或出现违法、负面事件或其他不正当行为被报道/被有权机关调查的，华为公司有权随时根据本规范及相关合作协议、鲸鸿动能平台规则采取一项或多项处罚措施，投放方应认可因此产生的相应处罚后果，同时也享有解释及申辩的权利。

2.2.4违规行为披露

华为公司若接到投诉、举报，或司法、执法机关或其他主管部门调查，有权将投放方的相关信息(包括但不限于注册信息、开户信息、投放情况、消耗数据等)向相关部门或第三方披露。

**2.3** **违规处理规则**

2.3.1违规处理措施

对于投放方的投放行为（包含但不限于素材、落地页、落地页二跳、私域、app、小程序等）违反法律法规、华为公司协议、鲸鸿动能平台规则的，华为公司将根据违规情节，依据本规范对投放方账号、投放方主体采取如下一种或多种处罚措施：

1. 通报：指在各类渠道、媒体具名或不具名通报投放方违规情况。
2. 限期整改：对违规内容/APP/商品提出整改要求。
3. 违规素材下线：对涉嫌违规、涉负面舆情或涉投诉举报的内容素材进行下线。
4. 全量素材下线：对违规账户内的所有素材进行下线。
5. 限制新建任务：指在规定时间内禁止投放方新建任务。
6. 禁止推广：对投放方提交的全部素材予以下线，账户禁止投放。
7. 冻结账户：指在规定时间内冻结涉嫌违规的鲸鸿动能账户，可支持解冻。
8. 封停账户：指永久封停涉嫌违规的鲸鸿动能账户，不支持解封。
9. 罚没账户余额：直接扣除投放方在鲸鸿动能平台账户的全部/部分剩余款项作为违约金，不予退还。
10. 封停该企业下所有账户：指永久封停涉嫌违规的鲸鸿动能账户及其同一主体注册的相关账户，不支持解封。
11. 限制新开账户：指禁止违规主体（公司）注册新的账户。
12. 终止合作：永久禁止违规主体（公司）注册成为投放方；永久禁止该同一主体（同一主体认定范围见附则3.1）企业注册鲸鸿动能账户。
13. 赔偿：因投放方违规给华为公司、消费者、流量供应方或其他任何主体造成损失（包括但不限于因处理消费者投诉、消费者索赔、行政处罚等支付的费用等），投放方应承担上述主体所遭受损失的全部赔偿责任。

2.3.3违规处理特殊细则

1. 加重原则

   同一投放方发生违规行为后，如造成以下影响，华为公司可对投放方进行升级处理。

   - 造成监管或司法影响：投放方违规行为导致华为公司被国家机关质询/调查/立案处理、与第三方发生诉讼/仲裁，给鲸鸿动能平台/流量供应方/其他任何主体造成负面影响，或需向第三方支付赔偿或补偿或罚金，平台将根据实际违规等级进行升级处理。
   - 造成舆情或用户影响：投放方违规行为严重损害消费者合法权益或第三方合法权益，造成群体性客诉、大V（粉丝数量＞1w）投诉、产生社会舆论、造成负面影响等情形，给鲸鸿动能平台/流量供应方/其他任何主体造成恶劣影响、重大舆情，平台将根据实际违规等级进行升级处理。
   - 恶意对抗平台，造成平台资源浪费：投放方在推广过程中存在恶意与平台对抗的行为（如撞审；违规篡改；同一主体下账户出现多次、重复违规的或在特定时间内几种违规的；利用技术手段对抗平台规则等），给鲸鸿动能平台、流量供应方及其他主体造成潜在风险或直接造成损失，平台将根据实际违规等级进行处理或升级处理。
   - 消极配合平台处理：包括但不限于电话接听后机器自动接听语言播报，自动挂断等；无人接听，关机、空号；接听人员态度恶劣、消极回应等行为。
   - 经华为公司合理判断出现其他高风险场景。
2. 减轻原则

   同一投放方发生非一级违规的违规行为后，若有以下某一种行为，鲸鸿动能平台可对投放方进行减轻处理，即按照违规行为降低一级所对应的处理方案进行处理，最低至取消处理。

   - 投放方主动承认违规行为，经核实未对华为公司带来不良影响的情形；
   - 投放方的违规行为，经投放方举证且由华为公司核实确认为非投放方主导的情形；
   - 投放方的违规行为被认定后，能采取积极补救措施，降低不良影响的情形。
3. 赔偿原则

   因投放方违规给华为公司、消费者、流量供应方或其他任何主体造成损失（包括但不限于因处理消费者投诉、消费者索赔、行政处罚等支付的费用等），投放方应承担上述主体所遭受损失的全部赔偿责任。

2.3.4申诉

违规投放方接到违规处理通知后，可在3个自然日内向平台提交包含相关申诉理由和证据的书面申诉请求。违规投放方接到违规处理通知后，未在3个自然日内提出申诉请求的，视为接受处理结果，并永久放弃任何申诉权利。

## 三、附则

**3.1** **关联公司中“同一主体”的认定范围**

若投放方之间存在以下关系或在与华为公司签署的数据推广协议或相关协议下享受同等权利义务的客户，将认定为同一个客户，也称同一个投放方。

1. 同一股东：A公司股东与B公司股东完全相同或拥有1名（包含）以上相同股东，且相同股东累计股权占比不低于50%，则A公司和B公司可视为同一客户。
2. 100%持股：A公司对B公司100%持股，则A公司和B公司可视为同一客户。
3. 同一集团：A公司和B公司同属于一个集团公司（在已上市或在上市进程中的集团公司或主体公司的财报/公开招股书中包含A、B公司），则A公司和B公司可视为同一客户。
4. 层级从属关系：A公司对B公司100%持股，同时B公司从属于C集团（从属定义见“同一集团”），则ABC三个公司可视为同一客户。
5. 以上场景信息均需可在第三方企业信用信息网站可查；若以上情形均不满足，华为公司通过调研或第三方渠道获取到的相关信息，也可作为“同一客户”的判定依据。

**3.2** **管理规范的效力**

本管理规则通过鲸鸿动能帮助中心公示，华为公司有权根据国家法律法规的变化及维护交易秩序、保护互联网用户权益需要，不定时修改本管理规范，并将在鲸鸿动能帮助中心通知或公示。

如投放方对已生效的修订管理规范不同意的，华为公司有权暂停为投放方提供投放服务；

如投放方在修订管理规范生效后仍继续使用华为公司服务，则视为投放方同意已生效的修订管理规范。

本管理规范未尽事宜，华为公司有权根据相关法律、法规、平台规则及具体合作协议的约定，给予相应违规认定及处理。

**3.3** **管理规范的实施日期**

本规范自发布之日起实施。
