# 教育

## 法律法规审核规范

<strong>教育、培训广告不得含有下列内容：</strong>

1.不得出现不符合考试制度及学制的宣传。

本科类：“一年制学本科”、“初高中也能拿本科”。

考研类：考研科目数量根据报考专业不同而数量不同，如出现“只考x门”描述，需明确具体专业。

2.不得明示或暗示有相关考试机构或其工作人员、考试命题人员参与教育、培训。

如：“高考命题组专家参与培训”。

3.不得利用科研单位、学术机构、教育机构、行业协会、专业人士、受益者的名义或者形象作推荐、证明。

如：“高考状元自述在校成长”——受益者推荐不允许使用。

4.不得对通过考试、升学、获得学位学历或合格证书，或者对教育、培训的效果作出明示或暗示的保证性承诺：

不得使用具体时间+具体效果描述；如：“1年拿本科证书”、“1个月拿本科”等。

不得对培训效果做出明示或暗示的保证性承诺；如：“毕业月薪过万”、 “不用上大学也能拿本科”、“免考”“免试”“免学”。

5.不对未来效果、收益或者与其相关的情况做出保证性承诺，不明示或者暗示保本、无风险或者保收益，广告不宣传“帮你解套”、“跟庄”、“涨停”、“稳定获利”等内容。

6.不得在中小学校、幼儿园内开展广告活动，不得利用中小学生和幼儿的教材、教辅材料、练习册、文具、教具、校服、校车等发布或者变相发布广告。（注：公益广告除外：如：直接或间接使用教育部官方教科书作为广告素材）。

7.不得使用借助国家、教育局法律法规进行宣传推广，如：“借助相关政策来宣传自己的课程项目”

1）不得模仿新闻消息、借用国家政策宣传自身服务，如：红头文件样式/教育部XX政策。

2）不得使用不符合实际的政策变化描述，如：考试报名即将截止/公考考试难度加大。

8.不得利用不满十周岁的未成年人作为广告代言人。

9.教育培训行业不得明确涉及“超前学/超纲学”

1）不支持学科类课程培训内容，如：同步教材/上课听不懂找XX。

2）不支持拍题搜题，如：教材XX页，题目详解。

## 平台审核规范

<strong>教育广告文案应当遵循以下规范：</strong>

1．不得有暗示双份收入、额外收入或双份工作的挂靠描述，如：“不坐班、每月增加额外收入”。

2．教育培训行业广告的落地页禁止出现加群、加好友等内容。

3．不得利用虚假政策或无法证实的内容误导用户，如“内部消息”。

4．使用“大学”、“学院”描述需注意：

1）广告内容涉及第三方院校名称或使用“大学”“学院”字样的做宣传，需补充校方授权后方可使用。

2）未获得国家认可的高等教育机构不得使用“大学”、“学院”名义。

附：全国高等学校名单查询链接： ``https://hudong.moe.gov.cn/qgcrmd/``；外国高等学校名单查询链接：``https://jsj.moe.gov.cn/n1/12018.shtml``。

5.不得在未提供具备金融资质的前提下使用通过金融手段进行招生的宣传如：“学费零首付”、“零利息”、“学费分期”等描述。

6．涉及学费分期等内容需提供有权从事相关业务的资质或者与银行的分期业务合作凭证。

学历教育不支持“0基础”描述。

7．语言培训类在使用外文作为文案时，不能使用小语种作为文案，若使用英文等语言作为文案，应需与中文翻译一起使用，且不能使用负面词汇。

8．不得使用易引人误解的模糊性师资描述，如：XX名师，可使用XX大学毕业名师。

9．不得利用卖惨，煽动悲情，焦虑营销等负面内容展开广告宣传 如：被同行针对，这是最后一期了/宝妈赚钱难。

10．国学课程推广内容不得涉及封建迷信相关内容，包括但不限于运势解读、暗示批命、改命、批卦、改变运势、占卜、预测未来、决定命运、做人生睁眼玩家、算卦等。

11．健康养生类课程不得暗示医疗/治疗效果，如：别再买成千上万的保养品了，来学站桩/中医术语：固本驻颜，矫正骨骼。

12．短视频/电商培训不支持使用非教育培训行业的描述，如：一手货源/对接厂家一手货源/扶持变现。

<strong>教育广告图片元素应当遵循以下规范：</strong>

1.不得出现消防员、武警等国家机关工作人员形象。

2.不得使用过于暴露、性暗示或突出敏感位置的人物形象。

3.不得使用“红领巾”素材。

4.广告内容中出现教师师资介绍，如涉及具体教师需提交该教师的教师资格证书。

5.不得直接或间接使用第三方院校或学校大门、校区、校徽、教学楼等标志性建筑，请删除或提供授权资质。

<strong>素材创意支持使用学历证书，使用学历证书作为创意时，还需满足以下条件：</strong>

1.不得使用证书内页。

2.不得使用国徽、教育部相关政府单位名称、院校名称、logo等元素的真实证书样式。
