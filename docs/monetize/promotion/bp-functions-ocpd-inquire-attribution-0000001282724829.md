# 查询归因报表

创建oCPD任务后，您可以在归因报表中查看您的归因转化报表数据。

## 操作步骤

1. 登录[华为应用市场应用推广平台](https://ads.huawei.com/cn/)。
2. 点击“工具”页签，在“资产管理”中选择“数据资产”，进入“数据资产”页面。

   ![](./img/0000000000011111111.20260212172440.04092087543822159330363304909715_50001231000000_2800_E5AAFCCF225A43F7B7EC8DC890583F0E47DF14E299528AF603FBC9651A8C1FBE_012232d00b1e.png "点击放大")
3. 在数据源“操作”中点击“详情”。

   ![](./img/0000000000011111111.20260212172440.10868016792407577442671391546985_50001231000000_2800_213E434FB0E6DE86547F1352FF51DABB8B6A30C2F7D1657308D17E3631898799_d6e3cb549fa4.png "点击放大")
4. 在“归因报表”中可以查看所有已选择归因目标的任务，包括CPD和oCPD任务以及任务的曝光、下载、出价、深度转化成本、注册、激活等数据。

   报表支持筛选时间周期、任务类型、计费类型、转化目标。
