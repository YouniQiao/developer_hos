# 2026年鲸鸿动能游戏行业激励政策

<strong>政策周期：</strong> <strong>2026</strong> <strong>年1月1日-2026年6月30日。</strong>

<strong>面向对象</strong>：游戏行业客户（仅限标准联运HMS游戏参加）。

<strong>参与条件：</strong>无需报名，默认符合规则的产品均可参与。

<strong>互斥政策：</strong>与年框政策、营销大促政策互斥。

<strong>激励内容 ：</strong>

<strong>1</strong> <strong>）IAA游戏：</strong>

|  |  |  |
| --- | --- | --- |
| <strong>月度投放消耗额X（万元）</strong> | <strong>现金消耗部分返货比例</strong> | <strong>时长达标返货比例</strong> |
| 10&lt;=X＜300 | 30% | 5% |
| 300&lt;=X | 40% |
| 1、通过鲸鸿动能竞价广告买量带来的IAA游戏新用户，对该用户在该IAA游戏内产生的广告变现收入分成比调整为华为：开发者=1:9；该分成比例持续生效365天，到期后恢复为基础分成。  2、若某游戏用户的平均月活跃时长<strong>&gt;=</strong>30min，该游戏投放现金部分可额外享受5%的返货激励。  3、该政策适用于全量华为终端用户。 | | |

<strong>2</strong> <strong>）IAP-小游戏：</strong>

|  |  |  |
| --- | --- | --- |
| <strong>月度投放消耗额X（万元）</strong> | <strong>现金消耗部分返货比例</strong> | <strong>礼包以及道具商城接入</strong> |
| 10&lt;=X＜100 | 25% | 5% |
| 100&lt;=X | 30% |
| 1、通过鲸鸿动能竞价广告买量带来的小游戏新用户，对该用户在该小游戏内产生的内购变现分成比调整为华为：开发者=2:8；该分成比例持续生效365天，到期后恢复为基础分成。  2、同时接入华为礼包以及道具商城的产品，投放现金部分可额外享受5%的返货激励。接入标准：  1）接入华为礼包能力，并至少有2个审批通过的正式礼包；  2）接入道具商城能力，并在小游戏内有商城入口露出。  3、该政策仅适用于HarmonyOS 4.x及以下版本，通过该政策获取的返货也仅能使用于HarmonyOS 4.x及以下版本的投放。 | | |

<strong>3</strong> <strong>）IAP-APK游戏：</strong>

|  |  |  |
| --- | --- | --- |
| <strong>档位</strong> | <strong>月度投放消耗额X（万元）</strong> | <strong>现金消耗部分返货比例</strong> |
| 第一档 | 10&lt;=X＜300 | 70% |
| 第二档 | 300&lt;=X＜600 | 80% |
| 第三档 | 600&lt;=X | 90% |
| 该政策仅适用于HarmonyOS 4.x及以下版本，通过该政策获取的返货也仅能使用于HarmonyOS 4.x及以下版本的投放。 | | |

<strong>激励发放形式和使用：</strong>

|  |  |  |
| --- | --- | --- |
| <strong>竞价投放</strong> | <strong>发放周期</strong> | <strong>有效期</strong> |
| 返利金充值 | 月度 | 180天 |

<strong>结算方式：</strong>

1. 仅有广告变现的游戏属于IAA游戏，有内购变现的属于IAP游戏。
2. IAP游戏按产品维度结算，IAA游戏按公司维度结算。月度投放消耗额（X）包括现金与返货消耗（返货消耗不含鸿飞、赠送金等），但只针对现金部分返货。
3. 结算周期以月度为单位（预约场景统计预约期投放总额累计至首发月统一结算），返货自下发之日起有效期180天。
4. 本活动仅针对鲸鸿动能竞价消耗，仅限鲸鸿动能DSP，综合DSP和直客DSP不适用本激励。
5. 账单和激励金额数据以鲸鸿动能数据为准。
6. 如发现不符合激励条件的情况，鲸鸿动能将对数据进行复核，并仅在符合激励条件的范围内发放激励，金额以复核调整后的为准。次月发放后，如果客户有异议，可以通过服务商发邮件向鲸鸿动能反馈问题，鲸鸿动能进行数据复核。
7. 如涉及数据差异的，数据差异在鲸鸿动能数据的10%以内时，以鲸鸿动能数据为准；如差异超过10%，请于五个工作日内发正式邮件反馈数据差异排查，双方友好协商一致确认最终数据；如未及时反馈，视为确认以鲸鸿动能数据为准。

<strong>补充说明：</strong>

1. 返利金由鲸鸿动能平台在推广产品达标后次月直接发放至客户账户，涉及多个账户的，按各账户现金消耗比例进行分摊返利金，原则上不接受调整。
2. 适用政策期间，推广产品及相关推广投放应保持合法合规运营，不得出现违反法律法规或侵犯第三方合法权益的情形，否则鲸鸿动能有权取消推广产品的政策适用资格并采取相关处罚措施。
3. 服务商、客户享受鲸鸿动能提供的返利金等权益，需以其账户处于有效且可操作状态为前提。因广告主或服务商相关账户注销等账户状态异常情形，致使鲸鸿动能客观上无法向其账户成功充值返利金或发放其他权益的，由此产生的返利金等无法发放的后果，由服务商或客户自行承担，并视为其默认放弃该等权益，鲸鸿动能将不再进行发放操作。
4. <strong>异常作弊、违规操作、恶意修改预算/出价、利用系统漏洞、滥用系统机制、可疑或非法交易等不当行为不可享受激励。经鲸鸿动能合理判定服务商或客户存在前述行为的，鲸鸿动能有权拒绝发放激励并追回已发放的激励，并采取相应处罚措施，由此造成的损失或赔偿责任由服务商或客户承担。</strong>
5. <strong>如因产品功能、计算错误、系统错误等任何原因导致激励超额发放、重复发放或错误发放，一经发现，鲸鸿动能有权追回多发部分或按核实后的应发激励调整发放。</strong>
6. <strong>对于应追回的激励金额，鲸鸿动能可按以下方式扣除或追缴：（1）从下期激励中扣除；（2）从服务商或客户激励账户中直接扣除；（3）从服务商或客户的充值账户等其他支付、结算账户中扣除；（4）从鲸鸿动能尚未向服务商或客户支付的各种激励、费用、分成、代收款等款项中直接扣除；（5）从服务商或客户在华为平台的其他激励、账户或待结算款项中扣除；（6）采取其他可行的方式要求服务商或客户在30个工作日内退回、支付或赔偿。</strong>
7. 如对本政策有任何疑问，请联系鲸鸿动能。
