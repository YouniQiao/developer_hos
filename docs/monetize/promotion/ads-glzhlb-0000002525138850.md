# 查询关联账户列表

【简介】查询Token关联账户列表信息

https://ads.cloud.huawei.com/openapi/v1/account/profile/query

<strong>请求方法</strong>

<strong>GET</strong>

<strong>请求参数</strong>

无

<strong>请求示例</strong>

GET openapi/v1/account/profile/query HTTP/1.1

Accept:application/json

Content-Type:application/json

Authorization:Bearer CgB6e3x9ERGComr9dENxZX22iBk+mLuf1yGtQVPUjPJUMrstfKlqpdXk+kfHU9J8ZJ/soYIZHZzT446GeSYumluQuhsK7jvz4kz1Bkms4CLI/rE=

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1[] | 返回数据 |

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| accountId | string | 账户ID |
