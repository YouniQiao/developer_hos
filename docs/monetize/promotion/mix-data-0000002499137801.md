# 投放端——报表

您可在页面左上方切换，分别查看应用市场应用推广或鲸鸿动能展示广告网络的报表数据。若您选择“应用市场应用推广”，支持查看的数据维度、数据明细等对齐现有应用推广的报表。账户所有存量数据，投放端整合升级后自动做数据迁移。
