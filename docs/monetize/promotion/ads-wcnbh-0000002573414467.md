# 未成年人保护规则专题解读

<strong>未成年人保护规则专题解读</strong>

本专题解读以下列法规、部门规章及通知为核心依据，覆盖广告全流程未成年人保护要求

《中华人民共和国广告法》（核心条款：第10、20、38、39、40条）

《互联网广告管理办法》（核心条款：第12条）

《未成年人网络保护条例》（2024年施行，核心条款：第10、15、17、20、21条）

《可能影响未成年人身心健康的网络信息分类办法》（2026年3月1日施行，核心条款：第3、4、5、6、9条）

补充依据：《中华人民共和国未成年人保护法》（网络保护专章）、《中华人民共和国个人信息保护法》（第31条）、国家市场监督管理总局相关监管要求等

<strong>相关案例仅用于展示审核规则，并不代表鲸鸿动能广告对此持认可态度</strong>

## 一、核心规则：坚守三大核心底线

所有互联网广告活动需遵循“最有利于未成年人”原则，坚守三大核心底线，贯穿广告策划、制作、投放、传播全流程；

底线一：不得损害未成年人身心健康、人格尊严，不得诱导未成年人实施不良行为、形成错误价值观；

底线二：严格限制面向未成年人的广告品类、内容形式及投放场景，杜绝违规营销行为；

底线三：尊重未成年人个人信息权益，规范信息收集、使用及广告推送行为，保障监护人监护权。

## 二、核心分类：四大类可能影响未成年人身心健康的网络信息

本办法（《可能影响未成年人身心健康的网络信息分类办法》）核心将相关信息划分为四大类，每类均明确具体情形，其中多条款直接关联广告创作、投放全流程，是广告素材审核的核心红线，具体如下：

<strong>（一）可能引发或者诱导未成年人模仿或者实施不良行为的信息</strong>

- 带有性暗示、性挑逗等易使人产生性联想的内容（包括不限于人物穿着、动作、神情，画面场景和文字描述等）；

- 存在指责嘲讽、贬低歧视等涉网络暴力的不良信息（广告内容不得包含此类内容或导向，避免引导未成年人参与网络暴力）；
- 煽动人群歧视、地域歧视、对立冲突等行为的内容（广告需传递正向价值观，不得引发未成年人对立情绪）；
- 通过刻意刺激、恶意引导等方式，引发未成年人产生过度强烈或者持久的愤怒、恐惧、抑郁等极端情绪的内容（广告不得使用恐吓、焦虑式话术，避免伤害未成年人心理）；
- 存在不安全驾驶等危险动作，诱导模仿高风险行为、进入危险区域等可能损害身体健康行为的内容（广告不得展示危险动作、危险场景，避免未成年人模仿，与《广告法》第四十条要求一致）；

<strong>（二）可能对未成年人价值观造成负面影响的信息</strong>

此类信息直接影响未成年人价值观塑造，广告需传递正向内容，具体限制包括但不限于以下内容：

- 宣扬漠视生命、自我贬损等观念的内容（广告需传递尊重生命、积极向上的理念）；
- 宣扬奢靡享乐、炫富拜金、消极颓废等不良价值观的内容（广告不得诱导未成年人攀比、炫富或宣导消极颓废等）；

|  |  |  |
| --- | --- | --- |
| <strong>违规素材</strong> | <strong>违规解读</strong> | <strong>修改要点</strong> |
|  | - 消极导向内容违规   素材内容不得宣扬消极、颓废、躺平等不良价值观内容，不利于未成年人身心健康发展；左侧素材涉及负面情绪、躺平等内容，不支持投放 | 替换推广内容，建议推广阳光、积极向上素材内容 |

- 宣扬畸形审美、低俗恶俗文化的内容（广告画面、形象设计需符合未成年人审美，避免成人化、低俗化、恶俗化）；
- 宣扬荒诞离奇、危言耸听等伪科学内容的内容（教育、科普类广告需坚守科学原则，不得传播伪科学）；
- 宣扬不良交友、恋爱观的内容（面向未成年人的广告不得涉及不当交友、早恋引导内容）；
- 宣扬“读书无用论”“唯分数论”“唯升学论”等观念的内容（教育类广告需规避此类表述，避免制造教育焦虑）；

- 其他宣扬违反公序良俗和社会公德的不良价值导向的内容（广告需符合社会公序良俗，传递正向价值观）。

<strong>（三）不当使用未成年人形象的信息（广告代言、出镜形象核心约束）</strong>

依据《广告法》第三十三、三十八条关于未成年人代言相关法条，广告代言人限制包括但不限于以下场景：

- 不得利用不满十周岁的未成年人作为广告代言人（含所有广告形式）。
- 使用未成年人（10周岁及以上）的名义或形象拍摄广告，需事先征得其监护人书面同意；16-18周岁未成年人参与直播营销、广告代言，必须取得监护人同意。

|  |  |  |
| --- | --- | --- |
| <strong>违规素材</strong> | <strong>违规解读</strong> | <strong>修改要点</strong> |
|  | - 未成年人代言场景违规   任何商品、服务均不得利用不满十周岁的未成年人作为广告代言人。广告中使用未成年人作为代言人的，则首先需要核查实该未成年人是否已经年满十周岁（未满十周岁禁止广告代言），其次，需要核查实该未成年人及其法定监护人的授权书。 | 建议使用成年人代言人作为广告素材进行广告宣传 |

《可能影响未成年人身心健康的网络信息分类办法》中第五条关于未成年人形象法条，不当使用未成年人形象限制包括但不限于以下场景：

- 利用未成年人声线传播不良内容（广告配音需规范，不得使用未成年人声线传递不良信息）；
- 通过恶搞未成年人、利用未成年人打造争议人设等方式博取关注（广告不得“啃小”炒作，不得恶搞未成年人形象）；
- 利用未成年人形象摆拍演绎含有不良价值观或者不当言行的剧情内容（广告中未成年人出镜需传递正向行为，不得演绎不良剧情）；

- 以不良方式或者目的对未成年人进行品行、道德方面测试（广告不得设计此类测试内容，避免误导未成年人）；
- 歪曲或者炒作涉未成年人违法犯罪行为（广告不得利用此类事件营销、炒作，避免传递不良导向）；
- 其他不当披露和使用未成年人形象的内容（广告使用未成年人不当的着装、镜头与画面）。

<strong>（四）未成年人个人信息保护规则（广告信息收集核心约束）</strong>

此类信息与《个人信息保护法》第三十一条衔接，规范广告中未成年人个人信息的收集与使用，具体包括但不限于：

- 未经监护人同意不当展示不满十四周岁的未成年人学习、生活等可能暴露未成年人个人信息的内容（广告不得随意展示未成年人的学校、家庭住址、照片等个人信息）；
- 诱导未成年人发布可能泄露本人或者他人个人信息的内容（广告不得诱导未成年人留资、发布个人信息）；
- 其他不当披露和使用未成年人个人信息的内容（广告收集未成年人信息需遵循“最小必要+监护人同意”原则，不得用于广告精准投放、数据共享）。
- 不得将未成年人个人信息用于广告画像、精准投放、数据共享或售卖，平台需提供未成年人广告相关数据的查询、删除功能，保障监护人控制权。

## 三、广告行业重点关注的义务要求（全流程合规要点）

广告投放中合规要求：

- 依据《广告法》第40条、《互联网广告管理办法》第12条：在针对未成年人的网站、网页、互联网应用程序、公众号等媒介上，严禁发布医疗、药品、保健食品、特殊医学用途配方食品、医疗器械、化妆品、酒类、美容广告，以及不利于未成年人身心健康的网络游戏广告。
- 依据《可能影响未成年人身心健康的网络信息分类办法》第9条：不得在专门以未成年人为服务对象的网络产品和服务中，发布可能影响未成年人身心健康的广告信息。

- 依据《可能影响未成年人身心健康的网络信息分类办法》第9条：不得在首页首屏、弹窗、热搜、榜单、推荐等醒目位置，呈现可能影响未成年人身心健康的广告。
