# 业务介绍

华为应用市场面向所有实名认证的企业开发者开放运营活动管理功能，开发者可自助创建运营活动，通过审核后即可在华为应用市场（含儿童中心、教育中心、游戏中心）正式上线，实现拉新、促活、创收的经营目标。

运营活动将根据福利力度及目的配置不同资源位，例：应用市场客户端首页的【福利】专区、应用页【探索】专区、应用详情页、游戏中心【福利】专区、应用内浮标、应用内公告等。
