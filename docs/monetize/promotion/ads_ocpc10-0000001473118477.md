# oCPC投放策略

## 功能简介

oCPC支持“稳定拿量、优先跑量、优先低成本”3种不同的投放策略，满足广告主对“成本”和“跑量”的不同诉求，投放策略默认为“稳定拿量”。

- <strong>稳定拿量：</strong>追求跑量稳定，转化成本尽量接近转化出价，适用于希望转化成本稳定且贴近期望出价的广告主。
- <strong>优先跑量：</strong>追求跑量优先，尽快地多获取转化，实际成本可能会略高于目标成本，适用于预算充足，接受成本正常波动、或者活动/促销期间，需要尽可能快地获取更多转化的广告主。
- <strong>优先低成本：</strong>追求成本控制，优先获取低成本流量，跑量速度可能受限，适用于预算有限，对成本波动较为敏感的广告主。
- <strong>最大转化：</strong>追求最大化跑量，系统会结合历史投放数据与实时竞价环境，由低到高自动探索出价，在投放周期内尽量均匀消耗完预算的前提下，为您争取尽可能多的转化量。

![](./img/note_3.0-zh-cn_f95d0fe122d1.png) 

1. 如您需要选择最大转化投放策略，可联系行业运营进行申请。
2. 计划日预算必须限额且小于等于账户日预算，投放策略才可以使用最大转化。
3. 考虑到不同客户对成本的接受程度不同，“转化单价上限”为可选项，若录入“转化单价上限”，系统则以不超过此转化单价作为优化目标；若未填，则在预算内自动探索最优。

## 操作步骤

1. 创建任务时，选择竞价目标为“转化”，付费方式为“oCPC”。
2. 选择<strong>投放策略</strong>，根据实际需求输入“转化目标”、“期望转化成本”等参数创建任务。
