# 创建创意

【简介】通过此接口可以创建创意。

<strong>请求地址</strong>

https://ads.cloud.huawei.com/ads/v1/promotion/creative/create

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| adgroup\_id | long | 是 | 任务ID |
| creative\_name | string | 是 | 创意名称，最大长度不超过100，支持名称里含有\n符号， 同一个广告主同一任务不能重复 |
| dyn\_ad\_text\_flag | string | 否 | 动态词包标识  限制条件  creative\_size\_sub\_type !=APP\_ICON  详见[【动态词包标识】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section36771238204317)创意文案、摘要创意元素支持动态词包拼接：比如日期地点性别动态拼接 |
| content\_struct | Struct3 | 是 | 创意详情 |
| creative\_size\_sub\_type | string | 否 | 版位子形式  详见[【版位子样式】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section459185654519)  全域智投版位，此字段非必填；非全域智投版位，此字段必填 |
| creative\_size | string | 否 | 尺寸  全域智投版位，此字段非必填；非全域智投版位，此字段必填 |
| creative\_mode | string | 否 | 创意模式  当campaign\_type=CAMPAIGN\_TYPE\_SHOPPING必填  详见[【创意模式】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section34361428379)  通过[【查询商品库】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_new_api50_dpa-0000001456586201)接口响应中的creative\_mode字段进行取值。查询到creative\_mode的与创建创意中请求参数creative\_mode的映射关系：   1. TEMPLATE -&gt; TEMPLATE\_MODE 2. DIRECT -&gt; DIRECT\_INVESTMENT\_MODE 3. TEMPLATE\_AND\_DIRECT -&gt; TEMPLATE\_MODE或DIRECT\_INVESTMENT\_MODE |
| element\_group | Struct16 | 是 | 全域智投元素组信息(此字段不为空时，同层级 content\_struct 字段内元素内容不生效) |

content\_struct (Struct3)定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| images | Struct4[] | 否 | 图片列表信息 最多十张  根据版位元素creative\_size\_element\_name为image的规则填写（约束见[查询版位元素](https://developer.huawei.com/consumer/cn/doc/promotion/ads_new_api25-0000001455600233)）  当campaign\_type=CAMPAIGN\_TYPE\_SHOPPING时不填写 |
| icon | Struct4 | 否 | icon信息  根据版位元素creative\_size\_element\_name为icon的规则填写 |
| video | Struct4 | 否 | 视频信息  根据版位元素creative\_size\_element\_name为video的规则填写 |
| title | Struct5 | 否 | 文案  根据版位元素creative\_size\_element\_name为title的规则填写如果使用动态词包  取值：  \\{地点\\}\\{日期\\}\\{性别\\}  商品广告额外支持动态词包如下：  \\{广告文案\\}\\{商品描述\\}\\{商品名称\\}\\{商品原价\\}\\{商品现价\\}\\{商品折扣\\}\\{一级分类名称\\}\\{二级分类名称\\}\\{三级分类名称\\}\\{品牌名称\\} |
| description | Struct5 | 否 | 描述  根据版位元素creative\_size\_element\_name为description的规则填写  如果使用动态词包，取值：\\{地点\\}\\{日期\\}\\{性别\\}  商品广告额外支持动态词包如下：  \\{广告文案\\}\\{商品描述\\}\\{商品名称\\}\\{商品原价\\}\\{商品现价\\}\\{商品折扣\\}\\{一级分类名称\\}\\{二级分类名称\\}\\{三级分类名称\\}\\{品牌名称\\} |
| corporate | Struct5 | 否 | 品牌名称  根据版位元素creative\_size\_element\_name为corporate\_name的规则填写，支持通配符填写，取值\\{brandName\\} |
| landing\_page | Struct6 | 否 | 落地页信息  根据版位元素creative\_size\_element\_name为landing\_page\_url的规则填写  支持通配符，由运营配置项控制，目前取值如下：  \\{targetUrlMobile\\}移动端H5商品落地页  \\{brandUrlMobile\\}移动端H5品牌落地页  任务中的推广产品为PROMOTION时，必填  计划中推广类型为微信小程序，广告主在白名单内可选填落地页 |

| ad\_button\_text | Struct10 | 否 | 按钮文案  根据版位元素creative\_size\_element\_name为ad\_button\_text的规则填写  （约束和取值见[【查询版位元素】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_new_api25-0000001455600233)） |

| industry\_labels | Struct12[] | 是 | 行业标签  [查询创意行业元数据](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260529160208.76233252273622099950931817654788:20260531101342:2800:3AD3B936884A8426168A7127A963A03959316A11D4FA8D26DC049691DA4F50B3.xlsx?needInitFileName=true)  最多3个. |
| template | Struct13 | 是 | 创意模板ID  campaign\_type=CAMPAIGN\_TYPE\_SHOPPING，  取值详见[【查询商品库动态模板】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_new_api59-0000001456746005) |
| splash\_txt | Struct14 | 否 | 开屏提示文案 |
