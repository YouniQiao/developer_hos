# 概述

为确保您在鲸鸿动能广告平台（以下简称 “平台”）推广的内容严格遵守国家法律法规及相关政策要求，保障广大用户的合法权益，维护平台合规有序的运营环境，平台基于法律法规要求和平台体验要求制定了广告审核规范，具体要求如下：

<strong>1. 合法性：</strong>推广内容需符合符合《广告法》等国家相关[法律法规规范](/docs/monetize/promotion/ads_shenhe32-0000001055095868)，不得存在任何违法违规情形。

<strong>2. 资质合规：</strong>推广的产品或服务不属于[禁投内容](https://developer.huawei.com/consumer/cn/doc/promotion/ads-jtnr-0000001057745564)，且广告主在开户环节已按平台要求提供完整、真实、有效的资质文件，满足平台[资质审核规则](https://developer.huawei.com/consumer/cn/doc/promotion/ads-zzgz-0000002473340168)。

<strong>3. 内容合规：</strong>广告创意（含标题、文案、视觉素材、音频内容等）需遵循[平台审核规范](https://developer.huawei.com/consumer/cn/doc/promotion/ads-pttyshgz-0000002505500131)，不得出现违规表述或违规元素。

<strong>4. 行业适配：</strong>广告主在投放广告时广告创意需满足其所属行业的[特定行业审核规范](https://developer.huawei.com/consumer/cn/doc/promotion/ads-hygz-0000002473340172)。

<strong>5. 版位适配：</strong>广告主在投放广告时广告创意需符合[版位规则](https://developer.huawei.com/consumer/cn/doc/promotion/ads-bwgz-0000002505500133)（如开屏、智慧屏开机的设计规范与内容要求）。

以上审核规范适用于广告创意中的所有元素，包括标题、文案、素材及落地页等，均需满足相应要求。

特别提示：您在平台投放时，应当遵守以上规范，经发现存在违反法律法规或以上规范时，依照[平台投放违规管理规范](/docs/monetize/promotion/ads-toufanguangliguifang-0000001793450445)的相关规定，对投放方处理或处罚。
