# 更新人群包内容

【简介】合作伙伴更新人群包,

人群包内容变化时可通过该接口更新。

特殊说明：增量更新可能存在失败情况，建议根据实际情况固定周期全量同步一次。

<strong>请求地址</strong>

https://svc-drcn.ads.huawei.com/opendmp/v1/audience\_package/update

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>长度</strong> | <strong>必填</strong> | <strong>说明</strong> |
| dmpID | String | 255 | Y | 三方DMP标识。 |
| audienceID | String | 128 | Y | 人群ID |
| addUrl | AudiencePackage | - | N | 新增人群信息包下载URL。  <strong>注意1：当updateType为1时，addUrl是必须的。</strong>  <strong>注意2：</strong> <strong>addUrl</strong> <strong>和</strong> <strong>deleteUrl</strong> <strong>不可同时使用</strong> |
| deleteUrl | AudiencePackage | - | N | 删除人群信息包的下载地址。  <strong>注意1：当updateType为1时，不能填写deleteUrl。</strong>  <strong>注意2：</strong> <strong>addUrl</strong> <strong>和</strong> <strong>deleteUrl</strong> <strong>不可同时使用</strong> |
| updateType | int | 1 | N | 更新类型。  0：增量更新（默认）  1：完全更新。  <strong>建议：定期进行全量同步（</strong> <strong>updateType=1</strong> <strong>），避免增量同步导致数据不一致。</strong> |

<strong>AudiencePackage</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>Parameter name</strong> | <strong>Type</strong> | <strong>Length</strong> | <strong>Mandatory</strong> | <strong>Description</strong> |
| locationType | int | 1 | N | 固定传1  1-华为OBS |
| downloadUrls | List&lt;String&gt; | - | Yes | 人群包地址，  每个人群不超过100个文件路径。  国内传：objectID（即DMPID/文件名称.zip） |
| deviceIDType | int | - | No | 人群包中的设备ID类型 |

<strong>请求示例</strong>

POST http://$\{base\_address\}/opendmp/v1/audience\_package/update

HTTP/1.1

Headers:

Content-Type: application/json

Authorization：Bearer DAEAAIX7IS\*\*\*\*\*Pd81PDmWiM1KS2

Body:

```
{
    "dmpID": "829411234539267456", // Ads运营提供的dmpID
    "audienceID": "联调测试test001", //人群特殊标志符，
    "addUrl": {
                "locationType": "1", //固定传1
                "downloadUrls": ["829411234539267456/A.zip"], //上传至OBS文件路径
                "deviceIDType": "3" //人群包中的设备ID类型
            },
    "deleteUrl": {
                "locationType": "1",//固定传1
                "downloadUrls": ["829411234539267456/B.zip"], //上传至OBS文件路径
                "deviceIDType": "3" //人群包中的设备ID类型
            },
    "updateType": "0" //更新方式：0-增量更新；1-全量更新
}
```

<strong>响应字段</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 参数名 | 类型 | 长度 | 是否必填 | 说明 |
| code | int | - | Y | 错误码 |
| message | String | 2048 | N | 描述 |
| value | object | - | N | NA |
