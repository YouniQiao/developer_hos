# 常见问题

<strong>Q1</strong> <strong>：当前oCPC</strong> <strong>使用是否存在门槛？</strong>

<strong>A：</strong>使用前需要接入[转化跟踪](/docs/monetize/promotion/ocpc-0000001310577317#ZH-CN_TOPIC_0000001310577317__table10977121311584)。

<strong>Q2</strong> <strong>：oCPC</strong> <strong>接入后会经历几个阶段，怎样才算度过了学习期？</strong>

<strong>A：</strong>oCPC广告接入后需要先进入学习期，学习期按照CPC出价，以激活为例，要求7天内累积达到30个转化量。学习成功进入智能出价阶段，学习失败将维持CPC出价。

<strong>Q3</strong> <strong>：oCPC</strong> <strong>学习完成期该如何出价？</strong>

<strong>A：</strong>按照实际目标转化成本设置即可，系统在保证实时成本的同时，会控制广告全生命周期成本与目标转化成本保持一致，所以不建议您频繁的调整出价。例如：如果广告学习完成后，您将目标成本由10元/转化改为8元/转化，系统在保证实时成本调整至8元/转化后，还会调整“学习期、学习完成期调价前、学习完成期调价后”的历史累计成本至8元。

<strong>Q4</strong> <strong>：学习期跑不出量怎么办？</strong>

<strong>A：</strong>学习期跑量逻辑和CPC出价完全一样，ECPM竞价能力差，可能的原因有以下几点：

（1） 定向过窄，展现量较少，可适当增加目标人群覆盖范围；

（2） 出价过低，竞价成功率低曝光较少，可适当提高点击出价；

（3） 创意较差，创意点击率低，选择投放高点击率、质量好的素材；

（4） 总预算少且较为分散，每个计划/任务预算均分较少，导致曝光量均不高，可以合理提高预算，或较为集中于几个把握较大的任务。

<strong>Q5</strong> <strong>：学习完成期跑不出量怎么办？</strong>

<strong>A：</strong>学习完成期竞价排序主要与转化出价、预估点击率、预估转化率相关，可以适当提高转化出价，优化点击率和转化率（优化广告创意、落地页），稳定后逐步放开定向获得更多流量。

<strong>Q6</strong> <strong>：学习期点击成本偏高怎么办？</strong>

<strong>A：</strong>优化逻辑与CPC相同，若定向设置不合理，导致点击率较低，可按照CPC优化方向圈选目标人群；若创意质量较差，可尝试更换更符合目标人群的物料。

<strong>Q7</strong> <strong>：实际转化成本偏高怎么办？</strong>

<strong>A：</strong>通常转化成本与期望成本会保持一致。单天超成本可能的原因有：

（1）由于回传延迟原因，当天转化成本可能会显示偏高，建议耐心等待转化数据回传完毕；

（2）系统模型学习期间，按照CPC方式出价，成本存在一定波动，建议保持投放，出学习期耐心观察；

（3）频繁/大幅调整出价，每次调整出价系统会重新学习，控制转化成本与新出价一致。当调整出价幅度或频次过多，实际成本会出现一定波动。

建议每次调整出价周期5-7天为宜，持续耐心观察，若曝光量有所增长，可以保持投放；若曝光量不足，建议停投并重新创建广告任务。

<strong>Q</strong> <strong>8：oCPC</strong> <strong>素材和定向能否与之前在投的CPC</strong> <strong>任务相同？</strong>

<strong>A：</strong>可以。oCPC任务与其他所有展示广告计费方式任务实行公平竞价，所以可能存在同时竞价的情况。

<strong>Q9</strong> <strong>：</strong> <strong>申请开通了oCPC权限，是否适用于所有转化目标？</strong>

<strong>A：</strong>您好，oCPC不适用所有转化目标。oCPC当前支持的转化指标为：激活、激活（HMS）、唤醒、注册、付费，激活次留双出价。如果您想使用以下指标：唤醒、注册、付费，激活次留双出价，需要申请特性通行名单。详情请参考：[oCPC](/docs/monetize/promotion/ocpc-0000001310577317)内容。

<strong>Q10：</strong> <strong>申请开通了oCPC权限，是否适用于所有版位？</strong>

<strong>A：</strong>您好，oCPC只适用于展示广告网络大部分版位。

<strong>Q11</strong> <strong>：oCPC</strong> <strong>广告可以修改落地页/</strong> <strong>创意吗？</strong>

<strong>A：</strong>在学习期建议不要修改，落地页/创意都是训练模型的特征，修改会使模型更难渡过学习期。

oCPC各类操作规范如下：

| 操作 | 学习期 | 学习完成期 |
| --- | --- | --- |
| 广告创意修改 | 不建议修改 | 稳定后可适当优化创意 |
| 落地页变动 | 不建议修改 | 稳定后可适当优化调整 |
| 定向调整 | 不建议修改 | 稳定后可逐步放开定向 |
| 出价调整 | 不建议修改 | 稳定后可适当调整转化成本 |
| 预算调整 | 不建议修改 | 稳定后可适当提高预算扩量 |

<strong>Q12</strong> <strong>：</strong> <strong>满足条件的CPC任务一键切换成</strong> <strong>oCPC</strong> <strong>任务后，点击出价能改吗？</strong>

<strong>A：</strong>不能修改，切换后的oCPC学习期出价自动填充当前的点击出价。
