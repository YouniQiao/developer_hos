# 认证服务商

如果您想查看非中国大陆地区业务已认证的服务商信息，单击此[链接](https://ads.huawei.com/usermgtportal/home/index.html#/partners)页面，语言选择英文或者俄文，选择“Partners”-&gt;“Find a partner”进行查看。

选择对应区域的服务商，进行查看。
