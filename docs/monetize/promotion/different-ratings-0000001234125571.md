## 鲸鸿动能广告分级审核规范

| 分级类别 | 分级详细说明 | 相关行业 | 审核规范 |
| --- | --- | --- | --- |
| W、PI、J | - 广告内容不得具有欺骗性、对目标受众群体不公或不适合其观看。 - 不得包含任何暴力血腥、恐怖惊悚、不良用语、暧昧场景等受限制内容，以及不利于未成年人身心健康的内容。 | 宠物 | 不允许出现生活场景中不常见的、且形象特殊易造成用户不适的动物形象（蛇、蜥蜴等）。 |
| 玩具 | 1、儿童相关的产品，全部需要CE认证标志，需要提供年龄警示信息。  2、玩具不得存在小于一定尺寸的易拆卸部件，以避免儿童将其误入口中造成噎塞和窒息危险。  3、不允许出现生活场景中不常见的、且形象特殊易造成用户不适的形象（蛇、可怕小丑等）。 |
| 汽车/机动车 | 不允许出现赛车、酒驾等危险行为。 |
| 房地产 | 不允许出现高空作业等危险行为。 |
| 教育培训 | 1、不允许煽动未成年人要求父母/成年人为其购买此课程。  2、以未成年人为服务对象的在线教育网络产品和服务，不允许推送与教学无关的内容。 |
| 游戏、视频、阅读 | 1、不允许出现血腥暴力（包括卡通暴力）和可能惊吓到未成年人的声音或场景（巨兽打斗、相互撕咬、肢体撕裂、咬食等）。  2、不允许使用消极、丑陋、恐怖、灵异等可能引起用户不适或反感的元素 （僵尸、骷髅头、尸骨等）。  3、不允许出现恶搞类人物形象、表情狰狞、美观度差的画面（如鬼怪面具、惊恐的万圣节装扮等）。  4、不允许宣传烟酒、抽烟、饮酒等行为，包含真人及卡通形象。  5、不允许出现性感裸露、着装暧昧的女性形象（不得穿着情趣内衣、肤色装、透视装、真空凸点、深V领、透视蕾丝装等刻意突出敏感部位的紧身衣，不允许穿着吊带袜、裸露内衣等衣衫不整的造型）。  6、不允许带有性暗示、性挑逗或其他低级趣味的行为，如仅用被子、浴巾、衣物等半裸露身体，背景不得为卫生间、浴室、卧室、床等敏感场景。  7、不允许出现粗话、性暗示、性联想等不适宜的语言，如go to hell、damn、shit、on fire、sleepless nights等。  8、不允许出现突出敏感部位的真人展示的服装图片或健身场景，如突出翘臀的热裤、裸露胸部的运动内衣等。  9、不允许宣传第一人称射击游戏和战争游戏：用户以第一人称视角射击其他人物的游戏、涉及在战争或战斗中控制军队或斗士的游戏。  10、不允许出现带有不良诱导的描述，如做黑客、刺客等，或沉迷游戏等描述。  11、不允许推送医疗健康、赌博博彩（包括免费竞赛和抽奖、赌场游戏）等不利于其身心健康的广告。  12、不允许宣传危险行为、诱导购买、煽动情绪、损坏长辈权威等广告。  13、不允许展示不符合该年龄段的、不安全的场景/物品/信息。  14、不允许宣传、推送付费服务。  15、不允许推送任何类型的政治广告，包括与政治候选人、政治行动委员会或其政策立场相关的信息。  16、不允许推送社会负面新闻，如车祸、火灾、疫情、自然灾害等。  17、不允许宣传涉及拳击、摔跤、狩猎及彩弹射击等危险行为。  18、不允许宣传涉及销售喷漆、气溶胶涂料、玻璃蚀刻材质、涂鸦产品等有关内容。  19、广告内容涉及未成年人代言，需提供未成年人及法定监护人的授权书。 |
| A | 仅适用于成人用户，但不得含有性暗示内容（例如两性笑话、裸露内容等）、虚拟赌博，不得含有激烈的暴力内容（如血腥、残肢、严刑拷打等场景）、惊悚/恐怖题材内容，也不得含有提及烟草/饮酒的内容 | 医疗、理财、购物、直播、恋爱交友、教育、汽车、房地产等 | 详见《[通用审核规范](https://developer.huawei.com/consumer/cn/doc/promotion/general-review-regulations-0000001189084544)》。 |
