# 管理素材库

素材库内素材支持删除、修改名称、预览等操作，请参考如下操作进行管理。

- 点击“缩略图”中的图片，可进行素材预览，查看已创建素材库中的图片。
- 点击![](./img/zh-cn_image_0000001399646153_b66def3406ba.png)可重新编辑素材名称。
- 点击“操作”列的“删除”，删除您已创建的素材。
- 在“批量操作”下拉框中，点击“批量删除”，可批量删除已创建的素材。
