# 概述

合约广告的目标是提升您企业产品与服务的知名度，并使用户产生积极的品牌联想。您可以选择的合约广告形式有：

- [应用市场合约广告](/docs/monetize/promotion/contract-ad-shown-on-ag-0000001294223968)：约定在某一时间段内、在某些广告位上固定投送该广告主的广告，无需竞价即可展示。
