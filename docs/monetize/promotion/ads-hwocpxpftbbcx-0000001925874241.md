# ocpx赔付天报表查询

您通过本接口通过广告主ID、任务ID查询Ocpx赔付天报表数据。

- <strong>请求地址</strong>：请根据您的就近区域进行选择

  <strong>亚非拉：</strong>https://ads-dra.cloud.huawei.com/openapi/v2/reports/ocpx/queryDailyCompensation

  <strong>俄罗斯：</strong>https://ads-drru.cloud.huawei.com/openapi/v2/reports/ocpx/queryDailyCompensation

  <strong>欧洲：</strong>https://ads-dre.cloud.huawei.com/openapi/v2/reports/ocpx/queryDailyCompensation
- <strong>请求方法</strong>：<strong>POST</strong>
- <strong>请求参数</strong>：

  | <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
  | --- | --- | --- | --- |
  | is\_abroad | Boolean | 否 | 是否为海外广告主. |
  | advertiser\_id | String | 否 | 广告主ID，经理账户或者服务商账户登录时，则此字段必填；当前操作的华为账号支持多个广告主账户时，此字段必填。 |
  | adgroup\_id\_list | list&lt;string&gt; | 否 | 任务ID List |
  | page | integer | 否 | 页数，默认值1，从1开始 |
  | page\_size | integer | 否 | 期望返回页大小。默认20，取值范围1-10000 |
  | ocpx\_event\_types | list&lt;OcpxEventType&gt; | 否 | OCPX任务转化目标类型 |
  | compensation\_status | list&lt;string&gt; | 否 | 保障状态:  COMPENSATING-保障中，  NOT\_MEET\_COMPENSATING-不满足保障，  NO\_NEED\_COMPENSATION-保障结束无需赔付，  NEED\_COMPENSATION-保障金额统计完成，  COMPENSATION\_CHECKING-保障核实中，  FREE\_OF\_COMPENSATION-不涉及 |
  | compensation\_periods | list&lt;integer&gt; | 否 | 保障周期，1-周期一，2-周期二 |
  | start\_time | string | 是 | 数据周期时间起始时间，北京时间  YYYY-MM-DD hh:mm:ss |
  | end\_time | string | 是 | 数据周期时间结束时间，北京时间  YYYY-MM-DD hh:mm:ss |

  <strong>ocpx\_event\_types (OcpxEventType)</strong> <strong>参数：</strong>

| <strong>参数名称</strong> | 类型 | 是否必选 | 描述 |
| --- | --- | --- | --- |
| shallow\_event\_type | string | 否 | [浅层转化目标](/docs/monetize/promotion/marketing-api-appendix1-0000001174597591#section125615641916) |
| deep\_event\_type | string | 否 | [深层转化目标](/docs/monetize/promotion/marketing-api-appendix1-0000001174597591#section1042612114174) |
| deep\_optimize\_type | integer | 否 | 深度优化类型  1-双出价；  2-双阶段；  3-单目标加强点 |

- <strong>响应参数</strong>

  | <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
  | --- | --- | --- |
  | result\_code | integer | 0：查询成功  1：起始索引超出查询结果范围  其他：未知失败 |
  | desc | string | 结果描述 |
  | total\_num | integer | 满足查询条件总条数 |
  | page\_size | integer | 每页大小 |
  | page | integer | 当前页 |
  | total\_page | integer | 总页数 |
  | summary\_data | OcpxCompensation | 汇总数据 |
  | data | list&lt;OcpxCompensation&gt; | 查询结果 |
