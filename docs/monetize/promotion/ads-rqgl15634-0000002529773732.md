# 删除人群

【简介】合作伙伴删除人群信息列表

若人群包被现有任务引用，则不可删除。

<strong>请求地址</strong>

https://svc-drcn.ads.huawei.com/opendmp/v2/audience/delete

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>长度</strong> | <strong>必填</strong> | <strong>说明</strong> |
| dmpID | String | 255 | Y | 三方DMP标识。 |
| audienceIDs | List&lt;String&gt; | - | Y | 人群包id列表。  最长100个。 |

<strong>请求示例</strong>

POST http://$\\{base\_address\\}/opendmp/v2/audience/delete

HTTP/1.1

Headers:

Content-Type: application/json

Authorization：Bearer DAEAAIX7IS\*\*\*\*\*Pd81PDmWiM1KS2

Body:

```
{
    "dmpID": " 829411234539267456",
    "audienceIDs": ["联调测试test001"]  //人群特殊标志符
}
```

<strong>响应字段</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 参数名 | 类型 | 长度 | 是否必填 | 说明 |
| code | int | - | Y | 错误码 |
| message | String | 2048 | N | 描述 |
| value | object | - | N | NA |
