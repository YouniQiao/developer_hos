# FAQ

<strong>1、什么是“预装”？</strong>

应用在终端设备出厂前被预先集成到ROM中，随设备发货的应用分发方式。终端设备包含但不限于手机、平板、可穿戴设备、家庭网络终端、M2M终端等。

<strong>2、预装的前提条件是怎样的？</strong>

（1）应用由在华为开发者联盟实名认证的企业开发者提供；

（2）应用在市场排行所属应用类别中，活跃用户数等维度排行位于TOP5之内；

（3）在国内各大应用市场排行中，持续保持领先的地位，并在该行业内有较好的口碑；

（4）开发者提供相关数据、资质或材料，华为预装团队从功能、体验、性能、受众群等多个维度对其进行综合测评，得到应用竞争力指数排名，由多方共同决策是否入围；

（5）与华为签署预装协议。

<strong>3、预装的要求是怎样的？</strong>

（1）付费预装：

一般情况下，经过双方商务沟通，会按照CPP、CPA等双方协商一致的结算方式进行预装合作；同时，需在签署合同后支付保证金，预装了APP的产品发货后，双方根据华为财务部门提供的发货量按月度进行结算；

（2）安全要求：

首先，所有预装APP都要经过货架化测试流程，该测试过程，包含了华为安全红线的相关要求和用例 ；

同时，预装APP须遵从工信部安全规定和要求，APP须在工信部备案通过；最后，双方签署的预装合同中，需包含安全条款和风险的责任方；

（3）质量要求：

预装APP需经过华为货架化测试，满足功能、性能、功耗、安全等多方面的测试，开发者须积极配合华为测试团队完成各项测试。通过货架测试的应用，方可安排预装。

<strong>4、预装应用和华为应用市场应用在同一个后台，二者是否会有冲突？</strong>

不会产生冲突，他们是两个不同的分发渠道，开发者可以使用不同的渠道包进行上传和分发，互不干扰。
