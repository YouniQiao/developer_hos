# 概述

如果您希望提升您应用的活跃度，让更多的用户使用您的应用，或者您希望对应用内的某个活动进行推广，可以使用鲸鸿动能广告提供的应用促活广告。

应用促活广告只会被展示给已经安装了您应用的用户，您需要提供一个指向应用内内容的[Deeplink链接](/docs/monetize/promotion/overview-cjjjgg-0000001182873508#section19208102614323)，用户点击广告后会进入到您提供的Deeplink链接上。

![](./img/note_3.0-zh-cn_545b122ccc34.png) 

根据安全规范要求，在投放应用促活广告前，您需要先检查下所投放的应用相关Activity都有声明exported=“true”, 如果存在exported="false"的情况，则不会使用该Deeplink拉起应用。

建议您先创建些试投放任务进行测试，如果能正常拉起应用，则可以启动正式投放。

<strong>应用促活任务只支持在展示应用广告、智能应用广告上进行推广，当前支持两种推广类型：</strong>

|  |  | 支持推广的应用所属平台 | | | | 广告展示位置 | |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 推广类型 | 说明 | 华为应用市场 | GooglePlay | 其他安卓应用商店 | 第三方监测渠道 | 华为自有媒体 | 三方媒体 |
| 智能应用推广 | 在接入鲸鸿动能广告变现服务的三方应用上及华为应用上展示您的应用广告。在智能推广模式下，您只需要输入关键元素，系统将对各个元素进行智能组合和优化，适配到各个不同广告类型。 | ✔ | - | - | - | ✔ | ✔ |
| 展示应用广告 | 在华为应用上或接入鲸鸿动能广告变现服务的三方应用上展示您的应用广告。您可以完全控制展示的广告位类型，素材的组合方式等，以达到您预期的投放效果 | ✔ | ✔ | ✔ | ✔ | ✔ | ✔ |
