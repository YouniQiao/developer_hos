# oCPC产品激励政策

<strong>【政策公示】2026年oCPC产品激励政策</strong>

oCPC产品激励政策是鲸鸿动能推出的，为保障广告前期能得到充分的探索，促进持续投放的激励政策。oCPC产品激励政策的推出是为了降低客户投放成本，积累数据，提高模型学习效率，持续优化投放效果。

<strong>适用范围</strong>

1. 政策覆盖范围：所有投放oCPC的客户，无需申请。
2. 政策有效期：2026年1月1日起

<strong>激励规则</strong>

| <strong>激励门槛</strong> | <strong>执行细则</strong> |
| --- | --- |
| 转化目标 | oCPC所有支持的转化目标 |
| 保障周期 | 任务首次投放起<strong>20个自然日</strong>  （10天为一个周期，共两个周期） |
| 保障要求 | 分投放策略，优先跑量：30%&lt;实际成本偏差&lt;=430%；非优先跑量：20%&lt;实际成本偏差&lt;=420%  其中，实际成本偏差=结算期实际消耗 / 结算期期望消耗–1  生效时间均以任务的首次曝光时间为准。举例：  稳定拿量的任务，适用(20%,420%]的规则，出价100元，实际成本120元，不予保障；实际成本若大于120元，则对120-520元之间的部分给予保障，对超出520元的部分不予保障。  如果一个任务在保障周期内使用过优先跑量的投放策略，则该保障周期适用优先跑量的保障范围。 |

## 常见问题

<strong>Q1：什么叫首次投放？具体怎么定义保障周期？</strong>

<strong>A：</strong>即广告正式投放后，首次曝光给用户时开始，比如计划4月1日新建，4月2日审核通过，4月3日产生首次曝光数据，则保障时间为4月3日，及其之后的20个自然日（4月3日~ 4月22日）。

<strong>Q2：保障周期说明</strong>

<strong>A：</strong>以10天为一个周期，共保障2个周期，每个保障周期的激励金额单独计算。

保障周期一：第1-10天投放，第11天后开始计算第1-10天投放产生的激励金额；

保障周期二：第11-20天投放，第21天后开始计算第11-20天投放产生的激励金额；

注：两个保障周期是独立的，保障周期一未满足激励门槛，不影响保障周期二；两个保障周期通用一套激励门槛标准。

<strong>Q3：哪些任务在政策有效期内</strong>

<strong>A：</strong>首次曝光时间在2026年1月1日及以后的任务可享受上述oCPC产品激励政策。

<strong>Q4：深度转化说明</strong>

<strong>A：</strong>转化数要求说明：深度转化如双出价等任务。

转化数要求说明：深度转化如双出价等任务，转化数按照深度转化目标要求。以激活次留为例，每个保障周期（10天）内次留数&gt;=6个才达到转化数要求（对激活数不做要求）。

总预期消耗说明：若保障周期（10天）内同时存在学习期和稳定期，则总预期消耗=学习期期间（激活目标成本1\*对应激活数+激活目标成本2\*对应激活数+……）+稳定期期间（次留目标成本1\*对应次留数+次留目标成本2\*对应次留数+……）。

修改次数说明：浅层目标和深度转化目标均统计，比如一个任务一天修改了1次深度转化出价，同时修改了2次浅层出价，当天广告出价修改次数记为3。

<strong>Q5：激活首日ROI说明</strong>

<strong>A：</strong>激活首日ROI转化数要求按照付费，即每个保障周期（10天）内付费次数&gt;=2个；总预期消耗=学习期期间（激活目标成本1\*对应激活数+激活目标成本2\*对应激活数+……）+稳定期期间（实际收入/期望ROI）

<strong>Q6：哪些操作会被视为不诚信行为？</strong>

<strong>A：</strong>恶意控制回传、恶意调整预算和出价等影响模型的行为都可能被系统判定为不诚信行为，鲸鸿动能平台有权独立作出判定结果。

<strong>Q7：全域智投任务的创意涉及元素和元素组，元素或元素组的处理该怎么计算修改创意的次数？</strong>

<strong>A：</strong>

1. 整个删除元素组或者同一次批量删除多个元素组---算一次修改创意

2. 新增元素组-----算0次修改创意

3. 一次批量替换某个元素并新增元素-----算1次修改创意

4. 一次批量替换多个元素并删除某几个元素----算1次修改创意

注：同一个任务下，不管是从任务编辑进入元素组还是从创意进入元素组，口径一致。
