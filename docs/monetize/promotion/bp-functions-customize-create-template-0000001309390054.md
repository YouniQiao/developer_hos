# 使用放量模板创建

## 前提条件

已完成投放任务的创建。

## 操作步骤

1. 登录[华为应用市场应用推广平台](https://ads.huawei.com/cn/)，点击【工具】页签，投放辅助--点击“自定义投放规则”，进入“自定义投放规则”页面。

   ![](./img/0000000000011111111.20260303164841.43332468758922493526021091393407:50001231000000:2800:72A0610ADCFBD170B2734738842879F2C49920E9B8DDC5DDF21F75AF8CE10ADC_7c2c14be3fad.png "点击放大")
2. 点击“放量模板”，配置相关任务的投放规则。

   ![](./img/0000000000011111111.20260303164841.97783067031736919934066301506521:50001231000000:2800:5DEF0E97FF66F3A3CA00C94664A10A08F53F2576C618462341D13A871A757BE9_50ebd3ce74d0.png "点击放大")

   ![](./img/0000000000011111111.20260303164841.44137611908726053197549343640212:50001231000000:2800:7EFC9238710415017CE6C09F812CF5AD9ECF842927AE32315E8BFFD786EB6D31_7d58c2220cac.png) 

   放量模板默认预置了当天下载量超过阈值的条件，以及调整出价和任务日预算的执行操作，即实现在指定的日期下载量超过指定阈值即调整出价和任务日预算。

   ![](./img/0000000000011111111.20260303164841.96015820700006410351242013897844:50001231000000:2800:E5EC6F2A47DC8F7723308E0DD888D20533D7C8CB305072AB82CD53DA973F1E45_d8342b1e5e8c.png "点击放大")

   具体规则设置项说明请参见[规则设置项说明](https://developer.huawei.com/consumer/cn/doc/promotion/bp-functions-customize-create-blank-0000001362109145#ZH-CN_TOPIC_0000001362109145__zh-cn_topic_0000001343075169_p17931549143111)。
3. 配置完成后，点击“提交”。系统弹出提示窗口，进行二次确认，点击“确定”。
