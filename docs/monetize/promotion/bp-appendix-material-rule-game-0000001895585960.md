# 游戏

## 游戏行业通用规范

<strong>1. 文案基础要求：</strong>

1）广告文案请勿出现沉迷游戏类描述，如：通宵打关、一玩就停不下来、这游戏可以玩一整天、玩到忘记时间。

2）广告文案不得使用挑衅或夸大类描述，如：不好玩算我输、必备手游等。

3）广告文案不支持有歧义或涉及不良诱导的内容。

如：终于找到一款大人玩的塔防了、这游戏18岁以下慎入、花前月下来双修太有意思了。

（注：若描述场景不暧昧，如果单纯描述“魔武双修”游戏功法，可使用)

<strong>2. 文案推广内容一致性要求：</strong>

1）与开户报备保持一致，不得开户报备A游戏，任务推广B游戏。

2）创意内容需保持一致，不得外层素材描述A游戏，落地页推广B游戏。

<strong>3. 文案推广内容真实性要求：</strong>

广告内容不支持以玩家性别占比为噱头作宣传。（注：换装类游戏可通过，如：这游戏女性玩家占90%）

<strong>4. 文案游戏密语使用要求：</strong>

1）游戏广告，支持使用游戏中玩家常使用的游戏密语，不支持带有贬低人群的词汇或沉迷游戏的词汇描述，如：

可使用：氪金、GG、开黑、落地成盒、吃鸡、猥琐发育等。

不支持使用：非洲人、欧洲人、欧皇、非酋、肝帝、肝爆、屌丝等。

<strong>5. 文案不得违背社会公序良俗和社会精神文明建设要求：</strong>

1）不得描述用户为了玩游戏不顾一切等不符合常理的内容，如：玩这游戏，才是真正的生活。

2）不得误导，诱发未成年人模仿违反社会道德和犯罪行为的内容；如：杀人换装备，一键当打手，最强山贼等。（注：古惑仔类帮派游戏）

<strong>6. 文案敏感元素审核要求：</strong>

1）广告内容不允许出现暗示或明示赚钱、以金币或积分等方式变相兑换现金、财物的描述。

具体限制如下：

赌博类描述：例如：一夜暴富、炸金花等。

货币或实物交易类描述：例如：打金、返利、装备回收换钱、游戏积分金币兑换或赢取现金、红包等活动。（注:可以描述是游戏中的虚拟货币、装备等，或其他形式的“抽奖得实物活动”如：抽幸运用户送电脑椅）

2）广告内容不能出现“摸金”“倒斗”“盗墓”等违法词汇。

3）广告文案不得出现以下关键词/敏感词：

如：私服、SF、外挂、QQ尾巴、木马病毒、魔兽世界插件、魔兽世界大脚等关键词；包含游戏类私服、外挂软件、私服服务器租赁的网站提交的游戏类关键词等。

4）广告内容禁止涉嫌赌博场景/内容。

如“老虎机、大转盘、百家乐、骰宝、21点、牌九、梭哈、扎金花、赢三张、牛牛、德州”类目禁止。

<strong>7. 图片法律法规相关要求：</strong>

1）若无授权，广告图片不得模仿或抄袭第三方游戏的人物形象、场景、标志性logo或代表性图标等作为创意或创意元素。

2）广告图片不得涉及烟草等相关内容。（含动漫、游戏抽烟形象）

3）不得额外添加非游戏内的涉嫌诱导点击的元素误导用户、诱骗用户点击广告；包括但不限于虚假关闭按钮、翻页按钮等。

注：可使用游戏截图中的自有元素，如关闭按钮、抽奖按钮可以使用。

<strong>8. 图片敏感元素类审核要求：</strong>

1）广告图片不得使用任何军用logo、标志、特殊符号等可代表各国军队的标识。（注：体育竞技类游戏可放宽）

2）广告图片不允许使用不完整中国地图、国旗元素等素材。

3）现代战争类游戏不允许使用战争实景图的素材。

<strong>9. 人物形象不得涉嫌性诱导内容，需遵守以下规范：</strong>

1）不得使用色情、挑逗且突出敏感部位的形象。

2）不得使用导向不良的暧昧场景+亲密动作的素材。

3）人物动作不得出现色情挑逗或特写敏感部位，如：托胸、俯身翘臀。

<strong>10. 素材不得涉及恐怖暴力类内容：</strong>

1）素材禁止出现腐烂、尸体、僵尸、骷髅头类等形象。

2）素材禁止出现巨兽打斗场景、相互吞噬、撕咬等血腥场景，如肢体撕裂、浴血、咬食等。

3）素材禁止出现多头、多手、多足的巨兽类形象。

4）素材禁止出现伤害身体的过程、溅血场景、爆头场景等。

<strong>11. 视频素材规范要求：</strong>

1）广告视频声音须完整且正常无杂音，广告播放时间内须持续有声音，不得出现突兀的声音，包括但不限于爆破音、尖叫声等。

2）广告视频不支持出现色情、亲密场景、形象暴露、洞房等画面。

3）广告视频里面的语音跟视频里面的字幕需要保持一致。

4）广告视频应完整填满广告位规格尺寸，内容显示需完整，画面比例需符合视频广告规范，视频上下、左右可增加图片设计来适应比例，但不可以将画面拉长或压扁使视觉变形。

<strong>12. 视频素材规范要求：</strong>

1）广告视频文字不得出现10秒以上的文字切割或截断展示，影响用户阅读。

2）广告视频需完整播放，不得出现突兀的内容中断。

3）视频在使用外语音频配音时，需添加中文字幕搭配使用。

## 宫廷游戏审核规范

<strong>1. 文案应倡导良好社会道德风尚，不得出现低俗、暴力、反动等引导：</strong>

1）不得出现弘扬封建制度如“回到明朝做皇帝”。

2）不得出现违反社会道德风尚如“三妻四妾”、 “后宫选妃”。

3）不得出现政治敏感词语如 “治理国家”“造反”等。

4）不得使用有歧义容易使人误解的文案。

5）禁止种族、宗教、文化歧视和性别歧视，不得带有对特殊群体的歧视、惩罚内容，如：对囚犯有特殊对待、惩罚女奴等描述。

6）文案不得使用带有不良社会价值观导向，如：做山贼，体验一把烧杀抢掠的快感。

7）不得带有政治不良诱导。如：“如果你是皇帝你会如何治理国家”等政治敏感文案或模拟现今社会中容易引起社会反响的内容。

<strong>2. 图片素材涉及需要遵循以下原则：</strong>

1）广告图片素材不得涉及色情，包括过于暴露的人物以及性诱导内容。

2）广告图片素材不得出现有悖社会主义良好风尚的场景“惩罚女囚”、“侍寝”、 “朝拜”、 “下跪”。

3）广告图片素材禁止宣导消极、丑陋、恐怖、血腥及含有暴力倾向等元素。

4）广告图片素材不得出现明示或暗示社会敏感内容的元素，如 ：暗示拆迁的“官民冲突”。

5）广告图片素材不得出现囚犯形象。

6）广告图片素材禁止出现妨碍社会安定的元素，如：人民揭竿起义、重写历史篇章。

7）广告不得使用洞房图及带有“生子”导向的图和文案描述，可以使用“结婚”图片及描述。

## 棋牌捕鱼类游戏审核规范

<strong>1. 捕鱼棋牌类游戏文案要求：</strong>

1）广告文案不允许出现暗示或明示赚钱、游戏积分交易、积分兑换或以金币等方式变相兑换现金、财物的描述,具体限制如下：

赌博类描述，例如：一夜暴富、炸金花。

货币或实物交易类描述，例如：返利、游戏积分金币兑换或赢取话费、现金、红包等活动。

（注：可以描述是游戏中的虚拟货币、装备等，或其他形式的“抽奖得实物活动”如：抽幸运用户送电脑椅）

2）广告文案不得以女性玩家为噱头作宣传；如：这游戏女性玩家占90%以上、美女24小时在线陪玩。

3）广告文案不得涉嫌暗示或明示引导玩家私下交易的描述；如：加微信、加QQ、进群有福利等。

<strong>2. 捕鱼棋牌类游戏图片要求：</strong>

1）广告图片禁止使用涉嫌赌博场景/元素；如：筹码、老虎机、大转盘、线下真实的扑克牌等

2）广告图片禁止出现暴露/暧昧/带有引诱性质、突出敏感部位、穿着透视/肤色/紧身、动作不雅的暗示或隐含性信息的形象。

3）广告图片不允许出现暗示或明示赚钱，变相兑换现金财物的素材，如：红包、话费、京东卡等。
