# FAQ

1. <strong>Push用户增长服务菜单能够看到，点击无法进入投放页面？</strong>

   请优先确认当前所在的项目是否为目标应用所在项目。若确认项目信息无问题，请提供应用的基本信息，联系华为方销售/运营处理。
2. <strong>为什么账户总金额和可用余额不匹配？</strong>

   应用市场推广基金会根据任务投放情况，进行资金冻结。您账户的可用余额，为用户增长服务实际可使用余额。
3. <strong>账户只有代金券，没有余额，是否支持投放？</strong>

   2022年1月8日后，账户只有代金券也支持投放，代金券扣除完后，会扣减现金账户余额。在此日期之前，您需要在投放前确保通用基金账户有充足的可用余额。
