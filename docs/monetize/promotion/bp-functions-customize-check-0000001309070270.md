# 查看自定义投放规则的运行数据

1. 登录[华为应用市场应用推广平台](https://ads.huawei.com/cn/)，点击【工具】页签，投放辅助--点击“自定义投放规则”，进入“自定义投放规则”页面。![](./img/0000000000011111111.20260303164840.79954041430305456804052862217533_50001231000000_2800_BD8D311532390D709AE9F43FDE1A994F60038B401764528B0154FD4886986960_992af5d735f9.png "点击放大")
2. 点击“运行记录”页签，即可查看当前所有自定义投放规则的执行结果。

   ![](./img/0000000000011111111.20260303164840.40404672664524475464731365705521_50001231000000_2800_0AF3DB338391286ACE77CD6130E97FD78AF8C5266938ACC6FFE3BEE0FB1CC10F_913909abdd0b.png "点击放大")

   ![](./img/0000000000011111111.20260303164840.83200131810324191496731451700843_50001231000000_2800_A2B619F86EAFD57158DAE447572B3F848FFD51C153AAAFFDE5F613108657CEC1_15f03a95dc9b.png) 

   - 支持基于“规则名称”和“查询日期”进行对应规则的查询。
   - 点击“下载表格”可以将当前查询结果导出为excel文件。
