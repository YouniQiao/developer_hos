# 业务介绍

自定义投放规则可以根据您设置的条件，由系统自动完成实时观察和调整推广任务的投放。自定义投放规则是投放监控的好助手和好管家。使用此功能，您不需要时刻盯盘。推广任务和账户如果有异常情况能由系统发送通知消息，及时告知您，大大节省了您在盯盘和调整任务上所花费的时间，尤其在节假日期间，此功能会让您更加轻松自如的管理推广任务。

![](./img/e06b3384a661.png) 

自定义投放规则仅适用于非合约的推广任务。
