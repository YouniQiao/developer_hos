# 区域审核规范（亚太）

## 概述

本章节适用于亚太的国家/地区，与通用审核规范而言，对亚太的国家/地区的行业内容做了更细化的说明规范。

## 国家/地区法律法规规定

1. <strong>不得投放含有违法犯罪内容的广告。</strong>
   1. 广告不得包含、倡导或推广违法商品、服务或活动(如毒品买卖、人口及人体器官贩运和买卖、濒危动植物及其制品售卖，洗钱等)。
   2. 广告不得鼓动或宣扬犯罪，对他人实施人身伤害，偷猎或出售濒危物种或其肢体。
   3. 广告不得提倡实施犯罪活动，如走私服务或鼓励参与高风险病毒攻击等。
   4. 广告不得出现暴力或煽动暴力，可诱发肢体伤害等内容。
   5. 广告不得宣传监视服务或间谍等内容。
   6. 广告不得宣传危险人物或恐怖组织。
   7. 广告不得推广投资少回报快的业务模式，包括多层次传销，“快速致富”计划等。
2. <strong>不得投放虚假或者引人误解的广告，所推广商品或服务需真实、有效，不得欺骗、误导消费者。</strong>

   不得对商品的性能、功能、产地、用途、质量、成分、价格、生产者、有效期限、允诺等，或者对服务的内容、提供者、形式、质量、价格、允诺等做虚假宣传：

   - 不得有任何虚构原价、虚假折扣等价格欺诈以及虚假营销的行为。
   - 对商品的描述应当有合理、有效的证明资料支持，不可捏造或夸大内容，不可使用不符合事实认知或逻辑的描述。
   - 不得投放与市场价格差异过大，价格超低或明显造假的商品。
   - 不得对产品或服务的效果做保证性承诺的表述，如：5天瘦10斤等。
3. <strong>不得投放含有歧视性内容的广告。</strong>
   1. 广告中不得含有歧视性内容，包括但不限于性别歧视、种族歧视、宗教歧视、肤色歧视、年龄歧视、性取向歧视、特殊人群歧视、家庭状况歧视。
   2. 不得基于个人/群体的特征，煽动对个人/群体的仇恨、宣扬对个人群体的歧视或者对个人/群体进行污蔑，不得声称某个人/群体对他人的人身安全、健康或生存构成威胁。
4. <strong>不得投放含有暴力内容的广告。</strong>
   1. 禁止煽动暴力、宣扬美化或纵容对他人采取暴力行为的内容。
   2. 禁止宣传或美化危险人物/组织的形象或言论、恐怖主义行为；禁止招募恐怖主义分子、引导配合实施犯罪等内容。
   3. 禁止宣扬自杀、厌食或其他自我伤害方法的内容。
   4. 禁止宣传枪支武器，包括枪支、仿真枪，弹药，刀具等具有伤害性质的武器或涉嫌暴恐的内容。
   5. 广告素材中禁止出现仿真度较高的枪支武器或战争场景。
5. <strong>不得投放含有侮辱诽谤贬低内容的广告。</strong>
   1. 广告内容中禁止出现侮辱性词汇（含变体）、手势等，不得使用特殊符号暗示不雅语言/词汇。
   2. 不得辱骂、诽谤、贬损、诅咒、恶意中伤任何个人/团体/他人的服务或作品等。
6. <strong>不得投放含有不正当恐慌内容的广告。</strong>
   1. 广告中禁止出现血腥、恐怖、暴力、惊悚等易引起用户恐慌的画面，如：案发/事故/战争现场、丧尸、灵异场景、吞噬/死亡场景等。
   2. 广告中禁止出现暗示用户可能处于战争、疾病、阴谋等危险处境之中的内容，禁止使用易引起用户恐慌的文字描述。
7. <strong>不得投放含有损害国家/地区尊严或者利益，危害国家/地区外交，泄露国家/地区秘密，妨碍社会安定，损害社会公共利益内容的广告。</strong>
   1. 广告中提及领袖、王室成员、使用国家/地区机关、国家/地区机关工作人员的名义或者形象时应严格遵循各国相关法律法规，不得对其进行丑化、污蔑、诽谤。
   2. 广告中使用国旗、国徽、王室徽章、国歌、军旗、军歌、军徽、文化遗产等国家/地区特殊标识时，应严格遵循各国相关法律，不得对其进行丑化、污蔑、诽谤。
   3. 广告内容中不得包含危害国家/地区外交、泄露国家/地区秘密、妨碍社会安定、损害社会公共利益等不当言论。
   4. 广告不得涉及危害国家/地区安全，必须严格遵守各国的国家安全或相关法律。
8. <strong>不得投放含有淫秽色情、低俗、迷信、赌博内容的广告。</strong>
   1. 广告中禁止出现暴露/暧昧/带有引诱性质、突出敏感部位、穿着透视/肤色/紧身、动作不雅的暗示或隐含性信息的形象，不得将人物色欲化。
   2. 广告不得包含意在唤起性欲的文字、图片、音频或视频。
   3. 广告中不得宣传未成年人、非自愿或其他非法性主题（无论是模拟的还是真实的）的内容。
   4. 广告中不得宣传迷信内容，不得使用带有迷信色彩的文案、图片、音频或视频；广告中不得明示或暗示通过超越自然科学解释范围内的手法可达到某种效果/目的。
   5. 广告中不得倡导、美化赌博。
9. <strong>不得投放含有妨碍环境、自然资源或者文化遗产保护、虐待动物内容的广告、不得贩卖濒危物种或销售濒危物种制品</strong>。
   1. 广告中不得宣传/倡导破坏、浪费自然资源，损坏、亵渎文化遗产。
   2. 广告中不得宣传虐待或无端残害动物的内容，如通过虐待动物达到娱乐的目的。
   3. 广告中不得宣传/倡导买卖濒危物种或濒危物种制品。
10. <strong>广告禁止诱导/鼓励不安全机动车驾驶和不安全机械设备操作等危险内容，如喝酒驾驶，超速驾驶等</strong>。
11. <strong>广告禁止仿冒产品，不得出现高仿品，假冒品，伪造品等</strong>。
12. <strong>广告使用数据、统计资料、调查结果、文摘、引用语等引证内容的，应当真实、准确，并表明出处</strong>。
    1. 引证内容有适用范围和有效期限的，应当明确表示适用范围和有效期限。
    2. 引用历史数据（比如在线人数、下载量、销售量、排名、通过率、成功率等），应当注明“历史数据，仅供参考”等提示说明，如：广告中含有评奖、评比或排名等信息（如“销量第XX”、“市场占有率X%”、“获得XX年XX单位评选的XX奖”等），需要广告主提供有效的相关证明资料后方可发布广告。
13. <strong>不得在广告中涉及虚假专利产品或者专利方法</strong>。
    1. 在广告中使用专利产品或专利方法，应当标明专利号和专利种类。
    2. 未取得专利权的，不得在广告中谎称取得专利权。
    3. 禁止使用未授予专利权的专利申请和已经终止、撤销、无效的专利作广告。
14. <strong>不得使用易引人误解的广告创意，故意欺骗、误导消费者</strong>。
    1. 广告应当具有可识别性，能够使消费者辨明其为广告。
    2. 大众传播媒介不得以新闻报道形式变相发布广告，不得使消费者产生误解。
    3. 按钮的使用，应符合以下使用规范：

       按钮的定义：（1）使用边框、对比色的独立区域（2）带有文案引导或箭头引导。同时满足条件（1）（2）定义为按钮，应满足以下规则。

       - 广告素材上不得利用图标或按钮的操作功能误导用户，诱骗用户点击广告。
       - 图标或按钮，包括但不限于虚假播放、关闭、暂停按钮、鼠标或者手势图案、假分页、翻页图标、点赞、评论、loading等，如：外层图片上放置了关闭按钮，用户点击后实际进入广告页，而非关闭广告，属于欺骗、诱导用户点击。
    4. 广告素材不得模仿系统或网站发出的消息或通知，使消费者误以为是系统行为，如：告警、对话框、错误消息，“恭喜你获得一个礼包”、“祝贺你获得免费领取的资格”等。
    5. 广告文案不得使用模拟好友关系链的文案，使消费者误以为是自己好友在进行邀请；如：“亲爱的XXX，您的好友XX邀请您进行一起进行活动”等。
    6. 广告页面不得仿照知名品牌网站链接或者知名品牌网站设计，以诱导用户点击或购买。
15. <strong>不得贬低其他生产经营者的商品或者服务</strong>。
    1. 广告不得直接捏造、虚构事实，从而对竞争对手的产品或者服务进行诋毁、贬低，如：虚构竞争对手的产品、服务存在瑕疵、质量安全问题等。
    2. 广告不得间接贬低竞争对手的产品、服务，如声称自己的产品、服务的质量、价格等都比竞争对手好，如：“本公司产品比XX产品有效率高30%”、“本店产品价格均低于XX平台”等。
16. <strong>广告不允许使用未经授权的版权内容</strong>。
    1. 未经授权不得使用第三方公司相关标识。
    2. 广告中使用他人的注册商标/版权作品（如照片、绘画等）内容，或知名人物、动漫形象，以及奖项荣誉称号等，需事先取得合法授权证明或合作协议。
17. <strong>广告不得损害未成年人和残疾人的身心健康</strong>。
18. <strong>不得违反投放国广告代言规则</strong>。

    广告代言人，是指广告主以外的，在广告中以自己的名义或者形象对商品、服务作推荐、证明的自然人、法人或者其他组织。广告中使用代言人需要遵循以下规则：

    - 广告代言人在广告中对商品、服务作推荐、证明，应当依据事实，符合本法和有关法律、行政法规规定，并不得为其未使用过的商品或者未接受过的服务作推荐、证明。
    - 医疗、药品、医疗器械、保健食品广告不得利用广告代言人作推荐、证明。
    - 对在虚假广告中作推荐、证明受到处罚的自然人、法人或者其他组织，一般情况不建议利用其再作为广告代言人。
19. <strong>禁止内容中出于竞争目的引用商标</strong>。
20. <strong>广告中避免出现敏感词、极限词（如best，first，No.1，safe，without risk等）和其他程度词（如new，improved，most preferred，价格低至，折扣高达等）；在允许使用程度词进行宣传的国家/地区中，要求应用了程度词的广告有确实的依据证实。</strong>
21. <strong>部分国家/地区在提及环保相关用词（environmentally friendly, green, sustainable, clean, climate-friendly）时，要求有切实的证据证实</strong> <strong>如菲律宾、马来西亚、新加坡等</strong>。
22. <strong>对于含有赌博内容的广告，限制其投放。</strong>

    广告中可以宣传赌博或具有赌博性质的游戏/活动，如德州扑克等卡牌游戏，或赌场游戏和抽奖活动。广告中不允许出现套现行为，且禁止向未成年人投放。
23. <strong>对于含有理财产品和服务的广告，限制其投放。</strong>
    1. 广告中允许宣传理财产品和服务，如综合金融服务平台、在线股票交易和个人贷款。
    2. 对于宣传投资理财产品和服务的广告，需要添加风险提示语。
    3. 宣传个人贷款产品和服务的广告必须提供详细的网站或应用信息。信息透明化可以保证消费者的知情权，并且帮助他们利用有效信息进行决策。

## 平台基本规则

1. <strong>广告内容禁止宣传政治内容及政治敏感内容，包括竞选广告，批判国家/地区/政治体制，领袖等内容</strong>。
2. <strong>广告素材禁止出现华为相关的负面内容。</strong>
3. <strong>广告素材禁止出现含占星术、超自然现象、神秘学、鼓吹运气等神秘学等内容。</strong>
4. <strong>广告禁止销售免费商品：</strong>
   1. 禁止宣传对可从政府或公共来源免费或较低价格取得的产品或服务加以收费的产品/服务广告，如出生证、护照或驾照申请服务；医疗保险申请；官方注册机构出具的文件，例如结婚证明或公司注册文件；考试成绩查询；税务计算工具。
   2. 免费商品/服务与有价商品/服务的捆绑产品、且不以免费商品/服务为主要宣传对象的广告可以投放。
5. <strong>广告内容不得涉及成人用品、计生用品、情趣用品、洁阴用品等产品。</strong>
6. <strong>广告应使用投放国支持的主流语言及官方货币单位。</strong>
   1. 广告创意所使用的语言需与投放国的主流语言保持一致。
   2. 广告语言定向需与广告创意语言保持一致。
7. <strong>对于包含约会交友内容的广告，限制其投放</strong>。
   1. 广告中不得包含带有性暗示的行为，包括婚外情、性交易、性接触等。
   2. 广告仅允许在某些场合投放，且存在年龄限制。禁止向未成年人投放该广告。

## 广告文案基础规范

1. <strong>广告文案不允许出现不可理解或无意义的字符。</strong>

   示例：广告文字没有意义或过于空泛；宣传内容过于空泛或含糊；广告文字被切断或不完整。
2. <strong>广告或附加信息不得超过字符语言的字符数限制。</strong>
3. <strong>广告不允许出现标点或符号使用不当或有违其本来用途，及不受支持的无效字符，如trees!!!</strong>

## 广告图片制作基础规范

1. <strong>不得使用低像素、模糊、变形的图片</strong>。
   1. 广告图片必须高精度（含logo），清晰可读，无锯齿无噪点，不凌乱不变形。
   2. 广告图片正立、广告不得翻转或倒立显示。
   3. 图片里的抠图素材与背景图的拼接需自然，避免模糊、边缘锯齿及像素噪点。
2. <strong>不得使用消极、丑陋、恐怖等可能引起用户不适或反感的元素</strong>。

   不能宣导消极、丑陋、恐怖的元素，应在感官上保证舒适的感官体验，如：形象不佳的人物，局部放大或病变的人体器官（黑头、手、脚、眼睛等）、手写文案等。
3. <strong>不得在图片中出现个人电话号码（招商加盟广告允许出现个人电话号码）、地址、URL链接、或其他社交软件号码等文字内容，马赛克图、条形码图等</strong>。
4. <strong>广告位素材不得直接使用手机软件客户端截图</strong>。
5. <strong>素材图片不得使用浮层、图片叠加堆砌，不得使用密集的元素进行堆砌</strong>。
6. <strong>不支持其他类型可能影响用户体验的图片：如整体设计感欠缺、如毫无构图、图片缺乏修饰等情况</strong>。
7. <strong>不允许出现诱导或引导用户点击图片、点击查看详情、诱导点赞评论的按钮或文案。</strong>
8. <strong>广告素材应具有明显的创意性，不得直接使用无设计的单品图片作为广告创意，可将其作为创意元素搭配使用。</strong>
9. <strong>广告素材需具有创意性，可使用海报型图片，不得直接使用实景照片、手机截图、表单或表格截图等</strong>。
10. <strong>广告素材如添加了产品商标logo或者广告主logo，Logo中的文字需显示完整、清晰、易阅读，还应满足如下要求：</strong>
    1. 数量：每条广告中仅允许出现一个Logo标志。
    2. 位置：Logo不允许贴边显示不全。
11. <strong>图片不得使用频闪、闪烁或会让用户分散注意力的其他效果</strong>。
12. <strong>广告不得延伸到框架之外或者侵占网站或应用的空间</strong>。

## 广告视频制作规范

1. <strong>不允许出现所含文字难以辨认、音质不佳、视觉效果模糊不清或无法辨识的视频</strong>。
2. <strong>有声广告视频的声音须完整且正常无杂音，不得出现突兀的声音，包括但不限于爆破音、尖叫声等</strong>。
3. <strong>广告视频里面的语音应跟视频里面的字幕需要保持一致</strong>。
4. <strong>广告不可自动播放有声音频、视频等</strong>。

## 广告详情页制作基础规范

1. <strong>落地页不允许出现“死链”，死链即点击后无法抵达的页面</strong>。
2. <strong>打开时长：网络最佳环境下，落地页加载时间，即用户从点击到完全打开落地页所需的时长不得超过5秒</strong>。

## 广告主信息规范

<strong>品牌名称</strong> <strong>，需满足以下条件:</strong>

- 推广名称应独立、完整的表达广告主权益。
- “推广名称”与开户主体一致。
- “推广名称”与推广素材、落地页具有关联性。
- 品牌名称请勿超过30个英文字符，需为品牌名称、广告主名称、广告主简称。
