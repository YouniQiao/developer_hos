# 招商加盟

## 一、招商加盟广告风险提示语

广告外层素材或落地页中需添加【投资有风险，加盟需谨慎】的提示语字体要求：大图字体最小尺寸16px，小图最小尺寸12px

## 二、广告素材审核要求

1.外层素材不得使用易引起误导的角标符号如：角标符号上标注“火”、“火爆”、“推荐”等易误以为是华为官方推荐行为的内容

2.不得使用真实证件或演绎证件（包括漫画形式）作为元素内容如： “营业执照”“合同”

3.不得出现转账付款信息，如支付宝、银行账户、微信付款二维码等

|  |  |
| ×问题解析：“火爆”易引起误导的角标符号 | ×问题解析：出现营业执照 |

## 三、广告内容不得出现对比、排他的描述

1.不允许出现和传统行业或其他行业进行对比，利用其他行业突出体现加盟开店的优势如：打工不如开店、开店比上班好、还在辛苦打工？不如开店等描述

2.不得与竞品做对比来表现品牌的项目优势与品牌优势如：加盟模式优劣对比、产品对比等

## 四、不得涉及收益与推荐证明类描述

1.不得涉及明示或暗示承诺收益、承诺加盟效果的描述，如: 店店排队、四季营收、回本快销高等相关的描述

2.不得涉及使用受益人名义进行宣传

3.不得利用学术机构、行业协会、专业人士等的名义或形象做出推荐、证明

## 五、不得涉及夸大描述类

1.在推介项目优势时，对所处领域的市场规模、市场前景等，不得夸大项目优势、或者歪曲、捏造事实根据

2.不得使用夸大/最高级词语描述去宣传品牌或服务如：引领潮流、领跑市场、冠军品牌等描述

3.不得使用其他夸大/绝对化/承诺性的描述

## 六、不得涉及负面、歧义描述

1.不得使用负面、消极描述如：死工资、月薪低等

2.不得使用存在争议、表达不明确、会引发用户不良的想法的描述如：偷偷告诉您、一般人我不告诉他

3.不得使用挑衅、绝对化、必须类类文案如： “赚钱必备”

## 七、其他要求

1.不得使用具体薪资、带有指向性特定人群的描述如：80后、90后、月薪5000也能加盟

2.不得出现模拟好友关系链的文案如：包括但不限于“亲爱的xxx，您的好友分享给您一个创业机会”

3.允许出现座机号码、400等服务号码，暂不支持个人联系电话号码。

4.不得出现个人二维码、小程序码、、条形码图、个人微信号、个人QQ号、QQ群号或其他社交软件号码等文字内容，禁止添加个人好友的行为
