# 2026年鲸鸿动能广告开门红激励活动

<strong>政策周期：</strong>2026年1月1日~2026年3月31日

<strong>面向对象：</strong>FSD模式下鲸鸿动能N2服务商

<strong>参与条件：</strong>对比2025年Q4，服务商于2026年Q1（1-3月）在鲸鸿动能的投放有增量现金消耗。

<strong>报名方式：</strong>开门红活动无需报名，所有FSD模式下满足条件的鲸鸿动能N2服务商默认参加并自动适用本政策。

<strong>互斥政策：</strong>无

<strong>激励内容：</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>激励名称</strong> | <strong>计算类型</strong> | <strong>计算基数</strong> | <strong>采买模式</strong> | <strong>返点细则</strong> |
| <strong>2026</strong> <strong>年新春开门红激励活动</strong> | 固定返点 | 现金消耗 | <strong>竞价/合约</strong> | <strong>对比2025年Q4，服务商于2026年Q1（1-3月）在鲸鸿动能投放的增量现金消耗可以享受额外6%返现激励。</strong> |

注：仅限鲸鸿动能DSP，综合DSP和直客DSP不适用本激励。

<strong>激励发放形式和使用：</strong>

|  |  |  |
| --- | --- | --- |
| <strong>竞价投放</strong> | <strong>合约</strong> <strong>投放</strong> | <strong>发放</strong> <strong>周期</strong> |
| 返现充值 | 应付款抵扣 | 单次 |

注：

1. 竞价投放的激励部分以返现形式充值进服务商的现金账户中，现金账户余额未经甲方确认不可提现、不得转让，该激励和其他现金充值业务合并开具发票。
2. 合约投放的激励部分予以应付款抵扣用于采买订单的费用支付和抵扣，合并开具发票，除在合同终止以及合作终止清算后，抵扣激励的金额不可中途提现。

<strong>结算方式：</strong>

1. 账单和激励金额数据以鲸鸿动能数据为准。
2. 如发现不符合激励条件的情况，鲸鸿动能将对数据进行复核，并仅在符合激励条件的范围内发放激励，金额以复核调整后的为准。激励发放后，如果服务商对激励数据有异议，可在五个工作日内通过服务商管理部邮箱mspadmin@huawei.com反馈。
3. 如涉及数据差异的，数据差异在鲸鸿动能数据的10%以内时，以鲸鸿动能数据为准；如差异超过10%，请于五个工作日内发正式邮件反馈数据差异排查，双方友好协商一致确认最终数据；如未及时反馈，则以鲸鸿动能数据为准。

<strong>补充说明：</strong>

1. 返现由鲸鸿动能平台在激励周期结束后，经华为侧核算完后发放至服务商账户，涉及多个账户的，按各账户消耗比例进行分摊返现，原则上不接受调整。
2. 适用政策期间，推广产品及相关推广投放应保持合法合规运营，不得出现违反法律法规或侵犯第三方合法权益的情形，否则鲸鸿动能有权取消推广产品的政策适用资格并采取相关处罚措施。
3. 服务商享受鲸鸿动能提供的返现等权益，需以其账户处于有效且可操作状态为前提。因广告主或服务商相关账户注销等账户状态异常情形，致使鲸鸿动能客观上无法向其账户成功充值返现或发放其他权益的，由此产生的返现等无法发放的后果，由服务商自行承担，并视为其默认放弃该等权益，鲸鸿动能将不再进行发放操作。
4. <strong>异常作弊、违规操作、恶意修改预算/出价、利用系统漏洞、滥用系统机制、可疑或非法交易等不当行为不可享受激励。经鲸鸿动能合理判定服务商或客户存在前述行为的，鲸鸿动能有权拒绝发放激励并追回已发放的激励，并采取相应处罚措施，由此造成的损失或赔偿责任由服务商承担。</strong>
5. <strong>如因产品功能、计算错误、系统错误等任何原因导致激励超额发放、重复发放或错误发放，一经发现，鲸鸿动能有权追回多发部分或按核实后的应发激励调整发放。</strong>
6. <strong>对于应追回的激励金额，鲸鸿动能可按以下方式扣除或追缴：（1）从下期激励中扣除；（2）从服务商激励账户中直接扣除；（3）从服务商的充值账户等其他支付、结算账户中扣除；（4）从鲸鸿动能尚未向服务商支付的各种激励、费用、分成、代收款等款项中直接扣除；（5）从服务商在华为平台的其他激励、账户或待结算款项中扣除；（6）采取其他可行的方式要求服务商在30个工作日内退回、支付或赔偿。</strong>
7. 如对本政策有任何疑问，请联系鲸鸿动能。
