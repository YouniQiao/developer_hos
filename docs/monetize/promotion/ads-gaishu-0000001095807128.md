# 概述

鲸鸿动能广告当前的广告资源分成应用市场、展示广告网络。

- 应用市场：应用市场广告可以帮助您在华为应用市场（包括应用市场搜索广告、应用市场展示广告）中推广您的应用。您只需添加您的应用、设置好定向、出价等，系统会自动优化其余部分，从而帮助用户找到您的应用。
- 展示广告网络：展示广告包括华为媒体资源、三方媒体资源和三方ssp资源。通过投放展示广告，您可以这些应用中推广您的网页或者应用，在用户浏览这些应用内容时，可以向他们展示您的广告，提升您的网页浏览量，增加应用下载数或者提升您应用现有用户的活跃度。

您在创建投放计划时，可以通过选择“投放网络”来指定将自己的广告投放到哪类广告资源上。
