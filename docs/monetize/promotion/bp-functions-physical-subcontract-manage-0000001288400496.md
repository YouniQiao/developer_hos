# 管理渠道包

在新建物理分包任务前，需要开通渠道包，配置好渠道包信息，并上传对应的软件包。

## 操作步骤

1. 申请开通渠道包（测试渠道包和极小渠道包功能暂停申请，如有疑问可咨询相关运营）

   具体请参见[申请开通](https://developer.huawei.com/consumer/cn/doc/development/AppGallery-connect-Guides/channel-commonpkg-apply-0000001193797006)。
2. 管理渠道包信息。

   具体请参见[管理渠道包](https://developer.huawei.com/consumer/cn/doc/development/AppGallery-connect-Guides/channel-commonpkg-manage-0000001131588316)。
3. 管理渠道包对应的软件包。

   具体请参见[管理软件包](https://developer.huawei.com/consumer/cn/doc/development/AppGallery-connect-Guides/channel-commonpkg-package-0000001177707829)。
