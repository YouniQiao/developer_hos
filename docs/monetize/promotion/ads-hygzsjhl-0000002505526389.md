# 社交&婚恋&直播

## 社交&婚恋&网络直播广告不得含有下列内容：

<strong>性暗示与低俗擦边词汇</strong>

1. 广告文案需符合社会良好风尚，不得使用涉及低俗、打擦边球、不良暗示、容易引起歧义的文案。

- 禁用低俗/擦边/不良暗示的文案：包括但不限于“深夜孤单”、“刺激/过瘾/爽”、“床上/深夜/秘密直播”等。
- 禁止人物居住状态+暗示需要慰藉、陪伴的状态进行结合描述，如：租房/在家+找人陪/孤独/寂寞等；支持场景：无聊描述。
- 禁止夜间时段暗示： 明示或暗示夜间时段（晚上、深夜、半夜）等词汇进行描述或引导。

2.不得含有不良导向词汇，如：“大波”、“艳遇”、“18禁”、“老司机才懂”等。

<strong>歧义和急迫邀约性词汇</strong>

1.不得使用容易产生歧义的“约”字的搭配，如：单身独居美女在家睡衣与您约聊；寂寞妹纸等您来约；不闲聊，直接见，快快约到您所爱；约会/可约/约一晚/约聊等。

2.不得使用强调急迫性的“求”字的搭配，包括但不限于“求爱”、“求约”、“求聊”等。

3.不得使用撩拨型口头语文案，如：“您过来，我想撩您一下！”。

## 人物形象与行为管理（禁止性暗示形象）

<strong>1.</strong> <strong>衣着</strong>：不得使用暴露/暧昧/带有引诱性质、突出敏感部位、穿着透视/肤色/紧身、动作不雅的暗示或隐含性信息的形象。

- 不得穿着情趣制服、情趣内衣、暴露装、肤色装、透视装、真空凸点、内衣外穿、刻意突出敏感部位的紧身衣、三角短裤或呈现三角状短裤、衣衫不整等。
- 不得裸露胸罩或内裤，或仅用被子、浴巾、衣物等遮盖身体。
- 不得穿着过于紧身导致敏感部位突出，不得对局部敏感部位聚焦或具有特殊修饰等镜头作为素材创意。
- 不得大尺度裸露身体，包括男性赤裸上身。

## 行业专项禁止

<strong>【社交婚恋专项禁止】</strong>

1.为避免不良导向，社交广告不得使用“摇一摇”、 “漂流瓶” 、“查看附近的人” 等介绍作为广告。

2.社交广告内容请勿涉及付费相关描述，易引起歧义。

<strong>【网络直播专项禁止】</strong>

1.不得对主播使用诱惑性称谓，如：“熟女”、“少妇”、“小妖精”等。

2.不得出现视频交友、视频邀约的描述，如：漂亮女主播给您发来一个视频邀约/视频电话。
