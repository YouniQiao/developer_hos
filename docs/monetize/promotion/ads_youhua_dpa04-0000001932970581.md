# 对接准备

<strong>申请权限：</strong>

当前动态商品广告（DPA）仅面向白名单广告主开启，若需投放请联系鲸鸿动能客户运营申请。

<strong>对接准备：</strong>

在使用DPA功能前，您需提前准备如下内容：

- 准备商品文件，根据鲸鸿动能提供的xml格式准备商品文件。
- 使用模板模式投放时，平台提供通用无装饰性的模板。您可定制高转化或通用性较强的创意模板，用于后期创意拼接，如投放不同版位需提供不同版位的创意模板（定制模板需要排期）；如投放直投模式，需保证商品库中的图片尺寸与鲸鸿动能广告版位尺寸一致。
