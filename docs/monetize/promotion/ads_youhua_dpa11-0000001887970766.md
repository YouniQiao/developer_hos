# 商品广告转化跟踪

鲸鸿动能平台提供商品广告任务中最终投放的商品ID信息用于转化归因。转化跟踪参数支持选择商品库ID、推广商品ID以及商品推荐来源，您可以通过转化跟踪API统计转化数据，具体对接方式请参考[事件资产管理（原转化跟踪）](/docs/monetize/promotion/ads_gongju08-0000001469108293)。
