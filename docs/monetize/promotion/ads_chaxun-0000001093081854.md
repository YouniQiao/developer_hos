# 查询定向包

【简介】通过此接口可以查询定向包

<strong>请求地址</strong>

https://ads.cloud.huawei.com/openapi/v2\_1/tools/targeting\_package/query

<strong>请求方法</strong>

<strong>GET</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| page\_num | integer | 是 | 搜索页码，  取值范围1~10000 |
| page\_size | integer | 是 | 一页展示数量，  取值范围 10~50 |
| targeting\_type | string | 否 | 定向包类型，  包含应用类、非应用类;  详见[【定向包类型】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_api73-0000001058884698#section1368511219507) |

<strong>请求示例</strong>

GET openapi/v2/tools/targeting\_package/query?page\_num=1&page\_size=10&targeting\_type=1 HTTP/1.1

Accept:application/json

Content-Type:application/json

Authorization:Bearer CgB6e3x9ERGComr9dENxZX22iBk+mLuf1yGtQVPUjPJUMrstfKlqpdXk+kfHU9J8ZJ/soYIZHZzT446GeSYumluQuhsK7jvz4kz1Bkms4CLI/rE=

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1 | 定向包列表 |

Struct1定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| total | integer | 定向包总条数 |
| data | Struct2[] | 定向包列表 |

Struct2定义

|  |  |  |
| --- | --- | --- |
| targeting \_id | long | 定向包ID |
| targeting\_name | string | 定向包名称 |
| targeting\_description | string | 定向包描述 |
| targeting\_type | string | 定向包类型，  包含应用类、非应用类  详见[【定向包类型】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_api73-0000001058884698#section1368511219507) |
| gender\_struct | Struct3 | 性别定向，值为gender定向中的value |
| age\_struct | Struct3 | 年龄定向，值为age定向中的value |
| series\_type\_struct | Struct3 | 手机系列定向，值为series\_type定向中的value |
| device\_price\_struct | Struct3 | 设备价格定向，值为device\_price定向中的value |
| installed\_apps\_struct | Struct3 | 已安装应用定向，如果选择此定向，value不为空 |
| not\_installed\_apps\_struct | Struct3 | 未安装应用定向，如果选择此定向，value不为空 |
| app\_category\_installed\_struct | Struct3 | 已安装App分类定向，值为app\_category定向中的value |
| not\_app\_category\_install\_struct | Struct3 | 未安装App分类定向，值为app\_category定向中的value |
| app\_category\_active\_struct | Struct3 | 一个月内活跃App分类定向，值为app\_category定向中的value |
| network\_type\_struct | Struct3 | 网络类型定向，值为network\_type定向中的value |
| pre\_defined\_audience\_struct | Struct3 | 自定义人群定向，值为pre\_define\_audience定向中的id |
| not\_pre\_defined\_audience\_struct | Struct3 | 排除自定义人群定向，值为not\_pre\_define\_audience定向中的id |
| current\_geo\_location\_struct | Struct3 | 当前在这里的人（地理划分）  值为region\_geo中的code |
| current\_level\_location\_struct | Struct3 | 当前在这里的人（发展划分）  值为region\_level中的code |
| residence\_geo\_location\_struct | Struct3 | 常驻在这里的人（地理划分）  值为region\_geo中的value |
| residence\_level\_location\_struct | Struct3 | 常驻在这里的人（发展划分）  值为region\_level中的value |
| app\_category\_of\_media\_struct | Struct3 | 投放媒体类型定向 |
| not\_app\_category\_of\_media\_struct | Struct3 | 投放媒体类型排除定向 |
| content\_id\_struct | Struct3 | 内容ID定向 |
| content\_label\_struct | Struct3 | 内容标签定向 |
| targeting\_scope | string | 标记定向包类型 |
| accept\_roaming\_flag | string | 漫游流量开关 |

Struct3定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| value | string[] | 定向选中的值 |
| additional | string |  |
