# 业务介绍

当前HarmonyOS应用推广支持oCPC投放模式，支持开发者以转化成本为目标进行智能出价。

oCPC按点击量计费，支持下载、平台激活、激活、次日留存、激活-次日留存、激活-7日留存、注册、授信、首次付费、每次付费、首日ROI、7日ROI等转化目标。

![](./img/caution_3.0-zh-cn_836c036fc8a7.png) 

激活目标全量开放，其余目标按白名单开放。投放前请提供对应的应用ID和应用名称，通过行业运营、客服邮箱（developer@huawei.com）与我们进行联系开通。

在创建oCPC任务前要求开发者先将转化数据进行回传，平台积累一段时间数据，在模型训练完成之后才能进行oCPC投放。使用oCPC，开发者无需再针对不同人群尝试性地进行出价，定向和出价过程由oCPC算法自动完成，为开发者节约了时间和投放成本。

请根据实际推广业务需求，创建对应的oCPC任务。

![](./img/note_3.0-zh-cn_0db16e5b273f.png) 

<strong>满足以下两个条件的应用将会自动开通oCPC功能：</strong>

- 已开通归因功能（[监测链接](https://developer.huawei.com/consumer/cn/doc/promotion/bp-hm-monitoring-link-0000002481479774)）。
- 已回传转化数据，且连续天日均回传转化量&gt;=5个。
