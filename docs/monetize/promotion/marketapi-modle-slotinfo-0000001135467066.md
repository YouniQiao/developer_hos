# SlotInfo

| 参数 | 必选(M)/可选(O) | 类型 | 描述 |
| --- | --- | --- | --- |
| slotId | M | String | 资源位ID。 |
| name | M | String | 资源位名称。 |
| category | M | String | 分类信息，对应站点信息。 |
| format | M | String | 资源位格式。 |
| floorPrice | M | String | 底价。 |
| bidAddMoney | M | String | 加价幅度。 |
