# 查询经理账户下属广告主账户列表

【简介】通过此接口可以查询经理账户下属广告主列表

<strong>请求地址</strong>

https://ads.cloud.huawei.com/openapi/v2/manager/advertiser/query

<strong>请求方法</strong>

<strong>GET</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| manager\_id | string | 否 | 经理账户 |
| page\_num | integer | 否 | 搜索页码 默认值1  取值范围1~1000 |
| page\_size | integer | 否 | 一页展示数量，默认值20  取值范围 10 ~ 50 |

<strong>请求示例</strong>

GET openapi/v2/manager/advertiser/query?manager\_id=123 HTTP/1.1

Accept:application/json

Authorization:Bearer CgB6e3x9ERGComr9dENxZX22iBk+mLuf1yGtQVPUjPJUMrstfKlqpdXk+kfHU9J8ZJ/soYIZHZzT446GeSYumluQuhsK7jvz4kz1Bkms4CLI/rE=

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1 | 广告主列表 |

data(Struct1)定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| advertiser\_info\_list | Struct2[] | 广告主列表 |
| total | integer | 总数 |

advertiser\_info\_list(Struct2)定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| advertiser\_id | long | 广告主ID |
| nick\_name | string | 广告主昵称 |
