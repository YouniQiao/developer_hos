# 业务介绍

华为分析（简称HA）作为监测平台为广告主呈现推广后端归因数据，广告主可通过HA便捷地进行推广归因，降低接入监测、回传归因操作成本。

![](./img/0000000000011111111.20251117124625.51891362895797005695565906552331_50001231000000_2800_CAF4EA6C43E3148D6A799332CDFAAEFA32D39A4DFCDC4AF236AFDF38252F8E30_196d8d2d0a4d.jpg "点击放大")

1. 广告主开通华为分析服务，华为分析服务可将行为数据回传到华为应用市场应用推广门户。
2. 广告主在华为应用市场应用推广门户选择HA归因监测。
3. 创建广告投放任务后任务会同步到华为应用市场应用推广平台。
