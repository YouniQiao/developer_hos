# 分享商品库

【简介】经理账户分享某一个商品库给广告主

<strong>请求地址</strong>

https://ads.cloud.huawei.com/openapi/v2/tools/dpa/product/share\_store/create

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| manager\_id | string | 是 | 经理账户 |
| store\_id | string | 是 | 商品库ID |
| advertiser\_ids | String[] | 是 | 该商品库共享的广告主id集合，不包括该商品库的所有者 |

<strong>请求示例</strong>

POST openapi/v2/tools/dpa/product/share\_store/create HTTP/1.1

Accept:application/json

Content-Type:application/json

Authorization:Bearer CgB6e3x9ERGComr9dENxZX22iBk+mLuf1yGtQVPUjPJUMrstfKlqpdXk+kfHU9J8ZJ/soYIZHZzT446GeSYumluQuhsK7jvz4kz1Bkms4CLI/rE=

```
{

```

| code | string | 返回码  200：成功  500：失败 |
| message | string | 返回描述 |
