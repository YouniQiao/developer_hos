# 查询RTA策略

【简介】通过此接口可以查询RTA策略信息，频率限制：每个账号&lt;=100次/min。

<strong>请求地址</strong>

https://ads.cloud.huawei.com/openapi/v2/tools/rta\_authority/query

<strong>请求方法</strong>

<strong>GET</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |

<strong>请求示例</strong>

========================================================================================================

GET openapi/v2/tools/rta\_authority/query?advertiser\_id=478911039618966272 HTTP/1.1

Accept:application/json

Content-Type:application/json

Authorization:Bearer CgB6e3x9ERGComr9dENxZX22iBk+mLuf1yGtQVPUjPJUMrstfKlqpdXk+kfHU9J8ZJ/soYIZHZzT446GeSYumluQuhsK7jvz4kz1Bkms4CLI/rE=

=========================================================================================================

<strong>响应字段</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| code | string | 是 | 返回码 |
| message | string | 否 | 返回描述 |
| data | Struct1 | 否 | RTA权限列表 |

Struct1定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| corp\_name | string | 否 | 企业名称 |
| advertiser\_id | long | 否 | 广告主ID |
| strategy\_of\_failure | string | 否 | RTA请求失败后的策略  详见[【RTA请求失败后的策略】](/docs/monetize/promotion/ads_api73-0000001058884698#section1332219296555) |
| rta\_cache\_time\_min | integer | 否 | RTA缓存时间窗 |
| rta\_id | string | 否 | RTA推广类型ID |
| rta\_url | string | 否 | RTA接口地址 |
