# 查询定向包

【简介】通过此接口可以查询定向包。

<strong>请求地址</strong>

https://ads.cloud.huawei.com/ads/v1/tools/targeting\_package/query

<strong>请求方法</strong>

<strong>GET</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| page | integer | 是 | 搜索页码，  取值范围1~10000 |
| page\_size | integer | 是 | 一页展示数量，  取值范围 10~50 |
| filtering | Struct1 | 否 | 过滤条件，若此字段不传，或传空则视为无限制条件 |

filtering (Struct1)定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| targeting\_type | string | 否 | 定向包类型，  包含应用类、非应用类;  详见[【定向包类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section1368511219507) |

<strong>请求示例</strong>

GET /ads/v1/tools/targeting\_package/query HTTP/1.1

Accept:application/json

Content-Type:application/json

Authorization:Bearer CgB6e3x9ERGComr9dENxZX22iBk+mLuf1yGtQVPUjPJUMrstfKlqpdXk+kfHU9J8ZJ/soYIZHZzT446GeSYumluQuhsK7jvz4kz1Bkms4CLI/rE=

```
{
"page": "1",
"page_size": "50",
"filtering":{
"targeting_type":"TARGET_TYPE_NOT_APP"
}
}
```

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1 | 定向包列表 |

Struct1定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| total | integer | 定向包总条数 |
| data | Struct2[] | 定向包列表 |

Struct2定义

|  |  |  |
| --- | --- | --- |
| <strong>参数字段</strong> | <strong>类型</strong> | <strong>描述</strong> |
| targeting\_id | long | 定向包ID |
| targeting\_name | string | 定向包名称 |
| targeting\_description | string | 定向包描述 |
| targeting\_type | string | 定向包类型，  包含应用类、非应用类  详见[【定向包类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section1368511219507) |
| gender\_struct | Struct3 | 性别定向，值为gender定向中的value |
| age\_struct | Struct3 | 年龄定向，值为age定向中的value |
| series\_type\_struct | Struct3 | 手机系列定向，值为series\_type定向中的value |
| device\_price\_struct | Struct3 | 设备价格定向，值为device\_price定向中的value |
| installed\_apps\_struct | Struct3 | 已安装应用定向，如果选择此定向，value不为空 |
| not\_installed\_apps\_struct | Struct3 | 未安装应用定向，如果选择此定向，value不为空 |
| network\_type\_struct | Struct3 | 网络类型定向，值为network\_type定向中的value |
| audience\_struct | Struct3 | 自定义人群定向，值为pre\_define\_audience定向中的id |
| not\_audience\_struct | Struct3 | 排除自定义人群定向，值为not\_pre\_define\_audience定向中的id |
| current\_geo\_location\_struct | Struct3 | 当前在这里的人（地理划分）  值为current\_dmp\_geo中的value |
| current\_level\_location\_struct | Struct3 | 当前在这里的人（发展划分）  值为current\_level\_geo中的value |
| residence\_geo\_location\_struct | Struct3 | 常驻在这里的人（地理划分）  值为current\_dmp\_geo中的value |
| residence\_level\_location\_struct | Struct3 | 常驻在这里的人（发展划分）  值为current\_level\_geo中的value |
| current\_custom\_location\_struct | Struct3 | 自定义地理位置定向 |
| is\_retain\_assistant | boolean | 是否为次留定向包  true为是  false为否 |
| ai\_target | Struct3 | 智能扩量  详见[【智能扩量】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section3184194510) |
| ai\_target\_flag | string | 智能扩量开关  详见[【智能扩量开关】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section3590746181218) |
| feature\_audience\_struct | Struct3 | 细分受众定向  值为feature\_audience的value |
| content\_id\_struct | Struct3 | 内容ID定向 |
| content\_label\_struct | Struct3 | 内容标签定向 |
| brand\_id\_struct | Struct3 | 品牌门店定向 |
| targeting\_scope | string | 标记定向包类型 |
| accept\_roaming\_flag | string | 漫游流量开关 |
| adgroup\_count | integer | 已关联任务数 |

Struct3定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| value | string[] | 定向选中的值 |
