# 查询渠道包投放数据报表

1. 登录[华为应用市场应用推广平台](https://developer.huawei.com/consumer/cn/service/apcs/app/home.html)，选择“报表”。

   ![](./img/610426bc4745.png)
2. 点击“自定义列”。

   ![](./img/49df915597c0.png)
3. 在“归因”区域中勾选“物理渠道包ID”，点击“确定”，即可查看渠道包投放数据。

   ![](./img/14429e8fb747.png)
