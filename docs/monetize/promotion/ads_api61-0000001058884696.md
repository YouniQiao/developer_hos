# 查询转化跟踪目标

通过此接口可以查询转化跟踪目标，需要向运营申请转化指标权限

<strong>请求地址</strong>

https://ads.cloud.huawei.com/openapi/v2/tools/effect\_tracking/query

<strong>请求方法</strong>

<strong>GET</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| page\_num | integer | 是 | 搜索页码  取值范围1~10000 |
| page\_size | integer | 是 | 一页展示数量  取值范围 10~50 |
| 以下为过滤字段 | | | |
| product\_unique\_flag | string | 否 | 推广产品的唯一标识，应用是包名，网页是URL链接 最大长度是2048位 |
| effect\_name | string | 否 | 转化指标名称 最大长度255位，满足^[\w\u4e00-\u9fa5\_ ]\\{1,20\\}$  正则校验 |
| effect\_type | string | 否 | 转化目标  详见[【转化目标】](/docs/monetize/promotion/ads_api73-0000001058884698#section895919210500) |
| product\_type | string | 否 | 推广产品  只能使用WEB、ANDROID\_APP  详见[【推广产品】](/docs/monetize/promotion/ads_api73-0000001058884698#section1526319223472) |
| imp\_tracking\_url | string | 是 | 曝光监测地址 |

<strong>请求示例</strong>

=========================================================================================================

Accept:application/json

<strong>响应字段</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| code | string | 是 | 返回码 |
| message | string | 否 | 返回描述 |
| data | Struct1 | 否 | 转化跟踪指标列表 |

Struct1定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>类型</strong> | <strong>参数名称</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| total | integer | 是 | 转化跟踪指标总条数 |
| data | Struct2[] | 否 | 转化跟踪指标列表 |

Struct2定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| tracking\_id | long | 是 | 转化跟踪指标ID |
| tracking\_status | string | 是 | 跟踪状态  详见[【转化跟踪状态】](/docs/monetize/promotion/ads_api73-0000001058884698#section128161035145020) |
| effect\_name | string | 是 | 转化跟踪指标名称 |
| effect\_type | string | 是 | 转化目标  详见[【转化目标】](/docs/monetize/promotion/ads_api73-0000001058884698#section895919210500) |
| click\_tracking\_url | string | 是 | 点击监测地址 |
