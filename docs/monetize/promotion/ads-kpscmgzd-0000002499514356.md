**精美度**

|  |  |
| --- | --- |
| 【**案例解析】：**保险行业的素材，整体设计比较简陋，未体现出该行业属性相关联的元素，缺乏氛围的营造和主体塑造，缺乏可信度与品质感。 | **【****正确指引****】****：**明确保险产品购买目标群体，使用轻松的插画风格营造温馨的家庭场景与目标群体产生共鸣。 |
|  |  |

**图片内容与推广内容无关联**

|  |  |
| --- | --- |
| 【**案例解析】：**保险行业使用动物/风景/水果等与行业属性无关联的内容，误导性强。 | **【****正确指引****】****：**鉴于保险行业没有实体的产品，建议选择用插画形式来展现，“盾牌”有“守护”的寓意，符合该行业属性。 |
|  |  |

**安全区**

|  |  |
| --- | --- |
| 【**案例解析】：**LOGO/文案超出安全区域（红框内）。 | **【****正确指引****】****：**主要内容LOGO、文案调整到安全区内。 |
|  |  |

**真实感**

|  |  |
| --- | --- |
| 【**案例解析】：**  1. 图片整体色调偏“灰”，产品不清晰；  2. 没有明显的主色调；  3. 该图片化妆品与背景搭配不和谐，缺 乏真实感（现实生活中不会有瓶子无牵引力的作用下浮空）；  4. 背景没有与产品属性相关的氛围烘托。 | **【****正确指引****】****：**  1. 原始产品颜色偏“灰”，调整对比度更清晰；  2. 提取产品颜色为主色调；  3. 添加体现真实感的倒影；  4. 背景没有与产品属性相关的氛围烘托。 |
|  |  |

**内容构图**

|  |  |
| --- | --- |
| 【**案例解析】：**照⽚未遵循了基本的拍摄原理，构图杂乱⽆章，使用了生活场景随处可拍、元素密集、环境嘈杂、复杂背景的图片。 | **【****正确指引****】****：**选择遵循基本拍摄原理、讲究构图角度和光线的图片。 |
|  |  |

**设计搭配协调**

|  |  |
| --- | --- |
| 【**案例解析】：**标题设计样式、颜色过多，属于过度设计，缺乏品质感。 | **【****正确指引****】****：**配色选择产品主色，不过多使用设计样式去繁就简，信息传达明确、清晰。 |
|  |  |

**调性**

|  |  |
| --- | --- |
| 【**案例解析】：**用利益点作为主标题（以字号和面积占比判断） | **【****正确指引****】****：**利益点减少字号和面积占比（明显比主标题小1/3以上） |
|  |  |
