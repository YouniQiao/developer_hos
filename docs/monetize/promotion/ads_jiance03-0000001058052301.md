# FAQ

<strong>Q1：为什么要使用第三方监测？</strong>

通过委托第三方监测，可以核对实际发生的展示或点击数量，以确保广告主的利益。

<strong>Q2：支持的监测指标有哪些？</strong>

第三方监测支持曝光和点击两项指标的监测。

<strong>Q3：第三方监测数据和投放端的数据是否存在较大差异？</strong>

通过与已经对接的第三方监测公司实测得到结果，数据差异一般不超过5%。这些差异主要是由于网络原因、各方反作弊机制等因素造成。

以投放端的数据为准结算实际广告费用，第三方监测公司提供的曝光/点击量仅供参考。
