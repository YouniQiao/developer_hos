# 创建素材

【简介】通过此接口可以创建素材，您也可以通过本接口对您的图片尺寸和大小进行预处理，target\_width、target\_height、target\_file\_size\_kb\_limit同时传才会触发图片裁剪压缩，PNG格式图片不能压缩。

<strong>请求地址</strong>

https://ads.cloud.huawei.com/ads/v1/tools/creative\_asset/create

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| file\_token | string | 是 | 文件上传凭证  详见[【获取文件上传凭证】](/docs/monetize/promotion/ads_new_api34-0000001405360444) |
| file | File | 是 | 文件内容  注：文件上限为50M  文件格式为：jpg，jpeg，png，mp4，webp  注：webp格式只允许上传尺寸低于512\*512并且宽高比例为1:1的图片 |
| asset\_name | string | 是 | 素材名称，不为空且长度不超过128  不能包含”’”,”\”,”\\{”,”\\}”,”%”,”\*”,”^”,”#”,”&lt;”,”&gt;”,”|”特殊字符 |
| target\_width | integer | 否 | 图片处理后的目标宽，范围在1-1080 |
| target\_height | integer | 否 | 图片处理后的目标高，范围在1-1920 |
| target\_file\_size\_kb\_limit | integer | 否 | 图片处理后目标文件大小最大值KB，范围在1-9000 |
