# 常见问题

1. <strong>图片广告曝光的统计口径是什么？</strong>

   目前鲸鸿动能广告平台统计广告曝光的口径参照了The Interactive Advertising Bureau的相关行业标准。具体口径为：广告内容露出50%以上且曝光0.5秒以上统计为一次有效曝光。
2. <strong>如何预览广告？</strong>
   1. 上传创意的时候，右侧默认展现广告预览效果。
   2. 若您需要在手机上预览，则需您创建[试投放](/docs/monetize/promotion/ads-adtest-0000001190031279)任务，若您仍无法预览广告，请通过[在线提单](https://developer.huawei.com/consumer/cn/support/feedback/#/)联系鲸鸿动能客服。
3. <strong>为什么有些功能文档上可见，但是广告平台界面上看不到？</strong>

   部分功能由[特性通行名单](/docs/monetize/promotion/addtongxing-0000001128278195)控制，请通过[在线提单](https://developer.huawei.com/consumer/cn/support/feedback/#/)联系鲸鸿动能客服。
4. <strong>为什么有些版位不可见？</strong>

   广告账户的注册国家/地区不同，允许投放的版位会存在差异。
5. <strong>广告平台的时区可以修改吗？</strong>

   暂不支持修改，广告后台的时区是由您在注册广告账户时所设置的。
6. <strong>落地页内支持增加跳转其他页面吗？</strong>

   以应用下载为目的的落地页暂不支持增加跳转其他页面，以网页推广为目的的落地页目前允许增加跳转其他页面。
7. <strong>使用人群包广告效果一定会变好吗</strong>？

   不一定，广告效果影响因素很多，只有根据人群包设置的目标人群与实际潜在用户人群匹配度较高时，才能够提升广告效果。
8. <strong>定向条件越多，广告投放效果会越好吗？</strong>

   不一定，影响广告投放效果的因素很多，若您所选择的人群包与实际潜在用户人群匹配度越高，广告效果好的概率就越高，但过多的定向可能影响触达人数，进而影响任务的曝光。
9. <strong>广告素材有什么需要注意的吗？</strong>

   素材内容：应满足[审核规则](/docs/monetize/promotion/review-0000001052064324)。

   素材语言：素材使用的语言可以根据投放需求来调整。如果广告定向设置了特定的语言，广告素材使用的语言应该与广告定向设置的语言一致。广告将会展示给手机上设置特定语言的用户。如果广告定向未设置特定的语言，素材使用的语言应该与投放地域主流语言一致。
10. <strong>为什么人均触达次数数据比任务频控设置大？</strong>

    由于人均触达次数指标和投放频次口径不同，不能在同一个场景中进行对比。投放频次指的是单一用户在指定时长内的广告展示次数。人均触达次数指的是总曝光/总触达用户数。触达用户数指的是多天单人展示多次记1个人。

    例如：任务A投放频次设置为单人单天展示<strong>4</strong>次，则人均触达次数如下表所示：

    |  |  |  |  |
    | --- | --- | --- | --- |
    | 时间 | 8月4号 | 8月5号 | 总计 |
    | 触达用户数（人） | 100 | 150（包含新触达50） | 150 |
    | 总曝光（次） | 400 | 4\*100+（4\*50）=600 | 1000 |
    | 人均触达次数（次） | 1000/150=6.67 | | |
