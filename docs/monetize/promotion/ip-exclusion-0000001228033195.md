# 排除IP管理

对投放到搜索广告网络的任务，如果您希望不向特定的IP地址投放，可以添加“<strong>排除IP</strong>”。

单击“<strong>工具</strong>”-&gt;“<strong>排除IP管理</strong>”-&gt;“<strong>添加IP</strong>”，支持批量添加/删除，每行一条，最多300行。
