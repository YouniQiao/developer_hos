# 2025年10月

<strong>典型驳回行业：</strong>电商、婚恋、陌生社交、游戏、软件应用。

<strong>典型驳回问题场景主要涵盖：</strong>

一、广告不得出现低俗、恶俗、荤黄段子、性暗示、性联想等不良导向内容及场景。

二、广告内容不得使用恶意诱导点击的设计元素、明显虚假的欺骗性内容、违背用户意愿、无法实现其功能的虚假设计。

三、创意推广产品需与开户报备信息保持一致。

四、广告文案、素材、落地页三者推广内容/活动/利益点/引导语等需保持一致，不得出现图文不符、外层素材与落地页不符、利益点不符等问题。

五、广告内容不得使用仿系统消息推送或仿系统操作功能。

六、广告不支持使用奇闻异事、耸人听闻类等容易引起用户心理不安的负面噱头式标题。

七、广告素材不得出现灵异鬼怪、惊悚恐怖、暴力血腥等易引起用户不适的描述或画面。

<strong>一、广告不得出现低俗、恶俗、荤黄段子、性暗示、性联想等不良导向内容及场景</strong>

1.不得使用居住状态+人物需要慰藉的状态的低俗导向内容；如：租房/在家+找人陪/孤独/寂寞等。

2.不得使用含有晚上时间描述；如“晚上”“夜聊”“夜晚”“深夜”“午夜”“半夜”“子夜”“凌晨”等。

3.不得涉及人物+举牌/白纸/黑板/手机/平板等场景画面。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> | <strong>通过示例</strong> |
| --- | --- | --- |
|  |  |  |
| ×问题解析：一个人+孤单 | ×问题解析：含有“晚上”描述 | √支持使用生活状态描述，无慰籍或其他歧义描述类的 |

4.不得涉及明显暗示性的动作和表情，如：刻意抚摸身体、舔舐食物、舔咬嘴唇、真人接吻及其他暗示性动作等。

5.不得使用突出敏感部位、穿着带有引诱性质的人物形象。

6.不得使用过于暴露的游戏、漫画人物形象。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> | <strong>驳回示例</strong> |
| --- | --- | --- |
|  |  |  |
| ×问题解析：导向不佳+穿着引诱 | ×问题解析：突出敏感部位 | ×问题解析：性暗示动作 |

<strong>二、广告内容不得使用恶意诱导点击的设计元素、明显虚假的欺骗性内容、违背用户意愿、无法实现其功能的虚假设计</strong>

1.禁止使用违背用户意愿、无法实现功能的虚假设计，如：关闭、暂停、取消、放弃、跳过等。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> | <strong>通过示例</strong> |
| --- | --- | --- |
|  |  |  |
| ×问题解析：无法实现忽略按钮 | ×问题解析：无法实现取消按钮 | √支持可实现其操作功能的描述，如：立即前往/领取，点击开抢等 |

2.禁止使用恶意诱导点击的设计元素，包括不限于以下设计样式。

<strong>三、创意推广产品需与开户报备信息保持一致</strong>

<strong>四、广告文案、素材、落地页三者推广内容/活动/利益点/引导语等需保持一致，不得出现图文不符、外层素材与落地页不符、利益点不符等问题</strong>
