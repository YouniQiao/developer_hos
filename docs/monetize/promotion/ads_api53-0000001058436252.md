# 查询渠道包

通过此接口可以查询渠道包，需要向运营申请渠道包白名单权限

注：此接口即将下线，如需继续使用请使用[【查询可用渠道包接口】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_api058-0000001262978362)

<strong>请求地址</strong>

https://ads.cloud.huawei.com/openapi/v2/tools/channel\_package/active\_package/query

<strong>请求方法</strong>

<strong>GET</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| store\_app\_id | string | 否 | 华为应用市场App ID 只能包含数字和字母，且最大长度为32位 |
| package\_name | string | 否 | 应用包名 长度1到128位 |

<strong>请求示例</strong>

GET openapi/v2/tools/channel\_package/active\_package/query?store\_app\_id=XXX HTTP/1.1

Accept:application/json

Content-Type:application/json

Authorization:Bearer CgB6e3x9ERGComr9dENxZX22iBk+mLuf1yGtQVPUjPJUMrstfKlqpdXk+kfHU9J8ZJ/soYIZHZzT446GeSYumluQuhsK7jvz4kz1Bkms4CLI/rE=

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1[] | 渠道包列表 |

Struct1定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| channel\_package\_id | string | 渠道包id |
| name | string | 渠道包名称 |
| package\_name | string | 应用包名 |
