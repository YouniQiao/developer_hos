# 2026年鲸鸿动能短剧端原生投流激励政策

<strong>政策周期：</strong>2026年4月1日-6月30日

<strong>面向对象</strong>：短剧端原生投流广告主

<strong>参与条件</strong>：通过华为鲸鸿动能广告渠道推广该剧目后，产生广告变现收益

<strong>报名方式</strong>：无需报名，满足条件的均可参加

<strong>互斥政策</strong>：与年框、星火、营销大促互斥

<strong>激励内容</strong>：针对短剧端原生投流竞价现金消耗部分给予30%-40%返利金激励

|  |  |
| --- | --- |
| <strong>月度买量现金消耗额X（万元）</strong> | <strong>现金消耗部分激励比例</strong> |
| 10&lt;=X＜300 | 30% |
| 300&lt;=X | 40% |

*注：仅限鲸鸿动能DSP，综合DSP不适用本激励*

激励发放形式和使用：

|  |  |  |
| --- | --- | --- |
| <strong>竞价投放</strong> | <strong>发放周期</strong> | <strong>有效期</strong> |
| 返利金充值 | 月度 | 返利金发放后90天 |

<strong>结算方式：</strong>

1. 账单和激励金额数据以鲸鸿动能数据为准。
2. 激励计算按照“短剧投流版位”的现金消耗部分计算月度累计消耗。
3. 如发现不符合激励条件的情况，鲸鸿动能将对数据进行复核，并仅在符合激励条件的范围内发放激励，金额以复核调整后的为准。次月发放后，如果客户有异议，可以通过服务商发邮件向鲸鸿动能反馈问题，鲸鸿动能进行数据复核。
4. 如涉及数据差异的，数据差异在鲸鸿动能数据的10%以内时，以鲸鸿动能数据为准；如差异超过10%，请于五个工作日内发正式邮件反馈数据差异排查，双方友好协商一致确认最终数据；如未及时反馈，视为确认以鲸鸿动能数据为准。

<strong>补充说明：</strong>

1. 返利金专款专用，仅用于短剧端原生投流，有效期自下发之日起90天
2. 激励计算按照“短剧投流版位”的现金消耗部分计算月度累计消耗。该版位投放需要开通白名单权限，请联系对应行业运营开通权限。
3. 返利金由鲸鸿动能平台在推广产品达标后次月直接发放至客户账户，涉及多个账户的，按各账户现金消耗比例进行分摊返利金，原则上不接受调整。
4. <strong>适用政策期间，推广产品及相关推广投放应保持合法合规运营，不得出现违反法律法规或侵犯第三方合法权益的情形，否则鲸鸿动能有权取消推广产品的政策适用资格并采取相关处罚措施。</strong>
5. <strong>服务商、客户享受鲸鸿动能提供的返利金等权益，需以其账户处于有效且可操作状态为前提。因广告主或服务商相关账户注销等账户状态异常情形，致使鲸鸿动能客观上无法向其账户成功充值返利金或发放其他权益的，由此产生的返利金等无法发放的后果，由服务商或客户自行承担，并视为其默认放弃该等权益，鲸鸿动能将不再进行发放操作。</strong>
6. <strong>异常作弊、违规操作、恶意修改预算/出价、利用系统漏洞、滥用系统机制、可疑或非法交易等不当行为不可享受激励。经鲸鸿动能合理判定服务商或客户存在前述行为的，鲸鸿动能有权拒绝发放激励并追回已发放的激励，并采取相应处罚措施，由此造成的损失或赔偿责任由服务商或客户承担。</strong>
7. 如因产品功能、计算错误、系统错误等任何原因导致激励超额发放、重复发放或错误发放，一经发现，鲸鸿动能有权追回多发部分或按核实后的应发激励调整发放。
8. 对于应追回的激励金额，鲸鸿动能可按以下方式扣除或追缴：（1）从下期激励中扣除；（2）从服务商或客户激励账户中直接扣除；（3）从服务商或客户的充值账户等其他支付、结算账户中扣除；（4）从鲸鸿动能尚未向服务商或客户支付的各种激励款项中直接扣除；（5）从服务商或客户在华为平台的其他激励、账户或待结算款项中扣除；（6）采取其他可行的方式要求服务商或客户在30个工作日内退回、支付或赔偿。
9. 如对本政策有任何疑问，请联系鲸鸿动能。
