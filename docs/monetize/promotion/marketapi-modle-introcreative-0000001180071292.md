# IntroCreative

| 参数 | 必选(M)/可选(O) | 类型 | 描述 |
| --- | --- | --- | --- |
| linkType | O | String | 创意类型。  取值范围：   - DEFAULT：默认的应用详情页 - DEEPLINK：开发者输入的DeepLink地址 - H5：自定义落地页 |
| pageId | O | String | 创意在应用市场落地页的ID。 |
| linkUrl | O | String | 创意落地页或者卡片的DeepLink访问地址。 |
