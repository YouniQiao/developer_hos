# 进入认证中心完成开户

以选择开通应用市场应用推广一级服务商为例，勾选完推广类型、签署协议后，您只需要在认证中心完成实名认证，并补充开票信息即可。

![](./img/zh-cn_image_0000002466019206_c054bbc67600.png "点击放大")

## 实名认证

点击【去认证】跳转华为开发者联盟进行实名认证。

1. 选择企业类型后，可通过对公银行打款认证或者企业资料人工审核认证完成实名认证。

   ![](./img/zh-cn_image_0000002499178169_5912c53afdb3.png "点击放大")
2. 对公银行打款认证，操作步骤参考文档：[对公银行打款认证](https://developer.huawei.com/consumer/cn/doc/start/atpopb-0000001062836624)。

   ![](./img/zh-cn_image_0000002499138217_a87703ea9717.png "点击放大")

   ![](./img/zh-cn_image_0000002466178858_7bc358f3abb0.png "点击放大")

   ![](./img/zh-cn_image_0000002466019210_69cb29b9e2d8.png "点击放大")
3. 企业资料人工审核认证，操作步骤参考文档：[企业资料人工审核认证](https://developer.huawei.com/consumer/cn/doc/start/mracoei-0000001062678404)。
   1. 上传手持身份证

      ![](./img/zh-cn_image_0000002499178177_17d9490a86d4.png "点击放大")

      ![](./img/zh-cn_image_0000002499138225_76059503fcc7.png "点击放大")

      ![](./img/zh-cn_image_0000002466178862_5d8a69c61807.png "点击放大")
   2. 人脸识别

      ![](./img/zh-cn_image_0000002466019218_52bd2db5b1dd.png "点击放大")

      ![](./img/zh-cn_image_0000002499178181_08e20c52253b.png "点击放大")

## 开票处理

点击【去补充】，跳转开发者联盟开通付费服务，并补充发票信息。官网指导文档：[付费服务](https://developer.huawei.com/consumer/cn/doc/start/payment-service-0000001052865979)。
