# 创建应用市场合约广告

## 概述

应用市场的一种合约广告形式，在开屏、banner等广告形式中固定投放广告，无需竞价即可展示，计费方式为CPD、CPT。

## 应用市场合约广告合作流程

![](./img/zh-cn_image_0000001903898176_66b4fe7793ce.png "点击放大")

## 应用市场合约广告操作步骤

1. 联系[鲸鸿动能运营](https://developer.huawei.com/consumer/cn/doc/promotion/customerservice-0000001052264359#section676133614717)确认合作信息。
2. 鲸鸿动能运营创建合约广告任务。
3. 广告主上传创意。

   应用市场合约广告任务创建完成后，任务将会同步至您提供的广告账户里，任务状态显示“待上传创意“，您需要按照不同广告样式上传不同的创意素材并提交审核。
4. 审核并投放广告。
   - 如果您需要修改、暂停应用市场合约广告任务，请联系华为运营。
   - 在广告投放前，如果您需要修改创意，系统将会重新审核创意，审核通过后才能投放。
   - 广告投放中，创意不可修改。
