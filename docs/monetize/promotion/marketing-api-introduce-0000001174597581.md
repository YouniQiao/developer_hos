# 术语介绍

| <strong>术语</strong> | <strong>详细描述</strong> |
| --- | --- |
| 客户端 ID | 鲸鸿动能广告平台Marketing API 客户端ID。 |
| 密钥 | 每一个客户端对应的密钥（用于后面获取access\_token）。 |
| scope | 广告主授予的权限列表。 |
| redirect\_uri | 授权后回调的URI，用于接收Authorization Code。 |
| access\_token | 使用/oauth2/v2/token接口会同时获得access\_token。access\_token是调用API接口的调用凭证,用于服务端对API请求鉴权，默认有效期为1小时，一旦access\_token失效，则需要重新获取。 |
