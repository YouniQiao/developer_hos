# 合作伙伴分级权益规则

华为应用市场为合作伙伴提供各项基础权益及增值权益，助力合作伙伴高效开展应用推广业务。合作伙伴可以在推广后台查看账户内应用的分级信息。

登录[华为应用市场应用推广平台](https://ads.huawei.com/cn/)，在顶部菜单栏点击【工具】页签，确认推广范围为“应用市场应用推广”。点击“账户辅助”——“客户分级权益”进入查看账户内应用的等级。

**其他说明：**

- 打**√**项目为分级客户可享有权益。
- 铂金级权益（四选一）：1天热词/5天安装打开提醒/5天营销语/5天角标。
- 金级权益（三选一）：3天安装打开提醒/3天营销语/3天角标。
- 上月为黑金级级别的客户，本月若继续保持黑金级，则可兑换的角标天数增加至10天。
- 如特殊原因无法进行资源兑换或兑换时间过期，权益将自动失效。
- 合作伙伴分级权益规则仅支持HarmonyOS 4.X及以下版本应用市场推广应用参与活动。
