# 查询账户充值记录

【简介】查询广告主的账户充值记录信息，一级服务商账户和直客广告主可以通过此接口查询充值记录。

请求地址

https://ads.cloud.huawei.com/openapi/v2/finance/recharge/query

<strong>请求方法</strong>

<strong>GET</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| start\_date | string | 否 | 付款起始时间 yyyy-MM-dd |
| end\_date | string | 否 | 付款结束时间yyyy-MM-dd  时间跨度最大一年 |
| page\_num | integer | 是 | 搜索页码  取值范围1~200 |
| page\_size | integer | 是 | 一页展示数据数量  取值范围10~50 |
| recharge\_type | string | 是 | 充值类型  详见[【充值类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#充值类型) |
| recharge\_status | string | 是 | 充值状态  详见[【充值状态】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#充值状态) |
| receipt\_status | string | 是 | 发票状态  详见[【发票状态】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#发票状态) |
| recharge\_usage | string | 否 | 充值用途。不填默认查全部充值用途  详见[【充值用途】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#充值用途) |
| user\_service\_type | string | 否 | 账户推广业务类型，不填默认查询ADS侧数据，详见[【账户推广业务类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#账户推广业务类型) |

<strong>请求示例</strong>

GET openapi/v2/finance/recharge/query?start\_date=2020-03-08&

end\_date=2021-03-08&page\_num=1&page\_size=2

&recharge\_type=ALL\_RECHARGE&recharge\_status=ALL\_RECHARGE

&receipt\_status=ALL\_RECHARGE&recharge\_usage=ALL&user\_service\_type=ADS

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| total | integer | 信息总数 |
| data | Struct1[] | 充值信息列表 |

Struct1定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| recharge\_id | string | 充值订单ID |
| recharge\_amount | bigdecimal | 充值金额 |
| currency | string | 币种 |
| created\_time | string | 订单创建时间  yyyy-MM-dd HH:mm:ss |
| pay\_time | string | 订单支付时间  yyyy-MM-dd HH:mm:ss |
| recharge\_status | string | 充值类型  详见[【充值类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#充值类型) |
| recharge\_type | string | 充值状态  详见[【充值状态】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#充值状态) |
| receipt\_status | string | 发票状态  详见[【发票状态】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#发票状态) |
| remark | string | 描述信息 |
| sign\_party | string | 签约主体  详见[【签约主体】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#签约主体) |
| account\_type | string | 账户类型标识  详见[【资金账户类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#资金账户类型) |
| media\_type | string | 适用媒体范围  详见[【虚拟账户类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#虚拟账户类型) |
