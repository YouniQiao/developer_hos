# 推广不合作应用类型

华为应用市场推广服务暂不合作应用类型如下表所示：

| 不合作应用类型 | | |
| --- | --- | --- |
| 手机基础类应用 | 桌面、锁屏类应用 | 安全辅助类应用 |
| 成人用品类应用 | 手机克隆类应用 | 网络电话类应用 |
| 数据恢复类应用 | 微商助手类应用 | 宗教类应用 |
| 街机类应用 | 纯网赚类应用 | 去水印应用 |
| 自动抢票类应用 | 游戏代练类应用 | 游戏账号交易类应用 |
| 手机号定位类应用 | 简单套壳，未提供实际服务的应用 | 第三方手机厂商官方手机商城APP |
| 跑分类应用 | 游戏攻略、游戏辅助工具及社区 | 按摩保健类应用 |

- 具体<strong>应用市场不收录应用</strong>请参见[不收录类型](https://developer.huawei.com/consumer/cn/doc/50117)。
- <strong>手机基础类应用</strong>即手机系统简易工具，如天气、手电筒、指南针、智能遥控等。
