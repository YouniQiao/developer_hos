# 法律法规禁投内容

|  |  |
| --- | --- |
| <strong>类别</strong> | <strong>描述</strong> |
| 医疗健康、保健 | 1. 不得宣传国家要求暂停发布医疗广告的疾病：尖锐湿疣、梅毒、淋病、软下疳、牛皮癣（或银屑病）、艾滋病、癌症（或恶性肿瘤）、癫痫、乙型肝炎、白癜风、红斑狼疮。 2. 广告中也不得出现上述疾病的专业性术语，如乙型肝炎的专业术语，包括HBV、大三阳、小三阳转阴、两对半等。 3. 麻醉药品、精神药品、医疗用毒性药品、放射性药品、药品类易制毒化学品，处方药、特殊医学配方食品中的特定全营养配方食品、特殊医学用途婴儿配方食品以及戒毒治疗的药品及医疗器械。 4. 军队特需药品、军队医疗机构配制的制剂。 5. 医疗机构配制的制剂。 6. 依法停止或者禁止生产、销售或者使用的药品、医疗器械、保健食品和特殊医学用途配方食品。 7. 未在监管部门报备的药品、保健品、医疗器械。 8. 医疗废物相关，如医疗废物回收等。 9. 提供海外就医中介服务，收取中介费的行为。 10. 倒卖医院挂号凭证，黄牛倒卖号源等加价牟利行为。 11. 非法医疗服务，如非法试药、代孕、胎儿性别鉴定、性别控制、售卖多子丸等违法生育产品或服务。 12. 非法医疗交易，如人体器官、遗体，动物尸体买卖交易等行为。 13. 推广“三品一械”及医疗服务相关的卡券或促销优惠等诱导性消费产品，包括但不限于：红包、现金券、体验券、礼品券、折扣券、特价券、换购券、提货券等。 |
| 金融 | 1. 仿冒银行、基金、证券、信托等金融类网站。 2. 违规使用“银行”字样，包括法人、商标名称、产品业务名称。 3. 复制/破解或回收银行卡、非银行官网代办银行卡、代办银行流水、买卖银行账户（银行卡）等。 4. 代理提取公积金、社保。 5. 伪造/变造/买卖信用卡等金融票证，信用卡套现、代还信用卡等业务。 6. 无证、网销POS机销售、利用POS机套现、刷卡返利。 7. 租机贷，指的是：以手机等电子设备租赁为名的变相非法高利贷。 8. 非法贷款产品，如校园贷、学生贷、首付贷、美容贷等。 9. 非法金融业务，如个股推荐、股票配资、股票杠杆、配资杠杆、高倍杠杆（3 倍及以上）、杠杆神器、以小博大、稳赚、高收益低风险等误导性/承诺收益。以及港美股交易、涉及证券基金等内幕消息、内部交易等业务、非法原油开采及原油衍生品等。 10. 私募基金、私募型金融产品。 11. 非法提供清债、欠债、讨债等相关服务或咨询的内容。 12. 提供反催收服务：不支持暴力对抗催收机构提供违法类服务内容，包括但不限于：假冒信息、征信修复、洗白等。 13. 典当类：涉及法律、行政法规禁止买卖的物品或财产权利，如抵押象牙、文物售卖等行业产品拍卖。 14. P2P投资理财内容，P2P（peer-to-peer lending）：指的是个人对个人网络借贷，属于互联网金融产品，通过互联网平台实现资金供需对接。 15. 非法公益募捐，包括但不限于：未备案公益平台，非“慈善中国”官网公示的公开募捐服务平台，公益募捐出现收集用户信息以及商业引流等。 |
| 电商 | 1. 发布招聘内容涉及网店刷信誉人员等。 2. 电商刷单、补单、刷流量、空包代发、黑搜（黑搜索)、黑车（黑科技直通车)、黑钻（黑科技钻展）等网络电商黑产服务。 3. 未履行正常进口手续的商品，比如"水货"、”欧水”、"港水"等类似产品。 |
| 软件应用 | 1. 存在扣费项目不明确、恶意扣费、暗设扣费程序等任何损害用户权益的情况，或含有盗号、窃取密码等恶意程序的产品。 2. 涉及恶意诱导、损害消费者权益的服务或业务，例如以免费、隐瞒、低价等方式恶意诱导消费者使用、消费其服务。 3. 宣传通过特定方式实现免费、无需支付传统电话费用即可直接拨打电话的软件或产品。 4. 利用非法网络技术突破网络限制，修改IP服务，侵犯他人隐私、财产或扰乱互联网平台正常经营秩序的违法违规行为，包括：“翻墙”服务、“翻墙”软件、越狱软件、修改IP所属地相关的方法、工具、教程等。 5. 时政类涉及境内外军事政治资讯、反动言论等软件或产品。 |
| 游戏 | 1. 赌博、色情等影响社会良好风尚类的游戏。 2. 涉及翻墙、越域、私服、外挂等非法功能的游戏。 3. 未经授权销售其他厂商游戏装备、冒充其他游戏官网、设立钓鱼网站等。 |
| 通讯虚拟 | 1. 已激活的手机卡、上网卡等违反国家实名制规定的商品。 2. 非法接入信号产品，包括但不限于：电视棒、信号接收器、卫星天线等非法信号接收设备。 3. 涉及非法制造/买卖/提供或使用用于实施电信网络诈骗等违法犯罪的设备/软件的违法违规功能或内容，包括但不限于：    - 电话卡批量插入设备；    - 具有改变主叫号码、虚拟拨号、互联网电话违规接入公用电信网络等功能的设备、软件，如应用为改号软件以及含有改号功能与内容；    - 批量账号、网络地址自动切换系统，批量接收提供短信验证、语音验证的平台，如应用为接码软件以及含有批量接码功能与内容；    - 电话、短信轰炸类软件；应用提供“防封号”、“防封卡”等方式侵扰他人的私人生活安宁；    - 其他用于实施电信网络诈骗等违法犯罪的设备、软件。 |
| 教育 | 1. 0～18岁以下非成人素质类教育，包括但不限于：体育（或体育与健康）、艺术（或音乐、美术）、综合实践活动（含信息技术教育、劳动与技术教育）、兴趣培训等。 2. 0～18岁以下非成人学科类教育：包括但不限于：道德与法治、语文、历史、地理、数学（包括思维培训）、外语（英语、日语、俄语）、物理、化学、生物、学科类家教辅导。 3. 面向中小学、幼儿园的校外培训招商加盟服务。 4. 涉及已取消或国家不认可的资格证考证培训的内容。已取消的资格证考证培训包括心理咨询师（ACI心理咨询师）、会计从业资格证、中国保健行业营养保健师资格、外贸会计专业证书、外贸英语专业证书、全国国际商务英语、金融专业英语证书、金融理财师（AFP）、国际金融理财师（CFP）、公共营养师、注册人力资源管理师、人力资源测评师等。 5. 地下违规教育类赛事（黑竞赛），如：大师赛（DSS）、希望杯（XWB）、华杯赛（HBS）、“数学花园探秘”、“数学大联盟线上考试"等。 6. 提供违规办理入学、提供泄题、买卖考题服务。 7. 顶替参加考试、代写论文、期刊论文代发等。 8. 灵修类非法培训课程。 9. 高校以“研究生”“硕士、博士学位”等名义举办课程进修班，面向社会举办的非学历教育冠以“领导干部”“总裁”“精英”“领袖”等名义 |
| 护肤彩妆 | 1. 含费洛蒙等成分的香水、香氛产品，含有人寡肽-1成分的产品，含PRP、PRP血清蛋白等成分的产品，含有比马前列素成分的产品，含有PRP、PRP血清蛋白等成分的产品等。 2. 干细胞技术相关产品。 3. 药妆相关产品。（在化妆品推广中“药妆”属于有违客观事实的虚假内容，是被明确管控的词汇，化妆品素材不得涉及此类表述。） |
| 生活服务 | 1. 虚假纪念币、仿制名人字画等非法文物买卖相关服务。 2. 合法的邮票（包括普通邮票和纪特邮票）交换和买卖可以进行。但不得从事《集邮市场管理办法》规定的下列活动：    - 经营伪造、变造的邮资凭证；    - 经营国家禁止流通的集邮票品；    - 经营1949年10月1日以后台湾地区发行的集邮票品。 3. 涉及盗墓及盗墓工具的产品或服务，如洛阳铲等。 4. 黄牛票务服务，包括但不限于：转让/倒卖演出票务、出行票务、政府消费券、体育比赛入场券或者其他有价票证、凭证。 |
| 房地产 | 1. 房产交易类型：非正常销售或不具备上市销售资格的房产、特殊类型房产及土地，如小产权、顶账房、土地交易、房产以旧换新、抵押房、公租房等。 2. 房产周边业务：超出房产正常居住和使用功能，强调有融资、投资、医疗服务等，包括房产投资或投资讲座、融资或者变相融资的内容、医疗服务、以房养老业务、房产投资移民等。 3. 返本销售、变相返本销售等违规房产销售行为，采取售后包租或者变相售后包租方式销售未竣工商品房的行为。 |
| 法律服务 | 非法法律服务，包括但不限于：信用卡逾期停息挂账服务、保险退保、网络欺诈维权、买房退定金等服务。 |
| 食品 | 1. 槟榔、1-2段奶粉。 2. 可食用玫瑰盐 / 喜马拉雅盐。 3. 添加金银箔粉的食品等。 4. 产品名称涉及医疗功效的产品，如明目酵素、散结茶、通便茶、排毒养颜茶等。 5. 织纹螺（俗称海丝螺、海狮螺、麦螺、白螺、割香螺、小黄螺、甲锥螺）。 |
| 烟/酒 | 1. 香烟及周边制品，如烟盒、烟标、电子香烟、戒烟产品、烟草替代品及辅助工具、烟草企业等。 2. 槟榔、槟葛及其制品。 |
| 汽车 | 1. 违章铲分，代办购车指标服务或信息。 2. 提供转让车辆号牌、摇号指标的租赁、销售、转让等交易服务或信息。 3. 提供无车报废、无车过户、无车年检、驾照收分等车辆等交易服务或信息。 4. 提供“粤港澳两地车牌”、北京等各地车牌代拍/代办/出售/租赁、过户挂靠相关等交易服务或信息。 5. 套牌车、走私车、罚没车、非法车、黑车、抵押车、“拼装机动车”、“组装机动车”等交易服务或信息。 6. 妨害交通安全秩序的汽车产品，包括但不限于：汽车反雷达测速仪、电子狗、汽车隐性喷雾、卷帘式车牌架、雷达干扰器、牌照遮盖设备、交通信号灯变换器。 7. 汽车非法改装相关内容，包括但不限于：改换减震器、轮载、刹车钳，改装尾翼，改变发动机动力参数，加宽轮胎，对进气系统、排气系统、改装涡轮增压，改装悬挂，私自加装座椅数，改装大灯、自行车的减限速产品。 |
| 农林牧渔 | 1. 长江禁捕：任何有关长江流域水产产品及长江流域专有水产，包括但不限于长江野生鱼、长江鲟、中华鲟、长江假饵、长江渔网等，高邮湖青虾、岷江鱼、四川哲罗鱼等。 2. 东北黑土/天然黑土。 |
| 商务服务 | 1. 利用GEO（生成式引擎优化）技术进行AI"投毒"或操控AI输出的行为（AI+SEO）。 2. 出现“洗产地”介绍及服务等内容，包括但不限于通过办理产地证灰清、转口贸易等方式从事违法活动。 3. 代开、出售、转让真假发票等，包括各类发票、财务票据。（不具有效性、流通性、为收藏目的转让的情况除外） 4. 涉及无实质商业贸易，以虚增营收、套取融资或骗取政策优惠为目的的贸易增量业务。 5. 提供黄牛代拍行为、代拍业务。 |

| 安全安保 | 1. 警用、军用设备，包括但不限于：警服、警徽、手铐、警灯、警笛、电击器等警用和军用器械。 2. 军用类软件。 |

| 涉毒 | 1. 包括各类毒品、易制毒化学品、毒品原料、制毒的书籍、提供涉毒类服务等。 2. 新型毒品及添加精神类药物、食品及饮料，如咔哇潮饮、咔哇氿、啪啪潮饮等。 |
