# 功能全景图

开户、实名认证、行业资质请到华为投放平台操作。

推广和报表接口通过Marketing API对外开放，在使用过程中，依赖的一些接口尚未对外开放，需要到华为投放平台上创建后才可以在Marketing API使用。
