# 简介

在广告推广过程中，通过第三方监测，广告主可获得由客观公正的第三方监测公司提供认证的广告数据，监测目标广告的曝光、点击等关键指标。目前，鲸鸿动能已与秒针、国双等多家第三方监测公司实现技术对接。

## 实现方式

在广告创建过程中，平台可通过广告主填写的监测地址将创意的曝光/点击数据发送到第三方平台服务器。

通过约定监测地址 URL 中指定格式和名称的字符串，确定每次曝光/点击发生时需要上报给第三方平台服务器的信息。

当前支持的监测字段和地址格式要求如下：

- <strong>支持字段</strong>

  | <strong>宏名称</strong> | <strong>字段描述</strong> |
  | --- | --- |
  | \_\_OS\_\_ | 操作系统名称，仅支持安卓 |
  | \_\_IP\_\_ | IP地址 |
  | \_\_APP\_\_ | 媒体名称 utf-8编码 |
  | \_\_REQUESTID\_\_ | 广告请求ID 替换为 \_\_REQUESTID\_\_ 或 \_\_HWPPSREQUESTID\_\_ |
  | \_\_OAID\_\_ | OAID，原值，如果用户关闭了个性化广告，OAID会被设置为全0，例如: 00000000-0000-0000-0000-000000000000，附加&optout=1 |
  | \_\_TERM\_\_ | 机型 |
  | \_\_CREATIVE\_ID\_\_ | 素材ID |
  | \_\_UA\_\_ | 设备的UA |
- <strong>监测地址格式</strong>

  监测地址链接是由第三方监测平台提供的、用于鲸鸿动能将曝光/点击数据传输给第三方监测的链接。

  监测地址格式由“前缀+服务器域名+参数”三部分构成，其中：

  ① <strong>前缀</strong>只能是 https 开头。

  ② <strong>服务器域名</strong>必须是能够正常访问的正式域名，不接受 IP 或下载链接。

  ③ <strong>参数</strong>必须是全大写字母，两边为双下划线（参数左右两边均为两个连续的英文字符"\_"），可根据实际选择需要的参数，并使用与号 (&) 分隔参数。

  示例：

  https://www.huawei.com/track\_click.ashx?k=2129556&p=7QJNf&ns=\_\_IP\_\_&tr=\_\_REQUESTID\_\_&m0=\_\_OS\_\_&m1=\_\_OAID\_\_
- <strong>多组监测链接</strong>

  <strong>鲸鸿动能支持在创意层级进行多组监测链接，用以将曝光/点击数据同时发送给不同的三方监测公司。您可以选择以下两种填写形式：</strong>

  ①秒针监测特有，不同的三方监测链接通过参数的形式进行串联，合并为一个链接，并发送曝光/点击数据。注意监测链接需以[https://e.cn.miaozhen.com](https://e.cn.miaozhen.com/)开头。

  示例：https://e.cn.miaozhen.com/r/k=000&dx=\_\_IPDX\_\_&rt=2&tr=\_\_REQUESTID\_\_&gc=\_\_GCID\_\_& o=https%3A%2F%2Ftrace.rtbasia.com%3D\_\_MAC1\_\_%26idfa%3D\_\_IDFA\_\_

  ②通用形式，不同的三方监测链接并联，以英文分号作为分割。鲸鸿动能将会同时对所有并联的三方监测链接进行<strong>曝光/点击数据的发送</strong>。

  示例：

  https://b.lnk0.com/osversion=\_\_OS\_\_&requestid=\_\_REQUESTID\_\_&oaid=\_\_OAID\_\_;https://g.cn.miaozhen.com/ osversion=\_\_OS\_\_&requestid=\_\_REQUESTID\_\_&oaid=\_\_OAID\_\_;https://trace.rtbasia.com/ osversion=\_\_OS\_\_&requestid=\_\_REQUESTID\_\_&oaid=\_\_OAID\_\_
