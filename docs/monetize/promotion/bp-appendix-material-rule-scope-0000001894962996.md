# 审核范围

1. 广告授权与证明资质：包括素材涉及的商标和著作权资质、人物肖像权和附加资质；
2. 广告内容：包括素材图片、视频、文案、落地页内容；
3. 广告链接：包括跳转地址及落地页的真实性和规范性；
