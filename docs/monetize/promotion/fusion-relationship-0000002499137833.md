# 客户投放伙伴体系升级对应关系

投放端整合升级后，原应用市场应用推广的客户投放伙伴体系层级不变，原客户投放伙伴的相关功能由鲸鸿动能服务商管理平台承载。

| Before | After |
| --- | --- |
| 客户投放伙伴主账户 | 一级服务商 |
| 客户投放伙伴子账户 | 子客服务商 |
| 投放操作账户 | 子客账户 |
