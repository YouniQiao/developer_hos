# 下载中心

您可通过下载中心对导出数据所生成的下载任务进行集中管理，通过筛选来源、文件名和导出状态等条件查看首页清单、转账记录和消耗统计等数据。

注：列表展示历史生成的导出文件数据记录且从创建时间起默认保留7天历史数据。
