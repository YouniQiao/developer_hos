# Material

| 参数 | 必选(M)/可选(O) | 类型 | 描述 |
| --- | --- | --- | --- |
| id | M | Long | 素材ID。 |
| fileName | M | String | 素材文件名称。 |
| fileUrl | M | String | 素材上传的地址。 |
| videoCode | O | String | 华为视频code编码。 |
| hwVideoUrl | O | String | 华为视频URL。 |
| fileType | M | String | 素材类型。  取值范围：   - PIC：图片素材 - VIDEO：视频素材 |
| height | O | Integer | 高度。 |
| width | O | Integer | 宽度。 |
| size | M | Integer | 文件大小。 |
| createTime | M | String | 素材创建时间。 |
| updateTime | M | String | 素材更新时间。 |
