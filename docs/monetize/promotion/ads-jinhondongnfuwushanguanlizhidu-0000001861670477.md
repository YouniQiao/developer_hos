# 鲸鸿动能服务商管理制度

<strong>生效日期：2026年4月1日</strong>

<strong>一、目的</strong>

为了更好的拓展鲸鸿动能业务，维护良好生态环境和市场秩序，保障华为及合作伙伴的长期利益，使双方能够长期、稳定的合作，特制定《鲸鸿动能服务商管理制度》（以下简称“本制度”）。

<strong>二、适用范围</strong>

本制度适用于通过认证并与华为正式确立合作关系的鲸鸿动能服务商。服务商的法定代表人、企业负责人、管理人员、业务人员、运维人员等员工或合作人员开展业务相关行为或活动视为服务商的行为。以上人员应参照本制度开展业务相关行为或活动，并承担相应责任。

<strong>三、名词解释</strong>

1. 服务商（或“一级服务商”）：指与华为签约并与华为正式确立合作关系，在合作期限及合作范围内，对客户在鲸鸿动能平台上的投放业务享有代理权的组织或实体。
2. 子客服务商：指在鲸鸿动能平台上账户直接关联在一级服务商账户下且具备代理服务权限的服务商。
3. 客户(或“子客”）：指委托服务商在鲸鸿动能平台上进行投放业务的个人、组织或实体。华为以服务商提交的客户的有权机关签发的证照等作为核对客户真实身份、地址和有效联系方式等信息的依据。
4. 白名单：指鲸鸿动能根据各业务发展需要，进行特殊授权管理的客户清单。

<strong>四、总则</strong>

1. 服务商应严格履行与华为签署的《鲸鸿动能N1/N2/N3领域服务商合作协议》等相关协议（以下简称“合作协议”），并按照合作协议中所约定的各项权利义务，履行对客户的服务义务。华为仅与一级服务商签署合作协议，与其名下的子客服务商及子客无协议责任和义务。
2. 本制度作为合作协议的补充协议，具有同等法律效力；本制度如与合作协议中相关管理办法或规定有冲突，在鲸鸿动能服务商管理事宜范围内以本制度为准；华为有权对本制度不定期进行修改，并于生效前30日通过鲸鸿动能平台发布，请及时查阅。如服务商拒绝接受变更，请务必在这30天内放弃使用鲸鸿动能服务；若服务商在前述30天期满后继续使用鲸鸿动能服务，则代表知晓、同意并将遵守变更后的制度。
3. 服务商应向客户如实说明其与华为的合作业务范围、合作权限和合作期限；服务商名下子客服务商及子客通过其在鲸鸿动能平台开展业务相关行为或活动视为服务商的行为，服务商应对此类行为或活动进行监督并承担相应责任。
4. 服务商应维护华为品牌形象和声誉，不得做出损害华为业务利益的行为。

<strong>五、文件内容</strong>

1.业务授权管理

（1）授权申请

服务商签署合作协议后，如需华为出具合作证明函，应向公邮mspadmin@huawei.com发送邮件申请，华为根据合作协议约定的合作范围等内容出具，证明函模板由华为统一提供。服务商因客户或项目等特殊要求，需提供专用模板/特殊内容证明函的，华为或根据实际情况必要性向服务商出具，决策主要参考点为

1）华为已与客户沟通，客户明确表明因入库等原因需要华为提供前述证明函或授权函；

2）前述证明函或授权函需由一级服务商向华为书面申请。

在合作过程中，服务商不得擅自转让、修改、终止前述证明函；如合作协议中的合作范围等内容出现变更，原证明函或授权函自动作废，服务商不得继续使用，华为或根据更新后的合作协议约定的相关内容出具相应证明函。

2.商务体系管理

（1）商务秩序管理

1）华为向服务商提供对应的产品报价表。在业务合作过程中，华为有权根据业务发展情况，对产品报价表进行调整。服务商不得破坏鲸鸿动能的商业合作生态秩序或损害鲸鸿动能整体市场策略、鲸鸿动能品牌其他服务商的正当利益。

2）服务商需严格按照合作协议中的相关约定进行对外商务合作，有涉及特殊商务报价需求的需按照本制度“特殊商务管理”执行。

（2）特殊商务管理

1）针对重点客户或重大项目，服务商可向华为申请特殊商务，特殊商务需至少提前五个工作日进行书面申请。

2）特殊商务申请需按要求提供申请材料，包括但不限于项目背景、客户介绍、业绩预估、报价模式、渠道通路、利润预估等。

3）华为会对服务商的特殊商务申请进行审核，审核通过后服务商方可对申请中的对应客户及项目进行特殊报价，不得用于其他客户或项目。

4）针对服务商提交的客户特殊商务申请项目，服务商需在申请时提供上游环节商务诉求书面证明或上游下单记录等证明材料，一经发现证据缺失、作假、赚取特殊商务差价或其它违规行为，华为将视违规情节轻重有权对服务商采取处罚措施。

3.子客服务商与客户管理

（1）一级服务商对子客服务商管理

1）鲸鸿动能一级服务商可在业务开展过程中，邀请子客服务商关联到其名下协助其进行业务开展。

2）一级服务商需根据合作协议及本制度规范管理其名下的子客服务商，子客服务商的一切相关业务行为或活动即视为一级服务商行为或活动，由一级服务商承担相应责任。

（2）客户拓展管理

1）服务商需严格按照各合作协议中约定的合作领域/行业的客户划分进行拓展，不得向非授权合作领域/行业内的客户进行报价或服务，如有违规行为，华为有权对服务商进行相应处罚。

2）华为对服务商可拓展客户的范围管理方式包括但不限于：制定客户NA名单、白名单和黑名单（名单内客户不可拓展）。各类型名单由华为定期或根据业务需要以邮件或官网公示等书面方式通知给服务商。

3）华为有权根据业务发展情况，不定期调整客户拓展管理制度，或将以邮件或系统公告等书面方式通知服务商且自通知之日起生效，华为有权根据管理制度调整服务商的客户拓展范围。

4）白名单管理机制：华为根据各业务发展需要，对部分客户进行白名单管理，包含行业白名单客户、区域白名单客户、定向授权白名单客户等，各类型白名单客户均以华为确定结果为准；服务商可根据相应业务管理规则进行有条件申请白名单客户合作。

5）黑名单管理机制：华为根据各业务合规安全需要，对部分客户进行黑名单管理，包含：工商处罚黑名单、采购黑名单、历史合作违规清退黑名单等，各类型黑名单客户均以华为确定结果为准。服务商需严格杜绝黑名单客户的拓展，并且不得通过关联主体、代理公司、授权等方式绕过华为黑名单管理，协助黑名单客户实际业务在华为进行开展。

4.合作审视

（1）华为有权例行对服务商合作情况进行审视。审视内容包括但不限于合作流水、客户拓展、投入人力、资金使用情况等。

（2）华为有权以自然月度或（及）自然年度为周期对服务商的合作流水、鲸鸿动能认证人数等情况进行审视，并根据审视结果进行合作级别匹配或调整，服务商需配合提供审视过程中所需的相关材料。具体合作级别对应的合作流水、鲸鸿动能认证人数等要求，以及匹配的合作级别均以鲸鸿动能官网公示为准。

（3）华为有权暂停或终止与欠款、失信、重大违法违规行为以及涉及重大负向舆论事件服务商的一切合作，并有权视情节严重程度扣罚服务商的返点、保证金等，直至取消合作资格。

a）华为在进行调查过程中有权对举报人、投诉人、被举报服务商的相关信息进行取证留存。

3）违规行为披露

华为若接到投诉、举报，或司法、执法机关或其他主管部门调查，有权将服务商的相关信息(包括但不限于服务商自身及其名下子客服务商和子客的注册信息、账户信息、投放情况、消耗数据等)向相关部门或第三方披露。

违规服务商接到违规处理通知后，可在3个工作日内向平台提交包含相关申诉理由和证据的书面申诉请求。<strong>违规服务商接到违规处理通知后，未在3个工作日内提出申诉请求的，视为接受处理结果，并永久放弃任何申诉权利。</strong>

3）违规加重原则。服务商有下列情形之一的，将加重一级处罚：服务商为谋取己方利益恶意违规；为绕开华为认证或审核要求，服务商故意伪造相关材料或通过技术手段绕过平台管控；违规事件造成重大影响（如客户用户投诉或舆情、监管机构处罚）；违规事项发生后服务商不配合调查和问题整改。

4）违规减轻原则。服务商有下列情形的，可以减轻处罚：服务商在该违规场景下的单类违规行为中初次违规，且属非主观恶意违规行为，违规事项发生后主动配合调查，积极推动问题整改。

5）违规赔偿原则：因服务商违规给华为、消费者、流量供应方或其他任何主体造成经济损失的（包括但不限于因处理消费者投诉、消费者索赔、行政处罚等支付的费用等），服务商应承担上述主体所遭受损失的全部赔偿责任；若实际责任方为其名下子客服务商或子客时，华为可为服务商追责提供协助支持；对服务商拒不赔偿的，华为有权扣罚其业绩返点或保证金，并保留进一步追索的权利。

6）合规扣分影响服务商规范运作返点激励。

7）对于服务商的所有违规行为，华为保留追究法律责任的权利。

<strong>八、附则</strong>

<strong>1.</strong> <strong>“关联主体”的认定范围</strong>

（1）同一股东：A公司股东与B公司股东完全相同或拥有1名（包含）以上相同股东，且相同股东累计股权占比不低于50%，则A公司和B公司可视为关联主体。

（2）100%持股：A公司对B公司100%持股，则A公司和B公司可视为关联主体。

（3）同一集团：A公司和B公司同属于一个集团公司（在已上市或在上市进程中的集团公司或主体公司的财报/公开招股书中包含A、B公司），则A公司和B公司可视为关联主体。

（4）层级从属关系：A公司对B公司100%持股，同时B公司从属于C集团（从属定义见“同一集团”），则ABC三个公司可视为关联主体。

（5）以上场景信息均需可在第三方企业信用信息网站可查；若以上情形均不满足，华为通过调研或第三方渠道获取到的相关信息，也可作为“关联主体”的判定依据。
