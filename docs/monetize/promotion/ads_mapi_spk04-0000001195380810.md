# 查询商品库列表

【简介】查询经理账户下属广告主的商品库列表

<strong>请求地址</strong>

https://ads.cloud.huawei.com/openapi/v2/tools/dpa/product/share\_store\_list/query

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| manager\_id | string | 是 | 经理账户 |
| store\_name | string | 否 | 商品库名称，支持模糊查询 |
| store\_type | string | 否 | 商品库类型，支持过滤：  详见[【商品库类型】](/docs/monetize/promotion/ads_api73-0000001058884698#section16203188825) |
| page\_num | int | 否 | 搜索页码 默认值1  取值范围1~1000 |
| page\_size | int | 否 | 一页展示数量，默认值20  取值范围 10 ~ 50 |

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1 | 商品库列表 |

data（Struct1）定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| stores | Struct2[] | 商品库列表 |
| total | integer | 总数 |

stores（Struct2）定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| advertiser\_id | string | 商品库所属的广告主账户ID |
| own | string | 广告主拥有此商品库关系  详见[【广告主与商品库关系】](/docs/monetize/promotion/ads_api73-0000001058884698#section1098142411217) |
| store\_name | string | 商品库名称 |
| store\_id | string | 商品库ID |
| store\_type | string | 商品库类型，  详见[【商品库类型】](/docs/monetize/promotion/ads_api73-0000001058884698#section16203188825) |
| upload\_method | string | 导入方式：  详见[【导入方式】](/docs/monetize/promotion/ads_api73-0000001058884698#section1475353815214) |
| create\_time | string | 商品库创建时间 |
| last\_update\_time | string | 商品库最后更新时间 |
| last\_update\_num | int | 最后一次更新商品库中商品条数 |
