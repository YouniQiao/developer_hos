# 查询资金账户日结算明细（新）

【简介】查询广告主或服务商的日结算明细，包括现金、赠送、返利账户分别支出的信息

请求地址

https://ads.cloud.huawei.com/openapi/v1/finance/consume/query

请求方法

GET

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，使用经理账户登录或授权账号关联了多个子客账户时，此字段必填：  1、若一级服务商登录：  i.传一级服务商直属子客，则查询一级服务商直属子客的消耗  ii.传子客服务商直属子客，则查询子客服务商直属子客的消耗  2、若子客服务商登录：  i.传子客服务商直属子客，则查询子客服务商直属子客的消耗 |
| agency\_id | long | 否 | 服务商ID，服务商登录时，可选填写该字段  1、若一级服务商登录：  i.传自身或不传，则查询一级服务商的所有消耗（同时不传advertiser\_id）  ii.传子客服务商，则查询子客服务商的所有消耗（同时不传advertiser\_id）  2、若子客服务商登录：  i.传自身或不传，则查询子客服务商的所有消耗（同时不传advertiser\_id） |
| start\_date | string | 是 | 起始时间 yyyy-MM-dd |
| end\_date | string | 是 | 结束时间 yyyy-MM-dd  时间跨度最大一年 |
| account\_type | string | 是 | 账户类型标识  详见[【资金账户类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#资金账户类型) |
| account\_media\_type | string | 否 | 适用媒体范围，不填默认查全部适用媒体范围  详见[【虚拟账户类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#虚拟账户类型) |
| sign\_party | string | 否 | 签约主体（不传查全部），详见[签约主体](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#签约主体)。 |
| user\_service\_type | string | 否 | 账户推广业务类型，不填默认查询ADS侧数据，详见[【账户推广业务类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#账户推广业务类型) |
