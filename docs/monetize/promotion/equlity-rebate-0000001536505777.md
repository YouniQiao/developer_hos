# 月度返利规则

<strong>该规则自发布之日起生效。</strong>

华为推送为答谢Push用户增长服务合作伙伴一直以来的支持，将实行如下月度返利规则。

## 月度返利规则

根据单款应用在华为Push用户增长服务自然月的现金消耗，按照如下比例赠送代金券，并在次月返还至该应用对应的现金消耗的账户内。若该应用在自然月内更换现金消耗账户，代金券将按各账户的现金消耗比例分配。

| 单款应用自然月现金消耗（元） | 次月代金券赠送额度 |
| --- | --- |
| &gt;=100万（包含100万） | 现金消耗的10% |
| 60-100万（包含60万） | 现金消耗的6% |
| 30-60万（包含30万） | 现金消耗的3% |

代金券到账时间：当月现金消耗满足上述条件，次月15日前（如遇节假日将顺延）将赠送代金券充值到相应账户。合作伙伴可通过AppGallery Connect账户中心查看应用的代金券赠送信息。

![](./img/note_3.0-zh-cn_965730b26915.png) 

现金：指开发者通过线上、线下充值的推广基金（代金券不属于现金）。

赠送代金券：指合作伙伴满足评定条件后，所获得的由华为推送赠送的可用于Push增长服务的代金券。

合作伙伴和代理服务商投放操作账户可以在AppGallery Connect首页“账户中心 &gt; 优惠券 &gt; 代金券”查看账户内的Push用户增长服务赠送代金券等。

## 常见问题

- <strong>我的应用现金消耗未达到30万元有赠送代金券吗？</strong>

  没有，该返利规则仅限当月现金消耗达到30万元以上（包含30万元）的应用参加。
- <strong>我的账户中有多款应用，本月多款应用累计消耗达到30万是否有月度赠送代金券？</strong>

  没有，以上返利规则只针对单款应用本月现金消耗达标后进行赠送代金券。
- <strong>为什么消耗达到了30万元却没有赠送代金券？</strong>

  赠送条件基于单款应用自然月的现金消耗计算，若开发者本月总消耗金额含代金券金额，需扣除代金券后的实际现金消耗达到30万元及以上，方可获得赠送；若扣除代金券金额后现金消耗不足30万元，则没有赠送代金券。
- <strong>该月度返利规则赠送的代金券是否只能给获得赠送的单款应用使用？</strong>

  是，2026年1月起，符合返利规则所赠送的代金券会充值到开发者账户的对应应用中，仅该应用可以使用这笔金额，其他应用无法使用。
- <strong>赠送的代金券能否自行选择在哪些任务中使用？</strong>

  可以，应用投放时默认优先使用账户中的 APP 代金券，但开发者可在创建任务时选择不使用APP代金券。
- <strong>赠送代金券后是否会通知？</strong>

  会，商务会告知当月满足现金消耗的应用开发者，接到通知的开发者可通过AppGallery Connect平台（AppGallery Connect首页-个人账户）中查看赠送的代金券。

  ![](./img/caution_3.0-zh-cn_756916c6d7b3.png) 

  赠送代金券的有效期为1年，请尽快使用，以免过期。
- <strong>赠送代金券可以申请退款吗？</strong>

  赠送代金券不支持退款和延期，请尽快合理安排使用。
- <strong>本次付费推广返利活动持续时间有多久？</strong>

  本次付费推广返利活动有效时间以华为Push对外发布为准，如有变更我们将另行通知。
