# 激励规则

华为应用市场为答谢应用推广合作伙伴一直以来的支持，将实行如下赠送激励规则。

## 月度激励

根据单款应用在华为应用市场应用推广平台自然月的现金消耗，按照如下比例赠送激励金，并在次月返还至该应用现金消耗相应账户内。若该应用在自然月内更换现金消耗账户，激励金将按各账户的现金消耗比例分配。

| 单款应用自然月现金消耗（元） | 次月赠送比例 |
| --- | --- |
| &gt;=300万 | 4% |
| 30万~300万 | 2% |

赠送推广基金到账时间：当月现金消耗满足上述条件，次月15日前（如遇节假日将顺延）将赠送激励金充值到相应账户。

![](./img/cda3081fa046.png) 

现金：指开发者通过线上、线下充值的推广基金（资源置换基金和赠送激励金不属于现金）。

赠送激励金：指合作伙伴满足评定条件后，所获得的由应用市场赠送的推广基金。

置换资源基金：指应用通过资源置换获得的推广基金。

直客和客户投放伙伴投放操作账户可以在推广后台-工具-账户管理-财务管理进入查看账户内各应用的现金消耗、赠送金消耗、置换金消耗和耀星券消耗。

以上激励规则适用于应用、快应用。

单款应用自然月现金消耗仅包含该应用在HarmonyOS 4.X 及以下版本应用市场整体现金消耗。

签约年框合作客户旗下应用，月度激励规则以年框合同约定为准。

## 常见问题

### 我的应用现金消耗未达到30万元有赠送激励金吗？

激励规则仅限当月现金消耗达到30万元以上（包含30万元）的应用参加。

### 单款应用自然月现金消耗是否包含该应用在鸿蒙应用市场的消耗？

否，自然月现金消耗仅包含该应用在HarmonyOS 4.X 及以下版本应用市场整体现金消耗。

### 我的账号中有多款应用，本月多款应用总消耗达到30万是否有月度赠送激励金？

没有，本次活动只针对单款应用现金消耗进行赠送激励金。

### 为什么消耗了30万元却没有赠送推广基金？

开发者账户余额最多可包括充值现金、赠送激励金、置换资源基金3种。账户消耗金额的先后顺序为赠送激励金、置换资源基金、充值现金，即当赠送激励金消耗完时，才能启动置换资源基金的消耗，故消耗30万元以上却未收到赠送金额的应用通常为本月总消耗金额含赠送激励金或置换资源基金，扣除后本月现金消耗不足30万元。

### 赠送激励金是否只能给获得赠送的单款应用使用？

不是，赠送激励金会充值到获得赠送的应用所在的开发者账户中，账户中所有应用都可以使用这笔金额。当账户中含赠送激励金时，账户下所有应用的推广费用都优先消耗赠送激励金。

### 赠送激励金后是否会通知？

会，商务会告知当月满足现金消耗的应用的开发者，接到通知的开发者可在后台（开发者联盟-管理中心-我的账户-余额-充值记录）中查看赠送的订单。注意，赠送激励金的有效期为1年，请尽快使用，以免过期。

### 赠送激励金可以申请退款吗？

赠送激励金不支持退款申请，请尽快使用，以免过期。

### 本次应用推广激励活动持续时间有多久？

本次应用推广激励活动有效时间以华为应用市场对外发布为准，如有变更我们将另行通知。
