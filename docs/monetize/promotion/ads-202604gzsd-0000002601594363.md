# 2026年4月

<strong>【规则</strong> <strong>变更</strong> <strong>】</strong> <strong>金融</strong> <strong>-</strong> <strong>文案描述</strong>

1. 金融行业支持使用“神器、黑科技、利器”描述

|  |  |
| --- | --- |
| ✅支持使用“黑科技”描述 | ✅支持使用“利器”描述 |
|  |  |

<strong>【规则</strong> <strong>新增</strong> <strong>】</strong> <strong>工具</strong> <strong>-</strong> <strong>虚假宣传</strong> <strong>场景</strong>

2. 虚假宣传：

a. WiFi类广告，不支持将5GHz WiFi简写为5G WiFi、5G网络等

|  |  |
| --- | --- |
| ❌不支持宣传“5G WiFi”描述 | ✅支持宣传“5GHz WiFi”描述 |
|  |  |

b.非运营商/虚拟运营商类工具软件，不具备网络资源与通信权限的，不支持以下描述

流量分发类：分发/领取/激活/赠送流量、升级5G、开通流量套餐等描述

技术功能类：网络质量优化、增强信号、解除网速限制、修改网络配置等描述

（注：如明确为推广网络加速器服务，需提供《增值电信业务经营许可证》）

|  |  |
| --- | --- |
| ❌不支持宣传流量“激活”描述 | ✅支持宣传流量管理、流流量用量监测、起超量预警功能 |
|  |  |
