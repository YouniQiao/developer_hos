# 创建推广产品

【简介】通过此接口可以推广产品

<strong>请求地址</strong>

https://ads.cloud.huawei.com/ads/v1/promotion/product/create

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| product\_type | string | 是 | 推广产品 枚举限制 详见[【推广产品】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_meijuzhi-0000001510863205#section1526319223472) |
| product\_info | Struct1 | 是 | 推广产品信息 |

product\_info (Struct1)定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| app | Struct2 | 否 | 应用信息，product\_type为ANDROID\_APP，PROMOTION时填写 |
| quick\_app | Struct3 | 否 | 快应用信息，product\_type为QUICK\_APP时填写 |
| web\_page | Struct4 | 否 | 网页信息，product\_type为WEB时填写 |
| mini\_app | Struct5 | 否 | 微信小程序信息，product\_type为MINI\_APP时填写 |
| harmonyos\_native\_app | Struct6 | 否 | 鸿蒙应用，product\_type为HARMONYOS\_NATIVE\_APP时填写 |
| harmonyos\_meta\_service | Struct7 | 否 | 鸿蒙元服务，product\_type为HARMONYOS\_META\_SERVICE时填写 |

app(Struct2)定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| app\_id | string | 是 | 华为应用市场App ID |

quick\_app (Struct3)定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| deeplink\_url | string | 是 | 快应用直达地址  最长2048  格式如下：  以hap://app/包名/ 开头 |

web\_page (Struct4)定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| dest\_url | string | 是 | 落地页地址 (国内) 最长2048  以``https://开``头 |

mini\_app (Struct5)定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| mini\_app\_id | string | 是 | 微信小程序ID，最长2048 |
| mini\_app\_link | string | 是 | 微信小程序链接，最长2048 |
| mini\_app\_name | string | 是 | 微信小程序名称，最长2048 |

harmonyos\_native\_app (Struct6)定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| app\_id | string | 否 | 华为应用市场APP ID。app\_id 和package\_name必填其一 |
| package\_name | string | 否 | 应用包名。app\_id 和package\_name必填其一 |

harmonyos\_meta\_service (Struct7)定义

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| app\_id | string | 否 | 华为应用市场APP ID。app\_id 和package\_name必填其一 |
| package\_name | string | 否 | 应用包名。app\_id 和package\_name必填其一 |
