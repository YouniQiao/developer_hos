# 工具软件虚假宣传合规专题解读

本专题聚焦工具软件行业（含系统清理、WiFi连接、地图导航等），围绕广告法中虚假宣传认定与合规红线，明确虚假宣传监管定义、高频违规场景与修改指引，助力合规投放、规避法律风险。

## 一、工具软件行业虚假宣传管控的法律依据

**《广告法》第一章 第四条**

广告不得含有虚假或者引人误解的内容，不得欺骗、误导消费者。广告主应当对广告内容的真实性负责。

**《广告法》第二章 第二十八条**

广告以虚构或者引人误解的内容误导消费者的，构成虚假广告。

## 二、虚假广告的5种法定情形

**商品 / 服务不存在：**宣传的产品或服务实际并未推出或不存在

**关键信息与实际不符且影响购买：**性能、功能、产地、质量、成分、价格、销售状况、荣誉、承诺等与事实不符，对购买行为有实质性影响

**使用虚构 / 伪造 / 无法验证的证明材料：**科研成果、统计资料、调查结果、文摘、引用语等虚假或无法核验

**虚构夸大使用效果：**夸大或编造产品 / 服务的实际使用效果

**其他虚假 / 引人误解情形：**兜底条款，覆盖其他欺骗、误导消费者的行为

## 四、工具行业合规投放TIPS

1.**功能真实**：只宣传**实际具备、可验证**的功能，不夸大、不承诺超出技术能力的效果。

2.**效果可证**：所有效果、数据、排名**必须有证据**（后台数据、检测报告、专利、授权）。

3.**隐私透明**：**明确告知权限、用途、安全措施**，不隐瞒、不诱导授权。

4.**广告合规**：福利真实、标识清晰、关闭按钮明显、**不强制跳转、不仿系统弹窗**。

5.**资质真实**：不虚构专利、认证、案例、背书，**所有材料可核验**。

6.**禁用绝对化用语**：最、第一、顶级、100%、根治、永不、万能等。
