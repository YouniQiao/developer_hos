# 获取设备意向分

【简介】合作伙伴查询联合建模后的设备意向分。

<strong>请求地址</strong>

https://svc-drcn.ads.huawei.com/opendmp/v3/label/intent-scoure

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>长度</strong> | <strong>必填</strong> | <strong>说明</strong> |
| <strong>advertiserId</strong> | String | 100 | Y | 广告主Id |

<strong>请求示例</strong>

POST http://$\\{base\_address\\}/opendmp/v3/label/intent-scoure

HTTP/1.1

Headers:

Content-Type: application/json

Authorization：Bearer DAEAAIX7IS\*\*\*\*\*Pd81PDmWiM1KS2

Body:

```
{
" advertiserId ": "829411234539267456"，// corpId 企业ID
}
```

<strong>响应字段</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>参数名</strong> | <strong>类型</strong> | <strong>长度</strong> | <strong>是否必填</strong> | <strong>说明</strong> |
| <strong>code</strong> | int | - | Y | 错误码 |
| <strong>message</strong> | String | 2048 | N | 描述 |
| <strong>value</strong> | List&lt;IntentScoreFile &gt; | - | Y | 意向分文件下载地址 |

IntentScoreFile

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| <strong>参数名</strong> | <strong>类型</strong> | <strong>长度</strong> | <strong>是否必填</strong> | <strong>说明</strong> |
| <strong>intentScoreLabelName</strong> | String | - | Y | 意向分标签名 |
| <strong>intentScoreFileName</strong> | String | 2048 | Y | 意向分文件名 |
| <strong>method</strong> | String | - | Y | HTTP请求方法：GET |
| <strong>url</strong> | String | - | Y | 请求的URL |
