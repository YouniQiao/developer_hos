# 2026年鲸鸿动能竞价广告五一大促激励政策

<strong>政策周期：</strong>2026年4月15日~2026年5月5日

<strong>面向对象：</strong>FSD-N1（不含CPA/CPS），及FSD-N2模式下客户，按照“推广产品”维度，“推广产品”定义请参考鲸鸿动能广告介绍文档，包含：HarmonyOS NEXT应用、Android应用、快应用、网页（不含维纳斯落地页）。

<strong>参与条件</strong>：对比2025年同期，广告主于2026年4月15日~2026年5月5日在鲸鸿动能的投放有增量现金消耗。

<strong>报名方式：</strong>子客页面-政策活动，选择【2026年鲸鸿动能竞价广告五一大促激励】进行报名。

<strong>报名截止时间：</strong>2026年4月14日

<strong>互斥政策：</strong>与星火计划（周期内）、年框政策，游戏政策互斥。

<strong>激励内容：</strong>

|  |  |  |
| --- | --- | --- |
| <strong>档位</strong> | <strong>政策周期日均流水增量X（万元）</strong> | <strong>增量现金部分返点比例C</strong> |
| 第一档 | 5&lt;=X | 20% |
| 第二档 | 3&lt;=X＜5 | 15% |
| 第三档 | 1&lt;=X＜3 | 10% |

*注：仅限鲸鸿动能DSP竞价部分参与本激励，综合DSP和直客DSP不适用本激励。*

计算方法如下：

- 日均流水增量X=[(政策周期内总流水消耗)-(去年同期总流水消耗)] /21天
- 现金增量Z=(政策周期内总现金消耗)-(去年同期总现金消耗)
- 达标门槛：单产品日均流水增量X&gt;=1万元
- 返利金额：返利金Y=Z\*C（根据日均流水增量X对应上表确认）。举例：根据X所在档位，享受相应档位的返点。如Z=50万元，X=2万元，则华为将于发放周期内将50\*10%=5万元的激励金充值到广告主账户
- 以上日均基线、流水增量、现金增量、返利计算等均不包含鸿飞消耗部分

<strong>激励发放形式和使用</strong>：

|  |  |  |
| --- | --- | --- |
| <strong>竞价投放</strong> | <strong>发放周期</strong> | <strong>有效期</strong> |
| 返利金充值 | 按活动节点发放 | 180天 |

<strong>结算方式：</strong>

1. 账单和激励金额数据以鲸鸿动能数据为准。
2. 如发现不符合激励条件的情况，鲸鸿动能将对数据进行复核，并仅在符合激励条件的范围内发放激励，金额以复核调整后的为准。激励发放后，如果客户有异议，可以通过服务商发邮件向鲸鸿动能反馈问题，鲸鸿动能进行数据复核。
3. 如涉及数据差异，数据差异在鲸鸿动能数据的10%以内时，以鲸鸿动能数据为准；如差异超过10%，请于五个工作日内发正式邮件反馈数据差异排查，双方友好协商一致确认最终数据；如未及时反馈，视为确认以鲸鸿动能数据为准。

<strong>补充说明：</strong>

1. 返利金由鲸鸿动能平台在推广产品政策期结束且消耗达标后，直接发放至子客账户，涉及多个账户的，按各账户现金消耗比例进行分摊返利金，原则上不接受调整。
2. 适用政策期间，推广产品及相关推广投放应保持合法合规运营，不得出现违反法律法规或侵犯第三方合法权益的情形，否则鲸鸿动能有权取消推广产品的政策适用资格并采取相关处罚措施。
3. 广告主享受鲸鸿动能提供的返利金等权益，需以其账户处于有效且可操作状态为前提。因广告主或服务商相关账户注销等账户状态异常情形，致使鲸鸿动能客观上无法向其账户成功充值返利金或发放其他权益的，由此产生的返利金等无法发放的后果，由服务商或客户自行承担，并视为其默认放弃该等权益，鲸鸿动能将不再进行发放操作。
4. 异常作弊、违规操作、恶意修改预算/出价、利用系统漏洞、滥用系统机制、可疑或非法交易等不当行为不可享受激励。经鲸鸿动能合理判定服务商或客户存在前述行为的，鲸鸿动能有权拒绝发放激励并追回已发放的激励，并采取相应处罚措施，由此造成的损失或赔偿责任由服务商或客户承担。
5. 如因产品功能、计算错误、系统错误等任何原因导致激励超额发放、重复发放或错误发放，一经发现，鲸鸿动能有权追回多发部分或按核实后的应发激励调整发放。
6. 对于应追回的激励金额，鲸鸿动能可按以下方式扣除或追缴：（1）从下期激励中扣除；（2）从服务商或客户激励账户中直接扣除；（3）从服务商或客户的充值账户等其他支付、结算账户中扣除；（4）从鲸鸿动能尚未向服务商或客户支付的各种激励、费用、分成、代收款等款项中直接扣除；（5）从服务商或客户在华为平台的其他激励、账户或待结算款项中扣除；（6）采取其他可行的方式要求服务商或客户在30个工作日内退回、支付或赔偿。
7. 如对本政策有任何疑问，请联系鲸鸿动能。
