# 概述

**行业资质审核中涉及到的行业为审核行业：审核行业是依据国家法律法规要求，推广开户主体经营范围内的行业，****为企业信息行业资质填写页面中所必选的行业**。

广告主在不涉及[禁投内容](/docs/monetize/promotion/ads_shenhe02-0000001058065564)的前提下，可根据本章节中的“行业描述”和“行业举例”，判断自己的行业是否准入，并可以通过“行业资质要求”判断准入需要提供的资质。

![](./img/note_3.0-zh-cn_6e6e09278732.png) 

如广告主体涉及第三方企业合作和推广非开户企业所有的产品，需在行业资质中提交《品牌授权书》或《销售合作相关的合作协议、供货协议或授权书》等。

如涉及***以营利为目的提供网络信息服务***，需提供《增值电信业务经营许可证》（B25互联网信息服务业务）。

如涉及***应用推广***，需要提供《计算机软件著作权登记证书》或《电子版权认证证书》。

**企业授权证明资质**要求如下：

- 资质的登记主体：应与系统中录入的、和华为签订的相关协议及其它相关文件中的出现的广告主的名称一致。
- 资质的有效期限：应涵盖广告的投放期限及产品、服务的提供期限。
- 资质的登记事项：应与要证明的事项一致。
- 授权人名称：应与广告主提交的反映原始生产商、提供商等原始所有者相关资质或权利证明中的主体名称一致。
- 被授权人名称：应与广告主的名称一致。
- 授权标的：广告所涉及的商品或服务应与授权的商品或服务一致；广告所涉及的商品或服务的品牌的商标应与授权的商品或服务的品牌的商标、授权人有权的商标权一致。
- 授权期限：应涵盖广告的投放期限及产品、服务的提供期限。
- 授权地域：应涵盖广告所提及的产品、服务的销售区域。
- 授权事项：应包含广告主可以自行销售被授权人的产品、服务等事项。
