# 华为账号绑定多账户类型，选择单个账户类型登录

如您的华为账号已绑定应用市场应用推广&鲸鸿动能广告不同的账户类型，在登录后，选择您想操作的账户类型，点击“进入系统”按钮，登录即可。以下场景需要选择账户类型登录，比如：

1. 您的华为账号分别绑定了应用市场应用推广和鲸鸿动能广告的直客账户和子客账户
2. 您的华为账号分别绑定了应用市场应用推广和鲸鸿动能广告的服务商账户
