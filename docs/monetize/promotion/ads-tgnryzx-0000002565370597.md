# 推广内容一致性&清晰明确要求

<strong>本专题聚焦广告内容“一致性&清晰明确”合规要求，重点涵盖创意内容与开户、广告三要素内容、行业一致性管理要点。结合典型违规案例与禁投场景深度解析，帮助商家规范投放、降低违规风险，保障广告投放稳定顺畅。</strong>

## 一、 广告推广内容信息传达需清晰完整

（1）广告标题要求：不得单独使用地域名、品牌名、模仿自然人称谓等不完整描述。

【修改建议】标题需明确行业、商品、活动等推广信息

|  |  |  |
| --- | --- | --- |
| ❌ <strong>单独使用“地域名”</strong> | ❌<strong>单独使用“品牌名”</strong> | ❌<strong>单独使用“模仿自然人称谓”</strong> |
|  |  |  |

（2）品牌名称要求：不得使用无意义符号、表意不明文字内容。

【修改建议】可使用品牌名称（如： app名称、商标名称）或企业主体简称

|  |  |  |
| --- | --- | --- |
| ❌<strong>不明确品牌名称</strong> | ❌<strong>不明确品牌名称</strong> | ❌<strong>不明确品牌名称</strong> |
|  |  |  |

（3）推广内容要求：不得使用推广不明确或与实际推广内容无相关性的广告。

【修改建议】广告素材需体现行业或产品特性，确保明确表达出所推广的内容

|  |  |  |
| --- | --- | --- |
| ❌<strong>未表达推广内容</strong> | ❌<strong>未表达推广内容</strong> | ❌<strong>未表达推广内容</strong> |
|  |  |  |

|  |  |  |
| --- | --- | --- |
| ✅<strong>明确表达推广内容</strong> | ✅<strong>明确表达推广内容</strong> | ✅<strong>明确表达推广内容</strong> |
|  |  |  |

## 二、一致性推广要求

（1）创意内容与开户行业一致性要求：不得出现创意推广内容与开户主体、行业不一致的情况，如：“开户为A行业，素材内容必须是A行业；不得出现B行业。

【修改建议】创意推广产品需与开户报备信息保持一致，若欲投放另一款产品/服务/应用类型，需重新开户

|  |  |  |
| --- | --- | --- |
| ❌<strong>行业不一致</strong> | <strong>❌ 行业不一致</strong> | <strong>❌ 行业不一致</strong> |
|  |  |  |

|  |  |  |
| --- | --- | --- |
| ✅<strong>行业一致</strong> | ✅<strong>行业一致</strong> | ✅<strong>行业一致</strong> |
|  |  |  |

（2）广告三要素内容一致性要求：不得出现图文不符、外层素材与落地页不符、利益点不符等、引导文案与落地页推广目的不符问题。

【修改建议】广告文案、素材、落地页三者推广内容/活动/利益点/引导语等需保持一致。且外层推广内容需在落地页体现

①文案与素材一致性

|  |  |  |
| --- | --- | --- |
| ❌<strong>商品不一致</strong> | ❌ <strong>金额不一致</strong> | ❌ <strong>称谓不一致</strong> |
|  |  |  |

|  |  |  |
| --- | --- | --- |
| ✅<strong>商品一致</strong> | ✅<strong>金额一致</strong> | ✅<strong>称谓一致</strong> |
|  |  |  |

②外层素材内容与落地页一致性

|  |  |  |
| --- | --- | --- |
| ❌<strong>测评不一致</strong> | ❌ <strong>立减活动不一致</strong> | ❌ <strong>指南不一致</strong> |
|  |  |  |

|  |  |  |
| --- | --- | --- |
| ✅<strong>内容一致</strong> | ✅<strong>内容一致</strong> | ✅<strong>内容一致</strong> |
|  |  |  |

（3）利益点一致性（关于价格、满减、折扣等）

|  |  |  |
| --- | --- | --- |
| ❌ <strong>满减不一致</strong> | ❌ <strong>活动不一致</strong> | ❌<strong>价格不一致</strong> |
|  |  |  |

|  |  |  |
| --- | --- | --- |
| ✅<strong>满减</strong> <strong>一致</strong> | ✅<strong>补贴</strong> <strong>一致</strong> | ✅<strong>价格一致</strong> |
|  |  |  |

（4）视频一致性：广告视频的语音需与字幕保持一致；若使用非汉语语音，需添加中文字幕

（5）任务推广目标一致性要求：

①推广目标为应用下载，落地页不得为网页推广，且不得二跳

②推广目标非应用下载类，落地页不得进行应用下载推广

③当推广产品为”快应用“或”网页“，落地页推广功能需一致，不得通过网页拉端到快应用

④外层引导文案需与落地页内容及功能相关联，如落地页推广促活，外层引导文案不得描述为下载

【修改建议】推广目的需与实际功能保持一致，如：推广目的为应用促活，广告点击后需具备打开到软件端内的功能

|  |  |  |
| --- | --- | --- |
| ❌<strong>推广应用下载，落地页使用网页</strong> | ❌<strong>快应用与网页功能不一致</strong> | ❌<strong>落地页为快应用，不得使用下载</strong> |
|  |  |  |

## 三、行业一致性注意事项（高频驳回场景）

（1）电商行业

价格一致性<strong>：</strong>使用单品素材投放广告，如外层素材商品在落地页无直接体现，且落地页为产品聚合页，在无法验证外层产品价格与实际售卖价格是否一致的情况下，外层素材请勿出现特定价格描述，请使用：xx元起、低至XX元或不出现具体价格。

|  |  |  |
| --- | --- | --- |
| ❌<strong>特定价格</strong> | ❌<strong>特定价格</strong> | ✅<strong>低至XX元</strong> |
|  |  |  |

（2）阅读短剧：广告创意不得误导用户，禁止将小说内容伪装成新闻资讯、娱乐八卦等非小说类内容。

（3）金融行业-内容与条款一致：互联网保险营销宣传内容（如保障范围、理赔条件、保费金额）需与保险合同条款保持一致。

（4）房地产行业-楼盘名称标注：若预售证名称与广告推广楼盘名称不一致，需在广告素材中同时注明【注册名称】和【推广名称】。

|  |  |  |
| --- | --- | --- |
| ❌<strong>小说涉及新闻资讯内容</strong> | ❌ <strong>金融保费不一致</strong> | ❌<strong>名称不一致未同时注明</strong> |
|  |  |  |

（5）盲盒：推广盲盒活动，外层素材中展示的奖品需与奖品池中的商品保持一致，包含品牌、规格、型号等。

（6）旅游资讯-目的地/景点素材匹配：

①广告文案、素材中使用目的地/景点/城市（尤其地标性建筑）描述时，需与配图严格匹配（如描述“日本东京”，配图不得为其他城市地标）。

②旅游攻略标题规范：使用“国内超好吃的五条美食街”“超值得去的9座古镇”等旅游攻略式标题时，落地页需同步呈现对应的内容介绍；若无法匹配，需修改为与落地页强关联的文案及素材。

|  |  |  |
| --- | --- | --- |
| ❌ <strong>文案与图片目的地不一致</strong> | ❌ <strong>标题内容在落地页无对应</strong> | ✅<strong>推广一致</strong> |
|  |  |  |
