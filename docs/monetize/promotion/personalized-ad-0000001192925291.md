# 个性化广告

## 概述

鲸鸿动能广告平台投放的广告包括个性化广告和非个性化广告。

1. 个性化广告：也称为基于用户兴趣投放的广告或行为广告。是指基于一段时间内、跨不同应用或服务的用户行为而显示的广告，包括但不限于受众特征定位、兴趣类别定向、再营销、目标客户匹配列表定位的广告。
2. 非个性化广告：是指不基于用户过去行为投放的广告，非个性化广告可能会基于上下文或背景信息来进行定位，例如基于设备上下文、内容上下文或大致的地理位置信息定位的广告。非个性化广告会禁止任何的受众特征定向、兴趣类别定向、再营销和目标客户匹配定向。

个性化广告是鲸鸿动能广告业务中重要的一块。它可以提高用户收到的广告相关性，也可以提高广告主的投资回报率。个性化广告是根据分析用户的数据来定向用户，并向用户展示与他们更为相关的广告内容，因此可以为用户和广告主都提供更好的体验。

在根据用户行为或兴趣数据来展示与他们更相关的广告内容时，需要妥善处理用户的这些信息。如某些兴趣属于敏感信息，根据这些兴趣来定向用户可能会对用户体验产生不利影响。

考虑到这一点以及向用户进行广告定向的敏感性，我们针对所有的个性化广告定向功能都制定了相关的政策标准。这些标准不会取代我们的其他广告政策（例如[广告协议](https://developer.huawei.com/consumer/cn/doc/promotion/ad-agreements-0000001169499170)），因此广告主除了要遵守个性化广告政策外，仍须遵守所有适用的其他广告政策。此外，广告主还须遵守我们相关的[广告审核规范](https://developer.huawei.com/consumer/cn/doc/promotion/overview-guangaoshenheguifan-0000001188925990)。

## 个性化广告政策原则

- <strong>政策原则</strong>

  个性化广告政策的适用性因所使用的定向功能而异。所有定向功能都包含两个要素：

  - 定向用户，这是指您选择了向哪些用户群体展示广告，具体包括创建或选择这些群体时所涉及的各个方面。
  - 广告素材，指的是您广告上展示的内容。

  根据使用的定向功能是广告主挑选的受众群体还是鲸鸿动能广告挑选的受众群体，宣传产品和服务时可能需要遵守个性化广告关于敏感兴趣类别的规定：

  - 对于所有个性化广告定向功能 ，我们都不允许广告主根据敏感的兴趣类别去定位用户。
  - 对于广告主挑选的受众群体（广告主可以自行指定、挑选或上传受众群体），广告主不得宣传属于敏感兴趣类别的产品和服务。
  - 对于鲸鸿动能广告挑选的受众群体（即鲸鸿动能广告进行受众群体的构建或挑选），广告主不可以宣传属于敏感兴趣类别的产品和服务。

  在个性化广告中限制使用敏感兴趣类别,具体的敏感类别可以参考[个性化广告敏感兴趣类别的限制](#section022419182716)原则。

- <strong>政策影响</strong>

  若广告主宣传的产品和服务属于敏感兴趣类别，则不得使用个性化广告定向功能。请参阅下面列出的定向功能（并非详尽无遗），以及个性化广告政策的适用情况。

  | <strong>个性化广告定向类型</strong> | <strong>详细功能列表</strong> |
  | --- | --- |
  | 广告主挑选的受众群体，不得宣传属于敏感兴趣类别的产品和服务 | - 再营销。   - 文件上传人群。   - 应用内行为人群。   - 广告行为人群。 - 拓展人群。 |
  | 鲸鸿动能广告挑选的受众群体，不得宣传属于敏感兴趣类别的产品和服务 | - 根据用户历史地理位置划分的细分受众人群。 - 有相同受众特征的细分受众人群。 - 有具体兴趣爱好的细分受众人群。 - 处于某个人生阶段中的细分受众人群。 |

## 个性化广告敏感兴趣类别的限制

- <strong>概述</strong>

  在个性化广告中，限制使用敏感的兴趣类别以及使用这些类别去定向用户。在判断什么是敏感兴趣类别时，鲸鸿动能广告会秉持以下内容定向的原则：

  - <strong>法律限制的内容</strong>：广告必须符合法律规定。
  - <strong>个人正经历的困境</strong>：广告不得通过利用用户所面临的艰难或困境来定向他们。
  - <strong>个人</strong> <strong>身份和信仰</strong>：广告不得根据有歧视含义的信息或不公平、污蔑的类别来定向用户。

- <strong>法律限制的内容</strong>

  无论在何处展示广告，广告都必须遵守所有适用的法律法规。您负责确保自己的广告遵守所有适用政策与法律规定。请参阅以下具体示例，了解我们不允许在广告中出现哪些内容。

  | <strong>受限制的敏感类别</strong> | <strong>限制详情</strong> |
  | --- | --- |
  | 涉及 13 周岁以下用户 | 1. 收集不满 13 周岁（或您所在国家/地区的法定年龄）的儿童的个人信息。 2. 将兴趣内容定向到不满 13 周岁（或您所在国家/地区的法定年龄）的儿童。 |
  | 医药相关内容 | 1. 宣传临床试验受试者招募。 2. 受限制或当地法律禁止的药品。 |
  | 酒精饮料 | 酒精饮料和类似于酒精饮料的饮品。 |
  | 赌博 | 1. 在线和线下赌博； 2. 与在线赌博有关的信息； 3. 以赢取现金或奖品为目的的在线非赌场游戏； 4. 在线赌场游戏， 5. 涉及金钱交换； 6. 宣传赌博的实体赌场。 |
- <strong>个人正经历的困境</strong>

  我们深知，用户不希望看到广告在利用他们所面临的艰辛、困难和苦难，因此我们不允许广告主依据这方面的信息投放个性化广告。个人所面临的这些困苦包括健康问题、治疗、手术、个人失败、艰辛或个人的创伤经历。此外，您也不得对用户做出否定评述。您负责确保自己的广告遵守所有适用政策。请参阅以下具体示例，了解我们不允许在广告中出现哪些内容。

  | <strong>受限制的敏感类别</strong> | <strong>限制详情</strong> |
  | --- | --- |
  | 健康问题 | - 身体或精神健康问题，包括疾病、性健康和慢性健康问题（即需要长期护理或控制的健康问题）。 - 治疗或控制慢性健康问题的产品、服务或手术，包括非处方药和医疗器械。 - 任何与身体私密部位或功能有关的健康问题，包括生殖器、肛肠或泌尿系统的健康。 - 侵入性医疗手术，包括整容手术。 - 残疾。 |
  | 负面财务状况信息 | 个人财务危机、财务困难或破产。 |
  | 关系困境 | 个人与家人、朋友的关系或其他人际关系陷入困境。 |
  | 涉及犯罪 | 个人犯罪记录、犯罪行为、犯罪指控或刑事诉讼。 |
  | 虐待和创伤 | 属于虐待、犯罪或其他创伤事件受害者的个人身份。 |
  | 否定评述 | 否定用户的内容，或者从负面角度或带偏见地宣传任何类别的内容。 |
- <strong>个人身份和信仰</strong>

  我们认为，身份和信仰体系有着强烈的个人色彩，它们来自于多样的文化习俗、地理环境、历史传统和个人生活经历。我们不允许根据身份和信仰来定向用户。您负责确保自己的广告遵守所有适用政策。请参阅以下具体示例，了解我们不允许在广告中出现哪些内容。

  | <strong>受限制的敏感类别</strong> | <strong>限制详情</strong> |
  | --- | --- |
  | 性取向 | 性取向，包括女同性恋、男同性恋、双性恋、疑性恋或异性恋取向。 |
  | 工会成员身份 | 工会以及暗示了解用户工会成员身份的广告。 |
  | 宗教信仰 | 个人宗教信仰，例如：宗教场所、宗教指导、宗教教育或大学、宗教用品。 |
  | 种族和民族 | 个人种族或民族，包括但不限于：面向特定种族或民族的出版物、面向特定种族或民族的大学、面向特定种族或民族的交友服务。 |
  | 边缘化群体 | 边缘化或弱势社会群体的成员，例如特定社会阶层、移民或难民。 |
  | 跨性别者认同 | 个人认同的性别不同于天生的性别，或者不符合非男即女的性别认同模式。例如：有关变性、跨性别者歧视性的信息。 |

## 个性化广告数据收集和使用政策

- <strong>数据的收集和使用中，不允许</strong> <strong>出现的情况</strong>

  我们希望用户相信，有关他们的信息会受到尊重并得到妥善处理。因此，我们的广告合作伙伴不应滥用这些信息，也不应在没有明确目的、未进行恰当披露或未采取适当安全措施的情况下收集这些信息。我们列举了一些示例来说明应避免在广告中出现的情况<strong>：</strong>

  | <strong>不允许</strong> <strong>出现的情况</strong> | <strong>详情</strong> |
  | --- | --- |
  | 数据安全保护力度不足 | 未能依据相关业界标准对所收集的信息类型采取适当的安全措施。 |
  | 滥用个人信息 | 1. 以未经用户同意的方式使用其个人信息。 2. 直接使用用户个人信息来称呼用户的广告。 3. 暗示知晓用户个人信息的广告。 4. 投放收集或包含个人身份信息的广告，包含收集个人的姓名、手机号、邮箱号、住址等等。 |
  | 使用定向的受众群体范围少于5000的再营销列表 | 这包括将再营销列表与其他定向条件（如地理位置限制或其他细分用户群）结合使用的情形，从而导致广告仅定向到人数很少的用户。 |

- <strong>关于再营销</strong>

  通过 鲸鸿动能广告平台中的再营销或相似受众群体功能，您可以定位到之前访问过您的网站或使用过您的应用的用户，并向合适的用户展示合适的广告内容。这些功能可以有效地吸引以前的访问者回访您的网站或应用，不过，您应该告知他们，您会在网站或应用中针对再营销或相似受众群体收集相关用户的信息。

  - 对用户而言：不想看到个性化广告的用户，可以关闭HUAWEI设备的个性化广告的开关。
  - 对广告主而言：未经您的同意，鲸鸿动能广告不会允许其他广告主使用您的再营销列表或相似受众群体列表。
- <strong>再营销隐私权政策应包含哪些内容</strong>

  当您在自己的网站或应用中使用再营销或相似受众群体功能时，需要在隐私权政策中包含以下信息：

  - 有关如何在在线广告中使用再营销或相似受众群体定位功能的说明。
  - 有关第三方供应商（包括鲸鸿动能广告）如何将您的广告在互联网中的网站或应用上展示的信息。
  - 有关第三方供应商（包括鲸鸿动能广告）如何根据用户过去访问您的网站或使用您的应用的情况，使用 Cookie 和/或[广告标识符](https://developer.huawei.com/consumer/cn/doc/development/HMSCore-Guides/oaid-0000001050783198)来投放广告的信息。
  - 有关访问者可以关闭HUAWEI设备的个性化广告的开关，让鲸鸿动能广告停止对其使用 Cookie 或[广告标识符](https://developer.huawei.com/consumer/cn/doc/development/HMSCore-Guides/oaid-0000001050783198)的这一信息。

## 如何遵守政策

1. <strong>了解并</strong> <strong>遵守条款中的政策</strong>

   你需要了解[个性化广告敏感兴趣类别的限制](#section022419182716)。确保您的广告、网站或应用符合个性化广告政策的规定。请注意，即使广告未使用敏感类别进行定位，某些类型的广告内容也属于禁止的范围。

   如果您无法及时解决这些违规问题，请移除您的广告，以免您的账号将来因为存在过多的拒登广告而被暂停。
2. <strong>从您的</strong> <strong>广告</strong> <strong>中移除违规的内容</strong>

   您可以查看在广告中我们[禁止的内容](https://developer.huawei.com/consumer/cn/doc/promotion/industry-admission-rules-0000001189244454#section18398121913)，请移除不符合此政策的所有内容。随后，您需要申请审核，之后才能进入下一步。

   您完成修改并提交投放申请后，系统会将其发送给我们进行审核。多数广告可在 1 个工作日内完成审核，但有些广告可能需要更长时间，原因是相关审核流程较为复杂。
3. <strong>从您的</strong> <strong>广告</strong> <strong>中移除违规的定向类型</strong>

   在我们审核完网站或应用后，定向人群将生效。请确保您的广告符合个性化广告政策的规定。如果广告不符合个性化广告政策的规定，请移除使用这些被定向的受众或再营销的用户。
