# 消息设置

## 功能简介

鲸鸿动能投放平台提供消息通知功能，通知方式包括站内信、手机短信和邮箱三种，消息类别包含系统消息、审核消息、账户消息和财务消息。

## 操作步骤

1. “消息设置”页面支持添加联系人，单击右侧“编辑”，可自定义发送通知的消息类别与通知方式。![](./img/zh-cn_image_0000001773689845_eaf6ce605c94.png "点击放大")
2. 通知方式如选择“站内信”，则可单击账户右上方“![](./img/zh-cn_image_0000001773809197_dcdac3463c8e.png "点击放大")”查看消息内容，在该页面也可进入“消息设置”界面。

   ![](./img/zh-cn_image_0000001726209466_258cf7bc2ef7.png "点击放大")
