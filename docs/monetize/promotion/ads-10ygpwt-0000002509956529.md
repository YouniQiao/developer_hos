# 2025年10月高频问题Q&A

<strong>Q1：通过Marketing API获取的版位与鲸鸿动能后台展示的版位不一致？</strong>

<strong>A：</strong>版位查询相关接口当前已全面支持全域智投版位，如果需要查询一致，需先添加过滤条件，具体可以查看文档：&lt;https://developer.huawei.com/consumer/cn/doc/promotion/ads_new_api24-0000001405520288&gt;。

<strong>Q2：通过Marketing API创建任务、版位元素时API接口文档里面均有“推广目的”字段，但是广告后台只有营销目的和应用推广，这是什么原因</strong> <strong>？</strong>

<strong>A：</strong>Marketing API里面推广目的字段是后台根据定向条件判断，只存在于应用任务，如果定向是已安装 ，推广目的就是促活；如果定向是未安装，推广目的就是拉新。

<strong>Q3：服务商账户如何接入Marketing API？</strong>

<strong>A：</strong>① 如果服务商以往使用经理账户或投放账户获取过鉴权的华为账号，在联盟完成开发者认证并创建凭证的客户端ID已申请开通权限，那么本次可以继续使用那个客户端ID，无需新申请客户端ID，直接看[接入流程文档第四步](https://developer.huawei.com/consumer/cn/doc/promotion/ads_api09-0000001058204667)开始操作（使用服务商账户密码登录获取access\_token）；

② 如果服务商以往从未通过任何方式对接过，本次使用自己的服务商账户密码接入，还是需要按Marketing API文档流程，重头开始完成并申请权限。

<strong>Q4：如果想要搭建任务做直播推广是需要另行申请吗？</strong>

<strong>A：</strong>当前有2种方式：

① 如果搭建任务的时候选择应用推广，然后放对应的直播间DP链接，用来给直播间引流，则不需要申请，直接搭建任务即可。

② 如果是直接在搭建任务的时候选直播推广，这个还是要找对应行业运营另行申请才行。

<strong>Q5：如果修改计划名称超过2次会影响oCPC赔付吗？</strong>

<strong>A：</strong>不影响。oCPC赔付是以单任务为维度的，修改次数要求也是：保障周期内，每天修改任务出价、定向、创意总次数不能超过2次。更多赔付规则可查看：&lt;https://developer.huawei.com/consumer/cn/doc/promotion/ads_jlzc_ocpc2-0000001880794312&gt;

<strong>Q6：如果想要测试同一小程序内不同页面的投放效果，但是后台同一个小程序做资产只能新增一次，那怎样可以实现我这个需求？</strong>

<strong>A：</strong>同一个资产在事件资产管理里面只能添加一次，可以在建任务时候，不同任务上的链接用不同页面的链接来填写；比如落地页A对应任务名称X，落地页B对应任务名称Y，资产用同一个小程序ID。

<strong>Q7：可以通过经理账户给不同的子客共享不同产品的人群包吗？</strong>

<strong>A：</strong>当前可以通过经理账户把关联账户下的自定义人群包、商品库、维纳斯落地页共享给其他关联广告账户使用。具体操作指引可参考：&lt;https://developer.huawei.com/consumer/cn/doc/promotion/ads-jinlizhanghu-0000001878420417#section257950185210&gt;

<strong>Q8：子客账户更换经理账户后，新的经理账户可以看到子客账户的历史数据吗？</strong>

<strong>A：</strong>可以的。

<strong>Q9：选择浏览器搜索直达这个版位，创建任务时并没有设置关键词，那么什么情况下会展示所投放的广告？</strong>

<strong>A：</strong>选择浏览器搜索直达这个版位不设置关键词，当前是根据app名称、创意内容，和用户搜索query（语义）匹配展示广告的。

<strong>Q10：账户显示资质过期是否影响账户投放？</strong>

<strong>A：</strong>账户显示资质过期30天内不影响账户投放，超过30天后不支持新建任务，建议提前更新资质上传过审。

<strong>Q11：想要投放鸿蒙应用，为什么后台没有鸿蒙应用可以选？</strong>

<strong>A：</strong>如果想要在广告平台搭建任务的时候能够选到鸿蒙应用，需要跟行业运营另行申请才行。
