## 修改限额约束

|  |  |
| --- | --- |
| 值 | 描述 |
| daily\_budget | 1.日限额不低于最小限额，现阶段为：500。  2.日限额修改幅度不低于最小修改幅度，现阶段为：10。  3.日限额修改不能大于每天最多修改次数，现阶段为10次。  4.日限额和出价比值应该在合理范围内。 |

## 日限额说明

|  |  |
| --- | --- |
| 值 | 描述 |
| daily\_budget | today\_daily\_budget有值，则daily\_budget为当日日限额。  tomorrow\_daily\_budget有值，此时daily\_budget为次日日限额。  出价不能大于日限额。  日限额和出价比值应在合理的范围内。 |
