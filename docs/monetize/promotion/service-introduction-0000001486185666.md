# 业务介绍

## 简介

Push用户增长平台基于智能大数据分析、华为消息推送中台和PushKit客户端构建了端、管、云、数相结合的用户增长平台，通过广泛的设备触达、精准的人群圈选、丰富的内容样式、智能的展示时机，助力开发者应用用户增长。

针对安装应用后从未打开过或多天未使用应用的用户，Push用户增长平台可以通过精准的人群匹配算法、个性化的文案内容、合理的展示时机和丰富的展示样式，提升消息曝光度，吸引用户点击，从而增加应用活跃用户数。

## 产品功能

Push用户增长平台主要包含以下功能：

- 客户接入

  用户增长平台提供[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)和Marketing API两种接入方式。客户可以通过登录[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)使用用户增长门户进行自助运营活动投放，也可以通过Marketing API对接Push用户增长平台完成任务投放和管理。
- 投放平台

  包含目标设定、人群圈选、文案创意、呈现时机和效果归因的活动推广的完整链路。

  - 目标设定：支持沉默唤醒和促激活，面向两种不同的场景的推广目标。
  - 人群圈选：支持标签人群圈定、私有人群上传、自定义人群直投和RTA筛选，保障目标人群更精准。
  - 文案创意：支持文本、图片、按钮基本样式，支持文案AB，支持自定义小图标、背景图、自定义标题颜色等高级样式，增加曝光效果，提升点击率。
  - 呈现时机：支持定义消息呈现的时机，立即显示、下拉通知栏显示、亮屏显示，增加曝光效果，提升点击率。
  - 效果归因：支持效果统计分析，点击回执上报。
- 消息中台

  提供稳定可靠和覆盖广泛的消息连接，以及大容量、高并发的消息发送能力。
- Push Kit

  以常驻服务运行在终端设备，提供稳定可靠的消息连接、消息收发和通知展示能力。针对增长服务提供多种高级显示样式和消息展示时机：

  - 支持手机亮屏时展示
  - 支持下拉通知栏时展示
  - 支持送达时展示
  - 支持铃声/振动
  - 支持自定义标题颜色
  - 支持自定义小图标
  - 支持背景图案

## 产品优势

- 设备触达广泛

  Push用户增长平台可覆盖华为手机。
- 人群圈选精准

  基于智能化大数据分析，提供丰富的基础标签、特有的用户行为标签和兴趣标签，人群圈定更精准。
- 显示样式丰富

  提供多种消息显示样式，包括振动、铃声、自定义小图标、自定义标题颜色、背景图、大图等，曝光效果更好，提升点击率。
- 展示时机智能

  提供多种智能化消息展示时机，包括亮屏时展示、送达时展示、下拉通知栏时展示等，有效提升消息点击率。
