# 2025年10月

<strong>【规则新增】通用-伪科学资讯内容</strong>

1.禁止使用无科学依据的营养、养生、护肤知识、生活窍门/建议，通过噱头、未经证实的知识吸引人关注

a.妖魔化”常见食物或日常行为直接与疾病划等号；如“睡前、早起喝白开水的人，肾脏可能都烂成渣了”“3种食物别再吃了，血管变硬变脆都是因为它”。

b.将普通食物或日常行为“神话”包装有医疗保健效果；如“血压稳如泰山，认准1块钱蔬菜，一吃安神、二吃利尿、三吃调血脂““西蓝花富含胆碱和叶酸，协同健脑，让你拥有年轻体态”。

c.通过分裂正常的健康行为和饮食，来创造耸人听闻的“知识点”，制造虚假健康对立与夸大风险；如“早上喝水和晚上喝水对身体效果那可不一样”“晨起空腹喝水和饭后喝水，效果差别太大了！”“大蒜尽量少吃，最好不吃？看完才明白”。

d.将复杂的健康问题简单归因于单一食物或行为，将其塑造为“万能钥匙”或“绝对禁忌”，使用绝对化指令且缺乏依据，来诱导受众群体类要不要吃某类食品、做某个动作；如“想要好身体，少吃鸡蛋/牛奶，多吃三果”。

e.无科学依据的中老年养生知识类，来制造健康焦虑/渴望，提供不现实的解决方案，去诱导用户使用广告所推广的商品；如“人老了，真正需要的是XX”。

2.禁止利用虚构事迹（尤其是高龄老人）推广产品或服务

a.通过虚构事迹（利用高龄老人来作为信任背书），制造健康焦虑/渴望，提供神化单一方案/食物，进行推广产品或服务；如“84岁悬崖跳水第一人，能做到这样的壮举全靠这一点”“105岁老人的长寿窍门，不靠顿顿补品，竟是这常见好物”等。

3. 禁止无科学依据的商品功效分析/建议/介绍

a.通过推荐常规健康认知中不被提倡（甚至被认为有健康风险）或无依据的饮食、行为和神化未知物品，来针对受众群体进行营销推广商品；如“年龄大了，建议多喝隔夜茶，现在知道也不晚”“人过50，多喝过夜水，多亏知道的早”。
