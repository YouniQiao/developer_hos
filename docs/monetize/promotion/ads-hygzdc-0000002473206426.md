# 房地产

## 一、房产广告投放资质要求

1. 房产涉及具体预售楼盘销售需提供《商品房预售许可证》，涉及现房销售需提供《商品房现售备案证明》，涉及二手房销售广告，须提供房屋产权证明以及商品房销售委托书。
2. 广告中涉及市政配套（楼盘周边的交通、公园、医院等市政类设施）描述，以推广楼盘为参照地点，请提供包含所提到的设施（如现有交通、公园、医院等）的地图截图（需为知名地图软件）。
3. 房产广告中涉及贷款服务的，应当载明提供贷款的银行名称及贷款额度、年期与合作证明；若广告主无法详细说明贷款细则与提供资质，可在广告内说明：“贷款规则详见销售现场介绍”等引导语。

## 二、房产广告信息备注要求

1.推广新盘广告，需在广告外层素材或落地页首屏标注以下信息（视频广告需全程标注）

a.楼盘预售证号或销售许可证书号；若推广楼盘名称与预售证中名称不一致，需同时标明注册名和推广名，如“注册名（预售证名）：A楼盘，推广名：B楼盘”

b.若为中介服务机构代理销售的，需载明该机构名称

c.楼盘开发商名称

2.广告宣传内容若涉及户型面积，应当表明为建筑面积或者套内建筑面积：

a.外层素材中出现户型面积介绍，均需清楚标明

b.落地页涉及户型面积介绍，可就近标明，或在落地页下方说明“广告中提及的面积均为建筑面积或套内建筑面积”

c.房产广告中使用建筑设计效果图或者模型照片的，应当在广告中注明

d.房产广告中涉及的交通、商业、文化教育设施及其他市政条件等，如在规划或者建设中，应当在广告中注明

e.广告中对价格有表示的，应该表示清楚为实际的销售价格及价格的有效期限

f.房地产广告中涉及房地产价格评估的，应当表明评估单位、估价师和评估时间

g.房产广告中涉及不限购不限贷，需与当地实际政策相符，并且需要在广告中说明“不限贷不限购”要求（三选其一）：

- 若为当地政策不限购，可说明“无任何限购条件”。
- 说明具体不限贷限购的条件，比如：“须购买当地社保”或“不能用住房公积金贷款，只能贷款50%”等。
- 标注说明：限购条件查询XX（当地政府）文件《XXX》。

## 三、房产广告房源信息应当真实，不得涉及违规营销

1.不得宣传“买一层送一层”等含有引导擅自改变房屋单位分割或者违章搭建的内容，或将同一套房产中不纳入建筑面积计算的空间宣传为“赠送面积”。

2.不得以项目到达某一具体参照物的所需时间表示项目位置，如“距离市中心仅20分钟”等。

3.房产广告中不得利用其他项目的形象、环境作为本项目的效果。

| <strong>驳回示例</strong> | <strong>驳回示例</strong> | <strong>通过示例</strong> |
| --- | --- | --- |
|  |  |  |
| ×问题解析：宣传“买一层送一层” | ×问题解析：违规展示到达参照物时间 | √仅供参考：支持使用真实距离描述 |

4.商业办公类房地产广告发布时，应当表现商业办公用途，不得出现“公寓”等带有居住功能的用语，不得使用“卧室”、“厨房”等混淆房屋使用性质的画面或者文字。

5.不得出现哄抬房价的内容, 如：“楼王”、“地王”、“帝王核心区”、“黄金屋”等。

6.不得涉及0首付、首付分期类或变相垫付首付款等违反国家有关价格管理的规定。

7.为避免产生价格虚假或误导宣传，楼盘宣传或整体平台业务宣传须采用模糊价格，如“xxxx 万起”。

8.不得涉及融资或变相融资的内容，如全额预收房款进行变相融资。

9.房产广告中请勿出现学区房、学位房描述。

10.不得直接或间接出现升值投资回报的内容，例如：投资回报率、“低投入，高回报”等类似措辞。

11.房产广告中不得含有广告主能够为入住者办理户口、就业、升学等事项的承诺。
