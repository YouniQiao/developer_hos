     6|# 创建推广产品
     7|
     8|通过此接口可以推广产品
     9|
    10|<strong>请求地址</strong>
    11|
    12|https://ads.cloud.huawei.com/openapi/v2/promotion/product/create
    13|
    14|<strong>请求方法</strong>
    15|
    16|<strong>POST</strong>
    17|
    18|<strong>请求参数</strong>
    19|
    20||  |  |  |  |
    21|| --- | --- | --- | --- |
    22|| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
    23|| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
    24|| product\_type | string | 是 | 推广产品 枚举限制 详见[【推广产品】](https://developer.huawei.com/consumer/cn/doc/promotion/ads_api73-0000001058884698#section1526319223472) |
    25|| attrs\_struct | Struct6 | 是 | 产品信息 |
    26|
    27|Struct6定义
    28|
    29||  |  |  |  |
    30|| --- | --- | --- | --- |
    31|| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
    32|| app | Struct7 | 否 | 应用信息，product\_type为ANDROID\_APP时填写 |
    33|| quick\_app | Struct8 | 否 | 快应用信息，product\_type为QUICK\_APP时填写 |
    34|| web\_page | Struct9 | 否 | 网页信息，product\_type为WEB时填写 |
    35|| harmonyos\_native\_app | Struct10 | 否 | 鸿蒙应用，product\_type为HARMONYOS\_NATIVE\_APP时填写 |
    36|| harmonyos\_meta\_service | Struct11 | 否 | 鸿蒙元服务，product\_type为HARMONYOS\_META\_SERVICE时填写 |
    37|
    38|Struct7定义
    39|
    40||  |  |  |  |
    41|| --- | --- | --- | --- |
    42|| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
    43|| store\_app\_id | string | 是 | 华为应用市场App ID |
    44|
    45|Struct8定义
    46|
    47||  |  |  |  |
    48|| --- | --- | --- | --- |
    49|| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
    50|| deeplink\_url | string | 是 | 快应用直达地址 regexp = "^[^@]+$" 最长2048 |
    51|
    52|Struct9定义
    53|
    54||  |  |  |  |
    55|| --- | --- | --- | --- |
    56|| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
    57|| dest\_url | string | 是 | 落地页地址 (国内) regexp = "^(https):\/\/[\w\-\_]+(\.[\w\-\_]+)+([\w\-\.\[\]\\&#123;\\&#125;,?^=%&;$:/~\+#]\*[\w\-\[\]\\&#123;\\&#125;\?^=%&;$/~\+#])?$" 最长2048 |
    58|
    59|harmonyos\_native\_app (Struct10)定义
    60|
    61||  |  |  |  |
    62|| --- | --- | --- | --- |
    63|| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
    64|| app\_id | string | 否 | 华为应用市场APP ID。app\_id 和package\_name必填其一 |
    65|| package\_name | string | 否 | 应用包名。app\_id 和package\_name必填其一 |
    66|
    67|harmonyos\_meta\_service (Struct11)定义
    68|
    69||  |  |  |  |
    70|| --- | --- | --- | --- |
    71|| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
    72|| app\_id | string | 否 | 华为应用市场APP ID。app\_id 和package\_name必填其一 |
    73|| package\_name | string | 否 | 应用包名。app\_id 和package\_name必填其一 |
