# 合作指引

## 1. 适合联运合作的内容及服务：

1. 游戏虚拟道具及数字内容：包括但不限于游戏虚拟道具（含游戏币、宝石）、额外的生命或回合、特殊道具或装备、角色或虚拟形象、更多关卡、游戏时间或信息内容产品；
2. 应用增值功能或内容：例如应用的无广告版本，免费版中不提供的新功能，或应用中的数字收费内容；
3. 订阅服务：例如在线播放音乐、视频、图书或其他媒体服务；数字出版物，包括与实体出版物捆绑销售的数字内容；以及社交网络服务；
4. 云软件商品：数据云存储服务、企业生产力软件和财务管理支持软件；
5. 会员权益：与非会员差别化的商品购买或服务使用体验。

## 2. 不适合联运合作的范围

手机基础类应用、桌面类应用、省电类应用、安全辅助类应用、刷机类应用、硬件评分类应用、应用市场类应用、Android助手类应用、恶意刷量应用、需使用VPN的应用、成人用品类应用、无资质金融理财、无资质医疗类、含积分墙的应用、强制ROOT权限、双开类、彩票类、抢红包类、夺宝类应用，以及华为认为不适合进行联运合作的其他应用类型。
