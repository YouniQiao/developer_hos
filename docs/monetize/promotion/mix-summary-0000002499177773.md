# 变化点汇总表

| 分类 | 涉及模块 | 现状 | 整合升级后 |
| --- | --- | --- | --- |
| 账户体系 | 账户管理变化 | 应用市场应用推广入口投放 | 1. 整合升级后仅保持鲸鸿动能广告入口，原应用市场应用推广入口登录后，会自动跳转鲸鸿动能平台。  2. 客户原在应用市场应用推广投放资产迁移到鲸鸿动能，如报表数据、投放数据、历史创建人群包等。  3. 账户开户流程、任务创建和效果查看、推广账户人员授权等，均在融合后平台操作查看 |
| 广告管理平台 | 开户流程——可推广范围 | 分别在应用市场应用推广和鲸鸿动能平台开户后，维护两个平台的账户 | 1. 存量账户支持拓展推广范围，仅支持：  应用市场应用推广：直客/直客管理者账户，支持拓展鲸鸿动能广告  鲸鸿动能广告：普通直客/ka直客，支持拓展应用市场应用推广  2. 新开账户可以同时勾选应用市场应用推广和鲸鸿动能广告。实现一个直客账户投放应用推广和鲸鸿动能。 |
| 广告管理平台 | 资金管理变化 | 资金账户类型——应用市场推广基金 | 鲸鸿动能广告基金（现金共用） |
| 广告管理平台 | 任务层级变化 | 三层（任务+子任务+创意） | 四层（计划+任务+子任务+创意） |
| 广告管理平台 | 推广 | 任务列表、子任务列表、创意列表 | 支持查看计划、任务、子任务、创意等任务创建详情，批量修改日预算功能迁移到计划层 |
| 广告管理平台 | 在线提单入口 | 在线提单 | 意见反馈 |
| 广告管理平台 | 账户余额查看 | 应用推广可用总余额、锁定金额、现金余额、赠送余额、耀星券余额 | 无推广可用总余额、现金账户、虚拟金、耀星券余额 |
| 广告管理平台 | 提醒设置 | 首页——提醒设置 | 小铃铛——点击所有消息——消息中心 |
| 广告管理平台 | 直客/子客 | —— | 新增概览页，查看效果总览和最近创建的任务 |
| 广告管理平台 | 直客/子客 | 双权限客户：应用市场应用推广和鲸鸿动能均有推广账户，具备推广权限 | 双权限客户：历史鲸鸿动能广告和应用市场应用推广均有直客推广账户。数据合并在一个账户里，可分别设置鲸鸿动能广告和应用市场应用推广账户预算  “投放网络”字段区分：应用市场应用推广——应用市场应用推广；展示广告网络——鲸鸿动能广告 |
| 广告管理平台 | 报表 | 可查看维度：整体数据、子任务数据、搜索数据、创意数据 | 可查看维度不变（整体数据、子任务数据、搜索数据、创意数据），双权限客户新增支持区分鲸鸿动能广告和应用市场应用推广的投放数据 |
| 广告管理平台 | 工具 | 只可查看应用市场应用推广工具能力 | 1. 双权限客户，工具入口支持区分鲸鸿动能广告和应用市场应用推广支持的投放工具能力  2. 商品管理工具，日落 |
| 广告管理平台 | 解决方案 | 贷款行业解决方案和卸载召回解决方案 | 不变，仅应用市场应用推广有贷款行业解决方案和卸载召回解决方案 |
| 广告管理平台 | 转账记录、财务记录 | 转账记录有单独入口，财务记录在工具 | 放在【小钱包】下，原财务记录、转账记录从工具入口转移到小钱包入口；支持查看账户支出、账户日预算、分资金账户余额； |
| 账户体系 | 开户流程——直客 | 1. 开发者注册、实名认证——开发者联盟  2. AppGallery Connect上架应用  3. 充值——开发者联盟  4. 签署协议开通直客账户——应用市场应用推广投放端  5. 申请推广评测，创建推广任务——应用市场应用推广投放端 | 1. 开发者注册、实名认证——开发者联盟  2. AppGallery Connect上架应用  3. 充值——开发者联盟  4. 选择投放区域、签署协议开通直客账户——鲸鸿动能平台  5. 申请推广评测，创建推广任务——鲸鸿动能广告平台 |
| 账户体系 | 开通流程——直客团队账户（经理账户） | 1. 联系运营开通直客团队白名单  2. 在投放端升级团队账户  3. 直客管理者新建直客协作者账户  4. 授权应用、授权华为账号  登录入口：选择角色类型为直客管理者的账户—进入经理账户平台操作 | 1. 联系运营开通直客团队白名单（应用市场应用推广、鲸鸿动能广告分别开通）  2. 直客登录投放端，工具——账户辅助——应用推广直客团队账户，点击确认升级。  3. 选择角色类型为直客管理者账户，点击登录经理账户平台  4. 经理账户首页关联新建直客协作者账户  5. 查看管理的直客协作者账户&转账等  入口：登录页选择角色类型为直客管理者账户或在投放端右上角切换账号进入。 |
| 账户体系 | 直客团队账户功能（经理账户） | 1. 关联关系绑定：直客管理者邀请直客协作者账户，填写登录直客协作者账户华为账号和授权投放应用即可 | 1. 关联关系绑定：直客管理者切换到经理账户管理，通过首页发起直客协作者绑定。  2. 绑定后，直客协作者账户使用被邀请的华为账号登录投放端，签署协议即可投放。  关联入口：应用市场应用推广直客管理者可通过：经理账户平台——首页——关联账户 |
| 广告管理平台 | 2. 转账：团队账户——选择直客协作者账户——转账 | 2.转账：经理账户管理平台首页，选择转账账户——点击【转账】，转账时可以选择资金类型（现金、赠送金、返利金、耀星券） |
| 广告管理平台 | 3. 直客转账记录 | 3. 查看转账记录：经理账户平台——财务记录——转账记录页面，查看和协作者账户的资金流转情况，对应原应用市场应用推广的转账数据详情 |

| 广告管理平台 | 5.原应用推广不支持——直客协作者账户关联创建/删除/变更授权应用/更换华为账号等、资产共享操作日志 | 5.（新增）操作日志：记录管理者账户对协作者账户授权相关的新增、修改、删除等操作 |
