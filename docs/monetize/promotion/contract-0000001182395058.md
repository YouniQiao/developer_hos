# 合约广告

合约广告是鲸鸿动能广告与广告主约定在某一时间段内、在某些版位上固定投放该广告主的广告，提升企业产品与服务的知名度，并使用户产生积极的品牌联想。合约广告由鲸鸿动能广告运营团队代为投放。

鲸鸿动能广告当前支持应用市场合约广告。

<strong>[应用市场合约广告](/docs/monetize/promotion/contract-ad-shown-on-ag-0000001294223968)</strong>：应用市场的一种合约广告形式，在开屏、榜单图标、banner、以及push中固定投放广告，无需竞价即可展示，计费方式为CPD、CPT。
