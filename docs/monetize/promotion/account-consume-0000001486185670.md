# 查看账户消费情况

登录[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)，在右上角帐号旁的下拉框中选择“账户中心，”选择“费用 &gt; 账户概览” ，即可[查看账户内的消费总览](https://developer.huawei.com/consumer/cn/doc/development/AppGallery-connect-Guides/agc-account-bill-0000001200817917)，包括：

- 查看账户余额等基本信息
- 查看账单详情
