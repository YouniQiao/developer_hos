# 添加创意标签

1. 创建或修改推广任务时，若该任务支持推广创意功能，勾选“编辑辅助创意”。

   ![](./img/zh-cn_image_0000002525059902_53abd805004c.png "点击放大")
2. 进入“推广创意”页面，在“创意展示”区域，点击![](./img/zh-cn_image_0000002555989527_686b655e093a.png)。

   ![](./img/note_3.0-zh-cn_f0060aba628d.png) 

   - 因创意标签较多，建议您根据创意内容按照关键词模糊搜索，高效勾选对应创意标签。
   - 当前最多可添加5个三级标签。

   ![](./img/zh-cn_image_0000002525069570_02add93e6ea9.png "点击放大")
3. 在弹出的“创意标签选择”窗口输入创意相关的关键词或下拉选项，勾选三级创意标签，完成后点击“确定”。

   ![](./img/zh-cn_image_0000002556109493_e6cba1cd4bf1.png "点击放大")
