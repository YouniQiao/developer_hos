# oCPC产品激励政策

<strong>【政策公示】2026年oCPC产品激励政策</strong>

oCPC产品激励政策是鲸鸿动能推出的，为保障广告前期能得到充分的探索，促进持续投放的激励政策。oCPC产品激励政策的推出是为了降低客户投放成本，保障客户预算消耗，积累数据，提高模型学习效率，持续优化投放效果。

<strong>适用范围</strong>

1. 政策覆盖范围：所有投放oCPC的出海及海外本地客户，无需申请。
2. 政策有效期：2026年1月1日-2026年12月31日

<strong>激励规则</strong>

| <strong>激励门槛</strong> | <strong>执行细则</strong> |
| --- | --- |
| 转化目标 | oCPC激活/激活（HMS）/唤醒/付费/次留/注册 |
| 保障周期 | 任务首次投放起<strong>20个自然日</strong>  （10天为一个周期，共两个周期） |
| 保障要求 | 成本偏差 &gt; 20%（超过部分全部保障）  举例：广告主投放激活指标，出价$1，实际成本$1.2，不予保障；实际成本若大于$1.2，则对超出$1.2的部分给予保障。 |
| 转化数要求 | 每个周期（10天）内转化数量要求：  激活/激活（HMS）/唤醒/次留/注册&gt;=6个  付费&gt;=2个 |
| 修改次数要求 | 保障周期内，每天修改广告出价、定向、创意总次数不能超过2次，出价修改幅度不能超过20%。超出要求的任务，投放全周期不享受此激励政策。 |

<strong>结算规则</strong>

| <strong>结算规则</strong> | <strong>执行细则</strong> |
| --- | --- |
| 深度优化 | 深度转化的转化数要求按照深层目标转化数要求。  一阶段（学习期）按照浅层目标进行保障。  二三阶段（稳定期）按照深度转化目标进行保障。 |
| 激励金额 | 激励金额=总实际消耗-总预期消耗（1+成本偏差要求） |
| 激励发放 | 按月进行结算。激励金额将以返利金的形式充值进广告主返利金账户，分阿斯比格和香港服务两个账户发放。不可提现退款，有效期一年。 |
| 其他 | 1）保障金额起步为$0.01，金额不足$0.01，不予保障。  2）广告主有异常作弊等操作行为，不予保障。  3）使用了RTA修改出价的广告等，不予保障。 |

1. 账单和金额以鲸鸿动能数据为准，数据统计时区：UTC+8。
2. 转化目标次留的激励覆盖范围：首次曝光在2026年1月1日-2026年12月31日的任务。
3. 如您对“oCPC产品激励政策”有任何意见和建议，请及时反馈至鲸鸿动能运营人员，我们将在后续的版本中不断优化，感谢您一直以来的关注与支持。
4. 本规范季度审视并刷新，最终解释权归鲸鸿动能所有。

<strong>常见问题</strong>

<strong>Q1：什么叫首次投放？具体怎么定义保障周期？</strong>

<strong>A：</strong>即广告正式投放后，首次曝光给用户时开始，比如计划 1月 1 日新建，1月 2 日审核通过，1月 3 日产生首次曝光数据，则保障时间为 1月 3 日，及其之后的20个自然日（1月 3 日~ 1月 22日）。

<strong>Q2：保障周期激励金额如何计算？</strong>

<strong>A：</strong>以10天为一个周期，共保障2个周期，每个保障周期的激励金额单独计算

保障周期一：第1-10天投放，第11天后开始计算第1-10天投放产生的激励金额；

保障周期二：第11-20天投放，第21天后开始计算第11-20天投放产生的激励金额。

注：两个保障周期是独立的，保障周期一未满足激励门槛，不影响保障周期二；两个保障周期通用一套激励门槛标准。

<strong>Q3：哪些任务在政策有效期内</strong> <strong>？</strong>

<strong>A：</strong>首次曝光时间在2026年1月1日-2026年12月31日的任务可享受上述oCPC产品激励政策。

<strong>Q4：深度转化有什么要求？</strong>

<strong>A：</strong>转化数要求说明：深度转化的任务，转化数按照深度转化目标要求。以付费为例，每个保障周期（10天）内付费数&gt;=2个才达到转化数要求（对激活数不做要求）；
