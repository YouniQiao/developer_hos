# 平台优势

<strong>精准投放</strong>

- 丰富的流量来源，覆盖各场景海量入口
- 完善的用户画像标签
- 精准的数据处理能力

- 多种计费模式：CPM、CPC、CPI、CPD、oCPC
- Remarketing再营销、lookalike等AI模型投放策略实时调整
- OAID统一归因口径更有利于效果评估

<strong>优质体验</strong>

- 静默安装，全屏体验
- 一致性视觉呈现，打造高端品牌体验
