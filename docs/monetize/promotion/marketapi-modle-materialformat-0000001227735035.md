# MaterialFormat

| 字段 | 必选(M)/可选(O) | 类型 | 描述 |
| --- | --- | --- | --- |
| materialTitle | O | String | 素材标题。 |
| materialTitleEn | O | String | 素材英文标题。 |
| materialKey | O | String | 素材字段key值。 |
| materialSize | O | String | 素材尺寸。 |
| materialFileType | O | String | 素材文件类型。 |
| materialFileSize | O | String | 素材文件大小。 |
| extendRule | O | String | 其他限制。 |
