# 2026年3月高频问题Q&A

<strong>Q1：当服务商账户被审核驳回后，如何重新触发审核？</strong>

<strong>A：</strong>可以在服务商账户-账号管理-账户信息页编辑账户名称就可以触发。

<strong>Q2：在投放过程中，如果浅层出价和深度转化出价在同一天都各修改2次，那么是算2次修改调整出价还是4次呢？</strong>

<strong>A：</strong>根据当前oCPC产品激励政策深度转化修改次数说明，浅层目标和深度转化目标的修改次数均统计，如果浅层出价和深度转化出价在同一天/同一时间都各修改2次，那么广告出价修改次数记为4。

<strong>Q3：账户后台显示资质过期，但是上传更改的时候为什么页面底部没有编辑按钮？</strong>

<strong>A：</strong>首先请确认该账户是否是复制账户，如果是复制出来的账户，那么更新资质需要去来源账户操作。来源账户的行业资质上传更新资质审核通过以后，极速开户的复制账户里面行业资质也会同步更新。

<strong>Q4：如何找到鸿蒙应用ID？</strong>

<strong>A：</strong>开发者通过AppGalleryConnect平台获取鸿蒙原生应用APPID，登录到AppGalleryConnect平台，在项目列表中找到对应的鸿蒙应用项目，点击进入，选择右侧菜单栏项目设置-常规-应用信息板块-查看App ID。

<strong>Q5：如何找到鸿蒙设备的OAID？</strong>

<strong>A：</strong>HarmonyOS NEXT设备OAID获取方式：打开手机-设置→隐私和安全→ 跨应用关联→页面底部可看到“查看 OAID”。

<strong>Q6：元服务产品开户时需要提供《计算机软著资质证书》资质吗？</strong>

<strong>A：</strong>元服务产品开户时可以不提供《计算机软著资质证书》。

<strong>Q7：开户时的资质证照可以使用截图上传吗？</strong>

<strong>A：</strong>主体资质和行业资质上传要求都不支持传摄/截屏图片，可上传原件、拍照件、扫描件或加盖公章的黑白复印件。

<strong>Q8：药品优惠权益可以在鲸鸿动能平台推广吗？</strong>

<strong>A：</strong>平台禁止推广“三品一械”及医疗服务相关的卡券或促销优惠等诱导性消费产品，包括但不限于：红包、现金券、体验券、礼品券、折扣券、特价券、换购券、提货券等。

<strong>Q9：开户时，行业资质支持授权吗？</strong>

<strong>A：</strong>企业资质是企业符合相关行业规定的，证明自身生产等能力的相关文件、证件，不得出借他人使用。可依据企业主体实际的业务开展和经营情况提供相关证明材料。

<strong>Q10：管理式华为账号的登陆手机号可以修改吗？</strong>

<strong>A：</strong>管理式华为账号是使用已实名认证的华为账号登录的，所以如需修改安全手机号，可以登录[华为账号官网](https://id1.cloud.huawei.com/AMW/portal/home.html)在账户与安全界面进行修改操作。

<strong>Q11：维纳斯落地页的功能组件可以在预览环境下测试使用吗？</strong>

<strong>A：</strong>需要搭建试投放或者在正式投放环境下进行测试，以确保正常使用。

<strong>Q12：资质过期多久要补充资质？</strong>

<strong>A：</strong>当资质到期30天内，账户中有弹窗提示您及时补充资质，30天内及时更新资质并审核通过不影响账户使用及搭建新任务，当资质过期超出30天，账户将限制搭建新任务投放，请及时更新资质。

<strong>Q13：同一个素材搭建到了不同创意下是否会按一个素材元素统计消耗？</strong>

<strong>A：</strong>如果直接在元素列表看元素数据（不通过单个创意跳转），元素数据是基于素材sha256值（属性值）聚合，不区分元素归属。
