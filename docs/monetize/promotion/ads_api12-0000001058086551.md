# 术语介绍

|  |  |  |
| --- | --- | --- |
| <strong>术语</strong> | <strong>中文名</strong> | <strong>详细描述</strong> |
| 鲸鸿动能 | 鲸鸿动能 | 鲸鸿动能是基于鸿蒙生态的智慧营销平台，依托于华为1+8+N全场景生态及自有媒体营销能力，汇聚全球三方优质流量，为全球广告主和三方网盟提供广告营销和流量变现服务。 |
| 客户端 ID | 鲸鸿动能 Marketing API 客户端 ID | 鲸鸿动能 Marketing API 客户端是指通过鲸鸿动能 Marketing API 获取数据的软件系统。 |
| 密钥 | 客户端密钥 | 每一个客户端对应的密钥（用于后面获取access\_token）。 |
| scope | 授权作用域 | 用户授予的权限列表 |
| redirect\_uri | 回调uri | 授权后要回调的URI，即接收Authorization Code或者token后的跳转url |
| access\_token | 授权令牌 | 使用/oauth2/v2/token接口会同时获得access\_token。access\_token是调用API接口的调用凭证,用于服务端对API请求鉴权，默认有效期为1小时，一旦access\_token失效，则需要重新获取。 |
