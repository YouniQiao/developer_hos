# 2025年8月高频问题Q&A

<strong>Q1：军队文职培训内容可以开户投放广告吗？</strong>

<strong>A：</strong>不可以。鲸鸿动能平台已对“教育-军队文职培训”类内容实施全面禁投管控，目前不支持此类培训相关广告的开户与投放，感谢您的理解与配合。

<strong>Q2：进行事件资产管理时，corp\_id这个参数对应的是账户id吗？</strong>

<strong>A：</strong>是的，corp\_id就是账户id。

<strong>Q3：用H5落地页做事件资产时，配置了corp\_id收到的参数是-1，但是用应用做事件资产时，收到的是正常的账户ID，这是为什么？</strong>

<strong>A：</strong>这是因为不同资产类型，手动联调的结果会有差异，如果想收到的都是账户ID，那么可以搭建试投放广告做测试。因为试投放或者正式投放时会替换成真实的值。

<strong>Q4：已经在事件资产管理修改了监测链接，为什么下发的还是之前旧？</strong>

<strong>A：</strong>这是因为修改了监测链接之后，对应资产的事件会变成修改中而非已启用，这个时候就表明需要重新手动联调成功才行。当事件再次变成已启用之后新的监测链接才会生效。

<strong>Q5：RTA的三个策略：创意推荐、商品推荐DPA、调整出价系数需要单独再次申请权限吗？</strong>

<strong>A：</strong>要的，都需要跟行业运营单独申请权限才能使用，其中调整出价系数这个功能还需要再次联调才能启用。

<strong>Q6：Marketing API是否支持添加多张图片素材？</strong>

<strong>A：</strong>目前通过Marketing API创建全域智投广告，上传图片/视频时，当前限制一次只能传一张/一个，后续会开放批量多个，敬请期待。更多详情可查看文档：&lt;https://developer.huawei.com/consumer/cn/doc/promotion/ad-mktchaunjianquanyuguanggao-0000001686929114&gt

<strong>Q7：事件资产管理的转化数据跟推广报表里面的转化数怎么对不上？</strong>

<strong>A：</strong>这是因为两个地方的统计口径不一致导致的。事件资产页面的转化事件量，即使回传失败，也会计算，但是推广报表只计算回传成功的数量。所以建议都以报表为准比较准确，可以不用关注事件资产页面的数据

<strong>Q8：鲸鸿动能支持投放搜索广告吗？</strong>

<strong>A：</strong>如果想在鲸鸿动能投放搜索广告，可以选择浏览器搜索直达这个版位投放，当前仅支持投放应用/快应用。具体操作指导可参考：&lt;https://developer.huawei.com/consumer/cn/doc/promotion/ads-chuanjiansousuoguangao-0000001773515437&gt

<strong>Q9：已经生效的鲸鸿动能账户，开户主体名称发生变更，能直接修改吗？</strong>

<strong>A：</strong>已经生效的账户是无法修改企业全称的，但是不影响这个生效账户的正常使用。当前能正常搭建任务过审，就可以正常投放。如果后续涉及到更新资质等，那就需要用新的资质重新开户。

<strong>Q10：事件状态为未启用，可以直接搭建任务选择投放吗？</strong>

<strong>A：</strong>如果新建资产的时候打开了智能跟踪按钮，那么对应资产下面的未启用事件可以正常搭建任务选到去投放。

需注意：开启智能跟踪后可以免联调，但是最终还是要做回传，不回传就不会启用指标。

<strong>Q11：针对已经生效的账户想要修改审核/客户行业，可以改吗？</strong>

<strong>A：</strong>已经生效的账户无法修改审核/客户行业，如果想要修改只能新开户。

<strong>Q12：关于ocpc赔付，为什么我的成本保障报表里面的转化数跟推广报表里面的转化数不一致？</strong>

<strong>A：</strong>如果不是在保障周期内回传的转化数，是不会计算到成本保障报表里面的。所以保障周期结束后看成本保障报表的转化数跟推广报表的转化数可能会存在差异。

<strong>Q13：开户时的“对公银行打款认证”，打款方（平台）的对公账号是多少？</strong>

A：开户时，对公银行打款认证环节，打款账户主体名称为银联商务支付股份有限公司，以实际打款情况为准，打款账号随机，还请注意查收验证。具体可查看子客注册流程介绍：&lt;https://developer.huawei.com/consumer/cn/doc/promotion/ads-zikezhuce-0000001814790812&gt

<strong>Q14：华为市场付费推广和鲸鸿动能投放，开户的时候能用同一个邮箱开户吗？</strong>

<strong>A：</strong>可以。

<strong>Q15：已开具的充值订单发票，应该在哪里查看？</strong>

<strong>A：</strong>有2种方式可以查看：

1. 可在[开发者联盟](https://developer.huawei.com/consumer/cn/)后台“我的账户-账单” 中查看对应的发票情况。在这里获取到开票日期+发票号后，可前往电子税务平台查看数电发票情况；

2. 2025年2月份之后开的数电发票，会发送到系统维护的邮箱中（若需要修改接收邮箱，可在[开发者联盟](https://developer.huawei.com/consumer/cn/)后台中进行修改后并及时同步至运营/客服），可直接进行查看。

更多充值发票查看路径，可参考：&lt;/docs/monetize/promotion/ads-fwskpczlc-0000001815143162#section111577017238&gt

<strong>Q16：试投放的曝光，点击、转化等数据在投放后台会有展示吗？</strong>

<strong>A：</strong>试投放的曝光、点击数--如果系统判断有效，就会体现在数据里面；

试投放的转化数：这个需要看是否成功归因，如果被判断无效，就不会有显示。如果归因成功，就会有数据显示。

<strong>Q17：22年、23年的历史消耗数据可以查到吗？</strong>

<strong>A：</strong>当前经理账户的消耗报表支持选择最近两年的查看，投放后台的账户报表支持选择最近一年的查看。超出对应时间将置灰无法选择。
