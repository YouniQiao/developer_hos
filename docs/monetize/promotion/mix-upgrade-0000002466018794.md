# 升级直客团队账户

- 如您需要升级直客团队账户，请参考[直客团队账户申请模板](/docs/monetize/promotion/bp-start-guestteam-admin-0000001526225136)申请。运营开通后，您在直客账户可见“应用推广直客团队账户”入口，点击确认开通即升级为直客管理者账户。

![](./img/zh-cn_image_0000002466018990_70ad4cde9bc0.png "点击放大")

![](./img/zh-cn_image_0000002499177937_6f8fbc539617.png "点击放大")

- 开通后，应用推广直客团队账户入口隐藏，您可以在右上角切换账户，选择直客管理者账户，进入经理账户平台管理协作者账户

![](./img/zh-cn_image_0000002499137989_f9b3f0f029ae.png "点击放大")
