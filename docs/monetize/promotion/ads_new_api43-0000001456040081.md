# 创建定向包

【简介】通过共享定向包，可以使同一广告主的多个任务共用一个定向包。

<strong>请求地址</strong>

https://ads.cloud.huawei.com/ads/v1/tools/targeting\_package/create

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| targeting\_name | string | 是 | 定向包名称，最大长度不超过100必填 |
| targeting\_description | string | 否 | 定向包描述，最大长度不超过100 |
| targeting\_type | string | 是 | 定向包类型，包含应用类、非应用类  详见[【定向包类型】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section1368511219507) |
| gender\_struct | Struct2 | 否 | 性别定向，值为gender定向中的value  单选值，不支持复选  填充不限时value为UNLIMITED |
| age\_struct | Struct2 | 否 | 年龄定向，值为age定向中的value  可同时选择不同年龄段的值  填充不限时value为UNLIMITED |
| series\_type\_struct | Struct2 | 否 | 手机系列定向，值为series\_type定向中的value，  series\_type\_struct和device\_price\_struct不能同时选择  填充不限时value为UNLIMITED |
| device\_price\_struct | Struct2 | 否 | 设备价格定向，值为device\_price定向中的value，  series\_type\_struct和device\_price\_struct不能同时选择  填充不限时value为UNLIMITED |
| installed\_apps\_struct | Struct2 | 否 | 已安装应用定向，如果选择此定向，value填写[“true”]  product\_type为ANDROID\_APP时，installedappsStruct和notInstalledappsStruct为二选一  当任务里面的object\_type=APP\_ACTIVE时 必填  填充不限时value为UNLIMITED |
| not\_installed\_apps\_struct | Struct2 | 否 | 未安装应用定向，如果选择此定向，value填写[“true”]  productType为ANDROID\_APP时，installedappsStruct和notInstalledappsStruct为二选一  当任务里面的object\_type=APP\_DOWNLOAD或object\_type=APP\_RESERVE\_DOWNLOAD时 必填  填充不限时value为UNLIMITED |
| network\_type\_struct | Struct2 | 否 | 网络类型定向，值为network\_type定向中的value，可以复选  填充不限时value为UNLIMITED |
| audience\_struct | Struct2 | 否 | 自定义人群定向，值为pre\_define\_audience定向中的id;  pre\_defined\_audience\_struct和not\_pre\_defined\_audience\_struct选择的值不能相同  填充不限时value为UNLIMITED |
| not\_audience\_struct | Struct2 | 否 | 排除自定义人群定向，值为not\_pre\_define\_audience定向中的id;  pre\_defined\_audience\_struct和not\_pre\_defined\_audience\_struct选择的值不能相同  填充不限时value为UNLIMITED |
| current\_geo\_location\_struct | Struct2 | 否 | 当前在这里的人（地理划分）  值为current\_dmp\_geo中的value  与自定义地理位置定向current\_custom\_location\_struct  不能同时选择  填充不限时value为UNLIMITED |
| current\_level\_location\_struct | Struct2 | 否 | 当前在这里的人（发展划分）  值为current\_level\_geo中的value  与自定义地理位置定向  current\_custom\_location\_struct  不能同时选择  填充不限时value为UNLIMITED |
| residence\_geo\_location\_struct | Struct2 | 否 | 常驻在这里的人（地理划分）  值为current\_dmp\_geo中的value  与自定义地理位置定向  current\_custom\_location\_struct  不能同时选择  填充不限时value为UNLIMITED |
| residence\_level\_location\_struct | Struct2 | 否 | 常驻在这里的人（发展划分）  值为current\_level\_geo中的value  与自定义地理位置定向  current\_custom\_location\_struct  不能同时选择  填充不限时value为UNLIMITED |
| current\_custom\_location\_struct | Struct2 | 否 | 自定义地理位置定向  与地理划分，发展划分中的current\_geo\_location\_struct  current\_level\_location\_struct  residence\_geo\_location\_struct  residence\_level\_location\_struct不能同时选择  填充不限时value为UNLIMITED |
| ai\_target | Struct2 | 否 | 智能扩量  如果填值不为空，智能扩量开关ai\_target\_flag必须打开  取值详见[【智能扩量】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section3184194510) |
| ai\_target\_flag | string | 否 | 智能扩量开关：开关打开ai\_target字段如果不填就是默认突破  取值详见[【智能扩量开关】](/docs/monetize/promotion/ads_meijuzhi-0000001510863205#section3590746181218) |
| feature\_audience\_struct | Struct2 | 否 | 细分受众定向  值为feature\_audience的value  填充不限时value为UNLIMITED |
| content\_id\_struct | Struct2 | 否 | 内容ID定向，与内容标签不能同时出现，格式不能为中文，不能包含分号  填充不限时value为UNLIMITED |
| content\_label\_struct | Struct2 | 否 | 内容标签定向，与内容ID不能同时出现，content\_label中的value  填充不限时value为UNLIMITED |
| brand\_id\_struct | Struct2 | 否 | 品牌门店定向  填充不限时value为UNLIMITED |

Struct2定义

| value | string[] | 是 | 定向选中的值 |
