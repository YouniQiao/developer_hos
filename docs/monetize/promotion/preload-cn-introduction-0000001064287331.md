# 业务介绍

在默认的初始状态、出厂设置下，应用已经在终端设备中由厂商预先集成到ROM中，并且第一次开机后用户可以直接使用。终端设备包含但不限于手机、Tablet、个人移动终端、家庭网络终端、M2M终端等。
