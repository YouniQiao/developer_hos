# 2025年12月

<strong>【规则放宽】通用-封建迷信管控规则</strong>

支持使用“八卦”、“罗盘”等传统文化元素，但严禁将其用于宣扬封建迷信，或与产品/服务建立任何不当关联。

|  |  |
| --- | --- |
| ❌不支持封建迷信描述，改变命运 | ✅支持作为传统文化、国学或背景元素出现 |
|  |  |

<strong>【规则放宽】教育-拍题解题描述管控规则</strong>

支持投放“拍照解题类”相关功能，但素材宣传需注意导向，应聚焦于产品作为搜题解题工具、作业检查助手与学习辅导资源的合规用途，不得涉及或鼓励任何形式的作业抄袭行为。

|  |  |
| --- | --- |
| ❌不支持直白的答案描述，或暗示可抄 | ✅支持作为辅助学习用途描述 |
|  |  |
