# 2025年9月

<strong>【规则新增】开户资质-营业执照扫码核验规则</strong>

1. 2025年10月15日起，所有提交的营业执照需通过扫码一致性核验：

|  |  |
| --- | --- |
| ✅ 扫码核验通过条件 | ❌ 驳回场景 |
| 提供的营业执照二维码清晰、可正常扫描且扫码获取的信息需与营业执照上所载内容完全一致。 | 二维码无法扫描或解析（核验无果）。  扫码结果与营业执照任何一项信息不一致。 |

<strong>【规则新增】教育-金融理财培训</strong>

1. 推广内容涉及理财培训需提供对应讲师金融行业从业资格证，如《证券分析师资格证》或《证券从业资格证》；
2. 不支持出现证券业务，如“证券开户、证券咨询、证券买卖”业务，如：实操营/实战描述；
3. 不得出现个股/板块推荐及操作建议、买卖点提示、主力动向分享等；
4. 不得涉及实盘数据分享/分析/讲解/教学，仅支持对历史数据做数据分析。

<strong>【规则细化】通用-未成年人形象使用要求</strong>

<strong>1.针对未成年人形象要求，不得有以下场景：</strong>

- <strong>着装要求：</strong>不得出现未成年人仅穿着或主要穿着泳装、内衣、居家睡衣等过于暴露的服装。
- <strong>镜头与画面要求：</strong>镜头不得聚焦、特写或刻意强调未成年人的胸部、腰部、臀部、腿部等特殊部位；不得采用带有性暗示的拍摄角度或姿势。
- <strong>美妆产品试用禁止：</strong>严禁出现在未成年人身上直接试用美妆产品的场景。包括但不限于，彩妆产品：为其涂抹口红、画眼影/眼线、打腮红/粉底等；美妆工具：为其粘贴双眼皮贴、假睫毛等。

<strong>2.针对不满十四周岁的未成年人的商品或者服务的广告不得含有下列内容：</strong>

- <strong>诱导购买：</strong>不得含有直接或间接劝诱、怂恿未成年人向其家长或监护人要求购买广告商品或服务。例如， “快让妈妈给您买一个”等描述。
- <strong>模仿不安全行为：</strong>不得含有可能引发未成年人模仿的、危及人身和财产安全的行为，例如，危险动作：没有保护措施的攀高、野外游泳、玩火、触摸电源等；不当行为：模仿成人吸烟、饮酒等。
