# 2026年2月高频问题Q&A

<strong>Q1：如何修改极速开户时接收开户链接的邮箱？</strong>

<strong>A：</strong>需要在对应的来源账户里修改联系人邮箱。

<strong>Q2：鲸鸿动能后台的数据会有缓存嘛？投放端消耗报表和财务报表数据为什么会不一样？</strong>

<strong>A：</strong>目前在投放的任务，由于转化事件上报回传的滞后性，展点消数据会有延迟，可能会存在数日后数据报表仍有变化的情况。如果是要看推广数据、优化等，直接看前端推广报表里面的数据；如果是要对账，以右上角的财务数据为准，两个地方的统计口径是不一样的。

<strong>Q3：浏览器搜索直达版位的关键词设置有数量限制么？最多加多少个关键词？</strong>

<strong>A：</strong>当前限制是1000个。

<strong>Q4：图标跟非图标投放有什么区别？</strong>

<strong>A：</strong>关于投放是否包含图标请以文档里的展示场景描述为准，如有描述（展示场景包含图标样式）则有可能投放到图标，且目前全域智投的版位，只要生成的通用创意的创意素材宽高比是1:1且只有icon唯一元素也是可能统计到图标竞价资源的。更多竞价资源可以查看文档：[鲸鸿动能竞价资源介绍](https://developer.huawei.com/consumer/cn/doc/promotion/ads-jhdnggjjsmzyjs-0000002485600781)

<strong>Q5：如何通过Marketing API开启创意的智能扩展？</strong>

<strong>A：</strong>通过Marketing API创建全域智投广告，投放端创意界面的智能拓展开关默认关闭，如果需要开启请联系对应的运营同学配置。

<strong>Q6：Marketing API调用接口时出现"code": "50000", "message": "invalid account,advertiser not found"报错，是什么问题？</strong>

<strong>A：</strong>出现上述报错，说明登录授权的华为账号没有在鲸鸿动能平台开户，请核实对应的登录授权账号情况。

<strong>Q7：oCPC成本保障报表中的转化数是以广告请求还是转化回传为统计口径？</strong>

<strong>A：</strong>请求口径。

<strong>Q8：竞价和合约在转化跟踪API联调上有区别吗？</strong>

<strong>A：</strong>没有区别，如果建立了资产，竞价任务和合约任务对于资产在联调上是一样的。

<strong>Q9：事件资产对应的监测链接更换，需要重新联调事件吗？</strong>

<strong>A：</strong>修改监测链接后需要重新手动联调生效。

<strong>Q10：当广告落地页需要设置二次跳转时，跳转链接可以随意设置吗？</strong>

<strong>A：</strong>落地页不得使用虚假链接，如涉及二次跳转，需要在页面上有明确的跳转引导提示；使用引导按钮跳转时需注意，跳转热区应限定在按钮范围，不能点哪里都能跳转。另外二次跳转页面内容需同时满足当前审核规则。

<strong>Q11：复制账户中的行业资质可以进行删除吗？</strong>

<strong>A：</strong>复制账户的行业资质是同步主账户（来源账户）的，不支持删除。如为临时补充资质，请将补充资质放到广告资质里，不要放行业资质，广告资质不会与主账户（来源账户）同步。广告资质提交路径：投放后台工具-广告账户管理-资质信息-投放资质（广告资质）。

<strong>Q12：同一个账户里，资产和事件可以重复多建吗？</strong>

<strong>A：</strong>同一个账户里，一个应用只能配置一个资产，一个资产内相同的事件也只能创建一个。

<strong>Q13：鲸鸿动能平台支持投放直营电商（二类电商）微信小程序吗？</strong>

<strong>A：</strong>暂不支持二类电商微信小程序里面的商家开户投商品。
