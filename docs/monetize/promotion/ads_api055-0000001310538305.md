# 上传渠道包

【简介】通过此接口可以分片上传渠道包

<strong>请求地址</strong>

<strong>从获取渠道包上传凭证接口的响应参数</strong> <strong>chunk\_upload\_file</strong> <strong>中获取</strong>

<strong>请求方法</strong>

<strong>POST</strong>

<strong>请求参数</strong>

|  |  |  |  |
| --- | --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>是否必选</strong> | <strong>描述</strong> |
| advertiser\_id | long | 否 | 广告主ID，当登录授权的华为账号为如下场景时此字段必填：  1）授权账号关联的是经理账户；  2）授权账号关联的是服务商账户；  3）授权账号关联了多个子客账户。 |
| action | string | 是 | 调用接口方法，取值为下列中的一种  check/upload/merge |
| authCode | string | 是 | 上传文件验证码 |
| chunk | string | 否 | 第几分片，不传此字段默认为不分片 |
| chunkSize | integer | 否 | 分片大小 |
| fileName | string | 否 | 文件名 |
| fileCount | string | 否 | 上传文件数 |
| fileId | string | 是 | 文件的唯一标识，同一个文件的不同分片必须保持一致才能正确合并，建议使用文件sha256值 |
| pkgName | string | 是 | 应用包名 |
| file | File | 否 | 文件内容  需要是APK文件  当action=upload是必填 |

<strong>请求示例</strong>

[渠道包分片上传demo](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260529160229.95267885551252627260391430705375:20260531101345:2800:B2467C6C7FE56F91321542C381733DFE0BC7D4FF8F7E796311E7E0C2EE6F6D9E.zip?needInitFileName=true)

<strong>响应字段</strong>

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| code | string | 返回码 |
| message | string | 返回描述 |
| data | Struct1[] | 渠道包列表 |

Struct1定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| result | Struct1[] | 上传响应信息 |

Struct1（result）定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| UploadFileRsp | Struct2[] | 成功时返回信息，失败时不返回此字段 |
| resultCode | string | 响应码 |
| CException | Struct3[] | 失败时返回信息，成功时不返回此字段 |

Struct2（UploadFileRsp）定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| ifSuccess | integer | 是否上传成功  0 失败  1 成功 |
| fileInfoList | Struct4 | 响应成功数据 |

Struct3（CException）定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| errorCode | long | 错误码 |
| errorDesc | string | 错误描述 |

Struct4（fileInfoList）定义

|  |  |  |
| --- | --- | --- |
| <strong>参数名称</strong> | <strong>类型</strong> | <strong>描述</strong> |
| fileDestUlr | string | 文件目标存储地址 |
| size | string | 文件大小 |
