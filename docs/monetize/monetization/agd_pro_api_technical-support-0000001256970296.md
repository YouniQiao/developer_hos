当您使用AGD Pro API遇到问题时，可以按照以下顺序需求帮助。

1. 查看[FAQ](https://developer.huawei.com/consumer/cn/doc/monetize/agd_pro_api_faq-0000001217354962)和[错误码](https://developer.huawei.com/consumer/cn/doc/monetize/agd_pro_api_if_error-code-0000001263223153)，来解答您的常见咨询问题。
2. 通过[智能客服](https://developer.huawei.com/consumer/cn/customerService/#/bot-dev-top/faq-top/faq-talk-top)查找问题解决方案。
3. 通过[在线工单系统](https://developer.huawei.com/consumer/cn/support/feedback/#/add/13?level2=111)与我们进行联系，提交后的工单可在开发者联盟页面，点击右上角头像图标，选择“我的客服”，查看工单处理进展。
4. 如果以上方法仍未解决您的问题，发送邮件至邮箱developer@huawei.com，联系对口商务或客服。
