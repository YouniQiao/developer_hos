#### 与隐私保护有关的信息披露（隐私声明要求）

作为发布商，您必须做到：

1. 您需**以应用自身名义发布用户隐私政策声明，并事先获得用户同意方可向用户推送鲸鸿动能广告平台的广告内容**。您需要在您的隐私声明或用户协议中根据实际使用情况进行描述并告知用户这些数据将分享给鲸鸿动能广告平台。您可以在隐私声明中显示链接，方便用户查阅当您使用鲸鸿动能广告平台的服务时，[鲸鸿动能广告平台如何使用相关数据](https://privacy.consumer.huawei.com/legal/ads/privacy-statement.htm?&code=HK&language=zh-CN&branchid=0&contenttag=default)。同时您必须保留最终用户同意记录。
2. 您还需要**告知用户如何关闭个性化广告和基于位置的广告**（在您的应用拥有获取用户位置权限时，鲸鸿动能广告平台会提供基于位置的广告）**。**
   * 如用户使用的是HMS设备，可通过在设备上进入以下路径后选择“限制广告跟踪”进行操作：

     “设置”>“隐私”>“广告与隐私”，

     “设置”>“HMS Core”>“广告”，

     “设置”>“隐私”>“HMS Core”>“广告”，

     “设置”>“安全和隐私”>“更多安全设置”>“匿名设备标识”（具体操作路径取决于最终用户的操作系统版本）。

     对于智慧屏设备，可前往“设置”>“系统”>“HMS Core”>“广告”进行操作。

     如果用户不同意接收个性化广告，广告标识符将被替换成一个全零非唯一值，以停止个性化广告的推送。用户可前往“设置”>“定位服务”系统服务界面关闭基于位置的广告开关。
   * 如用户使用的是不具备上述 HMS 设备设置项的 GMS 设备，可通过在设备上前往“设置”>“Google”>“广告”>“选择停用广告个性化功能”，使用谷歌的广告设置项进行操作。

#### 数据的收集和使用

* **数据的采集：**

  鲸鸿动能广告SDK可能收集以下用户信息用于创建广告细分组别并向用户提供广告服务：

  + **设备信息：**如鲸鸿动能广告标识符、设备标识符、品牌、型号和类型、操作系统版本、运营商信息、位置、屏幕分辨率、语言设置、CPU 处理器、内存、存储信息和设备使用时长。如果用户的设备安装了GMS，鲸鸿动能广告平台还将采集用户的GAID。（用户可随时前往 “设置”>“Google”>“广告”关闭基于 GAID 的广告个性化服务或重置 GAID。）
  + **网络信息：**如 IP 地址、网络类型，以及 Wi-Fi、蓝牙和互联网接入信息。我们会使用 IP 地址获取您的大致位置信息（国家/地区/城市）。Wi-Fi 热点和蓝牙信息可用于获取您的大致位置（例如机场和车站等）。
  + **服务使用信息：**如华为和第三方应用中的广告互动记录（如广告点击量），您在过去30天内以及180天内使用过的应用的信息（名称和分类），以及您在“华为视频”中观看过的视频。用户个人数据的存储位置及保存时间请参见[附录1](#section13365714155410)。
* **数据的使用：**

  基于我们的合法利益，我们将用户的个人数据用于以下用途：

  + **了解用户偏好，推送更符合用户兴趣的个性化广告。**有助于广告商在适合的时间向用户投放合适的广告。详细信息请查看关于鲸鸿动能广告与隐私的声明。
  + **统计广告点击和交互数据**，例如用户在华为和第三方应用中打开、下载或播放不同的广告内容，以及“华为应用市场”应用的安装和激活情况。我们将使用这些数据来分析我们与广告商之间的业务合作情况，并统计和衡量其广告效果。由于我们需要基于广告的投放和效果与广告商进行结算，我们会与广告商分享统计数据。
  + **统计分析及产品改进**：这将帮助我们更好地了解用户的需求，并改进我们当前及未来所提供产品和服务的质量及用户体验。
  + **检测欺诈行为**（例如攻击者使用自动化软件有组织地点击广告），避免广告商支付不合理费用。例如，我们可以分析用户与触摸屏的交互情况（例如，如何触摸屏幕）以及设备的位置移动情况（基于设备传感器数据），以检测和防止欺诈性广告点击。
  + **确保数据安全**，以便为您提供安全可靠的服务。

    如果用户同意我们推送基于位置的广告，我们可能会使用用户的地理位置信息为您提供适合用户所在区域的广告。我们不会在广告平台上存储用户设备的精确位置信息。

#### 个性化广告与非个性化广告

鲸鸿动能广告平台投放的广告包括个性化广告和非个性化广告。

* **个性化广告：**也称为基于用户兴趣投放的广告或行为广告。是指基于一段时间内、跨不同应用或服务的用户行为而显示的广告，包括但不限于受众特征定位、兴趣类别定向、再营销、目标客户匹配列表定位的广告。
* **非个性化广告：**是指不基于用户过去行为投放的广告，非个性化广告可能会基于上下文或背景信息来进行定位，例如基于设备上下文、内容上下文或大致的地理位置信息定位的广告。非个性化广告会禁止任何的受众特征定向、兴趣类别定向、再营销和目标客户匹配定向。

鲸鸿动能广告平台针对个性化广告和非个性化广告为发布商提供了多种选项。发布商可以

1. **选择投放个性化广告与非个性化广告，鲸鸿动能广告平台将根据广告主投放情况返回个性化广告或非个性化广告。**

   如果您选择投放个性化广告与非个性化广告，您需要根据各个国家/地区适用的法律要求，征求用户意见，详情见第4节[个性化广告与用户同意征求](#section1165116235534)。在需要获取用户同意的国家/地区 ，如未获取用户同意，则您不能请求个性化广告。您需要自行承担因请求错误而导致的错误投放所带来的一切后果。
2. **选择向用户投放非个性化广告。鲸鸿动能广告平台将仅返回非个性化广告。**

   如果发布商将广告请求标记为仅请求非个性化广告，则针对该广告请求只返回没有个性化定向的广告。性别定向、年龄定向、APP行为定向、APP兴趣定向、自定义人群定向都将不适用于您的广告请求。这意味设置此类定向的广告库存将不适用于您的广告请求，您广告请求的返回率和填充率将受到影响。

   发布商可以通过鲸鸿动能广告SDK中的setNonPersonalizedAd方法根据用户consent 状态向鲸鸿动能广告平台请求“个性化广告与非个性化广告”或仅“非个性化广告”。

   如果发布商集成了TCF2.0，还可以通过TC STRING告知鲸鸿动能广告平台的用户对各个使用用户个人数据的目的的同意情况，鲸鸿动能广告平台将根据各目的的同意结果决定投放非个性化广告还是个性化广告。详情请参见[IABTCFV2.0](#ZH-CN_TOPIC_0000001146675575__li1390113106184)。

#### 开发者：个性化广告与用户同意征求

当发布商通过鲸鸿动能广告SDK接入个性化广告服务时，为了投放个性化广告，需要收集和使用用户的个人数据并与鲸鸿动能广告平台共享。作为发布商您必须：

* 按照各个国家/地区适用的法律法规要求，就收集和使用并与可能的广告技术提供商共享用户个人数据进行意见征求。

  如果您的业务区域在以下范围内，您必须征得用户的同意才能使用其个人数据（例如OAID、GAID）来投放个性化广告或将个人数据分享给三方合作伙伴：
  + Aspiegel SE服务的所有业务区域。业务区域对应国家清单，详见《[华为开发者服务协议](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)》第14条“分发区域和签约主体”。
  + 华为服务（香港）有限公司服务的部分区域：阿拉伯联合酋长国，中国台湾，南非，阿根廷，巴西。

* 保存用户意见征求的记录，并告知鲸鸿动能广告平台。如因发布商没有及时告知鲸鸿动能广告平台而产生任何后果由发布商自行承担。
* 向最终用户提供撤销同意的明确说明。

如果最终用户同意接收个性化广告，则您可以请求“个性化广告与非个性化广告”，鲸鸿动能广告平台针对您的广告请求可以返回个性化广告和非个性化广告（根据广告商投放情况）；如果最终用户未同意接收个性化广告，则您只能请求非个性化广告，鲸鸿动能广告平台针对此广告请求只能返回非个性化广告；如果您的广告请求中没有设置用户同意的结果，默认情况下，鲸鸿动能广告平台将返回个性化广告，并根据以前收集的用户数据筛选广告。

鲸鸿动能广告平台有权对您的隐私和数据合规性进行监督，如鲸鸿动能广告平台发现存在不合规的情形，则有权要求您完善相应的合规机制。如未完善，我们有权限制、暂停或终止对您的服务。您关于用户数据隐私的全部义务和责任不因华为的风险评估权利而构成任何形式的豁免。

**我们提供如下两种方式协助您获取用户是否接受个性化广告的意见，您可以使用其中的任意一种方式。**

1. **鲸鸿动能广告平台 User Consent SDK**

   鲸鸿动能广告平台 User Consent SDK是一套适用于移动应用的用户意见征集工具。通过使用鲸鸿动能广告平台 User Consent SDK，您可以向您的用户展示您选择接入的广告技术提供商的信息（您可以在流量服务变现平台控制台的欧盟地区意见征求设置您接入的广告技术提供商），以便就收集、共享和使用个人数据来实现广告个性化征求用户的同意。

   这些可选择的广告技术提供商已经向鲸鸿动能广告平台提供相关信息，表明其遵守GDPR规定，并且遵循我们的数据使用政策，以保护您用户的数据。

   **具体的设置步骤如下：**

   1. 选择广告技术提供商

      请登录流量变现服务平台，并根据[欧盟地区意见征求](https://developer.huawei.com/consumer/cn/doc/distribution/monetize/usercontent-0000001061871027)设置指南选择您愿意接入的广告技术提供商。如果您选择了这些广告技术提供商，则代表它们可能会出于投放个性化广告和衡量广告效果的目的而使用您用户的个人数据。
   2. 使用Consent SDK设置意见收集

      请参阅[Consent SDK](https://developer.huawei.com/consumer/cn/doc/development/HMSCore-Guides-V5/publisher-service-consent-settings-0000001075342977-V5)开发指南，使用鲸鸿动能广告平台的意见征求对话框或自行创建意见征求对话框收集来自用户的个性化广告意见。**请根据本地适用的法律法规，对必须征求用户同意才能提供个性化广告服务的地区用户进行意见征求。**
2. **IAB TCF V2.0 （IAB透明度和用户意见征求框架2.0版本）**

   IAB Europe与IAB Tech Lab合作，于2019年8月21日宣布[透明度和同意框架2.0版本（TCF v2.0）](https://iabeurope.eu/tcf-2-0/)上线，该框架旨在帮助数字广告链中的所有各方在处理个人数据或访问用户的个人信息或在个人设备上访问和存储信息时，确保他们遵守欧盟的《通用数据保护条例》和《隐私权指令》。鲸鸿动能广告平台已于2020年9月18日加入TCF v2.0的全球供应商列表（GVL），供应商ID为856。

   已经集成和将要集成TCF2.0的发布商可以参考开发指南中的[IAB TCF V2.0用户同意信息传递](https://developer.huawei.com/consumer/cn/doc/development/HMSCore-Guides-V5/publisher-service-iab-tcf-0000001075694328-V5)了解鲸鸿动能广告平台将如何根据TCF v2.0的透明度和同意字符串（TC String）的内容对广告请求中包含的数据进行操作。

   如果您未集成TCF2.0，依然可以通过Consent SDK向鲸鸿动能广告平台传递用户同意信息。

* 关于阿斯比格的服务业务所覆盖的国家/地区列表，请参考[《](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)[华为开发者服务协议](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)[》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)中第14条“分发区域和签约主体”。

#### 面向儿童的广告

为满足《一般数据保护条例》GDPR及其他各国相关法律法规规定，如果您的应用或网站中，包含儿童相关的内容，您必须将用户是否是儿童的信息如实告诉鲸鸿动能广告平台。

**如果您告知鲸鸿动能广告平台当前广告请求的用户是儿童，则鲸鸿动能广告平台对该用户的请求：**

1. 将不会收集任何个人数据。
2. 不会向儿童投放个性化广告，但仍会向儿童提供非个性化广告。
3. 不向三方DSP请求广告。

您需要承担由于告知错误或没有告知导致广告误投放到儿童的手机上导致的一切后果。

**如果您需要通过鲸鸿动能广告平台面向儿童投放非个性化广告，您需要完成如下两个步骤：**

1. 在广告请求中标记您希望鲸鸿动能广告平台按照儿童用户的方式处理该广告请求。详细实现方式请参见[setTagForUnderAgeOfPromise](https://developer.huawei.com/consumer/cn/doc/development/HMSCore-Guides/publisher-service-audience-settings-0000001076462332)。
2. 对于标记为儿童的广告请求，您还需要设置对应的广告内容分级，以向儿童用户展示适合的广告内容。

   您可以参考以下链接设置内容分级：

   1）华为账号级设置：[为您的整个华为账号设置广告内容分级](https://developer.huawei.com/consumer/cn/doc/monetize/accountshielding-0000001062689335)。

   2）应用级设置：[为具体应用设置广告内容分级](https://developer.huawei.com/consumer/cn/doc/monetize/mediashielding-0000001062287179)。

   3）广告请求级设置：[为每个广告请求设置广告内容分级](https://developer.huawei.com/consumer/cn/doc/development/HMSCore-Guides/publisher-service-audience-settings-0000001076462332)。

   您可以同时设置上述任意1种或多种内容，优先使用广告请求级设置，没有广告请求级设置则使用应用级设置，没有请求级和应用级设置，则使用华为账号级设置。

   鲸鸿动能广告平台支持如下4种内容分级：

   |  |  |
   | --- | --- |
   | 级别 | 分级描述 |
   | W | Widespread:适合广泛受众的内容。 |
   | PI | Parental instruction：适合家长陪同观看的内容。 |
   | J | Junior：适合青少年及以上观看的内容 |
   | A | Adult：适合成人及以上观看的内容。 |

   鲸鸿动能广告平台将根据您的设置，返回您设置的级别及以上级别的广告内容，设置的内容分级与对应返回的广告内容级别关系如下：

   |  |  |
   | --- | --- |
   | 您设置的面向儿童广告的内容分级 | 可能返回的广告的内容级别 |
   | W | W |
   | PI | W、PI |
   | J | W、PI、J |
   | A或未设置 | W |

   标记为儿童的广告请求的广告内容分级只能设置为W、PI、J。

   如果您未设置广告内容分级或者将面向儿童的广告的内容分级设置为A，鲸鸿动能广告平台将强制返回W级别的广告内容。
