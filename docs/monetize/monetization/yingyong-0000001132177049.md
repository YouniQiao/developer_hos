#### Android应用

1. 鲸鸿动能SDK

[鲸鸿动能SDK接入指南](https://developer.huawei.com/consumer/cn/doc/development/HMSCore-Guides/publisher-service-integrating-sdk-0000001050066913)

提示：建议您优先使用上方的指南进行升级，但如果您的SDK为2021年5月前通过媒体服务平台"SDK下载"菜单获取接入的（包名：com.huawei.openalliance.ad接口包名），仍可使用如下指南平滑升级，详情请[点击链接跳转](https://developer.huawei.com/consumer/cn/doc/development/HMSCore-Guides/pps-commissioning-guide-0000001111513478?showHide=1)查看。

2. 鲸鸿动能JavaScript API

若您需要在App的Web页面呈现广告，请先集成鲸鸿动能SDK后，再集成[鲸鸿动能JavaScript API](https://developer.huawei.com/consumer/cn/doc/development/HMSCore-Guides/publisher-service-js-integrating-sdk-0000001133475862)。
