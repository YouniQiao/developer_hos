在正式以SDK方式接入AGD Pro服务前，您需要对集成后业务进行自测，自测完成后再提交给华为验收。

![](./img/agd_pro_sdk_andriod_test-online-0000001382413621_0.png)

如果请求不到广告，不能以报错的形式显示给用户。

1. 请对照《[AGD PRO SDK验收规范(Android)](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250314174012.02722492032530719311262337452275%3A50001231000000%3A2800%3A00F6BC4A0CFBFAF73145BE2F89C9DD7A9CFAFBFE2CF75E195692670E2632273F.xlsx)》交付件中的“应用转测自检项”页签核对检查。
2. 自测完成后，按《AGD PRO SDK验收规范》交付件要求填写check项要求的满足情况。
3. 按照如下邮件模板将填写好的《AGD PRO SDK验收规范》交付件发送邮件到**developer@huawei.com**邮箱申请开通上线。

   **邮件模板如下：**

   **邮件标题**：*APP名称*-AGD Pro验收申请

   **邮件内容：**

   APP名称：*\*\*\*\*\*\*\**

   APP ID：*\*\*\*\*\*\*\*\*\**

   申请验收：AGD Pro SDK
