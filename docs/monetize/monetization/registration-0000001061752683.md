#### 概述

开发者需要注册华为账号，完成实名认证，开通商户服务，才可以接入流量变现服务。

#### 操作步骤

详细的注册及认证流程，请参照：

1. **华为账号注册**

   详情可参考：[账号注册](https://developer.huawei.com/consumer/cn/doc/start/registration-and-verification-0000001053628148)。
2. **实名认证**

   注册地为中国大陆的企业开发者实名认证可以通过两种方式：[邓白氏认证](https://developer.huawei.com/consumer/en/doc/start/atpopb-0000001062836624)或者[企业对公打款认证](https://developer.huawei.com/consumer/cn/doc/start/atpopb-0000001062836624)。

   注册地为非中国大陆的企业开发者实名认证可以通过两种方式：[邓白氏认证](https://developer.huawei.com/consumer/en/doc/start/atpopb-0000001062836624)或者[营业执照](https://developer.huawei.com/consumer/en/doc/start/mracoei-0000001062678404)。

   个人开发者实名认证请参考：[个人开发者实名认证](https://developer.huawei.com/consumer/cn/doc/start/ibca-0000001062388135)。
3. **商户服务**

   开通商户服务是实现流量变现服务的前提。

   详情可参考：[商户服务](https://developer.huawei.com/consumer/cn/doc/start/merchant-service-0000001053025967)。

   ![](./img/registration-0000001061752683_0.png)

   * 中国大陆开发者在中国大陆区域以外进行流量变现业务，必须在商户服务中添加并维护跨境收款账户（即可接收外币的中国大陆银行收款账号）。
   * 目前跨境收款账户支持的银行：中国中信银行、中国银行、中国农业银行、中国民生银行、中国建设银行、中国工商银行、上海浦东发展银行、招商银行、交通银行、北京银行。
   * 支持维护的币种：EUR、HKD、JPY、CHF、GBP、USD。
4. **协议签署**
   * **官网入口**

     使用联盟实名认证的账号进入[流量变现服务平台](https://developer.huawei.com/consumer/cn/monetize/)，单击**开始变现**进入后台。
   * **变现服务卡片入口**

     登录[开发者联盟](https://developer.huawei.com/consumer/cn/monetize/)，在**管理中心**中，单击**应用服务**中的**流量变现**卡片。若进入后无卡片展示，请单击右上方**自定义**添加**流量变现**卡片。

     首次进入变现后台时，页面会弹出《鲸鸿动能媒体服务协议》，请您仔细阅读。如无异议，请单击“我已阅读并同意《鲸鸿动能媒体服务协议》”，协议立即生效。已签署服务协议，可在“**签约信息**”中查阅签约记录，并可下载合同文本。

     如果后期系统协议刷新，登录后系统会提示您需要重新签署协议，如果您开通了华为联盟提供的[团队账号](https://developer.huawei.com/consumer/cn/doc/start/team-account-guides-0000001053785552)功能，需要使用主账号登录并重新签署协议。

     ![](./img/registration-0000001061752683_1.png)

     + 在使用流量变现服务之前，请确保您浏览器的广告屏蔽功能已关闭，否则可能导致该服务无法正常访问。
     + 账号注册请选择正确的地址，详情可参考：[受限说明](/docs/monetize/monetization/restriction-0000001061730052)。
     + 华为开发者联盟提供了团队账号功能，供开发者管理成员账号。详情可参考：[团队账号](https://developer.huawei.com/consumer/cn/doc/start/team-account-guides-0000001053785552)。
