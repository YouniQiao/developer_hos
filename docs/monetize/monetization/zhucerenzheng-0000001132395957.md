1. 账号注册

   详情参考[注册账号](https://developer.huawei.com/consumer/cn/doc/start/registration-and-verification-0000001053628148)。
2. 实名认证

   详情参考[实名认证](https://developer.huawei.com/consumer/cn/doc/start/itrna-0000001076878172)。
3. 开通商户服务

   开通商户服务是开发者分发付费内容（付费应用/主题、应用内付费）的前提。商户服务里配置的银行卡账户、币种，用于开发者接受华为分成收益。详情参考[商户服务](https://developer.huawei.com/consumer/cn/doc/start/merchant-service-0000001053025967)。
4. 签署《鲸鸿动能媒体服务协议》

* 鲸鸿动能媒体服务平台
  + 开发者官网-变现-点击[【开始变现】](https://developer.huawei.com/consumer/cn/monetize/)进入。
  + 开发者官网右上角[【管理中心】](https://developer.huawei.com/consumer/cn/console#/serviceCards/)- 应用服务 - 流量变现。

* 进入媒体服务平台后，页面会弹出《鲸鸿动能媒体服务协议》，请您仔细阅读。如无异议，请点击“我已阅读并同意《鲸鸿动能媒体服务协议》”，协议立即生效。
* 如您已签署服务协议，可在“签约信息”中查阅签约记录，并可下载合同文本。
