1. 发布商限制

   实名认证成为华为开发者联盟的企业开发者，包括获得应用开发者合法授权的企业。
2. 接入媒体限制
   * 接入鲸鸿动能平台的应用/快应用/快游戏/元服务需在华为应用市场上架。
   * 与华为联运的游戏应用和快游戏禁止接入鲸鸿动能以外的其他广告内容/插件/SDK等。
3. 支持的设备限制

   |  |  |  |  |
   | --- | --- | --- | --- |
   | **操作系统** | **终端设备** | **设备类型** | **HMS版本** |
   | HarmonyOS | 华为终端设备 | 手机、Pad | 4.0.0及以上 |
   | HarmonyOS NEXT | 华为终端设备 | 手机、Pad | 不依赖 |
