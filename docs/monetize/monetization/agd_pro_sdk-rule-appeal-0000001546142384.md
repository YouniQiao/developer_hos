当您收到我们发送的违规处罚通知后，可以在5个工作日之内，按照平台的规则和要求将申诉材料发送至申诉邮箱**developer@huawei.com**，我们会在收到申诉邮件后尽快对申诉材料进行复核并给出回复。

申诉邮件标题格式：**AGD Pro应用变现服务申诉-***产品名称*

为了帮助我们更好地核实违规异常点，请提供尽可能详细的信息，包括但不限于以下内容：

1. 异常帐号：请填写异常帐号ID及名称，异常产品ID及名称，异常展示位ID及名称。
2. 交互形式：广告展示逻辑说明（什么情况下会展示广告），提供展示位置录屏或截图。
3. 异常展示位抓包数据及统计数据截图（例如：请求/曝光/点击的数据截图），如数据指标明显异常，请重点解释。
4. 广告请求、曝光、点击逻辑、缓存机制，以及参数回传是否正确。
5. 切量逻辑：流量请求是否存在定向（例如：人群、地域、厂商渠道等）。
6. 流量是否存在渠道推广：买量渠道、买量时间段、买量定向，以及相关数据。

申诉原因说明：若自查没有违规的行为，请举证说明原因；若出现可能违反相关规定的行为，说明违反原因及整改措施，整改后确保相关展示位已不存在异常，并提供相关数据或截图。

![](./img/agd_pro_sdk-rule-appeal-0000001546142384_0.png)

如果申诉被驳回或收到处罚通知5个工作日内未申诉，我们将不再受理您的申诉。
