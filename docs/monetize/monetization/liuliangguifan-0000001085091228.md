无效流量指通过作弊手段，导致广告客户费用或发布商收入虚增的点击或展示。而导致无效流量产生的活动称为无效活动。平台禁止任何不正当方式制造虚假或无效流量、点击量、下载量，或违规嵌入等行为，包括但不限于以下形式：

1. 违规推广流量

   禁止通过自行、强制、诱导、或程序访问，自动刷新等相关手段进行无效，或隐藏的推广行为，来赚取推广流量。

2. 修改推广内容
   1. 将用户点击的推广页面重定向到别处，与最终用户直接访问推广页所看到的页面不同；
   2. 对由推广内容任一部分、副本或衍生信息，采用“抓取”、“spider”、索引或任何其他方式加以存储或缓存；
   3. 推广内容显示后，改变推广内容属性，比如：改变推广位置、推广内容大小等， 推广元素的任何属性。

3. 诱导、强制用户点击
   1. 在非激励型广告合作形式中，对用户点击行为进行资源交换，奖励；
   2. 强制用户点击推广换取用户使用应用的次数或者继续应用的使用；
   3. 遮挡、隐藏推广显示区域的任何部分，包括logo、链接、描述等，造成推广内容与页面内容发生重叠；
   4. 禁止通过其他违规手段伪造点击量获取收益。

4. 反编译SDK

   禁止任何形式对鲸鸿动能SDK、华为移动基础服务SDK进行反编译，不得对SDK接口进行内容、功能、逻辑及界面的任何修改。
