优质媒体额外分成激励指针对接入流量变现服务平台的优质媒体实施的月度激励政策，以此增加优质媒体流量收益，促进媒体在流量变现服务平台的流量生态建设。

#### 评估条件

1. 通过变现接入的三方媒体，包括但不限于SDK、API、Prebid等方式接入。
2. 接入变现周期≥30天。
3. 月度流量质量评分≥75分。
4. 展示率≥30%。
5. 近期无违规行为。

#### 激励方式

对于满足上述条件的三方媒体，根据过去一个自然月的端内广告请求量进行评估，对于超出日均广告请求量1M的部分，给予该三方媒体原分成基础上额外10%的分成收入，该笔收入将随月度结算收入一起打款至相应开发者账号。

|  |  |  |
| --- | --- | --- |
| 三方媒体分成比例 | 日均广告请求＜1M | 日均广告请求≥1M |
| 30%：70% | 20%：80% |
| 根据过去一个自然月的日均广告请求量计算评估 | | |

#### 激励周期

2022年9月—2026年12月31号

#### 激励通知

鲸鸿动能广告平台每月将通过邮件及站内信的方式知会获得额外分成激励的开发者，敬请关注。
