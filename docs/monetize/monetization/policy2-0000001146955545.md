#### 禁止的内容

我们禁止在以下内容中展示广告，如果您的应用中含有以下内容，将不允许集成鲸鸿动能广告SDK并获取广告。一经发现您的APP中含有以下内容，我们将有权暂停或永久停止您的鲸鸿动能广告平台的账号。

* **非法内容**
  + 任何违反法律法规的内容，包括但不限于含有法律、法规禁止内容，宣传非法活动或侵犯他人权利的内容。
* **虚假内容**
  + 歪曲、谎报、隐藏或夸大自己的身份、内容和用途。
  + 通过虚假宣传、不实或欺骗性的信息宣传内容、产品和服务。
  + 通过诱导的词语或页面布局诱使用户互动。
* **涉恐涉暴内容**
  + 宣扬种族、民族、宗教仇恨，涉及性别、年龄、身份、国籍、残障等其他制度性歧视或边缘化相关的其他特征，煽动暴乱，煽动对个人或群体仇恨，宣扬对个人或群体歧视，宣扬对个人或群体进行污蔑的内容。
  + 骚扰、恐吓或欺凌个人或群体。
  + 威胁或宣扬对自己或他人施加伤害的内容
* **恶意应用或垃圾应用**
  + 危害用户机器安全，严重威胁用户的隐私和账号财产安全的恶意应用。
  + 未明确告知用户应用提供的功能或安装后影响；未提供“服务协议”或“最终用户许可协议”；在用户不知情的情况捆绑安装应用；未经用户允许更改系统；难以停止或卸载等其他骚扰用户的垃圾应用。
* **色情内容**
  + 含有大量暴露、色情、非自愿性爱主题的内容。
  + 宣传色情服务、情色交友服务的内容。
* **儿童性虐待和恋童癖**
  + 宣传对未成年人进行性虐待。
  + 引诱未成年人进行性行为。
  + 宣扬成年人以未成年人为性对象的内容或行为。
* **濒危或受威胁物种**
  + 推销、销售获取自濒危物种或受威胁物种的相关产品。
* **侵犯知识产权**
  + 侵犯版权的内容。
  + 推销或销售仿冒产品的内容。

#### 限制的内容

当应用中存在受限制的内容，我们将限制应用接收某些来源的广告。您可以选择利用受限制内容进行创收，但此类内容能投放的广告量将比其他不受限制的内容少，甚至在某些情况下，有可能不会有广告来源对您的流量进行出价，也不会有广告展示。

* **在线赌博**
  + 包含大量推广赌博的内容。
  + 提供线上赌博服务。

* **惊悚内容**
  + 包含血腥暴力、骇人或令人厌恶的画面或描述。
  + 涉及暴力行为。
  + 包含大量淫秽、亵渎性的内容。

* **烟草**
  + 推销烟草和烟草相关的产品。

* **酒精饮料销售或滥用**
  + 推销或销售酒精饮料。
  + 宣传不负责任的饮酒行为。

* **未获批准的药物和补充剂**
  + 推销或销售未经批准的药物和补充剂。
