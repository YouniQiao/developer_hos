为满足App不同场景下的内容体验需要，流量变现服务已提供Banner、原生、激励视频、插屏、开屏、贴片六种广告形式，未来还会持续推出更丰富、更有创意的广告形式。

* **Banner广告**

  ![](./img/formart-0000001051321897_0.png "点击放大")

  Banner广告即横幅广告，是以通栏式或矩形式出现在应用页面顶部、中部或底部，适合在用户停留较久或者访问频繁的页面。
* **原生****广告**

  ![](./img/formart-0000001051321897_1.png "点击放大")

  原生广告通常是指以“原生”的形式植入到应用本身的信息或服务内容中的广告，广告内容的植入和呈现不破坏页面本身的和谐，用户干扰度低，更愿意主动阅读。原生广告的应用场景较多，新闻信息流最为常见，支持视频和图片素材。视频素材时长限制为2-60s。
* **激励视频**

  ![](./img/formart-0000001051321897_2.png "点击放大")

  激励视频广告，即用户通过观看完整的视频广告，进而获得应用内相关的奖励。多用于游戏应用的通关、复活、道具获取、积分等场景中，阅读、影音等应用也有相关应用，视频时长限制为15-30s。
* **插屏广告**

  ![](./img/formart-0000001051321897_3.png "点击放大")

  插屏广告是一种在应用开启、暂停或退出时以半屏或全屏的形式弹出的广告形式，展示时机巧妙避开用户对应用的正常体验，尺寸大，曝光效果好；支持视频和图片素材，视频素材时长限制为2-60s。
* **开屏广告**

  **![](./img/formart-0000001051321897_4.png "点击放大")**

  + 开屏广告出现在APP启动加载时，一般会全屏展现3-5秒，展示完毕后自动关闭并进入应用主页面。通常展现形式为：静态图片、Gif图或视频。
  + 极速开屏广告在提升用户开屏体验的同时为开发者提供全新的广告变现方案，开发者无需集成SDK即可全版本接入极速开屏服务。
* **视频贴片**

  ![](./img/formart-0000001051321897_5.png)

  视频贴片广告可以在视频内容播放前、播放中或播放后展示，展现形式为视频，视频时长限制为15-30s。
