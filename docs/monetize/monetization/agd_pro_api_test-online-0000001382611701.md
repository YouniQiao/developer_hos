在正式接入AGD Pro API前，您需要对集成后业务进行自测，自测完成后再提交给华为验收。

![](./img/agd_pro_api_test-online-0000001382611701_0.png)

如果请求不到广告，不能以报错的形式显示给用户。

1. 请对照《[AGD PRO API验收规范](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250314174005.95284576672291560840369525805785%3A50001231000000%3A2800%3A3B81E72FFD3ED21337EA9254BA04AE77407D451DF57A72E9A821E9925470C2E7.xlsx)》交付件中的“应用转测自检项”页签核对检查。
2. 自测完成后，按《AGD PRO API验收规范》交付件要求填写check项要求的满足情况。
3. 按照如下邮件标题将填写好的《AGD PRO API验收规范》交付件发送邮件到**developer@huawei.com**邮箱申请开通上线。

   **邮件模板如下：**

   **邮件标题**：*APP名称*-AGD Pro验收申请

   **邮件内容：**

   APP名称：*\*\*\*\*\*\*\**

   APP ID：*\*\*\*\*\*\*\*\*\**

   申请验收：AGD Pro API
