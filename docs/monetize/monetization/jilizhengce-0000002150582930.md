#### 鲸鸿动能鸿蒙5及以上变现激励政策

HarmonyOS 5及以上版本自2024年10月22日正式开启公测以来，用户升级进程不断加速，鸿蒙应用及元服务的上架数量亦呈现迅猛增长态势。作为开发者重要的商业变现平台，鲸鸿动能同步开启公测以来，致力于为开发者开辟新的增收阵地。

为更好的助力开发者加入鸿蒙生态，生态红利惠及更多开发者，平台推出的HarmonyOS 5及以上版本变现激励重磅延续，为开发者提供高额现金激励、广告投放金激励等，具体如下：

**1. 激励周期：**2026年1月1日-2026年12月31日。

**2. 激励对象：**满足以下全部条件 1）已接入鲸鸿动能平台变现的开发者 2）HarmonyOS 5及以上版本设备的流量

**3. 激励政策：**

| **媒体类型** | **变现收益规模** | **激励总比例** | **激励构成配比** |
| --- | --- | --- | --- |
| 元服务 | 月收益≤3万元 | 100% | 100%现金激励 |
| 月收益＞3万元 | 100% | 30%现金激励+70%广告投放金 |
| 鸿蒙应用 | 月收益≤3万元 | 25% | 25%现金激励 |
| 月收益＞3万元 | 25% | 7.5%现金激励+17.5%广告投放金 |
| 鸿蒙游戏（IAA） | 仅通过鲸鸿动能广告买量产生的变现收益部分分成比例调整为华为:开发者=1:9，详情参照[《2026年鲸鸿动能游戏行业激励政策》](https://developer.huawei.com/consumer/cn/doc/promotion/ads-yxhyydjl-0000002160474788) | | |

**4. 激励发放：**

| **激励形式** | **发放及结算** |
| --- | --- |
| 现金分成 | 实时合并于基础变现收入中，并最终与月度收益合并结算 |
| 广告投放金 | 开发者在鲸鸿动能广告投放平台开户后，广告投放金充值到账以供投放使用 |

**5. 举例：**

A元服务开发者，激励前鲸鸿动能月变现收益2万元（2万＜3万），可额外获得2万元\*100%现金激励，合计4万元现金收益。

B鸿蒙应用开发者，激励前鲸鸿动能月变现收益40万元（40万＞3万），可额外获得40万元\*7.5%现金激励+40万元\*17.5%广告投放金激励，合计43万元现金收益+7万元广告投放金。

**6.** **其它说明：**更多详情可咨询鲸鸿动能商务/运营。
