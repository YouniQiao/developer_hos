快应用接入AGD Pro服务的SDK方式大致如下：

1. 将SDK集成到快应用引擎中。
2. 快应用通过接口调用快应用引擎中SDK能力。
3. 快应用引擎中推广SDK对接AGC进行推广请求与接收。
