**Q1：****测试ID是什么？**

A：测试ID是开发者在集成开发阶段，用于验证广告请求和展示是否正常。

**Q2：测试时使用正式ID还是测试ID？**

A：自测阶段请用测试ID，待验收通过替换为正式ID再提交应用市场上架。

**Q4：测试时拉取手机应用广告失败，可以如何处理？**

A：1）使用HarmonyOS NEXT手机进行测试；

2）自测阶段使用测试ID，获取到的是测试素材，非正式商业广告；

3）检查应用的“跨应用关联访问权限”是否为“允许”；

4）手机时间需同步当前时间，并将地域设置为中国；

若上述问题均自检后，仍然拉取广告失败，请联系客服处理。

**Q5：测试ID和媒体服务平台申请的正式ID有什么区别吗？**

A：测试广告位不区分应用和条件，都可以请求到广告；后台申请的广告位ID是正式ID，上线后才下发广告任务。

**Q6：手机应用广告的错误码表去哪里查看？**

A：Ads Kit接入请[点击此处](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ads-arkts-errcode)查看错误码列表。

**Q7：广告测试完成后需要提供什么信息给鲸鸿动能验收？**

A：将以下信息提交至鲸鸿动能客服验收：1）测试包；2）《HarmonyOS NEXT媒体接入自检表》交付件。

**Q8：《HarmonyOS NEXT媒体接入自检表》交付件是什么？必须提供吗？**

A：涉及广告位信息和测试自检项，验收以交附件信息为主，请按规范清晰描述广告位调用逻辑，转测自检项全部自检通过，以便加快上线进度。

**Q9：自测日志是什么？**

A：在windows CMD命令行中执行hdc shell hilog -b D命令打开debug日志：1）# 路径可以改为想要保存日志内容的文件路径；2）开始测试业务前执行命令hdc hilog > D:\hilog.log；3）测试结束后使用 Ctrl + C 终止脚本以停止抓取。

**Q10：提交测试包后需要多久可以上线？**

A：提交的测试包按顺序测试，上线进度取决于集成开发和交付件的质量，所以请务必按相关文档要求严格执行，避免返工。

**Q11：测试包通过验收后需要做什么？**

A：1）请将测试ID替换为正式ID，在应用市场提交审核；

2） 应用市场通过后，联系鲸鸿动能工作人员配置上线；

3）上线后，检查正式广告请求和展示是否正常。

**Q12：什么情况需要提交测试包给鲸鸿动能验收？**

A：1）首次接入应用，提交验收；

2）对于已接入的应用，如需接入新的广告样式，需要提交验收；

3）广告上线后，如需对与广告相关的内容做出修改，务必告知并提交测试包，待验收通过后再上架；如因修改后未提交验收，导致上架后出现数据问题，由此产生的损失由发布商承担。

**Q13：应用市场审核未通过怎么处理？**

A：联系华为应用市场客服询问处理，详情[点击此处](https://developer.huawei.com/consumer/cn/doc/distribution/app/agc-interaction-center)。

**Q14：上线后无广告返回怎么办？**

A：1）确保已联系客服配置广告；

2）确保您使用的手机是HarmonyOS NEXT手机；

3）若手机无广告展示，可能和投放定向等有关，可登录[鲸鸿动能媒体服务平台](https://developer.huawei.com/consumer/cn/service/ads/publisher/html/#/mainContent/dashbordContent)查看展示数据，如有展示数据则正常。

4）如以上问题均自检，长时间无返回数据，及时联系客服处理。
