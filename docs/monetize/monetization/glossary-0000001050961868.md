|  |  |
| --- | --- |
| **词汇** | **解释** |
| 测试广告 | 利用测试广告，在发布应用之前测试植入代码。在开发应用时，测试广告单元会向应用返回广告，以便了解植入代码是否在正常运行。 |
| 点击次数 | 用户点击应用上的广告的次数。 |
| 点击率 | 用户点击应用中展示的广告的次数除以广告在应用中展示的次数所得的值。 |
| 每次点击收益 | 客户每次点击广告都会向发布商支付相应的收益费用。 |
| 预计千次展示收益 | 广告在网络上展示了千次的收益费用。 |
| 展示率 | 在所有返回的广告中，实际在应用中展示给用户的广告所占的百分比。计算方法是用展示次数除以匹配请求数*。* |
| 无效活动 | 无效流量产生的活动称之为无效活动，无效流量指通过作弊手段，导致广告客户费用或发布商收入虚增的点击或展示。 |
