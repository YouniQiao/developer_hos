# 应用测试

应用测试覆盖三个核心维度，帮助你构建高质量应用：

- **[单元测试](./unit-test/guidelines)** — 使用自动化测试框架验证代码逻辑的正确性
- **[UI测试](./ui-test/arkts)** — 包含基于 ArkTS 和 Python 的 UI 自动化测试方案
- **[专项测试](./specialized-test/_index)** — 性能、稳定性、安全、UX、功能、功耗等专项质量检测

### 测试工具

- [SmartPerf](./tools/smartperf-guidelines) — 性能分析工具
- [Wukong](./tools/wukong-guidelines) — 稳定性压力测试工具
- [白盒性能测试](./tools/perftest-guideline) — 基于 ArkXT 的性能测试框架
- [Python 性能脚本](./tools/hypium-perf-python-guidelines) — 自定义性能测试脚本
