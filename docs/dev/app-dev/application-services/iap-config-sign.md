在DevEco Studio上，参考[创建一个新的工程](/docs/tools/coding-debug/ide-create-new-project)和[配置调试签名（自动签名）](/docs/tools/coding-debug/ide-signing#section18815157237)完成工程应用的创建及签名配置，从而确保能顺利完成项目的编译与开发。

![](./img/4b92d8ca.png)

如果开发者DevEco Studio版本低于6.0.0.848，请使用[手动签名](/docs/tools/coding-debug/ide-signing#section297715173233)完成工程应用的签名配置。
