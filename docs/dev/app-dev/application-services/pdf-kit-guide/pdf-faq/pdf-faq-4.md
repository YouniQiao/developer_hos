不可以，PDF Kit移除页眉页脚、水印或背景不会针对具体页面，请参考[页眉页脚](/docs/dev/app-dev/application-services/pdf-kit-guide/pdf-pdfservice-implements/pdf-add-headerfooter)，[水印](/docs/dev/app-dev/application-services/pdf-kit-guide/pdf-pdfservice-implements/pdf-add-watermark)或[背景图片](/docs/dev/app-dev/application-services/pdf-kit-guide/pdf-pdfservice-implements/pdf-add-background)。
