**问题现象**

来电过程中，横幅通知展示，但头像无法正常显示。

**解决措施**

1. 保证头像图片大小在196608bit以内。
2. 支持传入的最大图片大小为 221 x 221 像素，推荐传入的图片大小为 112 x 112 像素。
3. 如还未解决，请通过[在线提单](https://developer.huawei.com/consumer/cn/support/feedback/#/)提交问题，华为支持人员会及时处理。
