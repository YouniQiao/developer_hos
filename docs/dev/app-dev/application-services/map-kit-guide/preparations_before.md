## 华为地图服务使用协议

1. 登录[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)网站，选择“协议签署记录”。

   ![](./img/8beab6b0.png)
2. 在查看下载协议中选择可查看“华为地图服务使用协议”。

   ![](./img/ec9e86cf.png)
