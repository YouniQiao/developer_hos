在AppGallery Connect正式发布资源包下载任务前，可以指定测试设备进行本地自测，具体操作请参见[测试下载功能](/docs/dev/app-dev/graphics/graphics-accelerate-kit-guide/graphics-accelerate-assetdownload/graphics-accelerate-assetdownload-service/graphics-accelerate-assetdownload-release#测试下载功能)。在测试设备上测试下载功能符合预期效果后，再正式发布资源包下载任务。
