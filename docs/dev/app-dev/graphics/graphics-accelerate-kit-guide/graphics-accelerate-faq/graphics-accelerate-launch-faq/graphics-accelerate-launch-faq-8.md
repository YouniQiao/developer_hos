根据应用上架审核规则，建议游戏在秒级启动场景下增加游戏健康公告提示页（如健康游戏提示或防沉迷提醒），在展示完成后再进入秒级启动恢复的内存镜像界面，以确保用户能够正常看到相关提示信息，详细操作可参考[示例工程](https://gitcode.com/HarmonyOS_Codelabs/graphics-accelerate-kit-launch-acceleration-codelab-arkts)。
