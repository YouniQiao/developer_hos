模板参数MatmulConfig，用于配置Matmul模板信息以及相关的配置参数。

MatmulConfig的参数说明见[参数说明](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-template-parameters#参数说明)表2。在默认模板范围【CFG\_NORM、CFG\_MDL、CFG\_IBSHARE\_NORM】中选取模板，不支持自定义模板。默认情况下，使能Norm模板。其中，各个接口参数的含义见[参数说明](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-template-parameters#参数说明)表2。
