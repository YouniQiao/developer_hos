## 函数功能

## 函数原型

```
size_t GetSubgraphNamesCount() const;
```

## 参数说明

无

## 返回值

| 类型 | 描述 |
| --- | --- |
| size\_t | 返回当前算子子图个数。 |

## 异常处理
