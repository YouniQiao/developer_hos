## 函数功能

封装算子的inferFormat函数。

## 函数原型

```
IMPLEMT_INFERFORMAT_FUNC(op_name, func_name)
```

## 约束说明

无

## 参数说明

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| op\_name | 输入 | 算子类型。 |
| func\_name | 输入 | inferFormat函数名，开发者自定义。 |

## 返回值

无
