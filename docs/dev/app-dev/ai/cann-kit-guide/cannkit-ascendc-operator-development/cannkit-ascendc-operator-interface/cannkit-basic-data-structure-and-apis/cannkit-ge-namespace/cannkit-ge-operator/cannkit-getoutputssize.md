## 函数功能

获取算子所有Output的个数。

## 函数原型

```
size_t GetOutputsSize() const;
```

## 参数说明

无

## 返回值

| 类型 | 描述 |
| --- | --- |
| size\_t | 返回当前算子的Output个数。 |

## 约束说明

无
