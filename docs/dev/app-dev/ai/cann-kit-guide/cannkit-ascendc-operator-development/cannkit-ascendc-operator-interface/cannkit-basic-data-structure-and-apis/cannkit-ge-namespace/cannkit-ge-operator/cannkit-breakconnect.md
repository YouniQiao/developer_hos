## 函数功能

删除当前算子与前一个算子之间的所有连接关系，删除当前算子与下一个算子之间的所有连接关系。

## 函数原型

```
void BreakConnect() const;
```

## 参数说明

无

## 返回值

无

## 异常处理

无
