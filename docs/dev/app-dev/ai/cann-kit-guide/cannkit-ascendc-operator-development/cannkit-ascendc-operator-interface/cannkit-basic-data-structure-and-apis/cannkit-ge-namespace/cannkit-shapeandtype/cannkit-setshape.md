## 函数功能

向TensorDesc中设置Tensor的Shape。

## 函数原型

```
void SetShape(const Shape &shape)
```

## 参数说明

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| shape | 输入 | 需向TensorDesc设置的shape对象。 |

## 返回值

无

## 异常处理

无

## 约束说明

无
