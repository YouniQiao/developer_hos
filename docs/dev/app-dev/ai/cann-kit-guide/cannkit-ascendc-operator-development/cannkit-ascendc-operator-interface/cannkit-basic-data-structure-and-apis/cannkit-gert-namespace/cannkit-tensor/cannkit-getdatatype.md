## 函数功能

获取Tensor的DataType。

## 函数原型

```
ge::DataType GetDataType() const;
```

## 参数说明

无

## 返回值

| 类型 | 描述 |
| --- | --- |
| ge:: [DataType](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-ge-datatype) | 返回tensor的DataType值，默认值为DT\_UNDEFINED。 |

无
