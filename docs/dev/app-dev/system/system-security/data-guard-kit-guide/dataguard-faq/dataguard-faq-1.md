**问题现象**

DevEco Studio编译不通过，报错：install failed due to grant request permissions failed。

**解决措施**

1. 请按[开发准备](/docs/dev/app-dev/system/system-security/data-guard-kit-guide/dataguard-preparations)内容依次核对开发前置条件是否满足。
2. 如还未解决，请通过[在线提单](https://developer.huawei.com/consumer/cn/support/feedback/#/)提交问题，华为支持人员会及时处理。
