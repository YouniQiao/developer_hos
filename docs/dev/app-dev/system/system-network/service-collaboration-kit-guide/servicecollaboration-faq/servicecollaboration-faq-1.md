跨设备互通API支持根据特定调用策略调用设备。调用策略：PC/2in1设备可以调用Tablet和Phone，Tablet可以调用Phone，同类型设备不可调用。

1. 确保设备调用策略无误。
2. 设备选择列表仍没有周围设备，请检查双端设备是否开启WLAN、蓝牙，并且登录同一个华为账号。
