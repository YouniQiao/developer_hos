# DevEco Testing 版本说明 — 6.0.1(21)

当前为 DevEco Testing 最新版本说明文档，如需查看 DevEco Testing 其它历史版本的功能新增、变更情况，请在左侧文档目录中选择相应版本。

## DevEco Testing 6.0.1 Release（6.0.7.203）

1. 优化了任务创建、任务执行的稳定性。

2. 优化了mac版本的用户交互体验，减少了测试报告占用的磁盘空间。
