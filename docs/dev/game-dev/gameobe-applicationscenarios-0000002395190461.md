## 满足多种游戏玩法

联机对战服务提供多种匹配方式，支持玩家约战或快速匹配（包括房间匹配、在线匹配和组队匹配），满足多种游戏玩法。

* 您只需要创建一个游戏房间，即可通过分享该房间的ID或短码邀请朋友加入对战。
* 您还可以通过发起房间匹配、在线匹配或组队匹配的方式，直接匹配并加入到满足匹配规则的对战房间中。

## 适用多种游戏场景

联机对战服务支持快速构建玩家匹配规则，并提供了强大的帧同步能力，适用于棋牌休闲、格斗射击、MOBA、战役副本、大乱斗等多种游戏场景。

* 回合制游戏：支持棋牌、策略、角色扮演等回合制游戏。
* 实时游戏：支持MOBA、FPS、休闲对战等实时对战类游戏。
