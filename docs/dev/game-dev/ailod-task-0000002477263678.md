在游戏中，LOD并非越高越好，建议在性能需求和视觉质量之间找到平衡。LOD越高，会增加资源消耗，可能导致性能下降；LOD过低，可能影响画面细节表现。因此，合理配置LOD是关键，例如，减少非关键物体的多边形数量。

您可以使用AILOD生成不同细节级别的LOD模型。

![](./img/2799b0ea.png)

AILOD为端云协同工作，因此使用期间需要保持网络畅通。
