* 请确保游戏已进入Play Mode（运行模式）。
* 请确保World已正确初始化，并在Relationship Window左上角的World下拉栏中正确选择要观察的World。

* 本工具在Editor模式下通过反射和轮询获取数据，开启窗口会产生CPU开销，建议在进行性能Profiling时关闭此窗口，否则可能会影响游戏性能。
