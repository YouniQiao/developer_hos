当您遇到问题时，可以按照以下顺序寻求帮助。

1. 请先仔细阅读文档，部分功能可能有限制。
2. 查看[FAQ](/docs/dev/game-dev/games-quickgame-faq-0000002458772633)、[案例](/docs/dev/game-dev/games-quickgame-case-0000002318064148)，寻找类似问题的解决方案。
3. 通过[智能客服](https://developer.huawei.com/consumer/cn/customerService/#/bot-dev-top/faq-top/faq-talk-top)查找问题解决方案。
4. 登录[华为开发者论坛](https://developer.huawei.com/consumer/cn/forum/block/ag-connect)参与问题讨论。
5. 若以上方法仍未解决问题，您可以通过[在线工单系统](https://developer.huawei.com/consumer/cn/support/feedback/#/add/202200004?level2=91&level3=2022000030)与我们进行联系。提交后的工单可在开发者联盟“个人中心 &gt; 我的工单”查看工单处理进展。
