当您使用AILOD遇到问题时，可以按照以下顺序寻求帮助。

1. 请先仔细阅读业务文档，部分功能和系统有限制，请参见[使用限制](/docs/dev/game-dev/ailod-restrictions-0000002509045147)。
2. 自主查看[FAQ](/docs/dev/game-dev/ailod-faq-0000002509054397)。
3. 通过[智能客服](https://developer.huawei.com/consumer/cn/customerService/#/bot-dev-top/faq-top/faq-talk-top)查找问题解决方案。
4. 登录[华为开发者论坛](https://developer.huawei.com/consumer/cn/forum/block/gaming-hub)和[Stack Overflow社区](https://stackoverflow.com/questions/tagged/appgallery-connect)参与问题讨论。
5. 如果以上方法仍未解决您的问题，可以通过[在线工单系统](https://developer.huawei.com/consumer/cn/support/feedback/#/add/202200004?level2=91&level3=401691723511655727)与我们进行联系。提交后的工单可在开发者联盟“个人中心 &gt; 我的工单”查看工单处理进展。
