## manifestж–Үд»¶

manifest.jsonй…ҚзҪ®ж–Үд»¶еҢ…еҗ«дәҶеҝ«жёёжҲҸжҸҸиҝ°гҖҒжҺҘеҸЈеЈ°жҳҺгҖҒйЎөйқўи·Ҝз”ұзӯүдҝЎжҒҜгҖӮ

| еұһжҖ§ | зұ»еһӢ | еҝ…еЎ«(M)/йҖүеЎ«(O) | жҸҸиҝ° |
| --- | --- | --- | --- |
| package | string | M | еҝ«жёёжҲҸеҢ…еҗҚгҖӮ |
| name | string | M | еҝ«жёёжҲҸеҗҚз§°гҖӮ |
| appType | string | M | еҝ«жёёжҲҸзұ»еһӢпјҡ  еҝ«жёёжҲҸ**еЎ«еҶҷ****fastgame**гҖӮ |
| icon | string | M | еҝ«жёёжҲҸеӣҫж ҮгҖӮеҝ…йЎ»дёҺAGCжҺ§еҲ¶еҸ°дёҠжһ¶е®Ўж ёзҡ„еӣҫж ҮдҝқжҢҒдёҖиҮҙгҖӮ |
| versionName | string | O | еә”з”ЁзүҲжң¬еҗҚз§°гҖӮдҫӢеҰӮвҖң1.0вҖқгҖӮ |
| versionCode | integer | M | еә”з”ЁзүҲжң¬еҸ·гҖӮд»Һ1иҮӘеўһпјҢжҜҸж¬ЎйҮҚж–°дёҠдј еҢ…еҗҺеҝ…йЎ»versionCodeеҠ 1пјҢеҗҰеҲҷе°ҶеҪұе“ҚдёҠжһ¶зүҲжң¬зҡ„жӣҙж–°пјҢдҫӢеҰӮеҺҹзүҲжң¬дёә11пјҢжӣҙж–°зүҲжң¬зҡ„versionCodeеҝ…йЎ»дёә12гҖӮ |
| minPlatformVersion | integer | M | ж”ҜжҢҒзҡ„жңҖе°Ҹе№іеҸ°зүҲжң¬еҸ·пјҢй»ҳи®ӨеҖјжҳҜ1000гҖӮеұһдәҺе…је®№жҖ§жЈҖжҹҘпјҢйҒҝе…ҚдёҠзәҝеҗҺеңЁдҪҺзүҲжң¬е№іеҸ°иҝҗиЎҢеҮәзҺ°дёҚе…је®№зҡ„й—®йўҳгҖӮ  иҜҙжҳҺпјҡ  иӢҘеҝ«жёёжҲҸеј•ж“ҺдҪҺдәҺиҜҘеҖјж—¶пјҢжү“ејҖ**еҝ«еә”з”Ёдёӯеҝғ**жҲ–**иҠұз“ЈиҪ»жёё**зҡ„еҝ«жёёжҲҸе°Ҷжңүеј№жЎҶжҸҗзӨәз”ЁжҲ·жӣҙж–°еҲ°жңҖж–°зүҲжң¬гҖӮиӢҘз”ЁжҲ·йҖүжӢ©дёҚжӣҙж–°пјҢе°ҶзӣҙжҺҘйҖҖеҮә**еҝ«еә”з”Ёдёӯеҝғ**жҲ–**иҠұз“ЈиҪ»жёё**гҖӮ |
| config | object | M | зі»з»ҹй…ҚзҪ®дҝЎжҒҜпјҢиҜҰжғ…иҜ·еҸӮи§Ғ[configеҸӮж•°](#section829569145720)гҖӮ |
| display | object | O | UIжҳҫзӨәй…ҚзҪ®пјҢиҜҰжғ…иҜ·еҸӮи§Ғ[displayеҸӮж•°](#section6321102611211)гҖӮ |
| subpackages | object | O | е®ҡд№үе№¶еҗҜз”ЁеҲҶеҢ…еҠ иҪҪпјҢиҜҰжғ…иҜ·еҸӮи§Ғ[subpackagesеҸӮж•°](#section49093457219)гҖӮ |

### configеҸӮж•°

з”ЁдәҺе®ҡд№үзі»з»ҹй…ҚзҪ®е’Ңе…ЁеұҖж•°жҚ®гҖӮ

| еұһжҖ§ | зұ»еһӢ | еҝ…еЎ«(M)/йҖүеЎ«(O) | й»ҳи®ӨеҖј | жҸҸиҝ° |
| --- | --- | --- | --- | --- |
| logLevel | string | M | log | ж—Ҙеҝ—жү“еҚ°зӯүзә§пјҡ   * off * error * warn * info * log * debug |

### displayеҸӮж•°

з”ЁдәҺе®ҡд№үдёҺUIжҳҫзӨәзӣёе…ізҡ„й…ҚзҪ®гҖӮ

еӨ§еұҸи®ҫеӨҮд»…ж”ҜжҢҒfullScreenи®ҫзҪ®дёәtrueгҖҒorientationи®ҫзҪ®дёәlandscapeгҖӮ

| еұһжҖ§ | зұ»еһӢ | й»ҳи®ӨеҖј | жҸҸиҝ° |
| --- | --- | --- | --- |
| backgroundColor | string | #ffffff | зӘ—еҸЈиғҢжҷҜйўңиүІгҖӮ |
| fullScreen | boolean | false | жҳҜеҗҰе…ЁеұҸжЁЎејҸгҖӮе…ЁеұҸжЁЎејҸдёӢе°ҶиҰҶзӣ–зҠ¶жҖҒж ҸгҖӮ |
| menu(1076+) | boolean | false | жҳҜеҗҰжҳҫзӨәеҸідёҠи§’иҸңеҚ•жҢүй’®гҖӮд»Һ1076е№іеҸ°зүҲжң¬ејҖе§Ӣе°Ҷеӣәе®ҡжҳҫзӨәиҸңеҚ•пјҢдёҚж”ҜжҢҒиҮӘе®ҡд№үиҸңеҚ•еҶ…е®№гҖӮ |
| windowSoftInputMode(1030+) | adjustPan | adjustResize | adjustPan | иҪҜй”®зӣҳеј№еҮәж—¶пјҢдёәдәҶдҝқиҜҒйЎөйқўзҡ„иҫ“е…ҘжЎҶеҸҜи§ҒпјҢеҪ“еүҚйЎөйқўзҡ„и°ғж•ҙж–№ејҸпјҡ   * adjustPanпјҡдёҠз§»йЎөйқўгҖӮ * adjustResizeпјҡеҺӢзј©йЎөйқўзҡ„жҳҫзӨәеҢәеҹҹгҖӮиӢҘйЎөйқўе…ЁеұҸпјҢжӯӨи®ҫзҪ®йЎ№дёҚз”ҹж•ҲгҖӮ |
| pages | object | - | йЎөйқўзҡ„жҳҫзӨәж ·ејҸгҖӮ |
| orientation | string | portrait | йЎөйқўжҳҫзӨәж–№еҗ‘пјҡ   * portraitпјҡз«–еұҸгҖӮ * landscapeпјҡжЁӘеұҸгҖӮ * auto(1070+)пјҡеҸ—жүӢжңәзі»з»ҹиҮӘеҠЁж—ӢиҪ¬ејҖе…ізҡ„жҺ§еҲ¶гҖӮиӢҘи®ҫзҪ®autoпјҢе»әи®®дҪҝз”Ё[е“Қеә”ејҸеёғеұҖ](https://developer.huawei.com/consumer/cn/doc/quickApp-Guides/quickapp-responsive-layout-0000001082763996)ејҖеҸ‘гҖӮ |

### subpackagesеҸӮж•°

з”ЁдәҺе®ҡд№үеҲҶеҢ…зҡ„зӣёе…ій…ҚзҪ®гҖӮеҲҶеҢ…зҡ„дҪҝз”ЁжҢҮеҜјиҜ·еҸӮи§Ғ[еҲҶеҢ…еҠ иҪҪ](https://developer.huawei.com/consumer/cn/doc/quickApp-Guides/quickgame-subpackage-0000001159658271)гҖӮ

| еұһжҖ§ | зұ»еһӢ | жҸҸиҝ° |
| --- | --- | --- |
| name | string | еҲҶеҢ…еҗҚз§°гҖӮеҗҚз§°еҸӘиғҪжҳҜеӯ—жҜҚж•°еӯ—е’ҢдёӢеҲ’зәҝпјҢдёҚе…Ғи®ёеҢ…еҗ«е…¶е®ғз¬ҰеҸ·гҖӮ"base"жҳҜеҹәзЎҖеҢ…зҡ„дҝқз•ҷеҗҚз§°пјҢжӮЁж— йңҖдёәеҹәзЎҖеҢ…е®ҡд№үеҲҶеҢ…й…ҚзҪ®гҖӮ |
| resource | string | еҲҶеҢ…иө„жәҗж №зӣ®еҪ•пјҢзӣёеҜ№дәҺжәҗз Ғзӣ®еҪ•"src"зҡ„зӣёеҜ№и·Ҝеҫ„гҖӮж №зӣ®еҪ•еҸӘиғҪжҳҜеӯ—жҜҚж•°еӯ—д»ҘеҸҠ"\_"гҖҒ"-"гҖҒ"/"з»„жҲҗпјҢз¬¬дёҖдёӘеӯ—з¬ҰдёҚе…Ғи®ёдёә"-"е’Ң"/"пјҢдёҚе…Ғи®ёеҢ…еҗ«е…¶е®ғз¬ҰеҸ·гҖӮзј–иҜ‘ж—¶дјҡжҠҠиҜҘзӣ®еҪ•дёӢзҡ„жүҖжңүиө„жәҗйғҪжү“еҢ…еҲ°еҲҶеҢ…дёӯгҖӮ |
