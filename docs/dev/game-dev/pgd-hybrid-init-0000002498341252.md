```
public class SampleAuthoring : MonoBehaviour
{
    class Sample : PgdHybrid<SampleAuthoring>
    {
        // 在Handle方法中实现混合模式Entity初始化逻辑
        public override void Handle(SampleAuthoring authoring)
        {
            // 可自动基于该MonoBehaviour挂载的Prefab生成Entity
            var entity = GetHybridEntity();
            // 为Entity添加Component
            AddComponent<SampleComponent>(entity);
        }
    }
}
public struct SampleComponent : IComponent
{
}
```

将上述MonoBehaviour脚本挂载至Prefab资产，并将Prefab资产挂载至GameObject组件上，即可在初始化阶段自动生成包含Prefab属性的模板Entity。
