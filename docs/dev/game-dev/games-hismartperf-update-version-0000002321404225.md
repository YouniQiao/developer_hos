若打开工具时未升级工具版本，您可以在“版本升级”页面进行升级。

1. 打开“版本升级”页面，点击“立即升级”。

   ![](./img/bcf20ef2.png)

   若页面出现如下告警信息，建议您立即升级工具版本，否则可能影响您使用工具。

   ![](./img/49c8dd39.png)
2. 请耐心等待最新版本的下载。工具成功下载后，您可以根据提示安装工具。
