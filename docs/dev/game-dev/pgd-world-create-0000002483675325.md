## 创建World

```
using Pgd;

// 创建一个新的World实例
var world = new IECSWorld();
```

## 使用World属性

```
// 获取基本统计信息
Console.WriteLine($"实体数量: {world.EntityNum}");
Console.WriteLine($"原型数量: {world.ArchetypesCount}");
Console.WriteLine($"容量: {world.Capacity}");
Console.WriteLine($"最大ID: {world.NodeMaxId}");

// ID回收控制
world.RecycleIdsActive = false; // 禁用ID回收，确保ID连续性
world.RecycleIdsActive = true;  // 启用ID回收，提高内存效率
```
