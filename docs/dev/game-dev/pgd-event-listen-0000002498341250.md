对于PGD内置事件，开发者只需要注册对应的事件处理器，之后调用对应接口时会自动发布事件来触发事件处理器的逻辑，无需开发者手动发布事件。OnComponentChanged、OnTagChanged、OnChildEntityChanged分别在组件、标签、子实体的增删改等变更操作时会自动发布对应事件，OnEntityCreated和OnEntityDestroyed分别在实体创建和删除时会自动发布对应事件，OnSystemChanged是静态属性，会在系统注册、移除、名字变更、激活状态变更、移动时自动发布对应事件。

```
using Pgd.Event;

var world = new IECSWorld();
// 开启EventHub
world.EnableEventHub();

// 监听实体创建事件
world.OnEntityCreated += e => {Debug.Log($"Entity created: {e.Entity.Id}")};
// 监听实体销毁事件
world.OnEntityDestroyed +=e =>
{
    Debug.Log($"Entity destroyed: {e.EntityId}"),
};

// 监听组件变化事件
world.OnComponentChanged += e => {
    switch (e.ChangeType)
    {
        case ComponentChangeType.Add:
            Debug.Log($"Component added to {e.EntityId}: {e.ComponentType}");
            break;
        case ComponentChangeType.Remove:
            Debug.Log($"Component removed from {e.EntityId}: {e.ComponentType}");
            break;
        case ComponentChangeType.Update:
            Debug.Log($"Component updated on {e.EntityId}: {e.ComponentType}");
            break;
    }
};

// 监听标签变化事件
world.OnTagChanged += e=>
{
    Debug.Log($"Tag changed: {e.EntityId} | {e.Tags}");
};

// 监听子实体变化事件
world.OnChildEntityChanged += e => {
    Debug.Log($"Child entity {e.ChangeType}: parent={e.EntityId}, child={e.ChildId}, index={e.ChildIndex}");
};

// 监听系统变化事件。系统变化事件是静态属性
EcsWorldBase.OnSystemChanged += e => {
    Debug.Log($"System {e.System.SystemName} {e.ChangeType}: {e.Field} = {e.Value}");
};
```
