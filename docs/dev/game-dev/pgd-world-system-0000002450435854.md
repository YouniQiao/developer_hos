World负责管理游戏系统的注册和生命周期，系统的详细使用方法见[System](https://developer.huawei.com/consumer/cn/doc/games-references/system-0000002467576713)。

```
// 注册系统
world.RegisterSystem(new AISystem());
world.RegisterSystem(new MoveSystem());

// 更新所有系统
world.Update();
```
