## 自定义场景分析

通过自定义的场景如大厅/战斗/剧情，分别查看不同场景下游戏的流畅度指标，结合场景需求进行优化调整，来选择降低功耗或提高性能表现。

## 多机型性能对比

通过筛选观测不同机型的性能表现，精确定位性能存在瓶颈的机型，针对性地进行性能优化，使不同机型玩家都能有流畅的游戏体验。

## 新版本运行监控

游戏新版本上线后，通过筛选新版本与旧版本的性能数据，对比观察版本更新后的性能指标变化，保障新版本稳定运行。
