语音转文本功能具备快速识别实时语音并转录成文本内容的能力，可用于语音输入等使用场景。

![](./img/fe884c48.png)

* 目前，语音转文本功能支持识别中文和英文音频内容。
* 该服务为增值服务，具体收费方式请参考[服务价格](https://developer.huawei.com/consumer/cn/doc/hiai-Guides/ml-service-billing-0000001051010023)。
