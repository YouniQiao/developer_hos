## 高性能批量创建

通过Archetype直接创建实体，避免了实体结构变更的开销。

```
// 使用原型批量创建实体（推荐用于大量实体）
var archetype = world.GetCreateArchetype(
    IComponents.Get<PgdPosition, PgdScale>(),
    ITags.Get<EnemyTag>()
);

// 批量创建10万个实体
var entities = archetype.CreateEntities(100_000);
Console.WriteLine($"创建了 {entities.Count} 个实体");
```

## 复制实体

支持跨world复制实体的全部内容。

```
var sourceWorld = new IECSWorld();
var targetWorld = new IECSWorld();

// 在源world创建实体
sourceWorld.CreateEntity(new PgdPosition(1, 1, 1));
sourceWorld.CreateEntity(new PgdPosition(2, 2, 2), ITags.Get<NetworkTag>());
sourceWorld.CreateEntity(new PgdPosition(3, 3, 3));

// 查询特定实体并复制到目标world
var networkQuery = sourceWorld.Query().WithAnyTags(ITags.Get<NetworkTag>());
foreach (var entity in networkQuery.Entities)
{
    // 在目标world创建相同ID的实体
    var targetEntity = targetWorld.CreateEntity(entity.Id);
    // 复制所有组件和标签
    entity.CopyEntityTo(targetEntity);
}
```

## 删除实体

Archetype.DestroyAllEntities()是高效的批量删除方式，支持单次删除该原型下的所有实体。

* 方式1：通过实体获取Archetype。

  ```
  var world = new IECSWorld();

  // 创建一批实体
  for (int i = 0; i < 1000; i++)
  {
      world.CreateEntity(
          new PgdPosition(i, i, i)
      );
  }

  // 方式1：通过任意一个实体获取其Archetype
  var query = world.Query<PgdPosition>();
  var firstEntity = query.Entities.First();
  var archetype = firstEntity.Archetype; // 获取实体的原型

  // 删除该Archetype下的所有实体
  archetype.DeleteAllEntities();
  Console.WriteLine($"删除后实体数量: {world.EntityNum}"); // 0
  ```
* 方式2：通过组件和标签组合获取Archetype。

  ```
  var world = new IECSWorld();

  // 创建实体
  for (int i = 0; i < 1000; i++)
  {
      world.CreateEntity(
          new PgdPosition(i, i, i),
          new Velocity(),
          ITags.Get<EnemyTag>()
      );
  }

  // 方式2：通过组件和标签组合获取Archetype
  var components = IComponents.Get<PgdPosition, Velocity>();
  var tags = ITags.Get<EnemyTag>();
  var archetype = world.GetArchetype(components, tags);

  if (archetype != null)
  {
      Console.WriteLine($"该原型有 {archetype.EntityNum} 个实体");
      archetype.DeleteAllEntities();
      Console.WriteLine($"删除后实体数量: {world.EntityNum}");
  }
  ```
