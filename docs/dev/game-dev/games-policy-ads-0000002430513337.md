鲸鸿动能搭建鸿蒙生态全场景营销增长平台，全面升级HarmonyOS 5.0及以上系统级广告变现能力，提供多类专项激励与月度激励。汇聚亿级流量，激发营销新动能。

## 一、鲸鸿动能星火计划

![](./img/34f0b0f7.png)

鲸鸿动能星火计划的完整细则请以[鲸鸿动能互联网行业星火计划](/docs/monetize/promotion/ads-xhjh-0000002139646894)为准。

| **门槛** | **阶段** | **消耗门槛** | **投放时间要求** | **激励政策** | **激励上限** |
| --- | --- | --- | --- | --- | --- |
| 2025年报名之日起前90天无消耗的推广产品 | 政策1  （扶上马） | 无门槛 | 激励下发后的30天内 | 8000元赠送金 | 8000元 |
| 政策2  （送一程） | 总现金消耗≥30,000元 | 申请报名后的60天内 | 60天总现金消耗\*20% | 不限 |

## 二、鲸鸿动能游戏行业月度激励

![](./img/101b1bb0.png)

鲸鸿动能游戏行业月度激励的完整细则请以[2025年鲸鸿动能游戏行业月度激励政策](/docs/monetize/promotion/ads-yxhyydjl-0000002160474788)为准。

无需报名，默认符合规则的产品均可参与。

* 在架游戏

  | 月度投放消耗额X（人民币：万元） | 返货金额（人民币：万元）  X1为月度现金消耗额 |
  | --- | --- |
  | 10≤X\&lt;100 | 30%\*X1 |
  | 100≤X\&lt;200 | 40%\*X1 |
  | 200≤X\&lt;400 | 50%\*X1 |
  | 400≤X\&lt;600 | 55%\*X1 |
  | 600≤X\&lt;1000 | 60%\*X1 |
  | 1000≤X | 70%\*X1 |
* 预约游戏

  | 预约期总消耗额X（人民币：万元） | 返货金额（人民币：万元）  X1为预约期现金消耗额 |
  | --- | --- |
  | 10≤X\&lt;60 | 40%\*X1 |
  | 60≤X\&lt;100 | 50%\*X1 |
  | 100≤X | 60%\*X1 |

![](./img/863e49ad.png)

| 分类 | HarmonyOS 5.0及以上版本首年激励政策（截止到2025年12月31日） | |
