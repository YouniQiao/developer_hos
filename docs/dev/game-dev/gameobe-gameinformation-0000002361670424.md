1. 在项目列表中选择您的项目，进入“项目设置”的“常规”页签。
2. 记录项目下的“Client ID”和“Client Secret”，以及应用下的“APP ID”信息。

   ![](./img/303c2c45.png)

   “Client ID”、“Client Secret”和“APP ID”信息主要用于SDK接入校验。

   ![](./img/90845f16.png)

   | 参数 | 说明 |
   | --- | --- |
   | Client ID | 客户端ID，集成项目级SDK鉴权时的唯一标识。 |
   | Client Secret | 客户端密钥，集成项目级SDK鉴权时的密钥。 |
   | APP ID | 应用ID。 |
