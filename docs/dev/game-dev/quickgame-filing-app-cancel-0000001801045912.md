## 前提条件

已前往应用市场下载最新版本的华为云App。

## 操作步骤

1. 在华为云App的“ICP备案”界面取消快游戏接入。

   ![](./img/3a4eec3b.png)
2. 验证负责人的手机/邮箱，选择取消接入原因后点击“提交初审”，在弹出的“确定要取消接入互联网信息”窗口点击“确认”。

   ![](./img/674ba595.png)
3. 提交申请后，华为平台将自动审核您的申请，您需耐心等待通管局的审核。
