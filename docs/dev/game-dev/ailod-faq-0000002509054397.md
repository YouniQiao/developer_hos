## 点击“Test Connection”后显示连接失败，怎么办？

点击“Test Connection”后显示连接失败，并出现如下提示：

![](./img/b3c634c4.png)

```
[17:21:10] AILOD:"System.Net.Http.HttpRequestException: An error occurred while sending the request ---> System.Net.WebException: Error: ConnectFailure(由于连接方在一段时间后没有正确答复或连接的主机没有反应，连接尝试失败。)
```

解决方案如下：

* 情况一：AILOD服务异常，您可以通过[在线工单系统](https://developer.huawei.com/consumer/cn/support/feedback/#/ticketCard)与我们取得联系（“问题分类”请选择“游戏性能调优”）。

* 情况二：网络问题，您可以尝试检查网络。

## 此前已完成登录，为什么一段时间后提示未登录？

AILOD会在某些时刻向云端发出刷新令牌请求。

若此时遇到网络异常会导致登录状态丢失，出现如下提示：

```
Please sign in or register an account first.
```

此时无需重新登录，确认网络连接正常后重新打开AILOD窗口即可。
