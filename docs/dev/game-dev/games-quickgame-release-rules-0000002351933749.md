为减少提交上架审核被驳回的概率，请参考上架前自检CheckList进行自检，检查通过后，再前往AppGallery Connect上架快游戏。

| 一级分类 | 二级分类 | 检查项 | 影响 |
| --- | --- | --- | --- |
| 合规检查 | 隐私、游戏资质等 | 了解快游戏的内容要求与审核要求，例如游戏中的哪些交互行为属于不当内容、游戏如何更好的为用户提供付费内容等，确保您的快游戏符合**适用的法律法规**、**当代社会价值观**以及**华为终端分发渠道的审核要求**，帮助您向最终用户提供值得依赖的快游戏，审核规定详情请参见[华为应用市场审核指南](/docs/distribute/app-dist/app-market/x50000/x50104/x50104-overview)。 | 若不符合华为应用市场的审核要求，快游戏上架审核将被驳回。 |
| 病毒 | 前往[检测病毒网站](https://www.virustotal.com)检测快游戏是否存在病毒。 | 若快游戏存在病毒，快游戏上架审核将被驳回。 |
| 配置检查 | 应用类型 | 登录AppGallery Connect控制台，检查应用类型是否为**游戏**。 | 若不是游戏类应用，会导致游戏防沉迷异常，游戏上架审核将被驳回。 |
| 签名证书指纹 | 在AppGallery Connect中配置的签名指纹证书与快游戏实际的签名指纹证书保持一致。 | 若不一致，会影响快游戏的正常功能使用。 |
| 基本信息 | 启动 | 快游戏打开后出现标明核准（备案）号、游戏著作权人、出版单位、批准文号、出版物号、健康游戏忠告等信息，且标明上述信息的页面停留时间不宜过短。 | 若未出现相关信息，快游戏上架审核将被驳回。 |
| 第三方外链 | 不允许在快游戏内跳转至三方网址。 | 若出现游戏内跳转至三方网址，快游戏上架审核将被驳回。 |
| 第三方SDK | 不允许接入如下三方SDK：   * 第三方广告SDK。 * 第三方聚合SDK。 * 第三方支付SDK。 | 若快游戏集成了第三方SDK，游戏上架审核将被驳回。 |
| 代码检查 | 广告 | 参考[自测项](/docs/monetize/monetization/ceshiyanzhengjishangxian-0000001085219714#1-自测)使用测试IDE测试接入的广告能力。 | 若自测不通过，快游戏上架将被驳回。 |
| 登录 | 调用**防沉迷登录**接口。 | 若未接入防沉迷，将违反国家新闻出版署的相关规定。 |
| 仅允许使用**华为账号**登录。 | 若快游戏包含第三方账号登录，快游戏上架审核将被驳回。 |
| 支付 | **createPurchaseIntent**接口传入的商品ID、商品类型与在AppGallery Connect创建的商品ID、商品类型保持一致。 | 若商品ID、商品类型与AppGallery Connect不一致，将无法打开华为支付收银台界面。 |
| 在如下异常场景下进行消耗型商品的补单处理：   * 快游戏启动时。 * 支付接口返回错误码**-1**和**60051**时。 | 若未在适当时机进行补单处理，会导致在**异常场景**下不能正常发放消耗型商品。 |
| 升级 | 若升级在架版本的快游戏，manifest.json文件中的**应用版本号**或**versionCode**值是否自增1。 | 若版本号未自增，快游戏将升级失败。 |
| 账号注销 | 快游戏需允许用户删除华为账号和游戏数据之间的绑定关系。 | 若不具备华为账号销户能力，快游戏上架审核将被驳回。 |
| 游戏退出 | 快游戏调用**游戏退出接口**。 | 若不具备**游戏退出功能**，快游戏上架审核将被驳回。 |
