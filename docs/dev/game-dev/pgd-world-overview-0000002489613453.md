World是PGD ECS框架的核心容器，负责管理所有的实体、组件、系统和查询。IECSWorld类是框架的主要入口点，提供了创建、管理和查询实体的完整功能。

## 核心特性

* 实体生命周期管理：创建、删除、激活/禁用实体。
* ID管理：实体ID分配和回收机制。
* 内存管理：动态扩容和内存优化。
* 系统管理：注册和管理游戏系统。
* 查询入口：提供实体查询功能。
