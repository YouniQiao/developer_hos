HarmonyOS文件系统按文件所有者不同可分为：应用文件、用户文件和系统文件。文件系统适配主要针对应用运行时对文件的操作（读写、只读），帮助理解rawfile、resfile和应用沙箱的差异点。更多内容请参见[Rawfile开发指导](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/rawfile-guidelines)。
