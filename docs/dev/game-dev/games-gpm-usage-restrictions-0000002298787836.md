本章节将为您详细介绍游戏性能管理SDK在设备等方面的限制，这些限制可能会根据实际情况随时更新，请您注意文档变化。

## 设备限制

设备限制是指SDK对用户终端的限制。

| 限制项 | 限制 |
| --- | --- |
| Android OS | Android 5及以上（API Level 21及以上） |

## 工具依赖

工具依赖是指对游戏开发引擎的版本要求。

| 工具 | 版本要求 |
| --- | --- |
| Unity引擎 | 推荐2021及以上版本。 |
