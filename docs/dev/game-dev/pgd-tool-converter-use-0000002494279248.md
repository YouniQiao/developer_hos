* 请勿在转换时修改脚本。
* 请勿在转换过程中频繁关闭窗口或退出编辑器。在日志尚未出现Conversion completed!或明显错误结论前，请不要关闭ECS Converter窗口或强制退出编辑器，以免生成的日志和输出不完整。
* 确保当前工程可以在编辑器中正常编译。
* 确保当前工程已在IDE中打开并生成解决方案文件。ECS Converter会以当前打开的工程目录为“输入工程”，开发者只需指定“输出目录”，不需要再单独指定输入路径。
