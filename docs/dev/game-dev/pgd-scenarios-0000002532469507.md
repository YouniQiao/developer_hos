PGD适用于需要处理大量游戏对象和复杂交互的游戏项目。

## 实时战略游戏（RTS）

在实时战略游戏（RTS）中，大规模单位管理可以使用PGD进行优化，能够流畅处理数百个单位的同时移动、战斗和AI决策。

## 动作游戏和射击游戏

在动作游戏和射击游戏开发中，PGD的高性能特性能够支持密集的子弹系统、粒子特效和物理模拟；低延迟特性确保了玩家操作的即时响应；批处理系统能够高效处理大量相似对象的创建和销毁。

## 角色扮演游戏（RPG）

在角色扮演游戏中，PGD的组合式设计能够实现复杂的装备系统、技能树和角色属性管理。
