## 保留关键视觉特征

AILOD支持最大化降低高模和低模在引擎中的渲染差异，还原模型的原始视觉效果。

## 智能重塑顶点

AILOD支持智能优化网格上各顶点位置，简化后模型的几何轮廓优于传统LOD策略。

## 区域自适应减面

AILOD采用高度自适应的非均匀简化算法，支持根据模型不同区域的精细程度动态调整减面密度。
