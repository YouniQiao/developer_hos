当您在集成华为服务过程中遇到问题时，可以按照以下顺序寻求帮助。

1. 通过[智能客服](https://developer.huawei.com/consumer/cn/customerService/#/bot-dev-top/faq-top/faq-talk-top)查找问题解决方案。
2. 登录[华为开发者论坛](https://developer.huawei.com/consumer/cn/forum/block/gaming-hub)和[Stack Overflow社区](https://stackoverflow.com/questions/tagged/appgallery-connect)参与问题讨论。
3. 扫码联系华为技术支持人员。

   |  |  |  |
   | --- | --- | --- |
   | ![](./img/20eb9807.png) | ![](./img/6faa7b0a.png) |  |
4. 若以上方法仍未解决您的问题，可以通过[在线工单系统](https://developer.huawei.com/consumer/cn/support/feedback/#/add/101704353566310877?level2=101704353626565886&level3=101704354579010004)与我们进行联系。提交后的工单可在开发者联盟“个人中心 &gt; 我的工单”查看工单处理进展。
