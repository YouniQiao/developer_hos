Game Service Kit（游戏服务）为游戏提供游戏近场快传能力，支持设备在彼此靠近的情况下进行游戏数据交换。更多业务介绍及接入指导请参见[游戏近场快传](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/gameservice-nearbytransfer-dev)。
