游戏多媒体服务实时信令（Real-time Messaging，RTM）功能提供了客户端SDK以及服务端REST API，该功能基于华为RTN网络，支持通过网络实时发送和接收消息，提供了稳定可靠、低延时、高并发的通信服务。当前，RTM客户端SDK提供了点对点消息收发、频道订阅、频道消息收发等能力，可用于实现实时通信、全局聊天、游戏通知、指令同步等游戏场景。同时，RTM服务端API提供了点对点消息以及频道消息发送能力，可用于发送游戏公告、消息通知等游戏使用场景。
