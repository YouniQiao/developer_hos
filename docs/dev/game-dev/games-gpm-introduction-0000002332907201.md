## 业务描述

|  |  |
| --- | --- |
| 游戏性能管理服务（Game Performance Management，GPM）是华为游戏中心推出的一款功能强大、集成简便的性能管理解决方案。仅需接入游戏性能管理SDK即可实现低功耗、全方面地采集游戏运行时的性能数据，经过华为服务器深度分析，将在AGC控制台以不同维度的可视化图表进行展示。基于此报告，华为将协助您不断地优化游戏性能，以确保游戏能够运行在最佳状态，为用户提供流畅、稳定的使用体验。 |  |

## 主要功能

| 主要功能 | 功能描述 |
| --- | --- |
| 数据采集 | 支持游戏运行时的性能数据采集，涵盖流畅度指标、硬件相关指标、系统性能指标及场景相关指标。 |
| 可视化分析 | 支持不同机型、版本及场景下的数据过滤，支持可视化分析数据。 |
