若需要在System中删除派生出的Entity，需使用CommandQueue，示例代码如下：

```
public class DestorySystem : PgdSystem<SampleComponent>
{
    protected overried void OnUpdate()
    {
        GetQuery().ForEachEntity((ref SampleComponent _, IEntity entity) => {
            CommandQueue.DestroyEntity(entity);
        })
        CommandQueue.Apply();
    }
}
```

![](./img/4945fd79.png)

PGD混合模式底层依赖对象池的回收机制，实现GameObject和实体的高效复用。对于使用混合模式的GameObject和实体，请勿在工程中显式调用GameObject销毁或Entity删除接口，避免运行出现错误。
