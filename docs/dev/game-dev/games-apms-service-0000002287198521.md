应用性能监测服务（Application Performance Management Service，简称APMS）是为游戏提供的质量监测解决方案，支持监测稳定性指标，例如崩溃（CPP CRASH、JS ERROR）、应用无响应（AppFreeze）等，还能监测性能指标，例如游戏启动、页面加载、运行耗电等，同时支持提供单个问题发生时的环境信息、堆栈信息等分析数据，并支持基于堆栈关键行进行准确的同类异常汇聚，让开发者轻松发现、准确定位、快速解决。更多介绍及接入指导请参见[性能监测（APMS）](https://developer.huawei.com/consumer/cn/doc/app/agc-help-apms-0000002235870062)。
