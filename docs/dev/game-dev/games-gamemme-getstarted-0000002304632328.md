项目是您在AppGallery Connect（以下简称AGC）资源的组织实体，您可以将一个应用的不同平台版本添加到同一个项目中。当您的应用需要使用游戏多媒体服务时，您可以[创建项目](https://developer.huawei.com/consumer/cn/doc/app/agc-help-create-project-0000002242804048)，并在项目下[添加应用](https://developer.huawei.com/consumer/cn/doc/app/agc-help-create-app-0000002247955506)。
