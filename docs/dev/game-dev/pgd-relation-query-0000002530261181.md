## 获取指定目标的关系

```
// 方式一
var rel1 = entity.GetRelation<AttackRelation, IEntity>(target); // 获取关系
var rel2 = entity.GetRelation<InventoryItem, InventoryItemType>(InventoryItemType.Axe); // 获取关系

// 方式二
var flag1 = entity.TryGetRelation<AttackRelation, IEntity>(target, out AttackRelation rel1); // 获取关系
var flag2 = entity.TryGetRelation<InventoryItem, InventoryItemType>(InventoryItemType.Axe, out InventoryItem rel2); // 获取关系
```

## 获取关系集合相关操作

```
// 获取entity拥有的AttackRelation关系的集合
var relations = entity.GetRelations<AttackRelation>();
foreach (var relation in relations)
{
    /* 具体处理逻辑 */
    // ......
}

// 获取entity拥有的AttackRelation关系数量
var relationCount = entity.GetRelationsCount<AttackRelation>();

// 获取target拥有的被动AttackRelation关系集合
var beRelations = target.GetBeRelatedLinks<AttackRelation>();
```

## 查询关系

查询关系可通过QueryRelation接口获取到IQuery，通过IQuery进行查询操作。

| 迭代接口 | 支持情况 |
| --- | --- |
| IQuery.Entities | 完全支持。 |
| IQuery.ForEachEntity | 完全支持。  迭代提供共享组件值引用，可直接修改。 |
| IQuery.ArchetypeChunk | 不支持 |
| IQuery.ParallelForEach | 不支持 |

```
var query = world.QueryRelation<AttackRelation>();

// IQuery.ForEachEntity方式迭代
query.ForEachEntity((ref AttackRelation relation, IEntity entity) =>
{
    /* 具体处理逻辑 */
    // ......
});

// IQuery.Entities方式迭代
foreach (var entity in query.Entities)
{
    var relations = entity.GetRelations<AttackRelation>();
    /* 具体处理逻辑 */
    // ......
}
```

QueryRelation接口也可以带过滤条件，通过QueryFilter设置过滤条件。

```
var filter = new QueryFilter();
filter.WithAnyComponents(IComponents.Get<PgdPosition>());
var query = world.QueryRelation<AttackRelation>(filter);  // 带过滤条件的query

// IQuery.ForEachEntity方式迭代
query.ForEachEntity((ref AttackRelation relation, IEntity entity) =>
{
    /* 处理逻辑 */
    // ......
});

// IQuery.Entities方式迭代
foreach (var entity in query.Entities)
{
    var relations = entity.GetRelations<AttackRelation>();
    /* 处理逻辑 */
    // ......
}
```

## 判断是否拥有关系

```
// 判断entity是否拥有关系
var hasRelation = entity.HasRelationType<AttackRelation>();

// 判断target是否拥有被动关系
var hasBeRelation = target.HasBeRelatedLinkType<AttackRelation>();
```
