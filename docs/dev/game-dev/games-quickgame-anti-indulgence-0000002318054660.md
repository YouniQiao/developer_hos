根据国家新闻出版署的最新规定，所有网络游戏企业仅可在周五、周六、周日和法定节假日每日20时至21时向未成年人提供1小时网络游戏服务，其他时间均不得以任何形式向未成年人提供网络游戏服务。

## 实名认证

中国大陆地区发布的游戏必须要进行实名认证，玩家在使用华为账号登录游戏会引导玩家进行实名认证，游戏无需自行开发实名认证系统。玩家登录会弹出实名认证弹框。如果玩家点击取消认证，会返回错误码7021，游戏可再次调用登录接口弹出实名认证弹框，或直接强制玩家下线游戏。

## 防沉迷监控

根据国家新闻出版署的最新规定，所有网络游戏企业仅可在周五、周六、周日和法定节假日每日20时至21时向未成年人提供1小时网络游戏服务，其他时间均不得以任何形式向未成年人提供网络游戏服务。**接入账号登录服务后，快游戏将自动实现防沉迷的必要能力，无需您二次开发防沉迷功能。**

* 已实名的未成年人在非规定游戏时间内登录游戏，游戏服务会弹框提示玩家不允许游戏，玩家点击“确定”，将直接退出游戏。
* 已实名的未成年人在规定时间内登录游戏，当游戏进行到晚上21时，游戏服务会弹框提示玩家已到游戏时间，玩家点击“知道了”，将直接退出游戏。

## 支付限额

根据国家新闻出版署的要求，8周岁以下的未成年人不允许充值。8周岁以上未满16周岁的未成年人用户单次充值金额不得超过50元人民币，每月充值金额累计不超过200元人民币。16周岁以上的未成年人用户单次充值金额不得超过100元人民币，每月充值金额累计不得超过400元人民币。

用户在使用华为应用内支付时，华为会自动根据国家新闻出版署的要求进行支付限额控制，您无需处理。
