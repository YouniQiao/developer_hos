## 如何使用快游戏开发者工具进行试玩调试？

1. 在电脑上安装最新版本快游戏开发者工具（版本号大于等于15.0.2）。
2. 运行含试玩分包的主包项目，在运行主界面点击右上角四个点，在弹出的下拉框内查看最右边的试玩调试按钮，点击可以进入试玩分包。如果未出现试玩调试按钮，请检查试玩分包配置，当试玩分包配置错误时不会出现该按钮。

## 如何制作试玩分包？

建议使用游戏引擎发布Web版本，生成只含单个HTML的游戏文件，双击本地浏览器打开验证是否可玩。

## 试玩分包如何合入主包中？

试玩分包需要合入主包中，每次发布时必须随主包一起发布上架。目前有2种将试玩分包合入主包的方式可供选择。第一种是使用快游戏开发者工具，在项目目录中新建试玩分包目录，将试玩分包HTML拷贝到当前项目目录中的试玩分包目录内，并配置manifest.json中试玩分包的参数，然后构建正式RPK包。第二种是直接在支持配置分包的游戏引擎下配置分包，在构建时可直接导入第一种方式内配置好的manifest.json参数进行构建。
