PGD辅助工具是基于PGD基础能力，帮助开发者在团结引擎中进行PGD开发的工具集，旨在降低ECS的使用门槛，提升开发者的日常开发效率。

该工具集包含可视化调试工具、性能监控工具、代码转换工具、构建工具，帮助开发者在ECS开发过程中实现高效调试、快速构建、性能优化和代码迁移。

## 工具简介

| 工具 | 说明 |
| --- | --- |
| [Relationship Window](https://developer.huawei.com/consumer/cn/doc/games-guides/pgd-tool-window-0000002526353161) | 专为PGD设计的可视化调试工具。  实时展示System、Component、Entity三者之间的逻辑依赖与数据变化。 |
| [PGD Profiler](https://developer.huawei.com/consumer/cn/doc/games-guides/pgd-tool-profiler-0000002526433123) | 集成于引擎Profiler中System维度的PGD数据监控工具。 |
| [ECS Converter](https://developer.huawei.com/consumer/cn/doc/games-guides/pgd-tool-converter-0000002494353474) | 将游戏工程的ECS框架转换为PGD框架的辅助转换工具。 |
| [PGD Editor](https://developer.huawei.com/consumer/cn/doc/games-guides/pgd-tool-editor-0000002494193498) | 以引擎编辑器为核心的PGD构建工具，其包含以下GameObject组件：   * PGD World：世界管理模块。 * PGD Entity：实体操作模块。 * PGD Systems：系统操作模块。 * PGD Spawner：Prefab管理和派生模块。 |
