本章节将为您详细介绍游戏多媒体服务在设备、配额方面的限制，这些限制可能会根据实际情况随时更新，请您注意文档变化。

## API 限制

API限制是指API调用的限制。

| 限制项 | 限制 |
| --- | --- |
| 接口调用 | * mutePlayer   + 如需屏蔽指定玩家A，需确保A玩家正在当前语音房间内。 * muteAllPlayers   + 屏蔽所有玩家后，不允许解除单个指定玩家的语音屏蔽。   + 如需屏蔽所有玩家语音，需确保已解除房间内其他所有玩家的语音屏蔽。 |

## 设备限制

设备限制是指游戏多媒体SDK对用户终端的限制。

| 限制项 | 限制 |
| --- | --- |
| 设备品牌 | 支持所有品牌设备 |
| HarmonyOS | HarmonyOS 5.0及以上（API Version 11） |
| 华为快游戏 | 平台版本号：1107及以上 |
| HMS Core APK | 无限制 |
| 应用市场客户端 | 无限制 |

## 配额限制

* 实时语音

  | 配额类型 | 配额 |
  | --- | --- |
  | 房间数 | 最多同时加入5个  说明：  + 国战语音或范围语音房间最多加入1个，且不支持同时加入。 + 在已加入范围语音房间的情况下，最多允许再加入一个小队语音房间。 |
  | 同时发言人数 | 最多17人/房间 |
  | 小队语音房间人数 | 最多1000人/小队语音房间 |
  | 国战语音房间-指挥官人数 | 最多15人/国战语音房间 |
  | 国战语音房间-群众人数 | 最多10000人/国战语音房间 |
  | 范围语音房间-同时具备开麦权限人数 | 最多500人/范围语音房间 |

* 实时信令

  | 配额类型 | 配额 |
  | --- | --- |
  | 频道人数 | 最多10000人/频道 |
  | 频道数 | 最多同时订阅50个 |
  | 频道内玩家属性可查询量 | 最多支持100人/单次 |
