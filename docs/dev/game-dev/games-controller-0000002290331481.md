Game Controller Kit（游戏控制器服务）支持游戏适配控制器外设（当前仅支持手柄），解决玩家操控性问题，保障用户体验。游戏开发者可通过接入该服务轻松实现游戏外设的上下线和按键及轴事件监听等功能。更多业务介绍及接入指导请参见[Game Controller Kit（游戏控制器服务）](/docs/dev/app-dev/application-services/game-controller-kit)。
