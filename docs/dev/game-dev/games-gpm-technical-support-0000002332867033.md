当您接入游戏性能管理服务遇到问题时，可以按照以下顺序寻求帮助。

1. 请先仔细阅读文档，部分功能对设备和系统有限制，请参见[使用限制](https://developer.huawei.com/consumer/cn/doc/games-guides/games-gpm-usage-restrictions-0000002298787836#ZH-CN_TOPIC_0000002348293636)。
2. 查看[返回码](https://developer.huawei.com/consumer/cn/doc/AppGallery-connect-References/gpm-gpmerror-return-code-0000001754407780#section1936544012342)。
3. 通过[智能客服](https://developer.huawei.com/consumer/cn/customerService/#/bot-dev-top/faq-top/faq-talk-top)查找问题解决方案。
4. 登录[华为开发者论坛](https://developer.huawei.com/consumer/cn/forum/block/ag-connect)和[Stack Overflow社区](https://stackoverflow.com/questions/tagged/appgallery-connect)参与问题讨论。
5. 如果以上方法仍未解决您的问题，可以通过[在线工单系统](https://developer.huawei.com/consumer/cn/support/feedback/#/add/7?level2=102&keyWord=其他)与我们进行联系。提交后的工单可在开发者联盟“管理中心”的“开发者中心 &gt; 我的客服”页面查看工单处理进展。
