## 批处理生命周期管理

```
// 错误：使用已归还的批处理
var batch = world.GetBatchBuilder(true);
batch.AddComponent(new Health(100)).CreateEntity(); // 调用CreateEntity创建成功后batch自动归还
batch.CreateEntity(); // 再次使用会抛出异常

// 正确：使用 GetBatchBuilder(false) 或重新获取
var batch1 = world.GetBatchBuilder(false);
batch1.AddComponent(new Health(100));
batch1.CreateEntity();
batch1.CreateEntity(); // 正常工作
batch1.ReleaseBatch(); // 手动归还

// 或者重新获取
var batch2 = world.GetBatchBuilder(); // 新的批处理实例
batch2.AddComponent(new Health(100)).CreateEntity();
```

## 批处理内容修改

```
// 错误：尝试获取未添加的组件
var batch = world.GetBatchBuilder(false);
try
{
    ref var health = ref batch.GetComponent<Health>(); // 抛出 InvalidOperationException
}
catch (InvalidOperationException ex)
{
    Console.WriteLine($"组件未添加到批处理: {ex.Message}");
}

// 正确：先添加组件再获取
batch.AddComponent<Health>();
ref var health = ref batch.GetComponent<Health>();
health.maximum = 100;
batch.CreateEntity();
batch.ReleaseBatch();
```

## 性能陷阱

```
// 低效：频繁创建新的批处理对象
for (int i = 0; i < 1000; i++)
{
    var batch = new BatchProcessor(); // 每次都分配新对象
    batch.AddComponent(new Health(100));
    batch.ProcessOn(someEntity);
}

// 高效：复用批处理对象
var reusableBatch = new BatchProcessor();
reusableBatch.AddComponent(new Health(100));

for (int i = 0; i < 1000; i++)
{
    reusableBatch.ProcessOn(entities[i]); // 复用同一个批处理
}
```
