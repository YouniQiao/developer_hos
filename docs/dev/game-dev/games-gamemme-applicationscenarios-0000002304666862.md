## 对战开黑

低延时、流畅的实时语音、小队语音等能力，为对战开黑的游戏玩家提供更好的语音体验。

## 频道聊天

多房间语音能力适用于如开放世界等游戏的世界、好友或社团频道发送语音消息。

## 国战指挥

低时延的国战语音等能力，非常适合国战指挥、团队副本等游戏玩法。

## 实时通信与游戏公告

即时性、低延迟的RTM能力，支持高质量的游戏信令交互和互动消息场景，可用于实时通信、全局聊天、指令同步等多种游戏场景。

* 多人游戏实时通信：用于多人游戏中的实时通信，例如在线合作游戏、多人对战游戏或社交游戏。它可以支持玩家之间的即时聊天和实时消息传递，从而增强玩家之间的互动和沟通。
* 游戏内通知和事件：用于向玩家发送实时通知、更新和事件。例如，可以使用RTM向玩家发送成就达成通知、好友请求、游戏更新提示等信息，以提升游戏的社交互动性。
* 游戏全局聊天和大厅系统：支持游戏内的全局聊天和大厅系统，使玩家可以在游戏内部建立实时的聊天室和群组，以进行广播消息、交流和社交。
