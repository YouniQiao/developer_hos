游戏集成游戏性能管理SDK后将在运行过程中采集性能数据，并上报至AGC控制台。您可以登录[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)， 查看已上报的游戏性能数据。

* **[游戏性能分析](https://developer.huawei.com/consumer/cn/doc/games-guides/games-gpm-game-performance-analysis-0000002332870901)**
* **[单用户查询](https://developer.huawei.com/consumer/cn/doc/games-guides/games-gpm-single-user-query-0000002332911097)**
* **[崩溃管理服务](https://developer.huawei.com/consumer/cn/doc/games-guides/games-gpm-break-0000002298812696)**
* **[功耗分析](https://developer.huawei.com/consumer/cn/doc/games-guides/games-gpm-power-consumption-analysis-0000002298973088)**
