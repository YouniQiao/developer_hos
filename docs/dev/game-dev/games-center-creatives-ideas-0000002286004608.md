魔方创意是一个在线制作、轻松上手、简单轻量的设计网站，为您提升营销物料的内容质量和传播频次。魔方创意支持一键预览，支持复杂的交互或跳转，您可以自由搭配组件，一键复用优质内容，轻松设计运营活动的可视化H5落地页，高效解决内容生产问题，大大节省了沟通、设计和人力成本。

## 编辑器界面

![](./img/fada83bd.png)

### 左侧组件导航

“魔方创意”网站的左侧组件导航会根据不同的服务展示不同的组件。

* 平台组件

  | 组件 | 说明 |
  | --- | --- |
  | 文本 | + 您需要填写文本内容。 + 您可以添加点击事件。 |
  | 富文本 | + 您可以在富文本中插入表格。 + 您可以添加点击事件。 |
  | 单图 | + 您可以替换组件中默认的图片。 + 您可以设置点击事件。 |
  | 音频 | 您仅能上传MP3、WMA格式的音频。 |
  | 轮播图 | + 您可以替换组件中默认的轮播图片和轮播箭头。 + 您可以为每张轮播图添加点击事件。 |
  | 热区 | 您可以在热区所覆盖的范围添加点击事件。 |
* 模板默认组件

  | 组件 | 说明 |
  | --- | --- |
  | 头部视频 | 您仅能上传MP4格式的视频。 |
  | 应用分发 | 无法重复添加。 |
  | 九宫格 | 仅对“九宫格”抽奖样式展示。您可以设置九宫格的样式、事件、弹窗等。 |
  | 翻牌 | 仅对“翻牌”抽奖样式展示。您可以设置翻牌的样式、事件、弹窗等。 |
  | 中奖榜单 | 仅对“九宫格”和“翻牌”抽奖样式展示。您可以设置榜单动画和样式。 |

### 头部菜单导航

| 组件 | 说明 |
| --- | --- |
| 语言切换 | 您仅能切换成“中文”或“English”。 |
| 更换模板 | 您可以重新选择页面模板。 |
| 页面设置 | 您需要设置全局样式，例如页面名称、活动封面等。 |
| 保存 | 您可以随时保存页面内容。 |
| 预览 | 您可以随时[预览页面内容](#section31105420225)。 |

### 中间编辑区域

您可以设置页面高度，调整组件位置等。

### 右侧属性面板

您可以设置不同组件的不同属性，例如样式、事件等。

### 底部页面弹框

您可以新增、复制、删除页面。

## 创建H5页面

1. 跳转至“魔方创意”网站后，请在自动弹出的“创建页面”窗口中选择页面：
   * 您可以使用网站提供的主题模板。
   * 您可以复制历史页面。
   * 您可以使用空白页面。

   ![](./img/f8ebc551.png)
2. 根据运营需求使用网站组件并设置对应的组件属性，完成H5落地页的设计后点击“保存”。

   ![](./img/c94edf9c.png)

## 预览H5页面

魔方创意提供多种方式预览：

* 您可以点击 ![](./img/f4b4bedb.png) 或 ![](./img/35974975.png) 直接查看手机模拟器中的展示效果。
* 您可以复制一键生成的链接，前往移动端查看展示效果。
* 您可以直接扫描二维码查看展示效果。

  ![](./img/d19ab3e1.png)
