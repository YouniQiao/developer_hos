## 定义标签

标签是实现ITag接口的空结构体。

```
// 基础标签定义
public struct PlayerTag : ITag { }
public struct EnemyTag : ITag { }
public struct DeadTag : ITag { }
public struct BossTag : ITag { }

// PGD内置的Inactive标签
// public struct Inactive : ITag { } // 已内置
```

## 添加标签

```
var entity = world.CreateEntity();

// 添加单个标签
entity.AddTag<PlayerTag>();
entity.AddTag<AliveTag>();

// 批量添加标签
entity.AddTags(ITags.Get<PlayerTag, AliveTag>());

// 创建实体时添加标签
var enemy = world.CreateEntity(
    new Health(50),
    ITags.Get<EnemyTag, AIControlledTag>()
);
```

## 检查标签

```
// 检查单个标签
bool isPlayer = entity.HasTag<PlayerTag>();
bool isAlive = entity.Tags.Has<AliveTag>();

// 检查多个标签
bool isPlayerAndAlive = entity.Tags.Has<PlayerTag, AliveTag>();

// 检查是否包含任意标签
var enemyTags = ITags.Get<EnemyTag, BossTag>();
bool isAnyEnemy = entity.Tags.HasAny(enemyTags);

// 检查是否包含所有标签
var allRequiredTags = ITags.Get<PlayerTag, AliveTag>();
bool hasAllTags = entity.Tags.HasAll(allRequiredTags);
```

## 移除标签

```
// 移除单个标签
entity.RemoveTag<DeadTag>();

// 批量移除标签
entity.RemoveTags(ITags.Get<PlayerTag, AliveTag>());
```

## 标签集合操作

```
// 创建标签集合
var playerTags = ITags.Get<PlayerTag, AliveTag>();
var enemyTags = ITags.Get<EnemyTag, AIControlledTag>();

// 添加标签到集合
var combinedTags = playerTags;
combinedTags.Add(enemyTags);

// 移除标签从集合
combinedTags.Remove(ITags.Get<AliveTag>());

// 标签数量
int tagCount = entity.Tags.Count;
Console.WriteLine($"实体有 {tagCount} 个标签");
```
