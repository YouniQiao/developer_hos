Query（查询）是PGD中用于查找和迭代实体的核心系统。它提供高效的方式来筛选具有特定组件和标签组合的实体，并对它们进行批量处理。

## 核心特性

* 类型安全：编译时确保组件类型的正确性。
* 高性能：基于Archetype的内存布局优化。
* 灵活过滤：支持复杂的组件和标签过滤条件。
* 流式API：链式调用构建复杂查询。
* 多种迭代方式：适合不同性能需求的迭代方法。
