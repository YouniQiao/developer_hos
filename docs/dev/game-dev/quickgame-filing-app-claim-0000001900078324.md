## 前提条件

已前往应用市场下载最新版本的华为云App。

## 操作步骤

1. 打开华为云App，搜索并点击“ICP备案”。

   ![](./img/3eace87a.png)
2. 填写待认领的主办单位和快游戏信息后，提交初审。

   ![](./img/536b46a5.png)
3. 华为工作人员将在1~2个工作日内完成审核，审核结果将以短信或邮件形式通知，在此期间请耐心等待。

   ![](./img/1582751c.png)
