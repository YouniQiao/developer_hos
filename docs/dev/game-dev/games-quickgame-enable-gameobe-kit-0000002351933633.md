若游戏内提供虚拟商品，前往AGC控制台创建消耗型商品或非消耗型商品，具体步骤请参见[创建游戏内商品](/docs/distribute/app-dist/game-center/game-center-operation-0000001239502315/game-center-goods-management-0000001194462390/game-center-creating-product-0000001239502323#section10697151616123)。

| 商品类型 | 说明 |
| --- | --- |
| 消耗型商品 | 商品使用后即刻失效，再次使用需要再次购买。 |
| 非消耗型商品 | 商品购买一次，永久有效。 |
