## 准备资质文件

首次申请上架的快游戏，需准备如下资质文件：

* 游戏版权
* 游戏版号

资质文件的详细要求请参见[游戏版权、版号要求](https://developer.huawei.com/consumer/cn/doc/distribution/app/80301#h1-1584931854487-2)。

![](./img/09faa14a.png)

* 若上传游戏开发者非版号批文中的运营单位，需提供版号授权书，且注明是对快游戏的授权（若有多重授权，则每一重授权都需注明）。
* 若上传游戏开发者非《计算机软件著作权证书》中著作权人，需提供版权授权书，且注明是对快游戏的授权（若有多重授权，则每一重授权都需注明）。
* 若有同名APK游戏在架，且共用一个版号，则快游戏内容需与APK游戏一致才可上架。

## 提交上架申请

前往AppGallery Connect[配置快游戏的基本信息](https://developer.huawei.com/consumer/cn/doc/app/agc-help--release-fastapp-0000001099836868#section19724459249)，并[发布快游戏](https://developer.huawei.com/consumer/cn/doc/distribution/app/agc-help--release-fastapp-0000001099836868)。
