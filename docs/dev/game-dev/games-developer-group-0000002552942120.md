本文约定了开发者及其关联公司在使用华为数字商品及联运服务（以下简称“服务”）时，通过开发者群组使用服务和开展活动的具体规则。

**一、定义**

1. 关联公司：与开发者使用服务时所签署协议中的用语具有相同含义。
2. 开发者群组：开发者及其所有关联公司主体拥有一个唯一的开发者群组，简称群组。开发者及其所有关联公司主体的账号需如实申请加入群组。
3. 关联：指开发者创建或加入对应的开发者群组。

**二、关联关系申请与应用**

1. 开发者可在华为平台上申请加入已有开发者群组，或新建开发者群组并加入。
2. 为保证激励准确发放，每个开发者主体必须关联一个开发者群组，一个开发者群组内可以仅有一个开发者主体。
3. 开发者群组关联需经过华为审核，审核通过后次月生效。

**三、结算**

1. 为保证激励发放的公平公正，同时确保正确计算开发者的流水总额、分成结算金额及激励金额，开发者及其所有关联公司主体需要在华为平台上，如实进行关联。若开发者未关联开发者群组，为保障开发者的激励正常发放，开发者的与使用服务相关的结算将暂时停止，开发者关联开发者群组并生效后将一并计算流水总额、分成结算金额及激励金额。
2. 华为允许开发者及其所有关联公司主体进行关联关系变更及App转移，转移相关的流水总额会被同时计入变更/转移前后的开发者群组，用于计算对应开发者群组的累计流水总额及相应激励。

**四、其他**

1. 华为会不定期审视开发者关联关系是否正确，对于关系不正确的情形，华为将提出整改意见并暂停开发者与使用服务相关的结算，直至开发者申诉成功或修正关联关系。
2. 对于通过恶意行为（包括但不限于掩盖关联公司事实捏造独立开发者群组）获取激励的情形，华为有权采取取消开发者的激励资格，并追回开发者不当获利等措施。华为的审核不影响或免除开发者的责任。
