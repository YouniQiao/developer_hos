## 索引同步问题

```
// 问题：手动修改组件值没有更新索引
var player = world.CreateEntity(new PlayerLevel { level = 10 });
ref var level = ref player.GetComponent<PlayerLevel>();
level.level = 20; // 直接修改不会更新索引！

// 解决：使用Set方法更新组件
player.Set(new PlayerLevel { level = 20 }); // 正确，会自动更新索引
```

## 类型匹配

```
// 错误：类型不匹配
// var wrongQuery = world.Query<PlayerLevel>().HasValue<PlayerLevel, string>("25");

// 正确：确保值类型匹配
var correctQuery = world.Query<PlayerLevel>().HasValue<PlayerLevel, int>(25);
```
