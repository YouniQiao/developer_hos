## 前提条件

已在开发者联盟网站上[注册成为开发者](/docs/dev/game-dev/ailod-agc-0000002477235912#section61391351162414)。

## 操作步骤

1. 在顶部菜单栏选择“AILOD &gt; Settings”，点击“Sign in”。

   ![](./img/74928e77.png)
2. 在弹出的“华为账号登录”页面，输入已实名认证的华为账号，完成后点击“授权并登录”。

   ![](./img/dd686c17.png)
3. 登录成功后，请携带**Open Id**信息联系运营（QQ：2851508860）申请使用权限。

   后续再使用相同华为账号登录AILOD时，**Open Id**不会变，您无需重复联系运营申请权限。

   ![](./img/f47510dd.png)
