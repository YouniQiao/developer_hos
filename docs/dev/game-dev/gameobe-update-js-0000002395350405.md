通过本章的指导开发，您可实现获取房间最新信息的功能。

## 前提条件

玩家已进入房间。

## 开发步骤

调用[Room.update](https://developer.huawei.com/consumer/cn/doc/games-references/gameobe-room-js-0000002395195985#section784813131994)方法获取房间最新信息。

```
global.room.update().then(() => {
  // 获取玩家房间最新信息成功，做相关的游戏处理逻辑
}).catch(() => {
  // 获取玩家房间最新信息失败
});
```
