特殊实体是具有唯一标识符的实体，用于表示游戏中的重要对象，如玩家等。

```
// 创建特殊实体
var player = world.CreateEntity(new SpecialEntity{ uid = "Player"});

// 通过标识符获取特殊实体
var retrievedPlayer = world.GetSpecialEntity("Player");
Console.WriteLine($"玩家实体: {retrievedPlayer}");

// 获取所有特殊实体
foreach (var specialEntity in world.SpecialEntities)
{
    var specialComp = specialEntity.GetComponent<SpecialEntity>();
    Console.WriteLine($"特殊实体: {specialComp}");
}
```
