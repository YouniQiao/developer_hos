《协助修改核准（备案）信息在线服务条款》是华为软件技术有限公司为了帮助您在满足核准（备案）主体所在省市通信管理局审核要求的前提下，尽快完成核准（备案），避免反复退回修改，提高核准（备案）效率而提供的免费服务。当您选择同意《协助修改核准（备案）信息在线服务条款》，我们将按照条款协助您修改提交有误的核准（备案）信息。

协助修改核准（备案）信息在线服务条款：

在以下条款中，华为软件技术有限公司提供协助更改核准（备案）信息服务的核准（备案）主体类型，包括自然人、个人合伙和个体工商户、公司、企业、合伙企业和事业单位等。您同意此服务条款的效力如同您亲自签字、盖章的书面条款一样，对您具有法律约束力。

一、华为软件技术有限公司协助修改核准（备案）信息服务，仅限于您提交的电子版资料内容有明显错误的场景（如：您提交的网站核准（备案）信息中主办单位地址填写为：北京市东城区；但电子版有效证件上地址为：北京市西城区）；对于其他可能存在的错误信息，属于华为软件技术有限公司无法直接修改的信息，不予协助修改，但将及时联系您确认修改或将您的核准（备案）信息驳回自行修改。

二、选择此服务后，核准（备案）信息在"审核中状态"，如需要修改信息，您需要及时联系华为修改或自行撤回修改。

三、选择此服务后，您需要及时对所提交的核准（备案）信息、状态进行跟进，确认，如因您未及时更新核准（备案）信息而影响了核准（备案）进度，您理解并同意责任由您自行承担。

四、本服务不保证您的核准（备案）申请能够在核准（备案）主体所在省市通信管理局审核通过，为节约核准（备案）时间，华为建议您在提交核准（备案）信息时，应尽力做到信息的真实、有效、完整。

五、您提交所有核准（备案）信息（包括文字，图片，证件等），华为将按照国家监管要求提交到核准（备案）主体所在省市通信管理局，只用作华为核准（备案）使用，华为不会泄露您提交的核准（备案）信息。

请您确保已经完全阅读并理解了上述协助修改核准（备案）信息在线服务条款，并自愿正式进入协助更改核准（备案）信息申请程序，接受上述所有条款的约束。
