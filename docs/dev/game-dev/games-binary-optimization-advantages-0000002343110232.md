## 过程安全可靠

编译器优化技术，通过收集分析程序运行中的性能数据，进行程序so文件二进制指令重排，达到提高程序运行性能的目的，过程安全可靠。

## 负载降低显著

基于程序的局部性原理和CPU本身的特点，通过指令重排有效降低CPU的FE BOUND并提升IPC，在游戏重载情况下，可以显著降低整机功耗。
