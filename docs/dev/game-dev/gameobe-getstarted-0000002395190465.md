项目是您在AppGallery Connect（以下简称AGC）资源的组织实体，您可以将一个应用的不同平台版本添加到同一个项目中。当您的应用需要使用联机对战服务时，您可以[创建您的项目](https://developer.huawei.com/consumer/cn/doc/app/agc-help-create-project-0000002242804048)和[创建您的应用](https://developer.huawei.com/consumer/cn/doc/app/agc-help-create-app-0000002247955506)。

![](./img/20c2a5ff.png)

如果您需要开发一个小游戏，只需要创建一个Web类型应用，获取配置信息用于后续的初始化。
