AILOD脚本位于Editor目录下，核心实现逻辑存放于EditorWindow中。AILOD具备如下使用限制：

* AILOD采用端云协同工作方式，因此使用期间需要保持网络畅通。
* AILOD作为开发阶段工具，不会参与游戏应用包构建。
