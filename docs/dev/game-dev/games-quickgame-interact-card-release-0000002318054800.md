打包测试通过后，前往AGC控制台发布上架，为用户提供创新互动卡片服务。

## 了解上架规则

认真阅读[华为应用市场审核指南](https://developer.huawei.com/consumer/cn/doc/app/50104-overview)和[游戏审核FAQ](/docs/distribute/app-dist/app-market/x50000/x50118)，了解华为应用市场的上架规则。

## 提交上架申请

前往AGC控制台填写信息并提交上架申请，具体操作请参见[发布应用（APP）](https://developer.huawei.com/consumer/cn/doc/app/agc-help-releaseharmony-0000001933963166)和[审核与发布](/docs/dev/game-dev/games-quickgame-playable-subpackage-0000002351893469#section229320301422)。
