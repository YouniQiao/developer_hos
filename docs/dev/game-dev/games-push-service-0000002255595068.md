Push Kit（推送服务）是华为提供的消息推送平台，建立了从云端到终端的消息推送通道。所有HarmonyOS应用可通过集成Push Kit，实现向应用实时推送消息，使消息易见，构筑良好的用户关系，提升用户的感知度和活跃度。更多介绍及接入指导请参见[Push Kit（推送服务）](/docs/dev/app-dev/application-services/push-kit-guide)**。**
