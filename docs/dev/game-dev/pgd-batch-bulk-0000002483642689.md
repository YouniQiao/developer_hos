对于查询结果集、EntitySet等实体集合使用ProcessBatchOnAll来进行批处理操作。

## 查询结果批处理

为查询结果集中的所有实体进行批处理操作。

```
// 为所有敌人添加愤怒状态
var enemyQuery = world.Query<Health>().WithAllTags(ITags.Get<EnemyTag>());

var angerBatch = new BatchProcessor();
angerBatch.AddComponent(new AngerState { level = 5 })
          .AddTag<AngryTag>()
          .RemoveTag<CalmTag>();

// 对查询结果中的所有实体进行批量修改操作
enemyQuery.Entities.ProcessBatchOnAll(angerBatch);

Console.WriteLine($"已激怒的敌人数量: {enemyQuery.EntityCount}");
```

## 全数据域实体批处理

```
// 对整个world的所有实体进行批处理操作
var globalBatch = new BatchProcessor();
globalBatch.AddComponent(new TimestampComponent { value = DateTime.Now })
           .AddTag<ProcessedTag>();

world.Entities.ProcessBatchOnAll(globalBatch);
Console.WriteLine("已为所有实体添加时间戳");
```

## EntitySet批处理

对EntitySet中的所有实体进行批处理。

```
var entities = new EntitySet(world); // 创建实体集合

// 添加一些实体到集合中
var boss = world.CreateEntity(new Health(500), ITags.Get<BossTag>());
var minion1 = world.CreateEntity(new Health(100), ITags.Get<MinionTag>());
var minion2 = world.CreateEntity(new Health(100), ITags.Get<MinionTag>());

entities.Add(boss);
entities.Add(minion1);
entities.Add(minion2);

// 为实体集合中的所有实体都进行强化效果处理
var enhanceBatch = new BatchProcessor();
enhanceBatch.AddComponent(new BuffComponent { type = BuffType.Strengthen, duration = 30.0f })
            .AddTag<EnhancedTag>();

entities.ProcessBatchOnAll(enhanceBatch); // 调用ProcessBatchOnAll对所有实体进行批处理
Console.WriteLine($"已强化 {entities.Count} 个实体");
```
