游戏需要适配多种屏幕尺寸，当前游戏支持的主要产品形态包括手机、折叠屏和平板三种。通过监听断点变化，动态调整页面布局和元素。开发者可以从ArkTS侧获取断点值，并传入Native侧，使Native侧根据不同的断点值适配不同的页面及效果。更多内容请参见[Native一多适配](/docs/dev/app-dev/multi-device/bpta-multi_game#section181610762911)。
