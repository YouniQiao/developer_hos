证书指纹可以保证快游戏的完整性和安全性。快游戏在游戏引擎中生成证书指纹，详情请参见[生成并配置证书指纹](https://developer.huawei.com/consumer/cn/doc/games-guides/games-quickgame-environment-preparation-0000002351893453#section6561155114119)，生成后在AGC控制台进行配置。
