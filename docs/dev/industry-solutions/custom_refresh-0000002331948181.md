## 场景介绍

下拉刷新和上拉加载效果自定义是美食类应用高频使用的场景之一，如刷新首页推荐内容、加载更多新内容。

本示例基于[Refresh](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-refresh)实现下拉刷新效果，基于[WaterFlow](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-waterflow)实现上拉加载效果。

## 实现思路

1. 下拉页面时，通过设置[Refresh](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-refresh)的builder自定义刷新区域显示内容，通过reload()实现刷新。

   ```
   Refresh({ refreshing: $$this.refreshing, builder: this.refreshBuilder() }) {
   }
   .onRefreshing(() => {
     setTimeout(() => {
       this.dataSource.reload();
       this.refreshing = false;
     }, Constants.DELAY_TIME);
   });
   ```
2. 滚动到[WaterFlow](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-waterflow)底部时，展示footer()中自定义的加载内容，通过addLastItem()实现加载。

   ```
   WaterFlow() {
     FlowItem() {
       this.footer();
     };
   }
   .onReachEnd(() => {
     if (!this.isLoading) {
       this.isLoading = true;
       setTimeout(() => {
         for (let i = 0; i < Constants.ONCE_COUNT; i++) {
           this.dataSource.addLastItem();
         }
         this.isLoading = false;
       }, Constants.DELAY_TIME);
     }
   })
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
