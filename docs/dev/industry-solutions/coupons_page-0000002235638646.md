## 场景介绍

优惠券卡包是购物比价类应用中高频使用场景之一。

本示例基于[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)构建常用的优惠券卡包页面样式，提供当前可使用和已过期优惠券两个页面。

## 实现思路

1. 通过[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)和[ForEach](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-rendering-control-foreach)创建列表项，完成优惠卡券布局。

   ```
   List() {
     ForEach(this.couponCategory ? this.coupons : this.expiredCoupons, (info: CouponInfo) => {
       ListItem() {
         Row({ space: 18 }) {
           // 卡券布局
         }
       }
     })
   }
   ```

2. 使用[border](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-border#border)属性来绘制单侧虚线，实现卡券效果。

   ```
   .border({
     width: {
       start: LengthMetrics.vp(0),
       end: LengthMetrics.vp(1),
       top: LengthMetrics.vp(0),
       bottom: LengthMetrics.vp(0)
     }
   })
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
