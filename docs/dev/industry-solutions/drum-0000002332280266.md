## 场景介绍

电子鼓是影音娱乐类应用的高频使用场景之一，如用户可以点击鼓面进行演奏。

本示例基于[SoundPool](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-multimedia-soundpool#soundpool)、[ohos.vibrator](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-vibrator)实现音振协同功能，并通过监听点击事件触发音频播放和振动效果。

## 实现思路

1. 导入振动的json文件，内部参数用于控制振动的幅度、时长、频率。

   ```
   "Pattern": [{
     "Event": {
       "Type": "transient",
       "StartTime": 0,
       "Parameters": {
         "Frequency": 55,
         "Intensity": 75
       }
     },
     // ...
   }]
   ```
2. 传入对应音频文件和振动的json文件并调用方法[SoundPool](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-multimedia-soundpool#soundpool)、[ohos.vibrator](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-vibrator)实现音振协同功能。

   ```
   // 静态播放方法
   public static async playSound(uri: string, volume: number = 1, instance: SoundPoolUtil): Promise<number | undefined> {
     try {
       // 确保SoundPool已初始化
       if (!instance.soundPool) {
         await instance.init();
       }

       // 加载并播放声音
       await instance.loadSound(uri);
       const params: media.PlayParameters = {
         loop: 0,
         rate: audio.AudioRendererRate.RENDER_RATE_NORMAL,
         leftVolume: volume,
         rightVolume: volume,
         priority: 1
       };
       const result = await instance.play(uri, params);
       // ...
       return result;
     } catch (error) {
       // ...
       return undefined;
     }
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。

## 权限说明

获取振动权限：[ohos.permission.VIBRATE](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissionvibrate)。
