## 场景介绍

距离过大报警器是儿童教育类应用中的典型场景之一，如监护人可通过距离报警器实时掌握孩子离家距离，孩子超出安全范围时触发弹窗警告，从而降低孩子走失或发生意外的风险。

本示例使用[获取设备的位置信息开发指导](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/location-guidelines)实时更新坐标，通过[地图计算工具](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/map-calculation-tool)实现距离计算。

## 实现思路

1. 通过[获取设备的位置信息开发指导](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/location-guidelines)注册位置变化监听回调，在每次定位回调时通过[地图计算工具](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/map-calculation-tool)计算当前位置离家距离。

   ```
   onLocationChange(): void {
     try {
       // 注册位置变化监听
       geoLocationManager.on('locationChange', request, this.locationChange);
       ...
     }
   }
   locationChange = (location: geoLocationManager.Location): void => {
     let location1 = this.homeLocation... // 家的坐标
     let location2 = this.nowLocation... // 当前位置坐标
     this.distanceToHome = this.haversine(location1, location2)
   }
   ```
2. 超过安全距离时弹出警告。

   ```
   this.isSafe = this.distanceToHome < this.distanceAlarm
   if (!this.isSafe) {
     let message = `...`
     this.getUIContext().getPromptAction().showToast({
       message: message,
       duration: 2000
     });
   }
   ```

本示例需要[开通地图服务](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/map-config-agc#section16133115441516)，并进行相应配置，具体步骤如下：

1. 登录AppGallery Connect网站，选择“我的项目”。
2. 在项目列表中找到您的项目，在项目下的应用列表中选择需要打开“地图服务”的应用。
3. 选择API管理，找到“地图服务”开关，打开开关。
4. 确认已经开启“地图服务”开放能力，并完成[手动签名](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-signing#section297715173233)。

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。

## 权限说明

* 获取位置权限：[ohos.permission.LOCATION](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all-user#ohospermissionlocation)。
* 获取Internet网络权限：[ohos.permission.INTERNET](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissioninternet)。
* 获取后台位置权限：[ohos.permission.LOCATION\_IN\_BACKGROUND](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all-user#ohospermissionlocation_in_background)。
* 获取设备模糊位置权限：[ohos.permission.APPROXIMATELY\_LOCATION](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all-user#ohospermissionapproximately_location)。
* 获取设备后台运行权限：[ohos.permission.KEEP\_BACKGROUND\_RUNNING](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissionkeep_background_running)。
