## 场景介绍

分时线与日K线是理财保险类应用中高频使用场景之一，用于查看股票的短期走势和中期趋势。

本示例基于[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)组件绘制了分时图与日K图，实现了分时线与日K线的动态刷新，以及日K线的放大、缩小、左移与右移，也适用于周K线、月K线等相关场景。

## 实现思路

* 分时线实现思路：
  1. 使用[@ohos.resourceManager](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-resource-manager)的[getRawFileContent](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-resource-manager#getrawfilecontent9-1)方法读取rawfile文件夹下的模拟数据文件。

     ```
     let timeLineValue = await this.uiContext.resourceManager.getRawFileContent('MockTimeLineData.json');
     ```
  2. 使用[CanvasRenderingContext2D](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-canvasrenderingcontext2d)在[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)组件上进行绘制，使用[@Watch装饰器](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-watch)监听分时线数据，完成分时线图像（折线图与柱状图）的绘制。

     ```
     // 分时数据
     @Prop @Watch('redrawCanvas') data: TimeLineData[];
     // 重新绘制画布上的图表内容
     private redrawCanvas() {
       if (!this.context || this.canvasWidth === 0 || this.canvasHeight === 0) {
         return;
       }
       this.calculatePriceRange();
       // ...
       // 清空画布
       ctx.clearRect(0, 0, width, height);
       // ...
       // 绘制分时图部分
       this.drawMainChart(ctx);
       // 绘制成交量部分
       this.drawVolumeChart(ctx);
     }
     ```

* 日K线实现思路：
  1. 使用[@ohos.resourceManager](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-resource-manager)的[getRawFileContent](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-resource-manager#getrawfilecontent9-1)方法读取rawfile文件夹下的模拟数据文件。

     ```
     let dailyKLineValue = await this.uiContext.resourceManager.getRawFileContent('MockDailyKLineData.json');
     ```
  2. 使用[CanvasRenderingContext2D](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-canvasrenderingcontext2d)在[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)组件上进行绘制，使用[@Watch装饰器](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-watch)监听日K线数据，完成日K线图像（蜡烛图与柱状图）的绘制。

     ```
     // 当前显示的数据
     @Prop @Watch('onDataChange') data: KLineData[];
     // 数据变化时更新相关变量
     private onDataChange() {
       // 检查是否有新数据加入
       let hasNewData = this.data.length > this.previousDataLength;

       // 如果当前正在查看最新数据且有新数据，则更新显示
       if (this.isViewingLatest && hasNewData) {
         this.leftOffset = Math.max(0, this.data.length - this.displayCount);
       }

       this.previousDataLength = this.data.length; // 更新记录的数据长度
       this.updateDisplayData(this.isViewingLatest, false);
     }
     private updateDisplayData(autoUpdate: boolean, isZoom: boolean) {
       // 在查看历史数据或非放缩操作时，无需更新
       if (!autoUpdate && !isZoom) {
         return;
       }

       let startIndex = Math.max(0, this.leftOffset);
       let endIndex = Math.min(this.data.length - 1, startIndex + this.displayCount - 1);
       this.displayData = this.data.slice(startIndex, endIndex + 1);

       this.checkViewingLatest(); // 更新查看状态
       this.calculatePriceRange();
       this.calculateVolumeRange();
       this.calculateCandleWidth();
       this.redrawCanvas();
     }
     ```
  3. 实现蜡烛图的放大、缩小、左移与右移，详细请在代码中查看。

     ```
     // 蜡烛图放大函数，通过减少显示数据量的方式
     ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
