## 场景介绍

日历周视图、月视图切换是便捷生活类应用的高频使用场景之一，如用户可通过滑动手势在不同视图间切换。

本示例基于[Swiper](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-swiper)组件和[PanGesture](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-gestures-pangesture)实现日历在周、月以及详情视图间切换的功能。

## 实现思路

1. 使用[resourceManager](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-resource-manager)读取模拟数据文件，并初始化日历月视图、周视图数据。

   ```
   aboutToAppear(): void {
     let context = this.getUIContext().getHostContext();
       if (context) {
         this.data = RawfileUtil.readDataFromRawfile(context.resourceManager);
         this.monthRanges = CalendarUtil.getCalendarRange(this.data, CalendarViewType.MONTH);
         this.weekRanges = CalendarUtil.getCalendarRange(this.data, CalendarViewType.WEEK);
       }
   }

   Swiper(this.swiperController) {
     ForEach(this.viewType === CalendarViewType.WEEK ? this.weekRanges : this.monthRanges,
       (range: CalendarViewRange, index: number) => {
         CalendarView({
           viewType: this.viewType,
           monthData: this.data,
           range: range,
           // 其他传入参数
         });
     );
   }
   .index($$this.currentIndex);
   ```
2. 在日历视图组件上绑定[PanGesture](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-gestures-pangesture)滑动手势，实现上下滑动时月视图、周视图切换动画。

   ```
   Column() {
     // 日历视图组件
   }
   .gesture(
     PanGesture({ direction: PanDirection.Vertical })
       .onActionStart((event: GestureEvent) => {
         if (this.viewType === CalendarViewType.MONTH) {
           if (event.offsetY > PAN_GESTURE_OFFSET) {
             this.changeToDetailView();
           } else if (event.offsetY < -PAN_GESTURE_OFFSET) {
             this.changeToWeekView();
           }
         } else if ((this.viewType === CalendarViewType.DETAIL && event.offsetY < -PAN_GESTURE_OFFSET) ||
           (this.viewType === CalendarViewType.WEEK && event.offsetY > PAN_GESTURE_OFFSET)) {
           this.changeToMonthView();
         }
       })
   );
   ```
3. 视图切换后，因为数据源也随之发生变化，需要重新定位[Swiper](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-swiper)组件索引值到选中日期所在周或月，以实现平滑过渡效果。

   ```
   @State @Watch('refreshCalendarData') viewType: CalendarViewType = CalendarViewType.MONTH;

   refreshCalendarData() {
     let ranges = this.viewType === CalendarViewType.WEEK ? this.weekRanges : this.monthRanges;
     for (let i = 0; i < ranges.length; i++) {
       let monthRange = ranges[i];
       if (this.viewType === CalendarViewType.WEEK) {
         if (this.selectedWeekIndex === monthRange.weekViewIndex) {
           // 切换视图数据源后，重新定位索引到选中日期所在周
           this.currentIndex = i;
           // 调整月份范围为选中日期所在月
           // 便于在同一周跨月份时，实现周视图切换至月视图时平滑展开的动画效果
           CalendarUtil.adjustMonthRange(monthRange, this.data[this.selectedWeekIndex][this.selectedDayIndex]);
           break;
         }
     } else if (CalendarUtil.isWeekInMonthRange(this.selectedWeekIndex,
       this.data[this.selectedWeekIndex][this.selectedDayIndex], monthRange)) {
         // 切换视图数据源后，重新定位索引到选中日期所在月
         this.currentIndex = i;
         break;
       }
     }
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
