## 场景介绍

防沉迷控制是儿童教育类应用中的高频使用场景之一。

本示例通过控制应用使用时长，超时后锁定应用实现儿童防沉迷。

## 实现思路

1. 使用定时器[Timer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-timer)控制应用使用时长，并在页面退出时清除定时器。
2. 使用[TextTimer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-texttimer)显示休息时长，倒计时结束恢复使用。
3. 使用时长结束后，应用页面加锁，休息时长结束解锁。

```
// 配置定时器控制使用时长
setTimeout();
// 定时结束，立即取消
clearTimeout(this.returnTimerId);
// 获取当前时间秒数，便于设置使用倒计时
GetDateTime.getDateStringWithTimeStamp();
// TextTimer设置休息倒计时
TextTimer.onTimer();
// 锁定时触发休息倒计时
textTimerController.start();
```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
