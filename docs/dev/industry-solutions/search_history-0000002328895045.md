## 场景介绍

搜索及编辑搜索历史是购物比价类应用中高频使用场景之一，如用户需要购买指定商品，在搜索框中输入该商品，搜索后将本次搜索的商品自动添加至搜索历史，便于用户查看。

本示例基于[TextInput](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-textinput)组件和[ForEach](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-rendering-control-foreach)遍历实现搜索、添加或删除搜索历史记录的功能。

## 实现思路

1. [TextInput](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-textinput)组件实现自定义搜索栏，输入内容实时接收，通过[onChange](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-textinput#onchange)方法监听输入内容并进行处理。

   ```
   TextInput()
     .onChange((currentContent) => {
       this.currentSearchContent = currentContent;
       if (this.currentSearchContent.length !== Constants.ZERO_LENGTH) {
           // ...
       }
     });
   ```
2. 点击删除按钮，搜索历史进入删除模式，点击单个历史记录词语进行删除，点击完成后退出删除模式。

   ```
   ForEach(this.historyWords, (item: HistoryWordModel, index) => {
     Text()
       .onClick(() => {
         if (this.isDeleteState) {
            // ...
         }
       });
   });
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
