## 场景介绍

本示例基于[Stack](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-stack)层叠布局和组合手势实现文本局部放大效果，支持放大镜随触摸位置移动，通过放大镜辅助阅读，以满足更多用户阅读需求。

## 实现思路

1. 使用[Stack](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-stack)将放大镜展示层隐藏于文本上方，当用户长按后，截取当前阅读内容并等比例放大。
2. 通过获取用户触摸位置，完成放大镜和截取内容的移动，实现放大效果。

```
// 当长按之后进行拖动，PanGesture手势被触发
PanGesture()
  .onActionUpdate((event: GestureEvent | undefined) => {
    if (event) {
      this.posX = event.fingerList[0].localX;
      this.posY = event.fingerList[0].localY;
    }
  });
// 放大镜
if (this.isTouch && this.copyStyle === CopyOptions.None) {
  Row() {
    Image(this.pixmap)
      .height(this.uiContext.px2vp(this.pixmapH))
      .width(this.uiContext.px2vp(this.pixmapW))
      .translate({ x: (79 - this.posX) * 1.2, y: (332 - this.posY) * 1.2 }); // 放大镜视图的尺寸参数
  }
    .margin({
      left: this.posX - 79,
      top: this.posY - 165 // 放大镜宽度相关
    });
}
```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
