## 场景介绍

本示例基于[Reader Kit](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/reader-api)和[@ohos.file.fs](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-fs)实现将本地书籍文件导入书架的功能，同时支持编辑、删除、搜索及查看历史记录等功能。用户可将本地书籍文件导入书架阅览，并进行删除、搜索等操作。

## 实现思路

1. 使用[DocumentViewPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-picker#documentviewpicker)选择用户本地书籍文件。

   ```
   let documentSelectOptions = new picker.DocumentSelectOptions();
   documentSelectOptions.fileSuffixFilters = ['.txt'];
   let documentPicker = new picker.DocumentViewPicker();
   let documentSelectResult = await documentPicker.select(documentSelectOptions);
   ```
2. 获取选中文件信息。

   ```
   let srcPath: string = documentSelectResult[0];
   let file: fs.File|undefined = undefined ;
   let arrayBuffer: ArrayBuffer | undefined = undefined
   try {
     file = fs.openSync(srcPath, fs.OpenMode.READ_WRITE); // 根据需求选择模式
     let fileSize: number = 0
     fileSize = fs.statSync(file.fd).size
     arrayBuffer = new ArrayBuffer(fileSize)
     fs.readSync(file.fd, arrayBuffer)
   } catch (error) {
   } finally {
     fs.closeSync(file)
   }
   ```
3. 将文件信息读入沙箱。

   ```
   let segments = srcPath.split('/').filter(s => s.length > 0);
   let encodedFileName = segments[segments.length - 1];
   let fileName = decodeURI(encodedFileName)
   let destFile: fs.File| undefined = undefined;
   try {
     // 拷贝文件到沙箱
     destFile =
       fs.openSync(this.context.filesDir + `/${fileName}`, fs.OpenMode.CREATE | fs.OpenMode.READ_WRITE)
     if (arrayBuffer) {
       fs.writeSync(destFile.fd, arrayBuffer)
     }
   } catch (err) {
   } finally {
     fs.closeSync(destFile)
   }
   ```
4. 使用[Reader Kit](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/reader-api)能力显示书籍内容。

   ```
   let fileName: string[] = this.pathStack.getParamByName('ReadPage') as string[]
   let filePath: string =  this.context.filesDir + `/${fileName[0]}`
   let domPos: string = '';
   this.startPlay(filePath, domPos);
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
