## 场景介绍

雷达扫描是实用工具类应用的高频场景之一，如展示扫描周边蓝牙设备或扫描周围用户的动画。

本示例通过[transition](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-transition-animation-component)实现圆形波纹效果以及扫描结果逐渐显示效果，使用[animateTo](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#animateto)实现绕圈扫描动画效果。

## 实现思路

1. 通过[transition](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-transition-animation-component)实现圆形波纹效果，在组件消失的时候触发TransitionEffect.OPACITY转场，使组件变大，透明度变高。
2. 使用[animateTo](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#animateto)实现绕圈扫描效果，通过更改rotate属性实现组件旋转，通过iterations: -1实现动画无限循环。
3. 通过transition实现扫描结果逐渐显示效果，在组件出现的时候触发TransitionEffect.OPACITY转场，使组件透明度逐渐变低。

   ```
   Stack() {
     if (this.circleFlag) {
       // 圆形波纹效果
       this.scanCircle();
     }
     // 扫描结果显示效果
     this.scanResults();
     // 绕圈扫描效果
     this.scanEffect();
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
