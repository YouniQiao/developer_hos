## 问题现象

应用主界面为竖屏状态，点击进入某个横屏页面后，使用侧边返回退到主界面仍为横屏状态。

## 背景知识

[屏幕侧的rotation](https://developer.huawei.com/consumer/cn/doc/best-practices/bpta-multi-device-window-direction#section15598121101615)：显示设备的屏幕顺时针旋转角度。

## 问题定位

根据Hilog日志进行问题排查。

1. 在Hilog日志中检索关键字displayOrientation，发现有如下日志信息：


   此时说明侧滑动作已经响应，用户重写了onBackPress方法。继续查看后续的日志，发现没有设置窗口方向的相关日志。此时，窗口保持旋转为270°的状态。

## 分析结论

应用在响应用户操作，将屏幕旋转后。重写onBackPress方法的逻辑中，没有将屏幕重新旋转回正常的状态。

## 修改建议

在onBackPress方法中，设置屏幕旋转，将其调整回正常的状态。参考如下示例代码：

```
setOrientation(type: number) {
  try {
    window.getLastWindow(this.context, (err, data) => {
      //获取window实例
      if (err.code) {
        return;
      }
      let windowClass = data;
      try {
        windowClass.setPreferredOrientation(type, (err) => {
          if (err.code) {
            return;
          }
        });
      } catch (exception) {
      }
    });
  } catch (exception) {
  }
}
```

```
onBackPress(): boolean | void {
  this.setOrientation(window.Orientation.PORTRAIT);
  return false;
}
```

完整代码：

```
import { window } from '@kit.ArkUI';
import { common } from '@kit.AbilityKit';

@Entry
@Component
struct BackPressDemo {
  @State message: string = 'Hello World';
  private context = this.getUIContext().getHostContext() as common.UIAbilityContext;

  setOrientation(type: number) {
    try {
      window.getLastWindow(this.context, (err, data) => {
        //获取window实例
        if (err.code) {
          return;
        }
        let windowClass = data;
        try {
          windowClass.setPreferredOrientation(type, (err) => {
            if (err.code) {
              return;
            }
          });
        } catch (exception) {
        }
      });
    } catch (exception) {
    }
  }

  onBackPress(): boolean | void {
    this.setOrientation(window.Orientation.PORTRAIT);
    return false;
  }

  aboutToAppear(): void {
    this.setOrientation(window.Orientation.LANDSCAPE);
  }

  aboutToDisappear(): void {
    this.setOrientation(window.Orientation.PORTRAIT);
  }

  build() {
    RelativeContainer() {
      Text(this.message)
        .fontSize($r('app.float.page_text_font_size'))
        .fontWeight(FontWeight.Bold)
        .alignRules({
          center: { anchor: '__container__', align: VerticalAlign.Center },
          middle: { anchor: '__container__', align: HorizontalAlign.Center }
        })
        .onClick(() => {
          this.message = 'Welcome';
        })
    }
    .height('100%')
    .width('100%')
  }
}
```
