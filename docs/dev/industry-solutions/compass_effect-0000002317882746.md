## 场景介绍

指南针是实用工具类应用中的典型场景之一。

本示例通过[@ohos.sensor](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-sensor)、[@ohos.geoLocationManager](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-geolocationmanager)获取设备方向、大气压、海拔、磁场、经纬度等数据实现指南针效果，当设备移动时，指针始终指向设备所在位置的北方。

## 实现思路

1. 获取方向传感器数据：调用[sensor.on](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-sensor#sensoron)方法监听方向传感器[sensor.SensorId.ORIENTATION](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-sensor#sensorid9)，获取转动角度[sensor.OrientationResponse.alpha](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-sensor#orientationresponse)。

   ```
   // Subscribe to sensor data
   public getAngle() {
     try {
       sensor.on(
         sensor.SensorId.ORIENTATION,
         this.angleChange,
         {
           interval: Constants.SENSOR_INTERVAL
         });
     } catch (err) {
       Logger.error('sensor error: ' + err);
     }
   }
   // The callback for orientation sensor event data
   public angleChange = (data: sensor.OrientationResponse) => {
     this.angle = Math.round(data.alpha);
     this.notifyAngleChanges();
   }
   ```
2. 获取大气压、海拔数据：监听气压传感器[sensor.SensorId.BAROMETER](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-sensor#sensorid9)获取当前气压，调用[sensor.getDeviceAltitude](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-sensor#sensorgetdevicealtitude9)方法根据气压值获取当前设备海拔数据。

   ```
   // 获取大气压
   public getPressure() {
     try {
       sensor.on(sensor.SensorId.BAROMETER, (data: sensor.BarometerResponse) => {
         this.pressure = data.pressure
         this.notifyPressureChanges()
         this.getAltitude()
       }, { interval: 100000000 });
     } catch (error) {
       let e: BusinessError = error as BusinessError;
       Logger.error(`Failed to invoke on. Code: ${e.code}, message: ${e.message}`);
     }
   }
   // 获取海拔
   public getAltitude() {
     try {
       let seaPressure = 1013.2;
       let currentPressure = this.pressure;
       sensor.getDeviceAltitude(seaPressure, currentPressure, (err: BusinessError, data: number) => {
         if (err) {
           Logger.error(`Failed to get altitude. Code: ${err.code}, message: ${err.message}`);
           return;
         }
         this.altitude = data
         this.notifyAltitudeChanges()
       });
     } catch (error) {
       let e: BusinessError = error as BusinessError;
       Logger.error(`Failed to get altitude. Code: ${e.code}, message: ${e.message}`);
     }
   }
   ```
3. 获取磁场数据：监听磁力传感器[sensor.SensorId.MAGNETIC\_FIELD](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-sensor#sensorid9)获取X、Y、Z方向的磁场信息，并计算当前磁力信息。

   ```
   public getMagnetic() {
     try {
       sensor.on(sensor.SensorId.MAGNETIC_FIELD, (data: sensor.MagneticFieldResponse) => {
         this.magnetic = Math.sqrt((Math.pow(data.x, 2) + Math.pow(data.y, 2) + Math.pow(data.z, 2)))
         this.notifyMagneticChanges()
       }, { interval: 100000000 });
     } catch (error) {
       let e: BusinessError = error as BusinessError;
       Logger.error(`Failed to invoke on. Code: ${e.code}, message: ${e.message}`);
     }
   }
   ```
4. 获取经纬度数据：使用[geoLocationManager](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-geolocationmanager)获取latitude纬度、longitude经度和altitude海拔数据。

   ```
   // The callback for reporting the location result
   public locationChange = async (location: geoLocationManager.Location) => {
     this.latitude = location.latitude;
     this.longitude = location.longitude;
     if (location.altitude != 0) {
       this.altitude = location.altitude;
     }
     this.notifyPositionChanges();
   }
   private magnetic: number = 0;
   private angle: number = 0;
   private minute: number = 60;

   // Subscribe location changed
   public async getLocation() {
     let requestInfo: geoLocationManager.LocationRequest = {
       priority: geoLocationManager.LocationRequestPriority.ACCURACY,  // 快速获取位置优先，如果应用希望快速拿到1个位置，可以将优先级设置为该字段
       scenario: geoLocationManager.LocationRequestScenario.UNSET,  // 未设置场景信息
       timeInterval: 0,  // 上报位置信息的时间间隔
       distanceInterval: 100,  // 上报位置信息的距离间隔
       maxAccuracy: 100  // 精度信息
     };
     geoLocationManager.on('locationChange', requestInfo, this.locationChange);
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。

## 权限说明

* 获取网络权限：[ohos.permission.INTERNET](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissioninternet)。
* 获取模糊位置权限：[ohos.permission.APPROXIMATELY\_LOCATION](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all-user#ohospermissionapproximately_location)。
* 获取精确位置权限：[ohos.permission.LOCATION](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all-user#ohospermissionlocation)。
