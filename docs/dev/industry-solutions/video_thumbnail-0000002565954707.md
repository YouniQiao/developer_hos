## 场景介绍

拖动进度条时显示视频缩略图是影音娱乐类应用中的典型场景之一，如用户在视频播放时，可以拖动进度条预览对应时间点的视频缩略图。

本示例使用[AVPlayer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-media-avplayer)播放视频，使用[AVImageGenerator](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-media-avimagegenerator)获取视频指定时间点的缩略图，实现用户拖动视频进度条时，进度条上方对应时间点的视频缩略图随用户滑动的功能。

## 实现思路

1. 监听[Slider](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-slider)组件的[onChange](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-slider#onchange)事件，获取拖动进度条时滑块当前位置对应的视频时间。
2. 通过[AVImageGenerator](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-media-avimagegenerator)的[fetchFrameByTime](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-media-avimagegenerator#fetchframebytime12)接口，获取视频对应时间点的缩略图。

   ```
   async fetchFrame(timeMs: number): Promise<PixelMap | undefined> {
     if (this.avImageGenerator === undefined) {
       return undefined;
     }
     let timeUs = timeMs * 1000;
     let options = media.AVImageQueryOptions.AV_IMAGE_QUERY_CLOSEST;
     let videoAspectRadio = AppStorage.get('VideoAspectRadio') as number;
     let param: media.PixelMapParams = { width: POPUP_WIDTH, height: POPUP_WIDTH * videoAspectRadio };
     let frame = await this.avImageGenerator.fetchFrameByTime(timeUs, options, param);
     return frame;
   }
   ```
3. 根据[Slider](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-slider)滑块的当前位置，计算缩略图窗口的展示位置，实现跟随进度条滑动。

   ```
   calPopupOffsetX(value: number) {
     let sliderWidth = this.windowWidth * 0.7;
     let radio = value / this.videoDuration;
     let offsetX = sliderWidth * radio - POPUP_WIDTH / 2;
     if (offsetX < 0) {
       offsetX = 0;
     } else if (offsetX + POPUP_WIDTH > sliderWidth) {
       offsetX = sliderWidth - POPUP_WIDTH;
     }
     this.popupOffsetX = offsetX;
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
