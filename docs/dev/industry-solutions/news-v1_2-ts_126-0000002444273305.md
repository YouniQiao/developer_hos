## 场景介绍

精确时间间隔播放短音频是影音娱乐类应用中高频使用场景之一。

本示例主要是精确调整每分钟的节拍数，通过[Worker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-worker)线程的方式为应用程序提供一个多线程的运行环境，并与主线程通信实现精准定时，使用[SoundPool](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-multimedia-soundpool#soundpool)实现低时延短音播放，达到直观感受节拍器的节奏效果。

## 实现思路

1. 创建Worker：在对应的{moduleName}目录下任意位置，点击鼠标右键 > New > Worker，自动生成Worker的模板文件及配置信息，详见[创建Worker的注意事项](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/worker-introduction#创建worker的注意事项)。Worker线程接收宿主线程信息，判断是否开启定时器向宿主线程发送信息，以1s发送一次为例。

   ```
   import { ThreadWorkerGlobalScope, worker } from '@kit.ArkTS';

   const workerPort: ThreadWorkerGlobalScope = worker.workerPort;
   let timer: number = 0; // 定时器
   /**
    * Defines the event handler to be called when the worker thread receives a message sent by the host thread.
    * The event handler is executed in the worker thread.
    *
    * @param event message data
    */
   workerPort.onmessage = (e) => {
     if (e.data === true) {
       timer = setInterval(() => {
         workerPort.postMessage('timer');
       }, 1000);
     } else {
       clearTimeout(timer);
     }
   };
   ```
2. 在主线程中通过调用ThreadWorker的constructor()方法创建Worker对象，向Worker线程传递信息，并接收Worker线程信息，判断音频播放或者停止。

   ```
   onClick(() => {
     const workerInstance = new worker.ThreadWorker('entry/ets/workers/Worker.ets');
     if (this.isPlaying) {
       // 宿主线程向worker线程传递信息，停止播放
       workerInstance.postMessage(false);
       let item = this.soundFilesArr[0];
       this.stop(item.soundId);
     } else {
       // 宿主线程向worker线程传递信息，开始播放
       workerInstance.postMessage(true);
       // 宿主线程接收worker线程信息
       workerInstance.onmessage = (): void => {
         this.interval++
         let item = this.soundFilesArr[0];
         this.play(item.soundId);
       };
     }
   })
   ```
3. 创建SoundPool实例，获取加载本地音频文件。

   ```
   async create() {
       // 创建soundPool实例
       let audioRendererInfo: audio.AudioRendererInfo = {
         usage: audio.StreamUsage.STREAM_USAGE_MUSIC, // 音频流使用类型：音乐。根据业务场景配置，参考StreamUsage
         rendererFlags: 1 // 音频渲染器标志
       };
       soundPool = await media.createSoundPool(32, audioRendererInfo);

       this.loadCallback(soundPool); // 监听loadComplete
       await this.scanRawFile(); // 扫描rawfile音频资源
     }

     async scanRawFile() {
       let context = this.getUIContext().getHostContext() as common.UIAbilityContext;
       let resourceManager = context.resourceManager;
       // 获取本地音频文件
       let files = await resourceManager.getRawFileList('soundpool/');
       for (let i = 0; i < files.length; i++) {
         let soundFile: SoundFile = new SoundFile(files[i], -1);
         this.soundFilesArr.push(soundFile);
         await this.load(soundFile.fileName);
       }
       ;
     }

     async load(fileName: string) {
       let context = this.getUIContext().getHostContext() as common.UIAbilityContext;
       // 加载本地音频文件
       let rowFd = await context.resourceManager.getRawFd('soundpool/' + fileName);
       await soundPool.load(rowFd.fd, rowFd.offset, rowFd.length).then((soundId) => {
         this.updListArrSoundId(fileName, soundId);
       })
         .catch((e: BusinessError) => {
           console.error('load sound err: ', e.code, e.message);
         });
     }

     updListArrSoundId(fileName: string, soundId: number) {
       // 更新list记录soundId
       this.soundFilesArr.forEach((soundFile) => {
         if (soundFile.fileName === fileName) {
           soundFile.soundId = soundId;
         }
         ;
       });
     };
   ```
4. 调用play方法播放音频和stop方法终止指定流的播放。

   ```
   async play(soundId: number) {
       this.isPlaying = true;
       // 开始播放，这边play也可带播放播放的参数PlayParameters，请在音频资源加载完毕，即收到loadComplete回调之后再执行play操作
       await soundPool.play(soundId).then(async () => {
         console.info(`play sound success soundId is ${soundId}`);
       }, (err: BusinessError) => {
         console.info(`play sound Error: errCode is ${err.code}, errMessage is ${err.message}`);
       });
     }

     // 停止播放
     async stop(soundId: number) {
       this.isPlaying = false;
       this.interval = -1;
       await soundPool.stop(soundId).then(async () => {
         console.info(`stop sound success soundId is ${soundId}`);
       }, (err: BusinessError) => {
         console.info(`stop sound Error: errCode is ${err.code}, errMessage is ${err.message}`);
       });
     }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
