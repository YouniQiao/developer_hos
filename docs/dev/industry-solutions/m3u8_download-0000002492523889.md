## 场景介绍

后台下载m3u8音视频资源是影音娱乐类应用的高频使用场景之一，如用户不影响当前播放情况下，下载m3u8格式音视频资源至本地。

本示例通过创建与主线程并行的[Worker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-worker)线程，使用[request.downloadFile](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-request#requestdownloadfile9)启动下载任务，开启[backgroundTaskManager](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-resourceschedule-backgroundtaskmanager)后台任务管理并通过[NotificationManager](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-notificationmanager)通知模板提醒更新下载进度。

## 实现思路

1. 通过[getWantAgent](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-app-ability-wantagent#wantagentgetwantagent-1)方法获取WantAgent对象，申请数据传输类型长时任务并发送消息至[Worker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-worker)线程开启下载任务。

   ```
   let m3u8str = 'https://xxxxx.xxx.xxx/xxxxxxx.m3u8'; // 替换为实际地址
   workerInstance.postMessage({
     code: 200,
     context: this.context,
     m3u8Url: m3u8Str,
     filePath: filePath,
     fileName: fileName
   });
   this.startContinuousTask();

   // 申请数据传输类型长时任务方法
   startContinuousTask() {
     // 现已不推荐三方应用使用显式Want拉起其他应用，本示例需要获取WantAgent对象，用于申请长时任务
     let wantAgentInfo: wantAgent.WantAgentInfo = {
       // 添加需要被拉起应用的bundleName和abilityName
       wants: [
         {
           bundleName: 'com.example.m3u8download',
           abilityName: 'EntryAbility'
         }
       ],
       actionType: wantAgent.OperationType.START_ABILITY,
       requestCode: 0,
       actionFlags: [wantAgent.WantAgentFlags.UPDATE_PRESENT_FLAG],
     };
     // 通过wantAgent模块下getWantAgent方法获取WantAgent对象，用于申请长时任务
     wantAgent.getWantAgent(wantAgentInfo).then((wantAgentObj: WantAgent) => {
       let list: string[] = ['dataTransfer'];
       backgroundTaskManager.startBackgroundRunning(this.context, list, wantAgentObj)
         .then((res: backgroundTaskManager.ContinuousTaskNotification) => {
           // 使用res中返回的notificationId来更新通知
           this.notificationId = res.notificationId;
         });
     });
   }
   ```
2. 通过[HTTP数据请求](https://developer.huawei.com/consumer/cn/doc/atomic-guides/atomic-http-request)获取并解析m3u8索引内容。

   ```
   // M3u8Worker.ets
   // 下载解析m3u8一级索引，得到二级索引地址
   const videoContent = await downloadM3u8(m3u8Url);
   Logger.info('--------->\n', videoContent);
   const videoInfo = obtainM3u8Url(m3u8Url, videoContent);
   // 下载解析m3u8二级索引，得到ts分片地址
   const m3u8Content = await downloadM3u8(videoInfo[0].m3u8Url);
   Logger.info('--------->\n', m3u8Content);
   const m3u8Info = obtainTsUrls(videoInfo[0].m3u8Url, m3u8Content);
   tsCount = m3u8Info.tsUrls.length;

   // M3u8Utils.ets
   // 下载m3u8索引文件
   async function downloadM3u8(m3u8Url: string): Promise<string> {
     const httpRequest = http.createHttp();
     const response = await httpRequest.request(m3u8Url, {
       method: http.RequestMethod.GET,
       connectTimeout: 10000
     });
     return response.result.toString();
   }
   ```
3. 开启定时器，定时扫描当前下载状态。

   ```
   // 开启定时器，定时扫描当前下载状态
   intervalId = setInterval(() => {
       if (completedCount < tsCount) {
           if (!isDownLoading) {
           downLoadFile(context, m3u8Info.tsUrls[completedCount].tsUrl, filePath, `${completedCount}.ts`);
           }
       } else {
           let fileStr = saveM3U8Playlist(m3u8Info, filePath);
           Logger.info('------ end -------');
           workerPort.postMessage({ code: 200, data: JSON.stringify(fileStr) });
           clearInterval(intervalId);
       }
   }, 500);
   ```
4. 主线程接收到Worker线程发送的消息，对[NotificationManager](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-notificationmanager)进行通知刷新。

   ```
   workerInstance.onmessage = (e: MessageEvents): void => {
       if (e.data.code === 100) {
           this.updateDownloadProcess(e.data.percent, e.data.currentFile);
       }
       if (e.data.code === 200) {
           this.stopContinuousTask();
           this.isLoading = false;
           Loading.showToast($r('app.string.toast_message_2'));
           Logger.info('download succeeded');
           workerInstance.terminate();
       }
   };
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。

## 权限说明

* 使用Internet网络权限：[ohos.permission.INTERNET](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissioninternet)。
* 支持Service Ability在后台持续运行：[ohos.permission.KEEP\_BACKGROUND\_RUNNING](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissionkeep_background_running)。
