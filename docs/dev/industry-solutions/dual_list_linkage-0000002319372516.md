## 场景介绍

双列表联动切换是便捷生活类应用的高频使用场景之一。

本示例基于[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)组件实现双列表联动切换效果，当点击一级列表某一分类类型时，二级列表自动切换至对应的位置，当滑动二级列表时，一级列表自动切换至对应的分类类型。

## 实现思路

* 点击左侧一级列表：当点击左侧一级列表某一分类类型时，取消前一个选中项的状态标记，当前索引置为此次点击条目的索引，同时将此次索引的条目数据状态置为true，二级列表滚动控制器调用[scrollToIndex](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-scroll#scrolltoindex)方法将二级列表滚动至对应的位置。

  ```
  this.mList[this.currentIndex].isSelect = false;
  this.currentIndex = index;
  this.level2Scroller.scrollToIndex(this.currentIndex, true);
  this.mList[this.currentIndex].isSelect = true;
  ```
* 滑动右侧二级列表：右侧二级列表监听[onScrollIndex](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list#onscrollindex)方法，在该方法内部，将上次选中的条目数据状态置为false，当前索引置为此时二级列表显示区域内第一个子组件的索引，同时将此次索引的条目数据状态置为true。

  ```
  List({ scroller: this.level2Scroller })
    .onScrollIndex((start: number) => {
      this.mList[this.currentIndex].isSelect = false;
      this.currentIndex = start;
      this.mList[this.currentIndex].isSelect = true;
    });
  ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
