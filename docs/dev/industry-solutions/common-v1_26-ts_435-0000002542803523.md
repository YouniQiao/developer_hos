## 问题现象

在[场景化性能测试](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/specialized-testing#section8642101711299)脚本中右键“执行hypium用例”，出现测试资源加载异常[01100]：'NoneType' object has no attribute 'replace'报错。

![](./img/b44cded5.png "点击放大")

## 解决方案

点击工程中的main.py脚本，右键执行Run 'main'进行调试。

![](./img/137033c2.png "点击放大")
