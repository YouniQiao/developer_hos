## 场景介绍

学习进度折线图统计是教育类应用中的典型场景之一，如通过折线图展示用户某一周期（周、月）内的学习、复习词汇量以及学习时长等数据变化。

本示例使用[@ohos/mpchart](https://ohpm.openharmony.cn/#/cn/detail/@ohos%2Fmpchart)实现折线图（走势图）绘制与查看周期切换等功能。

## 实现思路

1. 根据对应的周期时间和图表需求，处理需要展示的数据。
2. 将需要展示的数据传入LineCharts组件，来绘制折线图。

   ```
   LineCharts({
     duringMinValue: 0,
     duringMaxValue: 100,
     timeStyle: 'week',
     trendData: this.vocabularyWeekLearnList,
     trendData1: this.vocabularyWeekReviseList,
     selectedIndex: 0,
     unit: $r('app.string.vocab'),
     textLeft: $r('app.string.today_learn'),
     numLeft: this.vocabularyWeekLearnList[this.vocabularyWeekLearnList.length-1],
     textRight: $r('app.string.today_revise'),
     numRight: this.vocabularyWeekReviseList[this.vocabularyWeekReviseList.length-1],
   });
   ```
3. 使用[@Watch](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-watch)监听当前日期与展示周期，若有变化，则调用对应函数更新数据。

   ```
   @Prop @Watch('onDataChange') trendData: Array<number>;

   onDataChange() {
     if (this.model) {
       this.model.setData(this.getLineData());
       this.model.notifyDataSetChanged();
       this.model.invalidate();
     }
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
