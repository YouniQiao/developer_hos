## 问题现象

元服务主包网络请求正常，子包网络请求失败，错误信息如下：

```
error:{"code":201,"message":"Permission denied"}
```

## 解决方案

元服务子包发起网络请求，未申请网络权限导致报错，需要在子包的module.json5配置文件的requestPermissions标签中声明ohos.permission.INTERNET权限，参考：[声明权限](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/declare-permissions)。
