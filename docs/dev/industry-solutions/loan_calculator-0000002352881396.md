## 场景介绍

贷款计算器是理财保险类应用中典型场景之一，如计算商业贷款、公积金贷款的月还款金额等。

本示例主要基于[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)、[TextPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-textpicker)和[NavDestination](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navdestination)实现贷款计算器，通过输入贷款金额、期限、年利率和还款方式自动计算月供金额，并分期展示月供总额、月供本金和月供利息。

## 实现思路

1. 利用[bindSheet](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-sheet-page)方法，为文本组件绑定半模态弹窗后，通过[TextPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-textpicker)实现在半模态弹窗内滑动选择贷款期限和还款方式。

   ```
   TextPicker({ range: this.getPickerRange(), selected: this.getSelectedIdx() })
     .onChange((value: string | string[]) => {
       // 选择器状态改变时，为对应的状态变量赋值
       this.updateFilledContent(value as string)
     })
   ```
2. 若填写项均已填充，则可点击“开始计算”按钮，携带用户填充的贷款信息跳转至详情页（[NavDestination](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navdestination)）。

   ```
   Button(CommonConstants.START_CALCULATE)
     .onClick(() => {
       // 若存在未填充的选项，则不可跳转
       if (!this.isAllFilled) {
         this.toastHasUnfilled()
         return;
       }

       // 构造用户输入的贷款信息对象
       let userLoanInfo: LoanInfo = {
         // 字段赋值
       }
       // 携带信息跳转详情页
       this.pathStack.pushPathByName('DetailPage', userLoanInfo)
     })
   ```
3. 进入详情页后，利用NavDestination的[onReady](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navdestination#onready11)回调方法，在渲染UI前，解析主页面携带的贷款信息。

   ```
   onReady((context: NavDestinationContext) => {
     this.pathStack = context.pathStack
     this.mLoanInfo = context.pathInfo.param as LoanInfo
     this.mTotalInterest = calEqualBaseInterest(this.mLoanInfo, this.mMonthInfoArr)
   })
   ```
4. 通过[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)组件展示每月待还款信息。

   ```
   List({ scroller: new Scroller() }) {
     ForEach(this.mMonthInfoArr, (item: MonthRepayInfo) => {
       ListItem() {
          // 展示每月还款信息
       }
     })
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
