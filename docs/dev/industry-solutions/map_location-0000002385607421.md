## 场景介绍

儿童定位与移动轨迹追踪是儿童教育类应用中的典型场景之一，如家长可实时查看孩子当前位置，并回溯其移动轨迹。

本示例基于[MapComponent](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-mapcomponent)组件和[addTraceOverlay](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-mapcomponentcontroller#section188905329242)方法绘制孩子移动的路线。

## 实现思路

1. 定义一系列经纬度坐标点，用来模拟孩子的轨迹。

   ```
   import { mapCommon } from '@kit.MapKit';

   export const MAP_LOCATION: mapCommon.LatLng[] = [
     { longitude: 121.44311, latitude: 31.185169 },
     { longitude: 121.443244, latitude: 31.18527 },
     { longitude: 121.443383, latitude: 31.185444 },
     { longitude: 121.443598, latitude: 31.185605 },
     { longitude: 121.4437, latitude: 31.18572 },
     { longitude: 121.44377, latitude: 31.18577 },
     { longitude: 121.443898, latitude: 31.185871 },
     // ...
   ```
2. 调用[addTraceOverlay](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-mapcomponentcontroller#section188905329242)方法初始化路线。

   ```
   let traceOptions: mapCommon.TraceOverlayParams = {
     // 轨迹点
     points: MAP_LOCATION,
     // 轨迹的动画时长
     animationDuration: 5000,
     // 相机是否跟随动画移动
     isMapMoving: true,
     // 轨迹的颜色
     color: 0xFFA5D61D,
     // 轨迹的宽度
     width: 20
   };
   let markers: Array<map.Marker> = new Array();
   markers.push(markerChild);
   // 新增轨迹点动画
   await this.mapController.addTraceOverlay(traceOptions, markers);
   ```
3. 使用[MapComponent](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-mapcomponent)组件初始化地图并绘制轨迹。

   ```
   MapComponent({ mapOptions: this.mapOptions, mapCallback: this.callback })
     .width(StyleConstants.FULL)
     .height(StyleConstants.FULL)
     .bindSheet(true, this.PositionText(), {
       showClose: false,
       backgroundColor: $r('app.color.gray_back'),
       detents: [137],
       enableOutsideInteractive: true,
       onWillDismiss: () => {
         return false;
       }
     });
   ```

本示例需要[开通地图服务](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/map-config-agc#section16133115441516)，并进行相应配置，具体步骤如下：

1. 登录AppGallery Connect网站，选择“我的项目”。
2. 在项目列表中找到您的项目，在项目下的应用列表中选择需要打开“地图服务”的应用。
3. 选择API管理，找到“地图服务”开关，打开开关。
4. 确认已经开启“地图服务”开放能力，并完成[手动签名](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-signing#section297715173233)。

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
