## 场景介绍

Native实现音频内录是实用工具类应用的高频使用场景之一，如音乐创作、线上音频会议。

本示例使用[AVScreenCapture](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-avscreencapture)实现音频内录功能，[AVPlayer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-media-avplayer)实现音频播放功能，以及通过[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)对音频波形实现实时绘制。

## 实现思路

1. 初始化音频内录器，注册内录状态回调和录制时长回调。

   ```
   // InnerRecorder.cpp
   int32_t InnerRecorder::InitConfig(const std::string &innerFilePath)
   {
       if (mIsInitialized) {
           OH_LOG_ERROR(LOG_APP, "Already initialized.");
           return AV_INNER_RECORD_ERR_INIT_FAIL;
       }

       mRecorder = new(std::nothrow) AVInnerRecorder(innerFilePath);
       if (!mRecorder) {
           OH_LOG_ERROR(LOG_APP, "Inner recorder new failed.");
           return AV_INNER_RECORD_ERR_INIT_FAIL;
       }

       mRecorder->SetStateCallback(std::bind(&InnerRecorder::OnStateChange, this, std::placeholders::_1));
       mRecorder->SetRecordDurationCallback(std::bind(&InnerRecorder::OnDurationChange, this, std::placeholders::_1));

       mIsInitialized = true;
       return AV_INNER_RECORD_ERR_OK;
   }
   ```
2. 启动音频内录，每次停止后再启动音频内录，都需重新创建[AVScreenCapture](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-avscreencapture)实例。

   ```
   // InnerRecorder.cpp
   int32_t InnerRecorder::Start()
   {
       if (mIsInnerRecording || !mRecorder) {
           OH_LOG_ERROR(LOG_APP, "Already InnerRecording or Recorder(%{public}d) is nullptr.", !mRecorder);
           return AV_INNER_RECORD_ERR_INIT_FAIL;
       }

       int32_t ret = mRecorder->Start();
       if (ret != AV_INNER_RECORD_ERR_OK) {
           OH_LOG_ERROR(LOG_APP, "Start inner record failed.");
           return AV_INNER_RECORD_ERR_INIT_FAIL;
       }

       mIsInnerRecording = true;
       return AV_INNER_RECORD_ERR_OK;
   }
   ```
3. 停止音频内录，释放资源。

   ```
   // InnerRecorder.cpp
   int32_t InnerRecorder::Stop()
   {
       if (mRecorder != nullptr) {
           mRecorder->Stop();
       }

       Reset();

       return AV_INNER_RECORD_ERR_OK;
   }
   ```
4. 音频内录状态通知，通过NAPI创建线程安全函数将音频内录状态通知给ArkTS层。

   ```
   // InnerRecorder.cpp
   void InnerRecorder::OnStateChange(const OH_AVScreenCaptureStateCode stateCode)
   {
       if (mStateChangeCallback) {
           auto result = new CallbackResult({CallbackResultType::STATE_CHANGE,
               new StateCallbackResult({stateCode})});
           napi_call_threadsafe_function(mStateChangeCallback->func, result, napi_tsfn_nonblocking);
       }
   }
   ```
5. 音频内录时长通知，通过NAPI创建线程安全函数将音频内录状态通知给ArkTS层。

   ```
   // InnerRecorder.cpp
   void InnerRecorder::OnDurationChange(const int64_t duration)
   {
       if (mRecordDurationCallback) {
           auto result = new CallbackResult({CallbackResultType::RECORD_DURATION,
               new RecordDurationCallbackResult({duration})});
           napi_call_threadsafe_function(mRecordDurationCallback->func, result, napi_tsfn_nonblocking);
       }
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
