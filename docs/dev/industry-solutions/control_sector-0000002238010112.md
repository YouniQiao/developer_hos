## 场景介绍

方位角与夹角测量是实用工具类应用的高频场景之一，如获取地图上对应位置朝向（方位角），或获取摄像方向与焦距下所能覆盖的范围（夹角）。

本示例通过[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)画布实现扇形绘制，[PanGesture](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-gestures-pangesture)实现手势处理，伴随拖动实现方位角与夹角的实时展示。

## 实现思路

* 通过[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)画布实现扇形绘制，在Draw()中实现具体绘画逻辑。
* 通过[PanGesture](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-gestures-pangesture)实现手势处理：对于拖动扇形两条边与拖动扇形本身分情况设置处理逻辑，实现扇形状态、方位角与夹角的实时展示。
* 设置扇形两边角度为-π到π，保证绘制扇形的角度小于180°。

```
// 扇形绘制
Canvas(this.context)
  .onReady(() => {
    this.Draw(); // 绘制函数
  })
// PanGesture手势处理
PanGesture()
  .onActionUpdate((event) => {
    if (this.judgeLocationFlag === Constants.LINE_ONE_CHANGE) {
      // 扇形一条边的角度与端点变化，角度变化在-π到π之间
    }
    this.Draw(); // 实时绘制扇形
    this.setArcAngle(); // 实时显示夹角
    this.setArcDirection(); // 实时显示方位角
  })
```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
