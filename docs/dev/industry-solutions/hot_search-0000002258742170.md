## 场景介绍

本示例基于[Tabs](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-tabs)构建了一个包含搜索栏、最近使用的应用列表和新闻热搜榜单的页面，用户可以搜索关键词、查看最近使用功能及每天的热门新闻排名。

## 实现思路

1. tabBuilder用于创建一个标签项，包含文本和底部指示条，使用selectedIndex控制文本颜色和指示条的可见性。

   ```
   @Builder
   tabBuilder(index: number, name: string) {
     Text(name)
       .fontColor(this.selectedIndex === index ? this.selectedFontColor : this.fontColor);
     Stack() {
       Divider();
       Divider()
         .opacity(this.selectedIndex === index ? 1 : 0);
     };
   }
   ```
2. 使用[Tabs](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-tabs)显示多个新闻标签页，每个标签页包含新闻标题和多个消息条目，TabContent中使用textColumn构建器显示消息条目。

   ```
   Tabs({ barPosition: BarPosition.Start, index: this.currentIndex, controller: this.controller }) {
     ForEach(newsArray, (item: number) => {
       TabContent() {
         Column() {
           Text(NEWS_DATA[item].title);
           ForEach(msgsArray, (item: number) => {
             this.textColumn(MESSAGES[item].num, MESSAGES[item].msg, MESSAGES[item].million, MESSAGES[item].img,
               item);
           });
         };
       };
     });
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
