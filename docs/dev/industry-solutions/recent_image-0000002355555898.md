## 场景介绍

在聊天界面提示用户可以发送最近保存图片是社交通讯类应用的高频使用场景之一。

本示例主要使用[@ohos.file.RecentPhotoComponent](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ohos-file-recentphotocomponent)实现获取图库最近新增的图片，使用[ohos.animator](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-animator)帧动画实现功能区的展开，并结合[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)实现协调的动效与交互逻辑。

## 实现思路

1. 在[RecentPhotoComponent](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ohos-file-recentphotocomponent#recentphotocomponent)组件的onRecentPhotoCheckInfo参数中判断是否存在最近一张图片，如有则展示最近图片供用户选择。

   ```
   // 最近图片组件
   RecentPhotoComponent({
     recentPhotoOptions: this.recentPhotoOptions,
     onRecentPhotoCheckResult: this.recentPhotoCheckResultCallback,
     onRecentPhotoClick: this.recentPhotoClickCallback,
     // 判断最近图片是否存在查询结果以及最近图片相关信息
     onRecentPhotoCheckInfo: this.recentPhotoCheckInfoCallback
   })
   // 最近图片查询结果回调函数，并且返回该照片的相关信息
   private recentPhotoCheckInfoCallback: RecentPhotoCheckInfoCallback =
     (recentPhotoExists: boolean, info: RecentPhotoInfo) => this.onRecentPhotoCheckInfo(recentPhotoExists, info);
   // 判断是否存在最近图片
   private onRecentPhotoCheckInfo(recentPhotoExists: boolean, info: RecentPhotoInfo): void {
     // 是否存在符合条件的照片或视频，若存在则可以拿到该照片或视频的相关信息，若不存在直接返回
     if (!recentPhotoExists) {
       return;
     }
     // 存在最近图片的话，可以实现业务需求，如去控制RecentPhotoComponent是否显示
     this.timeoutID = setTimeout(() => {
       this.hasRecentPhoto = true;
     }, 200);
   }
   ```
2. 使用onRecentPhotoClick方法实现点击后行为，关闭功能区，并跳转发送图片页面。

   ```
   // ChatPage.ets
   // 设置数据类型， IMAGE_VIDEO_TYPE：图片和视频（默认值）  IMAGE_TYPE：图片  VIDEO_TYPE：视频  MOVING_PHOTO_IMAGE_TYPE：动态图片
   this.recentPhotoOptions.MIMEType = photoAccessHelper.PhotoViewMIMETypes.IMAGE_TYPE;
   // 设置最近图片的时间范围，单位（秒），0表示所有时间
   this.recentPhotoOptions.period = 0;
   // 设置资源的来源  ALL：所有  CAMERA：相机  SCREENSHOT：截图
   this.recentPhotoOptions.photoSource = PhotoSource.ALL;
   private onRecentPhotoClick(recentPhotoInfo: BaseItemInfo): boolean {
     if (!recentPhotoInfo) {
       return false;
     }
     this.changeMenuStatus();
     this.getUIContext().getFocusController().clearFocus();
     this.pageInfos.pushPath({ name: 'sendPage', param: recentPhotoInfo });
     return true;
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
