## 场景介绍

文字转语音是新闻阅读类应用的典型场景之一，通过文字转语音可以实现听新闻的功能；定时关闭能够使用户控制听新闻结束的时间，给用户带来便利。

本示例使用[TextReader](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/speech-textreader-api)实现文本转语音功能，使用[Timer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-timer)实现倒计时功能，倒计时结束后关闭语音播放。

## 实现思路

通过[Timer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-timer)进行倒计时，使用[TextReader](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/speech-textreader-api)实现文本转语音播放，计时结束时播放停止。

```
start():void {
  TextReader.start(this.readInfoList!, this.selectedReadInfo?.id).catch((err:BusinessError) => {
    console.error(`Failed to start TextReader. code is ${(err as BusinessError).code}message is ${(err as BusinessError).message}`);
  });
  this.textTimerController?.start()
  this.isPlaying = true
  AppStorage.set('isPlaying', this.isPlaying)
  this.timeoutID = setTimeout(() => {
    TextReader.stop().then(async () => {
      console.info(`TextReader succeeded in stopping.`);
      emitter.emit('TextReader complete')
      TextReader.release()
    }).catch((e: BusinessError) => {
      console.error(`TextReader failed to stop. Code: ${e.code}, message: ${e.message}`);
    })
  }, this.timeCount)
  AppStorage.setOrCreate('timeoutId', this.timeoutID)
}
```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
