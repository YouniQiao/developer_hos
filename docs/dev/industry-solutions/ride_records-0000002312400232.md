## 场景介绍

乘车记录查询是公交地铁类应用的高频使用场景之一，如用户在网约车、公共交通等应用中查询历史行程。

本示例通过自定义组件实现时间筛选功能，通过[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)展示乘车记录数据。

## 实现思路

1. 传入起止时间，使用getData函数获取筛选后的乘车记录数据。

   ```
   // 获取筛选后的乘车记录数据
   static getData(begin: string, end: string): DataInfo[] {
     let tempModelAry: DataInfo[] = [];
     for (let i = 0; i < INPUT_TIMES; i++) {
       for (let item of jsonData) {
         let datePart = item.time.split(' ')[0];
         if (datePart >= begin && datePart <= end) {
           let model = new DataInfo(item.time);
           tempModelAry.push(model);
         }
       }
     }
     return tempModelAry;
   }
   ```
2. 通过[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)列表组件获取数据列表，展示乘车记录数据。

   ```
   List() {
     ForEach(this.vm.dataAry, (item: DataInfo, index) => {
       ListItem() {
         this.buildItem(item, index);
       };
       // ...
     });
   };
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
