## 场景介绍

在图片中使用矩形框标记人脸位置是拍摄美化类应用的高频使用场景之一，如在视频监控中定位人脸位置。

本示例使用[faceDetector](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/core-vision-face-detector-api)检测图片中的人脸位置信息。将识别的人脸位置信息的像素坐标转换为[OpenGL ES](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/opengles)的标准化设备坐标，然后使用[OpenGL ES](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/opengles)在图片上绘制矩形框标记人脸位置。

## 实现思路

1. 通过[PhotoViewPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-photoaccesshelper-photoviewpicker)拉起系统图库，让用户选择需要识别人脸位置的图片。使用[ImageSource](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-image-imagesource)将用户选择的图片文件解码为[PixelMap](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-image-pixelmap)。
2. 使用[faceDetector](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/core-vision-face-detector-api)检测图片人脸，获得人脸矩形框像素坐标位置信息。

   ```
   let rectArr: Array<Array<number>> = [];   // 人脸矩形框数组
   try {
     await faceDetector.init();
     // 检测图片人脸
     let visionInfo: faceDetector.VisionInfo = {
       pixelMap: this.pixelmap,
     };
     let faceArr: faceDetector.Face[] = await faceDetector.detect(visionInfo);
     // 获得人脸矩形框像素坐标位置信息
     for (const face of faceArr) {
       rectArr.push([face.rect.left, face.rect.top, face.rect.width, face.rect.height]);
     }
   } catch (error) {
     console.error(`failed to detect face, ${JSON.stringify(error)}`);
     return false;
   } finally {
     await faceDetector.release();
   }
   ```
3. 将像素坐标(x, y)转换为设备标准化坐标(m, n)，需要先获得绘制窗口的宽高(w, h)，然后根据窗口的宽高分别将水平像素位置x和垂直像素位置y分别映射到标准化设备坐标-1.0到1.0的范围内。需要注意像素坐标以水平向下为y轴正方向，而标准化设备坐标以水平向上为y轴正方向。具体转换方法为：m = 2.0 \* (x / w) - 1.0，n = 1.0 - 2.0 \* (y / h)。

   ```
   // 转换为OpenGL的标准化设备坐标，width为窗口宽度，height为窗口高度
   // 矩形左上角
   x = box.left;                                                         // 水平像素坐标
   y = box.top;                                                          // 垂直像素坐标
   vertices[0] = static_cast<float>(2.0f * (x / (double)width) - 1.0f);  // 水平标准化设备坐标
   vertices[1] = static_cast<float>(1.0f - 2.0f * (y / (double)height)); // 垂直标准化设备坐标
   ```
4. 绘制完成后，拷贝OpenGL ES的纹理像素数据，通过[ArrayBuffer](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arraybuffer-object)返回ArkTS侧，生成包含人脸矩形框的[PixelMap](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-image-pixelmap)。

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
