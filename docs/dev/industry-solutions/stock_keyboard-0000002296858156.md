## 场景介绍

自定义股票键盘是理财保险类应用中高频使用场景之一，如用户在股票交易应用中，使用股票键盘完成买入、卖出等操作，提高交易效率和准确性。

本示例基于[自定义键盘开发实践](https://developer.huawei.com/consumer/cn/doc/best-practices/bpta-custom-keyboard)构建股票键盘，点击搜索栏，弹出股票键盘，支持数字键盘（默认）、英文键盘和中文系统键盘切换。数字键盘支持快速输入股票代码（如600、601等），支持设置输入字符串长度，本示例设置长度限制为8。

## 实现思路

1. 在键盘Tab栏中，点击切换键盘类型，包括数字键盘、英文键盘和中文系统键盘。

   ```
   // StockKeyboardTabviewComponent.ets
   TabButton()
     .onClick(() => {
       // 转换至对应键盘类型
       this.keyboardController.keyboardType = this.type;
       // 转换至系统键盘时收回自定义键盘
       this.isCustomKeyboardAttach = this.type !== KeyboardType.SYSTEM_KEYBOARD;
     });
   ```
2. 通过[Grid](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-grid)组件实现股票数字键盘布局。

   ```
   // StockNumKeyboardComponent.ets
   Grid() {
     // numKeyboardData中存储按键高度属性，宽度均匀分配
     ForEach(numKeyboardData, (item: KeyboardMenu) => {
       GridItem() {
         Button(item.textOrImage, { type: ButtonType.Normal });
       };
     });
   };
   ```
3. 通过[Column](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-column)、[Row](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-row)组件实现英文键盘布局。

   ```
   // StockABCKeyboardComponent.ets
   Column() {
     // 每行高度均匀分配，ABCButton对特殊键的宽度进行特殊处理
     Row({ space: KeyboardConstants.KEYBOARD_MENU_SPACE }) {
       ForEach(abcKeyboardData[0], (item: KeyboardMenu, index: number) => {
         ABCButton({ item: item, rowIndex: 0, columnIndex: index });
       }, (item: KeyboardMenu) => JSON.stringify(item));
     };
     Row({ space: KeyboardConstants.KEYBOARD_MENU_SPACE }) {
       // ...
     };
     // ...
   };
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
