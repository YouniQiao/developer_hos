## 场景介绍

铃声设置是实用工具类应用的高频场景之一，如设置管理闹钟、来电、信息、通知铃声等音频信息。

本示例通过[Ringtone Kit](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ringtone-ringtone)实现铃声设置。

## 实现思路

1. 选择手机内音频文件或将rawfile文件加载进应用沙箱。

   ```
   // 手机音频文件加载进沙箱
   findAudioFromPhone() {
     let audioSelectOptions = new picker.AudioSelectOptions();// 设置选择选项
     let audioPicker = new picker.AudioViewPicker(context);// 获取音频选择器
     audioPicker.select(audioSelectOptions).then(async (audioSelectResult: Array<string>) => {
       // 选择手机内音频文件并加载入沙箱
     });
   }
   // rawfile文件加载进沙箱
   copyRawFileListToSdcard() {
     // 1. 从 rawfile 读取原始数据
     let rawFile = this.context?.resourceManager.getRawFileContentSync(Constants.MUSIC_PATH_1 + str);
     let arrayBuffer: ArrayBuffer = rawFile?.buffer as ArrayBuffer;
     // 2. 创建沙箱目标文件路径
     let fd = fs.openSync(sandboxPath, fs.OpenMode.READ_WRITE | fs.OpenMode.CREATE);
     // 3. 写入数据到沙箱文件
     let bytesWritten = fs.writeSync(fd.fd, arrayBuffer, {
       offset: 0,
       length: arrayBuffer.byteLength
     });
   }
   ```
2. 调用[Ringtone.startRingtoneSetting](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ringtone-ringtone#section5949453189)接口拉起弹窗设置铃声。

   ```
   ringtone.startRingtoneSetting(this.context, audioPath, fileName).then(res => {}) // 传入沙箱路径和文件名称
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
