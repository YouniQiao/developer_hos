## 场景介绍

公文审批是综合办公类应用中的典型场景之一，如在进行文件审批时需要预览待审批文件并签字确认。

本示例利用[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)组件和[CanvasRenderingContext2D](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-canvasrenderingcontext2d)实现画板签名功能，使用[DocumentViewPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-picker#documentviewpicker)和[filePreview](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/preview-arkts)实现在线文件的预览和下载保存功能。

## 实现思路

* 签名板绘制：
  1. 利用[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)组件构建画板，用户在画板上完成签名后保存签名路径，并使用[CanvasRenderingContext2D](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-canvasrenderingcontext2d)进行绘制；
  2. 采用清除所有线条再绘制的形式实现撤回功能。

  ```
  // 创建RenderingContext
  canvasContext: CanvasRenderingContext2D = new CanvasRenderingContext2D(new RenderingContextSettings(true)); // 初始化画板
  Canvas(this.canvasContext)
    // 设置绘画逻辑，将每一条绘画保存至pathArray
    .onTouch((event?: TouchEvent) => {
      if (event.type === TouchType.Down) {
      this.pathArray.push(this.tempPath)
      // ...
      }
      if (event.type === TouchType.Move) {
      this.tempPath.lineTo(event.touches[0].x, event.touches[0].y);
      this.canvasContext.stroke(this.tempPath);
      }
    })
  // 撤销：清空画板，弹出pathArray最后一条数据，重新进行画板绘画
  Button()
    .onClick(() => {
      this.canvasContext.clearRect(0, 0, 700, 350);
      this.pathArray.pop();
      for (let i = 0; i < this.pathArray.length; i++) {
        this.canvasContext.stroke(this.pathArray[i]);
      }
    })
  ```

* 文件预览和下载：
  1. 将要预览的文件下载保存至沙箱，并调用filePreview.openPreview()方法完成文件预览；
  2. 点击下载图标调用[DocumentViewPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-picker#documentviewpicker)选择下载路径，并调用fs.copyFile()方法完成沙箱到本地的文件拷贝，以实现文件的本地持久化保存。

  ```
  // 文件预览
  this.filePreview();
  // 下载到本地持久化保存
  documentViewPicker.save().then(()={
    writeToFile(documentSaveResult[0], filePath, uiContext);
  })
  ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
