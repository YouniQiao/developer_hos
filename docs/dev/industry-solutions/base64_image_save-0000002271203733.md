## 场景介绍

本示例在无需申请相册管理模块权限的情况下，利用[保存控件](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/savebutton)、[PhotoAccessHelper](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-photoaccesshelper-photoaccesshelper)和[Base64Helper](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-util#base64helper9)实现保存Base64格式图片到系统相册的功能，如用户保存当前浏览界面的Base64格式图片。

## 实现思路

1. Base64格式图片数据通过[base64Helper.decode](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-util#decode9)解码，并存储为ArrayBuffer类型。

   ```
   static async base64ToArrayBuffer(base64Str: string): Promise<ArrayBuffer> {
     let base64 = new util.Base64Helper();
     let uint8Array: Uint8Array = await base64.decode(base64Str);
     let buffer: ArrayBuffer = uint8Array.buffer.slice(0, uint8Array.byteLength);
     return buffer;
   }
   ```
2. 利用相册管理模块创建文件，并使用文件管理API完成文件的数据写入任务。

   ```
   static async saveImageToAlbum(uiContext: UIContext, imageBuffer: ArrayBuffer) {
     let file: fo.File | undefined = undefined;
     try {
       let helper = photoAccessHelper.getPhotoAccessHelper(uiContext.getHostContext());
       let uri = await helper.createAsset(photoAccessHelper.PhotoType.IMAGE, 'png');
       file = await fo.open(uri, fo.OpenMode.READ_WRITE | fo.OpenMode.CREATE);
       await fo.write(file.fd, imageBuffer);
       await fo.close(file.fd);
       // ...
     } catch (error) {
       // ...
     } finally {
       if (file) {
         fo.closeSync(file);
       }
     }
   }
   ```
3. 点击[SaveButton](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-security-components-savebutton)后，使用工具类PhotoUtils的saveImageToAlbum方法将Base64格式图片保存到本地。

   ```
   SaveButton({
     text: SaveDescription.SAVE_IMAGE
   })
     .onClick(() => {
       this.saveImageToAlbum(this.base64Image);
       // ...
     })
   async saveImageToAlbum(base64Str: string) {
     let imageBuffer: ArrayBuffer = await Base64Utils.base64ToArrayBuffer(base64Str);
     PhotoUtils.saveImageToAlbum(this.uiContext, imageBuffer);
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
