## 场景介绍

聊天快捷回复是社交通讯类应用的高频使用场景之一，如用户可以从快捷回复栏中选择预置的语句快速回复好友消息。

本示例基于[LazyForEach](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-rendering-control-lazyforeach)实现聊天快捷回复语窗口，通过[Array](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-arkts-collections-array)实现新增和删除常用语，可在常用语或快捷回复中选择并点击发送回复内容。

## 实现思路

1. 通过自定义弹窗实现[push](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-arkts-collections-array#push)新增常用语，在常用语中使用[splice](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-arkts-collections-array#splice-1)删除已有语句。

   ```
   dialogController: CustomDialogController = new CustomDialogController({
     builder: TextAreaDialog({
       confirm: (content) => {
         this.newTextInput = content;
         if (this.newTextInput) {
           // 新增
           this.commonReplyData.push(this.newTextInput);
           this.newTextInput = '';
         }
       }
     })
   });
   // 删除
   this.commonReplyData.splice(index, 1);
   ```
2. 通过getDataFromJSON函数读取“resources/rawfile/reply\_phrases.json”文件中的快捷回复语，写入到数据源quickReplyData中。

   ```
   export function getDataFromJSON(context: Context, fileName: string) {
     let result: QuickPhrases[] = [];
     try {
       let value: Uint8Array = context.resourceManager.getRawFileContentSync(fileName);
       // Compile Chinese UTF-8
       let decoder = util.TextDecoder.create('utf-8');
       let str = decoder.decodeToString(value);
       result = JSON.parse(str) as QuickPhrases[];
     } catch (err) {
       // ...
     }
     return result;
   }
   let phrasesData: QuickPhrases[] = getDataFromJSON(this.context, 'reply_phrases.json')
   this.quickReplyData.setData(phrasesData);
   ```
3. 通过[LazyForEach](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-rendering-control-lazyforeach)从数据源中读取数据，并创建相应组件。当组件滑出可视区域外时，销毁回收组件，以降低内存占用。

   ```
   LazyForEach(this.quickPhrases, (item: QuickPhrases, index: number) => {
     TabContent() {
       Column() {
         Divider()
           .strokeWidth(0.5)
           .color($r('sys.color.mask_fourth'));
         List({ scroller: this.listScroller }) {
           ForEach(item.phrases, (item: string, index: number) => {
             ListItem() {
               Text(item)
             }
           });
         }
       }
     }
     .tabBar(this.tabBuilder(index - 1, item.title));
   }, (item: QuickPhrases) => item.title);
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
