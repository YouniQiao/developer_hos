## 场景介绍

扫描并选择Wi-Fi进行连接是各类应用中的高频使用场景之一，需要进行定制化的Wi-Fi连接开发时，可参考此示例代码快速上手。

本示例使用[wifiManager.getScanInfoList](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-wifimanager#wifimanagergetscaninfolist10)获取扫描到的Wi-Fi网络，通过[wifiManager.connectToCandidateConfigWithUserAction](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-wifimanager#wifimanagerconnecttocandidateconfigwithuseraction20)选择添加过的候选网络并进行连接。

## 实现思路

1. 扫描Wi-Fi，在使用wifiManager.getScanInfoList进行扫描结果获取之前，需要用[wifiManager.startScan](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-wifimanager#wifimanagerstartscan21)进行一次扫描，确保能获取最新的扫描结果。由于扫描会切信道对实时性要求高的场景会有影响，因此这里设计多次获取扫描结果也只会发起一次扫描。

   ```
   isStartScan: boolean = false;
   // ...
   Toggle({ type: ToggleType.Switch, isOn: false })
     .id('wifi_Switch')
     .margin({ right: 12 })
     .onChange((isOn: boolean) => {
       this.isWifiSwitchOn = isOn;
       if (isOn) {
         if (this.isStartScan === false) {
           try {
             // 获取扫描结果前，先发起一次扫描，由于系统本身也会定期扫描，这里发起一次扫描即可
             wifiManager.startScan();
             this.isStartScan = true;
           } catch (error) {
             // ...
           };
         };
         this.wifiList = WiFiUtil.getScanResult();
       } else {
         this.wifiList = [];
       }
     })
   ;
   ```
2. 选择Wi-Fi进行连接，获取到扫描结果后，通过[wifiManager.addCandidateConfig](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-wifimanager#wifimanageraddcandidateconfig)将指定Wi-Fi网络添加到候选网络，并通过[wifiManager.connectToCandidateConfigWithUserAction](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-wifimanager#wifimanagerconnecttocandidateconfigwithuseraction20)进行连接。

   ```
   async connectWiFi(wifiScanInfo: wifiManager.WifiScanInfo, psk: string) {
     let config: wifiManager.WifiDeviceConfig = {
       ssid: wifiScanInfo.ssid,
       preSharedKey: psk,
       securityType: wifiScanInfo.securityType
     };
     let networkId = await wifiManager.addCandidateConfig(config);
     await wifiManager.connectToCandidateConfigWithUserAction(networkId);
   }
   ```

## 约束与限制

* 本示例支持API Version 21 Release及以上版本。
* 本示例支持HarmonyOS 6.0.1 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.1 Release及以上版本进行编译运行。

## 权限说明

* 获取Wi-Fi信息：[ohos.permission.GET\_WIFI\_INFO](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissionget_wifi_info)。
* 配置Wi-Fi：[ohos.permission.SET\_WIFI\_INFO](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissionset_wifi_info)。
