## 问题现象

使用消息推送服务，收到消息时，没有铃声和震动提醒。

推送代码如下：

```
{
    "payload": {
      "notification": {
        "category": "MARKETING",
        "title": "普通通知标题",
        "body": "普通通知内容",
        "clickAction": {
          "actionType": 1,
          "action": "****",
          "uri": "****",
          "data": {
            "securityData": "{
                "data": {
                    "batchNo": "",
                    "msgId": "****",
                    "userId": "****"
                },
                "click_action": "****"
            }"
          }
        }
      }
    },
    "target": {
      "token": ["*******"]
    },
    "pushOptions": {
      "testMessage": true
    }
  }
```

## 背景知识

[Push Kit（推送服务）](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/push-kit-introduction)是华为提供的消息推送平台，建立了从云端到终端的消息推送通道。所有HarmonyOS应用可通过集成Push Kit，实现向应用实时推送消息，使消息易见，构筑良好的用户关系，提升用户的感知度和活跃度。

## 问题定位

1. category是MARKETING，该消息分类为资讯营销类，消息提醒方式为静默通知，仅在通知栏展示。当category =IM （非MARKETING）时，按照[通知消息分类标准和提醒方式](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/push-apply-right#section1924971516101)，是有震动和铃声的，但是依然没有。
2. 确认是否申请了自分类权益。登录AppGallery Connect网站，点击“我的项目”。在项目列表中找到您的项目，通过“推送服务>配置”，在“配置”页签下选择需要申请自分类权益的应用，查看自分类权益是否已申请。

   ![](./img/3747ab8f.png "点击放大")

## 分析结论

根本原因是应用未申请推送场景化消息权益。未开通通知消息自分类权益的应用，通知消息类型将会默认归为资讯营销类消息，静默通知，仅在通知中心展示消息。

## 修改建议

1. [申请通知消息自分类权益](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/push-apply-right#section16708911111611)。
2. 适配云端category字段，使用对应的服务与通讯类取值，参考如下请求示例：

   ```

         },

         "image": "https://lf*******246.png"
       }
     },
     "target": {

         "MAAALgE4G98BAAAAst*******jg"
   ```

## 常见FAQ

Q：[Notification Kit](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/notification-overview?ha_source=sousuo&ha_sourceId=89000251)自定义铃声已经申请相关权益了，还是没有铃声是什么原因？

A：不同类型的通知渠道对应的通知提醒方式不同，有些通知渠道不支持铃声，详情见[通知渠道类型说明](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/notification-slot#通知渠道类型说明)，需要设置SlotType为SOCIAL\_COMMUNICATION、SERVICE\_INFORMATION或CUSTOMER\_SERVICE类型才支持铃声。
