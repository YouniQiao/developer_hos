## 问题现象

侧滑操作退出了应用，而不是关闭提示窗口。

## 背景知识

* [onBackPressed](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navdestination#onbackpressed10)：基于Navigation自定义弹窗实现，使用NavDestination的回调函数onBackPressed，当onBackPressed返回值为true时，表示重写返回键逻辑，返回值为false时，表示回退到上一个页面。
* [Class (WebviewController)](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-webview-webviewcontroller)：通过WebviewController可以控制Web组件各种行为（包括页面导航、生命周期状态、JavaScript交互等行为）。
* [runJavaScript](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-webview-webviewcontroller#runjavascript-1)：在当前显示页面的上下文中异步执行JavaScript脚本，脚本执行的结果将通过Promise方式返回。

## 问题定位

1. 从ArkUI Inspector页面发现这是一个Web页面。
2. 通过DevTools检查页面结构发现，弹窗由div元素实现，且与Web页面同级。因此侧滑操作会触发整个Web页面的返回事件，而Web页面位于应用首页，导致直接触发应用退出。

## 分析结论

弹窗与Web页面同层，导致侧滑返回时被识别为“返回上一级”，又因为当前已是首页，最终导致退出应用。

## 修改建议

在Web页面中定义关闭弹窗的JavaScript函数，并在onBackPress中通过WebViewController.runJavaScript调用该函数，优先尝试关闭弹窗；若弹窗未打开或已关闭，则执行默认返回逻辑（如终止应用）。

```
import { webview } from '@kit.ArkWeb';
import { common } from '@kit.AbilityKit';

@Entry
@Component
struct Index {
  @State private webViewController: webview.WebviewController = new webview.WebviewController();

  onBackPress() {
    let url: string = this.webViewController.getUrl()
    if (url === 'resource://rawfile/H5/index3.html') {
      this.webViewController.runJavaScript('closeDialog()').then((result) => {
        if (result === '"close dialog success"') {
          console.info('关闭弹窗成功');
        }else {
          console.info('关闭网页成功');
          // 若需要返回上一页，可使用this.webViewController.backward()
          let context = getContext(this) as common.UIAbilityContext;
          context.terminateSelf();
        }
      })
      return true;
    }
    return false;
  }

  build() {
    Column() {
      Web({
        src: $rawfile('H5/index3.html'),
        controller: this.webViewController
      })
        .javaScriptAccess(true)
        .width('80%')
        .height('50%')
        .backgroundColor('#CCF4F5')
        .domStorageAccess(true)
    }
    .backgroundColor('#CCF4F5')
    .justifyContent(FlexAlign.Center)
    .width('100%').height('100%')
  }
}
```
