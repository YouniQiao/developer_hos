## 场景介绍

前后台切换相机继续预览是拍摄美化类应用的高频使用场景之一。如用户使用相机时需要临时进入后台，从后台返回前台时需要相机继续预览。

本示例主要介绍[@Entry](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-entry#entry)场景、[Navigation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation)场景、使用[Emitter](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-emitter)线程订阅三种方法如何解决前后台切换后相机还能继续预览。

## 实现思路

1. 页面被[@Entry](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-entry#entry)组件修饰的场景下，通过[onPageShow](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-custom-component-lifecycle#onpageshow)和[onPageHide](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-custom-component-lifecycle#onpagehide)进行判断。

   ```
   @Entry
   @Component
   struct Index {
     async onPageShow() {
       cameraCreate(this.surfaceId, context);
     }

     async onPageHide() {
       releaseCamera();
     }

     build() {
      // ...
     }
   }
   ```
2. 使用[Navigation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation)导航框架管理页面的场景下，通过[onShown](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ui-design-hdsnavdestination#section18466638161314)和[onHidden](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ui-design-hdsnavdestination#section526154017153)进行判断。

   ```
   @Component
   struct NavigationPage {
     build() {
       NavDestination() {
        // ...
       }
       .backgroundColor(Color.Black)
       .height(Constants.FULL_PERCENT)
       .width(Constants.FULL_PERCENT)
       .hideTitleBar(true)
       .padding({
         top: this.getUIContext().px2vp(this.topRectHeight),
         bottom: this.getUIContext().px2vp(this.bottomRectHeight)
       })
       .onShown(() => {
         cameraCreate(this.surfaceId, context);
       })
       .onHidden(() => {
         releaseCamera();
       });
     }
   }
   ```
3. Foreground和Background状态分别在UIAbility实例切换至前台和切换至后台时触发，对应于[onForeground()](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/uiability-lifecycle#onforeground)回调和[onBackground()](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/uiability-lifecycle#onbackground)回调，因此可以在触发回调后使用[Emitter](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-emitter)进行线程间通信，通知相机初始化资源和释放资源。

   步骤一：在EntryAbility实现Emitter发送事件。

   ```
   onForeground(): void {
     let event: emitter.InnerEvent = {
       eventId: 1,
       priority: emitter.EventPriority.LOW
     };
     emitter.emit(event);
   }

   onBackground(): void {
     let event: emitter.InnerEvent = {
       eventId: 2,
       priority: emitter.EventPriority.LOW
     };
     emitter.emit(event);
   }
   ```

   步骤二：在相机页面实现Emitter订阅事件。

   ```
   @Component
   export struct EmitterPage {
     private event1: emitter.InnerEvent = {
       eventId: 1
     };
     private event2: emitter.InnerEvent = {
       eventId: 2
     };

     aboutToAppear() {
       let foregroundCallback = (): void => {
         cameraCreate(this.surfaceId, context);
       };

       let backgroundCallback = (): void => {
         releaseCamera();
       };
       emitter.on(this.event1, foregroundCallback);
       emitter.on(this.event2, backgroundCallback);
     }
     build() {
       // ...
     }
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
