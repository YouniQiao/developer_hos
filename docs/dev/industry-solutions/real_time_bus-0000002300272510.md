## 场景介绍

实时公交服务是公交地铁类应用的高频使用场景之一，如用户需要查看附近公交线路以及公交到站时间。

本示例基于[@ohos.geoLocationManager](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-geolocationmanager)获取当前位置信息，推送附近公交线路，实现实时公交服务能力。

## 实现思路

1. 申请位置权限。

   ```
   let atManager: abilityAccessCtrl.AtManager = abilityAccessCtrl.createAtManager();
   let requestStatus = await atManager.requestPermissionsFromUser(context, PERMISSION_ARRAY);
   ```
2. 调用[geoLocationManager.getCurrentLocation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-geolocationmanager#geolocationmanagergetcurrentlocation-2)方法，获取当前位置经纬度信息。

   ```
   geoLocationManager.getCurrentLocation(request).then((result) => {
     // ...
   })
   ```
3. 调用[geoLocationManager.getAddressesFromLocation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-geolocationmanager#geolocationmanagergetaddressesfromlocation)方法，将经纬度信息转化为地理描述。

   ```
   geoLocationManager.getAddressesFromLocation({ latitude: result.latitude, longitude: result.longitude },
     (err, data) => {
       if (err) {
         // ...
       } else {
         this.province = data[0].subAdministrativeArea ?? '';
       }
     });
   ```
4. 根据地理描述推送附近公交信息。

   ```
   Column() {
     ForEach(this.stationList, (station: StationModel, index: number) => {
       Card({ station: station });
       // ...
     })
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。

## 权限说明

获取设备模糊位置信息使用权限：[ohos.permission.APPROXIMATELY\_LOCATION](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all-user#ohospermissionapproximately_location)。
