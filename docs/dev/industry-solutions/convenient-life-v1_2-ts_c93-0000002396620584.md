## 问题现象

用户在应用中点击清理缓存功能，应用执行清理缓存动作后，界面上仍然显示有缓存数据。多次执行后，显示缓存数据清零。

## 背景知识

* [应用文件概述](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/app-file-overview)：应用文件包括应用安装文件、应用资源文件和应用缓存文件，文件所有者是应用。
* [应用文件访问(ArkTS)](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/app-file-access)：应用需要对应用文件目录下的应用文件进行查看、创建、读写、删除、移动、复制、获取属性等访问操作。
* [应用及文件系统空间统计](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/app-fs-space-statistics)：在系统中，可能出现系统空间不够或者cacheDir等目录受系统配额限制等情况，需要应用开发者关注系统剩余空间，同时控制应用自身占用的空间大小。
* [storageStatistics.getCurrentBundleStats](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-storage-statistics#storagestatisticsgetcurrentbundlestats9)：应用异步获取当前应用存储空间大小（单位为Byte），以Promise方式返回。
* [storageStatistics.getCurrentBundleStats](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-storage-statistics#storagestatisticsgetcurrentbundlestats9-1)：应用异步获取当前应用存储空间大小（单位为Byte），以callback方式返回。
* [BundleStats](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-storage-statistics#bundlestats9)：

  | 名称 | 类型 | 必填 | 说明 |
  | --- | --- | --- | --- |
  | appSize | number | 是 | 应用安装文件大小（单位为Byte）。 |
  | cacheSize | number | 是 | 应用缓存文件大小（单位为Byte）。 |
  | dataSize | number | 是 | 应用文件存储大小（除应用安装文件和缓存文件）（单位为Byte）。 |

   以上日志信息可以看到，应用分别获取了两次GetBundleStats信息。分别对应当为用户进入清理页面时，应用第一次调用来查询当前应用的缓存数据。当用户点击清理后，应用第二次调用查询应用的缓存数据。而在第二次调用前，应用仍然有缓存数据在写入，因此这里可以看到缓存数据不为零。因此，在界面上就可以看到仍然有缓存数据。

## 分析结论

storageStatistics.getCurrentBundleStats为异步调用，当清理结束时，仍然有新的缓存数据在产生。此时，再调用该接口仍然可以获取到缓存数据。

## 修改建议

* 在清理时调用getCurrentBundleStats接口获取缓存数据，用于展示。清理结束后，不用再调用该接口。
* 清理时，通过getCurrentBundleStats判断当前缓存是否为空，不为空时，继续执行。示例代码：

  ```

  }
  ```

完整的示例代码：

```
import { statfs,fileIo  } from '@kit.CoreFileKit';
import { BusinessError } from '@kit.BasicServicesKit';
import { application, common, contextConstant } from '@kit.AbilityKit';

@Entry
@Component
struct CacheStorageDemo {
  @State message: string = '点击开始清理应用缓存';

  build() {
    RelativeContainer() {
      Text(this.message)
        .fontSize('30fp')
        .fontWeight(FontWeight.Bold)
        .alignRules({
          center: { anchor: '__container__', align: VerticalAlign.Center },
          middle: { anchor: '__container__', align: HorizontalAlign.Center }
        })
        .onClick(async () => {
          let context = this.getUIContext().getHostContext() as common.UIAbilityContext;
          let path = context.filesDir;
          statfs.getFreeSize(path, (err: BusinessError, number: number) => {
            if (err) {
              console.error(`Invoke getFreeSize failed, code is ${err.code}, message is ${err.message}`);
            } else {
              console.info(`Invoke getFreeSize succeeded, size is ${number}`);
            }
          });

          let paths: Array<string> = [];
          let moduleContext: common.Context;
          moduleContext = await application.createModuleContext(context, 'entry');
          // 缓存路径/data/storage/el2/base/cache
          paths.push(moduleContext.cacheDir);
          // 缓存路径/data/storage/el2/base/haps/entry/cache
          paths.push(context.cacheDir);
          // 缓存路径/data/storage/el1/base/cache
          moduleContext.area = contextConstant.AreaMode.EL1;
          paths.push(moduleContext.cacheDir);
          // 缓存路径/data/storage/el1/base/haps/entry/cache
          context.area = contextConstant.AreaMode.EL1;
          paths.push(context.cacheDir);

          // 遍历paths数组，彻底删除每个文件夹下的缓存数据
          for (let i = 0; i < paths.length; i++) {
            let path = paths[i];
            fileIo.listFile(path).then((filenames) => {
              for (let i = 0; i < filenames.length; i++) {
                let dirPath = path + '/' + filenames[i];
                console.info(dirPath);
                // 判断是否为文件夹
                let isDirectory: boolean = false;
                try {
                  isDirectory = fileIo.statSync(dirPath).isDirectory();
                } catch (err) {
                  console.error(`remove file failed with error message: ${err.message}`);
                }

                if (isDirectory) {
                  fileIo.rmdirSync(dirPath);
                } else {
                  fileIo.unlink(dirPath).then(() => {
                    console.info('remove file succeed');
                  }).catch((err: Error) => {
                    console.error(`remove file failed with error message: ${err.message}`);
                  });
                }
              }
            });
          }
          this.message = '缓存清理完成';
        })
    }
    .height('100%')
    .width('100%')
  }
}
```
