## 问题现象

元服务未发布上架前，如何调试隐私声明弹框？

## 解决方案

1. 本地调试时可通过手动预置隐私链接信息模拟接入隐私托管和隐私管理服务，具体配置步骤参考[未上架应用接入隐私管理服务](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/store-privacy#section133476117117)。
2. 将待调试元服务安装到真机中，通过“hdc shell aa start -a EntryAbility -b bundleName”打开元服务，即可使用预置的隐私链接信息测试相关服务接口。
3. 若首次在真机上运行时未弹出隐私声明弹框，可通过如下方法排查解决：
   * 在真机上通过“设置->应用和元服务->元服务”中移除元服务，重新部署启动元服务，查看是否可弹框。
   * 在元服务安装部署一段时间后（比如5s），通过“负一屏->服务市场”中打开对应的元服务，查看是否可弹框。
   * 若经过上述尝试后，仍无法正常弹框，则需要获取相应日志做进一步分析，具体操作如下：
     + 应用市场->我的->帮助和反馈->问题与建议。
     + 在问题与建议页面填写问题信息，包含问题类型、问题描述、联系方式、共享日志勾选（一定要勾选共享应用日志才能上报日志）。
     + 点击“提交”按钮，提交反馈问题和日志。
     + 将生成的问题编号一起反馈。
