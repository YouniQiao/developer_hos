## 场景介绍

Navigation实现闪屏页是影音娱乐类应用中的典型场景之一，如应用启动时使用闪屏页过渡。

本示例基于[Navigation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation)组件实现闪屏页效果。

## 实现思路

1. 通过[Image](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-image)组件插入图片。

   ```
   // 图片
   Image($r('app.media.splash_bg'))
     .height(Constants.FULL)
     .width(Constants.FULL);
   ```
2. 通过[Navigation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation)组件设置闪屏时间为2秒，2秒后跳转至应用主页。

   ```
   // 导航组件包裹代码
   Navigation(this.pageInfos) {
     Flex({ direction: FlexDirection.Column, alignItems: ItemAlign.Center }) {
       Column() {
         Image($r('app.media.splash_bg'))
           .height(Constants.FULL)
           .width(Constants.FULL);
       }
       .flexGrow(1);
     }
     .height(Constants.FULL)
     .width(Constants.FULL);
   }
   .hideToolBar(true)
   .hideNavBar(this.isHideNavBar);

   // 两秒后跳转到主页
   this.timeOutId = setTimeout(() => {
     this.isHideNavBar = true;
     this.pageInfos.pushPathByName('MainPage', null, false);
   }, 2000);
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
