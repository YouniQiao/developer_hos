## 场景介绍

应用主题版本切换是各类应用中高频使用场景之一，如用户可自行选择切换标准版、大字版（关怀版、老年版）等主题版本，也可用于节日庆典主题更换等场景，提升用户体验与界面美感。

本示例使用[PersistentStorage](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-state-management#persistentstorage)实现数据持久化，并控制[Navigation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation)首页渲染，实现不同主题切换的功能。

## 实现思路

1. 初始化mode之后，通过[@StorageLink](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-appstorage#storagelink)双向绑定变量，即可持久记录所选择的主题版本。
2. 通过变量mode来控制[Navigation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation)根节点渲染，实现切换不同主题的功能。

```
// 初始化变量
PersistentStorage.persistProp('mode', STANDARD_MODE);

// 双向绑定变量
@StorageLink('mode') mode: number = STANDARD_MODE;

Navigation(this.pageStack) {
  if (this.mode === STANDARD_MODE) {
    HomeStandard();
  } else {
    HomeLargeWord();
  }
}
```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
