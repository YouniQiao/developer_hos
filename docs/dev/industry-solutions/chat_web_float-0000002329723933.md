## 场景介绍

历史网页浮窗是社交通讯类应用的高频使用场景之一，如用户在浮窗保存访问过的网页链接，点击浮窗进入历史网页链接列表页，点击链接可继续访问该网页。

本示例基于[Navigation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation)、[Stack](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-service-widget-container-stack)组件实现浮窗功能，通过[LocalStorage](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-localstorage)保存网页链接。

## 实现思路

1. 通过[Navigation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation)、[Stack](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-service-widget-container-stack)组件自定义浮窗，并实现吸边效果。

   ```
   Stack() {
     Navigation(this.navPathStack) {
       // To do sth.
     }
     // 自定义浮窗
     Row() {
       Image()
     }
     .gesture(
       // 实现浮窗拖动和吸边效果
       PanGesture()
         .onActionEnd((event: GestureEvent) => {
           // 拖动停止时，计算吸边位置
         })
         .onActionUpdate((event: GestureEvent) => {
           // 计算浮窗位置
         })
       )
         .onClick(() => {
           // 点击打开web链接列表页
         })
   }
   .size({ width: '100%', height: '100%' })
   ```
2. 通过[LocalStorage](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-localstorage)和[@LocalStorageLink](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-localstorage#localstoragelink)实现历史链接的保存和浮窗隐藏控制。

   ```
   @LocalStorageLink('webInfos') webInfos: WebInfo[] = [];
   @LocalStorageLink('showFloat') showFloat: boolean = false;

   refreshFloatVisible() {
     this.showFloat = this.webInfos.length > 0;
   }

   NavDestination() {
     // 聊天页面UI
   }
     .onWillShow(() => {  // 通过NavDestination生命周期回调，刷新浮窗的隐藏状态
       this.refreshFloatVisible();
     })
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
