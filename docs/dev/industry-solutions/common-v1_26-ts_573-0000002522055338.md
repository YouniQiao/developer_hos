## 问题现象

使用接口[networkManager.addDomainFilterRule](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-enterprise-networkmanager#networkmanageradddomainfilterrule)添加过滤规则，设置白名单，调用networkManager.getDomainFilterRules发现设置白名单成功，但是实际去访问不在白名单的域名，仍然可以访问。

## 解决方案

1. 查看设备IP地址：为设备添加域名过滤规则，从API version 22开始，支持IPv4和IPv6。API version 21及之前版本，仅支持IPv4。
2. 清空缓存，确保不受缓存影响，特殊情况下可通过恢复出厂设置彻底清缓存。
3. 添加过滤规则时可以指定规则链，且支持使用IP段来做拦截，具体请参考[domainfilterrule](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-enterprise-networkmanager#domainfilterrule)。
