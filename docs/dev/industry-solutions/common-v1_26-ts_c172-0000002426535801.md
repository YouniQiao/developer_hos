## 问题现象

快速连续点击拍照按钮时，应用发生闪退。

## 背景知识

* CppCrash进程崩溃检测基于操作系统信号机制，目前支持的崩溃信号参考[Cpp Crash（进程崩溃）检测](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cppcrash-guidelines)。
* CppCrash日志规格说明可以参考[日志规格](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cppcrash-guidelines#日志规格)说明。

* [packToFile](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-image-imagepacker#packtofile13)：指定编码参数，将PixelMap直接编码进文件。使用Promise形式返回结果。
* [crop](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-image-pixelmap#crop9)：根据输入的尺寸对图片进行裁剪，使用Promise形式返回。
* [async/await](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/async-concurrency-overview#asyncawait)：async/await是用于处理异步操作的Promise语法糖，使编写异步代码更加简单和易读。使用async关键字声明异步函数，并使用await关键字等待Promise的解析（完成或拒绝），以同步方式编写异步操作的代码。

2. 查看故障堆栈，应用在调用packToFile函数时出现内存访问异常，需要排查在调用packToFile函数时编码的PixelMap资源是否被释放和修改。
3. 排查应用在调用packToFile函数前是否调用crop函数修改PixelMap。

## 分析结论

在调用packToFile函数时编码的PixelMap资源被修改，导致应用闪退。

## 修改建议

使用await等待crop函数执行结束，再执行packToFile函数。

```
import { image } from '@kit.ImageKit';
import { fileIo as fs } from '@kit.CoreFileKit';
import { BusinessError } from '@kit.BasicServicesKit';

async function Packing(context: Context) {
  const resourceMgr = context.resourceManager;
  const rawFile = await resourceMgr.getRawFileContent('test.jpg');
  let ops: image.SourceOptions = {
    sourceDensity: 98,
  };
  let imageSource: image.ImageSource = image.createImageSource(rawFile.buffer as ArrayBuffer, ops);
  let pixelMap: image.PixelMap = await imageSource.createPixelMap();
  const imagePackerApi: image.ImagePacker = image.createImagePacker();
  let funcName = 'Packing';
  if (imagePackerApi != null) {
    try {
      let opts: image.PackingOption = { format: 'image/jpeg', quality: 98 };
      await pixelMap.crop({ x: 1, y: 1, size: { width: 50, height: 50 } });
      const path: string = context.cacheDir + '/pixel_map.jpg';
      let file = fs.openSync(path, fs.OpenMode.CREATE | fs.OpenMode.READ_WRITE);
      imagePackerApi.packToFile(pixelMap, file.fd, opts).then(() => {
        // 直接编码进文件。
      }).catch((error: BusinessError) => {
        console.error(funcName, `Failed to pack the image.code ${error.code},message is ${error.message}`);
      }).finally(() => {
        fs.closeSync(file.fd);
      });
    } catch (error) {
      console.error(funcName, `Failed to image.code ${error.code},message is ${error.message}`);
    }
  }
}
```
