## 问题现象

进入Web页面时，页面顶部的标题内容快速变化。

![](./img/12d224ea.png "点击放大")

## 背景知识

* [Web](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-basic-components-web)：提供具有网页显示能力的Web组件。
* [onTitleReceive](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-basic-components-web-events#ontitlereceive)：当页面文档标题<title>元素发生变更时，触发回调。若当前页面未显示设置标题，ArkWeb将在加载完成前基于页面的URL生成标题并返回给应用。
* [onPageEnd](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-basic-components-web-events#onpageend)：网页加载完成时触发该回调，且只在主frame触发，iframe或者frameset的内容加载时不会触发此回调。
* [onLoadIntercept](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-basic-components-web-events#onloadintercept10)：当Web组件加载URL之前触发该回调，用于判断是否阻止此次访问。

## 问题定位

查看Web组件相关设置，发现使用了onTitleReceive方法动态获取页面标题并实时展示，且没有使用onPageEnd在页面加载完成后再显示标题。

```
Web({ src: this.url, controller: this.controller })
  .onTitleReceive(()=>{
    // 获取页面标题
  })
  // 未使用onPageEnd
```

## 分析结论

应用使用了onTitleReceive方法动态获取页面标题并实时展示，页面跳转到目的页面的过程中需要跳转到其他的页面，页面标题实时显示从而发生闪变。

## 修改建议

页面加载过程中不显示标题，通过onLoadIntercept判断是否跳转到目的网页，并通过onPageEnd在页面加载完成后再显示标题。

```
import { webview } from '@kit.ArkWeb';

@Entry
@Component
struct Index {
  controller: webview.WebviewController = new webview.WebviewController();
  @State title: string = '';
  @State lastTitle: string = '';
  @State url: string = '';

  build() {
    Scroll() {
      Column() {
        Text(this.lastTitle)
          .height('5%')
          .width('100%')
          .textAlign(TextAlign.Center)
          .fontSize(20);
        Web({ src: $rawfile('transitions.html'), controller: this.controller })
          .height('95%')
          .width('100%')
          .fileAccess(true)
          .geolocationAccess(false)
          .initialScale(85)
          .onTitleReceive((event) => {
            if (event) {
              this.title = event.title;
            }
          })
          .onLoadIntercept((event) => {
            this.url = event.data.getRequestUrl(); // 获取当前页面的URL
            return false;
          })
          .onPageEnd(() => {
            // 跳转到目的页面并加载完成后才显示标题
            if (this.url === 'resource://rawfile/title.html') {
              this.lastTitle = this.title;
            }
          });
      }
      .height('100%')
      .width('100%');
    }
    .width('100%')
    .height('100%');
  }
}
```

resource://rawfile/transitions.html：

```
<!DOCTYPE html>
<html lang="zh">
<head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
    <title>过渡页面</title>
</head>
<body>

<script>
    setTimeout(() => {
        // "./text.html"需替换为开发者需要的网页资源文件
        window.location.href = "./title.html";
    }, 300);
</script>

</body>
</html>
```
