## 场景介绍

运动321倒计时是运动健康类应用中的高频使用场景之一，如用户跑步时，点击开始按钮，显示跑步开始倒计时。

本示例基于[animateTo](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#animateto)实现运动倒计时的数字动画效果。

## 实现思路

1. 使用[animateTo](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#animateto)，在显示动效的闭包函数中，改变数字和大小，完成倒计时中一个数字的动画效果。

   ```
   this.getUIContext().animateTo({
     // 动画效果相关参数配置
   }, () => {
     this.countdownFontSize = $r('app.float.font_size_240');
     this.countdownNumber--;
   })
   ```
2. 设置[AnimateParam对象说明](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-explicit-animation#animateparam对象说明)中的onFinish参数，在onFinish回调中嵌套animateTo，实现下一个数字动画效果。

   ```
   this.getUIContext().animateTo({
     duration: this.animationDuration,
     playMode: PlayMode.Normal,
     curve: Curve.EaseInOut,
     onFinish: () => {
       this.countdownFontSize = $r('app.float.font_size_150');

       this.getUIContext().animateTo({
         // 动画效果相关参数配置
       }, () => {
         this.countdownFontSize = $r('app.float.font_size_240');
         this.countdownNumber--;
       })
     }
   }, () => {
     this.countdownFontSize = $r('app.float.font_size_240');
     this.countdownNumber--;
   })
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
