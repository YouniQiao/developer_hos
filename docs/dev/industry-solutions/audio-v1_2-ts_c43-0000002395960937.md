## 问题现象

应用打开正常显示，点击某一视频横屏播放后，返回上一页，页面显示错位异常。

## 背景知识

* [横竖屏切换](https://developer.huawei.com/consumer/cn/doc/best-practices/bpta-landscape-and-portrait-development)：横竖屏切换功能实现应用内既支持竖屏显示也支持横屏显示的效果。横竖屏切换时，页面尺寸同步发生变化。
* [窗口方向](https://developer.huawei.com/consumer/cn/doc/best-practices/bpta-multi-device-window-direction)：横竖屏切换功能是指应用内同时支持竖屏和横屏显示。当不同页面需要采用不同显示方向时，需在应用逻辑中动态调整窗口方向以实现该效果。

## 问题定位

1. 当应用无论竖屏还是横屏启动，页面显示正常，说明应用初次加载监听了屏幕大小，并正确设置了页面尺寸。
2. 当应用从横屏的视频页面切换回竖屏状态时，建议检查代码中是否使用on('windowSizeChange')监听了设备尺寸变化，判断当前是横屏还是竖屏。
3. 检查异常的页面是否通过判断当前是横屏还是竖屏，分别展示不同的布局。

## 分析结论

应用在进入视频横屏时监听了屏幕方向变化，退出视频播放页面时，页面还根据横屏状态获取的屏幕尺寸进行页面布局，导致应用在返回上一页时，页面显示错位异常现象。

## 修改建议

在具体需要实现横竖屏切换的页面上，需要在进入对应的页面时，采用window窗口提供的设置窗口方向的能力，通过setPreferredOrientation()将窗口显示的方向修改为横屏或竖屏的状态，退出该页面时同样需要使用setPreferredOrientation()更改横竖屏状态。

```
import { window } from '@kit.ArkUI';

@Entry
@Component
struct VideoPage {
  @State videoSrc: Resource = $rawfile('example.mp4'); // 实际开发替换为实际视频资源
  @State curRate: PlaybackSpeed = PlaybackSpeed.Speed_Forward_1_00_X;
  private isAutoPlay: boolean = true;
  private showControls: boolean = true;
  private isShortcutKeyEnabled: boolean = false;
  controller: VideoController = new VideoController();

  build() {
    Column() {
      Video({
        src: this.videoSrc,
        currentProgressRate: this.curRate, // 设置播放速度
        controller: this.controller,
      })
        .width('100%')
        .height(600)
        .autoPlay(this.isAutoPlay)
        .controls(this.showControls)
        .enableShortcutKey(this.isShortcutKeyEnabled)
        .onFullscreenChange((e?: FullscreenObject) => {
          if (e != undefined) {
            if (e.fullscreen == true) {
              window.getLastWindow(this.getUIContext().getHostContext(), (err, data) => { // 获取window实例
                let windowClass = data;
                windowClass.setPreferredOrientation(window.Orientation.LANDSCAPE); // 点击全屏后，旋转为横屏显示
              });
            } else {
              window.getLastWindow(this.getUIContext().getHostContext(), (err, data) => { // 获取window实例
                let windowClass = data;
                windowClass.setPreferredOrientation(window.Orientation.PORTRAIT); // 退出全屏后，旋转为竖屏显示
              });
            }
          }
        });
    };
  }
}

interface FullscreenObject {
  fullscreen: boolean;
}
```
