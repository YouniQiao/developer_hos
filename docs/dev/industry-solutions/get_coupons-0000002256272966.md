## 场景介绍

优惠券防重复领取是购物比价类应用的高频使用场景之一，如商家需要限制用户领取优惠券的次数。

本示例基于[应用设备状态检测](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/devicesecurity-deviceverify-api)实现一台设备只能领取一次优惠券的效果。

## 实现思路

1. 查询设备的token。
2. 在DeviceSecurity服务器中查询该设备的标记状态，被标记代表已经领取过优惠券，不可再领取。

```
getDeviceStatus(): Promise<string> {
  // 获取设备的token
  this.queryDeviceToken();
  // 验证该设备是否为真实的华为设备，需要携带token
  this.verifyDeviceToken();
  // 查询设备的标记状态
  this.queryDeviceMarkStatus();
  // 更新设备标记状态，被标记代表已经领取过优惠券
  this.updateDeviceMarkStatus(true, false);
}
```

## 约束与限制

* 本示例支持API Version 21 Release及以上版本。
* 本示例支持HarmonyOS 6.0.1 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.1 Release及以上版本进行编译运行。
