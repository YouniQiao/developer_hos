## 场景介绍

PDF在线安全预览是综合办公类应用的高频场景之一，如向客户展示新产品特性时，需要添加水印、防截屏和防下载等功能来保证安全性。

本示例利用[Web](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-web)组件实现PDF在线加载显示，[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)画布实现蒙版水印，通过设置[隐私窗口权限](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissionprivacy_window)实现防截屏，顶部导航栏自定义实现防下载，切合实际应用场景中安全预览的需求。

## 实现思路

* 通过[Web](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-web)组件实现网络PDF文件在线加载显示。
* 通过[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)画布在所占区域上均匀绘制水印，最后在Web组件通过默认属性overlay实现组件级水印效果。
* 声明隐私界面后，在onWindowStageCreate生命周期函数中获取windowClass对象并保存，在需要防截屏的页面获取对象并调用[setWindowPrivacyMode()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-window-window#setwindowprivacymode9)方法设置隐私模式，实现防截屏效果。
* 通过在URL末尾添加'#toolbar=0'隐藏Web组件导航栏（其中包含下载按钮），自定义导航栏重写下载按钮，实现防下载效果。

```
// Web组件
Web({ src: this.path, controller: this.webController })
  .overlay(createWaterMarkView()) // 水印效果

// 蒙版水印
Canvas(this.waterMarkContext)
  .onReady(() => {
    // 水印具体实现
  })

// 防截屏
async onWindowStageCreate(windowStage: window.WindowStage): Promise<void> {
  windowStage.getMainWindow((err: BusinessError, data) => {
    // 获取windowClass对象并保存
    const windowClass = data;
    AppStorage.setOrCreate('windowClass', windowClass);
  })
}

aboutToAppear(): void {
  // 具体页面设置隐私模式
  this.isPrivacyMode = true;
  if (this.windowClass) {
    this.windowClass.setWindowPrivacyMode(this.isPrivacyMode);
  }
}

// 防下载
Image($r('app.media.download'))
  .onClick(() => {
    this.getUIContext().getPromptAction().showToast({ message: $r('app.string.Prohibited') }); // 自定义禁止下载弹窗
  })
```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。

## 权限说明

* 申请网络权限：[ohos.permission.INTERNET](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissioninternet)。
* 声明隐私窗口权限：[ohos.permission.PRIVACY\_WINDOW](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissionprivacy_window)。
