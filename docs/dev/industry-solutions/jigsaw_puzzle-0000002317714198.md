## 场景介绍

七巧板拼图是儿童教育类应用中的典型场景之一，用户可以拖拽旋转七块拼图拼成特定图形。

本示例基于[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)组件、[animateTo](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#animateto)和[PanGesture](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-gestures-pangesture)实现七巧板拼图，也适用于其他拼图类游戏的绘制。

## 实现思路

1. 使用[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)组件绘制七巧板七块拼图。

   ```
   @Builder
   TriangleShape(color: string, angle: number, strokeWidth: number) {
     Canvas() {
       Path()
         .commands(`${CanvasConstants.PIECES_START}${CanvasConstants.PIECES_LENGTH_0} ${CanvasConstants.PIECES_LENGTH_0} ${CanvasConstants.PIECES_STRAIGHT_LINE}${CanvasConstants.PIECES_LENGTH_100} ${CanvasConstants.PIECES_LENGTH_100} ${CanvasConstants.PIECES_STRAIGHT_LINE}${CanvasConstants.PIECES_LENGTH_0} ${CanvasConstants.PIECES_LENGTH_100} ${CanvasConstants.PIECES_END}`)
         .strokeWidth(strokeWidth)
         .stroke($r('app.color.black'))
         .fill(color)
         .rotate({
           z: 1,
           angle: angle,
         })
     }
     .width(20)
     .height(100)
   }
   ```
2. 通过[animateTo](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#animateto)、[PanGesture](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-gestures-pangesture)实现七巧板拖拽和旋转功能。

   ```
   PanGesture()
     .onActionStart(() => {
       // 拖拽开始
       this.currentX = this.getWidthScaledVp(this.offsetXItem[index]);
       this.currentY = this.getHeightScaledVp(this.offsetYItem[index]);
     })
     .onActionUpdate((info: GestureEvent) => {
       // 拖拽过程中更新位置
       let offsetY = this.currentY + info.offsetY;
       let offsetX = this.currentX + info.offsetX;
     })
     .onActionEnd(() => {
       // 拖拽结束
       this.checkPiecePlacement(item, this.offsetXItem[index], this.offsetYItem[index],
         this.rotateAngleTimes[index] % PiecesConstants.ROTATE_ANGLE_TIMES)
     })
   ```
3. 检测拼图是否正确放置，若正确放置则会自动吸附目标图形。

   ```
   checkPiecePlacement(piece: PuzzlePieceItem, x: number, y: number, angle: number) {
     const TARGET_X = piece.targetX
     const TARGET_Y = piece.targetY
     this.times[piece.id] = []
     for (let times = 0; times < TARGET_X.length; times++) {
       let distance =
         Math.sqrt(Math.pow(x - TARGET_X[times], 2) + Math.pow(y - TARGET_Y[times], 2))
       if (distance < 30 && angle === piece.targetAngle[times]) {
         piece.isPlaced = true
         this.times[piece.id].push(times)
         this.animatePieceToTarget(piece, TARGET_X[times], TARGET_Y[times])
       } else {
         if (this.times[piece.id].length === 0) {
           piece.isPlaced = false
         }
       }
     }
     this.checkAllPiecesPlaced()
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
