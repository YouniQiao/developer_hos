## 场景介绍

儿童绘画板是儿童教育类应用中的高频使用场景之一。

本示例基于[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)和[触摸事件](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-events-touch)实现儿童绘画板功能，支持选择画布背景色、画笔颜色、画笔粗细和简笔画图案。

## 实现思路

1. 使用[bindSheet](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-sheet-transition#bindsheet)和[Grid](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-grid)实现自定义颜色选择弹窗。
2. 基于[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)组件构建画板，通过监听[onTouch](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-events-touch#ontouch)触摸事件获取当前触点坐标，并使用[CanvasRenderingContext2D](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-canvasrenderingcontext2d)控制画笔路径、颜色及粗细。

```
Canvas(this.context)
  .onTouch((event) => {
    if (event.type === TouchType.Down) {
      this.x = event.touches[0].x;
      this.y = event.touches[0].y;
      this.context.beginPath();
      this.tempPath = new Path2D();
      this.tempPath.moveTo(this.x, this.y);
    }
    if (event.type === TouchType.Move) {
      if (this.isEraserMode) {
        // 橡皮擦模式
        this.context.clearRect(event.touches[0].x, event.touches[0].y, 20, 20);
      } else {
        this.isEmpty = false;
        // 画笔路径
        this.x = event.touches[0].x;
        this.y = event.touches[0].y;
        // 画笔颜色
        this.context.strokeStyle = this.penColorList[this.penColor];
        // 画笔粗细
        this.context.lineWidth = this.paintSize;
        this.tempPath.lineTo(this.x, this.y);
        this.context.stroke(this.tempPath);
      }
    }
  });
```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
