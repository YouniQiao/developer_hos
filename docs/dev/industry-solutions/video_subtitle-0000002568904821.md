## 场景介绍

自动识别并添加字幕是视频播放类场景的高频使用场景之一。

本示例主要基于三方库[@ohos/mp4parser(V2.0.6)](https://ohpm.openharmony.cn/#/cn/detail/@ohos%2Fmp4parser)从视频中提取音频，基于[speechRecognizer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/hms-ai-speechrecognizer)将音频转换为文本，实现了语音识别自动生成视频字幕功能。

## 实现思路

1. 使用三方库[@ohos/mp4parser(V2.0.6)](https://ohpm.openharmony.cn/#/cn/detail/@ohos%2Fmp4parser)从视频文件中提取出PCM音频文件，保存到沙箱目录下。

   ```
   // 从视频中提取音频
   let callback: ICallBack = {
     callBackResult: (code: number) => {
       if (code === 0) {
         hilog.info(DOMAIN, TAG, 'Extract audio succeed');
       } else {
         hilog.info(DOMAIN, TAG, 'Extract audio failed');
       }
     }
   };
   // videFilePath：输入视频文件的沙箱路径，audioFilePath：输出音频文件的沙箱路径
   let cmd = `ffmpeg -i ${videoFilePath} -vn -acodec pcm_s16le -ar 16000 -ac 1 -f s16le ${audioFilePath}`;
   MP4Parser.ffmpegCmd(cmd, callback);
   ```
2. 使用[speechRecognizer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/hms-ai-speechrecognizer)将PCM音频转换为文本，并记录每条字幕的起始时间与结束时间。

   ```
   // 语音识别结果回调
   onResult: (sessionId: string, result: speechRecognizer.SpeechRecognitionResult): void => {
     // 根据写入音频数据的长度计算当前时间
     let time = this.getTime(this.writeOffset);
     time = Math.max(0, time - Constants.RECOGNIZE_DELAY);
     // 如果是一条新的字幕，当前时间为这条字幕的起始时间
     if (this.startTime === -1) {
       this.startTime = time;
     }
     // 识别一句完整的字幕，当前时间为这条字幕的结束时间
     if (result.isFinal) {
       if (result.result.length > 0) {
         // 保存字幕的内容，起始时间与结束时间
         let subtitle: Subtitle = { startTime: this.startTime, endTime: time, content: result.result };
         // ...
       }
       this.startTime = -1;
     }
   }
   ```
3. 根据视频当前播放时间，选择对应的字幕显示。

   ```
   // AVPlayer监听资源播放当前时间回调
   this.videoPlayer?.on('timeUpdate', (time: number) => {
     if (this.currSubtitle === undefined || time < this.currSubtitle.startTime || time > this.currSubtitle.endTime) {
       let content = '';
       let length = this.subtitles.length;
       // 搜索当前播放时间对应的字幕
       for (let idx = 0; idx < length; idx++) {
         let start = this.subtitles[idx].startTime;
         let end = this.subtitles[idx].endTime;
         if (time >= start && time <= end) {
           // 获得当前播放时间对应的字幕
           content = this.subtitles[idx].content;
           this.currSubtitle = { content: content, startTime: start, endTime: end };
           break;
         }
       }
     }
   });
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
