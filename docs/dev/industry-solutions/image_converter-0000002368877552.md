## 场景介绍

图片格式转换是拍摄美化类应用的典型场景之一，如用户需将图片PNG格式转换为WebP、JPEG等格式，并保存本地。

本示例基于[image.createImageSource](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-image-f#imagecreateimagesource)方法创建[ImageSource](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-image-imagesource)实例，并使用[packToFile](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-image-imagepacker#packtofile11-1)方法指定编码参数，将ImageSource编码进文件，之后通过[DocumentViewPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-picker#documentviewpicker)将转换格式后的图片保存到本地。

* 图片解码：当前支持图片解码格式包括JPEG、PNG、GIF、WebP、BMP、SVG、ICO、DNG和HEIF（不同硬件设备支持情况可能不同 ）。
* 图片编码：当前支持图片编码格式包括JPEG、WebP、PNG和HEIF（不同硬件设备支持情况不同），用于保存、传输等后续处理。

## 实现思路

1. 根据图片uri创建[ImageSource](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-image-imagesource)实例。

   ```
   public imageToImageSource(imagePath: string): image.ImageSource {
     let imageFile: fs.File | null = null;
     try {
       imageFile = fs.openSync(imagePath, fs.OpenMode.READ_ONLY);
       const STAT = fs.statSync(imagePath);
       const BUFFER = new ArrayBuffer(STAT.size);
       fs.readSync(imageFile.fd, BUFFER);
       const IMAGE_SOURCE = image.createImageSource(BUFFER);
       return IMAGE_SOURCE;
     } catch (error) {
     } finally {
       if (imageFile) {
         try {
           fs.closeSync(imageFile);
         } catch (closeError) {
         }
       }
     }
   }
   ```
2. 使用[ImagePacker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-image-imagepacker)的[packToFile](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-image-imagepacker#packtofile11-1)方法，指定编码参数，将ImageSource编码进指定格式文件。

   ```
   public async imageFormatTrans(imageSource: image.ImageSource, targetPath: string, format: string): Promise<void> {}
   // 关键代码段
   let packOpts: image.PackingOption = {
     format: myFormat,
     quality: myQuality
   }
   let file: fs.File | null = null;
   const IMAGE_PACKER_API: image.ImagePacker = image.createImagePacker();
   file = fs.openSync(targetPath, fs.OpenMode.CREATE | fs.OpenMode.READ_WRITE);
   await IMAGE_PACKER_API.packToFile(imageSource, file.fd, packOpts);
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
