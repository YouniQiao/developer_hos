## 问题现象

在线短视频播放场景，用户切换视频时会短暂显示暂停按钮，影响用户观看体验。

## 背景知识

* [visibility](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-visibility#visibility)：控制组件的显示或隐藏。当未设置visibility时，组件默认为显示。
* [在线短视频流畅切换](https://developer.huawei.com/consumer/cn/doc/best-practices/bpta-smooth-switching)：使用LazyForEach进行数据懒加载，设置cachedCount属性指定缓存数量，搭配组件复用能力以提升性能。
* [controls](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-media-components-video#controls)：设置控制视频播放的控制栏是否显示。

## 分析结论

应用的暂停标志根据Video的播放状态进行显示，但是由于视频切换期间视频播放状态都会进入pause，导致了暂停标志的显示。

## 修改建议

Image组件默认可视状态设置为不可视，通过isPaused变量进行控制，当Video完成视频加载时，将isPaused置为false，实现切换视频时不显示暂停标志。

完整示例参考如下：

```
import { AVDataSource } from './AVDataSource';

@Entry
@Component
struct AudioDemoPage {
  private swiperController: SwiperController = new SwiperController();
  private videoController: VideoController = new VideoController();
  @State isPaused: boolean = false; // 默认为播放状态
  private data: AVDataSource = new AVDataSource();

  async aboutToAppear(): Promise<void> {
    for (let i = 0; i <= 20; i++) {
      this.data.pushData($rawfile('sample-5.mp4')); // 此处请填写实际的播放资源
    }
  }

  build() {
    // Swiper轮播视频
    Swiper(this.swiperController) {
      LazyForEach(this.data, (item: Resource) => {
        Stack() {
          Video({
            src: item,
            controller: this.videoController
          })
            // 在视频准备完成后将状态需修改为播放。
            .onPrepared(() => {
              console.info(`Video prepared.`);
              this.isPaused = false;
            })
            .autoPlay(true)
            .controls(false)
          Image($r('app.media.play')) // 此处请填写实际的按钮图标地址
            .width(60)
            .height(60)
            .visibility(this.isPaused ? Visibility.Visible : Visibility.None)
        }
      }, (item: string, index: number) => JSON.stringify(item) + index)
    }
    // 配置缓存使得视频提前缓存
    .cachedCount(2)
    .width('100%')
    .height('100%')
    .vertical(true)
    .loop(true)
    .curve(Curve.Ease)
    .duration(300)
    .indicator(false)
    .backgroundColor(Color.Black)
    .onClick(() => {
      this.isPaused = !this.isPaused;
      if (this.isPaused) {
        this.videoController.pause();
      } else {
        this.videoController.start();
      }
    })
  }
}
```

AVDataSource.ets文件如下所示，跟页面放在同级文件夹下即可。

```
class BasicDataSource implements IDataSource {
  private listeners: DataChangeListener[] = [];
  private originDataArray: Resource[] = [];

  public totalCount(): number {
    return this.originDataArray.length;
  }

  public getData(index: number): Resource {
    return this.originDataArray[index];
  }

  // 该方法为框架侧调用，为LazyForEach组件向其数据源处添加listener监听
  registerDataChangeListener(listener: DataChangeListener): void {
    if (this.listeners.indexOf(listener) < 0) {
      console.info('add listener');
      this.listeners.push(listener);
    }
  }

  // 该方法为框架侧调用，为对应的LazyForEach组件在数据源处去除listener监听
  unregisterDataChangeListener(listener: DataChangeListener): void {
    const pos = this.listeners.indexOf(listener);
    if (pos >= 0) {
      console.info('remove listener');
      this.listeners.splice(pos, 1);
    }
  }

  // 通知LazyForEach组件需要重载所有子组件
  notifyDataReload(): void {
    this.listeners.forEach(listener => {
      listener.onDataReloaded();
    });
  }

  // 通知LazyForEach组件需要在index对应索引处添加子组件
  notifyDataAdd(index: number): void {
    this.listeners.forEach(listener => {
      listener.onDataAdd(index);
      // 写法2：listener.onDatasetChange([{type: DataOperationType.ADD, index: index}]);
    });
  }

  // 通知LazyForEach组件在index对应索引处数据有变化，需要重建该子组件
  notifyDataChange(index: number): void {
    this.listeners.forEach(listener => {
      listener.onDataChange(index);
      // 写法2：listener.onDatasetChange([{type: DataOperationType.CHANGE, index: index}]);
    });
  }

  // 通知LazyForEach组件需要在index对应索引处删除该子组件
  notifyDataDelete(index: number): void {
    this.listeners.forEach(listener => {
      listener.onDataDelete(index);
      // 写法2：listener.onDatasetChange([{type: DataOperationType.DELETE, index: index}]);
    });
  }

  // 通知LazyForEach组件将from索引和to索引处的子组件进行交换
  notifyDataMove(from: number, to: number): void {
    this.listeners.forEach(listener => {
      listener.onDataMove(from, to);
      // 写法2：listener.onDatasetChange(
      //         [{type: DataOperationType.EXCHANGE, index: {start: from, end: to}}]);
    });
  }

  notifyDatasetChange(operations: DataOperation[]): void {
    this.listeners.forEach(listener => {
      listener.onDatasetChange(operations);
    });
  }
}

export class AVDataSource extends BasicDataSource {
  private dataArray: Resource[] = [];

  public totalCount(): number {
    return this.dataArray.length;
  }

  public getData(index: number): Resource {
    return this.dataArray[index];
  }

  public pushData(data: Resource): void {
    this.dataArray.push(data);
    this.notifyDataAdd(this.dataArray.length - 1);
  }
}
```
