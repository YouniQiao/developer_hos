## 场景介绍

跑马灯横幅通知是综合办公类应用的高频使用场景之一，如应用主页中置顶的告警提示信息，将信息通过跑马灯的形式进行展示，提升用户关注度。

本示例基于组件通用属性[linearGradient](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-gradient-color#lineargradient)和[Text](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-text)组件textOverflow属性，实现了两端渐变的跑马灯横幅效果。

## 实现思路

* 设置[Text](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-text)组件的textOverflow属性，实现跑马灯效果。

  ```
  Text($r('app.string.notification'))
    .textOverflow({ overflow: TextOverflow.MARQUEE })
  ```
* 设置组件通用属性[linearGradient](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-gradient-color#lineargradient)，实现颜色渐变效果。

  ```
  Column() {
  }
    .width(CommonConstants.NOTIFICATION_HEIGHT)
    .height(CommonConstants.FULL_PERCENT)
    .linearGradient({
      direction: GradientDirection.Left, // 渐变方向
      colors: [[$r('app.color.linear_gradient_left'), 0], [$r('app.color.linear_gradient_right'), 1]]
    })
  ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
