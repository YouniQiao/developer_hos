## 问题现象

页面标题处显示链接内容。

![](./img/206b810e.png "点击放大")

## 背景知识

* [Text](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-text)为显示一段文本的组件。

* [Web](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-basic-components-web)组件提供网页显示能力。
* H5页面使用title可设置页面标题，未设置时以链接地址作为标题。
* 使用[onTitleReceive](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-basic-components-web-events#ontitlereceive)可通知应用程序页面document标题已更改，如果加载的页面未设置title元素指定的标题，Web组件将基于URL生成标题并返回给应用程序。

## 问题定位

1. 使用DevEco Testing-实用工具-UIViewer查看页面布局，标题为Text组件。

   ![](./img/9db6dabf.png "点击放大")
2. 查看该Web组件的设置，该Web组件使用onTitleReceive获取H5页面的标题，并通过Text组件显示标题。

   ```
   Column() {
     Text(this.pageTitle)
       .width('100%')
       .height('10%')
       .textAlign(TextAlign.Center);

     Web({ src: $rawfile('video.html'), controller: this.controller })
       .height('90%')
       .width('100%')
       .fileAccess(true)
       .geolocationAccess(false)
       .onTitleReceive((event) => {
         if (event) {
           this.pageTitle = event.title;
         }
       });
   }
   ```
3. 使用Devtools或开发者工具查看该H5页面的设置，该H5页面未使用title设置页面标题。

   ```
   <!DOCTYPE html>
   <html lang="zh">
   <head>
       <meta charset="UTF-8" />
       <meta name="viewport" content="width=device-width, initial-scale=1.0" />
       <!-- 未使用title设置页面标题 -->
   </head>
   </html>
   ```

## 分析结论

H5页面的标题内容未设置，导致页面标题处显示链接内容。

## 修改建议

合理设置H5页面的<title>标签中的标题内容。

```
import { webview } from '@kit.ArkWeb';
import { window } from '@kit.ArkUI';
import { common } from '@kit.AbilityKit';

@Entry
@Component
struct WebVideo {
  controller: webview.WebviewController = new webview.WebviewController();
  @State pageTitle: string = '';

  aboutToAppear(): void {
    let context: common.UIAbilityContext = this.getUIContext().getHostContext() as common.UIAbilityContext;
    window.getLastWindow(context).then((lastWindow) => {
      lastWindow.setWindowLayoutFullScreen(true);
    });
  }

  build() {
    Column() {
      Text(this.pageTitle)
        .width('100%')
        .height('10%')
        .textAlign(TextAlign.Center);

      Web({ src: $rawfile('video.html'), controller: this.controller })
        .height('90%')
        .width('100%')
        .fileAccess(true)
        .geolocationAccess(false)
        .onTitleReceive((event) => {
          if (event) {
            this.pageTitle = event.title;
          }
        });
    }
    .height('100%')
    .width('100%')
    .padding({ top: '123px', bottom: '91px' })
    .expandSafeArea([SafeAreaType.SYSTEM], [SafeAreaEdge.TOP, SafeAreaEdge.BOTTOM]); // 扩展页面区域到导航栏和状态栏
  }
}
```
