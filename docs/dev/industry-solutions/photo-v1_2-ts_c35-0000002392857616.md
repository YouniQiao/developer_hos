## 问题现象

打开应用，点击证件照拍摄按钮，启用前置摄像头拍摄证件照，此时点击页面上的闪光灯按钮，应用闪退。

## 背景知识

* 在开发相机应用时，需要先[申请相机开发的权限](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/camera-preparation)。
* [7400102](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/errorcode-camera#section7400102-非法操作)：未按照既定方式操作。
* [FlashMode](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-e#flashmode)：枚举，闪光灯模式。
* [hasFlash](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-flashquery#hasflash11)：检测是否有闪光灯，通过注册回调函数获取结果。
* [isFlashModeSupported](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-flashquery#isflashmodesupported11)：检测闪光灯模式是否支持。
* [setFlashMode](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-flash#setflashmode11)：设置闪光灯模式。
* [CameraStatusInfo](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-i#camerastatusinfo)：相机管理器回调返回的接口实例，该实例存储相机状态信息。
* [CameraDevice](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-i#cameradevice)：相机设备信息。其中cameraId表示相机ID。
* [CameraStatus](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-e#camerastatus)：枚举，相机状态。

3. 查看相关代码，结合以上报错日志，发现问题出现在setFlashMode方法，而"com.hm.example is about to exit due to RuntimeError"表示应用闪退是因为运行时异常。

   ```
   // CameraService.ets
   hasFlashFn(flashMode: camera.FlashMode): void {
     // 检测是否有闪光灯
     let hasFlash = this.session?.hasFlash();
     console.info(`hasFlash success, hasFlash: ${hasFlash}`);
     // 检测闪光灯模式是否支持
     let isFlashModeSupported = this.session?.isFlashModeSupported(flashMode);
     console.info(`isFlashModeSupported success, isFlashModeSupported: ${isFlashModeSupported}`);
     // 设置闪光灯模式
     this.session?.setFlashMode(flashMode);
   }
   ```

## 分析结论

前置摄像头无法使用后置的闪光灯补光，点击闪光灯按钮时，应用没有判断是否支持闪光灯就设置闪光灯模式，也没有进行异常捕获，导致应用闪退。

## 修改建议

建议应用在进入相机页面时，对当前调用的是前置还是后置摄像头进行判断，如果是前置摄像头，可以采取以下措施：

* 页面不展示闪光灯按钮。
* 闪光灯按钮置灰，不可点击。
* 用户点击闪光灯按钮，应用提示“前置摄像头不支持闪光灯”。
* 在设置闪光灯模式前加入if条件判断逻辑。
* 给设置闪光灯模式的代码块加上try/catch语句用来捕获异常，防止应用闪退。

设置闪光灯可以参考[实现自定义相机功能](https://gitee.com/harmonyos_samples/CustomCamera)示例代码项目的[CameraManager.ets](https://gitee.com/harmonyos_samples/CustomCamera/blob/master/camera/src/main/ets/cameramanagers/CameraManager.ets)文件中的setFlashMode方法。
