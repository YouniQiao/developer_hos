## 场景介绍

点击返回聊天消息底部是社交通讯类应用中的典型场景之一，如用户在接收到新消息时，点击一键返回到聊天消息底部。

本示例基于[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)组件实现聊天消息列表，通过[scrollToIndex](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-scroll#scrolltoindex)实现点击跳转至聊天消息列表底部。

## 实现思路

1. 模拟接收新消息，如果收到新消息时滑动距离超过半屏，弹出新消息提醒；否则直接回到底部展示最新消息。

   ```
   if (this.currentYOffset < Constants.Y_OFFSET) {
     this.iconIsShow = true;
     this.isNew = true;
   } else {
     this.listScroller.scrollToIndex(this.data.length - 1, true, ScrollAlign.END);
   }
   ```

2. 点击提醒通过[scrollToIndex](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-scroll#scrolltoindex)跳转至消息列表最底部，查看最新消息。

   ```
   .onClick(() => {
     this.listScroller.scrollToIndex(this.data.length - 1, true, ScrollAlign.END);
     this.isNew = false;
     this.iconIsShow = false;
   })
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
