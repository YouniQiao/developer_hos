## 场景介绍

添加、删除以及预览邮件附件是综合办公类应用中的高频使用场景之一。

本示例基于[PhotoViewPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-photoaccesshelper-photoviewpicker)、[cameraPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-camerapicker)、[@ohos.file.picker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-picker)、[filePreview](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/preview-arkts)和自定义弹窗等组件实现附件上传、展示和预览的功能，支持从相册、相机拍摄和本地文件中选择上传附件。

## 实现思路

* 添加相机拍摄文件至邮件附件，通过[cameraPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-camerapicker)相机Picker组件拉起相机拍摄并获取拍摄文件，将文件添加至附件列表。

  ```
  cameraPicker.pick(this.getUIContext().getHostContext(), [cameraPicker.PickerMediaType.PHOTO], pickerProfile)
    .then((pickerResult) => {
      this.addToFileList(pickerResult.resultUri.toString());
    })
  ```
* 添加本地文件至邮件附件，通过[@ohos.file.picker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-picker)文件Picker组件获取本地文件，将文件添加至附件列表。

  ```
  documentViewPicker.select(documentSelectOptions)
    .then((documentSelectResult) => {
      this.addToFileList(documentSelectResult[0]);
    })
  ```
* 添加相册文件至邮件附件，通过[PhotoViewPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-photoaccesshelper-photoviewpicker)相册Picker组件获取相册文件，将文件添加至附件列表。

  ```
  photoViewPicker.select(photoSelectOptions)
    .then((photoSelectResult: photoAccessHelper.PhotoSelectResult) => {
      this.addToFileList(photoSelectResult.photoUris[0]);
    })
  ```
* 附件展示和预览，通过[filePreview](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/preview-arkts)预览附件，支持音视频、文本、HTML等常见文件类型预览。

  ```
  let previewInfo: filePreview.PreviewInfo = {
    title: this.filename,
    uri: this.uri,
    mimeType: this.mimeType
  };
  filePreview.openPreview(this.getUIContext().getHostContext(), previewInfo);
  ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
