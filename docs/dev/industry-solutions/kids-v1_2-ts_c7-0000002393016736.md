## 问题现象

应用启动横竖屏切换，导致宽图显示不全，存在显示差异。

## 背景知识

[横竖屏切换](https://developer.huawei.com/consumer/cn/doc/best-practices/bpta-landscape-and-portrait-development)提供常见场景下的横竖屏切换方案。

## 问题定位

应用启动后，进行横竖屏切换，图片由仅展示顶部切换为展示全图，清晰度也有变化。应用启动时竖屏显示，当进入横屏显示页面时，Image组件配置宽高填满，组件被等比例放大，导致竖屏状态清晰度下降，图片展示异常。

```
import { window } from '@kit.ArkUI'

@Entry
@Component
struct Index {
  build() {
    Column() {
      Image($r('app.media.landscape'))
        .objectFit(ImageFit.Cover)
        .height('100%')
        .width('100%')
        .expandSafeArea([SafeAreaType.SYSTEM], [SafeAreaEdge.TOP, SafeAreaEdge.BOTTOM])
    }
    .height('100%')
    .width('100%')
    .onClick(() => {
      window.getLastWindow(this.getUIContext().getHostContext()).then((win) => {
        win.setPreferredOrientation(window.Orientation.LANDSCAPE);
      });
    })
  }
}
```

## 分析结论

横竖屏宽高不一致，导致图片被异常拉伸。

## 修改建议

* 方案一：将应用从启动阶段保持横屏。将module.json5文件中“orientation”字段设置为“landscape”，保持横屏状态。
* 方案二：通过屏幕的宽高并进行对比，然后对应组件进行适配。使用[on('windowSizeChange')](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-window-window#onwindowsizechange7)进行窗口监控，在回调中对图片进行调整。如下使用图片旋转的方式，在横竖屏状态下图片维持不变，防止失真。

  ```
  import { display, window } from '@kit.ArkUI';

  @Entry
  @Component
  struct LandscapeImageExample {
    // 图片旋转角度
    @State angle: number = 90;
    @State displayHeight: number = 0;
    @State displayWidth: number = 0;

    aboutToAppear(): void {
      let screen = display.getDefaultDisplaySync();
      // 获取屏幕宽高，并调整
      this.displayHeight = this.getUIContext().px2vp(screen.height) + 20;
      this.displayWidth = this.getUIContext().px2vp(screen.width) + 30;
    }

    build() {
      Stack() {
        // 加载图片资源
        Image($r('app.media.landscape'))
          .objectFit(ImageFit.Cover)  // 保持宽高放大缩小，边界大于组件边界
          .height(this.displayWidth)
          .width(this.displayHeight)
          .expandSafeArea([SafeAreaType.SYSTEM], [SafeAreaEdge.TOP, SafeAreaEdge.BOTTOM])
          .rotate({ angle: this.angle })
      }
      .height('100%')
      .width('100%')
      .onClick(() => {
        // 调整为横屏
        window.getLastWindow(this.getUIContext().getHostContext()).then((win) => {
          win.on('windowSizeChange', () => {
            // 当屏幕旋转时，设置旋转角度
            this.angle = 0;
          });
          win.setPreferredOrientation(window.Orientation.LANDSCAPE);
        });
      })
    }
  }
  ```
