## 场景介绍

下拉跳转到指定页面是购物比价类应用中高频使用场景之一。

本示例基于[Refresh](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-refresh)组件实现页面下拉跳转，当页面下拉至特定距离后触发跳转功能。

## 实现思路

1. 通过[Refresh](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-refresh)组件的refreshOffset方法设置触发刷新的下拉偏移量。
2. 通过[Refresh](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-refresh)组件的onOffsetChange、onStateChange方法监听页面下拉距离和刷新状态，当页面下拉进入刷新状态时触发页面跳转功能。

```
Refresh({ refreshing: $$this.isRefreshing, builder: this.customRefreshComponent() }) {
  // ...
}
.refreshOffset(Constants.REFRESH_OFFSET) // 设置触发刷新的下拉偏移量
.onStateChange((refreshStatus: RefreshStatus) => {
  // 当前刷新状态变更时触发回调
})
.onOffsetChange((offset: number) => {
  // 下拉距离发生变化时触发回调
})
.onRefreshing(() => {
  // 进入刷新状态时触发回调
});
```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
