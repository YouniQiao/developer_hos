## 问题现象

元服务中，使用createWindow创建子窗口，顶部胶囊位置被状态栏遮挡，具体如下图所示：

![](./img/1eed5473.png "点击放大")

## 背景知识

* [窗口模块](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/window-overview)用于在同一块物理屏幕上，提供多个应用界面显示、交互的机制。将窗口界面分为系统窗口、应用窗口两种基本类型。
* [系统窗口](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/window-overview#窗口类型)指完成系统特定功能的窗口。如音量条、壁纸、通知栏、状态栏、导航栏等。
* [应用窗口](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/window-overview#窗口类型)区别于系统窗口，指与应用显示相关的窗口。根据显示内容的不同，应用窗口又分为应用主窗口、应用子窗口两种类型。
  + 应用主窗口：应用主窗口用于显示应用界面，会在“任务管理界面”显示。
  + 应用子窗口：应用子窗口用于显示应用的弹窗、悬浮窗等辅助窗口，不会在“任务管理界面”显示。应用子窗口的生命周期跟随应用主窗口。
* [窗口沉浸式能力](https://developer.huawei.com/consumer/cn/doc/best-practices/bpta-multi-device-window-immersive)：指对状态栏、导航栏等系统窗口进行控制，减少状态栏导航栏等系统界面的突兀感，从而使用户获得最佳体验的能力。沉浸式能力只在应用主窗口作为全屏窗口时生效。

## 问题定位

使用[createWindow()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-window-f#windowcreatewindow9)创建子窗口时，在非[自由窗口](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/window-terminology#自由窗口)状态下，子窗口创建后默认是[沉浸式布局](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/window-terminology#沉浸式布局)。沉浸式页面开发通常会将应用页面延伸至状态栏和导航条，进而产生UI元素的遮挡。可以使用[getWindowAvoidArea()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-window-window#getwindowavoidarea9)方法获取避让区域的高度，结合[resize()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-window-window#resize9)和[moveWindowTo()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-window-window#movewindowto9)避让状态栏。

## 分析结论

[createWindow()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-window-f#windowcreatewindow9)创建子窗口时默认是[沉浸式布局](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/window-terminology#沉浸式布局)。元服务胶囊和状态栏位置产生重叠，需调整页面布局，避免遮挡。

## 修改建议

使用getWindowAvoidArea()方法获取避让区域的高度，使用resize()重置窗口大小，并使用moveWindowTo()移动窗口位置，具体代码如下：

```
import { display, window } from '@kit.ArkUI';
import { Context } from '@kit.AbilityKit';

@Entry
@Component
struct Index {
  async createSubWindow() {
    try {
      let context = this.getUIContext().getHostContext() as Context;
      // 当创建窗口类型为window.WindowType.TYPE_FLOAT时需要申请ohos.permission.SYSTEM_FLOAT_WINDOW
      const win = await window.createWindow({
        name: 'sub_window',
        windowType: window.WindowType.TYPE_DIALOG,
        ctx: context,
      });

      await win.setUIContent('pages/SubWindowContent');
      win.setWindowBackgroundColor('#48f65555');
      let displayInfo = display.getDefaultDisplaySync();
      // 获取状态栏高度
      let mainWindow = await window.getLastWindow(context);
      const avoidAreaType = window.AvoidAreaType.TYPE_SYSTEM;
      let avoidArea = mainWindow.getWindowAvoidArea(avoidAreaType);
      let avoidAreaHeight = avoidArea.topRect.height;
      await win.resize(displayInfo.width, displayInfo.height - avoidAreaHeight);
      await win.moveWindowTo(0, avoidAreaHeight);
      await win.setWindowFocusable(true);
      await win.showWindow();
    } catch (e) {
      console.error('create window error:', e);
    }
  }

  build() {
    Column() {
      Button('创建子窗口').onClick(() => this.createSubWindow());
    }.width('100%').height('100%').justifyContent(FlexAlign.Center);
  }
}
```

子窗口代码：

```
@Entry
@Component
struct SubWindowContent {
  build() {
    Column() {
      Text('我是子窗口')
        .fontSize(20)
        .margin({ bottom: 20 });
    }
    .width('100%')
    .height('100%')
    .justifyContent(FlexAlign.Center)
    .backgroundColor('#F5F5F5');
  }
}
```
