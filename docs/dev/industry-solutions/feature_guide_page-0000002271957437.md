## 场景介绍

功能引导页是各类应用中的高频使用场景之一，如用户首次打开应用，需要基本功能引导。

本示例使用[@ohos/high\_light\_guide](https://ohpm.openharmony.cn/#/cn/detail/@ohos%2Fhigh_light_guide)第三方库实现高亮区域和背景蒙层的明暗对比，引导用户使用应用功能。

## 实现思路

1. 引入[@ohos/high\_light\_guide](https://ohpm.openharmony.cn/#/cn/detail/@ohos%2Fhigh_light_guide)第三方库的引导页组件。
2. 初始化引导页构建类：通过addHighLightWithOptions绑定组件Id，实现该组件高亮显示。
3. 设置引导层布局组件：通过引导页构建类的setHighLightIndicator方法绑定引导层布局组件。
4. 设置底层页面布局：通过HighLightGuideComponent组件的highLightContainer属性绑定底层页面。
5. 添加引导页布局：通过HighLightGuideComponent组件初始化引导层。

```
// 引入三方库引导页组件
import {HighLightGuideComponent, HighLightGuideBuilder, Controller, GuidePage, HighLightOptionsBuilder, HighLightShape,} from '@ohos/high_light_guide';

// 初始化引导页构建类
aboutToAppear(): void {
  this.builder = new HighLightGuideBuilder()
    // ...
}

// 设置引导层布局组件
HighLightGuideComponent({
  // 设置相关参数
})

// 自定义引导页布局
@Builder
private firstIndicator(): void {
  // 第1引导层布局内容
}
@Builder
private secondIndicator(): void {
  // 第2引导层布局内容
}

// 设置底层页面
@Builder
private HighLightComponent(): void {
  Column() {
    // 底层页面
  }.alignItems(HorizontalAlign.Center)
}
```

![](./img/82a34919.png)

[@ohos/high\_light\_guide](https://ohpm.openharmony.cn/#/cn/detail/@ohos%2Fhigh_light_guide)第三方库绘制圆角弧度的方式为Canvas圆角，使用[roundRect](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-canvasrenderingcontext2d#roundrect20)接口调用drawing的AddRoundRect接口进行绘制。Image组件borderRadius属性设置圆角是系统默认圆角，一般为G2圆角，非标准圆弧。由于Canvas和Image两种圆角绘制方式存在差异，故功能引导中圆角可能会存在白边问题，当前规格如此。

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
