## 场景介绍

网络状态实时呈现是教育类应用的典型场景之一，如用户在使用应用时需要实时查看当前使用的网络速率，以便获得更好的体验。

本示例通过[@ohos.net.connection](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-net-connection)和[netQuality](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/networkboost-netquality)监听网络状态，实时获取网络速率并显示。

## 实现思路

1. 通过[register](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-net-connection#register)订阅网络状态变化通知，然后利用[on('netConnectionPropertiesChange')](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-net-connection#onnetconnectionpropertieschange)订阅网络连接信息变化事件。

   ```
   this.netCon.register((err: BusinessError) => {
     if (err) {
       hilog.info(0x0000, '网络状态', `networkMonitor fail: ${JSON.stringify(err)}`);
     }
   })
   this.netCon.on('netConnectionPropertiesChange',(data:connection.NetConnectionPropertyInfo)=>{
     this.networkTypes = this.getNetworkConnectionType()?.bearerTypes;
     if (this.networkTypes?.includes(connection.NetBearType.BEARER_WIFI)) {
       this.isWifi = true;
     } else if (this.networkTypes?.includes(connection.NetBearType.BEARER_CELLULAR)) {
       this.isWifi = false;
     }
     hilog.info(0x0000,'网络状态',`netConnectionPropertyInfo:${JSON.stringify(data)}`);
   });
   ```
2. 通过[netQuality.on('netQosChange')](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/networkboost-netquality#section13106021163)订阅网络质量信息，并获取网速信息。

   ```
   async networkSpeedMonitor() {
     try {
       netQuality.on('netQosChange', (info: Array<netQuality.NetworkQos>) => {
         if (info.length > 0) {
           info.forEach((networkQos) => {
             if (networkQos.linkDownRate / 1000000 / 8 <= 1024) {
               this.network = Number(new Decimal(networkQos.linkDownRate / 1000000 / 8 * 1024).toDecimalPlaces(1));
               this.networkSpeed = String(this.network) + 'KB/s';
             } else {
               this.network = Number(new Decimal(networkQos.linkDownRate / 1000000 / 8).toDecimalPlaces(1));
               this.networkSpeed = String(this.network) + 'MB/s';
             }
           });
         }
       });
     } catch (err) {
       hilog.info(0x0000, 'networkSpeed',
         `errorCode:${(err as BusinessError).code},errorMessage:${(err as BusinessError).message}`);
     }
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。

## 权限说明

需要申请网络权限：ohos.permission.INTERNET。
