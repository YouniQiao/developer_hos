## 问题现象

打开应用，当侧滑退出应用时会有弹出提示“再侧滑一次退出应用”，再次侧滑还是无法退出。

## 背景知识

* [openCustomDialog](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-promptaction#opencustomdialog12)：创建并弹出自定义弹窗。推荐使用UIContext中获取到的PromptAction对象提供的openCustomDialog接口在相对应用复杂的场景来实现自定义弹出框，相较于[CustomDialogController](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-methods-custom-dialog-box#customdialogcontroller)优势点在于页面解耦，支持动态刷新[update](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-componentcontent#update)。
* [DialogOptions](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-promptaction#dialogoptions18)：自定义弹窗的内容，继承自[BaseDialogOptions](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-promptaction#basedialogoptions11)。
  + onWillDismiss：交互式关闭回调函数。在回调函数中可以通过reason得到阻拦关闭弹窗的操作类型（点击遮障层关闭、侧滑（左滑/右滑）、三键back、键盘ESC关闭），从而根据原因选择是否能关闭弹窗。
  + onWillAppear：弹窗显示动效前的事件回调。
* [showToast](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-promptaction#showtoast)：创建并显示文本提示框。

3. 查看相关代码，发现侧滑页面使用了openCustomDialog来实现自定义弹窗（默认为模态窗口）。onBackPress函数中判断两次侧滑操作的时间间隔，间隔大于1秒则显示弹窗，间隔小于1秒则关闭应用。

   ```

   import { ComponentContent, promptAction } from "@kit.ArkUI";
   import { BusinessError } from "@kit.BasicServicesKit";

   class Params {
     text: string = "";

     constructor(text: string) {
   	this.text = text
     }
   }

   @Builder
   function BuilderDialog(params: Params) {
     Column() {
       Text(params.text)
         .margin(30)
     }
     .backgroundColor(Color.Grey)
   }

   @Entry
   @Component
   struct Index {
     @State firstBackTimestamp: number = 0;
     uiContext = this.getUIContext();
     promptAction = this.uiContext.getPromptAction();
     contentNode: ComponentContent<Object> = new ComponentContent(this.uiContext, wrapBuilder(BuilderDialog), new Params('再侧滑一次退出应用'));

     onBackPress() {
       let now = Date.now();
       if (now - this.firstBackTimestamp < 1000) {
         let context = this.uiContext.getHostContext() as common.UIAbilityContext
         context.terminateSelf();
       } else {
         this.firstBackTimestamp = now;
         this.promptAction.openCustomDialog(this.contentNode).then(() => {
           console.info("OpenCustomDialog succeeded")
         }).catch((error: BusinessError) => {
           console.error(`OpenCustomDialog args error code is ${error.code}, message is ${error.message}`);
         })
       }
       return true
     }

     build() {
       Column() {
         Text("Hello World")
           .fontSize(30)
           .margin(30)
       }.width("100%")
     }
   }
   ```

## 分析结论

侧滑页面的onBackPress函数中实现了两次侧滑操作退出应用的逻辑，但是由于自定义弹窗默认为模态窗口，实际第二次侧滑不会触发onBackPress函数，而是关闭了弹窗，所以导致第二次侧滑无法退出应用。

## 修改建议

* 可以在自定义弹窗的生命周期函数onWillAppear、onWillDismiss进行两次侧滑退出应用的逻辑处理。
* 如果业务上不需要自定义弹窗，也可以使用showToast文本提示框提示用户二次侧滑。

  示例代码如下：

  ```
  import { common } from '@kit.AbilityKit';
  import { ComponentContent, promptAction } from '@kit.ArkUI';
  import { BusinessError } from '@kit.BasicServicesKit';

  class Params {
    text: string = '';

    constructor(text: string) {
      this.text = text;
    }
  }

  @Builder
  function BuilderDialog(params: Params) {
    Column() {
      Text(params.text)
        .margin(30);
    }
    .backgroundColor(Color.Grey);
  }

  @Entry
  @Component
  struct SwipeTwiceToExitApp {
    arr: string[] = ['solution1', 'solution2'];
    count: number = 3;
    @State text: string = this.arr[0];
    @State firstBackTimestamp: number = 0;
    solution: () => void = () => {
    };

    aboutToAppear(): void {
      this.solution = () => {
        this.solution1();
      };
    }

    uiContext = this.getUIContext();
    promptAction = this.uiContext.getPromptAction();
    contentNode: ComponentContent<Object> =
      new ComponentContent(this.uiContext, wrapBuilder(BuilderDialog), new Params('再侧滑一次退出应用'));
    dialogOptions: promptAction.DialogOptions = {
      showInSubWindow: false,
      onWillAppear: () => {
        this.firstBackTimestamp = Date.now();
        setTimeout(() => {
          this.closeDialog();
        }, 2000);
      },
      onWillDismiss: (dismissDialogAction: DismissDialogAction) => {
        if (dismissDialogAction.reason == 0) {
          this.closeDialog();
          let now = Date.now();
          if (now - this.firstBackTimestamp < 1000) {
            const context = this.uiContext.getHostContext() as common.UIAbilityContext;
            context.terminateSelf().catch(() => {
              console.error('terminateSelf failed');
            });
          }
        } else if (dismissDialogAction.reason == 1) {
          this.closeDialog();
        }
      },
    };

    closeDialog() {
      this.promptAction.closeCustomDialog(this.contentNode).then(() => {
        console.info('succeeded');
      }).catch((error: BusinessError) => {
        console.error(`CloseCustomDialog args error code is ${error.code}, message is ${error.message}`);
      });
    }

    openDialog() {
      this.promptAction.openCustomDialog(this.contentNode, this.dialogOptions).then(() => {
        console.info('succeeded');
      }).catch((error: BusinessError) => {
        console.error(`OpenCustomDialog args error code is ${error.code}, message is ${error.message}`);
      });
    }

    onBackPress() {
      this.solution();
      return true;
    }

    solution1() {
      this.openDialog();
    }

    solution2() {
      let now = Date.now();
      if (now - this.firstBackTimestamp < 1000) {
        const context = this.uiContext.getHostContext() as common.UIAbilityContext;
        context.terminateSelf().catch(() => {
          console.error('terminateSelf failed');
        });
      } else {
        this.firstBackTimestamp = now;
        try {
          this.promptAction.showToast({
            message: '再侧滑一次退出应用',
            duration: 1000
          });
        } catch (error) {
          console.error('showToast failed');
        }
      }
    }

    build() {
      Column() {
        Button(this.text)
          .fontSize(30)
          .margin(30)
          .onClick(() => {
            this.text = this.arr[this.count % 2];
            this.count++;
            if (this.text === 'solution1') {
              this.solution = () => {
                this.solution1();
              };
            } else {
              this.solution = () => {
                this.solution2();
              };
            }
          });
      }.width('100%');
    }
  }
  ```
