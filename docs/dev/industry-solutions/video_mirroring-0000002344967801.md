## 场景介绍

视频镜面播放是影音娱乐类应用的典型场景之一，如用户可通过镜面播放功能，还原视频真实视角。

本示例基于[AVPlayer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-media-avplayer)实现视频播放的功能，通过[rotate图形变换](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-transformation#rotate)实现组件镜面功能。

## 实现思路

1. 设置是否选择镜面播放，若选择镜面播放，镜面翻转角度为180°，按钮变为蓝色，否则镜面角度恢复，按钮变为白色。

   ```
   Image(this.isMirror ? $r('app.media.mirror_blue') : $r('app.media.mirror_white'))
     .height(Constants.IMAGE_18)
     .width(Constants.IMAGE_18)
     .margin({ right: Constants.MARGIN_10 })
     .onClick(() => {
       if (!this.isMirror) {
         this.isMirror = true;
         // 设置镜面角度
         this.angle = Constants.ROTATE_ANGLE;
         this.dialogDuration();
       } else {
         this.isMirror = false;
         // 恢复镜面角度
         this.angle = Constants.DEFAULT_ANGLE;
       }
     });
   ```
2. 设置播放控件镜面属性rotate，实现镜面功能。

   ```
   Stack() {
     // ...
   }
   .height(Constants.FULL_PERCENT)
   .rotate({ y: 2, angle: this.angle, perspective: Constants.ROTATE_CAMERA_DISTANCE });
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
