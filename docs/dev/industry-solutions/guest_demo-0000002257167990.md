## 场景介绍

访客管理是综合办公类应用中的典型场景之一，如邀请他人到公司面试、商谈业务、参与活动或其他事项时，需事先预约、登记访客来访信息，并生成访客通行证。

本示例基于[@ohos.contact](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-contact)、ArkTS弹窗实现访客信息登记功能，并可通过系统联系人快速登记访客相关信息；通过[generateBarcode](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/scan-generatebarcode)方法实现访客通行证二维码自动生成功能。

## 实现思路

1. 进入首页，展示当前已登记的访客邀请列表，并按照访客来访时间由近及远顺序排列。

   ```
   // 对访客邀请列表数据排序
   sortGuestList() {
     this.dataArr.sort((a: GuestData, b: GuestData) => {
       return b.selectedTimeMills - a.selectedTimeMills;
     });
   }
   List() {
     ForEach(this.dataArr, (item: GuestData) => {
       // 展示已有访客邀请列表
     })
   };
   ```
2. 点击首页右上角图标进入新增访客邀请界面，可选择自主填充访客身份信息，或通过[contact.selectContacts](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-contact#contactselectcontacts10)选择系统联系人信息进行快速填充。

   ```
   // 手动填写或拉起系统联系人自动填充
   TextInput({ text: this.guest.name });
   TextInput({ text: this.guest.phone });
   // 获取到系统联系人信息后自动填充访客姓名和手机号
   Row({ space: 10 }) {
     // ...
   }
   .onClick(() => {
     // 拉起通讯录信息
     let promise = contact.selectContacts();
     promise.then((data) => {
       let jsonString = JSON.stringify(data);
       if (data.length > 0) {
         let contact: contact.Contact = data[0];
         this.guest.name = contact.name?.fullName ?? '';
         if (contact.phoneNumbers && contact.phoneNumbers.length > 0) {
           this.guest.phone = contact.phoneNumbers[0].phoneNumber;
         }
       }
     });
   });
   ```
3. 通过[showTextPickerDialog](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#showtextpickerdialog)选择手机区号和访客到访事由，通过[showDatePickerDialog](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#showdatepickerdialog)选择访客到访时间。

   ```
   // 选择手机区号
   this.uiContext.showTextPickerDialog({
     // ...
     onAccept: (value: TextPickerResult) => {
       this.select = value.index;
       if (typeof value.index === 'number') {
         let selectedIndex = value.index as number;
         this.guest.areaCode = this.areaCodeList[selectedIndex];
       }
     }
   });
   // 选择到访事由
   this.uiContext.showTextPickerDialog({
     // ...
     onAccept: (value: TextPickerResult) => {
       this.select = value.index;
       this.guest.reasons = value.value as string;
     }
   });
   // 选择来访时间
   this.uiContext.showDatePickerDialog({
     // ...
     onDateAccept: (value: Date) => {
       this.selectedDate = value;
       this.guest.selectedTimeMills = value.getTime();
       this.guest.selectedTime = CommonUtil.formatDate(value);
     }
   });
   ```
4. 点击提交，保存时通过[generateBarcode](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/scan-generatebarcode)生成对应访客通行二维码。

   ```
   // 保存时生成访客通行二维码
   CommonUtil.generateScanCode(this.guest.name)
     .then((scanCode) => {
       // 成功生成二维码图像信息后，保存至对应访客邀请信息中，便于后续展示
       if (scanCode) {
         this.guest.scanCode = scanCode;
       }
       this.pageStack.pop(this.guest, true);
     });
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
