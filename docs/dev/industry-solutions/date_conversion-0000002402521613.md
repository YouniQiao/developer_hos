## 场景介绍

日期计算器是实用工具类应用中高频使用场景之一。

本示例利用[国际化-I18n](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-i18n)实现日期转换、计算日期间隔和日期推算功能。

## 实现思路

* 通过[i18n.getCalendar](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-i18n#i18ngetcalendar8)获取农历日历，利用[setTime](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-i18n#settime8)获取公历日期对应的农历日期，实现公历日期转换成农历日期的功能。

  ```
  let calendar: i18n.Calendar = i18n.getCalendar('zh-Hans', 'chinese');
  let dateArray: number[] = solarDate.split('/').map(Number);
  calendar.setTime(new Date(dateArray[0], dateArray[1] - 1, dateArray[2], 8, 0, 0,0));
  ```
* 通过Date.getTime()获取两个日期的时间戳，计算两个日期的间隔天数。

  ```
  let timeDifference: number = Math.abs(lastDate.getTime() - firstDate.getTime());
  let result: string = Math.floor(timeDifference / CommonConstants.MILLISECONDS_PER_DAY).toString();
  ```
* 通过获取选择日期的时间戳和日期间隔进行公历日期推算，并利用日期转换进行农历日期推算。

  ```
  this.estimationSolarDate = SwitchDate.getTime(new Date(value.getTime() + CommonConstants.MILLISECONDS_PER_DAY * Number.parseInt(this.solarGapDay)));
  ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
