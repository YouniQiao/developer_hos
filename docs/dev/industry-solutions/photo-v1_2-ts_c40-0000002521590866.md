## 问题现象

在进行自定义相机开发时，选择部分视频配置信息项[VideoProfile](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-i#videoprofile)进行视频录制，录制出的文件会损坏，无法在视频列表中找到录制的视频，如下图所示：

![](./img/1463939f.png "点击放大")

## 解决方案

打印目前所选择的视频配置信息项，结果如下：

```
Current video format is 2002 with width 640 and height 480
```

在选择视频配置信息项时，部分[VideoProfile](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-i#videoprofile)的[CameraFormat](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-e#cameraformat)为2002（CAMERA\_FORMAT\_YCRCB\_P010）类型，需要使用[HDR录像](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/camera-hdr-recording)的开发能力，不然会导致文件损坏。普通录像需要选择CameraFormat为1003（CAMERA\_FORMAT\_YUV\_420\_SP）的VideoProfile，重新选择VideoProfile后的录制效果如下：

![](./img/05fe32a4.png "点击放大")
