## 场景介绍

设备流量监控是儿童教育类应用的高频使用场景之一，如家长需要查看孩子电子设备的流量使用情况。

本示例基于[用户首选项](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-data-preferences)和[流量管理](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-net-statistics)获取流量使用情况，基于[refresh](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-components-container-refresh)实现下拉刷新获取剩余流量，基于[progress](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-service-widget-basic-progress)实现环形进度条动画，当检测到流量超额时，触发标红警示。

## 实现思路

1. 在第一次安装应用时，分别获取之前的下行和上行流量使用情况，相加后保存在[用户首选项](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-data-preferences)中，从此不发生改变。

   ```
   aboutToAppear() {
     this.firstData = PreferencesClass.getPreferenceInfo(this.context, this.flag, PreferencesClass.DEFAULT_STORE);
     this.firstDataMonitor().then(() => {});
   }
   async firstDataMonitor() {
     if (this.firstData === 0) {
       let rx = await statistics.getIfaceRxBytes('wlan0');
       let tx = await statistics.getIfaceTxBytes('wlan0');
       this.firstData = rx + tx;
       PreferencesClass.savePreferenceInfo(this.context, this.flag, PreferencesClass.DEFAULT_STORE,
         this.firstData);
     }
   }
   ```
2. 获取剩余流量：将当前流量数据与初始流量数据相减，即可获取剩余流量；同时获取系统时间，即为刷新时间。

   ```
   async tempDataMonitor() {
     let rx = await statistics.getIfaceRxBytes('wlan0');
     let tx = await statistics.getIfaceTxBytes('wlan0');
     let restData = rx + tx - this.firstData;
     this.tempData = this.dataNumber * Constants.GB_TO_MB - restData / Constants.BYTE_TO_MB;
     this.time = this.getTime();
   }
   ```
3. 在得到剩余流量后，与流量限额相除得到百分比显示在[progress](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-service-widget-basic-progress)中，当剩余流量为负，颜色变红。

   ```
   Refresh({ refreshing: $$this.isRefreshing }) {
     Stack() {
       Progress({
         value: this.tempData >= 0 ? this.tempData * 100 / (this.dataNumber * Constants.GB_TO_MB) : 100,
       })
       Column() {
         Text(this.tempData.toFixed(0))
         Text() {Span(this.time)}
       }
     }
   }
   .onRefreshing(() => {
     this.tempDataMonitor();
     setTimeout(() => {this.isRefreshing = false}, 500);
   })
   .backgroundColor(!this.isShow ? '#F1F3F5' : '#33000000')
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
