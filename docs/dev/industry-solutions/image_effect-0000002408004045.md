## 场景介绍

图片色彩调节是拍摄美化类应用的典型场景之一，如用户需要对图片进行基础调色。

本示例基于[图像效果](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-image-effect)、[SaveButton](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-security-components-savebutton)实现图片亮度、对比度和饱和度的调节，并可将处理后的图片保存至图库。

## 实现思路

1. 通过调控图片组件对应属性的参数，调节图片亮度、对比度和饱和度。
2. 通过使用[SaveButton](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-security-components-savebutton)保存控件实现临时获取存储权限保存图片。

```
Image($r('app.media.main_image'))
  .brightness(this.selectMode === CommonConstants.SELECT_MODE_LIGHT ? this.adjustValue / 200 + 1 :
    this.lightValue / 200 + 1)
  .contrast(this.selectMode === CommonConstants.SELECT_MODE_CONTRAST_RATIO ? this.adjustValue / 200 + 1 :
    this.contrastRatioValue / 200 + 1)
  .saturate(this.selectMode === CommonConstants.SELECT_MODE_EXPOSURE ? this.adjustValue / 200 + 1 :
    this.saturateValue / 200 + 1)

SaveButton({
  text: SaveDescription.EXPORT_TO_GALLERY
})
  .width(65)
  .onClick(() => {
    // ...
    PhotoUtils.saveImageToAlbum(getContext(this), arrayBuffer);
  })
```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
