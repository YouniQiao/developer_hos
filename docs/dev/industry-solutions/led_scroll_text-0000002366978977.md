## 场景介绍

LED滚动字幕是实用工具类应用中的典型场景之一，如用户在演唱会现场或好友过生日时，可通过手机屏幕展示滚动字幕，显示应援口号或祝福语。

本示例通过[setWindowLayoutFullScreen](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-window-window#setwindowlayoutfullscreen9)接口设置内容沉浸式展示，通过[Marquee](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-marquee)组件模拟文字跑马灯效果，实现了一个简易的LED显示屏，支持设置显示内容、字体大小、字体颜色、背景颜色、是否滚动及滚动速度等功能。

## 实现思路

1. 调用[setWindowLayoutFullScreen](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-window-window#setwindowlayoutfullscreen9)接口实现沉浸式展示效果。

   ```
    windowClass: window.Window = (this.context as common.UIAbilityContext).windowStage.getMainWindowSync();
    this.windowClass.setWindowLayoutFullScreen(true);
   ```
2. 调用[setPreferredOrientation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-window-window#setpreferredorientation9)接口实现横竖屏模式切换，竖屏配置参数，横屏展示内容。

   ```
   // 设置横屏模式
   this.windowClass.setPreferredOrientation(window.Orientation.LANDSCAPE);
   // 返回竖屏模式
   this.windowClass.setPreferredOrientation(window.Orientation.PORTRAIT);
   ```
3. 使用[Marquee](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-marquee)组件实现文字跑马灯效果。

   ```
   Marquee({
     start: this.whetherScroll,
     step: this.scrollSpeed,
     loop: Constants.INFINITE_LOOP,
     src: this.contentStrNormalize.toString(),
   })
     .width(Constants.FULL_PERCENT)
     .height(Constants.FULL_PERCENT)
     .fontSize(this.contentFontSize)
     .fontColor(this.contentFontColor)
     .align(Alignment.Center)
     .onClick(() => {
       this.isButtonVisible = !this.isButtonVisible;
     })
   ```
4. 当文字宽度小于屏幕宽度时，Marquee展示内容静止不动，需通过拼接空格的方式扩展字符串长度，从而达到跑马灯效果。

   ```
   if (this.whetherScroll) {
     return this.contentStr + ' '.repeat(this.screenHeight / this.spaceWidth) + this.contentStr;
   } else {
     return this.contentStr;
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
