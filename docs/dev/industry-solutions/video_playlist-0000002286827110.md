## 场景介绍

视频播放列表是影音娱乐类应用的高频使用场景之一，如用户浏览视频列表时自动播放视频。

本示例基于[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)、[AVPlayer](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/video-playback)组件实现视频播放列表页面，通过监听列表滑动，实现视频列表中首个可见视频自动播放的效果。

## 实现思路

1. 使用[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)、[XComponent](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-xcomponent)组件，搭配[AVPlayer](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/video-playback)模块和[LazyForEach](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-rendering-control-lazyforeach)渲染控制机制，实现视频播放列表页面。
2. 监听列表滚动停止事件，获得列表当前滚动偏移量，计算出第一个完全可见的视频项的索引值，播放该索引对应的视频资源。

```
.onScrollStop(() => {
  // 获取当前偏移量并计算第一个完全可见视频项的索引
  let yOffset = this.listScroller.currentOffset().yOffset;
  let curIndex = Math.floor(yOffset / (this.listItemHeight + LIST_DIVIDER_WIDTH));
  let offsetInItem = yOffset - curIndex * (this.listItemHeight + LIST_DIVIDER_WIDTH);
  if (offsetInItem > LIST_ITEM_PADDING) {
    curIndex += 1;
  }
  // 播放对应的视频
  this.curIndex = curIndex;
  if (curIndex !== this.playIdx && curIndex < this.dataSource.totalCount()) {
    setTimeout(() => {
      if (this.curIndex === curIndex && this.curIndex !== this.playIdx) {
        this.play(curIndex);
      }
    }, DELAY_MS);
  }
})
```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
