## 场景介绍

电子章添加和保存是综合办公类应用高频场景之一，如用户给PDF文件加盖电子公章。

本示例基于[PDF Kit](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/pdf-introduction)实现在文件中添加印章的效果，通过[PdfView](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/pdf-arkts-pdfview-component#section724521414201)预览PDF，生成盖章文件并保存至用户文档目录中。

## 实现思路

1. 预览PDF：[PdfController](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/pdf-arkts-pdfviewmanage#section15202162183817)调用loadDocument方法，加载PDF文件。

   ```
   private controller: pdfViewManager.PdfController = new pdfViewManager.PdfController();
   aboutToAppear(): void {
     (async () => {
       this.controller.registerPageCountChangedListener((pageCount: number) => {
       });
       let loadResult: pdfService.ParseResult = await this.controller.loadDocument(this.filePath);
       if (pdfService.ParseResult.PARSE_SUCCESS === loadResult) {
       }
     })()
   }
   ```

   调用[PdfView](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/pdf-arkts-pdfview-component)组件预览PDF文件。

   ```
   PdfView({
     controller: this.controller,
     pageFit: pdfService.PageFit.FIT_WIDTH,
     showScroll: false
   })
     .id('pdfview_app_view')
     .backgroundColor(Color.White)
   ```
2. 添加印章：通过[pdfService](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/pdf-arkts-pdfservice)调用[loadDocument](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/pdf-arkts-pdfservice#section167288392229)方法，加载PDF文件，调用[addImageObject](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/pdf-arkts-pdfservice#section699712112399)方法在指定位置插入印章图片，生成PDF盖章文件。

   ```
   private pdfDocument: pdfService.PdfDocument = new pdfService.PdfDocument();
   this.outPdfPath = this.context.filesDir + '/' + Date.now() + Constants.PDF_NAME;
   let res = this.pdfDocument.loadDocument(this.filePath);
   if (res === pdfService.ParseResult.PARSE_SUCCESS) {
     let pdfPage: pdfService.PdfPage = this.pdfDocument.getPage(0);
     let imgPath = this.context.filesDir + '/' + Constants.MARKER_NAME;
     pdfPage.addImageObject(imgPath, 370, 200, 120, 120);
     let result = this.pdfDocument.saveDocument(this.outPdfPath);
     this.isMarked = true
   }
   ```

   生成盖章文件后释放未盖章PDF文件，重新加载PDF文件路径。

   ```
   this.pdfDocument.releaseDocument();
   this.reloadDocument(this.outPdfPath)
   ```
3. 保存PDF：创建文件选择器[DocumentViewPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-picker#documentviewpicker)实例，调用[save](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-picker#save)接口拉起documentPicker界面，保存PDF至用户指定目录。

   ```
   async callFilePickerSaveFile(): Promise<void> {
     try {
       let documentSaveOptions = new picker.DocumentSaveOptions();
       documentSaveOptions.newFileNames = [Constants.PDF_NAME];
       let documentPicker = new picker.DocumentViewPicker();
       documentPicker.save(documentSaveOptions).then((documentSaveResult) => {
         if (documentSaveResult !== null && documentSaveResult !== undefined) {
           this.uriSave = documentSaveResult[0];
         }
         FileUtils.copyFile2Document(this.outPdfPath, this.uriSave);
         this.promptAction.showToast({
           message: $r('app.string.save_success'),
           duration: 2000
         })
       }).catch((err: BusinessError) => {
       });
     } catch (err) {
     }
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
