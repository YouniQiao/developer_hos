## 场景介绍

群聊页面置顶公告是综合办公类应用的高频使用场景之一，如用户发布群公告后，公告内容置顶展示，若点击查看公告详情并返回，则公告消失。

本示例基于[ComponentContent](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-componentcontent)将组件内容实体封装，并叠加在当前组件上作为当前组件的[浮层](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-overlay)，实现群聊页面公告置顶的效果。

## 实现思路

1. 在aboutToAppear中初始化[ComponentContent](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-componentcontent)并新增节点，点击公告后隐藏节点并跳转至公告详情页。

   ```
   aboutToAppear(): void {
     let componentContent = new ComponentContent(
       this.uiContext, wrapBuilder<[Params]>(OverlayNode), this.params;
     );
     this.overlayNode.addComponentContent(componentContent, 0);
     this.overlayContent.push(componentContent);
   }
   // ...
   // Click the announcement to hide the node and navigate to the announcement page
   let componentContent = params.overlayContent[0];
   params.overlayNode.hideComponentContent(componentContent);
   params.pageInfos.pushPath({ name: 'NoticePage' });
   ```
2. 在aboutToDisappear中删除节点。

   ```
   aboutToDisappear(): void {
     let componentContent = this.overlayContent.pop();
     this.overlayNode.removeComponentContent(componentContent);
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
