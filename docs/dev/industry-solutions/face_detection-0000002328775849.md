## 场景介绍

人脸位置识别是拍摄美化类应用的高频使用场景之一，如通过对人脸的定位，实现对人脸特定位置的美化修饰。

本示例使用[faceDetector](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/core-vision-face-detector-api)实现人脸位置识别功能并展示缩略图。

## 实现思路

1. 设置[Image](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-image)组件填充效果为ImageFit.MATRIX，并设置图片变换矩阵。

   ```
   Image(this.chooseImage)
     .objectFit(ImageFit.MATRIX)  // 设置图片填充效果
     .imageMatrix(this.matrixTransit)  // 设置图片变换矩阵
   ```
2. 初始化[faceDetector](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/core-vision-face-detector-api)并传入图片信息获取人脸识别结果。
3. 根据人脸识别结果，获取聚焦到人脸中心位置的变换矩阵，展示对应的人脸缩略图。

   ```
   static async detectFace(pixelMap: PixelMap): Promise<faceDetector.Face[]> {
     let res: faceDetector.Face[] = [];
     if (await faceDetector.init()) {  // 初始化faceDetector
       let visionInfo: faceDetector.VisionInfo = { pixelMap: pixelMap };
       res = await faceDetector.detect(visionInfo);  // 检测人脸
       await faceDetector.release();  // 完成后释放faceDetector
     }
     return res;
   }

   onFaceDataReceived() {
     if (this.faceData) {
       // 根据人脸识别结果，获取聚焦到人脸中心位置的变换矩阵
       let transit = Matrix4Util.getFaceCenterTransit(this.faceData, this.componentWidth, this.componentHeight);
       this.matrixTransit = Matrix4Util.getMatrix4(transit.scale, transit.translationX, transit.translationY);
     }
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
