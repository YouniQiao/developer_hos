## 问题现象

问题一：半模态弹框bindSheet在折叠屏展开状态和平板上没有半模态的效果。

问题二：半模态弹窗bindSheet如何固定内容高度，禁止跟手滑动。

问题三：半模态弹窗如何只支持侧滑关闭。

## 背景知识

* [bindSheet](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-sheet-transition#bindsheet)：给组件绑定半模态页面，点击后显示模态页面。
* [SheetType](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-sheet-transition#sheettype11枚举说明)：设置半模态弹窗的样式。
* [Scroll](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-scroll)：可滚动的容器组件，当子组件的布局尺寸超过父组件的尺寸时，内容可以滚动。
* [SheetOptions](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-sheet-transition#sheetoptions)：半模态页面内容选项，可通过实现onWillDismiss方法拦截半模态关闭的行为。

* **解决方案二：**

  在bindSheet的builder外层包裹Scroll容器，利用其手势拦截机制可有效屏蔽跟手滑动。但需特别注意的是，通过SheetOptions的title属性设置的标题区域与builder处于同级布局，该区域仍会独立消费手势事件，导致拖拽标题时触发跟手滑动。因此若要实现完全禁止跟手滑动的效果，应避免设置SheetOptions的title属性，具体代码如下：

  ```
  @Entry
  @Component
  struct Page1 {
    @State isShow: boolean = false;
    @State sheetHeight: number = 300;

    @Builder
    myBuilder() {
      // 在bindSheet布局外层用Scroll容器，即可实现禁止跟手滑动
      Scroll() {
        Button('改变高度')
          .margin(10)
          .backgroundColor('#0a59f7')
          .fontSize(20)
          .onClick(() => {
            this.sheetHeight = 500;
          });
      }
      .width('100%')
      .height('100%');
    }

    build() {
      Column() {
        Button('打开bindsheet')
          .onClick(() => {
            this.isShow = true;
          })
          .fontSize(20)
          .margin(10)
          .bindSheet($$this.isShow, this.myBuilder(), {
            height: this.sheetHeight,
            backgroundColor: '#f1f3f5',
            showClose: true,
          });
      }
      .justifyContent(FlexAlign.Center)
      .width('100%')
      .height('100%');
    }
  }
  ```

* **解决方案三：**

  通过bindSheet的SheetOptions提供的onWillDismiss方法，拦截bindSheet的关闭原因，从而可以控制bindSheet是否关闭，具体代码如下：

  ```
  @Entry
  @Component
  struct bindSheetDemo {
    @State isShow: boolean = false;
    @State sheetHeight: number = 300;

    @Builder
    myBuilder() {
      Column() {
        Button('改变高度')
          .margin(10)
          .backgroundColor('#0a59f7')
          .fontSize(20)
          .onClick(() => {
            this.sheetHeight = 500;
          });
      }
      .width('100%')
      .height('100%');
    }

    build() {
      Column() {
        Button('打开bindsheet')
          .onClick(() => {
            this.isShow = true;
          })
          .fontSize(20)
          .margin(10)
          .bindSheet($$this.isShow, this.myBuilder(), {
            height: this.sheetHeight,
            backgroundColor: '#f1f3f5',
            showClose: false,
            onWillDismiss: (dismissSheetAction: DismissSheetAction) => {
              // 侧拉关闭包含侧滑（左滑/右滑）、三键back、键盘ESC关闭时关闭弹窗：
              if (dismissSheetAction.reason === DismissReason.PRESS_BACK) {
                dismissSheetAction.dismiss();
              }
            }
          });
      }
      .justifyContent(FlexAlign.Center)
      .width('100%')
      .height('100%');
    }
  }
  ```

## 常见FAQ

Q：如何在bindSheet滑动改变高度的过程中刷新可视区域？

A：[ScrollSizeMode](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-sheet-transition#scrollsizemode12枚举说明)是半模态面板上下滑动时的内容更新方式，给bindSheet添加ScrollSizeMode.CONTINUOUS属性即可。

Q：bindSheet设置mode为EMBEDDED时背景如何才能覆盖状态栏？

A：目前bindSheet无法做到，可以参考NavDestination的dialog模式，具体请参考[NavDestinationMode枚举说明](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navdestination#navdestinationmode枚举说明11)。

Q：是否可以将两个半模态绑定在同一个组件上面？

A：bindSheet给组件绑定半模态页面，点击后显示模态页面。同一个组件不能绑定两个bindSheet，需将bindSheet绑定到不同的组件上。
