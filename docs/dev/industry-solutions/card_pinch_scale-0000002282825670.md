## 场景介绍

双指捏合缩放卡片是便捷生活类应用中高频使用场景之一，如用户切换备忘录、日历提醒、记忆卡片等，需要缩放卡片查看。

本示例基于[PinchGesture](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-gestures-pinchgesture)实现卡片的双指捏合缩放能力。

## 实现思路

1. 双指捏合缩小时，通过[PinchGesture](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-gestures-pinchgesture)获取缩放比例，若缩放比例小于1，则触发跳转至[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)选择列表。

   ```
   // 双指捏合缩小
   Column(){
   }
   .gesture(
     PinchGesture({ fingers: Constants.TWO_FINGERS })
       .onActionUpdate((event: GestureEvent) => {
         // 获取缩放比
       })
       .onActionEnd(() => {
         if (this.scaleValue < 1) {
           // 进入List选择
         }
       })
   );
   ```
2. 在List选择列表中，当点击或双指放大（缩放比例大于1）选中某一卡片时，退出当前List选择列表，并展示被选中的卡片。

   ```
   // 在List中点击或放大卡片
   ListItem() {
     CardView()
       .onClick(() => {
         // 点击展示卡片
       })
   }
   .gesture(
     PinchGesture({ fingers: Constants.TWO_FINGERS })
       .onActionEnd(() => {
         if (this.scaleValue > 1) {
           // 放大展示卡片
         }
       })
   );
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
