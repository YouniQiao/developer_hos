## 背景知识

应用无法直接调节系统音量，可以通过[系统音量面板](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ohos-multimedia-avvolumepanel)，让用户通过界面操作来调节音量。当用户通过应用内音量面板调节音量时，系统会展示音量提示界面，显性地提示用户系统音量发生改变。

## 解决方案

目前没有控制系统音量进度条位置或显隐的API，可以给页面增加[音量面板组件](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ohos-multimedia-avvolumepanel)并使用[Position](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-types#position)属性将面板移出屏幕外，以实现系统音量进度条和音量面板的隐藏效果。

```
import { AVVolumePanel } from '@kit.AudioKit';

@Entry
@Component
struct AVVolumePanelPage {
  @State volume: number = 0;

  build() {
    Row() {
      Column() {
        AVVolumePanel({
          volumeLevel: this.volume,
          volumeParameter: {
            position: {
              x: -100,
              y: -200
            }
          }
        })
      }
    }.width('50%').height('50%')
  }
}
```
