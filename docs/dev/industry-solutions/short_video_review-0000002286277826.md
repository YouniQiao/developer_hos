## 场景介绍

短视频评论是影音娱乐类应用的高频使用场景之一，如用户在播放短视频时可以弹出评论区，提升用户体验。

本示例利用[pushPathByName](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation#pushpathbyname10)函数跳转至评论区页面，通过[transition](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-transition-animation-component)实现自定义评论区动画弹出，利用[CustomDialog](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-methods-custom-dialog-box)实现评论时键盘的弹出。

## 实现思路

* 利用[pushPathByName](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation#pushpathbyname10)函数跳转至评论区页面。

  ```
  Column() {
    // ...
  }
  .onClick(() => {
    this.controller.stop();
    this.isPlay = false;
    let test = $r('app.string.shortvideo_test');
    this.pageInfos.pushPathByName('DialogFirst', test, (popInfo) => {
      let message =
        '[pushPathByName]last page is: ' + popInfo.info.name + ', result: ' + JSON.stringify(popInfo.result);
    });
  });
  ```
* 通过[transition](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-transition-animation-component)实现自定义评论区动画弹出。

  ```
  Column() {
    // ...
  }
  .transition(TransitionEffect.translate({ x: 0, y: '100%', z: 0 })
    .animation({ duration: 400, curve: Curve.Linear }));
  ```
* 利用[CustomDialog](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-methods-custom-dialog-box)实现评论时键盘的弹出。

  ```
  // 创建控制器
  dialogController: CustomDialogController | null = new CustomDialogController({
    builder: CommentDialog({
      textInputText: this.textInputText,
      textValue: this.textValue
    }),
    // ...
  });
  // 自定义键盘弹窗页面内容
  @CustomDialog
  struct CommentDialog {
    // ...
  }
  ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
