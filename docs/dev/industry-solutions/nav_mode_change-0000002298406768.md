## 场景介绍

双栏页面覆盖整个应用是各类应用中的高频场景之一，如宽屏机器在适配双栏模式后，进入某些特定页面时页面覆盖整个应用，而非以双栏形式展示。

本示例通过调整[Navigation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation)组件的mode属性及[@ohos.arkui.observer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-observer)监听页面更新事件实现双栏页面覆盖整个应用的功能。

## 实现思路

1. 进入认证页前设置导航栏显示模式为[NavigationMode.Stack](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation#navigationmode9枚举说明)，确保认证页能够铺满整个应用界面。

   ```
   this.pathStack.setInterception({
     willShow: (_from, _to, _operation, _isAnimated) => {
       if (to === 'navBar') {
         return;
       }
       let toInfo = to as NavDestinationContext;
       let pathInfo = toInfo.pathInfo;
       if (pathInfo && pathInfo.name === 'ConversationPage' && !this.verified) {
         this.pathStack.pop();
         this.pathStack.pushPath({ name: 'VerifyPage', param: pathInfo.param });
         AppStorage.setOrCreate('currentPage', 'VerifyPage');
         this.navMode = NavigationMode.Stack;
       }
     }
   });
   ```
2. 监听页面更新事件[navDestinationUpdate](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-observer#uiobserveronnavdestinationupdate)，在双栏模式下，认证页退出时将导航栏显示模式设置为[NavigationMode.Split](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation#navigationmode9枚举说明)。

   ```
   // 初始化监听设置
   uiContext.getUIObserver().on('navDestinationUpdate', (info) => {
     let currentBreakpoint = AppStorage.get<string>(BreakpointConstant.CURRENT_BREAKPOINT);
     if (info.name.toString() === 'VerifyPage' && info.state === uiObserver.NavDestinationState.ON_WILL_DISAPPEAR &&
       currentBreakpoint !== BreakpointConstant.BREAKPOINT_SM) {
       AppStorage.setOrCreate('navMode', NavigationMode.Split);
     }
   });
   ```
3. 双栏模式下，从认证页回退至上一页面时将空页压入路由栈中，避免导航栏显示模式切换时的动画效果导致页面抖动异常。

   ```
   if (this.currentBreakpoint === BreakpointConstant.BREAKPOINT_SM) {
     this.pathStack.pop();
   } else {
     this.pathStack.clear();
     this.pathStack.pushPath({ name: 'NonePage' });
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
