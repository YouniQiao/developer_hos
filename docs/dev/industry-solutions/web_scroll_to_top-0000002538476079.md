## 场景介绍

H5页面一键返回顶部是网页浏览时的高频操作之一，如用户上滑查看网页内容时，通过点击向上箭头图标实现返回至网页顶部。

本示例基于ArkUI组件能力实现的返回顶部按键，通过[Stack](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-layout-development-stack-layout)布局实现自定义的向上按键组件叠放在Web组件上面，通过[scrollTo()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-webview-webviewcontroller#scrollto)将网页滚动到顶部位置，实现同时适配所有H5页面。

## 实现思路

1. 要实现向上按键悬浮在Web组件上的效果，可以采用[Stack](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-layout-development-stack-layout)层叠布局，将自定义的向上按键组件叠放在Web组件上面。

   ```
   Stack({ alignContent: Alignment.BottomEnd }){
       Web({
           src: $rawfile('index.html'),
           controller: this.webviewController,
       })
           .fileAccess(false)
           .geolocationAccess(false)
           .expandSafeArea([SafeAreaType.SYSTEM], [SafeAreaEdge.TOP, SafeAreaEdge.BOTTOM])
           .onScroll((event) => {
           if (event.yOffset > this.showButtonLimit) {
               this.isScrollToTopShow = true;
           } else {
               this.isScrollToTopShow = false;
           }
       })
       ScrollToTopButton({
           webviewController: this.webviewController,
           isShow: this.isScrollToTopShow,
           containerWidth: this.containerWidth,
           containerHeight: this.containerHeight
       })
   }
   ```

   为了提高向上按键的可拓展性，采用Row+Image组件实现自定义按键组件，在onClick()事件中监听点击事件。

   ```
   build() {
       Row() {
           Image($r('app.media.top_button_icon'))
               .width($r('app.float.top_icon_size'))
               .height($r('app.float.top_icon_size'))
       }
       .justifyContent(FlexAlign.Center)
           .alignItems(VerticalAlign.Center)
           .width($r('app.float.top_button_size'))
           .height($r('app.float.top_button_size'))
           .borderRadius($r('app.float.top_button_radius'))
           .backgroundColor(Color.White)
           .visibility(this.isShow ? Visibility.Visible : Visibility.Hidden)
           .onClick(() => {
           this.webviewController?.scrollTo(0, 0);
           this.isShow = false;
       })
           .position({ x: this.offsetX, y: this.offsetY })

       // ...

   }
   ```
2. 在[gesture()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-gesture-settings#gesture)事件中监听拖动手势，实时修改按键位置，并通过[animateToImmediately()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-explicit-animatetoimmediately#animatetoimmediately)实现吸边动画效果。

   ```
   .gesture(
       PanGesture()
       .onActionStart((event: GestureEvent) => {
           this.buttonX = this.offsetX;
           this.buttonY = this.offsetY;
       })
       .onActionUpdate((event: GestureEvent) => {
           this.offsetX = this.buttonX + event.offsetX;
           this.offsetY = this.buttonY + event.offsetY;
       })
       .onActionEnd((event: GestureEvent) => {
           animateToImmediately({ curve: curves.springMotion() }, () => {
               // 限制按键在x方向上的最大、最小值，防止按键超出可视窗口
               if (this.offsetX >= 0) {
                   let x = this.offsetX + this.buttonWidth / 2;
                   this.offsetX =
                       x >= this.containerWidth / 2 ? this.containerWidth - this.buttonWidth - this.buttonPadding :
                   this.buttonPadding;
               } else {
                   this.offsetX = this.buttonPadding;
               }
               // 限制按键在y方向上的最大、最小值，防止按键超出可视窗口
               if (this.offsetY >= 0) {
                   this.offsetY =
                       this.offsetY + this.buttonHeight > this.containerHeight ? this.containerHeight -this.buttonHeight :
                   this.offsetY;
               } else {
                   this.offsetY = 0;
               }
           })
       })
   )
   ```
3. 通过[visibility](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-visibility#visibility)属性，控制按键的消失与出现。

   ```
   .visibility(this.isShow ? Visibility.Visible : Visibility.Hidden)
   ```
4. 在[onScroll()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-basic-components-web-events#onscroll9)事件中添加判断，如果页面滚动偏移量大于一定值时，返回顶部按键才出现。

   ```
   Web({
       src: $rawfile('index.html'),
       controller: this.webviewController,
   })
       .onScroll((event) => {
       if (event.yOffset > this.showButtonLimit) {
           this.isScrollToTopShow = true;
       } else {
           this.isScrollToTopShow = false;
       }
   })
   ```
5. 在按键的onClick()事件调用[scrollTo()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-webview-webviewcontroller#scrollto)将网页滚动到顶部位置，然后同时隐藏按键。

   ```
   .onClick(() => {
       this.webviewController?.scrollTo(0, 0);
       this.isShow = false;
   })
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
