## 问题现象

开通页面点击立即购买，弹出支付方式，选择华为支付，拉起收银台，取消支付后页面显示重新支付，点击重新支付不会跳转到开通页面。

## 背景知识

* 拉起通用收银台：商户客户端构建[PaymentInfo](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/payment-paymentservice#section1158325715256)参数调用[cashierPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/payment-paymentservice#section107051621123013)接口拉起Payment Kit通用收银台，用户选择支付方式并确认支付后，Payment Kit客户端将支付信息[PickerResult](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/payment-paymentservice#section1129353710263)返回给商户客户端 。
* 用户在拉起通用收银台后，如果用户在支付、签约等过程中取消退出，[cashierPicker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/payment-paymentservice#section107051621123013)接口会报[1001930000](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/payment-error-code#section1799018109586)错误码。

5. 根据页面内容以及日志中的url链接分析，重新支付页面跳转的是“未查询到缴费订单”页面，而不是开通付费项目的页面或者未支付订单页面。

## 分析结论

用户取消华为支付时，应用没有跳转到开通付费项目的页面或者未支付订单页面，而是访问与业务无关的url，跳转到“未查询到缴费订单”页面。

## 修改建议

需要优化取消华为支付时的页面跳转逻辑，建议使用try-catch捕捉错误码1001930000，使得应用根据业务逻辑正确处理用户取消支付后的页面跳转操作。

```
import { BusinessError } from '@kit.BasicServicesKit';
import { paymentService } from '@kit.PaymentKit';
import { common } from '@kit.AbilityKit';

@Entry
@Component
struct Index {
  context: common.UIAbilityContext = this.getUIContext().getHostContext() as common.UIAbilityContext;

  requestCashierPicker() {
    const paymentInfo: paymentService.PaymentInfo = {
      tradeSummary: '***',
      amount: 100,
      currency: 'CNY',
      extraInfo: '{"***":"***"}'
    };
    paymentService.cashierPicker(this.context, paymentInfo)
      .then((pickerResult: paymentService.PickerResult) => {
        // succeeded in paying
        console.info('succeeded in paying, picker result: ', pickerResult);
      })
      .catch((error: BusinessError) => {
        // failed to pay
        console.error(`failed to pay, error.code: ${error.code}, error.message: ${error.message}`);
        if (error.code === 1001930000) {
          // 应用侧需要根据业务逻辑，正确处理用户取消支付后的页面跳转操作，避免显示不相关的页面
        }
      });
  }

  build() {
    Column() {
      Button('requestCashierPicker')
        .type(ButtonType.Capsule)
        .width('50%')
        .margin(20)
        .onClick(() => {
          this.requestCashierPicker();
        });
    }
    .width('100%')
    .height('100%');
  }
}
```
