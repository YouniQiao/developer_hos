## 场景介绍

手机模拟NFC标签并设置标签信息是实用工具类应用的典型场景之一，如模拟门禁卡、手机碰一碰跳转应用等。

本示例使用[@ohos.nfc.cardEmulation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-cardemulation)模块实现手机模拟NFC卡片功能，利用[@ohos.nfc.tag](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-nfctag)模块实现读卡功能。基于[openLink](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-application-uiabilitycontext#openlink12)跳转模拟NFC卡片的应用信息。

## 实现思路

1. 在EntryAbility的onCreate函数中使用[canIUse](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-syscap#caniuse)判断是否具有NFC能力和使用[cardEmulation.hasHceCapability](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-cardemulation#cardemulationhashcecapability9)判断是否具有HCE能力，并新建[HceService](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-cardemulation#hceservice8)类以及初始化。

   ```
   onCreate(want: Want): void {
     this.context.getApplicationContext().setColorMode(ConfigurationConstant.ColorMode.COLOR_MODE_NOT_SET);
     hilog.info(DOMAIN, 'testTag', '%{public}s', 'Ability onCreate');
     GlobalContext.initContext(this.context);
     let nfcUtil: NfcUtil = NfcUtil.getInstance();
     // 判断设备是否支持NFC能力
     if (!canIUse('SystemCapability.Communication.NFC.Core')) {
       hilog.error(0x0000, 'testTag', 'nfc unavailable.');
       return;
     }

     NfcUtil.getInstance().init(want);
     nfcUtil.hceElementName = {
       bundleName: want.bundleName ?? '',
       abilityName: want.abilityName ?? '',
       moduleName: want.moduleName,
     };
     if (!cardEmulation.hasHceCapability()) {
       hilog.error(0x0000, 'testTag', 'hce unavailable.');
       return;
     }
     NfcUtil.getInstance().hceService = new cardEmulation.HceService();
   }
   ```
2. 在主页新增制作卡片tabBar和读卡tabBar，在[onwillshow](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-tabcontent#onwillshow12)中开启监听和HCE监听。

   ```
   // ...
   struct HomePage {
     pathStack: NavPathStack = new NavPathStack()
     private context: UIContext = this.getUIContext()
   // ...
     build() {
       Navigation(this.pathStack) {
         Tabs({ barPosition: BarPosition.End }) {
           TabContent() {
             ReadCardPage()
           }
           .tabBar(this.tabBuilder(0, $r('app.media.read_card_selected'),
             $r('app.media.read_card_normal')))
           .onWillShow(() => {
             if (!NfcUtil.getInstance().isNfcOpen()) {
               hilog.info(0x0000, 'testTag', 'nfc is not open');
               return
             }
             NfcUtil.getInstance().nfcTagOn(NfcUtil.getInstance().readerModeCb)
           })
           .onWillHide(() => {
             NfcUtil.getInstance().nfcTagOff()
           })
           TabContent() {
             HcePage()
           }
           .onWillShow(() => {
             if (!NfcUtil.getInstance().isNfcOpen()) {
               hilog.info(0x0000, 'testTag', 'nfc is not open');
               return
             }
             NfcUtil.getInstance().hceServiceOn(NfcUtil.getInstance().hceCommonTransmit)
           })
           .onWillHide(() => {
             if (!NfcUtil.getInstance()) {
               hilog.info(0x0000, 'testTag', 'getInstance null');
               return
             }
             if (NfcUtil.getInstance().hceElementName != undefined) {
               if (NfcUtil.getInstance().hceService != undefined) {
                 try {
                   let hceService = NfcUtil.getInstance().hceService as cardEmulation.HceService;
                   hceService.stop(NfcUtil.getInstance().hceElementName);
                 } catch (error) {
                   hilog.error(0x0000, 'testTag', 'hceService.stop error = %{public}s', JSON.stringify(error));
                 }
               }
             }
           })
           .tabBar(this.tabBuilder(1, $r('app.media.write_card_selected'),
             $r('app.media.write_card_normal')))
         }
       }
   // ...
     }
   }
   ```
3. 使用[start](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-cardemulation#start9)启动HCE业务功能，并动态注册AID，AID可自行编辑，HCE ElementName为HCE应用的相关信息，并使用[on](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-cardemulation#on8)接口用于接收对端读卡设备发送的APDU数据。

   ```
   public hceServiceOn(callback: AsyncCallback<number[], void>) {
     if (this.hceService != undefined) {
       // 调用tag模块中前台优先的接口，使能前台应用程序优先处理所发现的NFC标签功能
       try {
         // 调用接口使能前台HCE应用程序优先处理NFC刷卡功能
         let aidList = [CommonConstants.aidStr];
         this.hceService.start(this.hceElementName, aidList);
         // 订阅HCE APDU数据的接收
         this.hceService.on('hceCmd', callback);
       } catch (error) {
         hilog.error(0x0000, 'testTag', 'on readerMode error = %{public}s', JSON.stringify(error));
       }
     }
   }
   ```
4. HCE收到读卡器发送的包含AID数据的指令后，将跳转应用链接打包成二进制并使用[transmit](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-cardemulation#transmit9)发送APDU数据到对端读卡设备。

   ```
   public hceCommonTransmit(error : BusinessError, hceCommand : number[]): void {
     if (!error) {
       if (hceCommand == null) {
         hilog.error(0x0000, 'testTag', 'hceCommandCb has invalid hceCommand.');
         return;
       }
       if (NfcUtil.instance.hceService == undefined) {
         hilog.error(0x0000, 'testTag', 'hceService undefined.');
         return;
       }
       if (NfcUtil.getInstance().recordBytes == undefined) {
         hilog.info(0x0000, 'testTag', 'recordBytes undefined.');
         return;
       }
       NfcUtil.instance.hceService.transmit(NfcUtil.instance.recordBytes).then(() => {
       }).catch((err: BusinessError) => {
         hilog.error(0x0000, 'testTag', 'hceService transmit Promise error = %{public}s', JSON.stringify(err));
       });
     } else {
       hilog.error(0x0000, 'testTag', 'hceCommonTransmit error %{public}s', JSON.stringify(error));
     }
   }
   ```
5. 读卡器发送包含AID数据指令后接收应用跳转链接数据使用[openLink](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-application-uiabilitycontext#openlink12)拉起应用。

   ```
   isoDep.transmit(CommonConstants.CMD_DATA).then((response: number[]) => {
     let ndefMessage: tag.NdefMessage = tag.ndef.createNdefMessage(response);
     let ndefRecords: tag.NdefRecord[] = ndefMessage.getNdefRecords();
     for (let i = 0; i < ndefRecords.length; i++) {
       // 解析NDEF标签的数据
       let ndefRecord: tag.NdefRecord = ndefRecords[i];
       let rpl: number[] = ndefRecord.payload; // 获取NDEF记录的类型负载
       let url = NdefTagModel.getUriContent(rpl);
       let context = GlobalContext.getContext();
       context.openLink(url, { appLinkingOnly: true }).then(() => {
         hilog.info(0x0000, 'testTag', `readerModeCb Succeeded in opening link.`);
       })
         .catch((error: BusinessError) => {
           hilog.error(0x0000, 'testTag', `Failed to open link, code: ${error.code}, message: ${error.message}`);
         });
     }
   }).catch((err: BusinessError) => {
     hilog.error(0x0000, 'testTag', 'readerModeCb isoDep.transmit() err = %{public}s.', JSON.stringify(err));
     return;
   });
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。

![](./img/f447ae75.png)

模拟卡片中填写的跳转应用需预先安装。

## 权限说明

* 允许应用读写Tag卡片：[ohos.permission.NFC\_TAG](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissionnfc_tag)。
* 允许应用实现卡模拟功能：[ohos.permission.NFC\_CARD\_EMULATION](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/permissions-for-all#ohospermissionnfc_card_emulation)。
