## 场景介绍

首页下拉进入二楼是购物比价类应用的典型场景之一。

本示例基于[触摸事件](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-events-touch)和[animateTo](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#animateto)实现向下滑动首页进入二楼页面的功能。

## 实现思路

1. 通过监测[TouchType](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-appendix-enums#touchtype)中的Move事件，触发页面位置改变事件。

   ```
   .onTouch((event) => {
     switch (event.type) {
       case TouchType.Down:
         // Press event
         this.onTouchDown(event);
         break;
       case TouchType.Move:
         // Swipe event
         this.onTouchMove(event);
         break;
       case TouchType.Up:
       case TouchType.Cancel:
         // Touch lift or cancel touch event
         this.onTouchUp();
         break;
     }
   })
   ```

2. 当监测到下滑高度到达指定高度时，触发二楼展开动画。

   ```
   // (this.floorHeight - Math.abs(this.offsetY)) is the drag distance
   if ((this.floorHeight - Math.abs(this.offsetY)) > this.expandFloorTriggerDistance) {
     // Unfold the second-floor animation
     this.expandSecondFloor();
   }
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
