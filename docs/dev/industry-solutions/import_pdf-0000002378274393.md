## 场景介绍

PDF文件导入与打开是教育类应用中的高频场景之一。

本示例通过[@ohos.file.picker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-picker)选择本地PDF文档，使用[PdfView](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/pdf-arkts-pdfview-component)进行预览。

## 实现思路

1. 通过[DocumentViewPicker.select](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-picker#select)拉起文件选择器，选择PDF文档。

   ```
   ListItem(){
     // ...
   }
   .onClick(() => {
     if (item.name === this.context.resourceManager.getStringSync($r('app.string.import_pdf'))) {
       this.documentSelectOptions.fileSuffixFilters = ['.pdf'];
       this.documentPicker.select(this.documentSelectOptions).then((documentSelectResult: string[]) => {
         this.uri = documentSelectResult;
         this.uriPrint.push(this.uri[0]);
         this.index++;
         let file: fs.File | null = null;
         try {
           file = fs.openSync(this.uri[0]);
         } catch (e) {
         } finally {
           if (file !== null) {
             fs.closeSync(file.fd);
           }
         }
         if (file !== null) {
           this.fileName[this.index-1] = file.name;
           this.recentDocList.push(new RecentDocument(this.fileName[this.index-1]));
           this.uriMap.set(this.fileName[this.index-1], this.uri[0]);
         }
       });
     }
   });
   ```
2. 通过[fs.openSync](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-file-fs#fsopensync)获取到File对象，使用[loadDocument](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/pdf-arkts-pdfviewmanage#section073321844615)加载PDF文档，利用[PdfView](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/pdf-arkts-pdfview-component#section724521414201)进行预览。

   ```
   let file:fs.File|null=null;
   try {
     file = fs.openSync(this.uriMap.get(this.name));
   }catch (e){
     // ...
   }finally {
     fs.closeSync(file?.fd);
   }
   if (file !== null) {
     this.controller.loadDocument(file.path);
   }

   PdfView({
     controller: this.controller,
     pageFit: pdfService.PageFit.FIT_WIDTH,
     showScroll: true,
   });
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
