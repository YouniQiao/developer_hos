## 场景介绍

当需要持久化应用数据时，本地数据库是一种常用的方式，HarmonyOS支持键值型和关系型数据库，用户可以根据自己的业务复杂度自由选择，键值型数据库通常用于数据关系和业务关系较少的业务数据存储，关系型数据库则支持更复杂的场景。

本示例实现了使用本地数据库简单示例，包含键值型和关系型数据库的增删和查询操作，同时通过列表的形式展示数据库中已有的数据或展示查询结果，通过[distributedKVStore](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-distributedkvstore)接口实现键值型数据库功能，通过[relationalStore](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-data-relationalstore)实现关系型数据库功能。

## 实现思路

1. 使用[distributedKVStore](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-distributedkvstore)接口实现键值型数据库功能。

   ```
   /*
   创建键值型数据库
    */
   initKVManager() {
     // 配置KVManager
     const kvManagerConfig: distributedKVStore.KVManagerConfig = {
       context: this.context,
       bundleName: 'com.example.datatest'
     };

     try {
       // 创建KVManager实例
       this.kvManager = distributedKVStore.createKVManager(kvManagerConfig);
       console.info('Succeeded in creating KVManager.');

       // 配置KVStore选项
       const options: distributedKVStore.Options = {
         createIfMissing: true,
         encrypt: false,
         backup: false,
         autoSync: false,
         kvStoreType: distributedKVStore.KVStoreType.SINGLE_VERSION,
         securityLevel: distributedKVStore.SecurityLevel.S1
       };

       // 获取KVStore实例
       this.kvManager.getKVStore<distributedKVStore.SingleKVStore>('storeId', options, (err, store) => {
         if (err) {
           console.error(`Failed to get KVStore: Code:${err.code}, message:${err.message}`);
           return;
         }
         console.info('Succeeded in getting KVStore.');
         this.kvStore = store;
         this.getAllUsers();
       });
     } catch (e) {
       let error = e as BusinessError;
       console.error(`Failed to create KVManager. Code:${error.code}, message:${error.message}`);
     }
   }
   ```
2. 使用[relationalStore](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-data-relationalstore)实现关系型数据库功能。

   ```
   /*
    创建关系型数据库并新建数据表
    */
   initDatabase() {
     const STORE_CONFIG: relationalStore.StoreConfig = {
       name: 'database.db',
       securityLevel: relationalStore.SecurityLevel.S1
     };
     relationalStore.getRdbStore(this.context, STORE_CONFIG, (err, rdb) => {
       this.rdbStore = rdb;
       this.rdbStore.executeSql('CREATE TABLE IF NOT EXISTS users (id INTEGER PRIMARY KEY AUTOINCREMENT, name TEXT, UNIQUE(name))');
       this.queryAllUser();
     });
   }
   ```
3. 使用[半模态页面](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-sheet-transition#bindsheet)添加数据。

   ```
   Image($r('app.media.add'))
     .onClick(() => {
       this.isShowSheet = !this.isShowSheet;
     })
     .bindSheet($$this.isShowSheet, this.SheetBuilder(), {
       detents: [Constants.KVSORE_SHEET_HEIGHT + this.keyHeight],
       preferType: SheetType.BOTTOM,
       title: { title: $r('app.string.Add_User') }
     })
     .size({ width: Constants.BUTTON_HEIGHT, height: Constants.BUTTON_HEIGHT });
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
