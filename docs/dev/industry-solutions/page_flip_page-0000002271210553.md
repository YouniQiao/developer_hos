## 场景介绍

本示例通过[Swiper](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-swiper)、[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)和[Reader Kit](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/reader-kit-guide)分别实现了左右翻页、上下翻页、仿真翻页三种翻页方式，方便用户在阅读时选择合适的翻页方式。

## 实现思路

* 左右翻页：基于[Swiper](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-swiper)组件对页面进行左右翻页。

  ```
  Swiper(this.swiperController) {
    LazyForEach(this.data, (item: string) => {
      Text($r(item))
      // ...
    }, (item: string) => item)
  }
  ```
* 上下翻页：基于[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)组件对页面进行上下滑动翻页。

  ```
  List({ initialIndex: this.currentPageNum - CONFIGURATION.PAGEFLIPPAGECOUNT }) {
    LazyForEach(this.data, (item: string) => {
      ListItem() {
        Text($r(item))
        // ...
      }
    }, (item: string) => item)
  }
  ```
* 仿真翻页：基于[Reader Kit](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/reader-kit-guide)提供的API实现仿真翻页。
  1. 拷贝rawfile下的文件进入沙箱。

     ```
     myRawfileCopy(context: common.UIAbilityContext) {}
     ```
  2. 初始化组件控制器、默认设置项以及书籍解析器，并调用startPlay方法初始化阅读器页面。

     ```
     // 组件控制器，用于调用排版相关能力
     private readerComponentController: readerCore.ReaderComponentController = new readerCore.ReaderComponentController();
     // 默认设置项，用于初始化阅读器页面的默认属性
     private readerSetting: readerCore.ReaderSetting = {}
     // 书籍解析器，用于注册给组件控制器，供排版引擎调用
     private bookParserHandler: bookParser.BookParserHandler | null = null;
     // 初始化阅读器
     aboutToAppear(): void {
       this.startPlay(filePath, spineIndex, domPos);
     }
     ```
  3. 构建[ReadPageComponent（阅读页组件）](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/reader-api-readpagecomponent)，用于显示阅读内容。

     ```
     ReadPageComponent({
       // ...
     })
     ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
