## 场景介绍

刷题页面是教育类应用的高频使用场景之一，如用户需要刷题练习各类学科知识。

本示例基于[Swiper](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-swiper)、[LazyForEach](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-rendering-control-lazyforeach)组件实现可滑动刷题页面。

## 实现思路

1. 使用[Swiper](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-swiper)组件，搭配[LazyForEach](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-rendering-control-lazyforeach)渲染控制机制，实现可滑动刷题页面。

   ```
   Swiper(this.examPaperSwiperController) {
     LazyForEach(this.testData, (item: Question, index: number) => {
       QuestionItem({
         index,
         item,
         isSubmitted: this.isSubmitted[index],
         onSelectOption: () => {
           this.isSubmitted[index] = NOT_SUBMITTED;
         }
       });
     }, (item: Question) => item.title);
   }
   .index(this.index)
   .cachedCount(2)
   .width('100%')
   .indicator(false)
   .loop(false)
   .onChange((event: number) => {
     this.index = event;
   });
   ```
2. 通过Swiper组件的控制器[SwiperController](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-swiper#swipercontroller)，实现控制Swiper翻页功能。

   ```
   // 翻至上一页
   this.examPaperSwiperController.showPrevious();
   // 翻至下一页
   this.examPaperSwiperController.showNext();
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
