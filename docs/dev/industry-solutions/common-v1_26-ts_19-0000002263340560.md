## 问题现象

打开模拟器登录，提示登录失败'Login Failed!'，提示信息如下图：

![](./img/df44ab9f.png "点击放大")

进入创建模拟界面提示'Download the system image first.'，提示信息如下图：

![](./img/54fac5eb.png "点击放大")

## 背景知识

登录、使用模拟器需要考虑如下信息：

* 把账户登录网址加到白名单：DevEco Studio需要先登录，才能正常使用，因此需要按照公司要求在防火墙里把登录网址配置到白名单。
* 把镜像下载域名加到白名单：'Download the system image first'说明需要先下载镜像文件，按照公司要求将对应的域名配置到白名单。

## 解决方案

为了解决模拟器登录和下载镜像时的网络限制，按如下步骤添加白名单：

1. 把账户登录网址加到白名单，网址分别为：

   ```
   https://id1.cloud.huawei.com/
   ```

   ```
   https://cn.devecostudio.huawei.com/
   ```
2. 把镜像下载域名（访问的端口为：443）加到白名单，镜像下载域名分别为：
   * 获取镜像数据：

     ```
     devecostudio-drcn.deveco.dbankcloud.com
     ```
   * 下载镜像文件：

     ```
     updatebeta.hicloud.com
     ```
   * 下载更新镜像文件：

     ```
     update.dbankcdn.com
     ```
