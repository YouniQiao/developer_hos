## 场景介绍

聊天信息加密是社交通讯类应用中高频使用场景之一，如用户在发送消息时需要进行加密操作保证通讯信息安全。

本示例使用[SM2非对称密钥加解密](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/crypto-sm2-asym-encrypt-decrypt)实现聊天信息加密。示例中的密钥对为生成器生成，实际使用可以更换为使用者所提供的密钥对。

## 实现思路

1. 创建一个[使用SM2非对称密钥加解密](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/crypto-sm2-asym-encrypt-decrypt)的非对称密钥生成器，生成密钥对。

   ```
   let generator = cryptoFramework.createAsyKeyGenerator('SM2_256'); // 创建生成器
   let keyPair = generator.generateKeyPairSync(); // 生成密钥对
   ```
2. 对输入内容执行加密操作，并将结果转换为Base64编码的字符串。

   ```
   let encryptText = await this.encryptMessagePromise(keyPair.pubKey, plainText); // 加密
   encryptData = base64Helper.encodeToStringSync(encryptText.data); // 转换为Base64编码
   ```
3. 对密文执行解密操作，并将结果转换为UTF-8可读字符串。

   ```
   let decryptText = await this.decryptMessagePromise(keyPair.priKey, encryptText); // 解密
   decryptData = CipherUtil.uint8ArrayToString(decryptText.data); // 转为可读字符串
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
