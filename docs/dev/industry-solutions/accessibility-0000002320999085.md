## 场景介绍

无障碍读屏是各类应用中的高频使用场景之一，是面向视障人士开发的一项功能，可以在点击屏幕时朗读出对应的内容。

本示例基于[无障碍属性](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-accessibility)实现无障碍读屏时朗读文本内容。

## 实现思路

通过对组件添加[accessibilityText()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-accessibility#accessibilitytext)属性来给组件加上无障碍读屏时朗读的文本内容。

```
Row() {
  Search({ placeholder: CommonConstants.SEARCH })
    .width(CommonConstants.SEARCH_INDEX_WIDTH)
    .accessibilityText(CommonConstants.SEARCH)
  Image($r('app.media.scan'))
    .width(CommonConstants.SCAN_WIDTH)
    .margin({left:CommonConstants.SCAN_WIDTH_MARGIN_LEFT})
    .accessibilityText(CommonConstants.SCAN)
}
```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
