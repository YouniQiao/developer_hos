## 场景介绍

聊天消息引用及跳转是社交通讯类应用的高频使用场景之一，如用户对某条消息引用回复、跳转至引用消息。

本示例基于[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)列表实现对聊天消息的排序与精准定位，通过设置[Text](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-text)的[bindSelectionMenu](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-text#bindselectionmenu11)属性实现消息引用。

## 实现思路

1. 设置[Text](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-text)的[bindSelectionMenu](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-text#bindselectionmenu11)属性，自定义菜单项，引用聊天消息。

   ```
   // 自定义菜单项，引用聊天消息
   Text()
     .bindSelectionMenu(TextSpanType.DEFAULT, this.CustomMenu, TextResponseType.DEFAULT)
   ```
2. 执行列表滚动条[Scroll](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-scroll)的[scrollToIndex](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-scroll#scrolltoindex)方法，跳转至引用的聊天消息。

   ```
   // 跳转至引用的聊天消息
   this.listScroller.scrollToIndex(this.item.referenceIndex);
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
