## 场景介绍

关键词查找聊天记录是社交通讯类应用中高频使用场景之一，如用户查找聊天内容时，可输入关键词快速定位其所在位置。

本示例基于[正则表达式](https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-arkts-3)实现搜索关键词并高亮显示的功能，基于[Navigation](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation)和[List](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list)实现跳转到关键词所在位置的功能。

## 实现思路

1. 定义函数findKeyword用于查找与关键词匹配的聊天信息，将关键词提取出来并标记为高亮显示。

   ```
   Text() {
     ForEach(this.findKeyword(item.content, this.text), (item: formatString) => {
       // 搜索关键词高亮显示
       if (item.isAim) {
         Span(item.char)
           .fontColor('#0A59F7') // 高亮颜色
       } else {
         Span(item.char)
       }
     });
   };
   ```
2. 通过[PopInfo](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation#popinfo11)接收返回的index，利用[scrollToIndex](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-scroll#scrolltoindex)方法跳转到搜索关键词所在位置。

   ```
   Image($r('app.media.ic_public_ellipsis'))
     .onClick(() => {
       this.pathInfos.pushPathByName('chatDetailsPage', null, (info: PopInfo) => {
         this.index = info.result as number;
         // 跳转到搜索关键词所在位置
         this.listScroller.scrollToIndex(this.index);
       });
     });
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
