## 场景介绍

Canvas绘制固定图形是儿童教育类应用的高频使用场景之一。

本示例基于[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)实现在儿童绘画板中动态绘制矩形、圆形、三角形等固定图形。

## 实现思路

1. 基于[Canvas](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-components-canvas-canvas)组件构建画板，在Canvas组件的[onTouch](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-events-touch#ontouch)事件中进行绘制。

   ```
   Canvas(this.context)
     .onTouch((event) => {});
   ```
2. 使用[CanvasRenderingContext2D](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-canvasrenderingcontext2d)的rect函数绘制矩形，arc函数绘制圆形，使用moveTo和lineTo函数绘制三角形。

   ```
   this.context.beginPath();
   this.x = event.touches[0].x;
   this.y = event.touches[0].y;
   this.context.strokeStyle = this.penColorList[this.penColor];
   this.context.lineWidth = this.paintSize;
   if (this.shapeNum === 0) { // 绘制矩形
     this.context.rect(this.x0, this.y0, this.x - this.x0, this.y - this.y0);
   } else if (this.shapeNum === 1) { // 绘制圆形
     let length = (this.x - this.x0) * (this.x - this.x0) + (this.y - this.y0) * (this.y - this.y0);
     this.context.arc(this.x0, this.y0, Math.sqrt(length), 0, 360);
   } else if (this.shapeNum === 2) { // 绘制三角形
     let length = (this.x - this.x0) * (this.x - this.x0) + (this.y - this.y0) * (this.y - this.y0);
     this.context.moveTo(this.x0, this.y0 - Math.sqrt(length));
     this.context.lineTo(this.x0 + Math.sqrt(length * 0.75), this.y0 + Math.sqrt(length) / 2);
     this.context.lineTo(this.x0 - Math.sqrt(length * 0.75), this.y0 + Math.sqrt(length) / 2);
     this.context.closePath();
     this.context.stroke();
   }
   this.context.stroke();
   ```

## 约束与限制

* 本示例支持API Version 20 Release及以上版本。
* 本示例支持HarmonyOS 6.0.0 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 6.0.0 Release及以上版本进行编译运行。
