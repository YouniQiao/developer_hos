## 问题现象

进入应用时，一直停留在启动页，无法进入应用。

## 背景知识

* [Video](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-media-components-video)：用于播放视频文件并控制其播放状态的组件。
* [onFinish](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-media-components-video#onfinish)：播放结束时触发该事件，支持attributeModifier动态设置属性方法。
* [组件可见区域变化事件](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-component-visible-area-change-event)：组件可见区域变化事件是组件在屏幕中的显示区域面积变化时触发的事件，提供了判断组件是否完全或部分显示在屏幕中的能力，适用于广告曝光埋点之类的场景。

4. 判断应用监听了[Video](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-media-components-video)的[onFinish](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-media-components-video#onfinish)事件，在视频播放结束的回调函数中跳转到主页。在首次冷启动情况下，若用户在视频播放完成前通过侧滑操作将应用退至后台，后续热启动时视频将无法继续播放，导致[onFinish](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-media-components-video#onfinish)事件无法被正确触发。

## 分析结论

启动页视频未播放结束就退出应用到后台，视频暂停，后续热启动视频不会继续播放，无法触发页面跳转逻辑，导致无法进入应用。

## 修改建议

监听[组件可见区域变化事件](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-component-visible-area-change-event)，在Video组件重新可见时继续播放视频。

```
@Entry
@Component
export struct VideoPage {
  videoSrc: Resource = $rawfile('TestVideo.mp4'); //替换为启动页视频
  controller : VideoController = new VideoController();

  build() {
    Column() {
      Video({
        src : this.videoSrc,
        controller: this.controller
      })
        .autoPlay(true)
        .loop(false)
        .objectFit(ImageFit.Contain)
        .onFinish(() => {
          // 此处执行页面跳转
        })
        .onVisibleAreaChange([0.0, 1.0], (isExpanding: boolean, currentRatio: number) => {
          // 当Video组件全部可见时，继续播放
          if (isExpanding || currentRatio >= 1.0) {
            console.info('Test Video is fully visible.');
            this.controller.start();
          }
          // 当Video组件不可见时，暂停播放
          if (!isExpanding && currentRatio <= 0.0) {
            console.info('Test Video is completely invisible.');
            this.controller.pause();
          }
        })
    }
    .height('100%')
    .width('100%')
    .justifyContent(FlexAlign.Center)
  }
}
```
