## 场景介绍

查看通讯录是社交通讯类应用中高频使用场景之一，如用户需要查找联系人，并拨打电话或发送短信。

本示例基于[AlphabetIndexer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-alphabet-indexer)实现侧边栏字母排序索引，快速定位联系人。通过[@ohos.telephony.call](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-call)跳转拨号界面呼叫联系人，使用[UIAbilityContext](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-application-uiabilitycontext)拉起系统短信应用发送短信。

## 实现思路

1. 使用[AlphabetIndexer](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-alphabet-indexer)实现侧边栏字母排序索引，快速定位联系人：对联系人姓氏首字母进行字母表排序，然后使用三方库[pinyin4js](https://ohpm.openharmony.cn/#/cn/detail/@ohos%2Fpinyin4js)中 convertToPinyinString 方法提取出排序后的首字母列表，作为 AlphabetIndexer 组件的字符串数组入参。

   ```
   private preProcessContactList() {
     let sortedContactList = sourceContactList.sort((a: ContactInfo, b: ContactInfo) => a.name < b.name ? -1 : 1)
     for (let i = 0; i < sortedContactList.length; i++) {
       const lastName = sortedContactList[i].name.substring(0, 1)
       const firstLetter: string =
         pinyin4js.convertToPinyinString(lastName, '', pinyin4js.FIRST_LETTER).trim().toUpperCase()
       if (!this.contactList[firstLetter]) {
         this.contactList[firstLetter] = []
       }
       this.contactList[firstLetter].push(sortedContactList[i])
     }
     this.lastNameFirstLetters = Object.keys(this.contactList).sort()
   }
   ```
2. 通过[@ohos.telephony.call](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-call)的call.makeCall方法实现拨打电话的功能。

   ```
   call.makeCall(phoneNumber)
   ```
3. 使用[UIAbilityContext](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-application-uiabilitycontext)的[startAbilityForResult](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-application-uiabilitycontext#startabilityforresult)方法实现[应用内跳转到短信编辑界面](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/telephony-sms#应用内跳转到短信编辑界面)的功能。

   ```
   export function sendMessage(uiAbilityContext: common.UIAbilityContext, phoneNumber: string) {
     let params: Array<Record<string, string>> = [{ 'telephone': phoneNumber }];
     let want: Want = {
       bundleName: 'com.ohos.mms',
       abilityName: 'com.ohos.mms.MainAbility',
       parameters: {
         contactObjects: JSON.stringify(params),
         pageFlag: 'conversation'
       }
     };
     uiAbilityContext.startAbilityForResult(want)
   }
   ```

## 约束与限制

* 本示例支持API Version 16 Release及以上版本。
* 本示例支持HarmonyOS 5.0.4 Release SDK及以上版本。
* 本示例需要使用DevEco Studio 5.0.4 Release及以上版本进行编译运行。
