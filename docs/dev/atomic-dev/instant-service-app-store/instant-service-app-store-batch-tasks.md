1. 进入“门店管理”页签， 点击任务管理。

   ![](./img/9ebb503a.png "点击放大")
2. 可在此查询“门店导入/导出/修改/删除”的操作结果，并且能下载结果文件查看详情。

   ![](./img/ccd76615.png)

   导出时“导入成功”列对应导入成功数。

   ![](./img/477d8923.png "点击放大")
3. 回到门店管理页。

   ![](./img/15614a49.png "点击放大")
