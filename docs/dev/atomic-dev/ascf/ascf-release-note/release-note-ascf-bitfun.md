## 1.0.0-beta.1

**发布日期**：2026/03/31

插件首版本发布，该版本新增特性如下所示：

1. 新增ASCF开发助手，支持在鸿蒙电脑版DevEco Studio中开发元服务。
2. 支持创建、导入、打开ASCF元服务项目。
3. 支持快速创建app，page，component。
4. 支持编译构建，支持热加载运行。
5. 支持开启V8调试。
6. 支持查看设备日志。
