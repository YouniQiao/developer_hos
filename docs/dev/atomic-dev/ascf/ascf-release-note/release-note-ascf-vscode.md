## 1.0.10

**发布日期**：2026/04/30

该版本新增和增强特性如下所示：

1. 新增元服务签名配置功能，支持在项目配置中直接管理签名信息。
2. 新增集成HarmonyOS ArkTS插件，使开发者在VS Code环境中获得LSP级别的ArkTS语言开发能力，包括代码补全、类型检查等。
3. 新增typings配置项，支持为ASCF运行时新增接口提供代码提示，提升开发体验。

优化ASCF助手代码提示。

## 1.0.8

**发布日期**：2026/02/05

该版本新增和增强特性如下所示：

1. 优化ASCF助手开发者主页界面样式。
2. 优化调试能力，增加下载进度提示。
3. 新增支持HarmonyOS 4及以下设备上调试运行ASCF元服务。
4. 新增支持HarmonyOS 4及以下设备上投屏功能，启动运行阶段自动开启投屏显示。
5. 新增支持网络调试功能，支持在HarmonyOS 4及以下设备上调试，能看到网络请求响应信息。
6. 新增typings配置，支持ASCF运行时新增接口的代码提示。

## 1.0.1

**发布日期**：2025/07/31

1. 新增接口补充typings定义。

## 1.0.0

**发布日期**：2025/07/23

插件首版本发布，该版本新增和增强特性如下所示：

1. 新增ASCF开发助手，支持在VSCode中开发元服务。
2. 支持创建、导入、打开ASCF元服务项目。
3. 支持快速创建app，page，component。
4. 支持js/hjs/css/json/hxml编辑、语法高亮、代码提示等能力。
5. 支持编译构建，支持热加载运行。
6. 支持开启调试。
7. 支持查看设备日志。
