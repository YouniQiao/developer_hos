**问题现象**

ASCF运行时提示版本较低，需要升级 报错信息主要为：ASCF shared library version is too low，update to XXX。

**解决措施**

6.0及以上版本，用户本地设备已安装过ASCF共享库，如果ASCF 共享库有最新版本，开发者在调测阶段，可进入开发者选项，点击更新共享库完成自动升级。
