多项选择器，内部由多个checkbox组成。

**起始版本：** 1.0.0

## 属性

| 名称 | 类型 | 默认值 | 必填 | 描述 |
| --- | --- | --- | --- | --- |
| bindchange | eventhandle | - | 否 | 点击导致checkbox-group中选中项发生改变时触发change事件，detail = \\&#123;value: [选中的checkbox的value的数组]&#125;。 |

见[checkbox组件](/docs/dev/atomic-dev/ascf/components-form-components/components-checkbox#示例)示例代码。
