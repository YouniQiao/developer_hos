**问题现象**

在AGC使用云调试安装ASCF元服务app包失败，提示依赖的动态共享包（hsp）模块未安装。

**可能原因**

云调试设备未安装ASCF引擎。

**解决措施**

可使用云调试设备于负一屏或应用市场搜索“元服务ASCF示例”，随后直接运行以自动安装ASCF引擎。
