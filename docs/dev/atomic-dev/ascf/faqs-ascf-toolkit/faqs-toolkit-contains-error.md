**问题现象**

使用命令行转换工具时报错：“ERROR: XXX contains error, please check it.”

**解决措施**

方法一：确认XXX路径是否存在，检查报错提示的路径是否存在对应的json文件。

方法二：打开报错提示的json文件，检查文件中的json格式是否有误，修复为正确的json结构即可。
