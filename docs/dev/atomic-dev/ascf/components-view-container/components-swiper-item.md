swiper组件的子组件。

**起始版本：** 1.0.0

## 约束与限制

仅可放置在swiper组件中，宽高自动设置为100%。

## 属性

| 名称 | 类型 | 默认值 | 必填 | 描述 |
| --- | --- | --- | --- | --- |
| item-id | string | - | 否 | swiper-item的标识符。 |
