# 应用侵权投诉处理指引

## **1. 目的**

华为应用市场是提供应用程序信息存储的空间，其上应用均由开发者提供。为了保护权利人的著作权、隐私权、商标权、专利权等合法权益，特制订华为应用市场侵权投诉处理指引。

## **2. 处理指引**

### **2.1 官方受理投诉渠道**

**工作时间**：周一至周五 08: 30-12: 00，13: 30-18: 00 (法定节假日除外)。

**联系方式：**

请点击 [AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html#/interactive/appeal/appeal) 进行侵权投诉；若您的账号已被冻结，请发邮件至developer@huawei.com (应用类)，game.business@huawei.com(游戏类)进行投诉。

### **2.2 投诉通知**

如您发现上传至华为应用市场的应用或内容侵犯了您的合法权益，建议您先与开发者联系以解决问题。如果您无法与开发者达成解决方案，请填写并反馈《华为应用市场侵权投诉通知书》，通知书建议包含以下内容：

**2.2.1 投诉方信息**

|  |  |
| --- | --- |
| **投诉方信息** | **说明** |
| 自然人 | 有效身份证明文件 |
| 法人和其他组织 | 企业营业执照、组织的相应执照 |
| 代理人（如有） | 权利人出具的代理委托书、代理人有效证件 |
| 建议同步提供联系方式：联系方式可包含通讯地址、电话或Email等，非必填项。为了保护非利益相关投诉人的隐私，联系方式仅作为投诉进展通报，不对外公开。 | |

**2.2.2 侵权信息**

应用（或其内容）的名称、包名和地址等。

**2.2.3 侵权理由**

具体投诉的权利类型（如商标权、著作权、专利权），并简要说明侵权理由。

**2.2.4 侵权证明材料**

|  |  |
| --- | --- |
| **证明材料** | **说明** |
| 权属说明 | 请依据所投诉权利的类型提交证明材料：  1. 著作权：著作权权属证明，如相关有权机构颁发的著作权证书、作品首次公开发表或发行日期证明材料、创作手稿、经权威机构签发的作品创作时间戳、作品备案证书。  2. 商标权：商标注册证。  3. 专利权：专利权证书，技术特征或者设计特征对比的说明、实用新型或者外观设计专利权评价报告等材料。  4. 其他权利：视权利类型提供。 |
| 授权材料（如有） | 若权属说明中的权利人即为投诉方，则无需特殊说明；若权属说明中的权利人不是投诉方，需说明投诉方与权利人的关系，并提交相应材料予以佐证（如授权许可合同、授权委托书、关系公证书等）。 |
| 其他证据（如有） | 1. 争议应用（或其内容）为何侵权的材料，例如法律评估报告、近似度对比分析。  2. 证明权利稳定性的材料，例如商标使用情况证明材料等。  3. 行政、司法机关对于相关纠纷的处理材料，例如行政、司法机关的受理通知书、处理决定、判决书等。 |

**2.2.5 投诉通知要求**

指投诉方要求华为应用市场对被投诉方提供的内容采取何种措施，如内容整改或应用下架等。

### **2.3 投诉有效性判断**

在收到符合要求的《华为应用市场侵权投诉通知书》后，华为应用市场会依据相关法律法规的规定，将投诉内容转发给被投诉方，并初步核实投诉是否属实。

### **2.4 投诉反通知**

被投诉方在收到华为应用市场转发的投诉通知后，若不认可侵权，应在3个工作日内（“投诉反馈时限”）向华为应用市场提交侵权反通知，反通知书建议包含以下内容：

**2.4.1 被投诉方信息**

|  |  |
| --- | --- |
| **被投诉方信息** | **说明** |
| 开发者 | 有效身份证明文件（自然人）或企业营业执照（法人） |
| 代理人（如有） | 开发者出具的代理委托书、代理人有效证件 |
| 建议同步提供联系方式：联系方式可包含通讯地址、电话或Email等，非必填项。为了保护非利益相关投诉人的隐私，联系方式仅作为投诉进展通报，不对外公开。 | |

**2.4.2 被投诉的应用信息**

包括名称、包名和内容地址等。

**2.4.3 不认可投诉的理由**

包括被投诉方主张享有的权利、投诉方的权利及证明材料存在的问题、法律依据等。

**2.4.4 证明不存在侵权行为的材料**

|  |  |
| --- | --- |
| **证明材料** | **说明** |
| 权属说明 | 请依据被投诉权利的类型提交证明材料：  1. 著作权：著作权权属证明，如相关有权机构颁发的著作权证书、作品首次公开发表或发行日期证明材料、创作手稿、经权威机构签发的作品创作时间戳、作品备案证书。  2. 商标权：商标注册证。  3. 专利权：专利权证书，技术特征或者设计特征对比的说明、实用新型或者外观设计专利权评价报告等材料。  4. 其他权利：视权利类型提供，如有疑问请咨询。 |
| 权属证明 | 说明被投诉应用（或其内容）不侵犯投诉方所主张权利的材料：  1. 证明被投诉人已获得相应的授权的材料。  2. 证明争议应用（或其内容）未侵权的材料，例如律所出具的评估报告、内容不近似对比等材料。  3. 证明投诉人的权利稳定性存在瑕疵的材料。  4. 行政、司法机关对于相关纠纷的处理材料，例如行政、司法机关的受理通知书、处理决定、判决书等。 |

**2.4.5 反通知要求**

指被投诉方向华为应用市场提出的，对其提供的内容采取何种措施的要求。

**2.4.6 反通知处理**

在收到符合要求的《华为应用市场侵权投诉反通知书》后，华为应用市场会依据相关法律法规的规定，将反通知书及相关内容转发给投诉方，并告知投诉方可以向主管部门投诉或者向人民法院起诉。

### **2.5 投诉处理**

1. 依据构成侵权的初步证据和服务类型，华为应用市场将及时采取必要措施。在相关法律法规的要求下，华为应用市场将有可能维持或终止所采取的措施（视侵权情况、投诉方的投诉与起诉情况等）。

2. 如需通过法律途径，华为应用市场会配合国家机关的取证工作，争议内容的最终处置以双方协商结论或者国家机关的决定或判决为准。

3. 根据有关主管部门和人民法院的处理结论，或者双方达成的其他一致结果，投诉方和被投诉方均可据此对华为应用市场的投诉处理结果进行申诉，申诉依据包括但不限于胜诉判决、和解协议、谅解书等。

4. 对于开发者多次、故意侵害知识产权的，华为应用市场将对其采取账号关停等处理措施。

### **3. 温馨提示**

1. 为了确保投诉的真实性和有效性，投诉证明材料和反诉证明材料中涉及的相关资质文件，原则上应提供原件（扫描件），不能提供原件的，应提供权利人签字盖章的复印件。

2. 若投诉方已经就侵权内容向行政部门投诉或向法院起诉，建议将相关受理证明、行政机关或法院的决定或判决等一同提交给华为应用市场。

3. 投诉通知书、反通知书及所附证明材料，将有可能被依法转发给投诉方或被投诉方，建议提前对相关信息进行脱敏处理。
