# 应用年龄分级标准

为了方便开发者在上传应用时更准确地选择年龄分级，请参考如下华为应用市场年龄分级标准说明：

应用年龄分级标准：

|  |  |  |
| --- | --- | --- |
| **分级类别** | **分级描述** | **分级详细说明** |
| 3+ | 年满3周岁 | App内容适合所有年龄段的用户。此等级App应避免涉及可能引发儿童不适的暴力、恐怖、性暗示元素，并不得包含不良语言、低俗幽默或其他任何受年龄限制的内容。 |
| 8+ | 年满8周岁 | 此等级App可能包含不适合未满8周岁儿童的内容，包括  极少/极轻微的卡通的、幻想的暴力内容（但不得包含逼真的暴力内容）、  极少/极轻微的令人惊慌的元素（但不得包含令人感到恐怖的元素）、  极少/极轻微的性相关的内容（但不得包含暴露的服装、恋爱情节与功能、强调性别生理特征或生殖器的描绘）、  极少/极轻微的不良语言（不得包含轻微的谩骂或更严重的内容）和/或  极少/极轻微的粗俗幽默内容（不得包含呕吐、放屁）。 |
| 12+ | 年满12周岁 | 此等级App可能包含不适合未满12周岁儿童的内容，包括  偶尔/轻微的针对人物或动物的逼真暴力内容（不得含有经常/较多的暴力场面）、  轻微的血腥场景、  偶尔/轻微的令人惊慌或恐怖的元素（不得含有经常/较多的令人感到惊慌或恐怖的元素）、  偶尔/轻微的性相关内容（但不得含有挑逗性内容、暴露服装和裸体）、  偶尔/轻微的不良语言（不得含有咒骂或更严重的内容）和/或  偶尔/轻微的粗俗幽默内容。 |
| 16+ | 年满16周岁 | 此等级App可能包含不适合未满16周岁儿童的内容，包括  经常/较强烈暴力内容（不得鼓励玩家做出最激进、残暴或血腥的暴力行为）、  经常/较强烈血腥场景、  经常/较强烈令人惊慌或恐惧的元素（不得包含频繁/强烈的令人感到恐怖的元素）、  经常/较强烈性相关内容（但不得包含性行为、性暗示、大量或挑逗性的暴露服装和裸体内容）与亲密接触、  模拟赌博、  经常/较强烈不良语言（不得包含与性相关的脏话）和/或  对受限制物品的描写（如烟草、酒精、毒品等）。 |
| 18+ | 年满18周岁 | 此等级App可能包含不适合未满18周岁儿童的内容，包括  频繁/强烈的暴力内容、  频繁/强烈的令人恐惧的元素、  频繁/强烈的色情内容、  赌博、  性脏话和/或  对受限制物品的美化（如烟草、酒精、毒品等）。  请注意：18+等级的App有可能不能在中国大陆以及其他某些地区上架 |
