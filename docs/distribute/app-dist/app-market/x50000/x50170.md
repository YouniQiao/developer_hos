# 应用审核Checklist

发布应用前，请仔细阅读应用审核指南，了解发布应用至华为应用市场需要遵循的规则和要求，并提前准备发布所需的文件、资源，以便您能顺利、快速通过发布审核流程。

为了帮助您更好地准备应用提交审核，在进入审核流程前，请您花几分钟参考如下Checklist进行自检，以提前排查常见问题。请确认已完成所有检查项后，再前往[AppGallery Connect网站](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html#/)提交应用上架审核，并以最终审核意见为准。

## 一、开发前

|  |  |  |  |
| --- | --- | --- | --- |
| 阶段 | 模块 | 检查项 | 修改建议 |
| 开发前 | 应用名称- | 名称是否为泛词，如：免费壁纸 | 请修改您的应用名称，避免以电话、邮件、日历等广义归纳、普遍且不具有识别性的词汇来命名，且确保应用资质的软件名称与上传的应用名称一致。 |
| 应用名称、图标 | 名称、图标是否与其他APP相同/相似，是否为使用系统默认图标 | 1、请提供名称/图标权属证明或修改您的应用名称/图标。  2、编写代码时，统一替换软件包内的系统默认图标。 |
| 应用功能 | 应用市场是否已有较多类似功能的应用 | 请避免继续在已有较多类似应用的类别下进行开发，如敲木鱼、随机选择、计时类、计算器、手电筒、记事本、记账、天气、数字大小写转换等类别的应用。  您可参考：[应用审核指南 3. 应用功能](https://developer.huawei.com/consumer/cn/doc/app/50104-03) |
| APP备案 | 符合监管部门关于开展APP备案工作的相关要求 | 1、请您及时完成APP备案，确保APP备案时填写的主体信息、应用名称、应用包名与在架应用信息保持一致，并保证APP备案号的有效性；2、鸿蒙版APP备案：需在接入商备案系统填写材料时选择“鸿蒙”平台，并添加鸿蒙包名等信息。如存在多个包名，或同时有HarmonyOS应用和元服务，所有包名均需备案（可添加多个）；3、无需备案类型：单机应用、境外应用。单机应用定义：未通过连接公共互联网提供互联网信息服务的移动应用程序。境外应用定义：由境外主体运营且服务器仅放置在境外的移动互联网应用程序。  您可参考：[APP备案要求](https://www.miit.gov.cn/zwgk/zcjd/art/2023/art_39b4f1acc36745b98478e0ec3e07128d.html)、[APP备案FAQ](/docs/distribute/app-dist/app-market/x50000/x50130) |
| 特殊行业资质 | 应用内容是否包含特殊内容类型，如深度合成或生成式人工智能服务 | 请检索内容类型，参考资质要求准备资质文件。  您可参考：[资质审核要求](https://developer.huawei.com/consumer/cn/doc/distribution/app/80301)、[资质常见问题](https://developer.huawei.com/consumer/cn/doc/50111) |

## 二、开发中

|  |  |  |  |
| --- | --- | --- | --- |
| 阶段 | 模块 | 检查项 | 修改建议 |
| 开发中 | 功能开发 | 功能体验是否完整且无断点，是否具备实用价值且能为用户提供实质功能/服务 | 请确保您的应用能够为用户提供正常完整、稳定流畅、有吸引力的功能。若您的应用已有面向用户提供的其他版本，在开发鸿蒙版本时，请保持不低于其他版本的质量与完善程度，满足用户对体验一致性的期待。  您可参考：[应用审核指南 3. 应用功能](https://developer.huawei.com/consumer/cn/doc/app/50104-03) |
|  | 质量测试 | 测试 App 是否会发生崩溃、是否存在错误 | 为保障用户体验，软件包提前进行云测试、云调试，也可以通过DevEco Studio、DevEco Testing等本地测试工具自检，及早发现应用质量体验等问题。  您可参考：[云测试（HarmonyOS 5及以上）](/docs/distribute/agc/agc-help-cloudtest-0000002235710242/agc-help-cloudtest-introduction-0000002255036400)、[云测试（HarmonyOS 3.1/4.0及以下）](https://developer.huawei.com/consumer/cn/doc/AppGallery-connect-Guides/agc-cloudtest-introduction-0000001083002880)、[云调试（HarmonyOS 5及以上）](/docs/distribute/agc/agc-help-clouddebug-0000002235870046/agc-help-clouddebug-introduction-0000002254916514)、[云调试（HarmonyOS 3.1/4.0及以下）](https://developer.huawei.com/consumer/cn/doc/AppGallery-connect-Guides/agc-clouddebug-introduction-0000001057034023)、[DevEco Testing](/docs/dev/testing/deveco-testing) |
| UX设计 | 测试APP的UX设计是否存在影响用户体验，如：信息文本与背景色对比度小于3 | 请使用云测试、DevEco Testing工具自检，根据检测结果进行修改。请确保应用符合设计规范。  您可参考：[云调试（HarmonyOS 5及以上）](/docs/distribute/agc/agc-help-clouddebug-0000002235870046/agc-help-clouddebug-introduction-0000002254916514)、[云调试（HarmonyOS 3.1/4.0及以下）](https://developer.huawei.com/consumer/cn/doc/AppGallery-connect-Guides/agc-clouddebug-introduction-0000001057034023)、[设计入门（HarmonyOS 5及以上）](https://developer.huawei.com/consumer/cn/design/devstart/)、[通用设计基础（HarmonyOS 3.1/4.0及以下）](https://developer.huawei.com/consumer/cn/doc/design-guides-V1/overview-0000001053563071-V1)、[DevEco Testing](/docs/dev/testing/deveco-testing) |
| 隐私合规 | 是否以显著方式提示用户阅读隐私政策 | 请在应用首次启动时通过弹出隐私政策弹窗，或在应用注册、登录界面展示隐私政策等显著方式，提示用户阅读隐私政策。  您可参考： [标准化隐私声明托管服务FAQ](/docs/distribute/agc/agc-help-privacy-policy-0000002316794885/agc-help-privacy-policy-faq-0000002342315628)、[用户协议与隐私政策弹窗](https://developer.huawei.com/consumer/cn/doc/architecture-guides/user_agreement_and_privacy_policy-0000002331953689)、 |
| 隐私合规 | 是否在用户同意隐私政策前获取信息 | 自检APP或集成的SDK是否在用户同意隐私政策后再收集用户个人信息。请在用户同意隐私政策后，再申请获取用户个人信息及权限。  您可参考：[云测试（HarmonyOS 5及以上）](/docs/distribute/agc/agc-help-cloudtest-0000002235710242/agc-help-cloudtest-introduction-0000002255036400)、[云测试（HarmonyOS 3.1/4.0及以下）](https://developer.huawei.com/consumer/cn/doc/AppGallery-connect-Guides/agc-cloudtest-introduction-0000001083002880)、[个人信息保护合规性检测服务介绍](https://developer.huawei.com/consumer/cn/doc/app/FAQ-faq-10) |
| 人工智能生成合成内容标识 | 是否涉及提供深度合成服务 | 如涉及，请确保生成合成内容中已添加显式标识、文件元数据中已添加隐式标识。并按资质要求提供[人工智能生成合成内容标识相关材料](/docs/distribute/app-dist/app-market/x50000/x50111/x50111-10)。  您可参考：[资质审核要求](https://developer.huawei.com/consumer/cn/doc/distribution/app/80301) |

## 三、上架提审

|  |  |  |  |
| --- | --- | --- | --- |
| 阶段 | 模块 | 检查项 | 修改建议 |
| 上架提审 | 上架自检 | 提前检测应用是否存在功耗、性能、兼容性、稳定性、UX、隐私等问题 | 应用提交邀请测试时，可同步提交“上架自检”，提前感知上架可能存在的问题，以便提前修改快速实现应用上架  您可参考：[配置上架自检信息](/docs/distribute/agc/agc-help-apptest-invite-test-0000002258071220/agc-help-apptest-release-testapp-0000002292711385) |
| 可本地化基础信息 | 应用名称、应用图标是否与安装后的一致 | 请确保提交的应用名称、应用图标与应用安装后终端显示的应用名称、应用图标一致。您可参考：[名称图标一致性检测](https://developer.huawei.com/consumer/cn/forum/topic/0214198798960000231?fid=0102104600515103427) |
| 应用分类标签 | 应用分类/主标签与应用实际功能和用途是否相符 | 请选择与应用实际功能和用途相符的标签。  您可参考：[应用分类及标签（HarmonyOS 5及以上）](/docs/distribute/app-dist/app-services/classification-0000002068852289/classify-0000001960172909)、[应用分类（HarmonyOS 3.1/4.0及以下）](/docs/distribute/app-dist/app-market/x50103) |
| 软件包管理 | 软件包是否为正式版本 | 您可以对软件包提前进行云测试、云调试，及早发现问题，提升应用质量。  建议您在提交公开发布版本前，优先通过HarmonyOS应用内部测试和邀请测试功能进行验证，保障用户体验。  您可参考：[邀请测试](https://developer.huawei.com/consumer/cn/doc/app/agc-help-invite-test-0000002270829393)；[内部测试](https://developer.huawei.com/consumer/cn/doc/app/agc-help-internal-test-0000002270709477) |
| 可本地化基础信息 | 应用介绍、应用一句话简介、应用截图和视频是否与应用实际情况是否相符 | 1、请提交与应用实际功能内容相符的应用信息，应用信息包括应用名称、应用图标、应用一句话简介、应用介绍、应用新版本特性、应用截图及应用视频，避免误导用户。  2、请删除“国家级、最高级、最佳”等不符合广告法的相关内容。  3、请删除“test、demo、beta”等表明应用为测试版本/数据的词汇。  您可参考：[审核指南-应用信息章节](https://developer.huawei.com/consumer/cn/doc/app/50104-01) |
| 隐私标签信息录入 | 应用是否涉及个人信息收集 | 请您与软件包开发人员确认，根据应用收集的个人信息项和个人信息使用目的，如实填写隐私标签。  您可参考：[隐私标签服务说明](/docs/distribute/app-dist/app-market/privacy-label) |
| 版权信息 | 应用内容是否涉及包含特殊内容类型，如深度合成或生成式人工智能服务 | 请参考资质要求上传资质文件。  您可参考：[资质审核要求](https://developer.huawei.com/consumer/cn/doc/distribution/app/80301)、[资质常见问题](https://developer.huawei.com/consumer/cn/doc/50111) |
| 备案信息 | 是否已备案，鸿蒙版也需要备案 | （1）请您及时完成APP备案，确保APP备案时填写的主体信息、应用名称、应用包名与在架应用信息保持一致，并保证APP备案号的有效性；2、鸿蒙版APP备案：需在接入商备案系统填写材料时选择“鸿蒙”平台，并添加鸿蒙包名等信息。如存在多个包名，或同时有HarmonyOS应用和元服务，所有包名均需备案（可添加多个）；（3）如果您的应用为单机或您的应用服务器不在中国大陆，请您在AGC页面正确勾选备案信息。  单机应用定义：未通过连接公共互联网提供互联网信息服务的移动应用程序。  境外应用定义：由境外主体运营且服务器仅放置在境外的移动互联网应用程序。  您可参考：[APP核准（APP备案）指引](/docs/distribute/app-dist/app-market/x50000/x50130) |
| 备案信息 | 备案信息是否正确填写 | 如果您在AGC填写的“主体证件号”有误或APP备案的应用包名、应用名称、主体信息与在架信息不一致，均可能导致您的APP备案校验未通过。请注意，在填写“主体证件号”时，要区分“数字5”和“字母S”，“数字1”和“字母I”，“数字0”和“字母O”。  您可参考：[APP核准（APP备案）指引](/docs/distribute/app-dist/app-market/x50000/x50130) |
| 应用审核信息 | 测试账号信息是否正确填写 | 如应用需要登录才可使用，请记得在提审页面->应用审核信息->测试账号处提供测试账号。  请验证该测试账号可正常使用全部功能，无权限、角色、会员、付费等限制，确保审核人员可完成登录、浏览、操作、退出全流程测试。 |
