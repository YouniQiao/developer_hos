# 华为终端三方管家类应用软件质量检测和安全审查标准（试行）

**1. 总则**

第一条：为了给用户提供绿色、健康、高效的手机系统使用环境，同时为了保障移动智能终端应用软件在华为终端设备上具备良好的使用体验，华为统一对第三方移动智能终端管家类应用软件（下简称“三方管家类应用”）的软件质量和安全进行检测和管理。华为会在三方管家类应用上架华为应用市场过程中，依据检测情况和风险严重程度决定是否允许上架华为应用市场。华为也会在终端用户从外部来源进行三方管家类应用下载、安装过程中进行检测和管理，若三方管家类应用未告知其质量和安全是否达标或出现以下质量和安全情况，会进行风险提示、用户免责声明。

**2. 三方管家类应用范围**

第二条：三方管家类应用是指：基于Android（安卓）系统研发的一类应用，用于管理并优化手机系统，为用户提供更快、更省电、更流畅的用户体验，提供的功能包括手机体检、手机清理、手机杀毒、已安装应用的软件管理、骚扰拦截、防扣费、支付保护等系统优化及安全功能。

**3. 对三方管家类应用的质量检测与安全审查标准**

第三条：三方管家类应用必须达到《华为终端质量检测和安全审查标准（试行）》。若某三方管家应用未达标，或未告知是否达标，则华为会在终端用户从外部来源应用下载安装该应用过程中向用户进行风险提示、用户免责声明和华为系统手机管家使用提醒。

3.1 华为终端质量检测和安全审查标准包括但不限于以下项：

| 测试项 | 标准 |
| --- | --- |
| 兼容性测试结果 | 通过 |
| 通知栏广告 | 无 |
| Banner广告 | 无 |
| 积分墙广告 | 无 |
| 风险广告 | 无 |
| 含广告SDK数量 | 无 |
| 病毒木马 | 无 |
| 隐私(Y) | 无 |
| 漏洞(Y) | 无 |

第四条：三方管家类应用必须通过安卓绿色联盟兼容、稳定、功耗、性能及安全检测。若某三方管家应用未达标，或未告知是否达标，则华为会在终端用户从外部来源应用下载安装该应用过程中向用户进行风险提示、用户免责声明和华为系统手机管家使用提醒。

4.1 安卓绿色联盟兼容、稳定、功耗、性能及安全检测包括但不限于以下项：

| 测试项 | 标准 | 测试项 | 标准 |
| --- | --- | --- | --- |
| 稳定性测试结果 | 通过 | 冷启动响应时间（毫秒） | &lt;=2000毫秒 |
| 灭屏时wakelock时间(min) | &lt;=5分钟 | 热启动响应时间（毫秒） | &lt;=500毫秒 |
| 灭屏屏幕占用时间(ms) | =0 | 应用滑动帧率(FPS) | >=55fps |
| 灭屏时搜索WiFi次数(次/小时) | &lt;=60次/小时 | 内存占用(MB) | &lt;=200M |
| 灭屏时使用WiFi时间(min) | &lt;= 5分钟/小时 | 亮屏后台内存占用(MB) | &lt;=150M |
| 灭屏时音频的占用(ms) | =0 | 灭屏内存占用(MB) | &lt;=100M |
| 灭屏对摄像头的占用(ms) | =0 | 后台亮屏CPU占用(%) | &lt;=2% |
| 灭屏时定位传感器使用时间(min) | &lt;=5分钟 | 后台灭屏CPU占用(%) | &lt;=2% |
| 灭屏时动作传感器使用时间(min) | &lt;=5分钟 | 内存泄露(0) | 无 |

第五条：三方管家类应用必须通过管家类应用后台和静态广播质量标准。若某三方管家应用未达标，或未告知是否达标，则华为会在终端用户从外部来源应用下载安装该应用过程中向用户进行风险提示、用户免责声明和华为系统手机管家使用提醒。

5.1 管家类应用后台和静态广播质量标准包括但不限于以下项：

| 测试项 | 标准 | 测试项 | 标准 |
| --- | --- | --- | --- |
| 灭屏时应用后台时的自启动次数(次/小时) | &lt;=20次/小时 | 静态注册开关机事件类广播 | 无 |
| 灭屏时cpu占用(%) | &lt;=2.5 | 静态注册亮灭屏类广播 | 无 |
| 交叉唤醒 | 无 | 静态注册时间事件类广播 | 无 |
| 灭屏唤醒周期 | 无 | 静态注册外设变化事件广播 | 无 |
| 伪前台  1. 透明小像素。  2. 后台持续播放无声音乐或者录音。 | 无 | 静态注册数据变更类广播 | 无 |
| 静态注册网络事件广播 | 无 | 静态注册手机媒体状态类广播 | 无 |

第六条：三方管家类应用不能存在破坏手机系统完整性、明显破坏用户体验的功能和行为，并应该在合理合规的范围做商业推广。若某三方管家应用未达标，或未告知是否达标，则华为会在终端用户从外部来源应用下载安装该应用过程中向用户进行风险提示、用户免责声明和华为系统手机管家使用提醒。三方管家类应用的功能和行为需遵守并符合以下规范：

6.1 禁止误导、欺骗、强迫用户去获取ROOT权限。

6.2 禁止存在破坏手机系统和卸载系统预装软件的功能及行为，同时禁止诱导用户卸载系统预装软件。

6.3 禁止存在会频繁引起无法OTA升级、黑屏、卡死、重启等系统风险的功能，更严禁在明知道会引起系统风险的情况下依旧引导用户使用某些带风险或不完备的管家功能。

6.4 禁止存在会引起后台运行高耗电的常驻通知栏。

6.5 禁止存在会引起后台运行高耗电的常驻悬浮框。

6.6 禁止存在在用户不知情情况下或误导、欺骗、强迫用户开启劫持锁屏功能的行为。

6.7 禁止获取与管家核心功能不需要并会引起后台运行高耗电的重度高危权限，也禁止误导、欺骗、强迫用户开启高危权限。

6.8 禁止存在在用户不知情情况下或误导、欺骗、强迫用户在桌面创建桌面快捷方式的行为。

6.9 禁止存在安全功能不完善并容易引发信息泄露的功能。

6.10 禁止不经过用户同意或误导、欺骗、强迫用户上传用户数据。上传后的用户数据不能恶意传播，泄露用户隐私。

6.11 需标识和区分出广告和下载链接，需要在合理、合规、合情的情况下进行商业推广，需遵守以下规范和标准：

6.11.1 禁止误导消费者的行为，包括在三方管家类应用内部将商业推广和广告伪装成应用功能等。

6.11.2 禁止违法发布广告，包括广告无识别性，导致消费者无法辨明的行为等。

6.11.3 禁止通过利益诱惑欺骗用户点击、下载的商业化行为，包括但不限于：现金奖励、实物奖品、虚拟奖品（Q币、话费、流量、红包、免息贷、免息红包、优惠券、代金券、积分等）、集赞、拼团、下载可增加抽奖机会、下载可增加抽奖概率，以积分或金钱利益欺骗用户点击、下载的商业化行为； 通过前述方式诱导、欺骗用户修改、关闭、卸载其他经营者合法提供的网络产品或者服务。

6.11.4 禁止通过胁迫、恐吓促使用户点击和下载；禁止在广告宣传中通过虚假或者引人误解的内容欺骗、误导消费者；禁止通过“国家级”、“最高级”、“最佳”等用语进行广告宣传；禁止通过违背社会良好风尚的方式进行广告宣传。 例如：“最权威”、“大家都会装”等。
