# 华为钱包Wallet服务使用协议

点击下载[华为钱包Wallet服务使用协议](https://terms-drcn.platform.hicloud.com/agreementservice/developer/getAgreementTemplate?agrType=1024&country=cn&language=zh_cn&version=2020062901)。

最后更新日期：2020年6月29日

以下条款（以下简称“本协议”）是您与华为订立的关于“华为钱包 Wallet 服务”的具有法律约束效力的协议。本协议是[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)的补充协议，而[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)则是[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)的补充协议。本协议约定了您在使用华为钱包 Wallet 服务

时所享有的法律权利和应尽的法律义务。您通过华为AppGallery Connect页面弹出的对话框选择同意所有适用于AppGallery Connect服务的协议（包括本协议），或点击对应本协议的“我已阅读并同意”按钮，或使用本协议项下的华为钱包 Wallet 服务，即视为自您接受、使用之日（“生效日期”）起对本协议条款、[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)的充分理解和完全接受。本协议未尽事宜，按照[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)执行。如果您不同意本协议、[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)或[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)，您应当停止使用各个协议项下的服务。

## **1、定义**

除非本协议另有规定，本协议中的术语与[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)中的术语含义相同。如无特别说明，下列术语在本协议中的含义为：

“华为钱包 Wallet 服务” （又称 “Wallet服务”）：是由华为向用户提供的管理卡片和票券的服务，该服务允许开发者在华为的设备上集成卡券票证类的产品和/或服务：包括但不限于会员卡，一卡通，交通卡，身份证, 门票，电影票，登机牌，优惠券，折扣券，车钥匙，门钥匙，具体以本网站公布为准。

“华为Wallet Kit”（简称“Wallet Kit”）：指与华为向开发者提供的Wallet服务相关的API/SDK/Kit。

## **2、合作内容**

2.1 华为向开发者提供 Wallet Kit ，开发者将Wallet Kit 集成到开发者的产品中，以让用户在开发者产品中使用华为钱包 Wallet 的便捷服务。（以下简称“本服务”）

2.2 开发者可以通过集成WalletKit，来允许其用户在没有“华为钱包Wallet服务”的情况下，仍然可以使用开发者提供的卡券票证相关服务。

## **4、隐私**

当您集成WalletKit后，您的用户便可以使用“Wallet服务”，您了解华为会作为一个独立的数据控制者向用户提供对其卡券票证的访问和管理。

在不影响[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)“隐私保护”条款前提下，当您集成了WalletKit后，如果您的用户未使用“Wallet服务”，华为会按照[《AppGallery Connect数据处理附录》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20215)当中的要求来处理您的数据。
