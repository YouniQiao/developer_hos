# 华为机器学习服务使用协议

点击下载[华为机器学习服务使用协议](https://terms-drcn.platform.dbankcloud.cn/agreementservice/developer/getAgreementTemplate?agrType=1096&country=cn&language=zh_cn&version=2022090201)。

以下条款（以下简称“本协议”）是您与华为订立的关于“华为机器学习服务”的具有法律约束效力的协议。本协议是[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)的补充协议，《华为 APIs 使用协议》则是[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)的补充协议。本协议约定了您在使用华为机器学习服务时所享有的法律权利和应尽的法律义务。您通过华为AppGallery Connect页面弹出的对话框选择同意所有适用于AppGallery Connect服务的协议（包括本协议），或点击对应本协议的“我已阅读并同意”按钮，或使用本协议项下的华为机器学习服务即视为自您接受、使用之日（“生效日期”） 起对本协议条款、[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)和[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)的充分理解和完全接受。本协议未尽事宜，按照《华为开发者服务协议》和《华为 APIs 使用协议》执行。如果您不同意本协议、《华为 APIs 使用协议》或《华为开发者服务协议》，您应当停止使用各个协议项下的服务。

## **定义**

除非本协议另有规定，本协议中的术语与[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)和[《华为 APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)中的术语含义相同。如无特别说明，下列术语在本协议中的含义为：

1.1 **“ML Kit”**或**“华为机器学习服务”**（以下简称**“本服务”**）：指华为向开发者提供的，为开发者应用机器学习能力开发各类应用的机器学习套件。

1.2 **“语音数据”**：指通过机器学习服务的实时语音识别服务、音频文件转写服务获得的数据。

1.3 **“SDK”**：指华为机器学习服务提供的应用开发工具基础包。

1.4 **“Full SDK”**：指除了基础包外，同时包含算法能力或功能插件的应用开发工具包。

1.5 **“API”**:指华为机器学习服务提供的应用程序开放接口。

3.1 在向您提供本服务时，华为将作为您的数据处理方，根据[《AppGallery Connect 数据处理附录》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20215)（即《华为机器学习服务使用协议》和《华为开发者服务协议》中的一部分）来处理最终用户的个人数据。

3.2 最终用户数据隐私要求

为了保护最终用户上传数据的相关隐私，并考虑到《华为 APIs 使用协议》条款6 隐私保护，您应确保您的应用（i）提前通知了最终用户您从最终用户或最终用户设备收集的文本、图片、语音等数据类型，以及（ii）除非获得最终用户事先明确的可撤销的同意，否则不会获得或缓存最终用户的任何数据。

4.3 若华为决定不再提供本服务或者对本服务做重大变更，则华为有权提前向您发出通知后终止本协议。

4.4 协议双方签订的[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)和[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)不会随着本协议终止而自动终止。
