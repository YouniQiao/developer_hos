## **1. 定义**

除非本协议另有规定，本协议中的术语与底层协议中的术语意义相同。下列术语在本协议中的含义为：

1.1 华为主题：指可供最终用户浏览、下载、使用“作品”（定义见下）的一个或多个由华为提供和/或运营的平台，形式包括但不限于电脑网站和应用程序。

1.2 华为主题服务平台：指在华为的开发者联盟网站“主题中心”入口为开发者/发布者提供上传作品和管理作品的服务平台。

1.3 作品：指发布者在“华为主题服务平台”上发布或更新的任何信息或材料，包括但不限于主题、字体、壁纸、动态壁纸、表盘、音乐、文本及其它信息。作品为“付费产品”（定义见[**《华为开发者商户服务协议》**](https://developer.huawei.com/consumer/cn/doc/start/merchantserviceagreement-0000001052848245)）时，发布者可为作品定价。

1.4 发布者：指有效申请并经华为审核同意后，将自主创作作品、版权方作品或者以版权方提供的内容素材为基础制作作品发布在“华为主题服务平台”上的“开发者”，包括机构及个人。

1.5 版权方：指拥有作品部分或全部素材的知识产权或合法授权的“开发者”。

1.6 授权有效期：指版权方或发布者将作品通过华为主题服务平台授权给华为分发、推广其作品的期限。

1.7合作地域范围：中华人民共和国大陆地区（不包括香港和澳门特别行政区、台湾地区）。

1.8 授权范围：发布者及版权方授权华为及其关联公司在合作地域范围内、授权有效期期间（1）通过华为主题服务平台将作品分发至华为主题或其他华为运营平台或合作平台，供最终用户使用、下载；（2）在各种线上或线下的营销渠道，包括但不限于社交媒体平台、营销资料、海报、横幅、发布会、见面会等营销场景或渠道，为营销作品或华为主题或华为主题服务平台的目的，展示、传播作品的全部或其任何部分、发布者及版权方的名称、品牌、商标等标识。如需变更授权范围的，发布者和/或版权方应及时告知华为。

1.9 定价：发布者在“华为主题服务平台”上发布作品，并基于华为在其网站/开发者联盟平台发布的针对产品销售所制定的价格段对作品进行自主定价，最终定价以“华为主题”上显示的价格（含增值税）为准。

1.10 收入分成总额(RSRA)：遵照[**《华为开发者商户服务协议》**](https://developer.huawei.com/consumer/cn/doc/start/merchantserviceagreement-0000001052848245)4.1条描述。

1.11 扣除项：遵照[**《华为开发者商户服务协议》**](https://developer.huawei.com/consumer/cn/doc/start/merchantserviceagreement-0000001052848245)4.1条描述。

1.12 会员：购买华为主题会员套餐且在有效期内的最终用户。

1.13 非会员：未购买华为主题会员套餐或会员套餐已过期的最终用户。

## **2. 合作方式**

2.1 发布者在华为主题服务平台上传和管理作品。

2.2 华为及其关联公司根据本协议约定的授权范围分发和推广其作品，供最终用户通过华为主题或华为运营平台或合作平台下载和使用。发布者上传作品时需如实设置授权有效期。

3.1.2 版权方保证授予发布者制作、上传作品至华为主题服务平台所需的所有权利。

3.1.3 版权方同意华为最终用户可通过华为主题或华为运营平台或合作平台下载或使用作品，且最终用户一经购买可以永久使用。

3.1.4 如需终止授权，版权方需自行联系发布者下架作品，华为不承担责任。

3.1.5 如作品因采用版权方提供的素材内容涉嫌侵权或违法违规时，华为有权优先下架该作品，版权方有义务配合华为或发布者进行查证；若第三方因此要求侵权赔偿，版权方需承担侵权责任以及相关赔偿。若第三方通过诉讼、行政执法或其他法律途径直接向华为主张责任导致华为遭受损失的，华为有权向版权方追偿。

3.2 发布者的权利与义务

3.2.1 发布者保证上传的作品均系原创或已取得合法授权，保证作品内容不侵犯任何第三方的合法权益（包括但不限于知识产权、肖像权、名誉权），不违反任何适用的法律法规以及国家政策，尊重用户隐私，不存在盗取最终用户信息或盗取、破坏华为主题服务平台的隐藏内容。发布者保证不得干扰华为主题服务平台任何部分或功能的正常运行，或是同其构成竞争行为。

3.2.2 发布者保证对作品质量负责，在华为主题服务平台发布的作品不存在明显的缺陷，并应对已发布的作品进行及时的更新维护，保证作品的稳定运行。发布者应及时处理最终用户反馈的问题。如因作品本身造成与最终用户的任何纠纷，由发布者出面解决并承担责任。

3.2.3 如发布者发布的作品涉嫌侵权或违法违规或者违反华为协议约定时，华为有权优先下架该作品，发布者有义务配合华为进行查证。若第三方要求侵权赔偿，发布者需承担侵权责任以及相关赔偿。若第三方通过诉讼、行政执法或其他法律途径直接向华为主张责任导致华为遭受损失的，华为有权向发布者追偿。

3.2.4 如版权方与发布者之间约定的授权期终止，发布者应及时下架相关的作品。未经版权方许可，发布者不得擅自使用。如因发布者违反其与版权方的约定而造成版权方利益受损的，华为不承担任何责任。

3.3.1 华为有权就发布者在华为主题服务平台发布的作品按照本协议授权范围进行复制、传播和推广，供最终用户下载和使用；华为有权根据其需要对已发布作品的位置进行调整或删除作品。

## **4. 涉及人工智能生成合成内容（以下简称“AI生成内容”）的声明义务**

4.1 如发布者上传、发布或传播 AI 生成内容时，应通过平台指定入口提交《AI 素材声明》，声明内容至少包含：

（1）内容由 AI 工具生成的明确说明；

（2）AI 生成工具的名称及版本信息；

（3）是否经过人工修改及具体修改范围；

（4）内容生成的时间及用途说明。

发布者保证所提交的声明及标识信息真实准确，因虚假声明或标识不当引发的法律责任由发布者独立承担，包括但不限于第三方索赔、行政机关处罚等。

4.2 发布者保证不得出现以下任一情形：

（1）未按规定添加显式标识或虚假声明非 AI 生成内容；

（2）恶意删除、篡改、伪造或隐匿包含原始AI 生成内容平台携带的显式 / 隐式标识；

（3）为他人实施上述行为提供工具或技术支持；

（4）通过不正当标识手段误导公众或损害他人权益。

4.3 华为将对发布者上传内容进行以下核验，并根据核验结果对内容进行相应标注：

（1）元数据隐式标识检测：对含隐式标识的内容，在发布页面添加 "内容属于AI 生成"显性标识提示；

（2）发布者声明核验：对发布者主动声明为 AI 生成但无隐式标识的内容，标注 "内容可能AI 生成" 显性标识提示；

（3）技术特征识别：对未声明但平台检测到 AI 生成痕迹的内容，标注"内容疑似AI 生成"显性标识提示。

4.4 华为仅对发布者标识或声明进行形式审核，不保证内容真实性。因发布者违反本条（涉及人工智能生成合成内容的声明义务）约定导致华为遭受损失的，华为有权向发布者全额追偿。

## **5. 定价与结算**

5.1 两方结算模式

两方结算是指发布者上传自有版权内容作品或者有转授权内容作品（在华为主题服务平台只体现发布者作为结算分成主体）,华为将主题收入结算给发布者。

5.1.1 两方结算模式及分成比例：

|  |  |  |
| --- | --- | --- |
|  | **收入分成总额（RSRA）** | **分成比例（华为：发布者）** |
| **会员** | （最终用户支付的价格-扣除项）\*（1-运营成本费率）\*（发布者作品的会员有效使用价值/所有作品的会员有效使用总价值）\*收入确认系数 | 30%：70% |
| **非会员** | （最终用户支付的价格-扣除项）\*（1-运营成本费率） |

收入确认系数：按照支付日起至月末的权益天数占整个订购会员天数的比例计算，例如：最终用户A在1月19日订购3个月会员（1月19日到4月21日），则1月份的收入确认系数为13/93=0.14，2月份的收入确认系数为28/93=0.30，3月份的收入确认系数为31/93=0.33，4月份的收入确认系数为21/93=0.23

扣除项：按《华为开发者商户服务协议》第4.1条确定，但不包括渠道费用

运营成本费率：包含渠道及其他运营成本的综合费率。

5.1.2 当发布者与华为有特殊约定的情况下，双方可另行约定调整在华为开发者联盟页面上的分成比例。

5.2 三方结算模式：

三方模式指发布者上传版权方内容，且在上传时将版权方纳入结算分成主体之一，华为将主题收入分别结算给发布者和版权方。

5.2.1 三方结算分成比例：

|  |  |  |
| --- | --- | --- |
|  | **收入分成总额（RSRA）** | **分成比例（华为：发布者：版权方）** |
| **会员** | （最终用户支付的价格-扣除项）\*（1-运营成本费率）\*（发布者作品的会员有效使用价值/所有作品的会员有效使用总价值）\*收入确认系数 | 30%：40%：30%或者  30%：30%：40%或者  30%：20%：50%或者  30%：50%：20%；  以发布者上传作品时勾选的比例为准。 |
| **非会员** | （最终用户支付的价格-扣除项）\*（1-运营成本费率） |

收入确认系数：按照支付日起至月末的权益天数占整个订购会员天数的比例计算，例如：最终用户A在1月19日订购3个月会员（1月19日到4月21日），则1月份的收入确认系数为13/93=0.14，2月份的收入确认系数为28/93=0.30，3月份的收入确认系数为31/93=0.33，4月份的收入确认系数为21/93=0.23

扣除项：按《华为开发者商户服务协议》第4.1条确定，但不包括渠道费用。

运营成本费率：包含渠道及其他运营成本的综合费率。

5.3 本协议第4条定价结算未做明确约定的，各方遵照[**《华为开发者商户服务协议》**](https://developer.huawei.com/consumer/cn/doc/start/merchantserviceagreement-0000001052848245)执行。
