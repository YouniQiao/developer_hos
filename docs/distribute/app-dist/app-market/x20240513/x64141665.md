# 华为Nearby服务使用协议

点击下载[华为Nearby服务使用协议](https://terms-dre.platform.hicloud.com/agreementservice/developer/getAgreementTemplate?agrType=1077&country=cn&language=zh_cn&version=2020062401)。

以下条款（以下简称“本协议”）是您与华为订立的关于“华为Nearby服务”的具有法律约束效力的协议。本协议是[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/doc/20209)的补充协议。而[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/20209)则是[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)的补充协议。本协议约定了您在使用“华为Nearby服务”时所享有的法律权利和应尽的法律义务。您通过华为AppGallery Connect页面弹出对话框选择同意所有适用于AppGallery Connect服务的协议（包括本协议），或点击对应本协议的“我已阅读并同意”按钮，或使用本协议项下的华为Nearby服务，即视为自您接受、使用之日（“生效日期”）起对本协议条款、[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/doc/20209)的充分理解和完全接受。本协议未尽事宜，按照[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/doc/20209)执行。如果您不同意本协议、[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/20209)或[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)，您应当停止使用各个协议项下的服务。

## **1. 定义**

除非本协议另有规定，本协议中的术语与[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/20209)中的术语含义相同。如无特别说明，下列术语在本协议中的含义为：

1.1 “华为Nearby服务”（以下简称“Nearby服务”）：指华为通过Nearby Kit和Nearby平台向开发者提供的用于开发者应用程序提供设备间发布、广播、和查询消息等近距离通信服务。

1.2 “Nearby Kit”：指华为提供的、向开发者提供Nearby服务的相关API/SDK/Kit等。

1.3 “Nearby平台”：指华为提供的、用以使用和管理Nearby服务的相关华为网站。

1.4 独立SDK：指无需安装HMS Core APK，仅集成该SDK就可以使用Nearby服务.

## **2. 权利和义务**

### **2.1您的权利和义务**

(1) 您可使用Nearby Kit和Nearby平台将消息发布给您附近的设备，您仅可以在您的应用中使用Nearby服务。华为给您的应用分配了应用程序 APP ID、APP Secret，用于标识应用，您同意不与任何第三方共享您的应用程序 APP ID、APP Secret。

(2) 您需要用户在使用Nearby服务前应对该服务已经开启有合理预期。您应当在应用程序中结合必要的提示、交互，如要求用户执行明确的操作（如点击按钮或其他特殊的开关或进入应用程序的某个区域等）以激活Nearby服务。您保证不要在用户不知情的情况下在后台发布或订阅。

（3）您应当帮助用户了解您的应用将向附近的用户公开哪些数据。

（4）使用户易于停止消息的发布和订阅。

(5) 您不应使用Nearby服务发送包含任何淫秽、色情、令人反感的、诽谤内容或其他违反法律法规的材料（包含文字、图形、图像、照片和声音等）。

(6) 您承诺，不使用Nearby服务发送“钓鱼”消息和垃圾信息，包括但不限于含有仿冒网站或应用来欺诈用户、骗取用户私人资料的消息。

(7) 您理解并同意，发布消息或任何由您基于Nearby服务而开发、运营的服务，是由您独立控制、管理并发送的，华为仅为您提供基础能力，未向消费者提供服务。对于您通过使用Nearby服务而传输的任何数据或内容，华为在法律允许范围内不承担任何责任。

(8) 您承诺您向用户发送的内容不含敏感个人数据和用户的保密信息。如您向用户发送的内容包含任何个人信息，您承诺您对个人信息的收集、使用、传输和处理符合适用的数据保护法。

(9) 您需要遵守适用的数据保护法律法规，包括但不限于对个人数据的收集和处理、识别用户身份以及履行数据主体权力。您了解上述法律可能会要求您识别您的用户身份。

(10) 您应向华为提供必要的您的产品的使用场景，以使华为能够自行决定为您的产品提供针对性的解决方案，以符合您产品的市场定位。

(11) Nearby服务并非在所有国家/地区可用，而且不承诺可用于任何特定位置。您应自主地选择使用Nearby服务的范围，并且负责遵守任何适用的法律，包括但不限于任何当地法律。

(12) 您在使用独立（Standalone）SDK过程中，华为会对API的调用情况进行必要的数据收集，用于统计分析及服务改进。收集的数据不包含任何个人数据，包括：开发者应用包名、开发者应用ID、开发者应用版本号、HMS core版本号、Nearby SDK版本号、Nearby服务名称、API名称、接口调用结果、接口响应时延、接口调用时间、事务ID、设备型号、国家码、客户端EMUI版本号、消息发布结果、消息发布时延、消息订阅结果、消息订阅时延、最大消息ID、消息TTL、beacon类型、设备发现时延、连接结果、连接时延、数据传输结果、数据传输大小、数据传输时延、信道类型、连接策略、广播策略等。您可以在您应用的隐私政策中告知用户。

为了在最终用户的设备和服务器之间建立连接用以上报数据，我们收集了最终用户的IP地址。建立连接后，将立即删除IP地址。

## **3. 隐私**

3.1在向您提供Nearby服务时，华为将作为您的数据处理方，根据[《AppGallery Connect 数据处理附录》](https://developer.huawei.com/consumer/cn/devservice/doc/20215)（即《华为Nearby服务使用协议》和[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)中的一部分）来处理用户的个人数据。

3.2为了保护最终用户上传数据的相关隐私，并考虑到《华为APIs使用协议》“隐私保护”条款，您应确保您的应用（i）提前通知了最终用户您从最终用户或最终用户设备收集的文本、图片等数据类型，以及（ii）除非获得最终用户事先明确的可撤销的同意，否则不会获得或缓存最终用户的任何数据。

3.3 尽管有第3.1条的规定，如果您使用服务帐号功能并为您自己的客户分配服务帐号密钥，使他们能够使用Nearby服务的某些功能，则应适用以下规定：

3.3.1 您的客户通过服务帐号密钥使用Nearby服务时，您的客户（以下称为“服务帐号用户”）都应作为服务帐号上下文中处理的个人数据以及通过服务帐号注册的信标数据的数据控制者。

3.3.2 您应作为服务帐号上下文中处理的个人数据以及通过服务帐号注册的信标数据的数据处理者，并按照作为数据控制者的服务帐号用户的指示处理数据。

3.3.3 华为将作为您的服务帐号上下文中处理的个人数据以及通过服务帐号注册的信标数据的子处理者。

3.4 当您和/或服务帐号用户使用服务帐号功能时，[《AppGallery Connect数据处理附录》](https://developer.huawei.com/consumer/cn/doc/10130)中的条款适用于华为对服务帐号用户提交、上传、查询、请求，指导您或华为在Nearby服务上下文中进行的收集。但AppGallery Connect数据处理附录第3.1.1条不适用于服务帐号功能背景下的个人数据处理。若本协议第3.3至3.4条与AppGallery Connect数据处理附录发生冲突，则与服务帐号功能相关的个人数据处理以本协议为准。

3.5您应自行负责确保服务帐号功能中的个人数据处理符合所有适用的数据保护法律。

4.1 任何一方终止[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)或[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/20209)后，本协议将在同一终止日期自行终止。

4.2 您可通过关闭您所有应用的华为Nearby服务使用权限来终止本协议。

4.3 若华为决定不再提供Nearby服务或者对服务做重大变更，则华为有权提前向您发出通知后终止本协议。

4.4 协议双方签订的[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/doc/20209)不会随着本协议终止而自动终止。
