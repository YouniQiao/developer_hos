# 华为动态标签管理服务协议

点击下载[华为动态标签管理服务协议](https://terms-dra.platform.hicloud.com/agreementservice/developer/getAgreementTemplate?agrType=1109&country=cn&language=zh_cn&version=2021073101)。

以下条款（以下简称“本协议”）是您与华为订立的关于“华为动态标签管理服务”的具有法律约束效力的协议。本协议是[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)的补充协议，而[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)则是[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)的补充协议。本协议约定了您在使用华为动态标签管理服务时所享有的法律权利和应尽的法律义务。您通过华为AppGallery Connect页面弹出的对话框选择同意所有适用于AppGallery Connect服务的协议（包括本协议），或点击对应本协议的“我已阅读并同意”按钮，或使用本协议项下的华为动态标签管理服务，即视为自您接受、使用之日（“生效日期”）起对本协议条款、[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)的充分理解和完全接受。本协议未尽事宜，按照[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)执行。如果您不同意本协议、[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)或[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)，您应当停止使用各个协议项下的服务。

**1.定义**

除非本协议另有规定，本协议中的术语与[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)中的术语含义相同。如无特别说明，下列术语在本协议中的含义为：

1.1 “华为动态标签管理服务”（以下简称“DTM服务”）指华为通过DTM Kit和DTM服务器向开发者提供的标签管理服务。DTM服务帮助开发者基于Web界面轻松部署和更新测量代码，并向第三方分析平台报告数据。

1.2 “DTM Kit”指华为向开发者提供的DTM服务所含的API/SDK/Kit。

1.3 “DTM服务器”指用于配置和管理华为提供的DTM服务的网站。

1.4 “标签”指您或代表您的第三方在DTM服务器上配置的代码片段。

**2.权利和义务**

2.1 您的权利和义务

（1）您可以使用DTM服务提供的API、SDK以及网站来管理您的应用标签，但您仅可将DTM服务用于您开发的应用。

（2）您承认并同意，您不会将DTM服务用于任何非法行为、侵犯他人权利的行为、诽谤行为，或任何其他有害的行为。

（3）您承认并同意，标签由您而非华为全权控制、设置及管理。您应负责遵守与使用DTM服务以及通过DTM服务传输的任何数据或内容相关的适用法律要求和义务。

（4）如果您收集的数据包含任何个人数据，您保证您对此类信息的收集、使用、传输及处理符合一切适用的数据保护法律法规以及与使用Cookies和类似技术相关的法律法规。

（5）您承诺您不会攻击DTM服务网络及服务器，或以任何方式干扰其他开发者使用DTM服务。

（6）DTM服务并未在所有国家/地区提供，且华为并不保证您一定可以在某一特定地点使用该服务。您应独立决定DTM服务的使用范围，并负责遵守一切适用的法律法规，包括但不限于本地法律等。

2.2 华为的权利和义务

（1）华为有权拒绝或限制您对DTM的访问和使用。

（2）华为有权自行决定更新DTM服务的API和SDK。我们遵循软件设计原则尽可能保证兼容性。但是，在某些情况下，您可能需要修改或升级您的应用才可正常使用更新以后的API和SDK。您需要自行承担前述修改或升级应用产生的费用。

（3）如果DTM服务需要暂停以进行系统维护或API、SDK、网站更新，华为将尽可能通过其网站提前通知您。华为不承诺不间断地、无延迟地提供DTM服务，华为不承担因系统维护以及API、SDK和网站更新给您造成的损失。

**3.隐私**

您承诺，您将根据适用的法律法规以及您和华为签订的协议使用DTM服务和处理与DTM服务相关的用户个人数据。

3.1 数据处理角色

当华为向您提供DTM服务时，华为作为您的数据处理者处理用户个人数据。华为根据[《AppGallery Connect数据处理附录》](https://developer.huawei.com/consumer/cn/doc/20215)处理用户个人数据。[《AppGallery Connect数据处理附录》](https://developer.huawei.com/consumer/cn/doc/20215)是本协议以及[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)不可分割的一部分。

3.2 用户同意

（1）为了使用DTM服务，您必须明确指出可能由于您使用DTM服务而收集、接收和/或使用最终用户个人数据的各方。您必须向最终用户披露此类当事方，并将当事方收集的个人数据及其使用目的告知最终用户。您必须获得并记录最终用户的有效同意，例如最终用户对使用Cookies或类似技术（如果适用）的同意。您必须记录此类同意，并保证最终用户有能力撤销其同意。

（2）如果您不同意前述约定，我们可能会限制或暂停您对DTM服务的使用。

3.3 限制

您不得向DTM服务器上传任何可用于识别用户的信息（例如姓名、email地址、设备识别符或发票等）。

3.4 用户体验提升项目

华为可能会出于提升和维护DTM服务的目的而收集和处理您使用DTM服务及使用频率的信息。未经您的同意，华为不会与任何第三方分享此类信息，除非第三方按照合约代表华为行事。

**4.协议终止**

4.1任何一方终止[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)或[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)后，本协议将在同一终止日期自行终止。

4.2您可通过关闭您所有应用的DTM服务使用权限来终止本协议。

4.3若华为决定不再提供DTM服务或者对服务做重大变更，则华为有权提前向您发出通知后终止本协议。

4.4协议双方签订的[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)不会随着本协议终止而自动终止。

**5.分发区域和签约主体**

您承认并同意，本协议项下分发区域和签约主体按照[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)“签约主体”条款执行。

**6.法律适用和争议解决**

您承认并同意，本协议项下法律适用和争议解决按照[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)“法律适用和争议解决”条款执行。
