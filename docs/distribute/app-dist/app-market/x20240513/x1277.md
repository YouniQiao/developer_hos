# 华为视频编辑服务使用协议

以下条款（以下简称“本协议”）是您与华为订立的关于“华为视频编辑服务”的具有法律约束效力的协议。本协议是[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20209)的补充协议。而[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20209)则是[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20201)的补充协议。本协议约定了您在使用华为视频编辑服务时所享有的法律权利和应尽的法律义务。点击本协议下方的“我已阅读并同意”按钮（或类似按钮）或使用本协议项下的华为视频编辑服务，即视为自您接受、使用之日（“生效日期”） 起对本协议条款、[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20201)和[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20209)的充分理解和完全接受。

本协议未尽事宜，按照[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20201)和[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20209)执行。如果您不同意本协议、[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20209)或[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20201)，您应当停止使用本协议下的服务。

## **1.** **定义**

除非本协议另有规定，本协议中的术语与[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20201)和[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20209)中的术语含义相同。如无特别说明，下列术语在本协议中的含义为：

1.1华为视频编辑服务（以下简称“本服务”）：指华为提供的视频导入、编辑、导出、媒体资源管理等能力，开发者可集成构建应用的视频编辑能力和上传、管理应用内的素材。

1.2 您的素材：指您上传至[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html#/myProject/387511071394504218/9322780611179157231?appId=100330247)网站视频编辑服务栏目内的所有素材，包括但不限于图片、视频、贴纸、模板、文本等。

1.3 公共素材：指华为向集成视频编辑服务的开发者提供的、预置在视频编辑服务SDK内的素材，包括但不限于贴纸、特效、滤镜、模板、文本等。为免疑义，公共素材为《华为开发者服务协议》定义的“华为内容”。

3.1 当华为向您提供本服务时，华为将作为您的数据处理方，根据[《AppGallery Connect 数据处理附录》](https://developer.huawei.com/consumer/cn/devservice/doc/20215)（即《华为视频编辑服务使用协议》和[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20201)中的一部分）来处理最终用户的个人数据。

3.2 最终用户数据隐私要求

为了保护最终用户上传数据的相关隐私，并考虑到[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20209)条款6 隐私保护，您应确保您的应用（i）提前通知了最终用户您从最终用户或最终用户设备收集的数据类型，以及（ii）除非获得最终用户事先明确的可撤销的同意，否则不会获得或缓存最终用户的任何数据。

4.4 协议双方签订的[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20201)和[《华为 APIs 使用协议》](https://developer.huawei.com/consumer/cn/devservice/doc/20209)不会随着本协议终止而自动终止。
