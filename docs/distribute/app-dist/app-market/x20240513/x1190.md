# 华为国内短信服务使用协议

点击下载[华为国内短信服务使用协议](https://terms1.hicloud.com/agreementservice/developer/getAgreementTemplate?agrType=1190&country=cn&language=zh_cn&version=2021032202)

以下条款（以下简称“本协议”）是您与华为订立的关于“华为国内短信服务”的具有法律约束效力的协议。本协议是[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/20209)的补充协议，而[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/20209)则是[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)的补充协议。本协议与前述协议共同约定了您在使用华为国内短信服务时所享有的法律权利和应尽的法律义务。您通过华为AppGallery Connect页面弹出的对话框选择同意所有适用于AppGallery Connect服务的协议（包括本协议），或点击对应本协议的“我已阅读并同意”按钮，或使用本协议项下的华为国内短信服务，即视为自您接受、使用之日（“生效日期”）起对本协议条款、[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)的充分理解和完全接受。如果您不同意本协议、[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)或[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)，您应当停止使用相应协议项下的服务。

本协议未尽事宜，按照[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)执行。如本协议与[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)中的条款发生冲突的，仅在华为短信服务的范围内，以本协议为准。

## **定义**

除非本协议另有规定，本协议中的术语与[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)中的术语含义相同。如无特别说明，下列术语在本协议中的含义为：

1.1 **“华为国内短信服务”**（以下简称**“本服务”**）：指华为向开发者提供的一种通信服务能力，开发者可以使用本服务发送短信给拥有中国大陆地区手机号码的最终用户。

## **权利和义务**

2.1 您的权利和义务

2.1.1 您可使用本服务的API将短信发送到您的应用的最终用户。您仅可以在您的应用中使用本服务。

2.1.2 您通过本服务向最终用户发送任何短信之前，必须获取最终用户对接收此类短信的同意。如果最终用户拒绝接收或撤回对接收此类短信的同意，您不可以向该最终用户发送短信。

2.1.3 您通过本服务向最终用户发送的信息应包含您的企业标识和/或名称，明确告知最终用户信息来源和相关业务提供方。

2.1.4 您不能使用华为提供的短信通道提供违反法律、行政法规和电信管理机构相关规定的业务。您向最终用户发送的信息不得包含违法违规、侵犯第三方合法权益或违反社会公序良俗的内容或业务，包括但不限于信息中不得包含政治敏感词及法律法规规定的禁止性内容、不得包含侵犯第三方知识产权的业务或内容等。

2.1.5 您在使用本服务期间应加强对短消息内容的核查，并采取有效技术措施，防止黑客攻击、盗用密码、非法调用页面等危害短信业务信息安全的非法行为。

2.1.6 您应快速响应、积极配合华为处理关于您发送信息的投诉。若华为收到关于您发送信息的投诉并通知您，您必须在收到通知后2小时内给予华为初步回复，并在6个工作小时内查明原因，按华为规定的格式给予证明回复，停止违法违规信息的传播。

2.2 华为的权利和义务

2.2.1 华为提供短信资源、统一接口服务、通道安全保障服务、内容安全审核管理服务等服务内容。

2.2.2华为协助您在运营商完成备案等工作。

2.2.3华为有权审查您的发送内容及其链接。对不符合法律法规及华为规定的内容和表现形式，华为有权要求您修改或者拒绝发送。华为的内容安全审核服务不免除您保证您的业务活动合法合规的任何义务。

2.2.4 华为负责国内短信产品的技术支持和维护，以保障国内短信产品的稳定和畅通。

2.2.5 本服务按发送成功条数计费，并从您的付费服务账户余额中扣减应付金额。若因账户余额不足导致扣缴失败，华为有权暂时停止您对该服务的使用，并享有在您账户有余额时补扣及应收款追索的权利。您在使用本服务过程中所涉及的计费、付费事宜，还应遵从《[AppGallery Connect付费服务协议](https://developer.huawei.com/consumer/cn/doc/distribution/app/PaidService)》。

2.2.6 华为有权更新本服务的API。华为遵循软件设计原则尽可能保证兼容，但在特殊场景下，可能会需要修改或升级您的应用后才能使用更新后的API，您需要自行承担修改和升级应用产生的成本。

2.2.7 华为可能收集有关您使用本服务的诊断信息，以帮助改进本服务。

## **隐私**

在向您提供本服务时，华为将作为您的数据处理方，根据[《AppGallery Connect数据处理附录》](https://developer.huawei.com/consumer/cn/doc/20215)（即《华为国内短信服务使用协议》和[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)中的一部分）来处理最终用户的个人数据。

5.4 协议双方签订的[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/20201)和[《华为APIs使用协议》](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)不会随着本协议终止而自动终止。
