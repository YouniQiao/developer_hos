# AppGallery Connect付费服务协议

点击下载[AppGallery Connect付费服务协议](https://terms-drcn.platform.dbankcloud.cn/agreementservice/developer/getAgreementTemplate?agrType=1108&country=cn&language=zh_cn&version=2022101901)。

本协议是您与华为订立的《[华为APIs使用协议](https://developer.huawei.com/consumer/cn/doc/distribution/app/20209)》之补充协议，确立您使用华为AppGallery Connect付费服务时享有的权利和义务。前述《华为APIs使用协议》是您与华为订立的《[华为开发者服务协议](https://developer.huawei.com/consumer/cn/doc/start/20201)》之补充协议，与其构成一个整体（《华为APIs使用协议》和《华为开发者服务协议》下合称“原协议”）。您点击本协议页面下方的“我同意”按键或使用本协议下任何服务即视为自同意或使用之日（“生效日期”）起您对本协议的充分理解和完全接受。

本协议未尽事宜，按照原协议执行。如本协议的任何条款与原协议发生冲突的，仅在AppGallery Connect付费服务的范围内，以本协议条款为准。

## **1. 定义**

除非本协议另有规定，本协议中的术语与原协议中的含义相同。下列术语在本协议中的含义为：

1.1 “华为AppGallery Connect”或“华为AG Connect”：为通过应用市场展示和/或分发的产品提供一站式服务的华为平台。华为AG Connect向开发者提供包括但不限于：分发服务（应用和内容分发、预约、内测、首发等）、运营服务（产品运营、内容运营、活动运营、用户运营）、分析服务（分发分析、运营分析、质量分析、高级分析）和开发服务（商业开发、增长服务）、链接推广服务等华为服务。

1.2 “AppGallery Connect付费服务”（以下简称“AG Connect付费服务”）：华为在开发者使用华为AG Connect时向其提供的、会产生资源消耗的服务，华为会基于数据记录条数、存储空间和网络的占用情况等因素来进行收费。华为会在AG Connect服务页面具体列明付费服务的内容和计费方式。

1.3 “关联文档”：指华为提供给开发者的、用以说明AG Connect付费服务的相关文件。

1.4 “服务费用”：指开发者使用各项AG Connect付费服务所需支付的服务费及税费的总和。

1.5 “计费数据”：是指开发者使用需要付费的服务而产生的用量、账单等数据，如API的调用次数、套餐订购、付费账单等。

## **2．服务内容**

2.1本协议项下的AG Connect付费服务包括但不限于SaaS应用服务、人工定制服务、地图服务、定位服务、机器学习服务、云数据库服务、认证服务、云存储服务及云函数服务等涉及资源消耗的服务。

2.2华为有权在本协议关联文档中具体列明计费服务的类型和计费方式并对其进行不定期更新，具体内容参见《[服务计费说明](https://developer.huawei.com/consumer/cn/doc/development/AppGallery-connect-Guides/agc-service-billing)》。

2.3华为有权不定期地增加与计费服务相关的应用、工具、特性或功能。

## **3．计费类型及付款**

**3.1** 本协议提供以下两种计费类型的服务：

**3.1.1. 限额免费使用。**针对特定的AG Connect付费服务，华为将在约定的周期内向您提供一定的免费配额供其使用相关的AG Connect付费服务。在当期免费配额使用完后，您可以选择“按需付费使用”，否则将不能继续使用该付费服务。该免费配额会在下一服务周期自动恢复。

**3.1.2. 按需付费使用。**针对所有的AG Connect付费服务，华为将按您使用服务实际产生的使用量向您收费。您应预先对您的付费服务账户充值或确保微信/支付宝账户中有充足金额。华为以服务SKU（Stock Keeping Unit）为计费颗粒度统计付费服务使用量，并从您的付费服务账户余额或微信/支付宝账户中扣减应付金额。若因账户余额不足导致扣缴失败，华为会给您发送邮件充值提醒或支付失败提醒。若您在账户余额不足后仍继续使用付费服务，华为将在您充值后进行补扣款。如您一直未充值或未重新支付成功，华为有权在欠费48小时后，暂时停止您对该服务的使用。同时，华为会在暂时停止该服务的7天后将服务资源释放。华为会在释放资源前提前3天通过电子邮件或本网站通知您。

**3.2 付费服务账户**

3.2.1 为使用AG Connect付费服务，您需开通付费服务账户并提供向您开具发票所需的相关税务信息。您保证该等税务信息准确有效。开通付费服务后，开放平台将为您开通付费服务账户（“付费服务账户”）。

**i. 充值付费方式。**您可以随时登录到本网站及相关开放平台，根据相应指引向您的付费服务账户充值。充值付款方式根据您所在的国家不同，线上或线下支付均可。付费服务账户可以用于指定用途（仅限用于某一项付费服务），也可以设置为公用（可用于所有付费服务）。完成充值后，您可以消费AG Connect平台提供的AG Connect付费服务。

**ii. 信用卡付费方式。**华为根据其规则在部分国家提供信用卡付费方式，按邀请制向合作伙伴开放。受邀的合作伙伴可在付费服务账户下增加以合作伙伴名义开户的公司信用卡，并用于消费付费服务，如果因为您的账号使用了他人银行卡因此带来的盗刷、拒付等损失，华为有权利立即停止相关服务，您需要赔偿华为因此遭受的相关损失。为验证该信用卡是否有效，华为将向该信用卡扣款1个货币单位的金额，该笔金额将在您从付费服务账户下删除该信用卡后退还。您成功开通信用卡付费方式后，华为将在您的消费金额满500美金（或等值币种）时或每7个自然日（两者时间在先者）向您的信用卡扣款。

**iii.微信/支付宝付费方式。**您可以在服务购买页面，选择微信/支付宝进行支付，支付前请确保账户中有充足的余额。支付成功后，您可以消费AG Connect平台提供的AG Connect付费服务。

**iv. 消费付费优先顺序。**对于非微信/支付宝付费方式，系统将优先选择付费服务账户的充值余额付费，若余额不足，则通过信用卡付费。

3.2.2 在付费服务账户使用服务后，您可自行在平台查询账户余额和账户明细，华为不再另行发送对账单。

3.2.3 当您进行跨境汇款时，您需要遵守对应国家的跨境汇款外汇管制政策，并承担跨境汇款的银行汇费。

**3.3 税务与发票**

3.3.1 本协议下的所有结算金额均包括增值税。所有向华为或您征收的税款由各方根据所在区域的适用税法/税务条例相应承担。

您根据本协议向华为支付的所有款项均不得进行抵销、反申索或要求的代扣或扣减。若适用法律要求代扣税款，该代扣税由您承担，您向华为支付的金额不应扣除该代扣税。

3.3.2 付费服务账户充值或购买服务后，若您需要发票，您可在注册或更新发票所需信息后在开放平台上申请。华为将在订单成功后的规定期限内开具订单支付金额的发票。若您在订单成功后的十五（15）日内没有主动索取发票，开放平台将自动触发开票申请，开票信息以您在开放平台设置的信息为准。如因您的信息未及时更新导致发票错误，相关损失由您承担。发票的类型和您需向华为提供的信息详见附件A的相应条款。

3.3.3 如果任何税率提高、降低或者废除或者新税种生效，或者在本协议有效期内该税的解释或适用性发生任何变化，则双方将作出相应的调整，以保护您和华为不受上述变化的影响。

**3.4 退款**

3.4.1 付费服务一经售出及完成，不可退款。

3.4.2 付费服务账户余额退款要求：

i. 您需向华为申请支付款项及相应账户的退款。退款流程将在申请通过华为审核后启动。

ii. 在跨境退款流程中，您需要遵守您所在国家的跨境收款外汇管制政策要求，并承担相应的退款成本，包括但不限于跨境付款银行手续费、汇率损失及华为的税费。

3.5 改变服务费用。

华为保留不定期更新AG Connect付费服务的服务价格的权利。如有变更，将提前30日通过电子邮件或本网站通知开发者。

## **4. 协议终止**

4.1 任何一方终止原协议后，本协议将在同一终止日期自行终止。

4.2 您可以提前30天书面通知华为，并经华为书面同意后终止本协议。

4.3 若华为决定不再提供AG Connect付费服务，决定更改AG Connect付费服务或无意再与您就AG Connect付费服务签订协议，则华为有权在提前至少30天向您发出通知后随时终止本协议。
