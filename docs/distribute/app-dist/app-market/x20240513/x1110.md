# 华为开发者音频内容合作协议

**以下条款及条件（以下简称为“本协议”）是发布者与华为订立的华为开发者音频内容合作协议。点击本网站的“我同意”按钮即视为自您同意本协议或使用华为音频能力平台之日（“生效日期”）起对**[**《华为开发者服务协议》**](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)**和**[**《华为开发者商户服务协议》**](https://developer.huawei.com/consumer/cn/doc/20202)**条款的充分理解和完全接受。**

**发布者理解、知悉并同意，华为音频能力平台为内容分发渠道，华为有权将发布者内容进行推广、分发，包括但不限于华为自营平台。若发布者不接受本协议，请不要点击“我同意”。**

发布者就将发布者内容接入在华为音频能力平台的合作事宜, 与华为经双方友好协商，在平等互利的基础上，达成如下协议：

## **1.分发区域和签约主体**

按照[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)中 “分发区域和签约主体”条款执行。

## **2.定义**

如无特别说明，本协议内的术语与[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)及[《华为开发者商户服务协议》](https://developer.huawei.com/consumer/cn/doc/20202)中的术语意义相同，定义如下：

2.1发布者：指有效申请并经华为审核同意后，在华为音频能力平台上发布【音频内容】的“开发者”（“开发者”定义在[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)中）。

2.2发布者内容：指有效申请并经华为审核同意后，发布者在华为音频能力平台上发布的【音频内容】“产品”（“产品”定义在[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)中）。

2.3华为渠道：指由华为及其关联公司（以下合称“服务提供方”），以及服务提供方的合作伙伴生产和销售的终端设备产品，【包括但不限于手机、平板、电脑、便携设备、OTT盒子、智能电视、可穿戴设备、智能家庭产品、VR设备、车载终端等】。

2.4华为自营平台：指由华为及其关联公司通过华为渠道运营的为最终用户提供【音频内容】的各软件服务和渠道入口，包括但不限于华为视频、华为音乐、华为收音机等。

2.5最终用户：定义在[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)中。

2.6华为音频能力平台：指由华为基于开放平台运营的音频内容分发平台，（“开放平台”定义在[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)中）。华为将发布者发布至华为音频能力平台的发布者内容进行分发、推广（包括但不限于通过各类音视频点播、直播、按次收费、免费内容、免费试看等方式向最终用户提供）等。华为音频能力平台由发布者自主注册并独立承担全部责任。

2.7发布者账号：指在华为音频能力平台中，由发布者注册并完成企业实名认证的华为账号（“华为账号”定义在[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)中）。

2.8扣除项：遵照[《华为开发者商户服务协议》](https://developer.huawei.com/consumer/cn/doc/20202)4.2条描述。

## **3.合作模式**

3.1华为通过华为音频能力平台接收发布者内容，将发布者内容在分发区域内通过华为自营平台进行分发、推广，从而最终用户可以通过上述平台获取发布者内容。

3.2发布者可以有如下几种方式发布内容，并授权华为按照本协议约定进行分发、推广：

1）通过发布者账号在华为音频能力平台上传发布者内容；

2）通过平台的API标准接口进行音频内容接入。

上述方式均要求发布者应配合华为，进行相关技术或者内容的联调和测试，通过华为的联调测试后，华为在华为自营平台推广、分发发布者内容。

## **4.版权约定**

4.1华为有权将发布者通过华为音频能力平台所发布的内容（包括但不限于字、图片、图文、音频等）进行不改变原意的修改并进行分发、推广。发布者承诺，华为及其关联公司对发布者通过华为音频能力平台所上传发布的任何内容具有分发区域内、协议有效期内、不可撤销的使用权、推广权、分发权、信息网络传播权或者以其他方式为达到合作目的而使用上述发布者内容的权利。

4.2发布者保证其通过华为音频能力平台所发布的内容拥有完整的知识产权或合法授权，不存在任何违反法律、法规、政策的或者侵犯任何第三方合法权益（包括但不限于知识产权、肖像权、名誉权等）的情况。

4.3如果发布者内容有特殊的版权期限，需由发布者在版权到期后自行负责下架或删除发布者内容，如因版权到期而未及时下架产生的相关责任由发布者承担。

4.4发布者授权华为及其关联公司通过各种线上或线下的营销渠道展示、表演、分发、传输发布者内容或其任何部分、发布者的公司名、品牌商标、广告材料、海报、横幅等营销材料，在社交平台上推广与华为音频平台、华为自营平台和华为渠道相关的业务或用于其他合理的目的。

4.5发布者授权华为及其关联公司在华为渠道上使用发布者的商标、公司名或其他标签或标识，并指明内容和资料的来源或用于其他合理的目的。

## **5.合作期限**

5.1本协议有效期1年，发布者点击本网站的“我同意”按钮即视为自发布者同意之日起协议生效。（“生效日期”）。

5.2合同到期若双方无异议，则自动延续1年；若一方不同意自动延续，应当在合同到期之日两个月前提出。如无终止，合同可多次延续。

## **7.结算规则和结算条款**

7.1华为根据分发推广发布者内容收益（内容付费，不含广告收益）与发布者结算，发布者分成收益金额=（发布者内容产生收益-扣除项）\*50%。

7.2 会员模式结算：

① 字段定义（仅适用于会员分成模式）

音频资源：指本合同中发布者向华为提供的数据集成接口内所链接到的音频资源。

会员音频资源：指本合同中（根据实际情况选择：发布者向华为提供的数据集成接口内所链接到的标记会员标识/华为选定纳入合作会员包的发布者）的音频资源。

音频播放：指本合同中发布者提供音频资源在平台上的播放。

有效播放：指本合同中双方约定的单次音频播放达到一定时长为一次有效播放。有效播放时长以秒为计算单位，播放大于等于30秒或其他由华为自行设定的时长记为有效播放时长,按照会员用户实际播放时间计入，特别说明：有效播放时长包括在线和离线播放、重复播放内容。

合作会员：指本合同中包含发布者及其他发布者的音频会员畅听包。

会员可结算收入：购买合作会员的会员可结算收入之和，即SUM [用户单次支付金额\*（1-支付渠道费率）\*收入确认系数]。

收入确认系数是指从最终用户支付当日起到当月末占总计天数的比例，订购会员每个月按照31天计算。举例①：用户A在2019年1月19日支付了3个月的会员费用（2019年1月19日到2019年4月22日，统计为3个31天=93天，第一天1月19日不计算为结算日，从1月20日当天开始计算起始时间，结束日期为4月22日，不是4月21日），则1月份的收入确认系统是12/93=0.13，2月是28/93=0.30，3月是31/93=0.33,4月是22/93=0.24；如下表所示：

|  |  |  |  |
| --- | --- | --- | --- |
| 起始日 | 结束日 | 订阅天数 | 系数 |
| 2019年1月20日 | 2019年1月31日 | 12 | 13% |
| 2019年2月1日 | 2019年2月28日 | 28 | 30% |
| 2019年3月1日 | 2019年3月31日 | 31 | 33% |
| 2019年4月1日 | 2019年4月22日 | 22 | 24% |
| 合计 | - | 93 | 100% |

会员结算占比：发布者发布的会员音频资源（不包括用户单独购买后的播放）在结算周期内在平台的全量有效播放时长 / 合作会员作品库内音频资源（不包括用户单独购买后的播放）结算周期内在平台的全量有效播放总时长。

② 结算公式

双方就所获得的会员可结算收入进行分成。

|  |  |  |
| --- | --- | --- |
| 净收益分成 | 50%:50% | 合同有效期内，双方按累计净收益进行分成，具体公式如下：  发布者收益=（会员可结算收入-退款）\*会员结算占比\*分成比例 |

7.3广告分成：

华为通过发布者产品有效播放占比来计算发布者产品的的销售广告净收益（即销售广告收入扣除运营成本，包括可能涉及的坏账（如涉及）、广告税费、第三方收取的信息通道费、广告管理费等后的收益）。

①分成比例

无论是自有版权内容或非自有版权内容，发布者分成比例均为40%。

②结算公式

|  |  |  |
| --- | --- | --- |
| 结算方式 | 分成比例 | 计算公式 |
| 净收益分成 | 发布者:华为  40%:60% | 华为平台内发布者收益=（发布者产品在该华为平台有效播放时长/该华为平台有声领域的总有效播放时长）\*该华为平台专辑内广告位（不含公共页面广告位）销售广告净收益\*分成比例  各华为平台独立计算 |

有效播放时长：指非会员用户收听发布者内容的有效播放时长（不包括用户已经购买的内容的有效播放时长），以秒为计算单位，播放大于等于30秒或其他由华为自行设定的时长记为有效播放时长；按照实际播放时间计入。特别说明：有效播放时长包括在线和离线播放、重复播放内容。

7.4结算数据统计：双方在费用结算时以华为结算数据统计平台提供的数据为准。

7.5结算规则：本协议第7条定价结算未做明确约定的，各方遵照[《华为开发者商户服务协议》](https://developer.huawei.com/consumer/cn/doc/20202)执行。

7.6结算金额：以华为每期结算单金额为准。
