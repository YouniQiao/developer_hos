# 华为开发者HiAI服务协议

点击下载[华为开发者HiAI服务协议](https://terms-drcn.platform.hicloud.com/agreementservice/developer/getAgreementTemplate?agrType=1008&country=cn&language=zh_cn&version=2019122401).

本协议是您与华为订立的具备法律效力的协议，以便您通过本网站使用华为AI SDK/API/DDK服务。本协议为[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)的补充协议，与其构成一个整体。您点击本协议下方的“我同意”按键后或使用本协议下任何服务即视为自您接受、注册或使用之日（“生效日期”）起对本协议条款及[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)条款的充分理解和完全接受。本协议未尽事宜，按照[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)执行。如果您不同意本协议条款或[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)条款，您应当停止使用本协议下的服务或停止使用本网站。

## **1.** **定义**

除非本协议另有规定，本协议中的术语与[《华为开发者服务协议》](https://developer.huawei.com/consumer/cn/doc/start/agreement-0000001052728169)中的含义相同。以下术语在本协议中的含义如下：

1.1 “AI SDK/API/DDK”（简称“AI产品”）：开发者可以从[华为开发者联盟网站](https://developer.huawei.com/consumer/cn/)（“本网站”）获取到的AI相关的SDK、API或DDK，具体产品以本网站公布的清单为准。

1.2 “AI SDK/API/DDK服务”（以下又简称“AI产品服务”）：华为向您提供的AI产品的下载、升级、问题修复等服务。

1.3 “AI生态”：与AI产品相关的产品、服务等形成的链圈式生态体系。
