# 币种美化/更正规则

非美元币种国家/地区按照当时的汇率先将设置的“汇率换算基准价格”兑换为以美元为单位的价格，再由以美元为单位的价格兑换为该国家/地区币种为单位的价格，如涉及税率，还需加上对应税额，并按照如下币种美化/更正规则调整为最终商品价格，即“用户支付价格（含税）”。

* 通用币种要求国家/地区：按照汇率换算后，视价格不同情况进行如下调整。
  + 如汇率换算后价格正好为整数或仅有两位小数，则将此价格（保留两位小数）作为商品（或应用）的最终价格。如换算后价格为1.23，则默认选取1.23作为该商品（或应用）的最终价格。
  + 如汇率换算后价格含有三位及以上的小数，则自动截断第二位（不含）后面的小数部分，并根据第二位小数X的情况进行调整。
    - 如0≤X≤5，则第二位小数取值为6，如换算后价格为1.213，则最终价格为1.26。
    - 如6≤X≤8，则第二位小数取值为9，如换算后价格为1.264，则最终价格为1.29。
    - 如X=9，则将截断后的价格加上0.01作为最终价格，如换算后价格为1.2967，则最终价格为1.30。
* 特殊币种要求国家/地区：部分国家币种存在特殊要求，包括仅支持整数和以五分之一为最小单位（均保留两位小数）。
  + 仅支持整数的国家/地区，按照汇率换算后，若为非整数，则向上取整（保留两位小数）作为最终价格。如换算后价格为5.02，则默认选取6.00作为最终价格。
  + 以五分之一为最小单位的国家/地区，按照汇率换算后，若为非整数，则向上取符合五分之一要求的数值（保留两位小数）作为输入价格。如换算后价格为1.23，则默认选取1.40作为该商品的最终价格。
