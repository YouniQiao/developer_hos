# 错误码

开发联运游戏的过程中出现接口调用失败，开发者可以参考如下错误码：

| 业务 | 错误码 |
| --- | --- |
| 通用 | 调用Kit接口报错时请先参见[通用错误码](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/errorcode-universal)查看是否为框架错误。 |
| 基础游戏服务 | Game Service Kit错误码请参见[ArkTS API错误码](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/gameservice-error-code)。 |
| 应用内支付服务 | IAP Kit错误码请参见[ArkTS API错误码](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/iap-error-code)。 |
