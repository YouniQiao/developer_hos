# 应用内支付服务

若游戏支持玩家在游戏内购买各类虚拟商品，例如消耗型商品、非消耗型商品、自动续期订阅类商品，游戏必须接入华为应用内支付服务。

IAP Kit为开发者提供简便的接入流程，让开发者聚焦游戏本身的业务能力，快速助力开发者商业变现。具体接入流程请参见[IAP Kit（应用内支付服务）](`https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/iap-kit-guide`)。
