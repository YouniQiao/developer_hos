# 技术支持

1. 请先仔细阅读业务文档，部分功能对设备和系统有限制。
2. 查看[FAQ](`/docs/distribute/app-dist/app-services/intermodal-transport-services-0000001933253576/joint-operation-game-0000002024369570/joint-operation-faq-0000002024211370)。
3. 通过[智能客服](`https://developer.huawei.com/consumer/cn/customerService/#/bot-dev-top/faq-top/faq-talk-top`)查找问题解决方案。
4. 登录[华为开发者论坛](`https://developer.huawei.com/consumer/cn/forum/block/gaming-hub`)和[Stack Overflow社区](`https://stackoverflow.com/questions/tagged/appgallery-connect`)参与问题讨论。
5. 如果以上方法仍未解决您的问题，可以通过[在线工单系统](https://developer.huawei.com/consumer/cn/support/feedback/#/add/101704353566310877?level2=101704353626565886&level3=101704354579010004&keyWord=Game Service Kit)与我们进行联系，提交后的工单可在开发者联盟“管理中心”的“开发者中心 &gt; 我的客服”页面查看工单处理进展。
