# 参数说明

## 奖品素材

礼包奖品需准备如下素材内容：

| 准备项 | 说明 |
| --- | --- |
| 礼包名称 | 奖品名称。建议格式为“应用名+空格+礼包名称”，例如“永恒纪元 超值礼包”，要求1~100个字符。 |
| 奖品图片 | 礼包图片。宽高为216\*216px，圆角半径为32px，大小不超过500KB的JPG/PNG图片。 |
| 礼包兑换码 | 奖品兑换码。兑换码文件的格式与要求：   * 每条兑换码只能由大小写英文字母、数字、英文下划线和英文分隔符组成。 * 文件中每行排列1条兑换码，请勿添加其他注释文字。 * 文件内不能存在重复的兑换码。  * 文件类型为ANSI编码的txt格式。 * 文件大小不超过100M。 * 文件行数不超过200W。 |
| 礼包内容说明 | 礼包中所有的奖品名称。建议格式为“奖品名称\*对应数量”，用“、”分隔，例如“钻石\*4、经验卡\*2”，要求1~2000个字符。 |
| 礼包兑换地址 | 礼包兑换的地址。要求1~2000个字符。示例如下：   * 您进入游戏-点击头像-输入礼包码即可获得奖励。 |
