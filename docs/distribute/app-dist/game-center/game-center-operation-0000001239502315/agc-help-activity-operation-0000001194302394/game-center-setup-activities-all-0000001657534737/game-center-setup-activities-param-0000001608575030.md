# 参数说明

## 奖品素材

华为优惠券奖品需准备如下素材内容：

![](./img/cc055a5dce14.png)

华为优惠券必须接入[华为应用内支付服务](https://developer.huawei.com/consumer/cn/doc/HMSCore-Guides/introduction-0000001050033062)。

| 准备项 | 说明 |
| --- | --- |
| 奖品名称 | 华为优惠券名称。建议格式为“金额+优惠券”，例如“5元优惠券”，要求1~100个字符。 |
| 奖品描述 | 华为优惠券的领取说明和使用要求。要求1~2000个字符。   * “满减券”示例：当前华为优惠券需单笔消费达到指定金额可抵现。每笔订单限用1张，且不支持与其他券叠加使用。有效期为\*天，逾期将失效，您可进入“我的 > 优惠券”查询优惠券详情。 * “无门槛券”示例：华为优惠券可在指定应用中抵现，有效期为\*天，逾期将失效，您可进入“我的 > 优惠券”查询优惠券详情。 |
| 华为优惠券成本结算 | 活动结算模式。若活动奖品中包含“华为优惠券”时，请提前规划活动的结算模式：   * 收入抵减模式：在每月的结算日，联运分成收入将扣减上月用户实际使用优惠券的金额。 * 充值预购模式：在每月的结算日，账户下的冻结金额将扣减上月实际使用优惠券的金额，您需要提前[冻结预留金额](https://developer.huawei.com/consumer/cn/doc/app/game-center-setup-activities-faq-0000001608894534#section51418272517)。 |

礼包奖品需准备如下素材内容：

| 准备项 | 说明 |
| --- | --- |
| 礼包名称 | 奖品名称。建议格式为“应用名+送+礼包名称”，例如“华为送超值礼包”，要求1~100个字符。 |
| 奖品图片 | 礼包图片。宽高为216\*216px，圆角半径为32px，大小不超过500KB的JPG/PNG图片。 |
| 礼包兑换码 | 奖品兑换码。兑换码文件的格式与要求：   * 每条兑换码只能由大小写英文字母、数字、英文下划线和英文分隔符组成。 * 文件中每行排列1条兑换码，请勿添加其他注释文字。 * 文件内不能存在重复的兑换码。  * 文件类型为ANSI编码的txt格式。 * 文件大小不超过100M。 * 文件行数不超过200W。 |
| 礼包内容说明 | 礼包中所有的奖品名称。建议格式为“奖品名称\*对应数量”，用“、”分隔，例如“钻石\*4、经验卡\*2”，要求1~2000个字符。 |
| 礼包使用描述 | 礼包的领取说明和使用要求。要求1~2000个字符。示例如下：   * 您可以在华为应用市场/游戏中心客户端输入礼包码获得奖励。 * 您需要在礼包有效期内使用，逾期失效。 |

第三方卡券需准备如下素材内容：

| 准备项 | 说明 |
| --- | --- |
| 奖品名称 | 第三方卡券名称，例如“华为100个花币”，要求1~100个字符。 |
| 奖品图片 | 第三方卡券图片。宽高为216\*216px，圆角半径为32px，大小不超过500KB的JPG/PNG图片，详情请参考[第三方卡券图片规范](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20251031145026.80294560203460648340942757208184%3A50001231000000%3A2800%3A3C1F4F1C607EFCCFC4B082B884A507FAA634661A1A9E968587563F054699243F.zip?needInitFileName=true)。 |
| 奖品兑换码 | 第三方卡券兑换码。兑换码文件的格式与要求：   * 每条兑换码只能由大小写英文字母、数字、英文下划线和英文分隔符组成。 * 文件中每行排列1条兑换码，请勿添加其他注释文字。 * 文件内不能存在重复的兑换码。  * 文件类型为ANSI编码的txt格式。 * 文件大小不超过100M。 |
| 奖品描述 | 第三方卡券的领取说明和使用要求。要求1~2000个字符。示例如下：   * 您可以在第三方卡券对应的客户端获取奖励。 * 使用第三方卡券时请遵守第三方应用的要求。 |

实物奖品需准备如下素材内容：

![](./img/f953f6a399a0.png)

实物类奖品仅适用于“抽奖/签到”活动。

| 准备项 | 说明 |
| --- | --- |
| 奖品名称 | 实物奖品名称。要求1-100个字符，名称不能包含空格符、水平方向制表符、标签名和特殊字符。  说明：  若是华为的实物奖品，必须加上“HUAWEI”前缀，例如“HUAWEI Mate30”。  非华为产品禁止露出品牌名。  不可使用京东E卡/实体卡等具有现金价值的第三方发行的礼品卡。 |
| 奖品图片 | 实物奖品图片。宽高为216\*216px，圆角半径为32px，大小不超过500KB的JPG/PNG图片，详情请参考[实物奖品图片规范](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20251031145026.02067088641780433816574268397557%3A50001231000000%3A2800%3AE2DD060C2CEA9BF469C15696D24A01026911FB22F3AB4321D717500C5ACE7B9A.zip?needInitFileName=true)。 |
| 奖品描述 | 实物奖品的详细描述。要求1~2000个字符。示例如下：   * 颜色：随机 * 型号：P40 Pro * 内存：8GB+256GB * 版本：全网通5G * 图片仅供参考，所发奖品以实物为准。请填写准确的收件地址，收件信息仅用于寄送实物奖品，并将在中华人民共和国境内保存2年。请于获奖后7日内填写，逾期视为弃奖。 |
