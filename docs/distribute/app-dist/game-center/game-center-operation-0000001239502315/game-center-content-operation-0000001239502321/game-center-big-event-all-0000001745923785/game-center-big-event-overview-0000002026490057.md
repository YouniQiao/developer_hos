# 概述

游戏大事件是指游戏全生命周期中的重要事件节点，如大版本更新、首发、首曝、定档、特别活动、测试等。为方便您对游戏的事件节点进行运营，华为游戏中心向您开放游戏大事件自主运营的能力。您可以线上填报游戏事件内容，审核通过后，游戏大事件将在游戏中心固定位置展示，有效触达用户，促进新游预热下载、老游拉新促活。

## 游戏大事件类型

您可以基于不同的事件创建相应的游戏大事件，当前提供以下7种大事件类型供您选择。

| 游戏大事件类型 | 简介 | 操作指南 |
| --- | --- | --- |
| 大版本更新 | 已上架游戏的重要更新版本，如上线新玩法、新地图等。 | [大版本更新操作指导](`https://developer.huawei.com/consumer/cn/doc/app/game-center-big-event-version-update-0000001990090584`) |
| 首发 | 游戏首次面向用户公开正式运营。 | [首发操作指导](`https://developer.huawei.com/consumer/cn/doc/app/game-center-big-event-first-applyfor-0000001991401030`) |
| 首曝 | 游戏首次向玩家发布PV、实机视频等内容。 | [首曝操作指导](`https://developer.huawei.com/consumer/cn/doc/app/game-center-big-event-first-exposure-0000002101313586`) |
| 定档 | 游戏确定正式上线档期。 | [定档操作指导](`https://developer.huawei.com/consumer/cn/doc/app/game-center-big-event-scheduling-0000002136952745`) |
| 特别活动 | 游戏内的亮点活动。 | [特别活动操作指导](`https://developer.huawei.com/consumer/cn/doc/app/game-center-big-event-special-events-0000002137112369`) |
| 测试（内测） | 游戏在华为平台开启内测。 | [测试（内测）操作指导](`https://developer.huawei.com/consumer/cn/doc/app/game-center-big-event-special-test-0000002202109125`) |
| 测试（先锋测试） | 游戏在华为平台开启先锋测试。 | [测试（先锋测试）操作指导](`https://developer.huawei.com/consumer/cn/doc/app/game-center-big-event-pioneer-test-0000002523247654`) |
