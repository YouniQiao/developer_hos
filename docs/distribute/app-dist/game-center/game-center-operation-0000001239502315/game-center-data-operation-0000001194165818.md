# 数据运营

平台为您提供详细的产品数据，协助您了解、分析游戏在平台的下载、付费、活动运营等情况。

## 数据入口

1. 登录 [AppGallery Connect 网站](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)，点击“分析”。
2. 从列表中选择您的应用，进入页面。
