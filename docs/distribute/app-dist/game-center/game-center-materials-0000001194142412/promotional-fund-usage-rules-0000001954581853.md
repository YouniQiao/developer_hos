# 鸿飞计划推广金使用规则

<strong>政策周期：</strong>2026年1月1日-2026年6月30日。

<strong>申请有效期：</strong>2026年9月30日截止，开发者需在截止日前完成推广金激活申请流程。

<strong>消耗有效期：</strong><strong>推广金自激活日起，消耗有效期180天，超期未消耗则自动失效。</strong>

<strong>规则适用对象：</strong>本规定适用于已使用华为数字商品及联运服务并签署相关协议，按协议约定通过“【游戏】应用数字商品优惠政策—激励计划”获得推广金激励并申请推广金使用的开发者。

<strong>规则内容 ：</strong>

1. 生成和使用：华为月度按规则生成推广金具体额度，开发者按需按规则申请使用。
2. 使用对象：可用于获得激励的开发者公司主体或其所属的开发者群组下公司主体的在架游戏投放，不可用于非游戏类应用投放。
3. 使用范围：可用于鲸鸿动能或应用市场的推广。其中应用市场的推广仅限HarmonyOS 5及以上版本的竞价投放。
4. 使用限制：适用《华为应用市场联运服务协议》的游戏时，HarmonyOS 4.x及以下版本仅可用于联运包体投放。
5. 推广金发放：开发者按需向鸿飞计划推广金华为联系人BD邮件申请，申请时需明确为竞价或合约推广投放、申请金额、游戏、推广渠道（鲸鸿动能/应用市场）、推广账户信息，华为审核后发放到对应推广账户。开发者每次申请充值时，鲸鸿动能合约账户的充值金额需不低于当次充值金额的30%。
6. 账户管理：鲸鸿动能投放时，竞价与合约推广金采用相互独立专用推广账户，开发者需建立专用推广账户并在申请时将账户信息发送至华为审核，激励发放后激励及账户不可相互转化使用。应用市场投放时，开发者申请的返利金统一体现在“返利金”账户，与现金账户独立区分。

<strong>其它规则：</strong>

1. 推广金不可提现。
2. 推广金的使用需遵循具体使用场景下的对应管理规则。
3. 华为有权根据实施情况调整推广金的使用规则，如有任何疑问，请联系华为。
