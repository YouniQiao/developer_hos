# RPK游戏接入

## 华为快游戏特点

华为快游戏具有免安装、速度快、省电、省内存、省空间的特点：

* 底层图形加速：画质高、画面流畅，极速性能和体验。
* 开放设备能力：位置、传感器、蓝牙、网络、电量等。
* 网页跳转：游戏网站流量转换为可留存桌面的快游戏。

## 技术支持

华为快游戏为开发者提供了优质的工具和技术支持。

### 极速开发

华为快应用 IDE 提供真机云测，方便调试。

### 专业支持

开发者服务团队提供专人指导接入华为账号、支付能力等。

## 接入指导

开发华为快游戏的具体流程请参见官方文档[华为快游戏开发指南](`https://developer.huawei.com/consumer/cn/doc/quickApp-Guides/quickgame-doc-introduction-0000001073124845`)。
