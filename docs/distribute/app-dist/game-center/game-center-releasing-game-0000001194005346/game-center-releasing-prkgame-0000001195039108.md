# RPK游戏上架

当您的快应用开发和测试完成后，您可以在AppGallery Connect（以下简称AGC）正式提交[RPK游戏上架](`https://developer.huawei.com/consumer/cn/doc/distribution/app/agc-help--release-fastapp-0000001099836868`)，华为审核人员审核通过后应用就会变为“已上架”状态，用户可在华为应用市场搜索到您的快应用。
