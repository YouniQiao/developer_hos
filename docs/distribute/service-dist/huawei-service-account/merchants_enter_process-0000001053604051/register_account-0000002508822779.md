# 步骤1 注册并登录开发者联盟账号

进入[华为服务号](https://developer.huawei.com/consumer/cn/business-touch/)网页，点击“立即接入”。若您还没有华为服务号账号（不建议使用个人华为账号），请通过电子邮箱或手机号码完成华为账号注册；若已有相关账号，请输入开发者联盟账号及密码进行登录。
