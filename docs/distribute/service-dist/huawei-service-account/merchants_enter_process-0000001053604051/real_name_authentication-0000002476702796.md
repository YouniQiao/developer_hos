# 步骤2 实名认证

登录后显示认证页面，您应选择[对公银行打款](https://developer.huawei.com/consumer/cn/doc/start/atpopb-0000001062836624)或[人工审核](https://developer.huawei.com/consumer/cn/doc/start/mracoei-0000001062678404)的方式认证成为企业开发者，具体操作可参考[指导视频](https://developer.huawei.com/consumer/cn/training/detail/101597289897864105)（默认一个企业主体只能申请一个开发者联盟帐号，如已认证无需再次认证）。
