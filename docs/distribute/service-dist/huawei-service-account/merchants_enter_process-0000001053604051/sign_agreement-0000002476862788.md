# 步骤3 签署服务号平台协议

进入开发者联盟后台，选择“智慧服务”，点击“华为服务号”卡片（若无此选项，请点击右上角“自定义桌面”，勾选“华为服务号”）。

![](./img/65d46ee09cf5.png "点击放大")

点击“我的华为服务号 （支持Harmony5.0及以后版本）”进入服务号

![](./img/09fff3b51e8c.png "点击放大")

仔细阅读华为服务号平台合作协议后，勾选“我已阅读并同意《华为服务号平台合作协议》”，并点击“同意”。

![](./img/b73fac4a6da8.png "点击放大")
