# AppLinking跳转

## 概述

当用户配置HarmonyOS 5.0及以上跳转方式，链接类型选中AppLinking跳转，用户可以选择该项目下已配置生效的AppLinking，AirTouch会依据此链接作为跳转地址进行分发。

## 跳转商户AppLinking地址

用户通过标签拉起AirTouch预览界面后，点击跳转按钮会跳转商户AppLinking地址。

1. 商户需要在开发者联盟网站申请一个AirTouch应用。
2. 在“[鸿蒙AppLinking(应用或元服务均使用)](/docs/distribute/service-dist/AirTouch/create-service-0000002105172798#ZH-CN_TOPIC_0000002105172798__li5766246184413)”栏目选择AppLinking跳转链接。

   ![](./img/e67e57e4b2ae.png)

   ![](./img/9764862dbe1e.png)

   * AppLinking链接不区分元服务和应用
   * 若当前项目下无AppLinking，可以点击去设置AppLinking进行配置
   * 服务路径：非必填，若AppLinking支持多个path可以在此处填写，例如**/test**
   * 自定义参数：非必填，若AppLinking支持链接参数可以在此处填写。例如**?val=1**
3. 用户通过标签拉起AirTouch预览界面并点击按钮后，查看是否可以根据AppLinking配置的规则跳转到指定的应用或元服务。
