# 能力简介

AirTouch支持配置多种跳转方式：

* HarmonyOS 5.0及以上版本支持[普通链接跳转](/docs/distribute/service-dist/AirTouch/jump-0000002105332614/jump-2-0000002112095246/jump-normal-0000002552678999)和[AppLinking跳转](/docs/distribute/service-dist/AirTouch/jump-0000002105332614/jump-2-0000002112095246/jump-applinking-0000002521479036)两种方式。
* HarmonyOS 3.1/4.0及以下版本不支持AppLinking跳转。
* 支持同时配置HarmonyOS 5.0及以上和HarmonyOS 3.1/4.0及以下版本跳转方式。
* H5链接为兜底配置，只需配置一次，在HarmonyOS 5.0及以上和HarmonyOS 3.1/4.0及以下同时生效。
