# 参数回调接口

## 功能介绍

如果商户在[设置跳转参数来源](https://developer.huawei.com/consumer/cn/doc/service/create-service-0000002105172798#ZH-CN_TOPIC_0000002105172798__li1286220323515)中选择“根据TagId查询参数”时，商户服务器则需要实现该接口供华为服务器去开发者服务器获取跳转参数。

## 场景描述

华为钱包服务器调用此API去开发者服务器获取跳转参数。

## 使用描述

开发者在AGC网站上申请服务时，提供了回调地址。

## 接口原型

| 承载协议 | 接口方向 | 接口URL | 数据格式 |
| --- | --- | --- | --- |
| HTTPS POST | 华为钱包服务器->开发者服务器。 | 开发者在华为AGC网站上提供的回调地址。 | * 请求消息：Content-Type: application/json;charset=utf-8。 * 响应消息：Content-Type: application/json;charset=utf-8。 |

## 请求参数

Request Header：

| 参数 | 是否必选 | 参数类型 | 描述 |
| --- | --- | --- | --- |
| HMSSignType | 是 | String | 签名方式。值为固定值："SHA256WithRSA/PSS"。 |
| HMSSign | 是 | String | 签名值 |

Request Body：

**表1**

| 参数 | 是否必选 | 参数类型 | 描述 |
| --- | --- | --- | --- |
| eventId | 是 | String(32) | 事件ID。 |
| tagId | 是 | String(64) | 标签中的tagId。 |
| eventTime | 是 | String(32) | 事件发生时间（UTC时间）："yyyy-MM-dd'T'HH:mm:ss:SSS'Z'"。 |
| eventType | 是 | String(32) | 查询标签链接参数： 固定为“QUERY\_TAG\_PARAM”。 |

## 请求示例

```
POST /walletkit/callback/notify HTTP/1.1
Content-Type: application/json;charset=utf-8
HMSSignType: SHA256WithRSA/PSS
HMSSign: ***
Accept: application/json;charset=utf-8
Host: partmer.host.com
{
  "eventId": "0a1f133471c4102c8171f845919906f3",
  "tagId": "123",
  "eventTime": "2020-04-27T07:40:24.259Z",
  "eventType": "QUERY_TAG_PARAM"
 }
```

## 响应参数

| 参数 | 参数类型 | 描述 |
| --- | --- | --- |
| returnCode | String | * 成功：0。 * 失败：非0为失败。 |
| returnDesc | String | 错误描述。 |
| tagParam | String | 返回的跳转参数。 |

## 响应示例

```
HTTP/1.1 200 OK
Content-Type: application/json;charset=utf-8
{
  "returnCode": "0",
  "returnDesc": "success",
  "tagParam": "xxxx&xxxx"
}
```

## 调用示例

```
private BaseResponse callbackSp(CallbackRequest requestBody) {
    String priKey = "xxx";

    Map<String, String> header = new HashMap<>(8);
    header.put("Content-Type", "application/json;charset=UTF-8");
    header.put("Accept", "application/json;charset=UTF-8");
    header.put("HMSSignType", "SHA256WithRSA/PSS");
    String hmsSign = RSAUtils.signByRSA(ToStringUtil.signString(requestBody), priKey);
    header.put("HMSSign", hmsSign);

    HttpEntity<CallbackRequest> httpEntity = new HttpEntity(requestBody, header);

    return restful.postForObject(whiteCardCallbackTask.getCallbackUrl(), httpEntity, BaseResponse.class);
}
```
