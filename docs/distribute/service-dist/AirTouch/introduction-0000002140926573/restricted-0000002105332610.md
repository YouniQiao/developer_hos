# 受限说明

## 支持的终端设备

| 终端设备 | 软件版本 |
| --- | --- |
| 搭载HMS Core的终端设备（支持NFC能力） | HMS Core（APK）5.0.3及以上 |
| 搭载HarmonyOS 5.0及以上的终端设备（支持NFC能力） | 全系支持 |

## 支持的地区

当前仅中国大陆地区支持。
