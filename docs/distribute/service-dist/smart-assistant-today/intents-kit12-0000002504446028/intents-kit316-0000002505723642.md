# 功能简介

“付费添卡”是面向鸿蒙应用开发者提供的服务推广工具，支持根据场景、人群画像、和自定义的推广时间等条件为高效、精准推广服务卡片，并提升在负一屏探索流等多场地露出卡片的概率，引导用户把卡片添加到负一屏等，以便于用户便捷、稳定复访，而费用仅在用户成功添加卡片到负一屏等场地后产生，主要优势如下：

（一）流量优势

* 依托负一屏高频使用场景，卡片渗透率高、用户活跃度持续攀升，曝光基础稳固；
* 支持在探索流等多浅层入口添卡展示，覆盖用户主动浏览与被动推荐双路径；
* 根据出价进一步提升推荐概率，增加曝光度，并引导用户添卡。

（二）精准曝光

* 商家可自定义投放时机，自由圈定人群画像，实现精准推广，提高转化率；
* 根据用户所处场景、所在位置等，自动推荐，减少对用户的打扰，用户体验顺畅。

（三）投放自主可控

* 人群圈选：可自定义目标用户群体，推广更精准；
* 时段分发：支持按时间段设置投放计划，适配运营节奏与用户活跃周期，实现曝光资源最优配置；
* 推广预算：支持用户自定义设定投放金额，仅在用户成功添加卡片后才产生费用，实现成本与效果的直接挂钩。
