# 业务介绍

## 1.1 简介

智慧助手·今天（负一屏）是华为终端的系统级入口，是鸿蒙生态下服务分发的主阵地。

智慧助手·今天（负一屏）的主要功能如下图：

![](./img/531b858f5699.png "点击放大")

服务搜索：元服务搜索支持展示伙伴自主配置元服务的简介、优惠、服务品牌专区等，帮助伙伴提升品牌转化，实现品效合一。

服务动态：基于用户服务履约行为，以卡片形式通过负一屏、通知栏、锁屏等系统入口，提供服务实时状态及优惠提醒，伙伴可通过标准化接入能力，完成服务链路闭环。

鸿蒙有礼：营销专区，为伙伴提升品牌曝光，同时通过官方元服务权益和流量双补贴模式，帮助伙伴元服务用户和交易双增长。

常用服务：提供首屏高频曝光，用户可一键直达所需的服务。

卡片：用户可对感兴趣元服务或探索流卡片添加到负一屏，作为负一屏浅层入口和高频曝光区域，为伙伴提供用户增长、留存新机会。

资讯：基于华为浏览器，提供给用户最新、最热门的实时资讯。

探索元服务：负一屏新构建的流量主阵地，提供全新的用户体验。伙伴可将服务与内容无缝嵌入探索流中，通过直观体验，实现“人找服务”到“服务找人”。

## 1.2 业务特色

**丰富的触达入口**

负一屏作为系统级“智能入口”，正日益成为提升用户体验、增强用户粘性与推动生态价值转化的核心载体。提供**智能聚合、场景化服务、个性化推荐与生态协同**于一体的全域交互中枢。

**信息的极致聚合与高效触达**

用户无需打开多个 App，即可在负一屏中使用卡片的形式一站式查看天气预报、日程提醒、待办事项、快递物流、公交到站、新闻资讯等高频服务。这种“所见即所需”的设计，大幅减少了用户操作路径，提升了信息获取效率，真正实现了“极简交互，高效服务”。

**品牌与用户之间的高价值触点**

为应用开发者、内容创作者和广告主提供了流量与转化机会，通过在负一屏探索流嵌入个性化服务卡片、鸿蒙有礼等各种优惠活动，可有效提升用户活跃度、转化率与忠诚度，实现生态价值的倍增。
