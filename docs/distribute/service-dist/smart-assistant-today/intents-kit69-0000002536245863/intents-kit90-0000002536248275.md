# 卡片分发

## 卡片在探索流投放就要付费吗？

探索流是华为负一屏为活跃鸿蒙生态而构建的新业务，可以在探索流中投放商家的服务、商品、门店信息、服务号内容等，免费协助商家拓展客户，所有商家的推荐力度对等。只有在需要提权、加大推广力度，并吸引用户添卡到负一屏成功后，华为才会根据添卡成功的次数收取一定的费用。

## 我们的元服务能用么？应用能不能用？

只要是正常上架的元服务，都可以在探索流投放自己的服务、商品、门店、服务号内容等信息。目前不支持应用投放，我们在持续构建中。
