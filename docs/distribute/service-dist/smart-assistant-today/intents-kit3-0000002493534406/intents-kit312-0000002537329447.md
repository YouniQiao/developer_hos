# 配置指南

服务特点简介的配置指南如下：

（一）在服务分发增长>服务特点简介，点击新建。

![](./img/0bed6ca58bfe.png)

（二）商家可以填写申请名称、元服务描述、选择分发日期、上传元服务图标，内容填写完成后可以通过右侧看到预览效果，同时也可以通过现网测试进行现网查看效果，预览无误后点击提交审核。

![](./img/d28a153b762a.png)

（三）审核通过且在分发日期内就可以在负一屏搜索结果和服务市场生效。
