# 代码节点

代码节点支持通过编写代码来生成返回值。平台支持在代码节点内使用IDE工具，通过编写自定义代码逻辑，来处理输入参数并返回相应结果。注意：代码节点仅企业开发者可用。

./img/71e3008f2dce.png>

**配置代码节点**

代码节点的配置参数说明如下：

|  |  |
| --- | --- |
| <strong>配置</strong> | <strong>说明</strong> |
| <strong>输入</strong> | 声明代码中需要使用的变量。添加输入参数时需要设置参数名和变量值，其中变量值支持设置为固定值或引用上游节点的输出参数。  在代码中引用输入参数时，通过params=args.get('params')，params['input']取值即可。 |
| <strong>代码</strong> | 代码节点中需要执行的代码片段。你可以直接编写代码。  变量：直接使用输入参数中的变量，通过return一个对象来输出处理结果。  函数限制：代码必须保证有且只有一个main函数。即使仅有一个输出值，也务必保持以对象的形式返回。  暂时仅支持Python语言。 |
| <strong>输出</strong> | 代码运行成功后，输出的参数。你可以根据实际需求，在输出结构中只保留必要的参数。  确保此处定义的参数名、类型与代码的return对象的属性完全一致。以代码节点默认提供的代码为例，输出的参数与代码中定义的return对象属性完全一致。 |

**开发语言**

代码节点暂时只支持Python语言。

基于 Python 3.11.4 的标准库，目前支持的库有python内置的所有库以及开源选型预置的三方库。说明：不支持网络请求（基于安全要求）

**使用代码IDE编写代码**

平台提供了网页版代码IDE环境供开发者使用，开发者无需考虑代码部署等问题，只要关注代码逻辑即可。

开发者可选中代码节点，通过页面右侧按钮（如下图所示），拉起IDE编辑工具进行代码编写和测试代码。

./img/0f6a37776d2f.png>

点击【测试代码】，填写输入参数，并单击【运行】。平台会自动执行代码片段，并展示运行结果。运行结果中包括输入参数和输出内容。

./img/ca7955251d27.png>
