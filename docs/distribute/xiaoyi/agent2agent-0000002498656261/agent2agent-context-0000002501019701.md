# 设备上下文参数指令说明

Agent Server端可获取的受控系统上下文信息，需要先在小艺开放平台变量配置项中打开对应配置项：

![](./img/f8757a2f057a.gif "点击放大")

Agent Client请求Agent Server侧的data数据结构定义：

```
{
    "variables": {
        "clientVariables": [{}],
        "systemVariables": [{
             "app_ver": "{{小艺APP的版本号，在小艺开放平台系统变量开关打开后生效}}",
             "foreground_apps": "{{智能体运行的前台应用列表，在小艺开放平台系统变量开关打开后生效}}"
         }],
         "memoryVariables": [{}]
      }
}
```

Variables参数说明：

| <strong>字段名称</strong> | | | 类型 | 是否必填 | 字段描述 |
| --- | --- | --- | --- | --- | --- |
| kind | - | | string | 是 | 字段类型，此处固定为“data”。 |
| data | - | | object | 是 | 数据类型为data类型的结构体定义。 |
| variables | - | JsonObject | 否 | 获取Agent平台的一些上下文变量。 |
| clientVariables | JsonObject | 否 | 应用变量。 |
| systemVariables | JsonObject | 否 | 系统变量。 |
| memoryVariables | JsonObject | 否 | 用户变量。 |
