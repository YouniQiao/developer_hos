# 协议变更点

**更新时间：2025年10月20日**

**主要变更点：**

1、新增定义条款：

您的权利与义务章节新增，3.7 如您的产品涉及智能体服务，您同意附件三将作为最终用户在使用您的产品时与您及华为签订的具有法律约束力的协议条款，用于明确最终用户、您及华为的相关权利义务；同时，您可在此基础上另行订立与最终用户的其他协议条款；

2、新增附件，附件三《三方智能体服务协议》；

**更新时间：2025年3月26日**

**主要变更点：**

1、刷新协议名称，由《华为智慧分发平台合作协议》变更为《小艺开放平台合作协议》；

2、刷新平台名称，由“华为智慧分发平台”变更为“小艺开放平台”；

3、刷新/新增定义条款，如：

（1）刷新小艺开放平台、智慧服务、华为设备、华为应用或服务的定义；

（2）新增意图框架、智能体、关联公司、用户个人数据、个人信息处理者、个人信息受托方的定义；

（3）删除了数据控制者、数据处理者的定义；

4、刷新小艺开放平台意图框架智慧分发服务合作场景；新增智能体服务合作场景；

5、刷新官网链接；

6、刷新第五章数据处理与控制条款；

7、刷新附件，附件A《数据委托处理情况》、附件B《技术组织措施要求》变更为附件一《数据安全协议》、附件二《数据处理协议》

**更新时间：2023年10月24日**

主要变更点：

1、协议执行条件修改为同意协议且使用协议提供的服务；

2、将3.1条中官网地址由`https://developer.huawei.com/consumer/cn/doc/distribution/service/content-specifications-0000001239494199`修改为[HarmonyOS意图框架](https://developer.huawei.com/consumer/cn/huawei-hag)。

3、将6.3中华为终止协议内容修改为“华为决定不再提供智慧分发服务或者对本服务做重大变更时，华为有权提前向您发出通知后终止本协议“。

**更新时间：2023年8月24日**

**主要变更点：**

1.平台协议名称更改为小艺开放平台合作协议。

2.协议内容整体按智慧分发维度进行变更，详细变更点请查看[华为智慧分发平台合作协议](https://developer.huawei.com/consumer/cn/doc/service/harmonyos_agreement-0000001238515921)。

**更新时间：****2022****年****4****月****29****日**

**主要变更点：**

1. 将第3.1条“具体要求请参见小艺开放平台官网”中的超链接地址由`https://developer.huawei.com/consumer/cn/huawei-hag` 修改为`https://developer.huawei.com/consumer/cn/doc/distribution/service/content-specifications-0000001239494199`。
2. 将第6.2条中的“您可在停止使用本协议项下华为所提供的服务前**六（****6****）个月**向华为发出书面通知表示您希望终止本协议，本协议将在华为书面同意后终止”修改为“您可在停止使用本服务前**90****天**向华为发出书面通知表示您希望终止本协议，本协议将在华为书面同意后终止”。
3. 增加第2.6条：华为为您提供能力中心，您可自主选择是否使用该平台提供的内容，如果使用该平台提供的内容，您应遵守附件1《能力中心使用规则》。
4. 增加第5.4条：若您订阅或使用的能力涉及个人信息处理，您应审慎阅读该能力所对应的隐私政策，充分了解该能力对个人信息处理的情况，并在使用该能力处理个人信息前获取最终用户的同意或取得其他合法性基础。
5. 增加附件1：能力中心使用规则。
