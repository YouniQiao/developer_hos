# 请求data数据结构定义

客户端给服务器发送文本query时放在text中，发送文件/图片放在file中，发送端侧事件放在data.events中。

| <strong>字段名称</strong> | | <strong>类型</strong> | 是否必填 | 字段描述 |
| --- | --- | --- | --- | --- |
| kind | - | string | 是 | 字段类型，此处固定为“data”。 |
| data | - | object | 是 | 数据类型为data类型的结构体定义。 |
| authCode | string | 否 | 仅在执行authorize方法时必选。 |
| agentLoginSessionId | string | 否 | 仅在执行deauthorize方法时必选。 |
| events | array[EventObject] | 否 | 客户端给服务器侧上报事件时必选。 |
| userInputInfo | JSONObject | 否 | 客户端底部快捷指令点击产生的用户输入信息。 |
| variables | JSONObject | 否 | Agent Server可获取的小艺开放平台开放的用户变量、系统变量和客户端变量，在开关打开后才会传递给CP Server。 |

EventObject参数说明：

| 字段名称 | | | 类型 | 是否必填 | 字段描述 |
| --- | --- | --- | --- | --- | --- |
| events | - | - | - | - | - |
| header | - | Object | - | - |
| namespace | String | 是 | 按具体业务场景填充。 |
| name | String | 是 | 按具体业务场景填充。 |
| payload |  | Object | - | 按具体业务场景填充。 |
