# 用户问题建议

【用户问题建议】开关打开后，在每次智能体回复后，会为用户提供问题建议。

开关默认为关闭状态，打开开关后，可以选择【系统配置Prompt】或【自定义Prompt】，【系统配置Prompt】默认使用系统预置的提示词生成用户问题建议，【自定义Prompt】支持用户自定义Prompt来生成个性化的问题建议。
