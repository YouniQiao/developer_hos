# 角色声音

开发者可以在【角色声音】部分试听开放的音色并为智能体设置一个默认音色。

![](./img/ab0dda065103.png "点击放大")

另外平台为开发者提供了【终端用户可以切换声音】开关，开关默认为开启状态。

开关打开时，该智能体支持用户在终端自主切换可用的智能体音色，当开关关闭时，则不支持用户在终端自主切换智能体音色。
