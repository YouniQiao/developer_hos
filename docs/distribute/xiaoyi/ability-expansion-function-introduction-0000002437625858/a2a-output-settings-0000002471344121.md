# A2A输出设置

【A2A输出设置】用于绑定卡片输出，最多可添加20条，当进行A2A基础配置后才可进行输出设置。

**输出配置：**

【卡片名称】：响应卡片名称。

【卡片描述】：描述卡片的主要呈现内容。

【输出参数】：默认必须包含参数名为cardData、类型为Object的参数，并且必须添加子项。
