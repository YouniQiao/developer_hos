# 插件升级

**云插件**

智能体中已添加的云插件如果有版本更新，再次进入智能体时，会自动更新插件版本，注意：此处仅测试态智能体更新到插件新版本，需升级智能体，将智能体上架版本更新使用插件新版本。

插件版本更新后，智能体中对应提示：

![](./img/c7c4b5562e23.png "点击放大")

**端插件**

智能体中已添加的端插件若有版本更新，需在智能体编排界面手动移除旧版本并添加新版本端插件。
