# 给小艺对话或智能体发消息

给小艺对话或智能体发消息是指用户在点击某个节点时通知调用方用户的操作以进行后续的处理。

发送的消息可选择固定文本或选中的变量。

![](./img/622a4f06b725.png)

可通过"隐藏消息开关"来更改触发发消息事件时是否将发送的消息显示在对话框中。

![](./img/83ce6de7082a.png)
