# 设置变量值

触发事件时，将变量一栏中选中的变量更改为变量值一栏中的值。变量值一栏内可以选择输入固定内容或选择其他变量。

例如，图中的按钮绑定了变量test1，触发事件时，test1的值将被修改为test2。
