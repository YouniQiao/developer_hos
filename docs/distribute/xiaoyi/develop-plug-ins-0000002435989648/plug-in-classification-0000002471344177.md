# 插件分类

MCP插件：提供能力的是MCP服务，按MCP标准协议调用配置接口获取服务内容。

云插件：提供能力的服务在云侧，需要访问对应的服务端口获取服务内容。

端插件：提供能力的服务在HarmonyOS设备侧，端插件构建智能体与HarmonyOS设备侧应用的交互通路，从设备侧的应用获取信息或操控应用。
