# 运营看板

## 概述

小艺罗盘为开发者提供了智能化的观测功能，可自动采集并统计智能体上线的指标数据，让开发者实时监控智能体的使用情况。各项指标数据可在运营看板中查看。

## 查看统计数据

进入小艺开放平台，点击【小艺罗盘】-【观测】-【运营看板】，选择智能体后即可查看数据面板。
