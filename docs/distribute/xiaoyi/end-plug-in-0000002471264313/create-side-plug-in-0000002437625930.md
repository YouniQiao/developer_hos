# 创建端插件

在小艺开放平台工作空间的插件界面点击新建插件，选择端插件，下拉选择APP包名（仅支持选择本账号在AppGallery Connect中已上架应用的APP包名）。

插件描述尽可能描述清楚，大模型接收到用户的输入后，会根据插件的描述，调用与之匹配的插件。
