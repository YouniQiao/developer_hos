# 知识产权

请确保您的智能体及其内容具备完整权利或充分授权，不得侵犯他人知识产权（包括商标、版权、专利、商业秘密和其他专有权利）或其他合法权益。

1、开发者在智能体内使用他人拥有版权的内容时，应事先获得授权，不得在无授权的情况下使用。

2、未经授权，智能体不得是针对他人版权保护内容所开发的破解版、汉化版。

3、在不构成法定正当使用的情况下，开发者未经授权不得使用他人相同或近似商标，否则将有可能构成商标侵权或不正当竞争。

4、未经授权，开发者不得使用华为公司的商业名称、商标和其他标识。开发者不得虚假或引人误解地宣传与华为公司的关系，如暗示华为公司是应用的提供商或合作商，或者以任何形式表示华为公司认可或担保其质量或功能。

5、开发者不得实施混淆行为，误导用户认为智能体及其内容来源于他人或与他人存在特定联系（如在任何语言下均不得使用与其他开发者提交的智能体相同/相似的外观、名称、主题等）。

6、智能体不得销售或推广仿冒商品。

7、智能体及其内容不得侵犯他人合法权益（包括但不限于肖像权、名誉权），亦不得以鼓动、诱导或帮助等方式，间接实施或参与侵权行为。

8、尊重他人合法权益，不得侵害他人依法享有的知识产权 ，不得危害他人身心健康，不得侵害他人肖像权、名誉权、荣誉权、隐私权和个人信息权益。
