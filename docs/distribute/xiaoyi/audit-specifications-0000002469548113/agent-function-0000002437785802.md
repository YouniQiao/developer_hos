# 智能体功能

1、智能体应具备良好的兼容性，且需适配华为主流终端设备；智能体需能够正常启动返回，不得出现运行时频繁崩溃、闪退、无响应或其他功能异常情况。

2、智能体需具备实用价值，能为用户提供实质功能/服务，且需具备创意，不得为纯信息展示或为手机系统自带的简易功能。

3、智能体不得出现过度营销、诱导跳转等有损用户利益的行为。智能体需确保功能可在智能体内实现，即不得以使用其他应用为前提条件，且不得含有跳转至智能体外的恶意或不明链接，包括但不限于：

* 不得从事纯跳转等恶意导流行为，即不得在并不实际向用户提供特定服务的情况下，仅通过领红包、做任务领权益等形式，诱导用户点击跳转到第三方服务、应用、网页等相关内容或由其他方提供的页面。
* 不得在未实现基础功能、上架功能的情况下，诱导跳转至应用或应用下载界面。
* 不得对智能体或对其他应用实施推荐、推广、排行或集中设立跳转等诱导点击或跳转行为，也不得为上述行为提供任何协助或便利。
* 不得强行中断智能体内的功能或业务的完整流程、要求用户跳转/下载应用后才能继续下一步操作。如：跳转/下载应用后才可解锁相关功能、跳转/下载应用后才可完整体验业务流程、在智能体内有能力提供相关功能或服务时，仍要求跳转/下载应用等。

4、智能体不得提供虚假或具有误导性的功能、信息或声明。

5、智能体不得以任何方式协助用户误导他人或提供用于实现欺骗性行为或违法行为的功能，如生成或协助生成印章和身份信息等；也不得宣传或协助创建文字、图像和/或视频等以传达虚假或误导性的信息。
