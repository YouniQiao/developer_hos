# 智能体安全

1、智能体不得含有病毒、木马，包括但不限于通过可疑代码、文件及程序等形式对系统造成负面影响或侵害用户权益。

2、智能体不得存在劫持系统操作、利用漏洞或采取欺骗手段监控用户、窃取数据的恶意行为，包括但不限于劫持桌面、监听剪切板、设置钓鱼链接盗取用户账户或其他敏感信息。

3、智能体不得允许远程攻击者控制手机，接收攻击者下发的远程控制指令，在用户未授权、未知情的情况下，侵害用户隐私、窃取用户资产或者执行其他恶意行为。

4、智能体不得在用户不知情的情况下利用系统和资源进行获利，包括但不限于加密货币挖矿、通过模拟人工点击广告或链接、下载软件、修改软件业务逻辑进行刷量刷榜等。

5、智能体不得干扰、阻断、屏蔽或以未经授权的方式访问用户的通信功能、网络服务或其它合法业务，导致用户手机无法正常使用，损害用户利益。

6、智能体不得诱导或强制更改系统默认的功能，导致系统功能异常，包括但不限于进程无法停止、功能键失灵（如按“返回键”后无法退出智能体）等。
