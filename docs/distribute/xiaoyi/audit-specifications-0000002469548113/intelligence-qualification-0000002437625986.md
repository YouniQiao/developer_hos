# 智能体资质

**生成式人工智能服务上线备案**

需确认上传的智能体是否接入大模型，若接入需提供生成式人工智能服务上线备案号和算法备案编号。

1、若未接入大模型可忽略。

2、若接入大模型，需提供生成式人工智能服务上线备案号和算法备案编号。

3、若没有相关凭证或编号也可以提交，但需三个月内完成备案。

**各业务具体资质要求**

由于智能体内容包含的业务类型繁多和审核规则不同时期的调整更新，资质说明并未详尽，以最终审核意见为准，详情可见后文附录。
