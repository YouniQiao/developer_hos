# 基于已有意图转换创建插件

插件创建方式选择“端/云插件-基于已有意图转换”。

![](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260212181341.01416390077433322455455340054716:50001231000000:2800:5BD6AAD7550DA53E129872F6594EDEAC843415950E02E3DFA3BC081EB1D6BC52.png)

从已有的意图中选择一个后，APP包名、APP名称和服务类型会自动填充，只需要再补充一个插件描述即可。

![](./img/80f7ca356533.png "点击放大")

端插件创建成功后，添加版本和前面端插件章节中描述的操作一样，配置工具信息、输入和输出参数。

![](./img/75164eb76c01.png "点击放大")
