# 平台优势

1. **丰富的触达入口**

   智能体可以通过丰富的入口触达用户，包括小艺智能体市场、系统导航条、小艺APP、小艺搜索、小艺建议等。
2. **统一的应用智能体协同交互体验**

   全局体验一致，降低用户认知成本。
3. **丰富的智能体生态工具**

   大模型鸿蒙系统端云工具、生态MCP工具、自定义端云工具。
