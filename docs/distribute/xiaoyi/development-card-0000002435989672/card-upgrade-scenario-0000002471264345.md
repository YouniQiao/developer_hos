# 卡片升级场景

如果当前插件绑定的是A卡片的版本1，开发者新建了A卡片的版本2，则在“回复模板配置->我的卡片”中已绑卡界面上会显示“待更新提示”，点击之后会弹出升级提示，升级之后绑定的参数会被清空，需要开发者重新配置保存。
