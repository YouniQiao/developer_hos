# 回复模板配置

**查询插件/工作流输出参数**

在智能体配置页面的左侧面板为能力扩展页面，在此面板的插件和工作流区域可以配置回复模板。

点击插件参数设置按钮，在弹出的窗口中可以看到插件或工作流的输出参数。

![](./img/5e2ff14c9069.png "点击放大")

![](./img/1ef0564342b4.png "点击放大")

**卡片模板配置**

在需要配置回复模板的插件和工作流中，点击卡片模板配置按钮。（执行类型为前台执行和卡片执行的端插件不支持绑卡）

![](./img/b7e80a9cf971.png "点击放大")

![](./img/d7fb50e9a91b.png "点击放大")
