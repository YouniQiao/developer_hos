# 工作流执行设置

对于工作流执行耗时较长，执行完希望自动通知到用户结果的场景，可通过配置异步任务实现。

点击工作流设置，开启【是否包含异步任务】开关，手机端与智能体对话时，工作流执行完成后，若小艺APP不在前台，则自动向用户发送Push消息通知用户执行结果；开关关闭时，工作流执行完成后不发送通知消息；开关默认关闭。
