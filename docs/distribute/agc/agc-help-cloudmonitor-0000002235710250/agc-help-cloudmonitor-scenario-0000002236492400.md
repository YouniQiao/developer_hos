当您开通并使用了AGC云函数、云数据库、APMS等服务时，云监控可以帮助您监测这些服务的运行状况，主要在以下场景帮助您监控资源：

#### 提供云侧监控数据，助力优化改进

为您提供当前账号下云函数、云数据库指标的监测功能。您可以通过查看云函数、云数据库的监控图表，及时获悉资源的运行状况。

#### 日志数据实时分析

为您提供日志数据实时分析、监控图表可视化展示功能。云监控将本地日志通过日志服务进行收集，即可解决企业的监控运维与运营诉求。

#### 及时告警通知异常

云监控根据您设置的告警规则，在监控数据达到告警阈值时向您发送告警通知，帮助您及时获取异常，查询异常原因，并对异常进行处理。
