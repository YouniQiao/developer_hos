当您使用AppGallery Connect遇到问题时，可以按照以下顺序寻求帮助。

1. 请先仔细阅读文档中的说明和注意事项，检查是否满足限制条件。
2. 查看业务对应FAQ。
3. 通过[智能客服](https://developer.huawei.com/consumer/cn/customerService/#/bot-dev-top/faq-top/faq-talk-top)查找问题解决方案。
4. 登录[Stack Overflow社区](https://stackoverflow.com/questions/tagged/appgallery-connect)和[华为开发者社区](https://developer.huawei.com/consumer/cn/forum/block/ag-connect)参与问题讨论。
5. 如果以上方法仍未解决您的问题，可以通过在[互动中心](https://developer.huawei.com/consumer/cn/doc/app/agc-help-interaction-center-0000002276985946#section68459561)反馈问题的形式与我们进行联系。
