云监控日志服务功能可对云函数的日志数据进行实时查询和分析，以图表可视化方式展示分析结果，帮助您实时关注请求变化趋势，判断系统内部健康状态。您可通过设置不同的过滤条件筛选服务和业务运行过程中您关注的日志信息，在“查询结果”区域查看日志详情，评估风险并预测可能出现的问题，提升解决问题的效率。
