1. 登录[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)，选择“APP与元服务”。
2. 在应用列表中点击需要批量修改商品的应用。
3. 在“运营”页签下的左侧导航栏中，选择“产品运营>商品管理”。
4. 下载表格，批量修改商品信息并上传。

   ![](../img/agc-help-revise-goods-0000002391704482_0.png)

   请按照规范批量修改商品信息，否则会在上传时报错，具体错误信息可参见[常见错误说明](/docs/distribute/agc/agc-help-revise-0000002425262569/agc-help-mistakes-0000002425264353#ZH-CN_TOPIC_0000002425264353)。

   * 方法一：

   a）点击页面右上角![](../img/agc-help-revise-goods-0000002391704482_1.png "点击放大")，下载商品模板并保存到本地。

   b）按规范修改商品信息。

   c）点击“批量操作”下拉选项中的 “修改商品”，并在弹出的添加框中点击![](../img/agc-help-revise-goods-0000002391704482_2.png "点击放大")选择上传已填写的商品模板。

   ![](../img/agc-help-revise-goods-0000002391704482_3.png)

   * 方法二：

   a）点击“导出商品”，下载商品信息表格并保存到本地。

   b）按规范修改商品信息。

   c）点击“批量操作”下拉选项中的 “修改商品”，并在弹出的添加框中点击![](../img/agc-help-revise-goods-0000002391704482_4.png "点击放大")选择上传已修改的商品信息表格。

   ![](../img/agc-help-revise-goods-0000002391704482_5.png)
5. 在确认页面可查看商品信息修改前后对比图，点击“确定”。

   ![](../img/agc-help-revise-goods-0000002391704482_6.png "点击放大")

   ![](../img/agc-help-revise-goods-0000002391704482_7.png)

   已通过审核的数字商品，如果仅修改其价格和商品销售范围，则无需重新提交审核，新价格和新的商品销售范围立即生效；如果还修改了其他基本信息或审核信息，则需要再次提交审核，数字商品方可生效。
