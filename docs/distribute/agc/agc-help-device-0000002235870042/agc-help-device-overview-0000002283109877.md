进行调试和指定设备发布时，可以将HarmonyOS应用/元服务安装到指定的设备上。为了保证安全性，待安装的设备需要在AppGallery Connect上进行[注册](/docs/distribute/agc/agc-help-device-0000002235870042/agc-help-add-device-0000002283189937)。每个账号每年最多可以注册100个设备。

当后续不再使用某个已注册的设备时，您可以[将该设备从列表中删除](/docs/distribute/agc/agc-help-device-0000002235870042/agc-help-delete-device-0000002248111074)。如设备在添加后一年内被删除，该设备将置灰显示且不再可用，但其仍会占用这一年的设备数量限额。当设备添加的时间满一年后，被删除的设备会自动被清除，同时也不再占用年度的设备数量限额。

对于添加后一年内被删除的设备，您还可以重新添加回来，重新添加的设备的注册时间依然以首次注册时间为准。
