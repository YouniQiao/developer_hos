#### 为什么无法查看/下载Profile？

该Profile对应的应用可能已进行应用转移，目前不在您的账号下，因此Profile已无法使用，建议您删除该Profile。

#### 修改开放能力后，如何保证旧Profile和新Profile都能正常使用？

如果修改开放能力后，还需继续使用旧的Profile，请重新创建一个新的Profile，新Profile将包含最新的开放能力，切勿去下载旧的Profile。
