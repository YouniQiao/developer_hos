您可以将应用上架至华为应用市场，面向全网用户招募测试用户来测试您的应用。具体测试流程如下：

**第一步：****[创建并发布测试版本](/docs/distribute/agc/agc-help-public-test-0000002287814841/agc-help-publictest-release-app-0000002259033172)**

您需要将测试版本发布至华为应用市场，测试版本正式上架后，全网用户可以在应用市场的测试专区获取测试版本。

**第二步：[用户参与测试](/docs/distribute/agc/agc-help-public-test-0000002287814841/agc-help-publictest-user-test-0000002259136298)**

测试版本上架后，全网用户可以在应用市场的测试专区获取测试版本。当版本安装次数达到您设定的限制数量时，测试专区将不再显示此测试版本。您也可以通过分享邀请链接，让特定用户来测试体验。

**第三步：收集用户反馈**

您可以通过测试版本配置界面的“测试信息 > 测试说明”中预留的联系方式收集用户测试过程中发现的问题，进而针对问题进行优化。

**第四步：（可选）[停止测试](/docs/distribute/agc/agc-help-public-test-0000002287814841/agc-help-publictest-stop-test-0000002293706433)**

若您发现测试版本有问题，需要提前结束测试，您可以停止测试。停止后，新用户将无法安装测试版本，已参与测试的用户仍可以继续测试，直至安装时间超过90天。

**第五步：[测试版本转正式版本](/docs/distribute/agc/agc-help-public-test-0000002287814841/agc-help-publictest-to-formal-version-0000002293633341)**

当您公开测试的版本经过充分测试、已达到正式发布的标准后，您可以直接将此版本转为正式版本，从而免去重复审核的流程，提升您发布的效率。
