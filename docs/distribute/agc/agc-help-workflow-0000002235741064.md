通过AppGallery Connect，您可以向华为应用市场发布您的应用，管理您的应用，使用AppGallery Connect提供的各类服务辅助您开发和测试应用，应用上架后还可以查看应用的报表数据，监测应用的异常和性能等。您也可以通过[AppGallery Connect API](https://developer.huawei.com/consumer/cn/doc/app/agc-help-connect-api-0000002236015554)定制AppGallery Connect提供的能力，实现流程自动化，提高您的工作效率。

* **注册华为账号**

  在使用AppGallery Connect前，您需要[注册华为账号](https://developer.huawei.com/consumer/cn/doc/start/registration-and-verification-0000001053628148)，并进行[实名认证](https://developer.huawei.com/consumer/cn/doc/start/itrna-0000001076878172)，后续就可以使用此账号登录AppGallery Connect。
* **开通商户服务****（可选）**

  如果您的应用会与华为应用市场进行收益分成（付费应用或接入华为应用内支付），您需要[开通商户服务](https://developer.huawei.com/consumer/cn/doc/start/merchant-service-0000001053025967)并签署华为商户服务协议。
* **维护团队账号和分配权限（可选）**

  如果您有一个团队一起开发应用，作为账号持有者，您可以前往“用户与访问”板块添加团队其他人的账号，并为其分配权限，其他成员即可根据其账号被分配的权限，在AppGallery Connect中进行相关操作。详细内容请参见[管理团队账号](https://developer.huawei.com/consumer/cn/doc/app/agc-help-manageaccount-0000002306610129)。
* **创建应用**

  无论您的应用是否构建完成，都可以在AppGallery Connect[创建应用](https://developer.huawei.com/consumer/cn/doc/app/agc-help-app-0000002235710234)，后续在开发过程中，可以使用相关信息，并将开发完成的版本通过AppGallery Connect分发到华为应用市场。
* **测试应用**

  AppGallery Connect提供了多种测试服务，在发布应用前，您可以根据实际需求，选择相应的测试方案。

  + [云调试](https://developer.huawei.com/consumer/cn/doc/app/agc-help-clouddebug-0000002235870046)：使用丰富、高效的云端设备，直观了解应用在不同机型上的运行表现。
  + [云测试](https://developer.huawei.com/consumer/cn/doc/app/agc-help-cloudtest-0000002235710242)：在热门机型上自动进行兼容性测试、稳定性测试、性能测试、功耗测试和UX测试。您可以基于应用上架华为应用市场的标准进行上架测试，也可以根据自己关注的内容，自定义测试项目。
  + [指定设备发布](https://developer.huawei.com/consumer/cn/doc/app/agc-help-internal-test-0000002270709477)：将构建好的应用上传至自己的服务器或者第三方云上，团队成员可以下载到授权设备上进行测试。
  + [邀请测试](https://developer.huawei.com/consumer/cn/doc/app/agc-help-invite-test-0000002270829393)：邀请特定用户群组来测试您的应用。
  + [公开测试](https://developer.huawei.com/consumer/cn/doc/app/agc-help-public-test-0000002287814841)：面向全网公开招募部分用户测试您的应用。
* **发布应用**

  准备好发布包后，就可以准备提交应用发布审核。在提交审核前，您需要提前了解审核规则，准备好提交审核所需的材料。提交审核后，您可以在版本信息界面查看审核进度和结果。如果等待多日仍未完成审核，您可以进行催审；如果提交审核后，发布计划有变，您还可以撤销审核。
* **维护应用**

  当应用上架华为应用市场后，您还可以更新应用信息、升级版本、回退版本。
* **查看应用报表**

  AppGallery Connect提供了多种报表数据，您可以从“分析”板块进入，通过分析各项指标数据，了解应用的使用、销售情况，用户行为等。
* **监测应用异常和性能**

  您可以使用[APMS服务](https://developer.huawei.com/consumer/cn/doc/app/agc-help-apms-0000002235870062)，监测现网应用出现异常和性能问题的情况，从而快速识别、定位和解决问题。
