您可以通过Provisioning API管理证书、Profile以及设备等，相比在AppGallery Connect页面中手动管理提升您的工作效率。

与AppGallery Connect页面操作相同，支持如下API：

* 管理证书：[申请证书](/docs/distribute/agc/agc-help-provision-api-reference-0000002236041494/agc-help-provision-api-apply-cent-0000002236201302)、[查询证书](/docs/distribute/agc/agc-help-provision-api-reference-0000002236041494/agc-help-provision-api-query-cent-0000002271160629)、[删除证书](/docs/distribute/agc/agc-help-provision-api-reference-0000002236041494/agc-help-provision-api-delete-cent-0000002271000685)

* 管理Profile：[申请Profile](/docs/distribute/agc/agc-help-provision-api-reference-0000002236041494/agc-help-provision-api-apply-provision-0000002271000689)、[查询Profile列表](/docs/distribute/agc/agc-help-provision-api-reference-0000002236041494/agc-help-provision-api-query-provision-0000002236041506)、[修改Profile绑定的设备](/docs/distribute/agc/agc-help-provision-api-reference-0000002236041494/agc-help-provision-api-update-provision-0000002469198766)、[删除Profile](/docs/distribute/agc/agc-help-provision-api-reference-0000002236041494/agc-help-provision-api-delete-provision-0000002236201310)
