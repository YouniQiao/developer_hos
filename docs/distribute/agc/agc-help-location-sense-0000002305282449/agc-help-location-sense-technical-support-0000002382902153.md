当您使用近场服务遇到问题时，可以按照以下顺序寻求帮助。

1. 请根据接入场景，先仔细阅读各场景“[接入流程](https://developer.huawei.com/consumer/cn/doc/app/agc-help-petalmap-access-process-0000002407793981)”说明。
2. 查看[FAQ](https://developer.huawei.com/consumer/cn/doc/app/agc-help-location-sense-faq-0000002349181496)。
3. 通过[智能客服](https://developer.huawei.com/consumer/cn/customerService/#/bot-dev-top/faq-top/faq-talk-top)查找问题解决方案。
4. 登录[华为开发者社区](https://developer.huawei.com/consumer/cn/forum/block/ag-connect)参与问题讨论。
