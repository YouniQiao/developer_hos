| 限制项 | 限制 |
| --- | --- |
| 截屏操作 | 当前仅支持HarmonyOS 5.0.0及以上版本的所有真机设备。 |
| 文件管理 | 当前暂不支持文件管理功能。 |
| 应用安装完成后是否支持自动打开 | HAP和APP格式的HarmonyOS应用安装完成后均会自动打开，无需手动操作。 |
| 支持同时调试的设备数 | 单机调试场景下，每次调试仅支持申请1台设备，同一账号最多可同时占用2台设备。 |
| 调试设备权限修改 | 当前暂不支持修改调试设备高危权限相关的系统设置，即不支持进入“设置”界面修改配置。 |
| 应用包类型 | 当前仅支持上传和安装release版本的应用包，暂不支持In-house应用及debug版本的应用包。 |
