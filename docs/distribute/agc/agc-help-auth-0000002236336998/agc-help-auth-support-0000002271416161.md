当您集成AGC相关服务遇到问题时，可以按照以下顺序寻求帮助。

1. 请先仔细阅读文档，部分功能对设备和系统有限制，请参见[使用限制](https://developer.huawei.com/consumer/cn/doc/app/agc-help-auth-restrict-0000002236337002)。
2. 查看[FAQ](https://developer.huawei.com/consumer/cn/doc/app/agc-help-auth-faq-0000002271496205)和[错误码](https://developer.huawei.com/consumer/cn/doc/app/agc-help-auth-api-agcautherror-0000002273781653#section9906111314222)。
3. 通过[智能客服](https://developer.huawei.com/consumer/cn/customerService/#/bot-dev-top/faq-top/faq-talk-top)查找问题解决方案。
4. 登录[Stack Overflow社区](https://stackoverflow.com/questions/tagged/appgallery-connect)和[华为开发者社区](https://developer.huawei.com/consumer/cn/forum/block/serverless)参与问题讨论。
5. 如果以上方法仍未解决您的问题，可以通过[在线工单系统](https://developer.huawei.com/consumer/cn/support/feedback/#/add/89?level2=9016)与我们进行联系。提交工单后，可登录[华为开发者联盟官网](https://developer.huawei.com/consumer/cn/)，鼠标置于右上角头像处，在下拉框内点击“我的工单”查看工单处理进展。
