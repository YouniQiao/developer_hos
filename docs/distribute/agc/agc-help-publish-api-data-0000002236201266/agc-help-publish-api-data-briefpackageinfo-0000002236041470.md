| 参数名称 | 必选(M)/可选(O) | 类型 | 参数说明 |
| --- | --- | --- | --- |
| packageId | O | String | 软件包ID。 |
| versionName | O | String | 版本描述。  对应软件包中的**versionName**。 |
| versionCode | O | Integer(64) | 版本号。  对应软件包中的**versionCode**。 |
| buildVersion | O | String | 构建版本号，用于区分同一主版本下的不同测试子版本。 |
