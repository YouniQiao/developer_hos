| 参数名称 | 必选(M)/可选(O) | 类型 | 参数说明 |
| --- | --- | --- | --- |

| companyAddress | O | String(512) | 商业公司地址。 |
| customContactInfo | O | String(500) | 自定义联系方式。 |
