当您创建POI时，如果提示**该位置已被其他应用使用**，则表示该POI位置已被其他应用发布过。您需发送邮件申请使用该POI，华为方收到邮件后，将在15个工作日内完成审核，并邮件告知您审核结果。

申请邮件格式要求如下：

* **邮箱地址**

  agconnect@huawei.com

  ![](../img/agc-help-apply-publish-poi-0000002382902157_0.png)

  本邮箱仅用于处理位置信息反馈事宜，其他咨询类问题请勿发送至本邮箱。
* **邮件标题**

  【近场服务】-【申请使用已发布的POI】
* **邮件内容**

  需包含如下信息：

  + 应用/元服务名称
  + 应用ID：即APP ID，获取方法请参见[查看应用信息](https://developer.huawei.com/consumer/cn/doc/app/agc-help-view-app-info-0000002282674569)
  + POI信息：包括位置ID（新建POI界面点击“发布”后的弹框中会携带此信息）、位置名称、经纬度等
  + 应用和POI存在关联的相关证明截图
