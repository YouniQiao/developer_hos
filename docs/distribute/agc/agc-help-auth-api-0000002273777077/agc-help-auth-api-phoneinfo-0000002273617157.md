手机号码信息类，用于更新用户当前手机号码信息的操作。

#### Parameters

| Name | Type | Description |
| --- | --- | --- |
| phoneNumber | string | 手机号码。 |
| countryCode | string | 国家码，例如中国：“86”。 |
| verifyCode | string | 手机验证码。 |
| lang | string | 验证码文字语言，例如中文：“zh\_CN”。 |
