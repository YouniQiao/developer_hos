| email | string | Email账号。 |
| verifyCode | string | Email验证码。 |
| lang | string | 验证码文字语言，例如中文：“zh\_CN”。所有支持的语言类型请参考[认证服务验证码语言类型](/docs/distribute/agc/agc-help-appendix-0000002312305161/agc-help-auth-verifycode-langtype-0000002532176920)。 |
