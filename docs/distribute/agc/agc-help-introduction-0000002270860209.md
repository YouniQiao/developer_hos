| AppGallery Connect是华为推出的应用一站式服务平台，致力于为开发者提供应用开发、分发、运营、分析全生命周期服务，构建全场景智慧化的应用生态。  AppGallery Connect深度整合华为内部各项优质服务，将华为在全球化、质量、安全、工程管理等领域长期积累的能力向开发者开放，大幅降低应用开发与运维难度，提高版本质量，开放分发和运营服务，帮助您获得更多用户。 |  |
|---|---|

| 类别 | 服务 | 说明 |
| --- | --- | --- |
| 构建 | [认证服务](/docs/distribute/agc/agc-help-auth-0000002236336998/agc-help-auth-introduction-0000002271496181) | 基于预构建的托管式认证系统，更高效地保护您应用的数据安全。 |
| [Cloud Foundation Kit（云开发服务）](/docs/dev/app-dev/application-services/cloud-foundation-kit-guide/cloudfoundation-introduction) | 为应用提供云函数、云数据库、云存储、预加载等云端服务。应用运行所需的服务器和环境可以皆由云端平台提供，您只需关注应用的业务逻辑，而无需关心基础设施（例如：服务器、操作系统、容器等）。 |
| [AppGallery Connect API](/docs/distribute/agc/agc-help-connect-api-0000002236015554/agc-help-connect-api-introduction-0000002270974725) | 提供了一套RESTful API，通过这些API，您可以按需定制AppGallery Connect提供的服务，也可以实现流程自动化，从而提升您的工作效率。 |
| 增长 | [App Linking](https://developer.huawei.com/consumer/cn/doc/AppGallery-connect-Guides/agc-applinking-harmonyos-next-0000002016462204) | 提供了延迟链接能力、直达应用市场能力等有竞争力的特性，让用户直达您应用的体验更佳。 |
| 质量 | [AppGallery邀请测试](/docs/distribute/agc/agc-help-privacy-appgallery-invite-test-0000002292624409/agc-help-appgallery-invite-test-guide-0000002292711373) | 在正式版本发布之前，您可以邀请特定用户从华为应用市场提前获取您的应用，以收集测试用户的反馈意见，助您提前发现问题，及时修复问题和优化版本体验。 |
| [AppTest邀请测试](/docs/distribute/agc/agc-help-apptest-invite-test-0000002258071220/agc-help-apptest-invite-test-guide-0000002258174270) | 在正式版本发布之前，您可以邀请特定用户从AppTest提前获取您的应用，以收集测试用户的反馈意见，助您提前发现问题，及时修复问题和优化版本体验。 |
| [公开测试](/docs/distribute/agc/agc-help-public-test-0000002287814841/agc-help-public-test-guide-0000002293706429) | 在正式版本发布之前，您可以面向全网公开招募部分用户测试您的应用，以收集测试用户的反馈意见，助您提前发现问题，及时修复问题和优化版本体验。 |
| [指定设备发布](/docs/distribute/agc/agc-help-internal-test-0000002270709477/agc-help-internal-test-guide-0000002295325149) | 您可以将打包的HarmonyOS应用上传至自己的服务器或者第三方云上，团队内成员将应用下载到授权设备上即可进行测试。 |
| [云调试](/docs/distribute/agc/agc-help-clouddebug-0000002235870046/agc-help-clouddebug-introduction-0000002254916514) | 提供丰富、高效的云端设备，可解决您设备机型不足、bug无法复现等问题，您可以随时直观了解应用在不同机型上的运行表现。 |
| [云测试](/docs/distribute/agc/agc-help-cloudtest-0000002235710242/agc-help-cloudtest-introduction-0000002255036400) | 提供一整套完善的自动化真机测试流程，可对您的应用在系统上的兼容性、稳定性、性能、功耗、UX这几大方面进行无需人工干预的自动化测试。 |
| [云监控](/docs/distribute/agc/agc-help-cloudmonitor-0000002235710250/agc-help-cloudmonitor-introduction-0000002236332572) | 帮助您监测现网应用的崩溃（CPP CRASH、JS ERROR）、应用无响应（AppFreeze）等稳定性指标，以及应用的启动、页面加载、耗电等性能指标。 |
| [性能监测（APMS）](/docs/distribute/agc/agc-help-apms-0000002235870062/agc-help-apms-introduction-0000002236333914) | 帮助您监测现网应用的崩溃（CPP CRASH、JS ERROR）、应用无响应（AppFreeze）等稳定性指标，以及应用的启动、页面加载、耗电等性能指标。让您轻松准确快速发现、识别、定位和解决问题。 |
| 分发 | [发布HarmonyOS应用](/docs/distribute/agc/agc-help-release-app-0000002271695230/agc-help-release-app-guide-0000002287176372) | 您可以在AppGallery Connect提交HarmonyOS应用上架申请，审核通过后，用户即可在华为应用市场搜索到您的应用。 |
| [发布HarmonyOS游戏](https://developer.huawei.com/consumer/cn/doc/app/agc-help-release-game-0000002364930906) | 您可以在AppGallery Connect提交HarmonyOS游戏上架申请，审核通过后，用户即可在华为应用市场搜索到您的游戏。 |
| [发布元服务](/docs/distribute/agc/agc-help-release-atomic-0000002327731065/agc-help-release-atomic-guide-0000002293651514) | 您可以在AppGallery Connect提交元服务上架申请，审核通过后，用户即可在华为应用市场，或者通过负一屏搜索到您的元服务。 |
| [非公开发布](/docs/distribute/agc/agc-help-release-0000002235870050/agc-help-non-public-release-0000002278016482) | 当您的应用不适合在华为应用市场公开发布时，可以选择非公开发布，则用户仅可通过链接发现您的应用。 |
| [定向应用发布](/docs/distribute/agc/agc-help-release-0000002235870050/agc-help-private-release-0000002353433008) | 对于不适合面向所有用户公开分发，以及面向特定企业的私有化定制应用，您可以将应用的分发方式设置为“定向应用发布”。定向应用发布的应用不会出现在华为应用市场的任何类别、推荐、排行榜、搜索结果或其他列表中，仅能通过兑换码和链接供用户发现、下载和更新，且每个链接仅支持单次兑换下载。通过控制链接分发数量，实现安全可控分发。 |
| [发布In-house应用](/docs/distribute/agc/agc-help-release-0000002235870050/agc-help-inhouse-0000002281532696) | In-house发布仅适用于在企业内部网络环境中分发专属应用给内部员工的场景，In-house应用不适合在任何公开渠道发布。使用In-house发布，您无需提交应用至华为应用市场，直接将应用发布上传至您的服务器或者第三方云上、内部用户直接下载安装即可，便于您更灵活控制版本发布节奏。 |
| [分阶段发布](/docs/distribute/agc/agc-help-maintain-0000002270829401/agc-help-maintain-phased-release-7day-0000002279615726) | 在当前上架版本为全网发布时，您可以采用分阶段发布（7天内自动更新）的方式进行应用升级。采用此方式发布，您可以按照1%、2%、5%、10%、20%、50%、100%的比例向已在应用市场开启自动更新应用的用户发布更新的版本。通过阶段性的版本更新，您可以快速获取用户对新版本的反馈意见，降低该版本出现问题的风险。 |
| 分析 | [HarmonyOS应用分析](https://developer.huawei.com/consumer/cn/doc/app/agc-help-anaiyze-app-0000002271371749) | 通过分析各项指标数据，您可以了解用户安装和使用应用的情况、数字商品的销售订阅情况以及应用安装异常情况，从而及时调整产品策略，提升用户满意度。 |
| [元服务分析](https://developer.huawei.com/consumer/cn/doc/app/agc-help-anaiyze-atomic-0000002236332568) | 通过分析各项指标数据，您可以了解元服务的使用情况和用户行为，优化元服务功能和体验，提升用户对元服务的使用频率。 |
| 运营 | [元服务交易评价管理](/docs/distribute/agc/agc-help-users-operation-0000002338612342/agc-help-transaction-rating-0000002338669134) | 在元服务中产生有效交易的用户可以对交易体验进行打分、评论，您可以查看、回复评论，从而了解用户需求，优化服务体验。 |
