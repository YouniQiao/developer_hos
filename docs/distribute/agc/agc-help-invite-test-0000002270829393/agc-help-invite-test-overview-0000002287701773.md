在正式版本发布之前，您可以选择特定用户群组来测试您的应用，以收集测试用户的反馈意见，助您提前发现问题，及时修复和优化版本体验。

邀请测试和其他测试方式区别如下，您可以根据实际业务需求，选择合适的测试方式。

| 区别 | [AppGallery邀请测试](https://developer.huawei.com/consumer/cn/doc/app/agc-help-privacy-appgallery-invite-test-0000002292624409) | [AppTest邀请测试](https://developer.huawei.com/consumer/cn/doc/app/agc-help-apptest-invite-test-0000002258071220) | [公开测试](https://developer.huawei.com/consumer/cn/doc/app/agc-help-public-test-0000002287814841) | [指定设备发布](https://developer.huawei.com/consumer/cn/doc/app/agc-help-internal-test-0000002270709477) |
| --- | --- | --- | --- | --- |
| 面向对象 | 可邀请友好、信任的小范围用户 | 可邀请友好、信任的小范围用户和开发团队内部成员测试 | 面向全网所有用户公开招募测试 | 邀请开发团队内部成员测试 |
| 参与人数 | 上限10000人 | 外部测试用户上限10000人  内部测试用户上限100人 | 下载安装次数上限1000万次 | 安装设备上限100台 |
| 支持设备类型 | 手机、平板、PC/2in1 | 手机、平板、PC/2in1 | 手机、平板、PC/2in1 | 手机、平板、PC/2in1 |
| 支持同时测试的版本数 | 100个 | 100个 | 1个 | 由开发者部署决定 |
| 发布AppGallery测试专区 | 默认 | 否 | 可选 | 否 |
| 发布AppTest | 否 | 默认 | 否 | 否 |
| 是否支持分享链接 | 支持，可拼接邀请码链接 | 支持，公开链接无需拼接邀请码，直接可用 | 支持 | 支持 |
| 支持的应用类型 | HarmonyOS NEXT应用/元服务 | HarmonyOS NEXT应用/元服务 | HarmonyOS NEXT应用/元服务 | HarmonyOS NEXT应用 |
| 支持的国家/地区 | 支持分发中国大陆地区 | 仅支持分发中国大陆地区 | 仅支持分发中国大陆地区 | 仅支持分发中国大陆地区 |
| 是否支持极速审核 | 否 | 内部测试版本支持极速审核 | 否 | 不涉及 |
| 是否支持兑换邀请码 | 否 | 是 | 否 | 否 |
