数字商品类型分为消耗型、非消耗型、自动续期订阅、非续期订阅。

首次提审或追加新类型的数字商品时，必须同时提交新的小游戏版本，不能单独提交商品审核。

例如，老版本小游戏内仅有“消耗型”数字商品，新版本小游戏内追加了“自动续期订阅”数字商品，要求再次提审小游戏版本，并关联本次新增类型下的商品。

![](../img/agc-help-release-minigame-goods-0000002424923350_0.png)

* 若在已通过审核的商品类型下追加新的数字商品，可以单独在商品管理页面提交审核，具体操作请参见[提交已生效类型的数字商品](/docs/dev/game-dev/games-center-effective-digital-products-for-review-0000002320646005)。
* 若有多个商品共用计费点的情况（按档位计费），需要进行备注。

#### 前提条件

已根据[数字商品](/docs/dev/game-dev/games-center-products-manage-0000002286057092)，提前追加创建新类型的数字商品，且商品状态为“待审核”。

#### 操作步骤

登录[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)，点击“APP与元服务”，选择待上架的小游戏。左侧导航栏选择“应用上架 > 版本信息”，右侧页面进入“应用内商品”区域，点击“添加商品”选择待审核的商品类型。

![](../img/agc-help-release-minigame-goods-0000002424923350_1.png)
