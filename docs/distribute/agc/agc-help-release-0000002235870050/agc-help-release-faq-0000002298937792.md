#### 如何查看应用上架状态？应用上架状态有哪些？

登录[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)，点击“APP与元服务”，在应用列表找到您的应用，即可在“状态”列查看应用上架状态。

应用上架状态包含如下几种：

* 准备提交：表示应用已创建，但未提交审核。
* 正在等待审核：表示应用市场已收到您提交的版本，但尚未开始审核。
* 正在审核：表示应用市场正在审核您提交的版本。
* 待修改：表示提交审核的应用被驳回，需按照审核意见修改后重新提交审核。
* 待上架：表示应用已通过审核，但尚未达到您指定的上架时间。
* 已上架：表示应用已通过审核，成功上架应用市场。
* 已撤销上架：表示待审核的应用被您撤销审核。
* 下架处理中：表示应用已被您申请下架，华为应用市场正在受理。
* 被下架：表示应用已被华为应用市场下架。
* 被开发者下架：表示您的应用下架申请审核通过，应用成功下架应用市场。

#### 使用官方提供的应用加密能力后，我的应用代码就安全了吗？还需要做其它的代码保护措施吗？

应用加密特性可以有效提高应用代码文件逆向分析的难度，但应用代码防逆向是一个持续攻防对抗的过程，如对代码文件保护有更高的要求，需要结合其他安全加固措施，进一步提高逆向分析应用的难度。比如使能混淆能力、使能三方安全加固能力等。

#### 如果不使用应用加密能力，还能采取什么手段来保护我的代码资产？

使能混淆能力、使能三方安全加固能力等，可以参考：[应用资产保护设计](https://developer.huawei.com/consumer/cn/doc/best-practices/bpta-app-asset-protection-design#section24320461547)。

#### 应用加密会对我的应用包产生什么影响？

加密后的应用在程序启动和运行过程中可能会小幅度增加性能开销；加密后的应用相比于不加密的应用体积更大，可能会小幅度增加下载和安装时间。

#### 应用加密对应用的保护是侵入式的吗？对应用的启动、运行期间的性能会有影响吗？

加密后的应用在程序启动和运行过程中可能会小幅度增加性能开销；加密后的应用相比于不加密的应用体积更大，可能会小幅度增加下载和安装时间。

#### HAP包和APP包有什么区别，上架包是什么格式的？

HAP是用于本地调试的，APP包是用于上架发布的。根据不同的设备类型，一个APP包可以包含多个HAP包。

#### APP包能本地安装吗？

不能，APP包只能用于发布，即便使用调试证书和调试Profile进行签名，也无法安装。

#### 元服务软件包大小有什么要求？

* FA模型元服务要求如下：

  单个HAP包不得超过10MB。
* Stage模型元服务应用要求如下：
  + 同一设备类型下所有包大小总和不得超过10MB。
  + 一个APP包中任意类型的单个包大小都不超过2MB。
  + 单个包加上其采用dependency方式依赖的[动态共享包（Harmony Shared Package, HSP）](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-hsp)，总大小不超过2MB。

    ![](../img/agc-help-release-faq-0000002298937792_0.png)

    **若一个HSP包同时被entry包和分包依赖，则该HSP包仅计入entry包大小，不再重复计入分包大小。**例如，元服务包含entry包和分包1，entry包依赖HSP包1和HSP包2，而分包1依赖HSP包1和HSP包3，则：entry包大小 = entry包本身 + 依赖的HSP包1 + 依赖的HSP包2；分包1大小 = 分包1本身 + 依赖的HSP包3。
* Stage模型元服务小游戏要求如下：
  + 同一设备类型下所有包大小总和不超过25MB。
  + 主包体大小不超过5MB。

#### MAC压缩包自动生成的系统文件，是否需要手动清理？

MAC系统在生成压缩包时会自动添加一些系统文件，这些文件可能无法通过安全检测。当前AGC已支持自动清理这些系统文件，您无需手动删除即可完成上传。
