若您发现测试版本存在严重问题、或者已完成测试，可以在“版本列表”页面手动停止测试。测试版本停止测试后，状态会立即变为“已失效”。测试版本失效后，新用户将不再能下载安装测试版本，已安装的用户仍然可以继续使用，直到安装时间超过90天。

![](../img/agc-help-appgallery-stop-test-0000002292711381_0.png "点击放大")

用户安装到本地的邀请测试版本，在安装后最多可使用90天（不管期间测试任务是否停止），90天后应用将无法启动。

![](../img/agc-help-appgallery-stop-test-0000002292711381_1.png "点击放大")
