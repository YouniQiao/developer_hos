AppGallery Connect为您提供的服务分为免费和收费两种类别：

* 免费服务：用户不需要开通结算账户，也不需要签署付费服务协议就能使用的服务。
* 收费服务：使用收费服务需要开通结算账户并签署付费服务协议。收费服务通常会提供一定免费配额，您只需为超额使用的部分付费。

  下表列举了收费服务的常见订阅场景，您可根据实际需求进行操作。

  | 场景 | 说明 |
  | --- | --- |
  | [免费试用收费服务](https://developer.huawei.com/consumer/cn/doc/app/agc-help-free-trial-0000002277923061) | 收费服务通常会提供一定免费配额，供您体验试用。 |
  | [查看免费配额的使用情况](https://developer.huawei.com/consumer/cn/doc/app/agc-help-check-quota-0000002278002937) | 免费试用时，您可以在AppGallery Connect查询免费配额的使用情况，避免因配额用尽导致业务中断。 |
  | [升级到按量付费档套餐](https://developer.huawei.com/consumer/cn/doc/app/agc-help-upgrade-plan-0000002242804052) | 在计费周期内的免费配额即将用尽时，您可以选择升级到按量付费档套餐，以继续使用服务。或者，您也可以等到下个计费周期再使用服务，在此之前服务将不再可用。 |
  | [降级到免费档套餐](https://developer.huawei.com/consumer/cn/doc/app/agc-help-downgrade-plan-0000002247204812) | 升级到按量付费档套餐后，您也可以选择降级，将付费档关闭，仅使用免费配额。 |
  | [查看套餐详情](https://developer.huawei.com/consumer/cn/doc/development/AppGallery-connect-Guides/agc-service-billing-0000001058092595#section1854555510325) | 您可以在AppGallery Connect查看当前项目下订购的所有免费档和按量付费档套餐。 |
