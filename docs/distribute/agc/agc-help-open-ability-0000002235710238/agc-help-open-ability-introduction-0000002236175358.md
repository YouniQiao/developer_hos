除了使用HarmonyOS SDK提供的开发能力[开发HarmonyOS应用](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/application-dev-guide)/[开发元服务](https://developer.huawei.com/consumer/cn/doc/atomic-guides/atomic-development-journey)外，AppGallery Connect也开放了一些能力助力您降低开发难度，提升开发效率。您可以根据实际需求选择使用。

| 服务 | 说明 |
| --- | --- |
| [AppGallery Connect API](https://developer.huawei.com/consumer/cn/doc/app/agc-help-connect-api-introduction-0000002270974725) | AppGallery Connect API是一套RESTful API，通过这些API，您可以定制AppGallery Connect提供的服务，也可以实现流程自动化，从而提升您的工作效率。 |
| [认证服务](https://developer.huawei.com/consumer/cn/doc/app/agc-help-auth-introduction-0000002271496181) | 认证服务可以为您的应用快速构建安全可靠的用户认证系统，支持手机、邮箱、华为账号、自有账号等多种认证方式。 |
