#### 通用FAQ

#### [h2]元服务分析的所有报表什么时候刷新？

报表数据一般会在数据产生日次日上午10点左右刷新（例如，7日的数据将在8日更新），偶尔刷新时间会略有延迟，如遇异常情况，我们会提前通知您。
