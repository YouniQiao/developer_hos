|  |  |  |
| --- | --- | --- |
| **序号** | **步骤** | **详情** |
| 1 | [申请开通权限](https://developer.huawei.com/consumer/cn/doc/app/agc-help-location-sense-apply-permission-0000002382902149) | 当前近场服务处于灰度开放阶段，使用服务前需先申请开通近场服务权限。如已开通，可跳过本步骤。 |
| 2 | [元服务嵌入式运行开发](https://developer.huawei.com/consumer/cn/doc/atomic-guides/reusable-runtime-state-develop#section132994471191) | 元服务需支持在花瓣地图中嵌入式运行，页面顶部需避让系统状态栏和元服务导航按钮，避免重叠。 |
| 3 | [元服务嵌入式运行测试](https://developer.huawei.com/consumer/cn/doc/app/agc-help-atomic-embedded-running-test-0000002374154440) | 完成嵌入式运行开发后，需发送邮件申请，由华为侧协助完成测试验证。 |
| 4 | [创建全网态POI](https://developer.huawei.com/consumer/cn/doc/app/agc-help-petalmap-create-formalstate-poi-0000002407634113) | 创建需要全网的POI位置并完成发布。 |
| 5 | [创建全网态服务](https://developer.huawei.com/consumer/cn/doc/app/agc-help-petalmap-create-formalstate-service-0000002373994560) | 元服务嵌入式运行测试通过后，即可创建全网态近场服务并发布上线。 |
