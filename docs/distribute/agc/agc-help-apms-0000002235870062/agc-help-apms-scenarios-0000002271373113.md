#### 定位应用/游戏的闪退问题

应用质量对于用户体验至关重要，如果应用发生闪退等问题，可能直接导致应用评分下降。当应用发生崩溃事件时，利用异常管理模块提供的丰富的崩溃问题的详细信息进行处理，可以帮助您分析问题根因，快速修复，提升应用质量和用户体验。

![](../img/agc-help-apms-scenarios-0000002271373113_0.png "点击放大")

#### 采集应用性能数据，助力优化改进

APMS服务能自动采集关键性能数据，通过多个维度呈现，支持查看和分析性能数据，快速发现应用性能瓶颈。开发者可以根据呈现出的性能数据，实时全面掌握应用在线的性能表现，持续进行性能优化改进，提升应用的用户体验。
