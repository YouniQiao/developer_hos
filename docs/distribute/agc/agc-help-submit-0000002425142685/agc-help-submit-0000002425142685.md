在 AppGallery Connect 中创建数字商品后，便可将其提交至审核。为应用提交首个数字商品或新增数字商品类型时，必须同时提交新的应用版本。提交数字商品至审核前，请确保其处于“待提交”状态；否则，请先完善所有缺失信息。

您可以按下方两种场景类型进行提交审核操作：

![](../img/agc-help-submit-0000002425142685_0.png "点击放大")

* **[前提条件](https://developer.huawei.com/consumer/cn/doc/app/agc-help-submit-pre-0000002425264345)**
* **[首次提交数字商品或新增数字商品类型](https://developer.huawei.com/consumer/cn/doc/app/agc-help-submit-first-products-0000002391544606)**
* **[提交已生效类型的数字商品](https://developer.huawei.com/consumer/cn/doc/app/agc-help-submit-effective-products-0000002425144497)**
