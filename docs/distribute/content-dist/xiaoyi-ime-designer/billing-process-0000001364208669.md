# 结算流程

![](./img/dd715a991173.png)

<strong>1.账单出具时间</strong>

接入华为支付的应用（含游戏）才涉及到结算，结算周期为月度，华为将在每个结算周期的第15天提供您上一个日历月应得收入的结算数据（“结算单”）。

<strong>2.结算步骤</strong>

（1）进入自助结算页面

登录开发者联盟，进入“管理中心”，单击“我的账户”，选择“收益”，进入自助结算页面查看结算单并确认结算单。

根据已签订的合作协议，合作伙伴可以在结算单出具后15个自然日内同华为对账并发起争议，如在15个自然日内未发起争议申请则视同无异议，系统自动进入支付处理流程（合作伙伴仍需提供发票）。

（2）核对结算单

单击下载“结算单”或者“对账明细”，核对本月结算数据。

（3）确认结算单

如您对结算数据无异议，单击“确认结算单”提交结算申请，核对申付金额及收款银行等信息无误后，单击“提交”，提交后将无法取消。如需多个结算单合并开票，请确认同一个合同、同一业务类型相同的结算单方可进行合并开票。

（4）打印结算单

单击“下载结算单”按钮，企业开发者需打印结算单加盖印章（公章或财务章），个人需提供签字结算单，联同发票一同寄给华为（邮寄地址在结算单上有说明）。

合并开票要逐个下载结算单。

<strong>3.邮寄发票</strong>

（1）开具发票

按照确认的结算单金额开具发票，合并开票需将所选结算单中的“结算汇总信息-结算金额”加总，注意金额与发票金额要完全一致，不能有一分钱差异。

（2）开票信息

对于企业开发者，如您是一般纳税人，您需开具增值税专用发票，不接受其他类型的发票。如您是小规模纳税人请在税局代开增值税专用发票。

如您是个人开发者，需前往税务局申请代开增值税发票，增值税发票的规则声明。

（3）单据审核及付款

若您的付款单据被审核通过，结算页面的结算单状态会变为“付款中”（发起结算单付款）。

华为会收到合格有效的付款单据后30个自然日内付款。 若遇节假日可能顺延。付款完成后，结算单状态会变为“付款成功”。

（4）完成结算

以发票签收时间起计算付款赎期，华为公司会在合同约定的付款赎期及时完成结算款项支付。

（5）中国开发者海外分发业务

若您有应用或游戏分发到海外，您只需要按2.1，2.2，2.3操作即可，即点击“确认结算单”即可触发付款流程，无需开票。海外结算单请勿与国内结算单合并确认结算单。

（6） 其他注意事项

* 需要提交如下纸质材料，华为才会对您支付结算款：

与申付金额相等的增值税发票（原件）；

个税完税凭证复印件（如涉及）；

签字的结算单。

以上材料一并寄回华为（邮寄地址详见结算单）。

* 所有账单数据以开发者联盟出具数据为准，因涉及到付费用户的多次下载及运营活动等，不以客户端数据作为结算依据。
* 个人需要开具发票，打款时华为会代扣代缴个人所得税（如个人在税局代开增值税发票时，被扣缴了个税，请将个税完税证明复印件提供给华为，华为不再重复扣缴个税）。
* 上月账单一般在15号前出具（如遇节假日可能稍晚）。
* 当月15号之前请将上月结算数据提交结算，累计几个月同时申请结算可能会导致多交税，由此产生的损失由开发者自行承担。
