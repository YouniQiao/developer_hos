# FAQ

问题一： 设计师平台是什么？

设计师平台是小艺输入法为原创设计师提供一个公平、开放、创新的创作平台。设计师可以在这里上传原创的表情和语录作品，由小艺输入法后台审核通过后的创意内容可在小艺商店内上架展示，获得用户关注，并且可以获得一部分的创作收入。

问题二： 设计师平台入口在哪里？

入口1：进入开发者联盟官网-首页-产品-分发-华为内容分发-小艺输入法 由导航进入到小艺输入法的设计师平台后，会先来到一个设计师平台的落地介绍页。

入口2：在开发者联盟-管理中心-生态服务-内容服务-上架小艺输入法设计师平台服务卡片，设计师由服务卡片进入设计师平台不会遇到此落地介绍页。

问题三：如何修改个人信息？

设计师平台首页-个人中心-编辑个人信息。

问题四：设计师资格需要审核么，标准是什么？

不用，开发者联盟审核通过即可直接成为小艺输入法设计师平台作者。

问题五：个人信息图片尺寸要求是什么？

头像：PNG/JPG格式，文件大小不超过4M，画幅比1:1。

赞赏和致谢图片：JPG格式，尺寸366dp\*144dp，大小不超过300kb。

问题六：作品审核时间要多久？

3-5个工作日左右。

问题七：审核人是谁？

输入法运营人员后台审核。

问题八：已上架的作品支持在线编辑么？

不支持在线修改内容，设计师可申请下架作品后，修改作品并重新提交上架。

问题九：处于审核流程中的作品支持下架么？

不支持。

问题十：设计师收益在哪里查看？

收入报表。
