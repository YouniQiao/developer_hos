# 违规及侵权处理措施

首次出现侵权或违规情况的，平台将对其内容做删除或下架处理；如多次出现或情节严重的，平台有权根据其违规及侵权情况给华为造成的影响，采取包括但不限于下架内容、一定期限内或永久地限制功能、永久封号等措施。
