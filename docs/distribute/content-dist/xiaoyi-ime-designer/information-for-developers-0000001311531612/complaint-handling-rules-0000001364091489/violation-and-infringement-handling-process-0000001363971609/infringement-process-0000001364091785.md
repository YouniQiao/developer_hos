# 侵权处理流程

<strong>违规投诉通知</strong>

如您发现由设计师上传的内容出现本规则第2条所列举的违规类内容的，您可以向小艺输入法运营团队进行投诉，投诉通知包括但不限于以下内容：

（1）投诉方信息

| 投诉方信息 | 说明 |
| --- | --- |
| 用户 | 姓名，有效证件，证件号，通讯地址，电话和email等（非必填项，为了保护非利益相关投诉人的隐私，联系方式仅作为投诉进展通报，不对外公开）。 |
| 设计师 | 设计师姓名（公司名称），营业执照（单位）或身份证（个人），证件号，联系人，通讯地址，电话和email等。 |
| 代理人（如有） | 代理人姓名，律所名称，有效证件，证件号，通讯地址，电话和email等。 |

（2）违规内容信息

包括设计师名称、作品名称和侵权对象图片呈现等。

（3）投诉违规事实

被投诉内容属于哪种违规类型的客观描述。

（4）投诉违规证明材料

| 证明材料 | 说明 |
| --- | --- |
| 主体证明 | 如营业执照、组织机构证明、身份证、如授权他人投诉,还需提供授权委托书。 |
| 违规证明 | 如被投诉作品与投诉人作品的对比截图（附比较说明）、内容宣传、相关纠纷的行政、司法机关的受理通知书、处理决定、判决书等。 |

判断投诉通知有效性及内容是否违规

小艺输入法运营团队收到符合要求的投诉通知后，会尽快对被投诉内容是否违规进行核查。如核查属实，小艺输入法运营团队将对违规内容进行隐藏并要求设计师整改或者直接予以删除。
