<strong>图片素材概述</strong>

<strong>文案填写规范</strong>

文字信息尽量使用中文，必要时可使用英文，避免出现生僻字且不能使用表情符号和特殊字。

* <strong>作者信息</strong>

1. 名称

   不超过20个字，不能使用标点符号。
2. 简介

   不超过100个字。

   * <strong>表情包信息</strong>
3. 名称

   不超过30个字，9个字以内显示效果最佳，不能使用标点符号和空格。
4. 一句话简介

   不超过15个字，尽量避免包含标点符号，避免和表情包名称重复，能够展示出表情的故事或特点。
5. 标签

   1-5个标签，每个标签不超过10个字，不能使用标点符号和空格。

   * <strong>表情图信息</strong>
6. 名称

   不超过5个字，不能使用标点符号或空格。
7. 标签

1-5个标签，每个标签不超过10个字，不能使用标点符号和空格。
