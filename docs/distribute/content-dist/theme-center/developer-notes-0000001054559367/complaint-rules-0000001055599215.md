# 投诉处理规则

## 1. 目的

为维护市场公正，华为主题十分重视内容的版权、肖像权、商标权、隐私和专利等权益问题，所有在线内容不得侵权或违规。如有侵权或违规问题，将按照本规则予以严肃处理。

## 2. 投诉渠道、证据材料及处理流程

### 2.1 投诉渠道

工作时间：周一至周五 08：30-12：00，13：30-18：00（法定节假日除外）

反馈渠道： hwthemes@huawei.com

### 2.2 投诉通知

如您发现由设计师在华为主题上传的内容侵犯了您的合法权益，请填写并反馈[《华为主题平台侵权投诉通知书模板》](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250620110736.63297867248784438756696239112028%3A50001231000000%3A2800%3A8361E89A6C84C8629ABF5588F54A7FC2B820A874693B25152D579C9C1D7595BA.docx?needInitFileName=true)，通知书建议包含以下内容：

2.2.1 投诉方信息

|  |  |
| --- | --- |
| 投诉方类型 | 建议材料 |
| 自然人 | 有效身份证明文件 |
| 法人和其他组织 | 企业营业执照、组织的相应执照 |
| 代理人（如有） | 代理人出具的代理委托书、代理人有效证件 |

2.2.2 侵权信息

填写准确定位投诉对象的信息（如开发者昵称、作品名称、具体侵权画面信息等）

2.2.3 侵权理由

具体投诉的权利类型（如商标权、著作权、专利权），并简要说明侵权理由。

2.2.5 投诉诉求

建议反馈您希望华为主题平台对被投诉内容采取的处理措施，如整改、下架等。

### 2.3 投诉有效性判断

在收到符合要求的《华为主题平台侵权投诉通知书模板》后，华为主题平台会依据相关法律法规的规定，将投诉内容转发给被投诉方，并初步核实投诉是否属实。

### 2.4 投诉反通知

平台用户在收到投诉通知后，若不认可侵权，应在3个工作日内（“投诉反馈时限”）向华为主题平台提交[《华为主题平台侵权投诉反通知书模板》](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250620110736.72016018350344296968857568207118%3A50001231000000%3A2800%3A925C21A1C781D0270D0B25051334CB77676E86981586D98607177D9295B6B78D.docx?needInitFileName=true)，反通知书建议包含以下内容：

| 代理人（如有） | 代理人出具的代理委托书、代理人有效证件 |

2.4.2 侵权信息

填写准确定位被投诉对象的信息（如开发者昵称、作品名称、具体侵权画面信息等）

2.4.3 不认可投诉的理由

### 2.5 投诉处理

1.依据构成侵权的初步证据和服务类型，华为主题平台将及时采取必要措施。在相关法律法规的要求下，华为主题平台将有可能维持或终止所采取的措施（视侵权情况、投诉方的投诉与起诉情况等）。

2.如需通过法律途径，华为主题平台会配合国家机关的取证工作，争议内容的最终处置以双方协商结论或者国家机关的决定或判决为准。

3.根据有关主管部门和人民法院的处理结论，或者双方达成的其他一致结果，投诉方和被投诉方均可据此对华为主题平台的投诉处理结果进行申诉，申诉依据包括但不限于胜诉判决、和解协议、谅解书等。

4.对于平台用户多次、故意侵害知识产权的，华为主题平台有权采取终止交易和服务的措施。

## 3. 温馨提示

1.为了确保投诉的真实性和有效性，投诉举证材料和不侵权声明举证材料中涉及的相关资质证明，原则上应提供原件（扫描件），不能提供原件的，应提供权利人签字盖章的复印件。

2.若投诉方已就侵权内容向行政部门投诉或向法院起诉被投诉方，建议将相关受理证明、提交行政机关或法院的证据材料以及行政机关或法院的决定或判决一同提交给华为主题平台。

3.投诉通知书、反通知书及所附证明材料，将有可能被依法转发给投诉方或被投诉方，建议提前对相关信息进行脱敏处理。
