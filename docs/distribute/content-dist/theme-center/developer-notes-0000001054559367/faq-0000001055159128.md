# FAQ

<strong>[问题一]</strong> <strong>提交设计师资格审核申请文件时总提示上传文件类型非法，这个要怎么解决？</strong>

设计师资格认证申请zip文件包里只能有jpg、png、mp4格式文件。请检查zip包里的文件格式是否符合规范。如果仍然无法解决问题，通过联盟首页-“支持”-“技术支持”-“在线提单”进入工单系统，1-2个工作日内可获得客服回复。

<strong>[问题二]设计师资格认证申请被驳回，怎么办？</strong>

驳回都注明了原因，请参考驳回反馈意见改进作品，并遵守申请的时间间隔，请勿短期内重复提交申请。

<strong>[问题三]主题审核需要多久呢？</strong>

主题的审核周期一般为10个工作日。特殊情况下，根据审核过程中出现的问题适当延长审核周期。详见内容上架规范。

<strong>[问题四]主题审核不通过，请问在哪里可以看审核意见？</strong>

登录开发者联盟后台-进入主题中心后，查看上传的主题，点击主题名称就可以看到审核不通过的原因。

<strong>[问题五]资源定价错误如何解决？</strong>

目前，主题资源支持上架后修改定价，除主题资源外，其它资源上架后无法再修改价格；如果资源还未上架，可自行撤回或通过在线提单联系工作人员驳回。

<strong>[问题六]已上架的主题/壁纸/字体可以下架吗？</strong>

目前来说，同一华为账号下（在不同华为手机登录），购买/下载的主题、字体都是可以永久使用的。因此，付费作品不能下架，请在开发者联盟-主题中心处申请隐藏。

<strong>[问题七]做好的手机主题怎么测试？我的手机是9.0的版本只能测试9.0的主题。那10.0的主题怎么测试，有什么方法吗？</strong>

制作好的主题可以用对应系统版本的真机进行测试，如EMUI9.0版本的主题可以用EMUI9.0系统的真机进行测试。手机系统与主题版本不同则无法测试，但是制作主题时用制作工具可以预览主题效果。

<strong>[问题八]第三方图标如何命名？</strong>

三方图标按照包名命名放在icons文件夹下就能生效。手机可以装一个包名查看器的app查询所需应用的包名。

<strong>[问题九]我单月的收入比较少，可以几个月的账单合并结算吗？</strong>

可以的，如果金额较少，您可以选择几个月的订单合并结算，但最好不要超过三个月。累计几个月同时申请结算可能会导致多交税，由此产生的损失由开发者自行承担。

<strong>[问题十]我有其它问题应该去哪里问呢？</strong>

通过联盟首页-“支持”-“技术支持”-“在线提单”进入工单系统，1-2个工作日内可获得客服回复。
