# 杂志锁屏合作伙伴招募

## 业务介绍

华为杂志锁屏，以高科技赋予时尚新突破，实现锁屏与杂志的完美结合，为全球亿万用户提供震撼创意的视觉盛宴。一触之下即可直达图文资讯，第一时间捕捉热点信息，顷刻拉近世界，锁定精彩瞬间。

## 合作要求

1. 合作范围：杂志锁屏业务合作范围包括杂志锁屏的锁屏壁纸，以及与锁屏壁纸对应落地页的图文内容。
2. 合作模式：合作伙伴需每周稳定供应锁屏壁纸及图文内容，华为审核通过后上线华为杂志锁屏，用户点击后即进入图文内容落地页。锁屏壁纸及图文内容保存于华为服务器中。
3. 对接要求：合作伙伴需通过对接华为杂志锁屏API接口持续稳定地推送更新锁屏壁纸及图文内容，能根据市场变化和用户需求配合华为灵活调整内容。
4. 内容要求：参照[《杂志锁屏素材审核规范》](/docs/distribute/content-dist/theme-center/zazhisuoping-0000001206441666/zazhisuoping-scsh-0000001318654366)。

## 商务模式

华为与合作伙伴以广告收入分成的形式合作。

## 合作流程

有合作意向且具备上述能力的合作伙伴可以通过邮件申请的形式寻求合作。

合作流程如下：

1. 公司介绍：邮件发送公司介绍、合作项目运营能力、图库资源及成功案例介绍等信息至主题合作邮箱（hwthemes@huawei.com）。
2. 资格审核：成功提交邮件申请后，华为将在15个工作日内完成审核。
3. 合作洽谈：在审核通过后，华为工作人员会主动联系并跟进后续流程。
