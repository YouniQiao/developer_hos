# 1.0.27版本功能介绍（2022-06-10）

## 1. 版本更新特性

* [336\*480表盘上传校验AOD](#section676791153718)
* [联盟屏蔽三方分成作品分发到海外销售](#section719110281371)

## 2. 336\*480表盘上传校验AOD

336\*480表盘支持在联盟上传，必须制作表盘AOD。

## 3. 联盟屏蔽三方分成作品分发到海外销售

联盟的三方分成场景中，仅支持国内设计师配置三方分成作品，分发到国内地区。
