# 主题联盟校验规则更新说明

以下是主题联盟校验的新增规则：

1. <strong>APK规范</strong>

为了改善性能问题，不允许上传或更新任何包含apk格式文件的资源。

2. <strong>图标规范</strong>

因规范变动，主题包里从EMUI 10.1版本，icons图标不允许上传“com.huawei.android.hwouc.png”和“huawei.com.android.manager.png”两个图标。

3. <strong>表盘规范</strong>
   1. 为了避免表盘应用黑屏，表盘包版本为HWHD01时，版本号不能为3.2；版本为HWHD02时，版本号不能为2.2。
   2. Yoda表盘上传时，校验preview目录下是否含有aod.jpg这张图。

4. <strong>天气/日历xml脚本校验</strong>

为了避免锁屏不能正常拉起天气/日历APP，xml脚本规范可参考：

[天气数据变量集成&lt;Weather&gt;](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/data-open1-0000001694307045/weather-0000001079515110)

[日历数据变量集成&lt;Calendar&gt;](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/data-open1-0000001694307045/calendar-0000001126131857)

5. <strong>手机主题版本号规范</strong>

只允许上传EMUI 10.1的主题，主题包版本号为10.0.X（X为100及以上，阿拉伯数字顺序递增）。

只允许上传EMUI 11.0的主题，主题包版本号为11.0.X（X为0-99，阿拉伯数字顺序递增）。

且上传EMUI 10.1版本手机主题时需要同时上传EMUI 11.0版本的手机主题。

6. <strong>AOD规范</strong>

静态AOD不允许升级为动态，静态AOD升级为动态则无法上传成功。如需上架动态AOD，请制作完成后重新上传。

7. <strong>预览视频校验</strong>

| 资源类型 | 大小限制 |
| --- | --- |
| 手机主题 | ≤10M |
| AOD | ≤10M |
| 表盘 | ≤5M |
| 动态壁纸 | ≤20M |
