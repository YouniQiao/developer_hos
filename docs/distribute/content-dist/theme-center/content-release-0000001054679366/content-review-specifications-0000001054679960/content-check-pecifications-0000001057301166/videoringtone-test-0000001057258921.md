下载视频铃声后，在“我的下载”-“视频铃声”中默认取第一帧为封面图，因此视频铃声的第一帧不能是纯色背景或模糊的图片。

## 2. 规格检视

| 检查项 | 详情 |
| --- | --- |
| 命名规范 | 壁纸名称在前，后缀可以用数字区分，壁纸名称字数建议在2~8字以内。例：壁纸名称为：房子，同系列其他名称可以为：房子1。 |
| 壁纸名称中，一定不可包含特殊符号，如以下符号：# “ & / ? $ ^ \* : () \ &lt; &gt; , | % + 等。  壁纸名称不能带任意中文符号。 |
| 壁纸名称中，一定不可出现OS、Win及年限等关键字。 |
| 壁纸描述必须和内容相符，不能恶意占用不相干内容名称。例：资源内容为：树，不能命名为电影明星。 |
| 资源规格 | 壁纸尺寸规格符合规范。 |
| 版权检查 | 摄影图版权证明文件要求：  1.提供带有参数的摄影源文件，文件格式：png、 jpg、jpeg或MP4；  2.提供的原创声明中需包含摄影相关信息，包括但不限于拍摄机器名称，型号等内容或摄影作品登记书。 |
| 上传规范 | 壁纸上架平台时资源名称为唯一标识不能和其他已上架资源重复。 |
| 同一设计师禁止将同一套壁纸重复上架，或做简单更改后重复上架。 |
