## 2. 规格检视

| 发布到 中国大陆 | | |
| --- | --- | --- |
| 模块 | 检查项 | 详细描述 |
| description.xml | 表盘市场详情检查 | 1. 表盘名称不能包含特殊符号（ ' = # " & / ? $ ^ \* : () \ &lt; &gt; , | % + ?）；中文名称长度不超过7个字符（不含空格不超过6个字符，包含空格不超过7个字符）；英文名称长度不超过19个字符，包含空格。 |
| 2. 主题描述必须有文字内容，功能描述可以详细介绍表盘功能；描述文件不能出现任何售后联系方式（如QQ、微信微博、邮箱、二维码等）；描述不能带广告性质；表盘命名和简介描述不能带省电、续航、低功耗、低能耗、 超续航之类的能耗字眼；466\*466分辨率2.0.X版本中表盘简介不能出现熄屏显示和AOD的相关描述。 |
| 3. 国内设计师上传至国内必须有中文简介，海外设计师上传至国内时的简介可以全用英文。 |
| 表盘内容 | 内容检查 | 1. 内容良好，不得含人身攻击、色情、暴力、惊悚、恐怖、宗教、种族、政治相关内容，内容不得违反相关法律法规，存在反政府、反社会内容。 |
| 2. 不能侵权抄袭其他作品，IP形象需要上传版权证明文件。 |
| 3. 内容不得有广告嫌疑，如不能出现网址等。 |
| 4. 表盘素材切图数量限制：预览图大小不超过200kb。背景图大小不超过150kb。每组组合图尺寸大小请保持一致。  同组表盘切图在保证同尺寸情况下，切图四周尽量不要留有透明像素。 |
| 5. 表盘上Logo的限制：设计师可以在表盘中添加本人或自己公司的Logo，禁止添加具有他人版权或属于第三方的商业Logo，如HUAWEI等。 |
| 6. 表盘预览图及实际应用界面不能出现误导用户的标识，比如手表不支持的能力：信号、WiFi、蓝牙等。 |
| 上传规范 | 版本检查 | 1. 请参考[分辨率与版本号](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/watchface-0000001054571181/basic-concepts-0000001207883464/resolution-version-0000001252603441)。 |
| 2. 不能随意手工更改表盘的前两位版本号，比如工具已经确定导出是2.1版本，不能通过其他方式将2.1改为2.0或改成1.1版本，否则可能会导致手表黑屏。 |
| 版权检查 | 若不同的设计师拥有同一图片版权，该图片仅限最早上架的表盘使用，后续表盘不允许使用该图片创作，以避免引起版权纠纷。若使用IP形象，需提供版权或授权证明；若图片内容是设计师自创的，除了需提供原创声明外，还需提供能看清分层的源文件或源文件截图或源文件录屏。  摄影图版权证明文件要求：  1.提供带有参数的摄影源文件，文件格式：png、 jpg、jpeg或MP4；  2.提供的原创声明中需包含摄影相关信息，包括但不限于拍摄机器名称，型号等内容或摄影作品登记书。 |
| 重复上架检查 | 检查是否将同一表盘重复上架，同一资源不能跨能力集上架，例如同一设计上传了一个2.1.1表盘，加自定义或动态功能为2.6.1在不能再上传，否则账号将被记录违规行为，按平台违规规则处理且隐藏重复作品。 |
| 描述检查 | 制作跳转功能，需在简介第一句话描述：跳转功能依赖于手表版本，检查设计师作品与描述是否相符。 |
| 海外补充项 | 海外补充项 | 表盘上架海外需与国内同一表盘保持定价一致。 |
