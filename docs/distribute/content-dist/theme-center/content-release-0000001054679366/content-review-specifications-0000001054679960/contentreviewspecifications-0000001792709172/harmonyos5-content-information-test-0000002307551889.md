（3）作品上架平台时名称为唯一标识，不能和其他已上架作品重复；

（4）作品/资源名称前、后禁止存在空格，中间可存在一个空格，禁止出现无意义空格；

（5）作品/资源名称不可包含特殊符号（#\"&/?$^\*:\\&lt;&gt;,|%+ ~ ! &等）；

（6）同类型作品/资源名不能仅以数字（XX1，XX2）、时间（正午，傍晚，黄昏）等描述作区分；

（7）作品/资源名称不可包含敏感词汇，不可与官方资源名称雷同，不可包含第三方广告信息、其他产品名，不可与知名商品一致，无授权不可与知名影视、动漫IP强相关；

（8）作品/资源名称不可带有功能特效（充电、流体等）、系统（OS、原装OS、原生OS、Win等）、系统bug名（unknown、undefined、test等）、功耗（低功耗、低能耗、省电、超续航等）描述，不可添加组件、我的主题、年限等关键字；

（9）名称上线后无法更改，请按照规则要求命名。

通用的注意事项说明（如：主题APP升级至XXX版本以上才可使用XX功能，授权“主题服务”开启位置权限及服务等）无需添加至资源简介中，平台已统一说明使用注意事项。

（2）制作要求：

a.为更好地向用户推荐不同的主题资源，建议在简介中多描述关于画面所传递出来的气氛、情感等信息；

b.作品简介与资源简介内容不可重复；

c.作品下的不同资源会存在不同功能，因此作品简介不可添加功能描述，请将功能描述添加至资源简介；

d.资源简介中的功能或动效描述需与实际应用相符，不得出现与资源无关的描述；

e.简介要求语句通顺、断句合理，文字中不得出现错别字、缺字漏字、不文明用语、异形符号、无意义数字及空格；

f.表盘制作跳转功能，必须在简介（作品、资源简介皆可）第一句话描述：跳转功能依赖于手表版本；

g.上传海外的资源简介不可为纯中文，海外简介描述处必须有对应的英文描述或者对应的小语种描述；

h.若锁屏解锁方式未使用系统解锁方式（任意滑动解锁），需在简介中告知用户锁屏解锁方式；

i.主题内容命名避免与系统特性重名造成歧义，如“负一屏/正一屏”，锁屏命名为“控制中心”。

（3）合规禁止：

a.简介中不可出现福利、返现、赠送等诱导用户购买或下载的内容；

b.简介中不可出现联系方式、个人信息等字眼；

c.不可出现三方广告、公司名、地址链接；

d.不可出现华为品牌和三方品牌名称；

e.不可出现与作品无关的推销类的字眼。

（2）制作要求：

a.作品图建议上传能清晰展示作品详情的图片，不建议只上传AOD、图标等，以便更好的对作品质量进行判断；

b.预览图/预览视频清晰且保持主体元素完整，无黑边，无锯齿，无明显切割，无大面积留白；

c.考虑主题图标的可识别度，图标产品信息视觉区域尺寸不可小于108\*108，图标的background可视图尺寸建议大于216\*216，如在主视觉设计比较丰富突出时可考虑将桌面壁纸压暗设计等；

d.预览视频不同分辨率等比缩放，确保无拉伸变形，视频首尾衔接流畅，循环播放，不出现画面跳动或闪烁；

e.为保证用户体验，预览视频需展现主题的大体风格，不可为单张静态图样式或只展示AOD效果

f.信息内容无遮挡、各类信息易识别；

g.顶部不可以出现状态栏信息，底部不可带有横条快捷键；

h.文字内容需拼写正确，无错别字，各信息要素之间应当构图美观、布局合理，符合逻辑；

i.同一视觉的素材应用到不同作品时需再创作，同一套模板未经二次设计不得直接用在多套资源上；表盘方表不可直接使用圆表样式，需适配修改；

j.同一作品各元素风格应保持一致，例如一套主题中的AOD、图标、组件、锁屏、壁纸、耳机弹窗等元素风格一致；

k.除主题、百变卡片外，其他类型作品皆不可上架日历样式资源；

l.为保证用户的良好体验，请勿使用“说明书式”的预览图；

m.不可制作“手机框”样式的预览图和预览视频；

n.海外作品不展示农历日期，十二时辰等，不可展示纯中文（日期，星期等重要数据不可为中文）；

（3）合规禁止：

a.预览图/预览视频中不可出现任何品牌名、品牌logo、品牌商品；不得有广告嫌疑，不能出现第三方引流内容及网址，不可出现掌酷引擎等第三方引擎字样；

b.不得出现任何品牌标识（华为/荣耀/HUAWEI/HONOR等）和系统标识（EMUI/HarmonyOS/鸿蒙等）、系统版本信息；

c.预览图/预览视频上不能出现真实信息，真实手机号，未经授权的头像、联系人名称、第三方公司名称、真实的歌曲信息（歌名，封面图，演唱者等）；

## 4.标签规范

（1）上传作品时，设计师从以下维度增打标签：

|  |  |  |  |
| --- | --- | --- | --- |
| 标签 | 必打个数 | 备注 | 作品类型 |
| 画面标签 | 1个 | 依据画面主元素添加 | NEXT主题、百变卡片、  耳机弹窗、表盘、熄屏显示、静态壁纸、动态壁纸 |
| 风格标签 | 1个 | 依据主要风格添加 | NEXT主题、百变卡片、  耳机弹窗、表盘、熄屏显示、静态壁纸、动态壁纸 |
| 颜色标签 | 1个 | 依据主要颜色添加 | NEXT主题、百变卡片、  耳机弹窗、表盘、熄屏显示、静态壁纸、动态壁纸 |
| 卡片标签 | 1个 | 根据卡片实际元素添加 | 百变卡片 |
| AI绘图 | / | 若为AI绘图作品添加 | NEXT主题、百变卡片、  耳机弹窗、表盘、熄屏显示、静态壁纸、动态壁纸 |
| 表盘标签 | / | 手表/手环表盘具有相关功能添加 | 表盘 |
| IP类标签 | / | IP作品添加 | NEXT主题、百变卡片、  耳机弹窗、表盘、熄屏显示、静态壁纸、动态壁纸 |
| [字体标签]  字体风格  字重  内容  字库  功能特效（字体） | 各打1个 | 根据字体实际情况添加 | 字体 |
| AOD标签 | / | 根据AOD类型添加 | 熄屏显示 |

（2）为提升标签的准确度，不标注同义词标签和无意义标签，不标注与资源画面无关的标签，各标签要有一定的区分度；

（3）卡片标签说明

通过主题联盟上传百变卡片作品时，创作者需根据卡片的实际元素给作品打“卡片标签”标签，支持打多个卡片标签。

| --- | --- | --- |

| 时钟 | 包含当前时间功能的卡片，例如模拟时钟，数字时钟等。 | * 当前时间 * 时钟APP跳转链接 |
| 天气 | 包含天气数据功能的卡片，例如天气数据展示，卡片样式随天气变化等效果。 | * 当前温度 * 气象描述（多云，下雨等） * 华为天气APP跳转链接 |
