# 资源上架规范

|  |  |
| --- | --- |
| 资源上架规范  \*资源画面涉及以下内容不可上架 | |
| 相似资源 | 使用相似度高或相同素材的画面 |
| 使用相似度高或相同样式的模板 |
| 低劣质画面  （无上架空间） | 色彩单一：纯色或几个简单色块拼接的设计 |
| 配色不佳：配色艳俗或颜色搭配呈现效果不佳 |
| 画面单一：元素单一，内容单调，画面整体无设计 |
| 画面杂乱：画面元素过多过杂，遮挡信息功能或功能信息显示不清晰 |
| 画面低幼：适用于幼儿、低龄儿童的卡通图，简单绘制的画风幼稚的图片 |
| 笔触简陋：笔法粗劣，元素形象设计潦草 |
| 低质量图片：设计图及摄影图存在噪点、马赛克、清晰度低、高曝光等问题、 |
| 非主流画面：非主流风格图片或文字 |
| 同质化清单  （优质资源可上架） | 画面主体为人物、人物头像，交互无动画无创意 |
| 画面主体为建筑、街景、书桌，仅搭配昼夜切换功能，动画效果简单无创意 |
| 手持相机自拍、面部有遮挡物的自拍等 |
| 画面为太空人主体旋转、发光、简笔画等样式 |
| 画面主体为手绘插画风蝴蝶、蝴蝶结 |
| 膨胀、气球、毛绒等样式的纯文字或简单元素设计的图片 |
| 内容为纯文字，背景为纯色或光影 |
| 内容主体为单调的贴纸、对联、窗花、灯笼、烟花、鞭炮的画面 |
| 画面主体包含流水、瀑布、凉亭、群山、荷塘、锦鲤、鹤、鹰等元素构成的图片 |
| 画面由蓝天、白云、草地、郁金香、彩霞元素构成的图片 |
| 山、圆环样式的合成图 |
| 画面主体为海浪、海、沙滩、岩石等图片，无功能性交互 |
| 画面含星空、星云、星球、黑洞等宇宙及天象类摄影、合成图片，无功能性交互 |
| 多由射线或者粒子组成的点、线、面构成较为抽象的画面，无功能性交互，无创意特效 |
| 画面展示重复主体，多主体背景较暗，突出单个主体发光特效 |
| 插画手绘风格的钢琴、掌机、手机框、时钟等画面 |
| 画面为缺乏创意的碎屏效果、跑马灯样式 |
| 画面主体为表情包或笑脸的图片 |
| 画面以动物、玩偶为主体的摄影图或合成图，锁屏无创意动效 |
| 同质化清单（表盘）  （优质资源可上架） | 画面主体为无创意设计的人物形象 |
| 画面为单调无创意设计的太空人，包括主体旋转、发光、简笔画等样式 |
| 画面主体为无创意设计的简笔表情包、搞怪表情或笑脸的图片 |
| 画面主体为动物简笔画，搭配简单的数据展示，无创意动效 |
| 画面以碎花、钻石等装饰为主要元素的简单无创意设计 |
| 画面含简单设计、无创意动效的星空、星云、星球、黑洞等宇宙及天象类图片 |
| 无特殊设计、无创意动效的山、圆环样式的图片 |
| 画面为简单无创意的扫秒发光样式 |
| 画面为纯文字、文字配图、文字贴纸或仅为数据展示，背景多为纯色或光影 |
| 画面主体为无创意设计及动效的，简约油画、插画、摄影图等 |
| 画面为观感不佳的摄影图简单搭配数据展示 |
| 表盘以荧光色、彩色为主配色，或搭配跑马灯样式的无创意动效的设计 |
| 仅改变背景颜色，画面无创意设计及动效 |
| 画面数据以几何展示，无创意动效 |
| 颜色单一的液晶屏表盘，画面无创意设计及动效 |
| 颜色单一、无创意的商务风简易表盘 |
