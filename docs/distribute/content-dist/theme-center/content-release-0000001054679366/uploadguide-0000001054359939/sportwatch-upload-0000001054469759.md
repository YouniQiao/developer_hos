# 表盘

## 1. 表盘上传

第一次上传表盘，请按以下操作：

1. 点击“作品上传”，左侧导航栏选择表盘，在“创建作品”处编辑并保存好“表盘名称”。

   ![](./img/dba4942d5ab1.png)

   表盘包description.xml文件中的“&lt;title-cn&gt;”字段内容需要与编辑的“表盘名称”保持一致，否则无法上传成功。

   ![](./img/f140c3a93dea.png "点击放大")

   ![](./img/8fb114e2a508.png "点击放大")
2. 然后上传表盘包。表盘包上传完成后，补充上传表盘的预览视频。

   ![](./img/62abfac35f63.png "点击放大")
3. 上传表盘的版权文件或原创声明。

   ![](./img/08c0cd124517.png "点击放大")
4. 完善表盘付费设置、分发国家及区域、标签设置、发布等信息，点击下一步。

   ![](./img/954295291ae4.png "点击放大")

   ![](./img/c979d66733fd.png "点击放大")
5. 信息确认无误后，点击“提交”。

   ![](./img/856b6bebb3a1.png "点击放大")
6. “我的作品”列表页对应作品的状态显示为“审核中”，表示上传成功。

   ![](./img/65b926b8ed1c.png "点击放大")

## 2. 表盘升级

1. 在“我的作品”页找到需要升级的表盘，点击“升级”。

   ![](./img/986466f1c12f.png "点击放大")
2. 点击“上传表盘包”，上传准备好的表盘包。

   ![](./img/27fc78df0415.png "点击放大")
3. 选中新上传的表盘包，表盘付费设置不允许修改，勾选更新类型，点击“下一步”。

   ![](./img/dd4fe192c9d0.png "点击放大")

   ![](./img/512044fbf2f7.png "点击放大")
4. 信息确认无误后，点击“提交”。

   ![](./img/4e3caf2af7ca.png "点击放大")
5. “我的作品”列表页对应作品的状态显示为“升级中”，表示上传成功。

   ![](./img/41e70146c81d.png "点击放大")
