# 平板主题设计指导及规范

平板主题就是一套用于改变平板界面视觉效果的元素集合，包含熄屏显示、锁屏、桌面、图标、应用（含控制中心、通知中心、播控中心、音量条、文件夹、耳机弹窗、充电动效换肤）、百变卡片。
