# EMUI 9.1主题设计指导及规范

小主题：小主题是锁屏、桌面及图标范围进行换肤的主题。

大主题：大主题是锁屏、桌面、图标、联系人、信息、下拉开关、拨号范围进行换肤的主题。

锁屏主题：锁屏主题是仅锁屏范围换肤的主题。

图标主题：图标主题是仅图标范围换肤的主题。

以上所有类型主题均可直接使用主题工具直接制作。

## 3. 锁屏（unlock）制作指导及规范

EMUI9.1的解锁方式如下：滑动锁屏、杂志锁屏、视频解锁、百变锁屏。四选一。

所有类型的锁屏，unlock文件夹下都必须有theme.xml文件。

### 3.1 滑动锁屏

滑动锁屏为一张静态壁纸全屏滑动解锁。需在壁纸wallpaper文件夹下放置锁屏壁纸一张。

锁屏壁纸的尺寸为：1080×2160px，格式为.jpg，文件名为：unlock\_wallpaper\_X.jpg（X为阿拉伯数字按顺序排列，首张壁纸X为0）。

锁屏壁纸样板如下：

![](./img/5050d03d71e9.png)

滑动锁屏unlock文件夹下只有一个XML文件，显示内容为默认效果，设计师仅能修改壁纸，代码固定，如下所示：

```
<?xmlversion="1.0" encoding="UTF-8"?>
<HWTheme>
    <itemstyle="slide"/>
</HWTheme>
```

滑动锁屏应用效果如下图所示：

![](./img/eb45a04828ff.png)

滑动锁屏可通过工具可视化制作：

![](./img/b4533903ee5b.png "点击放大")

### 3.2杂志锁屏

杂志锁屏在设置中开启开关时，每次亮屏时显示不同的锁屏图片，首次亮屏显示设计师设计的锁屏壁纸。需在壁纸wallpaper文件夹下放置锁屏壁纸一张。

锁屏壁纸的尺寸为：1080×2160px，格式为.jpg，文件名为：unlock\_wallpaper\_X.jpg（X为阿拉伯数字按顺序排列，首张壁纸X为0）。

锁屏壁纸样板如下：

![](./img/7cbed2d05938.png)

杂志锁屏和滑动解锁相似，unlock文件夹下只有一个XML文件，显示内容为默认效果，设计师仅能修改壁纸，代码固定，如下所示：

```
<?xmlversion="1.0" encoding="UTF-8"?>
<HWTheme>
    <itemstyle="magazine"/>
</HWTheme>
```

杂志锁屏应用效果如下图所示，系统根据手机指纹解锁位置，自动判断时间布局：

![](./img/21b0ec087c10.png "点击放大")

杂志锁屏可通过工具可视化制作：

![](./img/2e297ae0ec4c.png "点击放大")

百变锁屏在一定需求下可用于开发风格多变的用户界面。可方便地通过更换皮肤改变界面风格、动画甚至交互方式。百变锁屏可通过工具可视化制作：

视频锁屏是在锁屏上面循环播放视频的锁屏，默认全屏滑动解锁。设计师可以制作视频锁屏壁纸，并能修改锁屏时间切图、日期及上下午颜色。视频锁屏可通过工具可视化制作：

## 4. 壁纸（wallpaper）

壁纸分为桌面壁纸和锁屏壁纸，大小均控制在1M以内；均可按照顺序放置多张壁纸。

由于9.1特性，锁屏壁纸实际效果会被放大10%，设计师在制作时请注意规避。

锁屏壁纸必须放在wallpaper中才会自适应兼容18:9及以上的手机。

## 5. 图标（icons）

图标包含了系统图标和第三方图标；其中系统图标又分为动态图标和静态图标；所有图标尺寸：192×192px；所有图标格式：png格式。

### 5.1 系统图标-静态图标

* 一共67个静态图标，其中下图绿色选框的图标com.huawei.hwmarket.vr.png（VR）为选做，其余66个为必做。
* 日历，时钟的静态图标不能出现带有误导性的数字。
* 不同应用不能使用相同图标。
* 系统图标必须经过重新绘制。其中，红色框选的图标需保留官方图标中心元素，设计师不可随意更改其形状。

![](./img/28d1d312212e.png "点击放大")

### 5.2 系统图标-动态图标

动态图标在icons下面的dynamic\_icons文件夹下，分别支持以下3个动态图标：

![](./img/8a2dcde04f65.png "点击放大")

![](./img/92eda4884faf.png)

* 以上动态图标均是选做图标。
* 动态日历只用提供背板，具体时间详情由系统自动写入，可通过launcher中的theme.xml文件修改显示的时间数字的颜色。
* 若没有适配动态图标，需将其所属文件夹删除，若三个动态图标都没有适配，则直接删除dynamic\_icons。
* 日历，时钟的静态图标不能出现带有误导性的数字。不同应用不能使用相同图标。

### 5.3 第三方图标-模板适配

![](./img/d585558b5eef.png)

我们希望设计师尽可能多的适配第三方图标，但是也会有没有适配的情况，为此，华为主题提供了一套解决方案：通过背板，遮罩，蒙版组合出一套底板使未适配的第三方图标风格趋近于统一。一组第三方图标底板，由icon\_background\_01.png、 icon\_border.png、 icon\_mask.png组成，三个资源缺一不可。

1. icon\_background\_01.png ——图标底板。可以做多张底板图片（如不同材质、颜色），系统会随机调取该底板放到第三方应用的下方。做这个底板的时候，要考虑到其它图标大小，视觉上要保持一致。

   ![](./img/80e547edde33.png "点击放大")
2. icon\_border.png——图标遮罩。统一罩在第三方图标的最上层，可以灵活应用，做出多种效果。比如下图中图标上的做旧划痕或者阴影过渡等效果。如果不需要用这种叠加效果，则在主题包中将这张图做成完全空白透明的。

   ![](./img/7797f9fe4144.png "点击放大")
3. icon\_mask.png——图标蒙版。该图片的图像区域必须为纯黑色，第三方图标只显示黑色图像范围的内容。

### 5.4 第三方图标-包名适配

![](./img/a2cdb355acb3.png)

主题图标作为锁屏模块外第二重要大模块，是考究主题质量好坏的重要一环。我们鼓励设计师在EMUI9.1主题上适配更多的第三方图标，官方将对系统图标和第三方图标适配的数量和质量重点标注为主题推荐标准之一。

通过查询图标包名，将相应适配好的图标命名为正确的包名放置在icons图标中，可以更好的适配第三方图标。第三方图标适配上限为2万个。

第三方图标包名可以通过手机安装第三方“包名查看器”应用查看，或在[以下链接](https://sj.qq.com/)查看：

如：文件管理器的包名为：com.dianxin.file.explorer 。

![](./img/ef5a713ee6af.png "点击放大")

## 15. 预览图（preview）

![](./img/0b3a6c0225c1.png)

预览图是主题的一个概览，用户通过预览图判断一个主题的大体风格。所有静态预览图均可使用主题工具直接截图制作。

除cover及图标预览图外预览图的尺寸均为 1080×2160px，cover图的尺寸为1080×1920px，格式为.jpg，图标预览图使用PSD文件替换图标即可。

因数据库图片文件名称长度有限制，主题包内preview文件夹下的预览图总张数需≤15张。

预览图可使用手机截图的方式制作，截图后需按照预览图样板进行部分修改。没有手机的设计师可使用提供的PSD模板制作预览图，详情见PSD文件说明。

所有上传至海外的主题包不能包含短信预览图，即以preview\_mms\_X.jpg命名的图片，但仍然需要保证对短信模块的设计、换肤等。

### 15.1 小主题预览图样板

![](./img/d356785a525b.png "点击放大")

![](./img/feb3faf41409.png)

1. cover预览图，顶部和底部80px的范围内，壁纸不允许出现任何内容，包括状态栏信息，按钮等，cover预览图内容应与锁屏内容保持一致。
2. 除cover，icon\_small 、icon\_small\_1以外的其他预览图状态栏只允许出现信号、电量和时间。
3. 预览图所展示的内容应该与手机的实际效果相同。

### 15.3 视频预览文件

主题支持视频动态预览，该文件无需打包在主题内，上传主题时，直接另外在联盟上上传。

* 视频尺寸为：1080×2340px（高端机型适用）、1080×1920px（低端机型适用）；
* 视频格式为MP4，编解码制式要求为H.264，视频大小建议在10MB以内，时长建议在15秒以内；
* 为保证预览效果，请提供能循环播放的视频，确保首尾衔接部分流畅，不会出现画面跳动或者闪烁；

## 18. 主题包制作注意事项

* 除了描述文件以外，主题包内各模块不能出现任何的中文字符。
* 所有的xml文件均要将另存编码修改为UTF-8，如下图：

  ![](./img/b00a075df691.png "点击放大")
* 点9文件均要进行编译才可以正确读取。
* 为方便新设计师适配主题，9.1在线大、小主题模板中的点9文件均已编译。
* 使用截图方式制作的预览图，百变锁屏预览图的周几全部要通过PS修改为星期几。
* 请结合9.1在线主题模板、 EMUI9.1小主题预览图模版psd 、 9.1 主题模板指导文档学习制作。
* 制作好的主题包，可通过电脑导入对应版本的华为手机或者荣耀手机（9.1的主题导入EMUI9.1手机）查看并测试，导入路径为手机内存/Huawei/Themes，之后便可在主题app-我的-我的主题里找到该主题。
* 如有疑问，可在[官方主题论坛](https://developer.huawei.com/consumer/cn/forum/blockdisplay?fid=0101211211623820128)发帖讨论。
