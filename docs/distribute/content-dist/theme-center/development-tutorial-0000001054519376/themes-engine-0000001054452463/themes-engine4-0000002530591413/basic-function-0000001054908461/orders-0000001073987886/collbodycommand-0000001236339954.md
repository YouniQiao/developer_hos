# 刚体变速受力命令&lt;CollBodyCommand&gt;

## 功能概述

刚体变速受力命令&lt;CollBodyCommand&gt;，可以改变刚体的运动速度、受力方向、刚体属性和刚体类型，可以配合[2D物理碰撞](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/x2d-advanced-animation-0000001055668422/collisionworld-0000001277995829)使用。

## XML规范

```
<CollBodyCommand collbodyid="body_5"  forceX="100" forceY="100" vx="100" vy="100"/>
```

## 参数说明

| 参数 | 类型 | 选项 | 注释 |
| --- | --- | --- | --- |
| collbodyid | 字符串 | 必填 | 要调整的刚体的id。 |
| forceX | float | 选填 | 刚体x方向受力调整，建议取值范围【-10000000——10000000】，正负值表示方向相反。 |
| forceY | float | 选填 | 刚体y方向受力调整，建议取值范围【-10000000——10000000】，正负值表示方向相反。 |
| vx | float | 选填 | 刚体x方向速度调整，建议取值范围【-10000000——10000000】，正负值表示方向相反。 |
| vy | float | 选填 | 刚体y方向速度调整，建议取值范围【-10000000——10000000】，正负值表示方向相反。 |
| isActive | int | 选填 | 刚体属性，0为非刚体，其他值及默认值1为刚体。  说明：  刚体不能被穿过，碰撞时反弹。非刚体可被穿过。 |
| type | int | 选填 | 改变刚体类型，2为动态刚体，其他值或默认值为静态刚体。  说明：  动态刚体：受力会运动，可以设置运动速度。  静态刚体：静止不动，质量无穷大的刚体。 |
