# 恒定帧率&lt;FrameRate&gt;

## 功能概述

指定引擎的帧率为固定值，在效果与功耗之间寻求平衡。

## XML规范

```
<Lockscreen frameRate="" vibrate="" screenWidth="" pressure="">
</Lockscreen>
```

## 参数说明

| 参 数 | 类 型 | 选 项 | 注 释 |
| --- | --- | --- | --- |
| frameRate | 数值 | 选填 | 屏幕刷新的帧率，数值越大，效果越好，功耗相对较高。默认60，推荐使用30 |

## 应用示例

<strong>示例一：</strong>演示屏幕刷新的帧率为固定值30时的示例。

```
<Lockscreen version="1" frameRate="30" vibrate="false" screenWidth="1080" id="">
</Lockscreen>
```
