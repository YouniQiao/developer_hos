# 日历数据开放&lt;Calendar&gt;

## 功能概述

开放了日历App中订阅管理模块的黄历、星座、历史上的今天大事件相关数据。配置Calendar标签后，需要给该标签配置card属性，card属性值分别对应黄历（Almanac）、星座（Constellations）、历史上今天大事件（BigEvent）。然后在Calendar标签里面声明var变量并使用指定的变量名，即可声明对应的日历数据变量。变量支持设置默认值，如果获取到接口返回数据会更新该值。

[](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/publicContent/011/111/111/0000000000011111111.20251218173503.23299979565415093111796557815681:20260601221729:2800:8AE047AA720D83BA77BE48A1F00B30ED2D6B2F5B307C67C2FCA367B2023379D8.mp4)

![](./img/c4981026f042.png)

目前该功能暂时只支持国内，后续支持海外会另行通知。

## XML规范

```
         <!--黄历字段声明-->
         <Calendar card="Almanac">
	     <Var name="Almanac.caishen" expression="正南" type="string"/>
             <Var name="Almanac.yi" expression="'打扫 签订合同 开业 安机械 安葬 开光 迁坟 上梁'" type="string"/>
	     <Var name="Almanac.ji" expression="'结婚 祈福'" type="string"/>
         </Calendar>

         <!--星座字段声明 声明星座字段前必须先声明星座id变量，变量名称固定为constellationsId并且设置默认值为十二星座Id的其中一个-->
         <!--星座对应Id和星座映射关系参看附表4.5-->
         <Var name="constellationsId" expression="0003001" type="string" persist="true"/>
         <Calendar card="Constellations">
	     <Var name="Constellations.signName" expression="魔羯座" type="string"/>
             <Var name="Constellations.today.loveDes" expression="'注意调养,适合独处,单身者别与过多的异性纠缠，那样只会让你身心疲惫。'" type="string"/>
             <Var name="Constellations.today.loveStar" expression="3" type="int"/>
         </Calendar>

         <!--历史上今天大事件字段声明-->
         <Calendar card="BigEvent">
	      <Var name="BigEvent_0.year" expression="'1888'" type="string"/>
              <Var name="BigEvent_0.event" expression="'今天大事1'" type="string"/>
	      <Var name="BigEvent_1.year" expression="'1921'" type="string"/>
              <Var name="BigEvent_1.event" expression="'今天大事2'" type="string"/>
	      <Var name="BigEvent_2.year" expression="'1949'" type="string"/>
              <Var name="BigEvent_2.event" expression="'今天大事3'" type="string"/>
	  </Calendar>
```

![](./img/c471abcdc11b.png)

1. 配置Calendar标签后，需要在给该标签配置card属性，card属性值分别对应黄历（Almanac）、星座（Constellations）、历史上今天大事件（BigEvent）。
2. 只有在Calendar标签里面声明的Var才能获取到接口返回的日历数据，才会对默认值进行刷新。Var用法参照[自定义变量&lt;Var&gt;](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/variant-0000001074067773/variant-0000001074315406)，日历变量名称参照[日历变量名称说明表](#section1687212418312)。
3. 声明星座字段前必须先声明星座constellationsId变量，变量名称固定为constellationsId并且设置默认值为十二星座Id的其中一个。切换星座可以通过VariableCommand修改constellationsId的值来切换刷新数据。constellationsId映射星座名称对应关系参照[星座数据](#section1745793717495)。切换星座实例参照[示例2：展示星座信息](#section1067192620106)。

## 使用说明

1. 如果使用了Calendar标签并定义了黄历变量，可以通过全局变量almanacRespCode的值，获取当前数据是展示的默认值还是日历接口返回的值。值为0表示默认值，1表示接口返回值。参照[示例1：展示黄历信息](#section183266183104)。
2. 如果使用了Calendar标签并定义了星座变量，可以通过全局变量constellationRespCode的值，获取当前数据是展示的默认值还是日历接口返回的值。值为0表示默认值，1表示接口返回值。参照[示例2：展示星座信息](#section1067192620106)。
3. 如果使用了Calendar标签并定义了历史上今天大事件变量，可以通过全局变量bigEventRespCode的值，获取当前数据是展示的默认值还是日历接口返回的值。值为0表示默认值，1表示接口返回值。参照[示例3：展示历史上今天的大事件信息](#section16925143514113)。
4. 当使用日历数据变量时，需要在锁屏上提供通过Intent命令的方式跳转至日历App，以便用户在想了解更多日历信息的时候，可以直接从锁屏跳转至系统日历App，提升用户体验。

   ```
   <!-- 示例：跳转至日历应用，跳转方式为click -->
       <Button y="0" x="0" w="940" h="940">
          <Trigger action="click">
             <IntentCommand action="android.intent.action.MAIN" package="com.android.calendar" class="com.android.calendar.AllInOneActivity" condition="1"/>
          </Trigger>
       </Button>
   ```

## 应用示例

### 示例1：展示黄历信息

展示黄历信息，并判断数据是网络返回值还是默认值状态。

```
        <!--黄历字段声明-->
        <Calendar card="Almanac">
	     <Var name="Almanac.caishen" expression="正南" type="string"/>
             <Var name="Almanac.yi" expression="'打扫 签订合同 开业 安机械 安葬 开光 迁坟 上梁'" type="string"/>
	     <Var name="Almanac.ji" expression="'结婚 祈福'" type="string"/>
        </Calendar>

        <Text x="40" y="100" alignV="top" color="#ffffff" size="30"   format="财神: %s" width ="1000" autoLineFeed="true" paras="#Almanac.caishen"/>
        <Text x="40" y="200"  alignV="top" color="#ffffff" size="30"   format="宜: %s" width ="1000"  autoLineFeed="true" paras="#Almanac.yi"/>
        <Text x="40" y="300"  alignV="top" color="#ffffff" size="30"   format="忌: %s" width ="1000"  autoLineFeed="true" paras="#Almanac.ji"/>
        <Text x="40" y="400"  alignV="top" color="#ffffff" size="30"   format="数据状态0-默认1-接口返回值: %s" width ="1000"  autoLineFeed="true" paras="#almanacRespCode"/>
```

展示魔羯座当天的爱情运势指数及描述信息，点击按钮切换为射手座数据，并判断数据是网络返回值还是默认值状态。

           <Calendar card="Constellations">
	       <Var name="Constellations.signName" expression="魔羯座" type="string"/>
               <Var name="Constellations.today.loveDes" expression="'注意调养,适合独处,单身者别与过多的异性纠缠，那样只会让你身心疲惫。'" type="string"/>

           <Text x="40" y="600"  alignV="top" color="#ffffff" size="30"   format="爱情运势指数: %s星" width ="1000"  autoLineFeed="true" paras="#Constellations.today.loveStar"/>
