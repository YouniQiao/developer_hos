# 流体增减命令&lt;ParticleCommand&gt;

## 功能概述

流体增减命令&lt;ParticleCommand&gt;，配合[流体动效](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/x2d-advanced-animation-0000001055668422/fluidsview-0000001194289627)使用，实现动态增加或减少流体数量。

## XML规范

```
<ParticleCommand type="" color="" condition="">
   <CircleShape radius="" xPosition="" yPosition=""/>
</ParticleCommand>
或
<ParticleCommand type="" deleteNum="" condition=""/>
```

## 参数说明

| 参数 | 类型 | 选项 | 注释 |
| --- | --- | --- | --- |
| type | 字符串 | 必填 | 仅支持填写“add”(增加流体数量)或“delete"（减少流体数量） |
| color | 表达式 | 选填 | 新增流体的颜色值，支持argb函数 |
| condition | 表达式 | 选填 | 条件判断，支持表达式。当condition里的条件判断为非0或者为true时，该命令执行，为false或者0则不执行 |
| deleteNum | 表达式 | 选填 | 需要删除的流体数量，最高为50000，支持表达式。当type为delete时必填 |
| CircleShape | 子元素 | 选填 | 当type为add时，与PolygonShape选择一个填写，具体参数见[流体动效](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/x2d-advanced-animation-0000001055668422/fluidsview-0000001194289627) |
| PolygonShape | 子元素 | 选填 | 当type为add时，与CircleShape选择一个填写，具体参数见[流体动效](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/x2d-advanced-animation-0000001055668422/fluidsview-0000001194289627) |

## 应用示例

<strong>示例1</strong>：点击按钮时，在坐标点（0.8，1.5）的位置添加圆心为（0.8，1.5）、半径为（0.2\* #aaa）且颜色为argb(255,210,100,22)的圆形区域流体。

```
<Button x="0" y="50" w="#screen_width" h="200">
    <Trigger action="down">
        <ParticleCommand type="add" color="argb(255,210,100,22)" condition="1">
            <CircleShape radius="0.2* #aaa" xPosition="0.8" yPosition="1.5"/>
        </ParticleCommand>
    </Trigger>
</Button>
```

<strong>示例2</strong>：点击按钮时，删除数量为#delete的流体。

```
<Button x="0" y="500" w="#screen_width" h="200">
    <Trigger action="down">
        <ParticleCommand type="delete" deleteNum="#delete" condition="1"/>
    </Trigger>
</Button>
```
