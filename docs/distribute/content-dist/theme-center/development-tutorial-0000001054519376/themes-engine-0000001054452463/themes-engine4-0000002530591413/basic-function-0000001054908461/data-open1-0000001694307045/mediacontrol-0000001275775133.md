# 音乐数据开放&lt;MediaControl&gt;

## 功能概述

开放以下音乐数据和播控功能，支持在锁屏、桌面和桌面万象小组件中使用以制作创意主题。

| 音乐数据 | 歌曲名称 | 专辑封面 | 艺术家名称 | 歌词 | 歌曲风格 |
| --- | --- | --- | --- | --- | --- |
| 华为音乐App | ✓ | ✓ | ✓ | ✓ | ✓ |
| 三方音乐App | ✓ | ✓ | ✓ | × | × |

| 播控功能 | 暂停/播放 | 音量调节 | 切换歌曲 | 歌词开关 | 震动开关 | 收藏/取消收藏 | 播放模式 | 进度条 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 华为音乐App | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ |
| 三方音乐App | ✓ | ✓ | ✓ | × | ✓ | × | × | ✓ |

## 功能说明

* 第三方音乐App支持：歌曲名称、专辑封面、艺术家名称、播放/暂停、音量调节、切换歌曲、歌词开关、震动开关和进度条。华为音乐App额外支持：歌曲风格、歌词、收藏/取消收藏和播放模式。
* 如需使用华为音乐App的歌曲风格、歌词、收藏/取消收藏和播放模式，华为音乐App需要开启储存权限（设置&gt;应用管理&gt;华为音乐&gt;权限&gt;存储）并签署用户协议，并要求华为音乐App12.11.23.302版本以上，主题App12.0.14.300版本以上。
* 设计师需要在预览图中明确告知用户：歌曲风格、歌词、收藏/取消收藏和播放模式仅华为音乐App支持，同时还需考虑用户使用第三方音乐App时不支持这些数据和播控功能的情况。
* 在锁屏或桌面万象小组件中使用了音乐数据开放功能的主题，必须配置华为音乐App的跳转链接，可以同时配置其他音乐App的跳转链接。

  ```
  <!--跳转至华为音乐App-->
  <Button y="0" x="0" w="960" h="1550">
      <Trigger action="double">
          <IntentCommand action="android.intent.action.MAIN" class="com.android.mediacenter.PageActivity" package="com.android.mediacenter"/>
      </Trigger>
  </Button>
  ```
* 在锁屏中，建议不制作播控功能，避免与系统的播控弹窗重叠。

## 音乐控制器&lt;MediaControl&gt;

在使用音乐数据和播控功能时，需要先声明音乐控制器&lt;MediaControl/&gt;。

### XML规范

```
<!--不指定音乐App，可以控制当前播放的音乐App-->
<MediaControl/>
```

或者

```
<!--指定华为音乐App，此时仅可以控制华为音乐-->
<MediaControl packageName="com.huawei.music"/>
```

![](./img/ec8a973d9770.png)

1. 同一个脚本中，声明一次&lt;MediaControl/&gt;即可。
2. 音乐控制器&lt;MediaControl/&gt;需写在&lt;Lockscreen&gt;或者&lt;CommonWallpaper&gt;下一个层级。
3. 制作华为音乐小组件时，需设置packageName="com.huawei.music" 指定华为音乐App。

### 参数说明

| 参 数 | 类 型 | 选 项 | 注 释 |
| --- | --- | --- | --- |
| packageName | 字符串 | 选填 | 指定音乐App的包名，当前仅支持指定华为音乐App（com.huawei.music）。 |

## 音乐数据变量

![](./img/ec2c12887efa.png)

在使用音乐数据变量时，需要先声明[音乐控制器&lt;MediaControl&gt;](#section19781154113311)。

| 变 量 | 类  型 | 注 释 |
| --- | --- | --- |
| media\_like\_status | 数值 | 音乐歌曲收藏状态，1为已收藏。 |
| media\_play\_status | 数值 | 播放状态，1为停止，2为暂停，3为播放。  说明：  停止：从头开始播。暂停：从当前进度开始播。 |
| media\_title | 字符串 | 歌曲名称。  说明：  当通过[文本&lt;Text&gt;](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/view-0000001073865717/text-0000001074068045)展示歌曲名称时，可设置文本内容超出文本框宽度后滚动/裁剪，避免歌曲名称过长时自动换行，影响其他内容展示的情况。 |
| media\_subtitle | 字符串 | 艺术家名称。  说明：  当通过[文本&lt;Text&gt;](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/view-0000001073865717/text-0000001074068045)标签展示艺术家名称时，可设置文本内容超出文本框宽度后滚动/裁剪，避免艺术家名称过长时自动换行，影响其他内容展示的情况。 |
| media\_genres | 字符串 | 歌曲风格，共以下17类：“纯音乐类”、“中国元素”、“流行”、“民谣”、“电子”、“摇滚”、“乡村”、“说唱”、“雷鬼”、“R&B”、“FUNK”、“拉丁”、“独立音乐”、“古典”、“英伦”、“抒情歌曲”和“其他”。  说明：  1. 歌曲风格仅华为音乐支持。 2. 通过media\_genres变量，可以获取到华为音乐的歌曲风格，然后根据不同的歌曲风格设计不同的动画效果。需要注意的是：为保证每个歌曲风格都有对应的动画，建议设计师将所有歌曲风格都写入其中，可一个动画对应多种歌曲风格。 |
| media\_mode\_status | 字符串 | 播放模式文字，"顺序播放"、 "随机播放"、 "列表循环"和"单曲循环"。  说明：  播放模式仅华为音乐支持。 |
| repeat\_mode | 数值 | 播放模式数值，0、1、2、3分别对应media\_mode\_status中的文字。 |
| media\_loading\_status | 数值 | 音乐播放前的加载状态。1为正在加载；2为加载完成。  说明：  1. 拉起华为音乐数据时，可能需要一定的加载时间，此时media\_loading\_status值为1。 2. 建议设计师为正在加载状态设计提示文字或动效，提示用户音乐数据正在加载中。 |

## 音乐控制命令&lt;MediaCommand&gt;

![](./img/45402b311fdd.png)

在使用音乐控制命令&lt;MediaCommand&gt;时，需要先声明[音乐控制器&lt;MediaControl&gt;](#section19781154113311)。

### XML规范

```
<MediaCommand command="" voiceStep100="" condition=""/>
```

### 参数说明

| 参 数 | 类 型 | 选 项 | 注 释 |
| --- | --- | --- | --- |
| command | 字符串 | 必填 | command可以设置的值如下：  mediaPlay：音乐播放命令  mediaPause：音乐暂停命令  mediaLike：音乐收藏命令（仅华为音乐支持）  mediaDislike：音乐取消收藏命令（仅华为音乐支持）  skipToPrevious：上一首  skipToNext：下一首  addVoice：增加音量  subVoice：减少音量  mediaRepeatMode：修改播放模式  说明：  mediaLike和mediaDislike仅华为音乐支持。 |
| voiceStep100 | 数值 | 选填 | 音量变化的步进值，默认为10，支持表达式。 |
| condition | 数值 | 选填 | 触发条件，1为触发，支持表达式。 |

### 应用示例

```
<CommonWallpaper>
        <MediaControl/>

        <Button h="200"  w="200" x="700" y="100">
		<Trigger action="up">
		<VibrateCommand id ="a" type="haptic.common.success" define = "false" />
			<MediaCommand command="mediaPlay" condition="1" />
		</Trigger>
	</Button>

	<Text align="center" color="#000000" size="60" text="停止" x="750" y="400" />
	<Button h="200"  w="200" x="700" y="400">
		<Trigger action="up">
			<MediaCommand command="mediaPause" condition="1" />
		</Trigger>
	</Button>

	<Text align="center" color="#000000" size="60" text="收藏" x="750" y="600" />
	<Button h="200"  w="200" x="700" y="580">
		<Trigger action="up">
			<MediaCommand command="mediaLike" condition="1" />
		</Trigger>
	</Button>

	<Text align="center" color="#000000" size="60" text="取消收藏" x="750" y="900" />
	<Button h="200"  w="200" x="700" y="890">
		<Trigger action="up">
			<MediaCommand command="mediaDislike" condition="1" />
		</Trigger>
	</Button>

	<Text align="center" color="#000000" size="60" text="上一首" x="200" y="1350" />
	<Button h="100"  w="200" x="180" y="1350">
		<Trigger action="up">
			<MediaCommand command="skipToPrevious" condition="1" />
		</Trigger>

			<MediaCommand command="subVoice" condition="1" />

	</Button>

	<Text align="center" color="#000000" size="60" text="切换模式" x="200" y="1950" />
```

## 音乐歌词 &lt;LrcView&gt;

提供两种歌词控件：

* <strong>多行歌词控件</strong>，最小展示三行歌词，适用于锁屏和桌面。
* <strong>一句歌词控件</strong>，仅展示一句歌词，适用于桌面万象小组件。

![](./img/28f5efc995ca.png)

1. 音乐歌词&lt;LrcView&gt;仅华为音乐支持。
2. 在使用音乐歌词 &lt;LrcView&gt;时，需要先声明[音乐控制器&lt;MediaControl&gt;](#section19781154113311)。

### 多行歌词控件

支持设计歌词控件的高度和宽度，获取的歌词数据在控件内进行填充，最小展示三行歌词。

* <strong>XML规范</strong>

```
<LrcView color="" size="" highColor="" highSize="" x="" y=""  w="" h=""/>
```

* <strong>参数说明</strong>

| 参 数 | 类 型 | 选 项 | 注 释 |
| --- | --- | --- | --- |
| color | 字符串 | 必填 | 普通歌词的颜色，支持表达式。 |
| size | 数值 | 选填 | 普通歌词的文字大小，支持表达式。 |
| highColor | 字符串 | 选填 | 高亮歌词的颜色，支持表达式。不填则与普通歌词保持一致。 |
| highSize | 数值 | 选填 | 高亮歌词的文字大小，支持表达式。不填则与普通歌词保持一致。 |
| w | 数值 | 选填 | 歌词控件的宽度。支持表达式。 |
| h | 数值 | 选填 | 歌词控件的高度，最小展示三行歌词。支持表达式。 |
| dividerHeight | 数值 | 选填 | 歌词的行间距，如果没有设置，默认为20。 |

* <strong>应用示例</strong>

```
<LrcView color="#0000ff" size="40" highColor="#ff0000" highSize="60" x="10" y="800" w="1080" h="100"/>
```

### 一句歌词控件

只展示一句歌词。支持设计歌词控件的宽度，歌词未超出控件宽度时，展示为一行；歌词超出控件宽度时，展示为两行且第二行居中显示，超出两行部分展示为省略号。

* <strong>XML规范</strong>

```
<LrcView  maxLines="2" highColor="" highSize="" x="" y=""  w="" />
```

* <strong>参数说明</strong>

| 参数 | 类型 | 选项 | 注释 |
| --- | --- | --- | --- |
| maxLines | 数值 | 必填 | 设置maxLines="2"，实现展示一句歌词：歌词未超出控件宽度时，展示为一行；歌词超出控件宽度时，展示为两行且第二行居中显示，超出两行部分展示为省略号。 |
| highColor | 字符串 | 选填 | 歌词的颜色，支持表达式。 |
| highSize | 数值 | 选填 | 歌词的文字大小，支持表达式。 |
| w | 数值 | 选填 | 歌词控件的宽度，支持表达式。 |
| dividerHeight | 数值 | 选填 | 歌词的行间距，如果没有设置，默认为20。 |

* <strong>应用示例</strong>

```
<LrcView maxLines="2" highColor="#ff0000" highSize="60" x="10" y="800" w="800"/>
```

## 音乐进度条&lt;MediaProgressbar&gt;

![](./img/199d865ecf06.png)

在使用音乐进度条&lt;MediaProgressbar&gt;时，需要先声明[音乐控制器&lt;MediaControl&gt;](#section19781154113311)。

### XML规范

```
<MediaProgressbar  topColor="" bottomColor="" bgSrc="" iconSrc="" x="" y="" h="" w="" iconWidth="" iconHeight=""/>
```

### 参数说明

| 参 数 | 类 型 | 选 项 | 注 释 |
| --- | --- | --- | --- |
| topColor | 字符串 | 必填 | 已进行进度颜色，支持表达式。 |
| bottomColor | 字符串 | 必填 | 未进行的进度颜色，支持表达式。 |
| bgSrc | 字符串 | 选填 | 进度条的背景图。 |
| iconSrc | 字符串 | 选填 | 跟随进度的小图标，支持表达式。 |
| iconWidth | 数值 | 必填 | 小图标的宽度，支持表达式。 |
| iconHeight | 数值 | 选填 | 小图标的高度，支持表达式。 |
| isRightAngle | 字符串 | 选填 | 表示进度条角度，值为true时进度条为直角，非true或不填为圆角。 |

x、 y、h和w这4个参数规定了&lt;MediaProgressbar&gt;控件的位置和大小。

### 应用示例

```
<MediaProgressbar  topColor="#550000" bottomColor="#ff0000" bgSrc="k1.png" iconSrc="tkr.png" x="100" y="1650" h="100" w="800"  iconWidth="60" iconHeight="60"/>
```

## 专辑封面&lt;Medialcon&gt;

获取到音乐App中的专辑封面图片时，展示获取到的专辑封面。

未获取到音乐App中的专辑封面图片时，展示通过&lt;MediaIcon&gt;设置的默认专辑封面。

![](./img/dd7ba257f439.png)

在使用专辑封面&lt;Medialcon&gt;时，需要先声明[音乐控制器&lt;MediaControl&gt;](#section19781154113311)。

### xml规范

```
<MediaIcon x="" y="" w="" h="" src=""/>
```

### 参数说明

![](./img/2223157f40e3.png)

&lt;Medialcon&gt;支持[图片&lt;Image&gt;](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/view-0000001073865717/image-0000001073948153)的参数设置。

| 参 数 | 类 型 | 选 项 | 注 释 |
| --- | --- | --- | --- |
| src | 字符串 | 必填 | 没有歌曲封面时展示的默认图。 |

### 应用示例

```
<MediaIcon x="40" y="580" w="400" h="400" src="bg.jpg"/>
```
