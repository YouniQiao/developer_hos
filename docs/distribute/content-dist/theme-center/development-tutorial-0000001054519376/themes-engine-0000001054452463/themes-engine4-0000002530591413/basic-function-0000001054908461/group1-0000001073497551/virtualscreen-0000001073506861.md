# 虚拟屏幕&lt;VirtualScreen&gt;

## 功能概述

虚拟屏幕允许一系列的View的复用，虚拟屏幕给一系列的View提供了容器，减少设计师的工作量。

## XML规范

```
<VirtualScreen name="">
  <Image x="" y="" src=""/>
  <Image x="" y="" src=""/>
</VirtualScreen>
```

## 参数说明

| 参 数 | 类 型 | 选 项 | 注 释 |
| --- | --- | --- | --- |
| name | 字符串 | 必填 | 虚拟屏幕的标注，Image的src属性调用的名称 |

![](./img/0456fd15eaa9.png)

其中Image标签的src属性的值是被调用的VirtualScreen的标签name的值，其可以将一系列的元素重复使用; 如果使用自定义宽高的虚拟屏幕，需要在使用虚拟屏幕的标签上同时填写宽高限制。

## 应用示例

<strong>示例一：</strong>将多张图片同时控制在虚拟屏幕中展示；Image标签的宽、高会设置为虚拟屏幕的宽和高，绘制起点的坐标为Image标签的x和y。

```
<VirtualScreen name="vs">
  <Image x="0" y="0" src="dianhua.png"/>
  <Image x="100" y="100" src="ty.png"/>
</VirtualScreen>
<Image x="0" y="0" w="1080" h="1920" src="vs" useVirtualScreen="true"/>
<Image x="200" y="200" w="1080" h="1920" src="vs" useVirtualScreen="true"/>
<Image x="400" y="400" w="1080" h="1920" src="vs" useVirtualScreen="true"/>
```
