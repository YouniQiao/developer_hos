# 运动健康数据开放&lt;Healthy&gt;

## 功能概述

开放运动健康App的步数、距离、卡路里、实时心率和静息心率数据，设计师可以根据这些数据制作创意主题。

![](./img/8a29236c55a3.png)

1. 用户需打开运动健康App，同意用户协议并登录账号，才能获取到步数、距离和卡路里数据。在此基础上，还需绑定支持心率功能的华为智能穿戴设备，才能获取到实时心率和静息心率数据。
2. 用户手机安装的华为运动健康App版本必须高于11.0.0.512。
3. 运动健康数据支持应用于桌面和桌面万象小组件，不支持应用于锁屏。

## 场景举例

* 在屏幕中设计一个人物形象，卡路里消耗的越多，人物形象越苗条。
* 当步数达到一定数值时，背景图更改。

## XML规范

```
<Healthy polling="" time="" >
    <Var name="" />
</Healthy>
```

## 参数说明

| 参 数 | 类 型 | 选 项 | 注 释 |
| --- | --- | --- | --- |
| polling | 布尔值 | 选填 | 轮询开关，true为开启轮询，false为关闭轮询。默认是false。 |
| time | 数值 | 选填 | 轮询时间间隔，单位（秒），最小值为1，仅在polling为true的时候有效。默认是5秒。 |
| name | 字符串 | 必填 | 参照下面运动健康数据类型取值。 |

## 运动健康数据类型

| 名称和取值 | 含义 | 最快更新频率 | 单位说明 |
| --- | --- | --- | --- |
| steps\_value | 步数 | 秒级 | 步 |
| distance\_value | 距离 | 秒级 | 米 |
| calories\_value | 卡路里 | 秒级 | 卡 |
| heart\_rate\_value | 实时心率（运动时的心率） | 5秒级 | 次/分钟 |
| resting\_heart\_rate\_value | 静息心率（休息时的心率） | 小时级 | 次/分钟 |

![](./img/112b0d294e65.png)

1. 通过建立Var标签，设置Var标签name参数的值调用运动健康数据。如需调用多种运动健康数据，则需建立对应数量的Var标签。
2. 开启轮询开关并设置轮询时间间隔后，将按照轮询时间间隔自动刷新运动健康数据。

## 运动健康数据刷新命令&lt;RefreshHealthyCommand&gt;

&lt;RefreshHealthyCommand&gt;主要应用于需要手动刷新运动健康数据的场景。

应用示例：通过点击事件刷新当天的运动健康数据。

```
<Image x="600" y="500" h="250" w="260" src="bj.jpg"/>
<Button x="100" y="1000" w="300" h="150">
     <Trigger action="click">
         <RefreshHealthyCommand/>
     </Trigger>
</Button>
```

![](./img/2866c8c9aa68.png)

&lt;RefreshHealthyCommand&gt;是针对所有运动健康数据相关的Var标签的手动刷新，一秒内重复点击只会触发一次刷新。

## 应用示例

每隔5秒刷新一次步数、距离、卡路里、实时心率、静息心率的值，并通过文本展示出来。同时支持手动点击按钮，刷新这些数据。

```
<Healthy polling="true" time="5" >
   <Var name="steps_value" />
   <Var name="distance_value" />
   <Var name="calories_value" />
   <Var name="heart_rate_value" />
   <Var name="resting_heart_rate_value" />
   <Var name="sport_value" />
</Healthy>

<Text name="t1" color="#ffffff" format="步数总值:%s" paras="#steps_value" size="50" x="100" y="100" />
<Text name="t2" color="#ffffff" format="距离总值:%s" paras="#distance_value" size="50" x="100" y="200" />
<Text name="t3" color="#ffffff" format="卡路里总值:%s" paras="#calories_value" size="50" x="100" y="300" />
<Text name="t4" color="#ffffff" format="实时心率:%s" paras="#heart_rate_value" size="50" x="100" y="400" />
<Text name="t5" color="#ffffff" format="静息心率:%s" paras="#resting_heart_rate_value" size="50" x="100" y="500" />

<Text name="t10" color="#ffffff" format="手动刷新" size="50" x="100" y="1000" w="300" h="150" />
<Button x="100" y="1000" w="300" h="150">
   <Trigger action="click">
      <RefreshHealthyCommand/>
   </Trigger>
</Button>
```
