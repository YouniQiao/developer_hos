# 动态图片&lt;APNG&gt;

## 动效概述

Image的增强功能，支持APNG格式图片循环播放。APNG是基于PNG格式的位图动画格式图片，是PNG的位图动画扩展，可以实现PNG格式的动态图片效果。

使用APNG，可以减少脚本量，提升性能。

## 参数说明

| 参数 | 类型 | 选项 | 注释 |
| --- | --- | --- | --- |
| src | 字符串 | 必填 | 图片文件名，如：image.png，也可在前面加文件夹名称，如wenjianjia/image.png。 |
| supportApng | 字符串 | 选填 | 当且仅当该参数为true时，&lt;Image&gt;支持Apng类型的资源文件播放，此时资源文件类型为APNG，后缀名应为".png"。说明：需自行制作.png格式的APNG动图。 |
| apngLoopCount | 数值 | 选填 | APNG动画循环播放次数，当该参数为-1时，无限循环播放APNG动画。 |
| isRefresh | 字符串 | 选填 | APNG动画隐藏后（visibility="0"），再次显示出来时（visibility="1"），是否刷新状态重新从第一帧开始播放，为true则重新从第一帧开始播放，默认为false。 |

参数

类型

注释

src

字符串

必填

图片文件名，如：image.png，也可在前面加文件夹名称，如wenjianjia/image.png。

supportApng

字符串

选填

当且仅当该参数为true时，&lt;Image&gt;支持Apng类型的资源文件播放，此时资源文件类型为APNG，后缀名应为".png"。

需自行制作.png格式的APNG动图。

apngLoopCount

数值

选填

APNG动画循环播放次数，当该参数为-1时，无限循环播放APNG动画。

isRefresh

字符串

选填

APNG动画隐藏后（visibility="0"），再次显示出来时（visibility="1"），是否刷新状态重新从第一帧开始播放，为true则重新从第一帧开始播放，默认为false。
