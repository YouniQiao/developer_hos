# 麦克风音量感知&lt;Microphone&gt;

## 功能概述

麦克风音量感知功能根据全局变量microphone\_state、microPhone\_volume和isSupportMicro实现。

通过调用这些全局变量，实现麦克风音量感知创意主题。

## 应用场景

通过对手机的麦克风吹气，触发锁屏和桌面上的创意动效。例如：

* 在桌面上，对着手机麦克风吹气，风车开始转动。
* 在桌面上，对着手机麦克风吹气，使火苗燃烧更加旺盛，照亮对面的小伙伴。

## 功能说明

* 在锁屏上使用麦克风音量感知会触发锁屏弹窗，因此为保障用户体验，该功能仅支持在桌面和桌面万象小组件中使用，不支持在锁屏上使用。
* 主题APP需开启麦克风权限（设置&gt;应用管理&gt;主题&gt;权限&gt;麦克风），才能正常应用麦克风音量感知功能。
* 主题APP仅使用麦克风权限，不会做录音行为，不会收集用户语音。

## 变量说明

| 变 量 | 类  型 | 选 项 | 注 释 |
| --- | --- | --- | --- |
| isSupportMicro | 数值 | 必填 | 开关变量（固定名称）：1 开启录音； 0 停止录音。默认为0。 |
| microPhone\_volume | 数值 | 必填 | 麦克风分贝值，实时获取。 |
| microphone\_state | 数值 | 选填 | 麦克风吹气状态：1 短吹 ；2 长吹；0 无状态。默认为0。 |

制作麦克风音量感知主题时，必须设置变量isSupportMicro，以开启录音开关，示例如下：

```
<Var name="isSupportMicro" expression="0" persist="false"/
<Image x="100" y="400" w="258" h="258" src="dian.png" />
<Button x="100" y="400" w="258" h="258" >
    <Trigger action="down">
	<VariableCommand name="isSupportMicro" expression="not(#isSupportMicro)" />
    </Trigger>
</Button>
```
