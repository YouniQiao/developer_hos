# 图片混合&lt;GroupImage&gt;

## 功能概述

基于Group标签，可以对组内的多张图片实现混合模式的效果。其中，Group作为容器，组内最后一张图片设置混合模式参数，用于对之前所有的图片进行一个混合操作（除最后一张图片外所有图片作为一个整体）。不支持Mask标签中的绝对位置方式。

## 应用场景

* 可用于将多张图片叠加显示组成一个新的图片。

* 可以通过图层混合叠加达到微立体的效果。

## XML规范

```
<Group x="" y="" w="" h="" layered="">
    <Image x="" y="" align="" alignV="" src=""/>
    <Image x="" y="" align="" alignV="" src="" hybridMode=""/>
</Group>
```

## 参数说明

| 参 数 | 类 型 | 选 项 | 注 释 |
| --- | --- | --- | --- |
| layered | 字符串 | 必填 | true/false，是否开启图片混合功能；开启后会对Group中的图片进行混合操作 |
| hybridMode | 字符串 | 必填 | 图片混合模式，支持12种混合模式，具体参考Mask标签 |

## 应用示例

<strong>示例一：</strong>该示例在Group标签中设置layered参数为true，开启了图片混合操作，并在最后一个Image标签中设置了xor混合模式。

```
<Group x="100" y="100" w="800" h="800" layered="true">
    <Image x="100" y="100" align="center" alignV="center" src="tgt1.png"/>
    <Image x="200" y="200" align="center" alignV="center" src="tgt2.png"/>
    <Image x="261" y="261" align="center" alignV="center" src="ori.png" hybridMode="11"/>
</Group>
```
