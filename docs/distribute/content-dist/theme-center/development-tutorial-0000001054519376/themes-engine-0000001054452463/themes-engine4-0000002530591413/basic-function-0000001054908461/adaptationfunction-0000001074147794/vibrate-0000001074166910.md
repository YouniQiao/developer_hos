# 震动设置&lt;Vibrate&gt;

## 功能概述

通过&lt;Vibrate&gt;参数，控制[slider](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/control-0000001073947889/slider-0000001073875027)，[unlocker](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/control-0000001073947889/unlocker-0000001073866952)和[externCommand](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/orders-0000001073987886/externcommand-0000001074644528)三种组件在达到一定条件时是否触发震动。使用时有一些限制：

* &lt;Vibrate&gt;作为Lockscreen的一个参数使用，通过为其赋值为true/false，控制震动的开关。
* 目前项目中只有此三种组件，支持在达到一定条件时候时触发震动：[Slider](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/control-0000001073947889/slider-0000001073875027)、[Unlocker](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/control-0000001073947889/unlocker-0000001073866952)和[ExternCommand](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/orders-0000001073987886/externcommand-0000001074644528)。
* 只能打开或关闭震动，打开后震动效果单一，震动时间固定为50ms。不能切换震动的类型、不能自定义设置震动的时长、强弱和间隔时间。
* 无机型限制。

![](./img/49d48d88cf93.png)

如果需要更多的组件支持触发震动，或者需要支持切换震动类型、自定义设置震动强弱和间隔时间，请使用[线性马达振动命令&lt;VibrateCommand&gt;](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/orders-0000001073987886/vibratecommand-0000001194207997) 。

## 应用场景

* 滑动解锁，伴随震动的触感。
* 执行解锁命令，伴随震动的触感。

## XML规范

```
<Lockscreen vibrate="" ></Lockscreen>
```

## 参数说明

| 参 数 | 类 型 | 选 项 | 注 释 |
| --- | --- | --- | --- |
| vibrate | 字符串 | 选填 | 震动的开关，只支持true/false字段，不设置的情况下，默认是false |

## 支持组件

| 组件名 | 触发时机 |
| --- | --- |
| [Slider](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/control-0000001073947889/slider-0000001073875027) | 1. 当Slider的StartPoint区域被触摸到的时候，StartPoint和EndPoint区域状态都被设置为Pressed\_state，并且有50ms的震动  2. 当Slider的触摸进入一个EndPoint区域的时候，产生50ms的震动  3. 当触摸区域在StartPoint区域，而抬手操作在EndPoint区域之外时，产生50ms的震动 |
| [Unlocker](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/control-0000001073947889/unlocker-0000001073866952) | 与Slider相同 |
| [ExternCommand](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/themes-engine-0000001054452463/themes-engine4-0000002530591413/basic-function-0000001054908461/orders-0000001073987886/externcommand-0000001074644528) | 当ExternCommand的command等于unlock时，触发ExternCommand时产生50ms的震动 |

## 应用示例

<strong>示例一：</strong>开启项目的震动开关，在以上支持组件中达到相应的条件会产生震动的效果。

```
<Lockscreen version="1" frameRate="30" screenWidth="1080" vibrate="true" ></Lockscreen>
```
