# 时间动画&lt;TimeAnimation&gt;

## 动效概述

时间元素刷新，当有用到time全局变量，会启动变量的刷新，time的值会按帧率进行刷新。

## XML规范

```
<Text x="" y="" align="" color="" size="" format="time：%d" paras="#time"/>
```

或

```
<Text x="" y="" size="" color="" text="#time"/>
```

| 参 数 | 类 型 | 选 项 | 注 释 |
| --- | --- | --- | --- |
| time | 数值 | 必填 | [0-~]，当前系统时间几小时几点，设置系统全局变量time，不小于0。 |
