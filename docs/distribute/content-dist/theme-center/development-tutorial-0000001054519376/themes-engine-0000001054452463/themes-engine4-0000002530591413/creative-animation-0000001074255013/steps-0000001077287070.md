# 步数换景

## 动效概述

通过全局变量#steps\_value控制图片的变化。例如当步数达到一定数值时，背景图更改。

可在主题App中搜索《遇见博物》进行体验和参考。

## 素材准备

（素材来源于《遇见博物》，可在主题App下载同款主题）
