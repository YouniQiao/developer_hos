# 穿越动效

## 动效概述

通过长按或者点击的交互方式，来实现左右浏览或者调节图层远近，实现多个图层叠加的立体效果。

可在主题App中搜索《森林脉络》进行体验和参考。

## 素材准备

部分素材如下所示，完整体验请在主题App中搜索《森林脉络》进行体验和参考。
