# 新充电动效&lt;EMUI11.0&gt;

## 动效概述

EMUI11.0及以上版本支持新充电动效。充电动效部分的资源添加在主题包unlock目录里。因引擎顺序解析导致图层的问题，xml中应依次添加锁屏资源和充电动效资源。

根据规范要求，界面必要元素为：充电动效、锁屏充电文字提示、充电图标、电量数值、超级快充文字提示、Max xxW的标签、背景蒙黑。包含充电动效的主题包，需要有对应新充电动效的锁屏预览图。

## 应用范围

新充电动效只支持在锁屏上使用，不支持在桌面和桌面万象小组件中使用。

## XML规范

unlock文件夹下有一个lockscreen文件夹和一个theme.xml文件。lockscreen文件夹中包含锁屏资源和充电动效资源。

theme.xml中新增字段&lt;item supportChargingAnim="true"/>，整体内容如下：

```
<?xml version="1.0" encoding="utf-8"?>
<HWTheme>
    <item style="slide" />
    <item package="com.huawei.ucdlockscreen" />
    <item dynamicPath="lockscreen" />
    <item minversion="2.0.24" />
    <item title-cn="华为官方主题" />
    <item title="Huawei Themes" />
    <item maxHeight="2160" />
    <item baseVersion="6.0" />
    <item engineType="HWThemeEngine" />
    <item supportChargingAnim="true"/>
</HWTheme>
```

支持开放的规格：

| 界 面 元 素 | 是否支持修改 | 规 格 |
| --- | --- | --- |
| 充电动效 | 支持修改充电动效；支持基于不同的初始电量区间，上传不同的充电动效；支持基于不同的充电类型上传不同的充电动效 | 时长：不超过9s，若超过9s或不足0s，按9s显示。 |
| 充电图标 | 不支持修改，见[充电图标模板](https://communityfile-drcn.op.hicloud.com/FileServer/getFile/cmtyManage/011/111/111/0000000000011111111.20201010180009.20112038905657885548848177070686%3A50511202002932%3A2800%3A683637EB944EE0B8BA2FFF69FC00DD38212F85E0A497BB3B497E61353075C9F8.zip?needInitFileName=true) | 模板提供图标，不能使用其他图标。 |
| 电量数值 | 支持修改字号，颜色 | 超级快充场景，显示2位小数。 |
| 超级快充文字提示 | 支持修改字号，颜色 | 非超级快充场景，不显示。 |
| Max xxW的Tag | 支持修改字号，颜色 | 带矩形框，功率大于40W时有放大动效。 |
| 背景 | 支持修改，需有透明度可看清锁屏时间 | 默认全屏75% #000000。 |

充电动效变量：

| 参 数 | 类 型 | 注 释 |
| --- | --- | --- |
| charge\_anim | 数值 | 当支持充电动效，并从非充电状态切换到充电状态时，该值为1，表示播放动效；动效播放停止，值为0 。 |
| charge\_type | 字符串 | 充电类型，值为有线和无线。 |
| charge\_type\_value | 数值 | 根据充电类型，划分的等级，有线充电时，值为1；无线充电时，值为2。根据规范要求，为了方便处理有线和无线两种情况，划分的等级。 |
| charge\_mode | 字符串 | 充电速率，取值为：有线普充. 有线快充. 有线超级快充. 无线普充. 无线快充. 无线超级快充。 |
| charge\_mode\_value | 数值 | 根据充电速率，划分的等级，1. 2. 3. 4. 5. 6。分别对应上面的充电速率。根据规范要求，对应的充电速率显示对应充电图标。 |
| charge\_max\_power | 字符串 | 最大充电功率，显示的值为：Max xxW，值为空时不显示。针对超级快充场景，需要手机支持才能显示（目前有线超级快充有40W，无线超级快充有27W和40W）。 |
| charge\_power\_value | 数值 | 根据充电功率，划分的等级，当功率为27W时，值为1；功率为40W时，值为2；大于40W时，值为3。根据规范要求，当值为3（即功率大于40W）时，最大功率标签需要有放大的动效。 |
| charge\_level | 数值 | SystemUI传输的电量值，针对超级快充，会显示2位小数。 |
| wireless\_center\_X | 数值 | 无线充电中心x轴，单位px。 |
| wireless\_center\_Y | 数值 | 无线充电中心y轴，单位px。 |
| super\_charge | 字符串 | 超级快充，显示“超级快充”的字样。值为空时不显示。 |
| charge\_info | 字符串 | 充电信息，显示信息为：正在(无线)充电/(无线)快速充电/(无线)超级快充 xx%。包含反向充电的信息。根据规范要求，需要充电中显示该字样。 |

<strong>锁屏充电交互规则</strong>

* 充电动效播放过程中能被用户手势（点击、滑动操作、来电）打断，打断后，动效消失，显示锁屏。
* 切换到大主题后，解锁后桌面不再显示原生充电动效。
* 从锁屏拉起输入密码界面时，会销毁锁屏引擎，所以输入密码界面不支持换肤充电动效。
* 其他如省电模式、应用杂志锁屏等场景，未调用锁屏引擎，均不支持换肤充电动效。
