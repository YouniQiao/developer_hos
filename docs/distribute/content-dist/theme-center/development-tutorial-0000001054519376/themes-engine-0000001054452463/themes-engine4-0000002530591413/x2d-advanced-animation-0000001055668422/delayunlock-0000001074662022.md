# 延时解锁&lt;DelayUnlock&gt;

## 动效概述

延时解锁基于Slider和ExternCommand功能实现，允许用户在执行解锁操作一定时间后（延迟时间）执行系统解锁命令。关键在于执行ExternCommand命令时增加延时参数delay，用于控制延迟解锁的时间。

## 场景举例

* 融入俄罗斯方块小游戏，实现趣味解锁。
* 点击燃放鞭炮，鞭炮燃放完毕后执行解锁。
* 点击喂食宠物，宠物吃完食物后执行解锁。

## XML规范

```
<ExternCommand command="unlock" delay=""/>
```

| 参数 | 类 型 | 选 项 | 注释 |
| --- | --- | --- | --- |
| command | 字符串 | 必填 | 通用命令标签，用于向外部程序发送解锁命令。 |
| delay | 数值 | 必填 | 通用命令标签，用于向外部程序发送解锁命令，delay表明延迟解锁时间，单位为ms。 |
