## 3.静态AOD的制作设计指导

### 3.1 数字AOD设计指导

数字时钟AOD可以对时间数字进行设计，同时也可修改时间日期颜色，数字时钟支持在EMUI10.0以上的手机版本显示。

1. <strong>设计AOD样式，制作切图</strong>。

   设计和切图要求和注意事项

   * 时钟切图非填充区域应为透明镂空效果，单位.png。
   * 切图需要同时切单时钟和双时钟，注意尺寸。
   * 设计师寄语图片，需要有“设计师寄语”或者“设计说明”等说明字样，无需包含数字时钟。

   ![](./img/fc34cf1a4e39.png "点击放大")
2. <strong>工具制作AOD</strong>。
   1. 打开工具选择→新建→项目类型选择“AOD”。

   ![](./img/8b63dac67f1a.png "点击放大")

   2. 替换单双时钟数字切图（单双时钟都需替换)。

   ![](./img/c69713e41a7e.jpg "点击放大")

   3. 调整文字颜色与设计的数字和谐。

   ![](./img/81bd02f38c76.jpg "点击放大")

   4. 截屏生成预览效果，preview\_aod\_2.jpg上传设计师寄语。

   ![](./img/86b07d9f326b.jpg "点击放大")

   ![](./img/1180d6090fa2.jpg "点击放大")

   5. 导出主题包。

### 3.3 模拟时钟AOD设计指导

模拟时钟是可设计时钟表盘样式，以及时针和分针的样式，模拟时钟AOD支持在EMUI10.0以上的手机版本显示。

1. <strong>设计AOD样式，制作切图。</strong>

   设计和切图要求和注意事项

   * 时钟切图非填充区域应为透明镂空效果，单位.png。
   * 切图需要同时切单时钟和双时钟，注意尺寸。
   * 时针，分针的切图必须指向12点方向，显示内容区域必须为透明镂空效果。
   * 当前只支持普通顺时针显示满12小时刻度的表盘样式，其他特殊规格样式不支持。
   * 设计师寄语图片，需要有“设计师寄语”或者“设计说明”等说明字样，无需包含数字时钟。

     ![](./img/b33b22e69d07.png "点击放大")
2. <strong>工具制作AOD。</strong>
   1. 打开工具选择→新建→项目类型选择“AOD”。

      ![](./img/5d6556ed8adb.png "点击放大")
   2. 导入表盘，时钟和分钟，单双时钟都需要替换并调整文字颜色与设计的数字和谐。

      ![](./img/5ba59fb55af2.jpg "点击放大")

      ![](./img/a8cff87dd1f8.jpg "点击放大")
   3. 截屏生成预览效果，preview\_aod\_2.jpg上传设计师寄语。

      ![](./img/bea1c08b63b5.jpg "点击放大")

      ![](./img/e62f76fecd80.jpg "点击放大")
   4. 导出主题包。

### 3.5 微立体AOD设计指导

微立体AOD，3D视感，栩栩如生。

微立体时钟AOD为10.1新增样式，只支持在EMUI10.1以上的手机版本显示。

   设计和切图要求和注意事项
   * 切图非填充区域应为透明镂空效果，单位.png。
   * 双时钟下，时钟的背景设计样式需不一样。
   * 设计不能影响时钟的正常显示，保证所有时间的识别度。
   * 设计师寄语图片，需要有“设计师寄语”或者“设计说明”等说明字样，无需包含数字时钟。

## 5.自检自查

为了确保您尽快通过审核，在上传联盟之前，需要您自检自查。

1. 请根据[AOD自检规范](https://developer.huawei.com/consumer/cn/doc/content/aod-test-0000001056821215#section1655743143119)进行检查。
2. 检查黑色像素：Theme Studio已支持检测非黑色像素，制作时请按Theme Studio提示进行修改。非黑色的像素点所占整个屏幕像素点的比例需在15%以下。

## 6.上传联盟

自检完成后，通过[AOD上传指南](https://developer.huawei.com/consumer/cn/doc/content/themes-upload-0000001055029726#section96731130204020)在华为开发者联盟上传AOD，等待审核。

### 7.3 description文件规格说明

![](./img/397badf0eac0.png "点击放大")

![](./img/0bbfb9a73115.png)

1. 主题英文名，中文名，开发者名称，设计师名称四项待主题上线后均不可修改。
2. 设计师名称与设计师的开发者联盟账户一致。
3. 主题分辨率，主题英文字体，中文字体均采用默认不可以修改。
4. 主题版本号第一版为10.0.0，后续有更新则更改为10.0.X（X为阿拉伯数字）；大图AOD和微立体AOD，版本号第一版填写10.0.100，后续有更新则更改为10.0.10X（X为阿拉伯数字）；动态AOD，版本号第一版填写11.0.0，后续有更新则更改为11.0.X（X为阿拉伯数字）。

### 7.5 预览视频文件规范

* 视频尺寸为：1080×1920px；
* 视频格式为MP4，编解码制式要求为H.264，无音轨；
* 视频大小建议在10MB以内，时长建议在5秒以内；
* 预览视频首帧效果请与动态AOD的cover图保持一致，建议使用AOD的单时钟显示效果；
* 为保证预览效果，请提供能循环播放的视频，确保首尾衔接部分流畅，不会出现画面跳动或者闪烁。

  ![](./img/24725ea1f82f.png)

  1. 动态AOD支持上传预览视频文件，预览视频可以让你的动态AOD作品被更好地展示，有利于动态AOD销售。
  2. 请按上述规范准备动态AOD预览视频文件。该文件无需打包在动态AOD文件包内。上传时，直接在主题联盟上传。
