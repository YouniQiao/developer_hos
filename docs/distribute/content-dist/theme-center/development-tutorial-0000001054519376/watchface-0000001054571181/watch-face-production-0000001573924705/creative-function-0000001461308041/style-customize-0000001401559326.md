# 样式自定义

## 功能介绍

<strong>样式自定义</strong>：支持设计师为<strong>背景</strong>、<strong>时间</strong>和<strong>日期</strong>制作多个不同的自定义选项，然后将这些自定义选项自由组合起来，形成多个自定义样式。用户实际应用时，可自行切换组合好的自定义样式。

![](./img/434483939d42.png)

“样式自由组合”表盘与[“颜色自由组合”表盘](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/watchface-0000001054571181/watch-face-production-0000001573924705/creative-function-0000001461308041/color-customize-0000001459125433)功能互斥，只能选择其中一种进行制作。例如：当制作了“样式自由组合”后，再切换到“颜色自由组合”，会提示“暂未达到使用该功能条件”。反之亦然。

![](./img/ed768571fd8e.png "点击放大")

<strong>样式自定义表盘</strong> <strong>示例</strong>：以下示例表盘中，背景制作了2个自定义选项，时间也制作了2个自定义选项，将它们自由组合为4个自定义样式，提供给用户切换。

![](./img/35fe1deafbfb.png "点击放大")

<strong>样式自定义表盘使用方法</strong>：长按表盘 &gt; 点击“设置” &gt; 上下滑动，切换样式。

![](./img/78ffbcc23cb8.png "点击放大")

## 制作实操

本文以466\*466分辨率“样式自定义”表盘为例进行演示。

1. **新建一个466\*466表盘制作项目。**

   点击【+】，选择手表类型为【智能手表】，分辨率为【466 \*466】，然后输入表盘英文名。点击【确认】后，即可新建一个表盘制作项目。

   ![](./img/6a5f269b6c03.png "点击放大")
2. <strong>制作2个背景自定义选项。</strong>

   ① 选择“背景”&gt;“自定义选项”，然后点击“+”为背景-自定义选项1添加效果图。

   ![](./img/4917bf89def0.png "点击放大")

   ![](./img/271dadb3206b.png "点击放大")

   ![](./img/a5ed1e8eefda.png)

   背景的效果图，会与组合中的时间/日期的效果图叠加形成自定义样式的效果图，上表后在样式选择界面提供给用户预览。背景/时间/日期的效果图请以实际效果为准。

   ![](./img/8328359a1906.png "点击放大")

   ② 选择“背景”&gt;“单图”，然后点击“+”上传背景1的图片资源。

   ![](./img/7d0c0f5caade.png "点击放大")

   ③ 重复以上操作，制作背景-自定义选项2。

   选择“背景”&gt;“自定义选项”，然后点击“+”为背景-自定义选项2添加效果图。

   再选择“背景”&gt;“单图”，然后点击“+”上传背景2的图片资源。

   ![](./img/e26d1e122eac.png "点击放大")

   ![](./img/eecf60d5eee6.png "点击放大")

   ![](./img/64e014c4b83f.png)

   本示例中2个背景自定义选项都使用的单图控件。实际操作过程中，可添加更多背景自定义选项，也可以选择其他控件。
3. <strong>制作2个时间自定义选项</strong>。

   ① 选择“时间”&gt;“自定义选项”，然后点击“+”为时间-自定义选项1添加效果图。

   ![](./img/a7855959da56.png "点击放大")

   ② 按照常规表盘的制作方法，在时间-自定义选项1下，制作模拟时钟，包括时针、分针、秒针。

   ![](./img/8dcc766ee2b6.png "点击放大")

   ③ 重复以上操作，制作时间-自定义选项2。

   选择“时间”&gt;“自定义选项”，然后点击“+”为时间-自定义选项2添加效果图。

   按照常规表盘的制作方法，在时间-自定义选项2下，制作模拟时钟，包括时针、分针、秒针。

   ![](./img/5d66e17e15ab.png "点击放大")

   ![](./img/de578957fb09.png "点击放大")
4. <strong>创建样式自由组合组合。</strong>

   ① 背景和时间的2个自定义选项制作完成之后，点击“自由组合”&gt;“样式自由组合”，进入样式自由组合制作页面。

   点击“创建新组合”，即可开始创建样式自由组合组合。

   ![](./img/c56a29b7400d.png "点击放大")

   ② 将背景-自定义选项1和时间-自定义选项1勾选，然后点击“保存”，组合成自定义样式1。

   ![](./img/bb9ecea24530.png "点击放大")

   ![](./img/f8ef16486f9f.png "点击放大")

   ③ 继续点击“创建新组合”，创建新的组合。示例表盘中创建了四种不同的组合，即制作了4种自定义样式，同时可以设置默认显示的自定义样式。

   * 自定义样式1：背景-自定义选项1 + 时间-自定义选项1。
   * 自定义样式2：背景-自定义选项1 + 时间-自定义选项2。
   * 自定义样式3：背景-自定义选项2 + 时间-自定义选项1。
   * 自定义样式4：背景-自定义选项2 + 时间-自定义选项2。

   ![](./img/0fb097248f24.png "点击放大")

   ![](./img/02cdce1452d8.png "点击放大")
5. <strong>制作AOD表盘。</strong>

   ① 点击“半亮模式”，进入AOD表盘制作页面。

   ![](./img/f12e1ff95d9e.png "点击放大")

   ② 复制全亮表盘中的需要的图层，点击“确定”保存。

   直接复制的背景图层非黑色像素占比过高，不适用于AOD表盘，我们将其替换成合适的背景图片。

   AOD中的背景和时间，将跟随全亮模式中设置好的自定义组合方式进行展示。

   ![](./img/f213c2041a7a.png "点击放大")

   ![](./img/459ac649a8f7.png "点击放大")
6. <strong>预览图制作。</strong>

   表盘制作完成后，可在动态预览区点击 ![](./img/b85bf5476406.png "点击放大") 或拉动进度条，预览表盘动态效果。

   预览无误后，点击 ![](./img/ded76c52eac1.png "点击放大") 将数据恢复至默认状态（截图之前必须将数据恢复至默认状态）。

   然后点击“预览”-“截屏生成展示图”，自动生成三张预览图（cover.jpg、aod.jpg、icon.jpg）和一张缩略图（res.png）。

   ![](./img/f44290b88125.png "点击放大")
7. <strong>表盘导出。</strong>

   截图完成后，点击“导出”，填写以下信息后，点击“确认”导出。

   ![](./img/2970b42daeb1.png "点击放大")

   导出后获得以下1个文件：xxxx.hwt （xxxx为表盘英文名称）。

   xxxx.hwt：表盘资源包，后续将对其进行测试、自检与上传到联盟。

## 常见问题

<strong>1、提示“自定义样式未保存、请在图层区自有组合中进行保存”如何解决？</strong>

Theme Studio提示：

![](./img/b625aa8effb6.png "点击放大")

问题原因：自定义样式表盘需要对自定义样式的组合进行保存，以达到上表之后自由切换的效果。

Theme Studio 解决方式：

1. 点击【表盘图层区】边上的【自由组合】标签，点击【创建组合】。

   ![](./img/5580c2ddfd2c.png "点击放大")
2. 选择【栏目预览】对自定义样式进行组合，点击保存。

   ![](./img/1f2bcdbf337e.png "点击放大")
3. 点击【创建新组合】可创建多个样式。

   ![](./img/659936f01f88.png "点击放大")
