# 景深

## 功能介绍

<strong>景深功能</strong>：轻晃手表时，表盘上使用了景深功能的图层能够随着手表晃动，产生位移和景深，带来手表屏幕内有3D深度的视觉感受。

<strong>景深表盘</strong> <strong>示例</strong>：轻晃手表时，左上方的树叶、下方的老虎、草堆、花朵等图层就随着手表晃动，产生了位移和景深，带来生动有趣且富有互动性的动态效果。

## 制作实操

1. <strong>新建表盘制作项目。</strong>

   点击【+】，选择手表类型为【智能手表】，分辨率为【466 \*466】，然后输入表盘英文名。点击【确认】后，即可新建一个表盘制作项目。

   ![](./img/8393bf196178.png "点击放大")
2. <strong>打开景深效果开关。</strong>

   当前仅智能系列支持制作“景深”表盘，图层的“景深效果”默认关闭。

   点击【切类型】，取消勾选【轻智能系列】，点击【确定】保存。此时图层的“景深效果”入口就会展示出来。

   ![](./img/990a2f552a45.png "点击放大")
3. <strong>制作表盘背景。</strong>

   ① 点击【背景】&gt;【单图】，新建一个单图图层。

   ![](./img/ad617a48b051.png "点击放大")

   ② 选中单图图层，然后点击【+】导入表盘背景图。

   ![](./img/ad7962d05e68.png "点击放大")

   ![](./img/06419911ac93.png)

   在背景中添加图片资源时，该图片资源超出表盘圆形区域的部分会被裁剪。如果对该图片资源设置景深效果，则晃动手表图片位移时，可能会出现漏出黑色边缘的情况。因此，<strong>如果设计中有超出表盘圆形区域的图片资源，并且想要为该图片设置景深效果，则该图片建议不添加在背景中，可以根据设计需要，添加在时间、日期或控件中</strong>，这样超出表盘圆形区域的部分不会被裁剪，实现比较好的景深效果。
4. <strong>制作时间。</strong>

   ① 点击【时间】&gt;【选图】，数值类型选择【时钟高位】。然后点击【+】，批量导入0-2的数字切图，并调整其位置为（250,40）。

   ![](./img/4ec62ccd9cbf.png "点击放大")

   ② 再次点击【时间】&gt;【选图】，数值类型选择【时钟低位】。然后点击【+】，批量导入0-9的数字切图，并调整其位置为（325,40）。

   ![](./img/9d36f4e62b22.png "点击放大")

   ③ 再次点击【时间】&gt;【选图】，数值类型选择【分钟高位】。然后点击【+】，批量导入0-5的数字切图，并调整其位置为（250,130）。

   ![](./img/ef45a362289b.png "点击放大")

   ④ 再次点击【时间】&gt;【选图】，数值类型选择【分钟低位】。然后点击【+】，批量导入0-9的数字切图，并调整其位置为（325,130）。

   ![](./img/73b43867056a.png "点击放大")

   ⑤ 点击【时间】&gt;【单图】。然后点击【+】，导入树叶切图，并调整其位置为（0,-30）。然后打开【景深效果】按钮，设置参数为0.8。

   ![](./img/0413183a7b80.png "点击放大")

   ![](./img/21bc510eb725.png)

   1. 为树叶图层设置景深，想实现树叶图层移动后，盖住时间的效果，因此将树叶图层放在时间图层上面。
   2. 景深效果的值范围为-2到2，正值表示图层移动方向与手表晃动方向相同，负值则相反。设置的值越靠近-2或2，图层移动的幅度越大。
   3. 景深效果最多支持4个层级，一个参数值即一个层级。例如，当前设置了4个景深层级（-1.2 -0.6 0.8 1.2）以后，其他图层的景深效果只能从这4个层级中进行选择， 不能再新建其他景深层级。此时如果想要修改景深参数，则需要先删除不需要的景深层级，再新建新的景深层级。
   4. 弧形图、视频、GIF控件不支持景深效果。
5. <strong>制作日期。</strong>

   ① 点击【日期】&gt;【连接文本】，第一个数值类型选择【月数据】，第2个数值类型选择【日期】，连接属性选择【点.】，并调整其位置为（163,98），大小为宽68。

   ![](./img/4866f6ef0b8a.png "点击放大")

   ② 点击【日期】&gt;【文本】，数值类型选择【星期文本】。并调整其位置为（165,132），大小为宽42。

   ![](./img/7fd5c6403e5c.png "点击放大")

   ![](./img/5a6b34369986.png "点击放大")

   ② 新建【单图-1】图层后，同时会新建一个【容器2】图层。调整【容器2】的位置为（66,99），大小为宽100高100。

   ![](./img/304d4782ec17.png "点击放大")

   ② 新建【单图-1】图层后，同时会新建一个【容器3】图层。调整【容器3】的位置为（47,233），大小为宽88高40。

   ![](./img/dcc0e8999636.png "点击放大")

10. <strong>制作AOD表盘</strong> <strong>。</strong>

    ① 点击【半亮模式】，进入AOD表盘制作页面。

    ![](./img/0a1fd78eb91e.png "点击放大")

    ② 点击【复制全亮图层】，勾选时间和日期图层后，点击【确认】，即可将选中的图层复制到AOD表盘中来。

    ![](./img/57be7ca3337c.png "点击放大")
11. <strong>截图生成预览图。</strong>

    表盘制作完成后，可在动态预览区点击 ![](./img/f956334e0929.png "点击放大") 或拉动进度条，预览表盘动态效果。

    预览无误后，点击 ![](./img/6b3e67a48e42.png "点击放大") 将数据恢复至默认状态（截图之前必须将数据恢复至默认状态）。

    然后点击“预览”-“截屏生成展示图”，自动生成三张预览图（cover.jpg、aod.jpg、icon.jpg）和两张缩略图（res.png）。

    ![](./img/27e0754757dc.png "点击放大")
12. <strong>导出表盘资源包。</strong>

    截图完成后，点击“导出”，填写以下信息后，点击“确认”导出。

    版本号说明：制作了景深功能的表盘版本号为1.1.X。

    ![](./img/673553ebd32d.png "点击放大")

    导出后获得以下1个文件：xxxx.hwt （xxxx为表盘英文名称）。

    xxxx.hwt：表盘资源包，后续将对其进行测试、自检与上传到联盟。
