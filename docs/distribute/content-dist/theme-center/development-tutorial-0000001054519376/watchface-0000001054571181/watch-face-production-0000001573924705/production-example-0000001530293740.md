# 制作实例

## 说明

1. 下文以466\*466分辨率为例进行演示，其他分辨率的制作方法类似，只是支持的能力集有一些差异，详见：
   * [466\*466能力集](https://developer.huawei.com/consumer/cn/doc/content/466-capability-0000001881726154)
   * [390\*390/454\*454能力集](https://developer.huawei.com/consumer/cn/doc/content/454-capability-0000001580733885)
   * [280\*456/336\*480能力集](https://developer.huawei.com/consumer/cn/doc/content/280-capability-0000001592176765)
   * [194\*368能力集](https://developer.huawei.com/consumer/cn/doc/content/194-capability-0000001591976781)
2. 466\*466有两种制作模式：WATCH系列+GT系列（2.y.z）和WATCH系列（1.y.z）制作模式。两种制作模式支持不同的能力集，需点击【切类型】进行切换，详见制作模式切换。

   ![](./img/758823aafb87.png "点击放大")

## 背景

### 单图

<strong>通过单图控件，制作表盘静态背景。</strong>

<strong>切图准备</strong> <strong>：</strong>

![](./img/9aecfe3e9a41.png "点击放大")

![](./img/d093f5bb13e5.png)

<strong>切图说明：</strong>

* 背景避免白色像素过多，饱和度/亮度过高，否则会导致手表功耗过大，耗电量较快。Theme Studio支持亮度校验，亮度过高不可导出。
* 设计背景时，静态元素尽量与背景融合以减少图层及图片大小。

<strong>制作步骤：</strong>

1. 添加“背景”&gt;“单图”控件。
2. 导入表盘背景图。

![](./img/0518b9befb39.png "点击放大")

### 选图

<strong>通过选图控件，实现表盘静态背景切换，最多支持切换5个静态背景。</strong>

<strong>切图准备：</strong>

![](./img/365200b01a0a.png "点击放大")

<strong>制作步骤：</strong>

1. 添加“背景”&gt;“选图”控件。
2. 数值类型选择“背景切换”。
3. 导入5张表盘背景图。

![](./img/423b18356490.png "点击放大")

### 动图

<strong>通过动图控件，上传动图资源，制作表盘动态背景。</strong>

<strong>动图准备：</strong>

![](./img/4330ce00eb3a.png)

![](./img/540a2062a39f.png)

<strong>动图规格：</strong>

* 动图尺寸：466\*466
* 动图大小：≤10M
* 动图时长：5-10s
* 动图格式：GIF
* 动图帧率：15-30帧/秒
* 最多可以添加1个动图
* 建议素材首尾能相接循环，不能有明显跳帧痕迹

<strong>制作步骤：</strong>

1. 添加“背景”&gt;“动图”控件。
2. 导入动图资源和动图预览图。

![](./img/3e0cd204e018.png "点击放大")

### 视频

<strong>通过视频控件，上传视频资源，制作表盘动态背景。</strong>

<strong>视频准备：</strong>

![](./img/8b8f9ade192a.png)

![](./img/ab50faf53318.png)

<strong>视频规格：</strong>

* 视频尺寸：466\*466
* 视频大小：≤10M
* 视频时长：5-10s
* 视频格式：mp4
* 视频编码：H.264
* 视频帧率：25-30帧
* 最多可以添加3个视频
* 建议素材首尾能相接循环，不能有明显跳帧痕迹

<strong>制作步骤：</strong>

1. 添加“背景”&gt;“视频”控件。

   ![](./img/5f3e5f70712e.png "点击放大")
2. 导入视频资源。

   “+”：点击添加视频资源，最多支持添加3个。

   “播放”：预览视频效果。

   滑动白条选择合适的画面，生成视频预览图（必须生成预览图）。

   如果添加了多个视频，须通过“视频切换”，为每一个视频生成一张预览图。

   ![](./img/4f63659a89f6.png "点击放大")

### 单组序列帧

<strong>通过单组序列帧控件，上传单组序列帧资源，制作表盘动态背景。</strong>

<strong>单组序列帧准备：</strong>

![](./img/1d546bf1655f.png "点击放大")

![](./img/2967f3a695f6.png)

<strong>单组序列帧规格：</strong>

* 单组序列帧帧数：“WATCH系列+GT系列”须≤30帧，“WATCH系列”可&gt;30帧
* 单组序列帧单张大小：建议小于40KB
* 单组序列帧格式：PNG
* 单组序列帧每帧的尺寸大小必须保持一致
* 单组序列帧尺寸：“WATCH系列+GT系列”须≤200px\*200px，“WATCH系列”可&gt;200px\*200px
* 素材首尾能相接循环，不能有明显跳帧痕迹

<strong>制作步骤：</strong>

1. 选择“背景”&gt;“单组序列帧”控件，导入单组序列帧资源。

   播放模式：单次播放：在表盘上序列帧只播放一次；循环播放：在表盘上循环播放不暂停。

   位置：单组序列帧在表盘中的位置。

   帧间隔：每帧与每帧的间隔时间。

   延时播放：在设置的时间过后，再播放序列帧。

   ![](./img/31af76370b9d.png "点击放大")
2. 检测单组序列帧播放是否流畅<strong>。</strong>

   <strong>播放</strong>：点击播放按钮，预览单组序列帧动效是否流畅。

   ![](./img/c8381fbe92f2.png "点击放大")

## 时间

时间一般有两种展示方式：模拟时钟和数字时钟。

### 模拟时钟

<strong>通过表盘和指针显示时间，需制作时针、分针、秒针（选做）。</strong>

![](./img/93b7588914d4.png "点击放大")

<strong>使用控件：</strong>指针。

<strong>切图准备：</strong>时针、分针、秒针切图。

![](./img/1215ef1ea2a1.png "点击放大")

![](./img/4ffd964ca084.png)

<strong>切图说明：</strong>

1. 指针切图需指向0点方向（指针旋转的初始方向）。
2. 如果表盘背景制作了刻度，刻度须正确无偏差，可使用Theme Studio进行刻度校验。

<strong>制作步骤：</strong>

1. 添加“时间”&gt;“指针”控件。

   ![](./img/7e4532585a14.png "点击放大")
2. 数值类型选择“时钟比例12”。

   ![](./img/c44fd76cdfee.png "点击放大")
3. 点击“+”导入时针切图，并修改指针中心点为（233,233）；角度为（0,360）；旋转中心点为（17,104）。

   ![](./img/34f01c35fa80.png "点击放大")
4. 再次添加“时间”&gt;“指针”控件，数值类型选择“分钟比例”。点击“+”导入分针切图，并修改指针中心点为（233,233）；角度为（0,360）；旋转中心点为（17,196）。

   ![](./img/5b95f5603ab5.png "点击放大")
5. 再次添加“时间”&gt;“指针”控件，数值类型选择“秒针比例”。点击“+”，导入秒针切图，并修改指针中心点为（233,233）；角度为（0,360）；旋转中心点为（4,165）。

   ![](./img/7cdf11714c3c.png "点击放大")
6. 添加“时间”&gt;“单图”控件，点击“+”导入装饰的圆点切图，并调整好位置。

   ![](./img/e7044737f2ce.png "点击放大")

![](./img/fdb53ef36244.png)

<strong>参数说明：</strong>

1. 指针中心点：指针在表盘画布上的旋转中心点，一般为画布的中心。466\*466分辨率，指针中心点一般为（233,233）；280\*456分辨率，指针中心点一般为（140,228）。
2. 角度：指针的旋转角度，模拟时间中一般为360°。
3. 旋转中心点：指针自身的旋转中心点，根据指针设计而定，可在设计软件中进行确定。

   ![](./img/bfec850de32c.png "点击放大")

### 数字时钟

<strong>通过数字0-9显示时间，需制作时钟、分钟、秒钟（选做）。</strong>

![](./img/5d3b9ba65f4e.png "点击放大")

<strong>使用控件：</strong>选图。

<strong>切图准备：</strong>数字0-9切图。

![](./img/86c7189cd52e.png "点击放大")

<strong>制作步骤：</strong>

1. 添加“时间”&gt;“选图”控件，数值类型选择“时钟高位”。然后点击“+”，批量导入0-2的数字切图，并调整好位置。![](./img/719e4dc2240a.png "点击放大")
2. 再次添加“时间”&gt;“选图”控件，数值类型选择“时钟低位”。然后点击“+”，批量导入0-9的数字切图，并调整好位置。

   ![](./img/fe7a2a4fa7ee.png "点击放大")
3. 再次添加“时间”&gt;“选图”控件，数值类型选择“分钟高位”。然后点击“+”，批量导入0-5的数字切图，并调整好位置。

   ![](./img/9ea5aa91b1f0.png "点击放大")
4. 再次添加“时间”&gt;“选图”控件，数值类型选择“分钟低位”。然后点击“+”，批量导入0-9的数字切图，并调整好位置。

   ![](./img/482ed1b5bd7f.png "点击放大")

![](./img/70fdf742f64b.png)

时钟高位、时钟低位、分钟高位、分钟低位的位置，需符合常规阅读习惯，避免给用户造成困扰。

## 日期

日期一般包含月、日、星期，支持使用文本类控件或图片类控件制作。

### 日期-文本类控件制作

<strong>使用文本类控件，制作月、日、星期。</strong>

![](./img/0cdeb76a25cc.png)

<strong>使用控件：</strong>连接文本、文本。

<strong>切图准备：</strong>无需准备切图，直接使用文本类控件。

<strong>制作步骤：</strong>

1. 添加“日期”&gt;“连接文本”控件，第1个数值类型选择“月数据”，第2个数值类型选择“日期”，连接属性选择“点.”，并调整好位置和大小。

   ![](./img/5cd112f7ca28.png "点击放大")
2. 添加“日期”&gt;“文本”控件，数值类型选择“星期文本”。并调整好位置和大小。

   ![](./img/d7123944a585.png "点击放大")

### 步数

<strong>一般包含步数图标、步数数值、步数目标达成度。</strong>

[](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/publicContent/011/111/111/0000000000011111111.20250710142653.00285666163511451250841665164755:20260601222918:2800:60D7BF0597E05356E5EDF0CF91D067694339B255D1B3D7A17BA276CF2162DA6E.mp4)

<strong>使用控件：</strong>单图、弧形文本、弧形图。

<strong>切图准备：</strong>步数目标达成度弧形图、步数目标达成度底图。

![](./img/ef25a4417318.png "点击放大")

![](./img/005b91d4d3e3.png)

<strong>切图说明：</strong>

使用弧形图控件显示进度/比例时，需准备一张弧形图和一张底图。

* 弧形图：显示当前进度/当前比例，一般颜色较亮，需单独制作一个图层。
* 底图：显示总进度/总比例，一般颜色较深，可以是一个单图层，也可以直接制作在背景图上。

![](./img/a6567be94b21.png "点击放大")

<strong>制作步骤：</strong>

1. 导入步数背景图。

   添加“控件”&gt;“单图”控件，导入步数背景图。

   ![](./img/c91e3d066aae.png "点击放大")
2. 制作步数文本。

   添加“控件”&gt;“弧形文本”控件，第1个数值类型选择“步数”第2个数值类型选择“无数据”，连接属性选择“空格”，并调整好位置、大小、对齐方式、字体、字号、字间距、角度、颜色等参数。

   ![](./img/f292a807c1fe.png "点击放大")
3. 制作步数目标达成度弧形图。

   添加“控件”&gt;“弧形图”控件 ，数值类型选择“步数目标达成度”，导入相应弧形图，并调整好位置、角度、中心点、半径、弧宽等参数。

   ![](./img/1e1792ad48ab.png "点击放大")

![](./img/8c72e24384cb.png)

<strong>参数说明：</strong>

1. 角度：弧形起始点和终止点的角度，以弧形图的实际角度进行设置。一圈为360度，支持顺时针（例：60°至145°）或逆时针（例：300°至215°）计算。
2. 中心点：圆心。弧宽：弧形的宽度。半径：弧形的半径（内圆半径+弧宽一半）。
3. 中心点、弧宽和半径3个参数结合起来生成一个圆环，圆环的范围须覆盖弧形图的范围，再结合设置好的角度参数，即可实现使用弧形图显示当前进度/当前比例的效果。例：当前步数目标达成度为20%，则弧形图显示20%；当前步数目标达成度为90%，则弧形图显示90%。

   ![](./img/2941bd5d0a7f.png "点击放大")

### 心率

## 熄屏表盘制作

![](./img/0fe363c15aac.png "点击放大")

默认为[亮屏表盘](https://developer.huawei.com/consumer/cn/doc/content/watch-face-introduction-0000001566918497#section0147798445)制作模式，点击【半亮模式】则切换到[熄屏表盘](https://developer.huawei.com/consumer/cn/doc/content/watch-face-introduction-0000001566918497#section2023216179448)制作模式。

<strong>熄屏表盘要求：</strong>

1. 图层数量不超过30层。
2. 序列帧、视频、GIF控件不可用。
3. 熄屏表盘不可以单独导出，必须与亮屏表盘一起导出。
4. 屏幕非黑像素点不超过整体屏幕面积的10%或20%，详见[分辨率与版本号](https://developer.huawei.com/consumer/cn/doc/content/resolution-version-0000001252603441#section1756815468317)。
5. 非黑像素点区域尽量分散，不要出现大块集中展示。任意200px\*200px范围，非黑像素点占比不能超过60%。

<strong>熄屏表盘制作方法：</strong>

方法1：像制作亮屏表盘一样，逐一制作每个图层，以上已详细讲解。

<strong>方法2：一键复制亮屏表盘的图层。</strong>

1. 点击切换至“半亮模式”。

   ![](./img/57b9a0e744c3.png "点击放大")
2. 点击复制全亮图层，选中需要的图层。

   ![](./img/088b8900f9a0.png "点击放大")
3. 点击确认，选中的图层就会复制到半亮模式中<strong>。</strong>

   ![](./img/cee8c7ec9cd5.png "点击放大")
4. 点击播放或移动下方进度条查看表盘效果（测试后点击恢复按钮，表盘恢复到默认数值）。

   ![](./img/d781fbf0d345.png "点击放大")
5. 点击“AOD校验”，进行非黑像素点占比校验。

   ![](./img/6c8c1b717dfd.png "点击放大")

   ![](./img/18bd5ba75272.png)

   非黑像素点占比超过后如何修改？详见[aod提示非黑像素点占比过高](https://developer.huawei.com/consumer/cn/doc/content/watchface-faq-0000001174035539#section595359155210)解决方法。
