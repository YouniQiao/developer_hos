# 控件自定义

## 功能介绍

<strong>控件自定义</strong>：支持在表盘上设置自定义区域并制作多个手表预装应用数据。用户通过表盘市场下载并安装了自定义表盘后，可以通过表盘自定义功能，为每个自定义区域更换预装应用数据。

<strong>控件自定义表盘示例</strong>：示例表盘设置了4个圆形的自定义区域，每个自定义区域制作了12种预装应用数据，用户可以为每个自定义区域更换制作好的12种预装应用数据之一。

![](./img/b631e64e026c.png "点击放大")

<strong>控件自定义表盘使用方法</strong> <strong>：</strong>长按表盘 &gt; 点击“设置” &gt;点击“自定义区域”&gt; 更换预装应用数据。

![](./img/d83e8feb8ddc.png "点击放大")

## 制作说明

1. 一个表盘最多同时支持制作16个自定义选项，每个自定义选项下的图层不超过5层。
2. 每一个自定义容器中，至少制作两个自定义选项。

## 制作实操

本文以466\*466控件自定义表盘为例进行制作演示。

1. <strong>按常规方法制作非自定义区域</strong> <strong>。</strong>

   在本示例表盘中，非自定义区域包括时间、日期、步数和卡路里，按照常规方法进行制作即可。

   ![](./img/cda0b805f4a2.png "点击放大")
2. <strong>为背景添加自定义蒙版。</strong>

   选中“背景”图层，导入自定义蒙版。

   ![](./img/47a61525dbcd.png "点击放大")

   ![](./img/3376a7d00aec.png)

   自定义蒙版尺寸需与表盘分辨率一致，制作时将自定义区域挖空，并设置60%透明度。

   ![](./img/578751a689db.png "点击放大")
3. <strong>添加自定义容器1。</strong>

   ① 选择“控件”&gt;“自定义选项”。

   ![](./img/80514bd65784.png "点击放大")

   ② 自动生成自定义容器3和自定义选项1。

   选中自定义容器3，关闭“自适应”，设置好位置和大小。

   ![](./img/c668484d63f3.png "点击放大")
4. <strong>为自定义容器3制作自定义选项。</strong>

   在本示例表盘中，共制作了4个自定义选项：天气、心率、电量和压力。

   通过为“自定义选项”添加图层的方式，制作天气、心率、电量和压力数据。

   ![](./img/bd7dc700c793.png "点击放大")

   <strong>① 自定义选项1-天气</strong>

   制作天气数据，包括气温、温度类型和28种天气类型。

   ![](./img/1bf526c30061.png "点击放大")

   再选中“自定义选项1-天气”，添加天气预览图和边框图。

   ![](./img/2aa82c2cc3bd.png "点击放大")

   ![](./img/2566e08044a5.png)

   1. 自定义区域必须添加一个边框，标明每个可自定义区域的范围。自定义边框的形状和大小建议与当前自定义容器的边框和大小保持一致。
   2. 自定义选项预览图需与实际制作的数据样式保持一致。

   ![](./img/3d163d0b7a8e.png "点击放大")

   <strong>②</strong> <strong>自定义选项2-心率</strong>

   选中“自定义容器3”，添加一个新的自定义选项2。

   ![](./img/2a109d699292.png "点击放大")

   制作心率数据，包括心率数值与心率比例。

   ![](./img/6d0ddcab2846.png "点击放大")

   再选中“自定义选项2-心率”，添加心率预览图和边框图。

   ![](./img/d7af7f25d8ea.png "点击放大")

   <strong>③ 自定义选项3-电量</strong>

   重复以上操作，选中“自定义容器3”，添加一个新的自定义选项3。

   制作电量数据，包括电量数值、电量枚举和电量比例。

   ![](./img/11a80c8d2a8b.png "点击放大")

   再选中“自定义选项3-电量”，添加电量预览图和边框图。

   ![](./img/afac26f5c7dc.png "点击放大")

   <strong>④ 自定义选项4-压力</strong>

   重复以上操作，选中“自定义容器3”，添加一个新的自定义选项4。

   制作压力数据，包括压力背景图和压力数值。

   ![](./img/6d8c984161ff.png "点击放大")

   再选中“自定义选项4-压力”，添加压力预览图和边框图。

   ![](./img/64698499ec44.png "点击放大")

   <strong>⑤ 为</strong> <strong>自定义容器</strong> <strong>设置默认显示的自定义选项。</strong>

   4个自定义选项制作完成后，将显示在当前自定义容器中。

   可为当前容器设置默认显示的自定义选项，这里设置默认显示项为自定义选项1-天气。

   ![](./img/cce5dfdf4e4a.png "点击放大")
5. <strong>添加更多自定义容器。</strong>

   选中“控件”，添加一个新的自定义容器4。

   ![](./img/49748c3a1868.png "点击放大")

   自定义容器4默认添加一个空白的、新的自定义选项，同时将已制作完成的自定义选项同步过来。

   ![](./img/b28992b8b103.png "点击放大")

   在新的自定义容器中，可选择制作新的自定义选项，也可复用已制作完成的自定义选项。

   本示例表盘中，自定义容器4复用了已制作完成的4个自定义选项，设置默认显示项为自定义选项2-心率。

   ![](./img/a4d5c0c3273c.png "点击放大")

   重复以上操作，添加一个新的自定义容器5，设置默认显示项为自定义选项3-电量。

   ![](./img/ebe6dcae810b.png "点击放大")

   重复以上操作，添加一个新的自定义容器6，并设置默认显示项为自定义选项4-压力。

   ![](./img/5e508520fe8f.png "点击放大")
6. <strong>制作熄屏表盘。</strong>

   熄屏表盘不支持控件自定义。

   切换至“熄屏模式”，通过添加图层或复制图层的方式，制作熄屏表盘即可。

   ![](./img/aa3c6ec186b2.png "点击放大")

经过以上步骤，控件自定义表盘制作完成，后续按照常规步骤，[导出表盘](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/watchface-0000001054571181/watch-face-production-pro-0000001633846449/start-production-pro-0000001583807166/watch-face-production-pro-0000001583647406#section1783463344019)即可。

## 相关问题

<strong>1、提示“制作自定义容器，至少制作两个自定义选项”如何解决？</strong>

Theme Studio Pro提示：

![](./img/23bb7e820c4b.png "点击放大")

Theme Studio Pro解决方式：

<strong>方式一</strong>：选中“控件”并右键，“添加”&gt;“自定义选项”，添加新的自定义选项。

![](./img/72f42f3c97e7.png "点击放大")
