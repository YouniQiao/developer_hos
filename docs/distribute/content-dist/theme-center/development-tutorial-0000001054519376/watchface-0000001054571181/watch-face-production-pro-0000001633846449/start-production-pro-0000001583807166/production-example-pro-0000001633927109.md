# 制作实例

## 说明

466\*466设备类型支持两种规范版本：“2.y”和“1.y”。

两种规范版本支持不同的能力集，请根据需要进行选择。

![](./img/39e6516983d0.png "点击放大")

## 背景

### 单图

<strong>通过单图控件，制作表盘静态背景。</strong>

<strong>切图准备</strong> <strong>：</strong>

![](./img/d451e335e476.png "点击放大")

![](./img/10e6aaf9ece6.png)

<strong>切图说明：</strong>

* 背景避免白色像素过多，饱和度/亮度过高，否则会导致手表功耗过大，耗电量较快。Theme Studio Pro支持亮度校验，亮度过高不可导出。
* 设计背景时，静态元素尽量与背景融合以减少图层及图片大小。

<strong>制作步骤：</strong>

1. 添加“背景”&gt;“单图”控件。

   ![](./img/c01e92b4e304.png "点击放大")
2. 导入表盘背景图。

   ![](./img/1525f9249032.png "点击放大")

### 选图

<strong>通过选图控件，实现表盘静态背景切换，最多支持切换5个静态背景。</strong>

<strong>切图准备：</strong>

![](./img/9a126e5d6d99.png "点击放大")

<strong>制作步骤：</strong>

1. 添加“背景”&gt;“选图”控件。

   ![](./img/daa40e269b56.png "点击放大")
2. 数据类型选择“背景切换”，再导入5张表盘背景图。

   ![](./img/e85537b2e12a.png "点击放大")

### 动图

<strong>通过动图控件，上传动图资源，制作表盘动态背景。</strong>

<strong>动图准备：</strong>

![](./img/6a59a4e18a97.png)

![](./img/69a81d56f9db.png)

<strong>动图规格：</strong>

* 动图尺寸：466\*466
* 动图大小：≤10M
* 动图时长：5-10s
* 动图格式：GIF
* 动图帧率：15-30帧/秒
* 最多可以添加1个动图
* 建议素材首尾能相接循环，不能有明显跳帧痕迹

<strong>制作步骤：</strong>

1. 添加“背景”&gt;“动图”控件。

   ![](./img/bf09387ce68f.png "点击放大")
2. 导入动图资源和动图预览图。

   ![](./img/cbb12f6a651e.png "点击放大")

### 视频

<strong>通过视频控件，上传视频资源，制作表盘动态背景。</strong>

<strong>视频准备：</strong>

![](./img/d6f4a87544de.png)

![](./img/d2cc6d9db79b.png)

<strong>视频规格：</strong>

* 视频尺寸：466\*466
* 视频大小：≤10M
* 视频时长：5-10s
* 视频格式：mp4
* 视频编码：H.264
* 视频帧率：25-30帧
* 最多可以添加3个视频
* 建议素材首尾能相接循环，不能有明显跳帧痕迹

<strong>制作步骤：</strong>

1. 添加“背景”&gt;“视频”控件。

   ![](./img/af090a669c0b.png "点击放大")
2. 导入视频资源，最多支持添加3个。

   滑动进度条选择合适的画面，生成视频预览图（必须生成预览图）。

   如果添加了多个视频，须为每一个视频生成一张预览图。

   ![](./img/0bf3b7461583.png "点击放大")

### 单组序列帧

<strong>通过单组序列帧控件，上传单组序列帧资源，制作表盘动态背景。</strong>

<strong>单组序列帧准备：</strong>

![](./img/de6bd6377c56.png "点击放大")

![](./img/bc1784d11261.png)

<strong>单组序列帧规格：</strong>

* 单组序列帧帧数：“WATCH系列+GT系列”须≤30帧，“WATCH系列”可&gt;30帧
* 单组序列帧单张大小：建议小于40KB
* 单组序列帧格式：PNG
* 单组序列帧每帧的尺寸大小必须保持一致
* 单组序列帧尺寸：“WATCH系列+GT系列”须≤200px\*200px，“WATCH系列”可&gt;200px\*200px
* 素材首尾能相接循环，不能有明显跳帧痕迹

<strong>制作步骤：</strong>

1. 选择“背景”&gt;“单组序列帧”控件。

   ![](./img/b8f1544998f3.png "点击放大")
2. 导入单组序列帧资源。

   位置：单组序列帧在表盘中的位置。

   播放模式：单次播放：在表盘上序列帧只播放一次；循环播放：在表盘上循环播放不暂停。

   帧间隔：每帧与每帧的间隔时间。

   播放延时：在设置的时间过后，再播放序列帧。

   ![](./img/54f0d6d97475.png "点击放大")

## 时间

时间一般有两种展示方式：模拟时钟和数字时钟。

### 模拟时钟

<strong>通过表盘和指针显示时间，需制作时针、分针、秒针（选做）。</strong>

![](./img/0df070c66b9b.png "点击放大")

<strong>使用控件：</strong>指针。

<strong>切图准备：</strong>时针、分针、秒针切图。

![](./img/f9087f4052ca.png "点击放大")

![](./img/13fb04f770ec.png)

<strong>切图说明：</strong>

1. 指针切图需指向0点方向（指针旋转的初始方向）。
2. 如果表盘背景制作了刻度，刻度须正确无偏差，可使用Theme Studio Pro进行刻度校验。

<strong>制作步骤：</strong>

1. 添加“时间”&gt;“指针”控件，数据类型选择“时钟比例12”。导入时针切图，并设置好位置、角度和旋转中心点参数。

   ![](./img/6317695ea17a.png "点击放大")

   ![](./img/c6fc6c4c4da0.png "点击放大")
2. 再次添加“时间”&gt;“指针”控件，数据类型选择“分钟比例”。导入分针切图，并设置好位置、角度和旋转中心点参数。

   ![](./img/76b83e956904.png "点击放大")
3. 再次添加“时间”&gt;“指针”控件，数据类型选择“秒针比例”。导入秒针切图，并设置好位置、角度和旋转中心点参数。

   ![](./img/2dccc2310082.png "点击放大")
4. 添加“时间”&gt;“单图”控件，导入装饰的圆点切图，并调整好位置。

   ![](./img/cbd1204655d4.png "点击放大")

![](./img/a7826c923b7a.png)

<strong>参数说明：</strong>

1. 位置：指针在表盘画布上的位置，一般为画布的中心。466\*466分辨率，指针位置一般为（233,233）。
2. 角度：指针的旋转角度，模拟时间中一般为360°。
3. 旋转中心点：指针自身的旋转中心点，根据指针设计而定，可在设计软件中进行确定。

   ![](./img/2beeddc95a1d.png "点击放大")
4. 在“时间”上右键，可直接添加图层。

   ![](./img/874e3ffc49ac.png "点击放大")

### 数字时钟

<strong>通过数字0-9显示时间，需制作时钟、分钟、秒钟（选做）。</strong>

![](./img/67c772b90626.png "点击放大")

<strong>使用控件：</strong>选图。

<strong>切图准备：</strong>数字0-9切图。

![](./img/beb06db24434.png "点击放大")

<strong>制作步骤：</strong>

1. 添加“时间”&gt;“选图”控件，数据类型选择“时钟高位”。然后批量导入0-2的数字切图，并调整好位置。

   ![](./img/621ec7e4ea25.png "点击放大")

   ![](./img/8eb6c97a2d50.png "点击放大")
2. 再次添加“时间”&gt;“选图”控件，数据类型选择“时钟低位”。然后批量导入0-9的数字切图，并调整好位置。

   ![](./img/6a4ec23d8ff8.png "点击放大")
3. 再次添加“时间”&gt;“选图”控件，数据类型选择“分钟高位”。然后批量导入0-5的数字切图，并调整好位置。

   ![](./img/35ea93b19c64.png "点击放大")
4. 再次添加“时间”&gt;“选图”控件，数据类型选择“分钟低位”。然后批量导入0-9的数字切图，并调整好位置。

   ![](./img/7e89a4b8644f.png "点击放大")

![](./img/f90e785580b3.png)

时钟高位、时钟低位、分钟高位、分钟低位的位置，需符合常规阅读习惯，避免给用户造成困扰。

## 日期

日期一般包含月、日、星期，支持使用文本类控件或图片类控件制作。

### 日期-文本类控件制作

<strong>使用文本类控件，制作月、日、星期。</strong>

![](./img/d5242a74b81d.png "点击放大")

<strong>使用控件：</strong>连接文本、文本。

<strong>切图准备：</strong>无需准备切图，直接使用文本类控件。

<strong>制作步骤：</strong>

1. 添加“日期”&gt;“连接文本”控件，数据类型-1选择“月数据”，数据类型-2选择“日期”，连接属性选择“斜线/”，并调整好位置和大小。

   ![](./img/3ca86677693a.png "点击放大")

   ![](./img/d1f313066b23.png "点击放大")
2. 添加“日期”&gt;“文本”控件，数据类型选择“星期文本”。并调整好位置和大小。

   ![](./img/52c46a3b4a9f.png "点击放大")

### 日期-图片类控件制作

<strong>使用图片类控件，制作月、日、星期。</strong>

![](./img/c61f6ebd70cf.png "点击放大")

<strong>使用控件：</strong>选图、组合图。

<strong>切图准备：</strong>星期、月份、日期切图。

![](./img/c54b5a0eee5d.png "点击放大")

![](./img/7178b8f63258.png)

<strong>切图说明：</strong>

* 星期切图可设计为中文全称、英文全称、英文简写等。
* 月份切图可设计为中文全称、1-12数字、英文全称、英文简写等。
* 组合图无效资源：设计一张“-”图片，未有数据同步时，显示无效资源“-”，与组合图0-9数字切图尺寸保持一致。

<strong>制作步骤：</strong>

1. 添加“日期”&gt;“选图”控件。数据类型选择“月”，批量导入月份切图，并调整好位置。

   ![](./img/9cbeec838db1.png "点击放大")

   ![](./img/f255f1c47d35.png "点击放大")
2. 添加“日期”&gt;“单图”控件，导入连接符号.切图，并调整好位置。

   ![](./img/7d07f6edb8ac.png "点击放大")
3. 添加“日期”&gt;“组合图”控件，数据类型选择“日期”，批量导入0-9数字切图，并调整好位置。

   ![](./img/3142d32b5322.png "点击放大")
4. 数据类型选择“星期数据”。批量导入星期资源切图，并调整好位置。

   ![](./img/c63eb1c77e34.png "点击放大")

### 步数

<strong>一般包含步数图标、步数数值、步数目标达成度。</strong>

<strong>使用控件：</strong>单图、弧形文本、弧形图。

<strong>切图准备：</strong>步数目标达成度弧形图、步数目标达成度底图。

![](./img/554e7fb00920.png "点击放大")

![](./img/2e2f5e3218f5.png)

<strong>切图说明：</strong>

使用弧形图控件显示进度/比例时，需准备一张弧形图和一张底图。

* 弧形图：显示当前进度/当前比例，一般颜色较亮，需单独制作一个图层。
* 底图：显示总进度/总比例，一般颜色较深，可以是一个单图层，也可以直接制作在背景图上。

![](./img/a9d1be1dfdc7.png "点击放大")

<strong>制作步骤：</strong>

1. 导入步数背景图。

   添加“控件”&gt;“单图”控件，导入步数背景图。

   ![](./img/cf3bac022091.png "点击放大")
2. 制作步数文本。

   添加“控件”&gt;“弧形文本”控件，数据类型-1选择“步数”，数据类型-2选择“无数据”，连接属性选择“空格”，并调整好位置、大小、对齐方式、字体、字号、字间距、角度、颜色等参数。

   ![](./img/f9b29e3c47b3.png "点击放大")
3. 制作步数目标达成度弧形图。

   添加“控件”&gt;“弧形图”控件 ，数据类型选择“步数目标达成度”，导入相应弧形图，并调整好位置、角度、中心点、半径、弧宽等参数。

   ![](./img/777fc4fd696a.png "点击放大")

![](./img/8fb1ee9b65a6.png)

<strong>参数说明：</strong>

1. 角度：弧形起始点和终止点的角度，以弧形图的实际角度进行设置。一圈为360度，支持顺时针（例：60°至145°）或逆时针（例：300°至215°）计算。
2. 中心点：圆心。弧宽：弧形的宽度。半径：弧形的半径（内圆半径+弧宽一半）。
3. 中心点、弧宽和半径3个参数结合起来生成一个圆环，圆环的范围须覆盖弧形图的范围，再结合设置好的角度参数，即可实现使用弧形图显示当前进度/当前比例的效果。例：当前步数目标达成度为20%，则弧形图显示20%；当前步数目标达成度为90%，则弧形图显示90%。

   ![](./img/18b61a7d6898.png "点击放大")

## 熄屏模式制作

默认为全亮模式，点击切换至“熄屏模式”。

![](./img/f2406b50f515.png)

<strong>熄屏模式要求：</strong>

1. 图层数量不超过30层。
2. 序列帧、视频、GIF控件不可用。
3. 熄屏表盘不可以单独导出，必须与亮屏表盘一起导出。
4. 屏幕非黑像素点不超过整体屏幕面积的20%，详见[分辨率与版本号](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/watchface-0000001054571181/basic-concepts-0000001207883464/resolution-version-0000001252603441#section1756815468317)。
5. 非黑像素点区域尽量分散，不要出现大块集中展示。任意200px\*200px范围，非黑像素点占比不能超过60%。

方法1：像制作全亮模式一样，逐一制作每个图层，以上已详细讲解。

   ![](./img/ccf6bee455ab.png "点击放大")
