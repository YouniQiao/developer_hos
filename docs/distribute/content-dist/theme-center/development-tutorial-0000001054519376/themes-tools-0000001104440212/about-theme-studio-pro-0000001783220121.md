# Theme Studio Pro版本更新说明

## Theme Studio Pro 1.0.18（2025-10-15）

<strong>主题工具</strong>

<strong>新增特性</strong>

* 工具支持直板机引擎能力实时预览。
* 工具直板机新增设计者模式。
* 工具支持通用直板机资源转Pura X资源。
* 主题素材管理支持拖拽上传。

<strong>优化特性</strong>

* 解决因变量动画重复次数格式类型问题，导致项目异常。

<strong>表盘工具</strong>

<strong>新增特性</strong>

* 表盘增加能耗智能检测分析与一键优化功能。

## Theme Studio Pro 1.0.17（2025-08-18）

<strong>主题工具</strong>

<strong>新增特性</strong>

* 支持在NEXT主题编辑页导入其他设备类型的主题资源包。
* 支持换肤编辑区通过右键菜单查看所有可换肤项。

<strong>优化特性</strong>

* 优化桌面时钟样式、应用名称、分页导航栏反色效果。
* 优化导出预览图大小，与联盟上传规范保持一致。
* 新增对百变卡片的图片＆视频最大分辨率校验。
* 新增30个必做三方应用图标。
* 新增静态图标&实时图标&三方图标的前景图、背景图不允许全透明校验。
* 新增静态图标&实时图标&三方图标的前景图、背景图文件大小不超过500KB校验。

<strong>表盘工具</strong>

<strong>新增特性</strong>

* 466\*466分辨率表盘开放情绪数据。

## Theme Studio Pro 1.0.16（2025-06-30）

<strong>新增特性</strong>

* 支持全景换肤主题制作。
* 开放引擎新能力：跟手粒子、延时解锁、空间翻转、蓝牙耳机数据开放。
* 支持百变卡片制作。
* 计算NEXT主题、百变卡片资源包的耗电等级。

<strong>优化特性</strong>

* 深/浅模式复制功能优化，支持按照模块维度复制。
* 控制中心新增“耳机降噪图标颜色”换肤项。
* 视频桌面、耳机弹窗支持选择视频指定帧作为预览效果。
* 优化Pura X外屏锁屏上时钟、快捷工具的颜色显示逻辑。

## Theme Studio Pro 1.0.8（2024-5-11）

<strong>表盘主题新增特性</strong>

* 新增408\*480分辨率表盘可视化制作。

## Theme Studio Pro 1.0.7（2024-4-7）

<strong>表盘主题新增特性</strong>

* 优化控件中位置属性的计算方式。

![](./img/a8ae2c87f31a.png)

将容器原点为参考计算，优化为画布原点为参考计算。

* 修复刻度线覆盖顶部下拉菜单的问题。
* 修复表盘图层大小计算错误的问题。

## Theme Studio Pro 1.0.6（2023-11-15）

<strong>表盘主题新增特性</strong>

* 占位图资源显示逻辑优化：当编辑预览区处于播放状态时，或当属性区资源图片上传后，将不展示占位图。
* 支持表盘资源存储状态可视化显示。
* 组合图上传优化。
  1. 系统自动填充组合图中的透明图片，无需单独上传。
  2. 无数据时默认显示图（“--”）和特殊资源负数图（“-”），需与数字图片一起批量上传。

## Theme Studio Pro 1.0.5（2023-9-15）

<strong>表盘主题新增特性</strong>

* 支持466\*466分辨率表盘制作。
* 支持高级功能制作：景深、颜色自定义、样式自定义。
* 支持元素栏，图层栏、属性栏、高级功能栏的拖拽和吸附。
* 支持背景单图上传裁剪功能，可上传超出规范的图片，通过裁剪功能生成符合规范的图片。
* 支持[快捷键](/docs/distribute/content-dist/theme-center/development-tutorial-0000001054519376/watchface-0000001054571181/watch-face-production-pro-0000001633846449/accessibility-pro-0000001583647410#section16163482563)使用。
