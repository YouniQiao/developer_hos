# 制作工具下载

## Theme Studio Pro

访问网址： &lt;https://themestudio.cloud.huawei.com.cn/&gt;

登录账号：拥有华为主题认证设计师-表盘权限的华为账号。

网络环境：连接互联网Internet环境。

![](./img/f8b39f5b86ab.png)

1. 支持466\*466和408\*480分辨率表盘制作。
2. 请使用Chrome浏览器95以上版本访问。
3. 如何申请<strong>“主题认证设计师-表盘权限”</strong>？详见[入驻指导](https://developer.huawei.com/consumer/cn/doc/content/settlement-guidance-0000001056348857)。

## Theme Studio

华为主题&手表表盘制作工具：Theme Studio。

Windows及macOS：点击下载 [Theme Studio工具包](https://developer.huawei.com/consumer/cn/doc/Tools-Library/theme_download-0000001050424897)。

![](./img/b08ade5c6c39.png)

1. Theme Studio支持制作EMUI 10.0及以上版本的手机主题，包括小主题、大主题、锁屏主题、图标主题、AOD主题。如需制作EMUI 5.0~EMUI 9.1版本的手机主题，请下载hwtTool。
2. Theme Studio同时还支持以下分辨率表盘的制作：466\*466、390\*390、454\*454、280\*456、336\*480和194\*368。
3. 安装macOS版本时，请在“系统偏好设置 -&gt; 安全性与隐私 -&gt; 通用”中选择“App store 和被认可的开发者”选项。
