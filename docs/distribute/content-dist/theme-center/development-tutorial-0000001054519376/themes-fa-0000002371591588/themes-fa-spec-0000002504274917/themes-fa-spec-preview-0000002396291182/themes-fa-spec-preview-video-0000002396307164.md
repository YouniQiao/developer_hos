# 预览视频

视频尺寸为1440×2160 px。

视频格式为MP4，编解码制式要求为H.264/H.265，视频大小建议在10MB以内，时长建议在15秒以内。

视频背景要求采用统一的背景样式，背景渐变色值：#FFFFFF——#E0E2E6，背景渐变角度：上下垂直180度。

为保证预览效果，请提供能循环播放的视频，确保首尾衔接部分流畅，不会出现画面跳动或者闪烁。
