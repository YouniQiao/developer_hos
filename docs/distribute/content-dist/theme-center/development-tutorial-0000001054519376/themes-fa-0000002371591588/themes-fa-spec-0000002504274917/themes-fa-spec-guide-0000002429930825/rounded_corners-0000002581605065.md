# 卡片圆角

1. 卡片内容尽可能保证在安全范围内，在圆角裁剪时避免对内容造成影响。

2. 创作者在进行设计时如果需要保证最终效果，可以通过Theme Studio Pro工具进行圆角预览效果，交付时确保直角即可。

3. 在交付卡片时请保证背景是直角矩形，系统会提供对应的圆角裁剪，保证在不同设备上卡片圆角效果一致。

4. 因卡片圆角会根据场景动态变化，不建议使用卡片内嵌套圆角矩形的设计方式，确保卡片整体精致度。
