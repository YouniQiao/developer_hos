# 卡片标签

通过主题联盟上传百变卡片作品时，创作者需根据卡片的实际元素给作品打“卡片标签”，一个百变卡片作品支持打多个标签，比如卡片包含时钟和天气信息时，需要选择“时钟”和“天气”标签。

<strong>表1</strong> 卡片标签

| 标签名称 | 标签描述 | 必备要素 |
| --- | --- | --- |
| 时钟 | 包含当前时间功能的卡片，例如模拟时钟，数字时钟等。 | * 当前时间 * 时钟APP跳转链接 |
| 天气 | 包含天气数据功能的卡片，例如天气数据展示，卡片样式随天气变化等效果。 | * 当前温度 * 气象描述（多云，下雨等） * 华为天气APP跳转链接 |
| 日历 | 可以让用户看到当前日期和星期（可以在日历卡片上添加时间及时钟APP跳转链接）。 | * 至少包含当前一周内的日期情况 * 华为日历APP跳转链接 |
| 音乐播放器 | 使用音乐播放/暂停以及歌曲名等功能的卡片。 | * 艺术家名称 * 歌曲名 * 收藏功能 * 暂停/播放/上下首等播控功能 |
| 趣味 | 具备趣味性，如电冰箱、电风扇、木鱼、点赞、每日心情、今天吃什么、路牌、签名等 | 有趣 |
| 工具 | 各类实用工具，如课程表，电量，系统数据，备忘录，目标打开，快捷跳转（应用程序：支付宝、微信；便捷操作：手电筒、相机） | 好用 |
| 倒计时 | 包含倒计时功能的卡片，可以让用户设置关键日期倒计时 | 包含倒计时功能 |
| 步数 | 包含用户当前步数数据的卡片，例如做步数换景或者简单的步数显示等。 | * 今日步数 * 华为运动健康APP跳转链接 |
