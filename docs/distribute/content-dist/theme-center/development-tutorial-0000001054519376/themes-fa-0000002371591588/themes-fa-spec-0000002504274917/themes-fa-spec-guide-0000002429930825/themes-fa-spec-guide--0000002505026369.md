# 功能要求

卡片应用成功并添桌后可以正常使用，功能要求如下：

1. 卡片的实际显示效果需与预览图一致，若卡片中包含隐式功能，则必须在预览图中给出具体的玩法说明；若卡片包含跳转链接，且跳转对象为非预装应用，需在预览图和资源简介中明示用户，需要安装相应应用后才可正常跳转。
2. 原始卡片或卡片被缩小或放大后，卡片背景图片不得出现拉伸变形或锯齿。任何设备分辨率下，不能出现显示不全或断层情况。
3. 不得存在影响用户使用的重叠、互相遮挡、布局错乱。
4. 卡片图片、字体等内容显示清晰，若卡片引用全局变量#darkMode提供深色模式设计时，需要确保卡片上内容显示清晰。
5. 卡片上信息需正确显示，不得出现和系统的时间、日期、步数等明显不一致的情况。
6. 不得含有隐藏链接，所有跳转链接必须符合用户预期，譬如日历卡片，可以跳转到日历APP，不能跳转至相机APP。
7. 操作卡片或播放动效体验流畅，不得出现卡顿现象。
8. 卡片上提供的功能，均可正常使用，譬如点击播放按钮，可以正常播放音视频；所提供的功能，需提供复位，确保相同的操作下，用户体验一致，譬如电风扇卡片，需提供开启和关闭功能，抽签卡片，抽完签后所有签需能够回到签筒。
