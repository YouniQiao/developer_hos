# 分类定义

百变卡片分为单卡-中卡、单卡-大卡和卡片套装，一个百变卡片作品归属其中一种分类。

<strong>表1</strong> 百变卡片分类定义

| 分类 | 定义 |
| --- | --- |
| 单卡-中卡 | ①仅含1张卡片  ②卡片尺寸2×4  ③设计风格和谐统一，设计精美，功能不限 |
| 单卡-大卡 | ①仅含1张卡片  ②卡片尺寸4×4  ③设计风格和谐统一，设计精美，功能不限 |
| 卡片套装 | ①卡片数量不少于4张，最多20张  ②卡片尺寸不限  ③设计风格和谐统一，设计精美，功能不限 |

![](./img/45fd7eb56e65.png)

一张卡片内可以包含多种功能，比如一张卡里同时展示时钟和天气信息。

不允许将多个虚拟卡片组合在一张卡片上，以免误导最终用户。
