# 卡片规格

百变卡片支持以下四种卡片规格：

<strong>表1</strong> 卡片规格

| 卡片规格 | **对应宫格数（高\*宽） | 图片&视频素材允许最大分辨率** |
| --- | --- | --- |
| 微卡片 | 1×2 | 265×686 px |
| 小卡片 | 2×2 | 692×692 px |
| 中卡片 | 2×4 | 320×686 px |
| 大卡片 | 4×4 | 720×692 px |
