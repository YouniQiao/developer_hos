# 使用限制

资源限额

| 资源 | 限制 |
| --- | --- |
| 直接上传单个视频文件。 | 视频128G/个。 |
| 提供外部下载链接，让平台自行下载视频文件。 | 外部下载链接对应的视频文件5G/个。 |
| 单张图片大小（视频封面图片）。 | 10M/张。 |
