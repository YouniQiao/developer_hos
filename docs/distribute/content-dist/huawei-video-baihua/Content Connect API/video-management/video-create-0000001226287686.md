# 视频创建

该接口用于创建百花号视频，创建成功的视频会在华为视频APP上展示，创建提交后可通过/video/detail接口来获取详情和状态，创建过程是个异步过程，过程中会有很多检查项，在成功创建之前/video/list接口获取的列表中会不包含。

Scope：https://www.huawei.com/contentconnect/video

<strong>请求地址</strong>

POST /mosopenapi/v2/foruser/video/create

<strong>请求参数说明</strong>

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 参数名称 | 参数类型 | 参数描述 | 参数示例 | 是否必须 |
| upResourceId | string | UP主的id标识，upResourceId和tpUserCode不能同时为空。 | 492456389055106944  为/list/users接口获取的upInfos中的upResourceId。 | false |
| tpUserCode | string | UP主的第三方标识，upResourceId和tpUserCode不能同时为空。 | hwvrmgjxmt | false |
| videoId | string | 视频唯一id。 | 492456389055106956 | true |
| tpVideoId | string | 第三方视频标识 | video123456 | false |
| imageId | string | 图片唯一id，如果不填后台会自动生成海报封面。 | 492456389055106956 | false |
| language | string | 视频语言，枚举值范围参照：附录A。 | zh\_CN | true |
| contentName | string | 视频标题。 | 后来的我们：通往成功的路上，一个人太孤单，两个人刚刚好 | true |
| description | string | 视频描述信息。 | 后来的我们：通往成功的路上，一个人太孤单，两个人刚刚好 | false |
| mediaType | string | 一级分类，为分类的id，取值需联系运营获取。 | 1 | true |
| theme | string | 二级分类，为分类的id，取值需联系运营获取。 | 142 | true |
| oparea | string | 视频分发国家，可选范围为版权内国家码子集，多个以英文逗号（,）分隔。 | CO,SG,CN | false |
| original | integer | 是否声明成原创， 0：非原创，1：原创。 | 0 | true |
| tags | string | 标签，多个以英文分号（;）分隔。 | 悬疑;破案 | false |
| videoType | integer | 视频类型，1：横版视频，2：竖版视频。 | 1 | true |

<strong>请求样例</strong>

```
{
  "upResourceId": "674482848291436800",
  "videoId": "854955834189519744",
  "imageId": "854954091556871040",
  "language": "zh_CN",
  "contentName": "OpenAPI-13813919198-2022-0325-YC-000000000000000000000001",
  "description": "OpenAPI-13813919198-2022-0325-YC-000000000000000000000001",
  "mediaType": "20",
  "theme": "650",
  "oparea": "CN",
  "original": 1,
  "tags": "111;aa;AA",
  "videoType": 1
}
```

|  |  |  |  |  |
