地址：&lt;https://developer.huawei.com/consumer/cn/&gt;

![](./img/cfce8545c2ca.png)

第二步：如未进行实名认证需进行企业级实名认证。

![](./img/c4d42686c003.png)

第三步：进入管理中心。

成功通过申请的应用，client key，client secret，权限均可前往【管理中心】查看。

![](./img/87f6bc392441.png)

<strong>client secret</strong> <strong>切勿泄露</strong>。

![](./img/c48eddcbc2d5.png)

![](./img/12fd1539668f.png)

如未创建应用可点击创建第三方应用。

![](./img/843e3a8a0083.png)

第四步：搜索华为视频百花号，申请开放平台调用权限。

![](./img/c1a812acb547.png)

![](./img/9e4000639a98.png "点击放大")
