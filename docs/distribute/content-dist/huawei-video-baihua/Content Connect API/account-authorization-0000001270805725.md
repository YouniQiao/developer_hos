# 帐号授权

华为帐号登陆是基于OAuth2.0协议标准构建的授权登录系统，OAuth2.0授权登录让用户安全登陆接入的第三方应用或网站，在用户授权登录已接入华为开发者联盟OAuth2.0的第三方应用或网站后，第三方可以获取到用户的接口调用凭证（access\_token），通过该凭证可以实现获取百花号用户基本开放信息和其他已授权的相关信息等，参照[获取AccessToken](https://developer.huawei.com/consumer/cn/doc/development/HMSCore-Guides/open-platform-oauth-0000001053629189)说明中的授权码模式。
