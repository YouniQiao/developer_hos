# 术语

| 术语 | 解释 |
| --- | --- |
| 华为视频百花号 | 指由华为基于开放平台运营的内容分发渠道，（“开放平台”定义在《[华为开发者服务协议](https://terms-drcn.platform.dbankcloud.cn/agreementservice/developer/getAgreementTemplate?agrType=1000&country=cn&language=zh_cn&version=2022030201)》中）华为将发布者发布至华为视频百花号平台的发布者内容进行分发、推广等。华为视频百花号平台由发布者自主注册并独立承担全部责任，发布者须同时遵守《[百花号内容合作协议](https://terms-drcn.platform.dbankcloud.cn/agreementservice/developer/getAgreementTemplate?agrType=1056&country=cn&language=zh_cn&version=2020053014)》（如签署）。 |
| 发布者 | 指有效申请并经华为审核同意后，在华为视频百花号平台上发布【包含但不限于视频、文字、图片、图文、音频内容】的“开发者”（“开发者”定义在《[华为开发者服务协议](https://terms-drcn.platform.dbankcloud.cn/agreementservice/developer/getAgreementTemplate?agrType=1000&country=cn&language=zh_cn&version=2022030201)》中）。 |
| 发布者内容 | 指有效申请并经华为审核同意后，发布者在华为视频百花号平台上发布的【包含但不限于视频、文字、图片、图文、音频内容】产品（“产品”定义在《[华为开发者服务协议](https://terms-drcn.platform.dbankcloud.cn/agreementservice/developer/getAgreementTemplate?agrType=1000&country=cn&language=zh_cn&version=2022030201)》中）。 |
