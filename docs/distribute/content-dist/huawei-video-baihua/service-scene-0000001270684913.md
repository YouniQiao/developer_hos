# 触达场景

工具类应用

开发者可通过华为视频百花号开放服务将内容创作者生产的视频内容上传至华为视频百花号，并由华为视频百花号分发至华为品牌的各个APP（例如华为视频APP），让终端用户进行消费，并且和平台分享收益。
