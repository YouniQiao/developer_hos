# 服务条款

使用华为视频百花号开放服务，向华为视频百花号上传视频内容，需要遵守如下条款：

1、《[华为开发者服务协议](https://terms-drcn.platform.dbankcloud.cn/agreementservice/developer/getAgreementTemplate?agrType=1000&country=cn&language=zh_cn&version=2022030201)》

2、《[百花号平台服务协议](https://terms-drcn.platform.dbankcloud.cn/agreementservice/developer/getAgreementTemplate?agrType=1046&country=cn&language=zh_cn&version=2020053013)》

使用华为视频百花号开放服务，向华为视频百花号上传视频内容，并且要和平台分享收益，需要遵守如下条款：

1、《[华为合作伙伴商户服务协议](https://terms-drcn.platform.dbankcloud.cn/agreementservice/developer/getAgreementTemplate?agrType=1003&country=cn&language=zh_cn&version=2022030205)》

2、《[百花号内容合作协议](https://terms-drcn.platform.dbankcloud.cn/agreementservice/developer/getAgreementTemplate?agrType=1056&country=cn&language=zh_cn&version=2020053014)》
