# 上传任务管理

为提升开发者上传内容的效率，方便用户快速创建多个章节或多个课程而不被上传文件阻塞。当您添加或编辑课程时，添加课程文件或章节文件后文件随即开始上传。

具体上传情况可在“上传任务管理”中查看和管理，您可以查看上传的文件信息、上传状态和上传百分比，另外您也可以删除上传任务或者重新上传失败的任务。
