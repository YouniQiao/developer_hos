# 业务介绍

## 1.简介

音频能力平台是华为旗下的一站式内容分发平台，致力于帮助媒体、自媒体、企业、机构获得更多曝光与关注，持续扩大品牌影响力和商业变现能力，扶植优质内容生产者做大做强，建立合理、健康、安全的内容生态体系。

音频能力平台面向音频行业提供开放能力，音频内容提供商可以通过音频能力平台开发者门户为华为音乐提供各种音频内容。

华为音乐既为开发者提供签约注册、合同签署、（音乐/有声）内容提供、数据查看等完整的端到端流程；也为华为内部服务提供音频内容托管服务。

## 2.平台业务

有声内容导入

音频能力平台提供有声内容导入能力。用户或企业可以根据自身的需求将有声书批量导入到华为音乐，进行内容托管服务。

有声内容管理

音频能力平台提供有声内容管理能力。用户或企业可以对已经导入的有声书进行管理，具体功能有：专辑的查询、新增、内容的修改、上下线、价格变更等；专辑下章节的查询、新增、内容的修改、上下线、连载等。
