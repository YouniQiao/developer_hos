# ohpm-repo --version

查询ohpm-repo版本。

## 命令格式

```
ohpm-repo --version
alias: -v
```

## 功能描述

打印ohpm-repo的版本号。

## 示例

执行以下命令，查看版本信息：

```
ohpm-repo -v
```
