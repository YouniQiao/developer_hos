# ohpm clean错误码

## 00626001 清理模块失败

<strong>错误信息</strong>

Clean Module Failed.

<strong>错误描述</strong>

清理模块失败。

<strong>可能原因</strong>

文件可能被占用导致清理失败。

<strong>处理步骤</strong>

确保文件未被占用。
