# 下载与安装

## 环境准备

DevEco CLI支持在Windows和macOS上运行。

* 下载和安装[DevEco Studio](https://developer.huawei.com/consumer/cn/deveco-studio/) 6.1.0及以上版本的DevEco Studio。
* 安装[Node.js](https://nodejs.org/)，推荐使用22及以上版本。

## 安装和更新

* 下载和安装DevEco CLI（稳定版）。

  ```
  npm install -g @deveco/deveco-cli@stable
  ```
* 下载和安装DevEco CLI（尝鲜版）。

  ```
  npm install -g @deveco/deveco-cli
  ```
* 更新DevEco CLI。

  ```
  devecocli update
  ```

![](./img/note_3.0-zh-cn.png)

下载和安装命令中的标签@stable是可选项，加上标签@stable表示下载安装稳定版本，不加标签@stable表示下载安装最新版本，可以体验最新功能。
