# 工程问答

从DevEco Studio 6.1.0 Beta2版本开始，CodeGenie新增工程问答的功能。工程问答通过分析用户代码库，快速定位文件、类、函数、变量、常量、UI元素等的具体位置，对开发者提出的本地工程代码检索类问题作出精准回答，并完成具体的代码解读。

选择<strong>HarmonyOS Ask</strong>智能体，在对话区域输入“<strong>/</strong>”后，在弹出的菜单中选择“<strong>Project Ask</strong>”，在对话框中输入本地工程代码检索类问题，点击![](./img/zh-cn_image_0000002602066497.png)发送后等待回复：
