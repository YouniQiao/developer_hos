# 华为支付

华为支付是一种方便、安全和快捷的支付方式。

### 场景介绍

华为支付图标通常在收银台等界面展示，如下图所示：

### 图标大小

在核心使用场景下的实际尺寸通常有 48x48 vp、24x24 vp，请确保该尺寸下图标清晰易识别：

![](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250619224915.65381434784993439957564822100027:50001231000000:2800:CD77A823A75D42170E5C2A488B05D3834FF584B5FFC0E0094ED4E2EF0B439EA3.png "点击放大")

### 背景控制

在使用华为支付图标时，请注意图标颜色和背景颜色的反差，避免图标要素不可读或者不易读。

辅助色背景控制 (在有色背景上使用，尽量避免使用蓝色系，以免造成图标边缘不清晰)：

![](https://communityfile-drcn.op.dbankcloud.cn/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20250619224915.94708541629532100067788945846678:50001231000000:2800:D8B310A3F6476B2AC788E9910473240D38E4E47F0BCAE8E64D15A4BECA62D319.png "点击放大")
