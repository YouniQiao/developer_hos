# 华为账号开放登录

华为账号提供登录体验设计规范，帮助用户使用已有的华为账号登录HarmonyOS生态内全部应用。遵循规范设计可享有以下好处：

**(一) 简单易用：**

登录界面的重点信息和操作突出，减少用户的理解成本。

**(二) 体验一致：**

多应用，多设备登录体验一致，为用户提供一致且熟悉的登录体验。

**(三) 开发者友好：**

**1. 开发者拥有可自定义的范围：**使用华为账号提供的登录按钮时，开发者可以调整按钮的大小、圆角、位置等。

**2. 调用合适的登录组件：**华为账号提供两种登录类型，开发者可以根据需要调用合适的登录组件。

### 概述

## 使用原则

**使用华为账号开放登录请遵循以下原则：**

**(一) 建议仅在有价值时要求用户登录：**

允许用户有足够的时间和机会体验应用的用途后，再决定是否注册/登录使用。禁止在用户进入 HarmonyOS 应用还未开始了解基础功能时，就强制要求注册/登录。若特定行业有国家相关要求，则要求注册/登录后才可以使用。

**(二) 用户应当有拒绝登录的权利：**

当 HarmonyOS 应用调用账号登录功能时，应提供清晰的取消或拒绝选项按钮，不得强制要求用户登录账号才能继续使用。如用户拒绝登录，可在用户表现出向 HarmonyOS 应用授权的意向后再次提示登录。

**(****三)** **使用华为账号登录时，需作为首个登录页。**

## 使用规范

调用【华为账号一键登录】按钮时，页面必须包含以下【必选】项。推荐使用图中登录页布局。

**应用图标/品牌标志：**

① 应用图标：可以使用应用图标或者品牌标志凸显品牌特性。

**内容区：**

⑥【必选】隐私与用户协议：必须展示[《华为账号用户认证协议》](https://privacy.consumer.huawei.com/legal/id/authentication-terms.htm?code=CN&language=zh-CN)，可选择展示其他协议，需支持用户查看详情。

**I.** **多语言规则**

双向&镜像原则

阿拉伯语、波斯语、乌尔都语、希伯来语等语言的书写方向为从右向左书写，和当今世界的主流语言（如英语）等从左向右书写的方向相反。受文字书写方向的影响，阿拉伯等语言的用户对左右逻辑的认知和英文等LTR（Left to Right）完全相反，比如习惯将右侧作为开始，左侧作为结束。为了支持RTL语言的这种语言和用户习惯特点，应用在本地化时，除了常规的翻译、区域特性适配等工作外，还需要在文本、界面布局、手势操作和动画、图标等交互元素中满足RTL的特殊要求。

**G.** **多语言规则**

双向&镜像原则

阿拉伯语、波斯语、乌尔都语、希伯来语等语言的书写方向为从右向左书写，和当今世界的主流语言（如英语）等从左向右书写的方向相反。受文字书写方向的影响，阿拉伯等语言的用户对左右逻辑的认知和英文等LTR（Left to Right）完全相反，比如习惯将右侧作为开始，左侧作为结束。为了支持RTL语言的这种语言和用户习惯特点，应用在本地化时，除了常规的翻译、区域特性适配等工作外，还需要在文本、界面布局、手势操作和动画、图标等交互元素中满足RTL的特殊要求。
