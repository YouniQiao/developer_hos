# 下拉按钮

下拉按钮可让用户在多个选项之间选择。开发相关描述请参考 [Select](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-select) 文档。

### 如何使用

需过滤当前界面内容、快速切换类型 (如存储位置) ，或选项内容 (如设置密保问题) 时，使用下拉按钮。
