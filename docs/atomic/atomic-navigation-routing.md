Navigation是路由容器组件，一般作为首页的根容器，适用于模块内和跨模块的路由切换。Navigation子页是NavDestination包含的子组件，使用NavPathStack实现页面路由。

* **[跨模块页面路由](https://developer.huawei.com/consumer/cn/doc/atomic-guides/atomic-inter-module-page-routing)**
