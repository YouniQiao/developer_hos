## 注册账号与实名认证

若在[华为云核准（备案）系统](https://beian.huaweicloud.com/?utm_source=HUAWEI%2BDeveloper&utm_adplace=AdPlace099034)中核准（备案）过网站、APP、元服务等，您无需注册新的华为云账号，请继续使用以前的华为云账号核准（备案）元服务。

若首次在[华为云核准（备案）系统](https://beian.huaweicloud.com/?utm_source=HUAWEI%2BDeveloper&utm_adplace=AdPlace099034)中进行核准（备案），或同时核准（备案）多个主体信息，或转移主体及主体下的互联网信息，您需要注册新的华为云账号并完成实名认证。步骤如下：

1. 注册账号。个人或企业注册华为云账号的操作指导请参见[注册华为云账号并开通华为云服务](https://support.huaweicloud.com/usermanual-account/account_id_001.html)。
2. 实名认证。个人或企业实名认证华为云账号的操作指导请参见[实名认证](https://support.huaweicloud.com/usermanual-account/account_auth_00001.html)。

## 获取元服务ID

元服务ID是元服务的唯一标识，在AGC控制台创建元服务后生成。元服务ID将在填写元服务信息中使用，您需提前记录下元服务ID。获取元服务ID的路径：[我的元服务](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html#/myApp)-进入相应元服务-应用信息-基本信息。

## 选择元服务的服务内容

在核准（备案）系统中填写元服务信息时，需要选择元服务的服务内容。请在**工信部**提供的目录分类中进行选择。

![](./img/d6328ab1.png)

若暂无精准匹配项，可选择类似项，并在互联网信息备注内填写元服务详细用途，类目/标签详见《[HarmonyOS元服务类目标签](https://developer.huawei.com/consumer/cn/doc/app/ability-0000002032931302)》。

## 准备附件材料

![](./img/8bb47b5a.png)

* 所有材料均需要为拍照/彩色扫描证件原件。
* 证件必须在有效期内，身份证有效期须≥90天。
* 证件不可隔屏拍摄，拍照/扫描的证件需边角完整，内容清晰（国徽不得遮挡）。
* 色调背景一致，证件内容无反光。
* 可以添加一行描述准确的水印，但需文字简洁且不遮挡证件上的重要信息。

### 办理前置审批文件

若元服务从事新闻、出版、药品和医疗器械、网约车等须经有关主管部门审核同意的互联网信息服务，在履行核准（备案）手续时，应提供业务对应前置审批文件。

通用前置审批类目参考如下：

| **前置审批类目** | **对应材料** | **办理部门（参考）** |
| --- | --- | --- |
| 新闻 | 《互联网新闻服务许可证》 | 省级及以上互联网信息办公室 |
| 出版（非游戏） | 《网络出版服务许可证》 | 国家新闻出版署、省级新闻出版部门 |
| 教育（校外培训） | 教育部门允许开展校外培训的审批文件 | 所在地教育行政管理部门 |
| 药品和医疗器械 | 《互联网药品信息服务资格证书》 | 省级食品药品监督管理局 |
| 文化 | 《网络文化经营许可证》 | 省级文化和旅游厅 |
| 广播电影电视节目 | 《信息网络传播视听节目许可证》 | 省级省委宣传部、新闻出版广电局 |
| 网络预约车 | 《网络预约出租汽车经营许可证》 | 所在地交通管理委员会 |
| 互联网金融类 | 金融监管部门审批许可文件 | 所在地金融监督管理局、人民银行、证监会 |
| 宗教类 | 《互联网宗教信息服务许可证》 | 省级以上人民政府宗教事务部门 |
