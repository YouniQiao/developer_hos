1. 登录[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)，点击“APP与元服务”。
2. 在HarmonyOS列表，选择您已发布的应用。
3. 在左上角应用名称右边点击 "运营"。
4. 在左侧导航栏选择“服务经营&gt; 服务直达”。
