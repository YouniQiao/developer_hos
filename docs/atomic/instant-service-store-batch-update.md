title: "批量变更门店"
original_url: /docs/dev/atomic-dev/instant-service-store/instant-service-store-batch-update
format: md
---


1. 在服务直达主界面，选择“门店管理”页签，根据门店状态或门店名称/编码筛选出需要变更的门店。

   ![](./img/450c0a5a.png "点击放大")
2. 点击“导出”。
   * 没有勾选项时，点击“导出”可导出全部门店。
   * 存在已勾选的门店时，点击“导出”仅导出已勾选门店。

   ![](./img/9a163560.png "点击放大")
3. 导出完成后，点击“任务管理”，下载导出结果。

   ![](./img/0a59ca37.png "点击放大")
4. 编辑表格，修改门店信息。
5. 点击“服务直达”,返回“门店管理”页签。

   ![](./img/df7fe543.png)
6. 点击“批量变更”，上传编辑后的表格文件，点击“上传”按钮。

   ![](./img/517cb22c.png "点击放大")

   ![](./img/1df8ab43.png "点击放大")
