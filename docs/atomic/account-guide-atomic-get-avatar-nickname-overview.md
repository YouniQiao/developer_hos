当元服务需要获取用户头像昵称时，可使用下述能力：

1、可使用Account Kit提供的头像昵称授权API接口能力，快速获取头像昵称。

2、可通过场景化控件[选择头像Button](/docs/dev/app-dev/application-services/scenario-fusion-kit-guide/scenario-fusion-button/scenario-fusion-button-chooseavatar)，实现头像信息获取与展示。
