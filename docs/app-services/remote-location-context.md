# RemoteLocationExtensionContext（定位扩展Context）

RemoteLocationExtensionContext是[RemoteLocationExtensionAbility](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/remote-location-ability)的上下文环境，继承自[ExtensionContext](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-application-extensioncontext)，未新增内容。RemoteLocationExtensionContext模块提供访问RemoteLocationExtensionAbility资源的能力。

模型约束： 属性仅可在Stage模型下使用。

系统能力： SystemCapability.Push.PushService

起始版本： 4.1.0(11)

#### 导入模块

```
import { RemoteLocationExtensionContext } from '@kit.PushKit';
```
