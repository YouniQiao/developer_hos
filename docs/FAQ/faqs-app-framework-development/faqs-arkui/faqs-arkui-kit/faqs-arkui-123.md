original_url: /docs/FAQ/faqs-app-framework-development/faqs-arkui/faqs-arkui-kit/faqs-arkui-123

系统规格要求rgba格式必须完整书写，如rgba(0, 0, 255, 0.5)中透明度值前导0不可省略。
