---
title: "编译报错“The service widget file contains one or more references to HSPs”"
original_url: https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-compiling-and-building-135
---

**错误描述**

服务卡片文件包含一个或多个HSP模块的引用。

**可能原因**

服务卡片文件中引用了HSP模块类型。

![](./img/f82bf469.png)

**解决措施**

在服务卡片文件中，移除关于HSP类型模块的引用。
