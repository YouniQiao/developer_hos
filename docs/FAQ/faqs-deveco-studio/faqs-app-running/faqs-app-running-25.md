x86模拟器支持已x86化的C/C++三方库调试运行，未x86化的三方库开发者可以参考[tpc\_c\_cplusplus](https://gitcode.com/openharmony-sig/tpc_c_cplusplus)。

目前已经x86化的三方库如下：

| 三方库 | 备注 |
| --- | --- |
| [MMKV](https://gitcode.com/openharmony-tpc/MMKV) | 自2.0.4-rc.4版本开始支持x86编译。 |
| [ImageKnife](https://gitcode.com/openharmony-tpc/ImageKnife/tree/master) | 自2.1.1版本开始支持x86编译。请重新安装imageknife再进行编译。 |
| [ohos\_videocompressor](https://gitcode.com/openharmony-sig/ohos_videocompressor) | 自1.0.3版本开始支持x86编译。 |
| [ohos\_coap](https://gitcode.com/openharmony-tpc/ohos_coap) | 自2.0.5版本开始支持x86编译。 |
| [Unrar](https://gitcode.com/openharmony-tpc/openharmony_tpc_samples/blob/9c65d80263f08049a64d23617e7089d914f876cf/unrar/README.md) | 自2.0.2版本开始支持x86编译。 |
| [commons-compress](https://gitcode.com/openharmony-tpc/CommonsCompress) | 自2.0.2-rc.0版本开始支持x86编译。 |
| [Smack](https://gitcode.com/openharmony-tpc/openharmony_tpc_samples/tree/master/ohos_smack) | 自2.0.1-rc.0版本开始支持x86编译。 |
| [ijkplayer](https://gitcode.com/openharmony-sig/ohos_ijkplayer) | 自2.0.3-rc.0版本开始支持x86编译。 |
| [ohos\_mqtt](https://gitcode.com/openharmony-tpc/ohos_mqtt) | 自2.0.5版本开始支持x86编译。 |
| [Mars](https://gitcode.com/openharmony-sig/mars) | 自2.0.1-rc.1版本开始支持x86编译。 |
| [mp4parser](https://gitcode.com/openharmony-tpc/mp4parser) | 自2.0.2-rc.0版本开始支持x86编译。 |
