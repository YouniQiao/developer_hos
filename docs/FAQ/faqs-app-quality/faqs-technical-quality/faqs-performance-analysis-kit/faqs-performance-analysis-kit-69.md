历史版本日志打印不规范，后续版本已修正日志打印级别。请参考[hdc版本配套表](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/hdc#hdc版本配套表)升级hdc版本到 3.1.0a或以上的版本。
