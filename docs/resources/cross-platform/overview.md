# 跨平台框架

HarmonyOS 生态支持多种跨平台开发框架，开发者可使用熟悉的框架和技术栈快速构建鸿蒙应用。

| 框架 | 项目地址 | 文档地址 |
|------|---------|---------|
| **ArkUI-X** | [arkui-x](https://gitcode.com/arkui-x) | [文档](https://gitcode.com/arkui-x/docs/tree/master/zh-cn) |
| **Flutter** | [flutter_flutter](https://gitcode.com/openharmony-tpc/flutter_flutter) | [文档](https://gitcode.com/openharmony-tpc/flutter_samples/tree/master/ohos/docs) |
| **React Native** | [ohos_react_native](https://gitcode.com/openharmony-sig/ohos_react_native) | [文档](https://gitcode.com/openharmony-sig/ohos_react_native/tree/master/docs/zh-cn) |
| **uni-app / uni-app x** | — | [uni-app 文档](https://uniapp.dcloud.net.cn/tutorial/harmony/intro.html) / [uni-app x 文档](https://doc.dcloud.net.cn/uni-app-x/app-harmony/) |

## ArkUI-X

ArkUI-X 是华为官方推出的跨平台框架，基于 HarmonyOS 的 ArkUI 声明式开发范式，将 ArkUI 能力扩展到 Android、iOS 等平台，实现一套代码多端运行。

## Flutter

Flutter for OpenHarmony 是 Flutter 框架在鸿蒙生态的适配版本，由社区维护。开发者可使用 Dart 语言和 Flutter 组件库构建 HarmonyOS 应用。

## React Native

React Native for OpenHarmony 将 React Native 框架适配到 HarmonyOS，支持 JavaScript/TypeScript 开发，复用 React 生态丰富的第三方库。

## uni-app / uni-app x

uni-app 是 DCloud 推出的跨端开发框架，支持一套代码编译到 HarmonyOS、iOS、Android、Web 等多个平台。uni-app x 是基于 uts 语言的下一代版本，提供更原生的鸿蒙开发体验。
