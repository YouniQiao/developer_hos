# InputMethod_TextEditorProxy

```
typedef struct InputMethod_TextEditorProxy InputMethod_TextEditorProxy
```

#### 概述

输入法文本编辑器代理类。提供了获取来自输入法应用的通知和请求的方法。当输入法向编辑器发送请求或通知时，这些方法将被调用。

起始版本： 12
